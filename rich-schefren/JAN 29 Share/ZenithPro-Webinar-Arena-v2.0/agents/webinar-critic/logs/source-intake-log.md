# Webinar Skills - Source Intake Log

Log of new webinar methodology material reviewed and integrated into the skill system.

---

## How to Use This Log

When new webinar expert material is discovered:

1. **Log the source** using the template below
2. **Analyze against existing frameworks** — what's new vs. already captured?
3. **Propose additions** to relevant skills
4. **Get approval** before integrating
5. **Update skill-changelog.md** when changes are made

---
