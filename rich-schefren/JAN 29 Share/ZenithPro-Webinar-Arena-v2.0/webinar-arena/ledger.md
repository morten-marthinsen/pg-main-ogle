## Purpose

Central record of all Arena learning: competitions, patterns detected, skill updates, research completed, and system evolution. This is the institutional memory of the Webinar Arena.

---

## Ledger Structure

### Competition Records

Each competition is logged with:
- Date and project name
- Competitors (experts + synthesis + wild cards)
- Winner and runner-up
- Judge reasoning summary
- Key insights extracted
- Outcome link (when reported)

### Pattern Records

Patterns detected across competitions:
- Pattern description
- Supporting evidence (which competitions)
- Affected experts
- Recommended action
- Status (proposed, validated, applied)

### Skill Update Records

Changes to expert skills:
- Update description
- Source (pattern, research, user feedback)
- Expert affected
- Version number
- Approval status

### Research Records

Deep research completed:
- Topic
- Triggered by (pattern, user request, gap)
- Findings summary
- Applied to (which skill)
- Quality score

## Ledger Queries

Use `/arena-ledger` to query this ledger:

- `/arena-ledger --expert=brunson` - Show Brunson's history
- `/arena-ledger --patterns` - Show detected patterns
- `/arena-ledger --recent=10` - Show 10 most recent entries
- `/arena-ledger --search="skeptical"` - Search ledger content
