## Purpose

Measure whether critic feedback actually improves draft quality. Track which critics and which critique types lead to genuine improvements.

---

## Effectiveness Calculation

### Improvement Rate

```
Improvement Rate = (Revisions Scored Higher) / (Total Revisions)
```

A critique is "effective" when:
- Revised draft scores higher than original on same criteria
- Revised draft addresses specific critique points
- Final version retains the revisions (not reverted)

### Critique Categories

| Category | Description | Example |
|----------|-------------|---------|
| Structural | Organization, flow, sections | "Transition feels abrupt" |
| Persuasion | Psychological impact | "Needs stronger social proof" |
| Voice/Tone | Style and delivery | "Too formal for this audience" |
| Technical | Mechanics, timing, pacing | "Close section runs too long" |
| Strategic | Overall approach | "Wrong methodology for this context" |
