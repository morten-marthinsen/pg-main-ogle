#### Phase 2: BUILD THE OPEN (Checkout Prevention)
1. Create checkout prevention elements
2. Establish who/why (even for known audiences - ASSUME NOTHING)
3. Stack stay-tuned tactics
4. **VALIDATE:** Will they stay past the first 5 minutes?

**Document:** [Key Open elements]

#### Phase 3: BUILD THE MIDDLE (Commitment Creation)
1. Map E-Factors relevant to offer
2. Design commitment creation sequence
3. Handle all possible objections DURING content
4. Plan 7-21 repetitions of key ideas
5. Mix content types for interest retention
6. **VALIDATE:** Are objections handled as teaching, not as defense?

**Document:** [Key Middle elements]

#### Phase 4: BUILD THE BRIDGE (No False Borders)
1. Plan bridges (acknowledged) and segues (seamless)
2. Integrate trial closes throughout
3. Design content-to-commercial bridge
4. Eliminate physiological change tells
5. **VALIDATE:** Does it feel like continued help, not a "pitch"?

**Document:** [Key Bridge elements]

#### Phase 5: BUILD THE CLOSE
1. Create clearly understood offer
2. Build price-to-value demonstration
3. Create believed scarcity (REAL, not manufactured)
4. Remove all buying friction
5. Plan risk reversal (it's just math)
6. **VALIDATE:** Is buying the logical conclusion?

**Document:** [Key Close elements]

## KENNEDY'S CORE PRINCIPLES

1. **No False Borders** - Content IS selling, selling IS content
2. **Checkout Prevention** - Stop them leaving before you start
3. **Assume Nothing** - They fell from sky into room, know nothing
4. **Media Agnostic** - Structure doesn't change by delivery method
5. **Repetition** - 7-21 times for key ideas to stick
6. **Believed Scarcity** - Real, enforced, legitimate
7. **No Tells** - Zero physiological change at transition
8. **Evergreen Core** - Timeless structure, replaceable hooks
