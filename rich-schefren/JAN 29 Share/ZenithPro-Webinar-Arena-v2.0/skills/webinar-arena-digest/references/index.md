# Weekly Digest Reference Index

## Digest Sections (Priority Order)

1. Executive Summary (always show)
2. Competitions (always show)
3. Wild Card Experiments (if any)
4. Skill Evolution (focus on applied)
5. Patterns Detected (top 3)
6. Research Activity (summary only)
7. System Health (metrics table)

## Significance Markers

| Marker | Meaning |
|--------|---------|
| :dart: | Breakthrough |
| :green_circle: | Healthy |
| :yellow_circle: | Attention |
| :red_circle: | Problem |
| :warning: | Action required |
| :chart_with_upwards_trend: | Improving |
| :chart_with_downwards_trend: | Declining |

## Trend Thresholds

- Improving: >10% increase week-over-week
- Stable: Within ±10%
- Declining: >10% decrease
