# Naming Formulas & Templates

> **Purpose:** Fill-in-the-blank templates and systematic formulas for generating 50+ mechanism name candidates for any concept.

---

## The Master Naming Process

### Phase 1: Extract the Core
Before naming, answer these questions:

1. **What is the mechanism?** (The underlying process/cause)
2. **What does it do?** (The action it takes)
3. **What's the benefit?** (The outcome for the prospect)
4. **Who is the villain?** (What causes the problem)
5. **What's the market?** (Health, wealth, dating, etc.)

### Phase 2: Generate Word Banks
Create lists of relevant words in these categories:

### Phase 3: Apply Templates

### Phase 4: Filter and Test
Score candidates using the criteria at the end of this document.

## Scoring and Selection Matrix

Rate each candidate 1-5 on these criteria:

| Criterion | Weight | Score (1-5) | Weighted |
|-----------|--------|-------------|----------|
| Memorability | 25% | ___ | ___ |
| Clarity | 20% | ___ | ___ |
| Credibility | 15% | ___ | ___ |
| Uniqueness | 15% | ___ | ___ |
| Emotional Impact | 15% | ___ | ___ |
| Speakability | 10% | ___ | ___ |
| **TOTAL** | 100% | | ___ |

### Scoring Guidelines:

**Memorability (25%)**
- 5 = Unforgettable after one hearing
- 3 = Remembered after a few exposures
- 1 = Easily forgotten

**Clarity (20%)**
- 5 = Instantly communicates what it does
- 3 = Requires brief explanation
- 1 = Confusing or unclear

**Credibility (15%)**
- 5 = Sounds completely legitimate
- 3 = Acceptable but could be questioned
- 1 = Sounds made-up or gimmicky

**Uniqueness (15%)**
- 5 = Completely ownable and differentiable
- 3 = Somewhat distinct
- 1 = Generic or already used by competitors

**Emotional Impact (15%)**
- 5 = Triggers strong curiosity or desire
- 3 = Moderately interesting
- 1 = Emotionally flat

**Speakability (10%)**
- 5 = Flows perfectly, easy to say and spell
- 3 = Minor awkwardness
- 1 = Tongue twister or hard to spell

## Quick Validation Tests

### Test 1: The Cocktail Party Test
"Would you mention this to a friend at a party?"
- Pass = "I learned about this thing called [Name]..."
- Fail = You wouldn't bring it up naturally

### Test 2: The 5th Grader Test
"Can a child understand the general concept?"
- Pass = Basic meaning is clear
- Fail = Requires technical knowledge

### Test 3: The Google Test
Search the exact phrase:
- Pass = Distinct from competitors
- Fail = Cluttered with similar results

### Test 4: The Spell Test
"Can people spell it correctly after hearing it?"
- Pass = Phonetically consistent
- Fail = Multiple spelling possibilities

### Test 5: The Share Test
"Is this something they'd share/repeat?"
- Pass = Inherently shareable
- Fail = Too complex or forgettable

---

## Final Selection Process

1. Generate 50 candidates using templates
2. Score top 20 using the matrix
3. Run top 10 through validation tests
4. Select top 3 for A/B testing
5. Test with real audience feedback
6. Finalize the winner
