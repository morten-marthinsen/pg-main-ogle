# Action-Oriented Names

> **Pattern Definition:** Mechanism names built around strong verbs that imply movement, transformation, and change - conveying that the mechanism DOES something powerful.

---

## Why Action Names Work

### The Psychology of Action

1. **Implied Dynamism**
   - Action words suggest something is happening
   - Creates sense of movement and progress
   - Contrasts with static, stagnant current state

2. **Agency and Control**
   - Verbs imply the mechanism takes action
   - Prospect feels they're enlisting an active force
   - "This will DO something for me"

3. **Speed and Results**
   - Action implies immediacy
   - Suggests the mechanism works quickly
   - Creates expectation of change

4. **Emotional Resonance**
   - Action words trigger emotional responses
   - More engaging than passive descriptions
   - Creates urgency and excitement

---

## The Core Formula

```
ACTION NAME = [Power Verb] + [Target/Outcome]
```

or

```
ACTION NAME = [Target] + [Action Noun]
```

## Intensity Calibration

### When to Use High-Intensity Verbs
- Frustrated, tried-everything prospects
- Pain-heavy markets (weight loss, chronic conditions)
- When differentiation requires boldness
- For breakthrough/discovery positioning

**High-Intensity Examples:**
- Destroy, Blast, Explode, Unleash, Supercharge, Turbo

### When to Use Medium-Intensity Verbs
- Health-conscious but not desperate
- Maintenance/optimization messaging
- When credibility is paramount
- Professional/medical-adjacent markets

**Medium-Intensity Examples:**
- Boost, Restore, Activate, Shield, Reset, Balance

### When to Use Low-Intensity Verbs
- Wellness-focused audiences
- Gentle/natural positioning
- When not making aggressive claims
- For sensitive health topics

**Low-Intensity Examples:**
- Support, Nurture, Enhance, Improve, Aid, Help

## Common Mistakes with Action Names

### Mistake 1: Weak Verbs
**Bad:** "Help Weight Management"
**Better:** "Torch Stubborn Fat"

### Mistake 2: Passive Voice
**Bad:** "Fat is Reduced"
**Better:** "Reduce Fat Fast"

### Mistake 3: Overclaiming
**Bad:** "Instantly Destroy All Disease"
**Better:** "Activate Your Body's Defense System"

### Mistake 4: Generic Actions
**Bad:** "Improve Health"
**Better:** "Ignite Cellular Energy"

### Mistake 5: Wrong Intensity for Market
**Bad:** "Annihilate Stress" (too aggressive for wellness market)
**Better:** "Calm the Cortisol Response"
