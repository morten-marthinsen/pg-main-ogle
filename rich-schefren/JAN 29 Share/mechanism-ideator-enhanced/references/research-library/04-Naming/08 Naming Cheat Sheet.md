## Instant Formula Templates

### Fill-in-the-Blank Starters

**Alliterative:**
- _____ _____ Formula (same letter)
- _____ _____ System
- _____ _____ Protocol

**Benefit-Embedded:**
- [Outcome] + [Process Word]
- Fat Flush, Sleep Switch, Energy Surge

**Scientific:**
- [Scientific Term] + Resistance/Dysfunction/Activation
- Leptin Reset, Cellular Restoration, Hormonal Balance

**Action:**
- [Power Verb] + [Target]
- Trigger Sleep, Blast Fat, Shield Health

**Numbered:**
- [Number]-[Time/Step] + [Benefit] + System
- 3-Step Fat Flush, 7-Day Reset, 5-Minute Method

## Sound Pattern Quick Guide

### Consonants

| Sound Type | Letters | Impression | Use For |
|------------|---------|------------|---------|
| Plosives | P, B, T, D, K, G | Power, impact | Strength, action |
| Fricatives | F, S, V, Z | Speed, flow | Quick results |
| Nasals | M, N | Soft, nurturing | Gentle, calm |

### Vowels

| Type | Letters | Impression | Examples |
|------|---------|------------|----------|
| Front | E, I | Small, light, fast | Slim, Fit, Speed |
| Back | O, U | Large, powerful | Boost, Bold, Force |

### Optimal Structure
- **Word length:** 6-8 characters ideal
- **Syllables:** 2-3 per word
- **Total words:** 2-3 maximum

## 5-Minute Name Generation Process

### Step 1: Core Concept (30 seconds)
Write in one sentence what the mechanism does:
_________________________________________________

### Step 2: Word Dump (1 minute)
List 5 words in each category:

**Action words:** _____, _____, _____, _____, _____
**Benefit words:** _____, _____, _____, _____, _____
**Target words:** _____, _____, _____, _____, _____

### Step 3: Pattern Application (2 minutes)
Generate 2 names per pattern:

| Pattern | Name 1 | Name 2 |
|---------|--------|--------|
| Alliterative | | |
| Benefit-Embedded | | |
| Scientific | | |
| Action | | |
| Personification | | |
| Numbered | | |
| Paradox | | |

### Step 4: Quick Filter (1 minute)
Circle your top 3 based on:
- Memorability
- Clarity
- Emotional impact

### Step 5: Speak Test (30 seconds)
Say each aloud 3 times. Winner = easiest to say.

## Validation Checklist

Quick yes/no for final candidates:

- [ ] Can I say it 3 times fast without stumbling?
- [ ] Would I mention this at a cocktail party?
- [ ] Could a 12-year-old understand the concept?
- [ ] Is it different from competitors?
- [ ] Can someone spell it after hearing it once?
- [ ] Does it trigger curiosity or desire?

**6/6 = Winner | 4-5/6 = Good | <4/6 = Revise**

## Red Flags to Avoid

| Problem | Example | Fix |
|---------|---------|-----|
| Too long | Revolutionary Metabolic Optimization | Metabolism Reset |
| Too generic | Health Improvement | Cellular Restoration |
| Unpronounceable | Gluconeogenetic | Glucose Guard |
| Negative lead | Disease Fighter | Health Shield |
| No imagery | Better Results System | Fat Flush Formula |
| Overclaiming | Miracle Cure-All | Vitality Restore |

## Emergency Name Generator

If stuck, use this formula:

```
[Adjective from market] + [Key benefit word] + [System word]
```

**Examples:**
- Cellular + Energy + Protocol = Cellular Energy Protocol
- Deep + Sleep + System = Deep Sleep System
- Metabolic + Reset + Method = Metabolic Reset Method
- Neural + Clarity + Formula = Neural Clarity Formula
