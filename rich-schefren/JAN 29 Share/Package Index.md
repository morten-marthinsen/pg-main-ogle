## 7. Dispatcher-Manager Agent

**Folder:** `Dispatcher-Manager-Agent/` | **Zip:** `Dispatcher-Manager-Agent.zip`
**Given to:** ZenithPro

An orchestration pattern that turns Claude into a project manager. Instead of you directing every step, you give Claude an objective and constraints, and it: (1) decomposes the objective into a task graph, (2) spawns persona-optimized sub-agents for each task (Dr. Elena Vasquez for analysis, Marcus Chen for creative, Sarah Okonkwo for validation, etc.), (3) enforces quality gates at three thresholds — Standard 70%, Elevated 85%, Critical 95%, and (4) reports back with options and recommendations. It never does the work itself — it manages sub-agents who do.

**What's inside (exact contents):**
- **AGENT.md** — the complete agent definition with personas, quality thresholds, task graph logic, advisor integration
- **README.md** — how to use the pattern
- **how-it-works.md** — visual explanation with diagrams
- **install.sh** + **install.ps1** — installers
