# ZenithPro Copy Arsenal - Workflow Selection

## Choosing Your Path

```mermaid
flowchart TD
NEED[What Do You Need?] --> CHOOSE{Choose Path}
CHOOSE --> SINGLE[Single Master]
CHOOSE --> ELEMENT[Element Skill]
CHOOSE --> ARENA[Arena Competition]
```

---

## Path 1: Single Master

```mermaid
flowchart TD
SINGLE[Single Master] --> DRAFT[Draft with Master]
DRAFT --> CRITIC[Critic Review]
CRITIC --> REVISE[Revise]
REVISE --> FINAL[Final Copy]
```

**Use when:** You know which approach fits best

---

## Path 2: Element Skill

```mermaid
flowchart TD
ELEMENT[Element Skill] --> TARGET[Target Specific Need]
TARGET --> APPLY[Apply Framework]
APPLY --> IMPROVE[Improved Element]
```

**Use when:** You need surgical improvement to one section

---

## Path 3: Arena Competition

```mermaid
flowchart TD

COMPETE --> BEST[Best Entry Wins]
BEST --> FINAL[Final Copy]
```

**Use when:** Maximum quality matters more than speed
