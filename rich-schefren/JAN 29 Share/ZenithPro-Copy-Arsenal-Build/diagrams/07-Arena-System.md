# ZenithPro Copy Arsenal - The Arena System

## What is the Arena?

The Arena is a competitive evolution engine where copywriters compete, learn, and improve automatically.

```mermaid
flowchart TD
ARENA[The Arena] --> COMPETE[Competition]
COMPETE --> JUDGE[Judgment]
JUDGE --> LEARN[Learning]
LEARN --> EVOLVE[Evolution]
EVOLVE --> BETTER[Better Copy]
```

```mermaid
flowchart TD
ARENA[arena folder] --> LEDGER[ledger.md]
LEDGER --> CONFIG[config.md]

```

| File | Purpose |
|------|---------|
| ledger.md | Overall statistics |
| config.md | Arena settings |
| win-records/ | Per-copywriter wins |
| synthesis-registry/ | Combination records |

## Win Record Tracking

```mermaid
flowchart TD
COMP[Competition Ends] --> WINNER[Winner Declared]
WINNER --> RECORD[Record Updated]
RECORD --> ANALYZE[Pattern Analysis]
ANALYZE --> INSIGHT[New Insights]
```

Each copywriter has their own win record file tracking what types of projects they win.
