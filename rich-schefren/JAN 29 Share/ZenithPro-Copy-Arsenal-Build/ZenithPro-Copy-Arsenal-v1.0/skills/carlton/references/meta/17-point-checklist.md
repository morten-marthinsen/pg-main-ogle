# Carlton 17-Point Checklist

Complete verification checklist for the Simple Writing System.

---

## Phase 1: Research & Foundation (Steps 1-4)

### Step 1: Market Research
- [ ] Competition identified and analyzed
- [ ] Market size validated
- [ ] Money flow confirmed (will they pay?)
- [ ] Positioning opportunity identified
- [ ] Hot Seat questions answered (Who are you? What do you sell? Why? What do you do for customers?)

### Step 2: Avatar
- [ ] Specific person defined (not general audience)
- [ ] Demographics complete (age, gender, location, income, occupation)
- [ ] Psychographics deep (pain, desire, fears, frustrations)
- [ ] Language documented (how they actually talk)
- [ ] Could describe them to a stranger

### Step 3: USP
- [ ] Unique positioning developed
- [ ] Non-clever slogan format ("We help [group] do [benefit] [better]")
- [ ] Id Ticklers present (faster, easier, simpler, unfair advantage, inside info)
- [ ] NOT fallen in love with USP (it's just setup, not the sale)
- [ ] Targets average prospect only (not everyone)

### Step 4: Features & Benefits
- [ ] All features listed
- [ ] "That means..." applied to each
- [ ] Benefits gone deep (chained multiple levels)
- [ ] 4 Horsemen checked (fast, easy, simple, cheap)
- [ ] Real desire identified (not surface desire)

---

## Phase 2: Core Elements (Steps 5-7)

### Step 5: Bullets
- [ ] One-two punch present in each bullet
  - [ ] Punch One: Practical use OR positioning
  - [ ] Punch Two: Shock/delight OR assault on common sense
- [ ] Multiple templates used
- [ ] Named techniques where appropriate
- [ ] Giveaway bullets for goodwill
- [ ] Specifics, not generalities

### Step 6: Testimonials
- [ ] At least 3 testimonials
- [ ] Each is SHORT (gets to point quickly)
- [ ] Each is SPECIFIC (real numbers, real results)
- [ ] Each is SPICY (emotional punch, memorable)
- [ ] Different voices (sound like different people)
- [ ] Results-based, not feeling-based

### Step 7: Hooks
- [ ] Incongruous juxtaposition present
- [ ] Two elements that don't belong together
- [ ] Creates cognitive dissonance
- [ ] Demands resolution (must keep reading)
- [ ] Immediately transitions to credibility
- [ ] Would jolt someone from zombie state

---

## Phase 3: Salesmanship (Steps 8-10)

### Step 8: Salesmanship Mentality
- [ ] Go-To Guy positioning established
- [ ] NOT selling from heels
- [ ] "I have what you need" confidence
- [ ] Money that belongs to you mindset
- [ ] If you have the goods, share them

### Step 9: Voice & Story
- [ ] Bar room conversation feel
- [ ] First person throughout
- [ ] Specifics over generalities
- [ ] No throat clearing (starts where action is)
- [ ] Story structure (setup, plot, punchline)
- [ ] Iconic situation established

### Step 10: Headlines
- [ ] Derived from hook (not separate)
- [ ] Would shock out of zombie state
- [ ] Appropriate length for message
- [ ] Visual hierarchy (headline + deck if needed)
- [ ] Gets them to read first sentence (single purpose)

---

## Phase 4: Structure (Steps 11-12)

### Step 11: Sales Funnel
- [ ] Campaign type defined (evergreen, launch, JV, etc.)
- [ ] Delivery system chosen (email, video, website, etc.)
- [ ] Platform limitations known
- [ ] Audience expectations considered
- [ ] Offer elements planned (price, value, bonuses, guarantee)

### Step 12: Basic Sales Message
- [ ] Here's who I am (established)
- [ ] Here's what I've got for you (clear)
- [ ] Here's why it's important right now (compelling)
- [ ] Here's what to do now (specific)
- [ ] So What? test passed (every paragraph earns its place)

---

## Phase 5: Persuasion & Close (Steps 13-17)

### Step 13: Huge Promise
- [ ] Truthful promise (can deliver)
- [ ] Big promise (worth caring about)
- [ ] Specific promise (not vague)
- [ ] Fixes what's wrong
- [ ] Eliminates fears
- [ ] Fortifies confidence

### Step 14: Risk Reversal
- [ ] Guarantee in right place (after desire, before close)
- [ ] Not used as selling point (unless USP)
- [ ] Specific timeframe
- [ ] Specific result expected
- [ ] Tone of insistence (YOU insist, not they request)
- [ ] Risk truly reversed ("You are not risking anything")

### Step 15: Turbulence
- [ ] At least one tool deployed:
  - [ ] Urgency (time-based)
  - [ ] Limitations (quantity/access)
  - [ ] Takeaway (fear of missing)
  - [ ] Opportunity (unique chance)
- [ ] Believable for context
- [ ] Would cause sleepless night
- [ ] Creates mental turbulence (can't let it go)

### Step 16: Knock 'Em Off Fence
- [ ] Bonuses add value
- [ ] Each bonus worth buying alone
- [ ] Stack creates overwhelming value
- [ ] At least one "order just for this" bonus
- [ ] Fast-action bonus if appropriate

### Step 17: Close
- [ ] 5-Part structure:
  - [ ] Here's the deal (summary)
  - [ ] Here's what you get (list)
  - [ ] Here's what it costs (with contrast)
  - [ ] Here's your guarantee (risk reversal)
  - [ ] Here's what to do now (walk-through)
- [ ] Assumes the sale (no hedging)
- [ ] First person intimacy
- [ ] Specific action steps
- [ ] Walk-through instructions

---

## Final Validation

### Voice Check
- [ ] Passes bar room test (sounds natural spoken)
- [ ] First person throughout (especially close)
- [ ] Specific details (not generalities)
- [ ] No throat clearing (starts where action is)
- [ ] Contractions used ("you're" not "you are")

### One-Two Punch Check
- [ ] Rational reasons present (for spouse/neighbor)
- [ ] Emotional A-Brain triggers present
- [ ] Both working together

### So What? Test
- [ ] Every paragraph answers "So what?"
- [ ] No sections that could be deleted
- [ ] Every section serves reader's self-interest

### Zombie State Check
- [ ] Hook would jolt them awake
- [ ] Energy maintained throughout
- [ ] No flat/boring sections
- [ ] Would remember it tomorrow

### Go-To Guy Check
- [ ] Positioned as THE authority
- [ ] Selling forward, not from heels
- [ ] Confidence without arrogance
