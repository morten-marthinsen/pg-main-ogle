# Evaldo Close Frameworks Index

Quick reference for all close frameworks.

## Core Close Methods

| Framework | Description | Use When |
|-----------|-------------|----------|
| close-stacking | Building value systematically | Presenting offer |
| close-price-anchor | Making price feel small | Price presentation |
| close-guarantee | Risk reversal presentation | Removing barriers |
| close-urgency | Creating immediate action | Final push |

## Advanced Close

## The Close Answers Questions 8-10

8. What do I get? (The Stack)
9. What does it cost? (Price Anchor + Drop)
10. What do I do now? (CTA + Urgency)

## The Close Sequence

1. Transition from body
2. Reveal what they get (Stack)
3. Establish value (Price Anchor)
4. Drop actual price (Contrast)
5. Add bonuses (More value)
6. Present guarantee (Risk removal)
7. Create urgency (Act now)
8. CTA (Simple action)
9. Re-close (Second pass)

## The Most Important Truth

**The close doesn't sell - it makes the sale official.**

By the time you reach the close, they should already want it. The lead created desire. The body built belief. The close just removes the final barriers and makes it easy to act.
