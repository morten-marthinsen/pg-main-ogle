## How to Install (5 minutes)

### Step 1: Unzip This Package

If you're reading this, you probably already did this. The folder should contain:
- `install.sh` (Mac installer)
- `install.ps1` (Windows installer)
- `skills/` folder
- `agents/` folder
- `docs/` folder
- Other supporting files

### Step 2: Put the Folder Where You Want It

Move the `ZenithPro-Copy-Arsenal-v1.0` folder wherever makes sense for you. If you use Obsidian, put it in your vault with your other copywriting materials.

**Remember where you put it.** You'll need the location for the next step.

### Step 3: Open Claude Code CLI

Launch the Claude Code desktop application on your computer.

### Step 4: Tell Claude to Run the Installer

**On Mac**, type this in Claude Code:

```
Run this command in bash: /path/to/ZenithPro-Copy-Arsenal-v1.0/install.sh
```

Replace `/path/to/` with wherever you put the folder.

**On Windows**, type this in Claude Code:

```
Run this PowerShell command: C:\path\to\ZenithPro-Copy-Arsenal-v1.0\install.ps1
```

Replace `C:\path\to\` with wherever you put the folder.

### Step 5: Enter Your Information

The installer will ask for your name and email. This is for license tracking. Type your responses when prompted.

### Step 6: Done

The installer copies all skills and agents to the correct locations. You'll see a summary of what was installed.
