# Deutsch Critic - Critique Log

This log captures findings from every Deutsch Critic evaluation. Used for pattern detection and skill improvement.

---

## How to Use This Log

1. **After each critique**, append a new entry using the template below
2. **Periodically review** (every 5-10 critiques) to detect patterns
3. **If same failure appears 3+ times**, it's a SKILL GAP — propose update to skill
4. **Link fixes to skill-changelog.md** when skills are updated

---

```markdown
## [DATE] | [PROJECT/COPY NAME] | Grade: [X]

### Key Failures

- [Failure 2]: [Brief description]
- [Failure 3]: [Brief description]

### Root Cause Analysis

### Skill Gap Identified?
- Skill: [which skill - deutsch or deutsch-critic]
- Gap: [what's missing]
- Frequency: [first time / 2nd time / 3+ times = UPDATE NEEDED]

### Fixes Applied to Output
- [Fix 1]
- [Fix 2]

### Notes
[Any additional observations]
```

---

## Pattern Detection Thresholds

| Occurrence | Classification | Action |
|------------|---------------|--------|
| 1st time | Note | Log in critique-log.md |
| 2nd time | Watch | Flag as potential pattern |
| 3rd time | SKILL GAP | Propose skill update |
| 4th+ time | CRITICAL | Update skill immediately |
