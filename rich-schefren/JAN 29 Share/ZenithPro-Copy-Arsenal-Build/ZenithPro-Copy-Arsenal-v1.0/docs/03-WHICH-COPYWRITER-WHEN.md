# Which Copywriter When
## The Decision Tree for Choosing Your Approach

---

## The Quick Answer

If you're in a hurry:

| Situation | Use This |
|-----------|----------|
| Need emotional punch | Clayton |
| Need to stop the scroll | Carlton |
| Sophisticated/educated audience | Deutsch |
| Writing a VSL | Evaldo |
| Not sure | Run the Arena |

## When to Use the Arena

**Always consider the Arena when:**

1. You're not sure which approach is best
2. You're writing something important (main sales page, VSL)
3. You want to see multiple approaches compared
4. You want the skills to improve over time
5. You have time for a thorough process

**Skip the Arena when:**

1. Quick surgical fix (just the headline)
2. You know exactly which style fits
3. Time-sensitive (need it in 10 minutes)

---

## Combination Strategies

Sometimes the best approach combines multiple masters:

### The "Stop and Build" Combo
- **Carlton hook** (stop them)
- **Evaldo body** (build belief)
- **Clayton close** (drive action)

### The "Sophisticated Punch" Combo
- **Deutsch opening** (trust building)
- **Clayton proof** (overwhelming evidence)
- **Deutsch close** (future fork)

### The "VSL Power" Combo
- **Evaldo structure** (10 Questions)
- **Carlton hooks** (pattern interrupts throughout)
- **Clayton emotional beats** (intensity)

**The Arena's Synthesis competitor automatically explores these combinations.**
