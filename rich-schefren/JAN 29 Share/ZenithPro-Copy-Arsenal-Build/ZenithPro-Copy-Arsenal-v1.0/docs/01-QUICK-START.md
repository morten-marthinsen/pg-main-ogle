## Your First Command (2 minutes)

Let's improve a headline right now.

### Step 1: Open Claude Code CLI

Open your terminal and start Claude Code:
```
claude
```

### Step 2: Run the Headlines Skill

Type:
```
/clayton-headlines
```

### Step 3: Give It Your Headline

When Claude prompts you, paste in a headline you want to improve. For example:

```
"How to Get More Clients for Your Consulting Business"
```

### Step 4: Watch the Magic

Clayton's methodology will:
1. Evaluate your headline against his "A-Brain" test (does it trigger primal emotion?)
2. Check for emotional trigger words
3. Generate 5+ improved versions
4. Explain what makes each one stronger

**Example output:**
```
Original: "How to Get More Clients for Your Consulting Business"
Clayton Rating: 3/10 (engages thinking brain, not lizard brain)

Improved versions:
1. "THEY LIED TO YOU About Why You're Not Getting Clients"
2. "The $47,000 Mistake Every Consultant Makes (Are You Next?)"
3. "WARNING: Your 'Marketing' Is Repelling Your Best Clients"
4. "EXPOSED: The Real Reason Consultants Struggle While Amateurs Thrive"
5. "SHAMELESS Gurus Left You With a Graveyard of Failed Tactics"
```

## Try a Different Style (1 minute)

Each copywriter has a different approach. Try the same headline with Carlton:

```
/carlton-hooks
```

Carlton specializes in "incongruous juxtaposition" - surprising combinations that stop the scroll.

**Carlton might produce:**
```
"The Consultant Who Made $2M Last Year... Working 3 Days a Week... From a Coffee Shop in Bali... While His 'Competitors' Burned Out Chasing Clients"
```

Different style. Different strengths. Same goal: better copy.

## Run Your First Competition (2 minutes)

Want to see all four copywriters compete on your project?

### Step 1: Start the Arena

```
/arena my-headline 1
```

### Step 2: Provide the Brief

When prompted, describe what you need:

```
I need a headline for a landing page.
Target: Strategic consultants who've tried marketing tactics without success.
Offer: A free positioning assessment.
Tone: Sophisticated, not hype-y.
```

### Step 3: Watch Them Compete

The Arena will:
1. Have all 4 copywriters draft their version
2. Have critics evaluate each draft
3. Have copywriters revise based on feedback
4. Have the Marketplace Judge declare a winner
5. Show you all 5 versions (4 pure + 1 synthesis)

### Step 4: Pick Your Winner

You get to see 5 different approaches and choose the one that resonates most.

## Pro Tips

### 1. Be Specific
The more context you give, the better the output:
- Who is the audience?
- What have they tried before?
- What's the price point?
- What's the awareness level?

### 2. Use Element Skills for Surgical Fixes
Don't rewrite your entire landing page if only the headline is weak. Use `/clayton-headlines` just for headlines.

### 3. Use the Arena for Big Projects
Writing a full sales letter? Run the Arena. You'll get 5 complete versions and learn what works best for your specific project.

### 4. Read the Output Explanations
Each skill doesn't just give you copy - it explains WHY it works. This teaches you the methodology.
