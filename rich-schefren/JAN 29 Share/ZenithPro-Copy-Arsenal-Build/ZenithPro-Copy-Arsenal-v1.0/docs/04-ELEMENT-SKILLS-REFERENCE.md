## How to Use This Reference

Each skill is designed for a specific copy element or task. Use this reference to find exactly what you need:

```

```

The skills are organized by copywriter, then by element type.

## Skill Naming Convention

All skills follow this pattern:
```
[copywriter]-[element]
```

Examples:
- `clayton-headlines` = Clayton's headline frameworks
- `carlton-hooks` = Carlton's hook frameworks
- `deutsch-leads` = Deutsch's lead frameworks
- `evaldo-close` = Evaldo's close frameworks

The main orchestrator for each copywriter is just their name:
- `/clayton` = Full Clayton workflow
- `/carlton` = Full Carlton workflow
- `/deutsch` = Full Deutsch workflow
- `/evaldo` = Full Evaldo workflow
