## Email Sequence Architecture

### The Launch Pattern
```
1. Intrigue (Coming Soon)
2. Story (Why This Exists)
3. Proof (Who It's Worked For)
4. Open (The Offer Revealed)
5. Deep Dive (Features → Benefits)
6. Objections (Handle Resistance)
7. Urgency (Deadline Pressure)
```

### The Evergreen Pattern
```
1. Value First (Pure teaching)
2. Story Bridge (Connect to offer)
3. Soft Pitch (Low pressure CTA)
4. Proof Stack (Build belief)
5. Direct Offer (Clear ask)
```

### The Re-Engagement Pattern
```
1. Pattern Interrupt (Unexpected subject)
2. Acknowledgment (Address silence)
3. New Angle (Fresh perspective)
4. Value Bomb (Immediate usefulness)
5. Simple CTA (Easy next step)
```
