# Workflow: Landing Page Creation
## From Brief to High-Converting Page

---

## When to Use This Workflow

Use this workflow when you need to:
- Create a new landing page from scratch
- Rewrite an underperforming landing page
- Create multiple landing page versions to test

---

## Option 1: Single Copywriter Approach

If you know which copywriter fits your audience, use their full workflow:

### For Skeptical/Burned Audiences
```
/evaldo

BRIEF:
Product: [What you're selling]
Price: $[Amount]
Audience: [Description - emphasize they've been burned before]
Key mechanism: [What makes this work]
Desired action: [What you want them to do]
```

### For Emotional/Desperate Audiences
```
/clayton

BRIEF:
Product: [What you're selling]
Price: $[Amount]
Audience: [Description - emphasize their pain]
Key promise: [What transformation they'll experience]
Available proof: [Testimonials, stats, credentials]
```

### For Sophisticated/B2B Audiences
```
/deutsch

BRIEF:
Product: [What you're selling]
Price: $[Amount]
Audience: [Description - emphasize their sophistication]
Key story: [Transformation story if available]
Trust elements: [Credentials, client list, etc.]
```

### For Jaded/Seen-Everything Audiences
```
/carlton

BRIEF:
Product: [What you're selling]
Price: $[Amount]
Audience: [Description - emphasize what they've seen before]
Unique angle: [What makes this different]
Personality: [Brand voice direction]
```

## Option 2: Arena Competition (Recommended)

Let all four masters compete and see what wins:

```
/arena landing-page 1

BRIEF:
Product: [What you're selling]
Price: $[Amount]
Target market: [Who they are, what they've tried, what they believe]
Key promise: [The transformation]
Available proof: [Testimonials, case studies, stats]
Key objections: [What prevents them from buying]
Constraints: [Tone requirements, length limits, etc.]

This is for a landing page. Each copywriter should produce a complete page.
```

**What You Get:**
- 5 complete landing pages (4 copywriters + synthesis)
- Critical evaluation of each
- Revised versions after feedback
- Judge's winner prediction with reasoning
- Learning briefs for understanding what worked

## The Complete Landing Page Structure

Regardless of which copywriter you use, a landing page needs:

### 1. The Hook/Headline
```
/[copywriter]-headlines
```

### 2. The Opening/Lead
```
/[copywriter]-leads
```

### 3. The Body (Story, Proof, Mechanism)
```
/[copywriter]-body
```

### 4. The Bullets/Benefits
```
/[copywriter]-bullets
```

### 5. The Offer
```
/[copywriter]-offers
```

### 6. The Close
```
/[copywriter]-closes
```

## Mix-and-Match Approach

Sometimes the best landing page combines multiple masters:

### "Stop and Build" Formula
Best for: Jaded audiences who need belief building

```
Step 1: /carlton-hooks
Create an attention-grabbing headline

Step 2: /evaldo-leads
Build belief gradually in the opening

Step 3: /evaldo-body
Continue belief building with mechanism

Step 4: /clayton-proof
Add overwhelming evidence

Step 5: /clayton-closes
Drive urgent action
```

### "Sophisticated Punch" Formula
Best for: Educated audiences who distrust marketing

```
Step 1: /deutsch-headlines
Sophisticated, non-salesy headline

Step 2: /deutsch-leads
Trust-building story opening

Step 3: /clayton-proof
Proof cascade (made elegant)

Step 4: /deutsch-close
Future-fork emotional close
```

### "VSL Power" Formula
Best for: Video sales letter landing pages

```
Step 1: /evaldo (full workflow)
10 Questions VSL structure

Step 2: /carlton-hooks
Add pattern interrupts throughout

Step 3: /clayton-bullets
Intensify the bullet sections
```

## Landing Page Brief Template

Use this template when requesting any landing page:

```
PRODUCT/SERVICE:
[Name and description]

PRICE POINT:
$[Amount]

TARGET MARKET:
- Who they are:
- What they've tried before:
- Why those things failed:
- What they believe now:
- What they secretly want:

THE PROMISE:
[What transformation they'll experience]

THE MECHANISM:
[Why/how this works when other things didn't]

AVAILABLE PROOF:
- Testimonials:
- Case studies:
- Statistics:
- Credentials:
- Expert endorsements:

KEY OBJECTIONS:
1. [Objection]
2. [Objection]
3. [Objection]

CONSTRAINTS:
- Tone: [e.g., sophisticated, urgent, friendly]
- Length: [if limited]
- Design: [if specific format needed]

CALL TO ACTION:
[What you want them to do]
```

## After the Landing Page

Once you have your landing page:

1. **Run it through the relevant critic**
   ```
   /[copywriter]-critic
   Evaluate this landing page against the methodology
   ```

2. **Test headlines separately**
   ```
   /arena headline-test 1
   ```

3. **Check for holes**
   - Does it address all major objections?
   - Is there enough proof for the claims?
   - Is the CTA crystal clear?

| Need to stand out | `/carlton` |
