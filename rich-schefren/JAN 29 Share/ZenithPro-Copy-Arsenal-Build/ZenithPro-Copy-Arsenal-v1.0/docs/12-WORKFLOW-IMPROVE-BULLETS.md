# Workflow: Improve Bullets
## Transform Weak Fascinations Into Irresistible Curiosity

---

## When to Use This Workflow

Use this workflow when you:
- Have bullet points that aren't creating desire
- Need to strengthen a bullet section
- Want to test multiple bullet approaches
- Are optimizing an existing sales page

---

## The Four Masters of Bullets

Each copywriter approaches bullets differently:

| Copywriter | Bullet Philosophy | Best For |
|------------|-------------------|----------|
| **Clayton** | Emotional triggers | Desire activation |
| **Carlton** | One-Two Punch specificity | Credibility + curiosity |
| **Deutsch** | Polished sophistication | Educated audiences |
| **Evaldo** | Open loops | Pure curiosity |

---

## Step 1: Diagnose Your Current Bullets

Before improving, understand what's missing:

```
Analyze these bullets for effectiveness:

[Paste your current bullets]

What's working?
What's missing?
Which would make someone stop and read more?
Which fall flat?
```

---

## Step 2: Choose Your Bullet Approach

### For Maximum Curiosity (Evaldo)

```
/evaldo-bullets

Here are my current bullets:
[List them]

Topic: [What you're selling]
Audience: [Who they are]

Transform these using curiosity mechanics:
- Open loops that MUST be closed
- Hints at mechanisms without revealing
- "I have to know" triggers
```

**Evaldo Bullet Pattern:**
> "The strange [specific detail] that explains why [unexpected result]... (page X)"

### For Emotional Triggers (Clayton)

```
/clayton-bullets

Here are my current bullets:
[List them]

Audience pain points: [What they fear/desire]
Key emotions to trigger: [Fear, greed, curiosity, etc.]

Transform these to trigger emotional response.
```

**Clayton Bullet Pattern:**
> "How to [specific benefit] that triggers [emotional state]—even if [obstacle they fear]"

### For Specificity + Credibility (Carlton)

```
/carlton-bullets

Here are my current bullets:
[List them]

Product mechanism: [What makes it work]
Specific details available: [Numbers, stories, facts]

Transform using One-Two Punch methodology:
- First punch: Specific detail that creates credibility
- Second punch: Benefit that creates desire
```

**Carlton One-Two Punch Pattern:**
> "The 23-minute ritual a bankrupt accountant from Toledo used to [specific result]—and why it works even if [objection]"

### For Sophisticated Audiences (Deutsch)

```
/deutsch-bullets

Here are my current bullets:
[List them]

Audience sophistication level: [High]
What they've seen before: [Types of marketing]

Polish these for sophisticated readers who hate "marketing speak."
```

**Deutsch Bullet Pattern:**
> "Why [counterintuitive insight] often matters more than [expected factor]—and how to use this to [specific outcome]"

## Bullet Diagnostic Checklist

Before finalizing, verify each bullet:

**Curiosity Check:**
- [ ] Creates an open loop
- [ ] Makes them NEED to know more
- [ ] Doesn't reveal the answer

**Specificity Check:**
- [ ] Contains specific details (numbers, names, places)
- [ ] Feels real, not generic
- [ ] Has credibility markers

**Benefit Check:**
- [ ] Clear "what's in it for me"
- [ ] Connects to their desires
- [ ] Promises transformation

**Uniqueness Check:**
- [ ] Couldn't be said about competitors
- [ ] Sounds fresh, not clichéd
- [ ] Has an unexpected element

## Example Transformations

**Original (Weak):**
> "Learn how to write better copy"

**After Evaldo (Curiosity):**
> "The counter-intuitive 'backwards writing' method that makes sales copy write itself... even if you've never written a word of marketing in your life"

**After Clayton (Emotional):**
> "How to write copy that reaches into your prospect's chest and squeezes their heart—until buying is the only relief they can imagine"

**After Carlton (Specificity):**
> "What a 67-year-old retired shoe salesman in Akron discovered about writing copy after failing for 11 years... and why it works when nothing else does"

**After Deutsch (Sophistication):**
> "Why the copy that 'sounds like copy' almost always fails—and the elegant principle that separates amateur persuasion from professional influence"

## Bullet Section Architecture

Strong bullet sections follow patterns:

### The Crescendo Pattern
Start mild, build intensity:
```
• Informational bullet
• Benefit bullet
• Emotional bullet
• "Can't live without" bullet
```

### The Variety Pattern
Mix different trigger types:
```
• Curiosity bullet (open loop)
• Proof bullet (specific result)
• Fear bullet (what to avoid)
• Hope bullet (future vision)
```

### The Proof-Stacking Pattern
Build credibility through specifics:
```
• Specific number result
• Named person testimonial
• Unusual detail that proves depth
• Unexpected benefit
```

## How Many Bullets?

| Copy Type | Typical Count | Notes |
|-----------|---------------|-------|
| Landing page | 7-15 | Key benefits only |
| Sales letter | 15-30 | More comprehensive |
| VSL | 12-20 | Need to be speakable |
| Email | 3-7 | Shorter, punchier |
