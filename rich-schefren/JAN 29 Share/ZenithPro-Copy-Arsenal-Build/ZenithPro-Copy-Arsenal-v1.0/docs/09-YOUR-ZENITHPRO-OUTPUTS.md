## Example: Full Integration

**Your Strategic Work:**
```
Avatar: "The Frustrated Expert"
- Core desire (L1): Significance, mastery
- They believe: "I'm great at what I do but terrible at marketing"
- They've tried: Courses, agencies, tactics
- Why it failed: Applied tactics without strategy

  engineering demand through belief architecture"
```

**Feeding to Copy Arsenal:**
```
/arena landing-page 3

STRATEGIC CONTEXT:

TARGET: The Frustrated Expert
- Experts earning $5K-50K/month
- Exceptional at delivery, struggling with demand generation
- Tried: Online courses, marketing agencies, social media tactics
- All failed because: Applied tactics without understanding strategy
- Now believe: "I must be bad at marketing" or "Marketing is sleazy"

CORE DESIRE: Significance, mastery (L1)
- Want to be recognized for expertise
- Want predictable client flow without feeling salesy

CORE CONCEPT:
"Marketing isn't about tactics—it's about engineering demand
through belief architecture. You're not bad at marketing—
you've been using the wrong mental model entirely."

POINT A → POINT B:
A: Believes marketing is a necessary evil, feels inauthentic
B: Sees marketing as extension of expertise, feels aligned

PRODUCT: [Your offer]
PRICE: $[Amount]

Create a landing page that moves them from Point A to Point B.
```

## Pro Tips

### 1. Always Include Point A → Point B
The copywriters work better when they understand the belief journey:
```
POINT A: [Current beliefs/state]
POINT B: [Required beliefs for purchase]
```

### 2. Name the Enemy (If Identified)
If your strategic work identified an enemy positioning:
```
THE ENEMY: [What or who represents the failed past approach]
```

### 3. Include Proof Inventory
List what you have available:
```
AVAILABLE PROOF:
- Testimonials: [Y/N, how many]
- Case studies: [Y/N, topics]
- Credentials: [List]
- Statistics: [List]
```

### 4. Share the Sophistication Level
From your psychographic work:
```
MARKET SOPHISTICATION:
[1-5, with 5 being "seen everything"]
```
