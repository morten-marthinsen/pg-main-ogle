# Workflow: Improve Headlines
## Transform Weak Headlines Into Scroll-Stoppers

---

## When to Use This Workflow

Use this workflow when you:
- Have a headline that's not converting
- Need to test multiple headline approaches
- Want to strengthen an existing headline
- Are optimizing a control

---

## The 3-Step Headline Improvement Process

### Step 1: Diagnose the Current Headline

Before improving, understand what's wrong. Ask Claude Code:

```
Analyze this headline using Clayton's emotional triggers framework:
"[Your current headline]"

What's working? What's missing? What emotional triggers could be added?
```

**What You'll Learn:**
- Which of the 29 emotions it triggers (or doesn't)
- Whether it passes the "lizard brain" test
- Specific weakness areas

### Step 2: Generate Alternative Headlines

Run your headline through multiple masters:

**For Emotional Punch:**
```
/clayton-headlines

Here's my current headline: "[Your headline]"
Target audience: [Describe them]
Key promise: [What they get]

Generate 10 headline alternatives that trigger stronger emotional response.
```

**For Stopping Power:**
```
/carlton-hooks

Here's my current headline: "[Your headline]"
Target audience: [Describe them]

Generate 10 headline alternatives using incongruous juxtaposition
to create pattern interrupts.
```

**For Sophisticated Audiences:**
```
/deutsch-headlines

Here's my current headline: "[Your headline]"
Target audience: [Sophisticated B2B buyers / educated professionals]

Generate 10 headline alternatives that pass the 4 C's + 2 H's test.
```

**For VSL Headlines:**
```
/evaldo-headlines

Here's my current headline: "[Your headline]"
This is for a VSL about: [Topic]

Generate 10 headline alternatives with open loops and curiosity triggers.
```

### Step 3: Competitive Selection

If you want ALL four masters to compete:

```
/arena headline-test 1

BRIEF:
Current headline: "[Your headline]"
Audience: [Description]
Product: [What you're selling]
Current performance: [If known - CTR, conversion, etc.]

Each copywriter should generate their best 3 headline alternatives.
```

The Arena will:
1. Have each master generate alternatives
2. Critics evaluate against their frameworks
3. Revisions made
4. Judge selects the predicted winner

## Headline Diagnostic Checklist

Before finalizing, verify your headline:

**Clayton Test (Emotional Triggers):**
- [ ] Triggers at least one of the 29 emotions
- [ ] Passes the "croc brain" filter (simple, concrete, emotional)
- [ ] Creates urgency or curiosity
- [ ] Promises a specific benefit

**Carlton Test (Stopping Power):**
- [ ] Would make someone stop scrolling
- [ ] Contains an unexpected element
- [ ] Speaks in their language (not marketing-speak)
- [ ] Has personality

**Deutsch Test (4 C's + 2 H's):**
- [ ] Clear - They understand what it's about
- [ ] Compelling - They want to know more
- [ ] Credible - They believe it's possible
- [ ] Customer-focused - It's about THEM
- [ ] Hope - It promises a better future
- [ ] Help - It offers assistance

**Evaldo Test (Curiosity Mechanics):**
- [ ] Opens a loop that must be closed
- [ ] Doesn't give away the answer
- [ ] Creates a "must know" feeling
- [ ] Hints at a mechanism or secret

## Testing Your Headlines

Once you have alternatives, test systematically:

1. **Internal gut check:** Which makes YOU want to read more?
2. **Audience check:** Show to someone in your target market
3. **A/B test:** If you have traffic, test the top 2-3
4. **Arena prediction:** The judge's predicted conversion can guide testing priority
