# Sample Brief: Designed to Trigger Synthesis Wins

## Copy This Into Your Arena

```
Brief: Create a webinar selling a $997 12-week program teaching corporate professionals how to transition to entrepreneurship. Target audience is successful managers earning $150K+ who feel trapped but are risk-averse. They need transformation but hate hype. They want practical steps but also emotional permission to make the leap. The webinar must work both live AND as an automated evergreen funnel.

Price: $997
Market sophistication: 4
Format: Hybrid (must work live AND automated)
Rounds: 5
```

---

## Why This Brief Triggers Synthesis

This brief has deliberately conflicting requirements:

| Requirement | Favors |
|-------------|--------|
| Transformation story | Brunson, Joon |
| No hype | Kennedy, Kern |
| Practical steps | Fladlien, Kennedy |
| Emotional permission | Brunson, Cage |
| Works live | Brunson, Joon |
| Works automated | Fladlien, Kennedy |

**No single expert can satisfy all requirements.**

---

## Expected Outcome

If this brief works as designed:
- Pure methodologies will each have weaknesses exposed
- Synthesis will combine strengths to address all requirements
- Synthesis has a real chance to win all 5 rounds

---

## If Synthesis Wins All Rounds

Watch for the spawning trigger:

1. Synthesis-critic will assess codification potential
2. If B+ or higher, spawning sequence starts
3. You'll be asked to name the new methodology
4. A new permanent expert will be created

---

## What to Watch For

- Which elements Synthesis pulls from each expert
- How the combination addresses conflicting requirements
- The synthesis-critic's codification assessment

---

## Running Multiple Times

If you run this brief multiple times and synthesis doesn't win all rounds, it means the combination wasn't consistently better. That's valuable data too.

If synthesis DOES win all rounds, you may have discovered a new methodology worth codifying.

---

## Try Variations

If synthesis keeps winning with similar combinations, that's a signal the combination addresses a common need that pure methodologies miss.
