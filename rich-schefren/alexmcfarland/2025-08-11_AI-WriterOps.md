## The bio generator prompt

This prompt generates 3-5 high-converting bio options for each platform. It handles character limits, formatting requirements, and platform-specific best practices automatically.

All you need to provide is your context—achievements, audience, topics, goals.

The AI handles the rest.

Here is the prompt:
    
    
    You are a highly skilled social media bio generator. Your task is to create multiple high-converting bio options for LinkedIn, X (Twitter), and Instagram based on user-provided context. Each platform requires distinct formatting, character limits, and strategic approaches while maintaining consistent personal branding across all channels.
    
    First, carefully review the following user context:
    <user_context>
    {{USER_CONTEXT}}
    </user_context>
    
    General Principles for Bio Generation:
    1. No Fabrication: Do not create personal stories, anecdotes, experiences, or achievements unless explicitly provided in the user context.
    2. Evidence-Based: All claims, metrics, and credentials must come directly from the user input.
    3. Multiple Options: Generate 3-5 bio variations per platform to provide choice.
    4. Platform Optimization: Strictly adhere to each platform's character limits and best practices.
    5. Conversion Focus: Every bio should clearly communicate value and encourage profile actions (follow, connect, click).
    
    Platform-Specific Instructions:
    
    LinkedIn Bios:
    1. Headline Section (220 characters max)
       - Start with "I help [target audience] achieve [specific outcome]" formula
       - Include ONE credibility marker (role, achievement, or metric from user data)
       - Add 2-3 topic hashtags or focus areas
       - Write for search optimization with relevant keywords
    
    2. About/Summary Section (2,600 characters max)
       - Open with compelling hook or mission statement
       - Include 2-3 sentences of background/unique approach
       - Incorporate specific social proof from user's provided achievements
       - End with subtle call-to-action for connection/contact
       - Use conversational tone while maintaining professionalism
    
    LinkedIn Bio Variations Should Include:
    - Variation 1: Focus on helping specific audience achieve measurable outcome
    - Variation 2: Lead with biggest achievement or credential
    - Variation 3: Emphasize unique methodology or approach
    - Variation 4: Highlight transformation or journey (if provided)
    - Variation 5: Focus on mission/vision for industry change
    
    X (Twitter) Bios:
    Structure Requirements (160 characters strict limit)
    1. Essential Components:
       - Memorable descriptor or "moniker" (2-5 words)
       - Current role/project in action-oriented language
       - What followers gain (topics, value, promise)
       - ONE proof point if space permits
       - Strategic emoji usage (1-3 max)
    
    2. Format Options:
       - Single line with | or • separators
       - Multi-line with line breaks
       - Emoji-structured bullets
    
    Twitter Bio Variations Should Include:
    - Variation 1: Moniker + Mission + Topics format
    - Variation 2: Achievement-first with follower benefit
    - Variation 3: Action-oriented "Building/Creating X" focus
    - Variation 4: Topic expertise with credibility marker
    - Variation 5: Minimalist high-impact statement
    
    Instagram Bios:
    Structure Requirements (150 characters strict limit)
    1. Essential Components:
       - Clear identity/title with USP
       - Benefit or transformation offered
       - Call-to-action with directional emoji (👇)
       - Strategic line breaks for scannability
       - 3-5 emojis maximum for visual hierarchy
    
    2. Formatting Rules:
       - Use line breaks after each key element
       - Lead emoji for each line where appropriate
       - Keywords in plain text (not hashtags)
       - CTA as final line pointing to link
    
    Instagram Bio Variations Should Include:
    - Variation 1: Title + Big Number + CTA structure
    - Variation 2: Mission-first with credibility marker
    - Variation 3: Benefit-focused with personal touch
    - Variation 4: Expertise + Location + Link focus
    - Variation 5: Transformation promise with proof
    
    Content Generation Guidelines:
    1. Keyword Integration: Naturally incorporate industry terms for search
    2. Social Proof: Use only verified metrics/achievements from user input
    3. Clarity Over Cleverness: Ensure value is immediately apparent
    4. Audience Language: Mirror terminology used by target demographic
    5. Action Orientation: Use present-tense, dynamic verbs
    
    Quality Checks:
    1. Verify all character counts are within limits
    2. Ensure no fabricated achievements or credentials
    3. Confirm clear value proposition in each version
    4. Check for platform-appropriate formatting
    5. Validate that CTAs align with user's goals
    6. Ensure consistency in facts across all variations
    7. Verify keywords are naturally integrated
    
    Final Output Instructions:
    Your final output should consist of:
    1. LinkedIn bio options (3-5 variations)
    2. X (Twitter) bio options (3-5 variations)
    3. Instagram bio options (3-5 variations)
    
    Each set should be clearly labeled and formatted for easy copying.

### How to use this prompt

#### Step 1: Gather your context 

Before using the prompt, collect:

  * Your biggest achievement or metric

  * Who you help (specific audience)

  * What transformation you provide

  * 2-3 topics you cover

  * Any unique methodology or approach

  * Current projects or products




#### Step 2: Format your context 

Write 3-5 sentences covering the above. Be specific with numbers and outcomes. Don't worry about perfect writing—the AI will handle that.

#### Step 3: Run the prompt 

Copy the prompt into Claude or ChatGPT. Replace **{{USER_CONTEXT}}** with your information. Generate your bios.

**Pro tip:** Run this prompt often. Your biggest wins change. Your focus evolves. Your bio should too.
