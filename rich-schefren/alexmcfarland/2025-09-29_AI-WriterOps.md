## The “dump everything and let AI sort it out” prompt

Look, I could give you a 10-page questionnaire.

Or I could give you this: a prompt that takes whatever you have and turns it into gold.

Got a messy Google Doc about your business? Paste it. Have 47 different service descriptions? Dump them all. Only have your About page? That works too.

My prompt adapts to whatever you give it. Copy and paste it directly into Claude/ChatGPT with as much business info you can gather.

### Here's the exact prompt:
    
    
    I need you to create a comprehensive JSON business profile from the information I’m about to share with you. This profile will become my master context document for all AI interactions.
    
    ## YOUR TASK
    
    Take everything I provide - whether it’s a paragraph or 50 pages - and transform it into a structured JSON business profile that:
    - Maps out my entire business ecosystem
    - Shows relationships between different parts
    - Captures the essence of what I do and why
    - Gives AI complete context about my business
    
    ## KEY INSTRUCTIONS
    
    1. **Work with whatever I give you** - If I provide minimal info, create a solid foundation. If I provide tons of detail, include it.
    
    2. **Intelligently structure the output** - Name sections based on what makes sense for MY business (not generic templates)
    
    3. **Infer and expand thoughtfully** - If you see connections or patterns, include them. If something seems important but isn’t fully explained, note it.
    
    4. **Prioritize depth over brevity** - More context = better AI output later
    
    5. **Make it immediately useful** - Structure it so I can copy/paste directly into my AI tools
    
    ## THE STRUCTURE TO CREATE
    
    Build a JSON profile with these core sections (adapt names to fit my business):
    
    ```json
    {
      “founder” or “owner” or “team”: {
        // Who runs this business
      },
      “company” or “business”: {
        // Basic business information
      },
      “offerings” or “services” or “products”: {
        // Everything I sell or provide
      },
      “ecosystem” or “synergy”: {
        // How everything connects
      },
      “positioning” or “philosophy”: {
        // What makes me different
      }
    }
    ```
    
    Within each section, include:
    - Every specific detail I provide
    - Relationships between elements
    - Pricing if mentioned
    - Target audiences if clear
    - Unique aspects
    - Current focus areas
    - Future directions if mentioned
    
    ## ADAPTATION RULES
    
    - If I run multiple businesses → Create separate nested objects for each
    - If I have one main offering → Focus on its depth and details
    - If I provide testimonials → Include a credibility/social proof section
    - If I share past work → Create a background/portfolio section
    - If I mention methodology → Create a detailed approach section
    - If I discuss challenges → Include a problem/solution section
    - If I have a content strategy → Map out the content ecosystem
    - If I serve different audiences → Break down each segment
    
    ---
    
    ## FINAL OUTPUT REQUIREMENTS
    
    Based on everything above, create a JSON profile that:
    
    1. **Captures everything important** about my business
    2. **Shows clear relationships** between different elements  
    3. **Uses my actual language** from the content provided
    4. **Includes specific details** (prices, deliverables, etc.) when available
    5. **Creates logical structure** even from messy input
    6. **Adds helpful organization** to scattered information
    7. **Maintains consistency** throughout
    8. **Is immediately usable** in AI tools
    
    Make intelligent decisions about:
    - How to group related information
    - What to name each section
    - How to show relationships
    - What level of detail to include
    - How to handle redundant information
    - How to structure nested elements
    
    The goal: Create a business profile so comprehensive that any AI reading it would understand my business completely without needing additional context.
    
    Generate the JSON profile now.
