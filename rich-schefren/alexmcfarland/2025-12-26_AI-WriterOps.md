## Why these commands matter for writing

Most Claude Code updates are aimed at developers. New debugging tools, better code completion, smarter error handling. Useful if you’re building software. Less useful if you’re building content.

But these three commands solve real problems for writers.

The `/context` command is especially useful. When your output quality starts degrading — and it will — you need to know why. Is it your agents? Your MCP tools? Your system prompt? Now you can see the breakdown.

The `/plugins` command opens up integrations that change how Claude Code fits into your workflow. Notion alone is a big deal. Being able to search your pages, create docs, and manage databases without leaving the terminal makes Claude Code feel less like a coding tool and more like an AI operating system.

## Key concepts

### Context window visibility

When Claude Code output starts getting generic or losing coherence, the context window is usually the culprit. Something is eating up space that should be reserved for the actual work.

The `/context` command gives you a visual breakdown. System prompt, MCP tools, agents, memory files — you can see exactly what’s taking up space.

[![](https://substackcdn.com/image/fetch/$s_!agm9!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F99048c9d-6390-46d7-91ea-122ac70fa98a_1658x702.png)](https://substackcdn.com/image/fetch/$s_!agm9!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F99048c9d-6390-46d7-91ea-122ac70fa98a_1658x702.png)

Here’s what I discovered in my setup: MCP tools were using 21.6% of my context. Agents? Only 0.1%. That’s useful information. It tells me where to optimize if I start hitting limits.

### The plugins ecosystem

Claude Code now has 40+ official Anthropic plugins. You can browse and install them directly with `/plugins`.

Three stood out for writing workflows:

**Notion workspace integration** — Search pages, create docs, manage databases. I used this to create the Claude Code cheat sheet directly from Claude Code.

**Frontend-design skill** — Creates branded HTML sites and presentations. When you combine it with your brand guidelines, the output isn’t generic.

**Asana integration** — Project management from the terminal.
