## **Why Claude Code beats regular Claude for writing**

If you’re still using the regular Claude website or app, you’re behind.

Claude Code is just different.

I’ll be honest—I put off switching to Claude Code for way too long. I thought it was too technical. A coding thing. Not for writers.

I was wrong. It’s taken my operations to a completely different level over the past few months.

* * *

## **Step 1: Download the Claude desktop app**

Go to [claude.ai/download](https://claude.ai/download).

Download the app for Mac or Windows. Install it, open it up.

You’ll see the regular Claude interface you’re used to.

## **Step 2: Toggle on Claude Code**

In the top left corner, you’ll see a toggle to switch between regular Claude and Claude Code.

Toggle it on.

Now you’re in Claude Code mode—inside the same desktop app. No terminal. No Cursor. No extra setup.

[![](https://substackcdn.com/image/fetch/$s_!1_64!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F40d679b0-600a-454d-927f-15d82431ebba_3840x2160.png)](https://substackcdn.com/image/fetch/$s_!1_64!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F40d679b0-600a-454d-927f-15d82431ebba_3840x2160.png)

## **Step 3: Create your writing folder**

Before connecting Claude Code to anything, create a folder on your computer. This is going to be your writing system.

You can use Obsidian (what I use), or just a regular folder in Finder. Doesn’t matter.

What matters is that you start putting everything in there:

  * Writing samples

  * Newsletters

  * Notes and ideas

  * Context profiles

  * Archives




This folder becomes Claude’s brain. Everything you store there, Claude Code can access.

[![](https://substackcdn.com/image/fetch/$s_!1o3N!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fc3990aac-b6a7-45ee-a9f6-af71df06b1d9_930x368.png)](https://substackcdn.com/image/fetch/$s_!1o3N!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fc3990aac-b6a7-45ee-a9f6-af71df06b1d9_930x368.png)

* * *

## **Step 4: Connect Claude Code to your folder**

Back in the Claude desktop app (with Claude Code toggled on), click to select a folder.

Navigate to the writing folder you just created. Select it.

Now Claude Code has access to everything inside.

**Important:** Always back up your folder. Claude Code can read, write, and execute files. Back it up so you don’t lose anything if something goes wrong.

[![](https://substackcdn.com/image/fetch/$s_!Yx8M!,w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Ff94cf4d2-ecfa-4b2e-9197-3ac7d44aea6c_2608x1320.png)](https://substackcdn.com/image/fetch/$s_!Yx8M!,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Ff94cf4d2-ecfa-4b2e-9197-3ac7d44aea6c_2608x1320.png)

## **Step 5: Start building**

You don’t need a complete system on day one.

Start documenting. Start storing. The first newsletter you write, save it in there. The first voice sample, add it. Your notes, your ideas, your archives—all of it.

Over time, this compounds. Claude gets smarter because it has more context to work with.

I can literally ask Claude: “Go through my entire system and tell me what you understand about my writing business.”

And it does. It reads my curriculum, my newsletters, my context profiles, my skills—everything. Then it gives me a complete picture.

That’s the power here. Claude actually knows you.

## **The one thing you need: Voice DNA**

If you’re going to use Claude Code for writing, you need a voice DNA profile.

This is a JSON document that captures how you write—your tone, your patterns, your personality. Not the specific words you use, but the overall picture of what you sound like.

Without it, Claude will still write generic stuff. With it, the output actually sounds like you.

### **→[Get the Voice DNA prompt here](https://chemical-clematis-3fe.notion.site/voice-dna-creator-prompt-2b2aade8881480c798b9e31ff95cb836)**

Run it in Claude. Attach 3-5 of your best writing samples. It’ll generate your voice DNA profile.

Save it in your folder. Now Claude Code can read it every time you write.
