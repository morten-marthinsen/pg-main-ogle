# Cognitive Biases for Copy

Cognitive biases are mental shortcuts our brains use to make decisions quickly. In marketing, these biases are leverage points—ethical ways to reduce friction and align messaging with how humans actually think.

## High-Impact Biases

### Anchoring Bias
**What it is**: First piece of information heavily influences subsequent judgments.

**Copy Application**:
- Introduce higher price point before revealing actual price
- Start with large number before showing smaller one
- Use "compared to" framing

**Examples**:
```
"Agencies charge $15,000/month for this. You get it for $997."
"Save $847 (normally $1,294)"
"Most people spend 40 hours learning this. You'll get it in 4."
```

### Scarcity Bias
**What it is**: We value things more when they're rare or diminishing.

**Copy Application**:
- Limited time offers
- Limited quantity
- Exclusive access
- Disappearing bonuses

**Examples**:
```
"Only 12 spots left at this price"
"Cart closes Friday at midnight—no exceptions"
"First 50 buyers get [bonus]. 23 remaining."
```

**Warning**: False scarcity destroys trust. Only use when genuine.

### Authority Bias
**What it is**: We defer to perceived experts and high-status sources.

**Copy Application**:
- Lead with credentials
- Quote recognized experts
- Associate with trusted institutions
- Use specific numbers (more authoritative than round numbers)

**Examples**:
```
"Based on research from Stanford's behavioral lab..."
"As featured in Forbes, Inc., and Entrepreneur..."
"After 847 client engagements across 12 industries..."
```

### Social Proof
**What it is**: We look to others' behavior to determine our own.

**Copy Application**:
- Testimonials with specific results
- User counts
- "Most popular" labels
- Real-time activity notifications

**Examples**:
```
"Join 47,000+ marketers who..."
"Rated 4.9/5 by 2,847 customers"
"Sarah from Austin just enrolled (2 minutes ago)"
```

## Social Proof Biases

### Bandwagon Effect
**What it is**: Tendency to adopt behaviors/beliefs as more people do.

**Copy Application**:
- Show growing momentum
- Emphasize community size
- Highlight trending status

**Examples**:
```
"The strategy sweeping through the marketing world"
"Why 1,000 new members join every week"
"This is becoming the new standard"
```

### Groupthink
**What it is**: Desire for conformity within a group.

**Copy Application**:
- Create in-group identity
- Show consensus
- "Everyone knows" framing

**Examples**:
```
"Top performers agree: this is the missing piece"
"The approach elite agencies have quietly adopted"
```

### Outgroup Homogeneity Bias
**What it is**: We see our group as diverse, other groups as uniform.

**Copy Application**:
- Position competitor users as "all the same"
- Highlight individuality within your community

**Examples**:
```
"While they all follow the same outdated playbook..."
"Our community attracts mavericks, not copycats"
```

## Decision-Making Biases

### Loss Aversion
**What it is**: Losses feel ~2x more painful than equivalent gains.

**Copy Application**:
- Frame as what they'll lose without action
- Emphasize cost of inaction
- "Don't miss" > "Get access"

**Examples**:
```
"Every day you wait costs you $247 in lost revenue"
"Don't let another month slip by with the same problems"
"Stop leaving money on the table"
```

### Zero-Risk Bias
**What it is**: Preference for eliminating small risks entirely over reducing large risks.

**Copy Application**:
- Offer "risk-free" guarantees
- Remove all friction points
- Eliminate objections completely

**Examples**:
```
"Zero risk. Full refund if you're not satisfied."
"No credit card required. No strings attached."
"If you don't see results, you don't pay."
```

### Default Effect
**What it is**: People stick with pre-selected options.

**Copy Application**:
- Pre-select recommended option
- Make desired action the default
- Opt-out > Opt-in framing

**Examples**:
```
"Most popular" badge on middle tier
Pre-checked "Yes, I want the bonus training"
```

### Effort Justification
**What it is**: We value things more when we work harder for them.

**Copy Application**:
- Add (reasonable) application process
- Create qualification criteria
- Position offer as "earned"

**Examples**:
```
"Applications are reviewed within 48 hours"
"We only work with businesses doing $1M+ in revenue"
"This program requires commitment. Not everyone qualifies."
```

### Ben Franklin Effect
**What it is**: We like people more after doing them a favor.

**Copy Application**:
- Get small favor before asking for sale
- Ask for feedback, input, or opinion
- Create micro-engagements

**Examples**:
```
"Help us improve—take this 30-second survey"
"Reply with your #1 challenge"
"Share this with someone who needs it"
```

## Memory & Attention Biases

### Zeigarnik Effect
**What it is**: We remember incomplete tasks better than completed ones.

**Copy Application**:
- Open loops in storytelling
- Cliffhangers between emails
- "Continue to the next page to discover..."

**Examples**:
```
"But here's what they didn't tell you..."
"In the next email, I'll reveal the technique that..."
"The third step might surprise you (page 2)"
```

### Peak-End Rule
**What it is**: We judge experiences based on peak moment and ending.

**Copy Application**:
- End sales pages with strongest testimonial
- Make last email in sequence most compelling
- Create memorable close

**Examples**:
Strong close: Powerful case study + inspiring vision + clear CTA
Peak moment: "This is the part where everything changes..."

### Humor Effect
**What it is**: We remember funny things better.

**Copy Application**:
- Strategic humor in hooks
- Memorable analogies
- Pattern breaks with humor

**Examples**:
```
"Your email list is like a houseplant. Ignore it and... well, you know."
"Their 'marketing strategy'? Praying the algorithm shows mercy."
```

### Generation Effect
**What it is**: We remember information better when we generate it ourselves.

**Copy Application**:
- Ask questions they answer mentally
- "Imagine if..." scenarios
- Interactive elements

**Examples**:
```
"What would you do with an extra $10K/month?"
"Picture your business 12 months from now..."
"How much time do you waste on [task]?"
```

## Risk & Loss Biases

### Hyperbolic Discounting
**What it is**: We prefer smaller immediate rewards over larger future rewards.

**Copy Application**:
- Emphasize immediate benefits
- "Start seeing results today"
- Instant gratification hooks

**Examples**:
```
"See your first results within 24 hours"
"Start implementing tonight"
"Instant access—start learning in 3 minutes"
```

### Pseudocertainty Effect
**What it is**: We make riskier choices when outcomes are framed as certain.

**Copy Application**:
- Frame stages as guaranteed progression
- "When you [action], you will [result]"
- Certainty language on intermediate steps

**Examples**:
```
"Step 1 guarantees you'll understand the fundamentals. Step 2 ensures you can implement..."
```

### Illusory Truth Effect
**What it is**: Repeated statements are perceived as more true.

**Copy Application**:
- Repeat core claims across touchpoints
- Consistent messaging in campaigns
- Key phrase repetition within copy

**Examples**:
```
Email 1: "Speed wins in this market."
Email 3: "Remember: speed wins."
Email 5: "Here's why speed wins—every time."
```

## Application Matrix

### Bias Selection by Goal

| Goal | Primary Biases | Application |
|------|---------------|-------------|
| Increase urgency | Scarcity, Loss Aversion | Limited time/quantity + cost of waiting |
| Build credibility | Authority, Social Proof | Expert credentials + user numbers |
| Drive action | Default Effect, Zero-Risk | Pre-selected options + guarantees |
| Improve recall | Zeigarnik, Humor | Open loops + memorable lines |
| Increase perceived value | Anchoring, Effort Justification | Price comparison + qualification |
| Reduce objections | Zero-Risk, Authority | Guarantees + expert validation |

### Bias Stacking Examples

**High-Converting Sales Page Stack**:
1. **Hook**: Zeigarnik (open loop)
2. **Social Proof**: Authority + Bandwagon
3. **Value Stack**: Anchoring (high comparison price)
4. **Guarantee**: Zero-Risk (eliminate fear)
5. **CTA**: Scarcity + Loss Aversion
6. **Close**: Peak-End (strongest testimonial)

**Email Sequence Stack**:
1. **Email 1**: Zeigarnik (open loop) + Generation Effect (question)
2. **Email 2**: Authority + Social Proof (case study)
3. **Email 3**: Loss Aversion (cost of inaction)
4. **Email 4**: Ben Franklin (ask for favor) + Bandwagon
5. **Email 5**: Scarcity + Zero-Risk (deadline + guarantee)
