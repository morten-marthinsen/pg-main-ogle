## Critical Decisions

### Architecture (S001)
- **Hub-and-spoke orchestration** (NOT sequential pipeline like Tess/Veda)
- **8 sub-agents in 3 layers**: Context Gatherer, Audience Intelligence, Ad Angle Ideation, Ad Hook Generation, Ad Script Generation, Influencer Brief Generation, Static Image Brief Generation, Quality Validator
- **Six-Axis Discipline** = cross-cutting operating principle (Focus → Suggestibility → Compliance)
- **Golf suggestibility insight**: "For golfers, emotion IS in the education"
- **Scope**: Ads only — no email/funnels/landing pages
- **Sub-agent pattern**: Boris Cherny methodology + backstory pattern
- **Static Image Brief**: Absorbed from global CLAUDE.md into Sub-Agent #7
- **Influencer Brief Template**: SSP brief (Google Doc 1gtD7OJ0nS9FfXHIC5zVz5YoOIyjsL27ZEiRUoLVMQos)
