## Key Tactical Takeaways

1. **Three distinct roles, never combined**: Strategist (data analysis, scripts, shot lists, QA), Creator (raw footage only, no editing), Editor (assembly, effects, post-production). Combining roles degrades output quality.

2. **Weekly production cadence**: Monday-Tuesday = shoot, Wednesday-Thursday = edit, Friday = client delivery. 4-12 ads per client per week depending on package size.

3. **Creator whitelisting is standard**: Every ad can run from both the brand's page and the creator's page. This is built into every engagement, not an add-on.

4. **Long-term creator relationships over one-offs**: Some creators have been with Savannah since 2021 (5+ years). Consistent work, on-time pay, bonus programs, and comprehensive briefs retain top talent.

5. **Cross-client hook testing**: When a hook concept works for one client, test it across 3+ other clients/industries simultaneously. If it performs broadly, it becomes a "slam dunk hook" added to the winning concept library.

6. **Visual hook variations > text-only changes**: Meta's algorithm treats ads as "different creations" only when the visual in the first 3-5 seconds is meaningfully different. Changing only overlay text does NOT count as a new variation.

7. **Andromeda adaptation = persona + value driver diversity**: The core response to Andromeda is more buyer persona testing (broad opener vs. specific persona callout) and more diverse value driver scripts (same product, different purchase motivations).

8. **Master hook list with linked examples**: Maintain a running database of every hook tested, linked to the actual ad. Searchable, organized, prevents repetition, and provides instant inspiration when stuck.

9. **Transition and font databases**: Keep curated libraries of transitions (with example links) and fonts/colors that can be copy-pasted into creator and editor briefs. This is what enables speed at scale.

10. **Winner iteration framework**: When a winner emerges, test in four directions: (a) same script + different hook, (b) same script + different creator, (c) same ad + different clip sequence, (d) same ad + shorter duration (30s → 20s → 15s). Isolate one variable per test.

### Winner Expansion Framework (from Q&A)

When a winner emerges, test in this order:
1. **Same creator, same value drivers, same messaging → new hook** (visual or spoken)
2. **Different creator → same script, same shot list, same everything** (isolates: was it the creator or the concept?)
3. **Same ad → rearranged clip sequence or shortened duration** (30s → 20s → 15s)
4. **Accept that some winners have "magic sauce"**: BlendJet 2020 example — best performer for years, all iterations underperformed the original. "Sometimes the algorithm just picks something and likes it."
