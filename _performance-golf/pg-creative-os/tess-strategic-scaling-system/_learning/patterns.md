# Tess Patterns

> Recurring patterns observed across sessions. When a pattern reaches L2 (confirmed across 2+ sessions), it's logged here. When promoted to L4+ (human approved), it becomes a rule in CLAUDE.md or TESS-MASTER-AGENT.md.

## Pattern Template

```yaml
pattern_id: "PAT-YYYY-MM-DD-NNN"
date_first_observed: YYYY-MM-DD
occurrences: 1
l_level: "L1"  # L1-L6
j_class: "J1"  # J1 (judgment-free, testable) or J2 (judgment-required, needs human taste)
pattern: |
  [What keeps happening — specific behavior or outcome]
evidence: |
  [Sessions where this was observed]
proposed_action: |
  [What should change to codify this pattern]
status: "observing"  # observing, testing, promoted, embedded
```
