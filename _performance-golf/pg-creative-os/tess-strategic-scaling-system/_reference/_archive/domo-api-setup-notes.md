## Recommended Approach

### Connect directly to Meta Marketing API

**What's needed:**
1. Create a Meta Developer App (takes ~5 minutes)
2. Generate an Access Token with `ads_read` permissions
3. Get the Ad Account ID(s)

**Steps to create Meta API access:**

1. **Go to:** https://developers.facebook.com/
2. **Log in** with the Facebook account that has ad account access
3. **Create App:**
   - Click "My Apps" → "Create App"
   - Select "Other" for use case
   - Select "Business" as app type
   - Name it (e.g., "PG Ad Analysis")
   - Connect to Performance Golf's Business Manager
4. **Get Access Token:**
   - Go to Tools → Graph API Explorer
   - Select your app
   - Click "Generate Access Token"
   - Grant permissions: `ads_read`, `ads_management`, `business_management`
5. **Get Ad Account ID:**
   - Found in Meta Ads Manager URL: `act_XXXXXXXXX`
   - Or: Business Settings → Ad Accounts
