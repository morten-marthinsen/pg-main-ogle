# 2.2 — Media Mention Writer

**Skill:** LP-08-trust-elements
**Position:** Layer 2, Microskill 2
**Purpose:** Write the "As Seen In" section copy — section header, per-publication callout sentences, and optional pull-quote framing. If no media mentions exist in the trust inventory, this microskill produces a `not_applicable` output and terminates.
**Effort Level:** high (when applicable)
**Dependencies:** `trust-classification.md` (1.1), `trust-placement-map.md` (1.3), `trust-inventory.md` (0.2)
**Output:** `media-mention-copy.md`

---

## Input Schema

```yaml
input:
  media_mention_status: "[from trust-classification.md — included | not_applicable]"
  media_mentions: "[from trust-inventory.md — list of publications with available data]"
  placement: "[from trust-placement-map.md — which section]"
  page_type: "[type_a | type_b | hybrid]"
```

---

## Execution Logic

### If `not_applicable`:
Write `media-mention-copy.md` with:
```
Status: not_applicable
Reason: No verifiable media mentions in trust inventory.
```
Terminate this microskill.

### If `included`:

1. **Select section header copy.** Options:
   - "As Featured In" (most professional)
   - "As Seen In" (most recognized — default choice)
   - "In the Press" (editorial feel)
   - "Media Coverage" (neutral)
   - Custom: "[Brand] in the News"

2. **For each publication, write a 1-sentence callout** that pulls the most credible claim from the actual mention:

   **Callout writing rules:**
   - Maximum 1 sentence per publication
   - Lead with the publication name (bolded for scanning)
   - Pull the single most credible, specific claim from the mention
   - If a direct quote exists, use it with quotation marks
   - If only the topic is known, frame the coverage topic
   - NEVER fabricate a quote or claim that is not in the trust inventory

   **Callout formats:**
   - With quote: `**Forbes:** "[Direct quote from article]"`
   - With claim: `**Men's Health:** Named "Best [Category] of [Year]" in their annual guide`
   - With topic: `**CNN:** Featured for [specific topic — e.g., "pioneering third-party transparency in supplement manufacturing"]`

3. **Apply page type differentiation:**
   - **Type A:** Full callout sentences with quotes where available. Placed as a dedicated section or inline block.
   - **Type B:** Logo strip with minimal copy (publication names only, or 3–5 word tags). Placed below fold.

4. **Verify every mention is in the trust inventory** — no fabrication.

5. **Write `media-mention-copy.md`.**

## Quality Criteria
- `not_applicable` output produced cleanly if no media mentions exist (no fabrication to fill the gap)
- Each callout is exactly 1 sentence — no multi-sentence callouts
- Direct quotes used where available (more credible than paraphrasing)
- Every claim traceable to trust inventory
- Page type differentiation applied (Type A gets full callouts, Type B gets logo strip)

## Failure Modes
- Fabricating media mentions for publications that never covered the brand → If not in inventory, it does not exist. `not_applicable` is the correct output.
- Writing generic callouts ("Leading publication praises [Brand]") → Must pull the specific claim or topic from the actual mention.
- Multi-sentence callouts → 1 sentence max per publication. Callouts must be scannable.
- Missing the `not_applicable` exit → If no media mentions exist, do not attempt to write this element. Produce the `not_applicable` output and terminate.
