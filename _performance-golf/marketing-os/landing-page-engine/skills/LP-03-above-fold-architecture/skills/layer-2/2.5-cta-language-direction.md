# 2.5 — CTA Language Direction

**Skill:** LP-03-above-fold-architecture
**Position:** Layer 2, Microskill 5
**Purpose:** Draft 3+ CTA (Call-to-Action) variants with distinct emotional appeals for the above-fold primary action. Each variant includes button text and supporting micro-copy. LP-14 (CTA Copy Optimizer) will finalize; this microskill sets the strategic direction.
**Effort Level:** medium
**Dependencies:** 2.1 headline-type.md, 1.1 traffic-analysis.md, 1.2 awareness-map.md, 1.3 pattern-selection.md, 0.1 input-verification.md
**Output:** `cta-direction.md`

## Execution Logic

**Step 1:** Read dependency files. Extract headline type, traffic temperature, awareness stage, selected pattern, and any available offer data (price, guarantee, product name).

**Step 2:** Determine CTA function based on page type:
- **Type A:** CTA above fold may be "Watch Video," "Read More," "Get Started," or "Order Now" depending on funnel stage. Long-form pages often use soft CTA above fold (engagement) with hard CTA below fold (purchase).
- **Type B:** CTA above fold is ALWAYS purchase-oriented: "Add to Cart," "Buy Now," "Subscribe & Save." No soft CTAs on PDPs.
- **Hybrid:** Primary CTA is purchase-oriented. Optional secondary CTA for "Learn More" scroll trigger.

**Step 3:** Draft minimum 3 CTA variants, each targeting a distinct emotional appeal:

**Variant 1: Confidence/Desire Appeal**
- Emotional lever: "I want this" / "This will work for me"
- Button text style: Benefit-forward ("Get My [Benefit]," "Start [Transformation]")
- Supporting micro-copy: Reinforces ease or certainty

**Variant 2: Consequence/Fear Appeal**
- Emotional lever: "I can't afford NOT to act" / "What happens if I don't"
- Button text style: Loss-aversion framing ("Don't Miss This," "Claim Your [Product]")
- Supporting micro-copy: References risk of inaction or limited availability

**Variant 3: Urgency Appeal**
- Emotional lever: "I need to act NOW" / "This won't last"
- Button text style: Time-pressure ("Get Today's Price," "Order Before [Deadline]")
- Supporting micro-copy: Specific urgency trigger (not fake scarcity)

**Step 4:** For each variant, write:
- Button text (3-6 words max)
- Supporting micro-copy (1 line below or beside button: guarantee reminder, shipping note, risk reversal)
- Emotional appeal classification
- Awareness stage fit (which stage does this CTA serve best?)

**Step 5:** Apply page type constraints:
- Type B: Button text must include transactional verb (Add, Buy, Order, Subscribe, Get)
- Type A soft CTA: Button text should lower commitment ("Watch Free Video," "See How It Works")
- All types: Micro-copy should address the #1 purchase objection for the vertical

**Step 6:** Recommend the primary CTA variant based on awareness stage:
- Unaware/Problem-Aware → Soft CTA or Confidence appeal (they need education first)
- Solution-Aware → Confidence or Consequence appeal (they're comparing)
- Product-Aware/Most-Aware → Urgency appeal (they need final push)

**Step 7:** Write `cta-direction.md`.

## Quality Criteria
- Minimum 3 CTA variants with DISTINCT emotional appeals (not 3 versions of the same appeal)
- Each variant has both button text AND supporting micro-copy
- Button text is 3-6 words (not a sentence)
- Type B pages have transactional verbs in button text
- Recommended primary CTA justified by awareness stage
- Micro-copy addresses a real purchase objection, not generic reassurance
- File exists after execution

## Failure Modes
- Fewer than 3 variants → REJECT, write additional variants
- All variants use the same emotional appeal → REJECT, diversify appeals
- Button text exceeds 6 words → Shorten to action phrase
- Type B page with soft CTA ("Learn More") as primary → REJECT, must be transactional
- Generic micro-copy ("100% Satisfaction") → Replace with specific claim
- headline-type.md not found → HALT, run 2.1 first
