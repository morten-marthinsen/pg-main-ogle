# 4.2 — Summary Writer

**Skill:** PDP-03-hero-carousel-buybox
**Position:** Layer 4, Microskill 2
**Purpose:** Write a human-readable summary of the PDP above-fold blueprint for review. This file is the primary document humans read to understand and approve the above-fold architecture before downstream skills execute.
**Effort Level:** low
**Dependencies:** 4.1 pdp-above-fold-blueprint.json
**Output:** `PDP-ABOVE-FOLD-SUMMARY.md`

---

## Input Schema

```yaml
input:
  blueprint: "[path to pdp-above-fold-blueprint.json]"
  eight_second_test: "[path to eight-second-test.md]"
  buy_box_audit: "[path to buy-box-audit.md]"
  carousel_check: "[path to carousel-check.md]"
```

---

## Execution Logic

**Step 1:** Read `pdp-above-fold-blueprint.json`.

**Step 2:** Write the summary following this exact structure:

### Section 1: Overview
- Product name
- Page type
- Total carousel slides (must be 10)
- Buy box components (must be 10)
- Audit score

### Section 2: Carousel Summary
- Table of all 10 slides: Position | Story Position | Purpose (1 sentence) | Video? (Y/N)
- Note any slides with particularly strong or weak copy directions

### Section 3: Buy Box Summary
- Product title (full text)
- Rating strip behavior
- Price display (current price, savings, shipping)
- Variant selector type (must say "chips")
- Subscription availability (OTP default, not pre-selected)
- Quantity selector type (must say "stepper")
- ATC button label and post-click behavior
- Micro-trust signals (list all 3-4)

### Section 4: Audit Results
- 8-second test results (all 3 questions)
- Buy box audit score
- Carousel completeness result
- Anti-slop check result

### Section 5: Downstream Handoffs
- PDP-04: Uses carousel and buy box context for BTF section planning
- PDP-06: Uses buy box architecture as spec for writing actual copy
- LP-15: Uses blueprint for page assembly

### Section 6: Gaps and Risks
- Any data gaps flagged during Layer 0-2
- Any audit points that barely passed
- Any assumptions made due to missing brief data

**Step 3:** Write `PDP-ABOVE-FOLD-SUMMARY.md`.
