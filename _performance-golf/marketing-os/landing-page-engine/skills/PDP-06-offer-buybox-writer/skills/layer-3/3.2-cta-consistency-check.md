# 3.2 — CTA Consistency Check

**Skill:** PDP-06-offer-buybox-writer
**Position:** Layer 3, Microskill 2
**Purpose:** Verify that the ATC button label is identical across ALL placements: main buy box, sticky ATC bar, and mini-cart. Also verify that no contradicting offers or prices appear across these three touchpoints. Users interact with the purchase flow through all three, and inconsistency creates subconscious friction.
**Effort Level:** low
**Dependencies:** 2.1 atc-copy.md (main ATC), 2.4 sticky-atc-copy.md (sticky bar), 2.5 mini-cart-copy.md (mini-cart), 1.1 pricing-copy.md (prices)
**Output:** `cta-consistency-check.md`

---

## Input Schema

```yaml
input:
  atc_copy: "[path to atc-copy.md]"
  sticky_atc_copy: "[path to sticky-atc-copy.md]"
  mini_cart_copy: "[path to mini-cart-copy.md]"
  pricing_copy: "[path to pricing-copy.md]"
  subscription_copy: "[path to subscription-copy.md or 'N/A']"
```

---

## Execution Logic

**Step 1:** Read `atc-copy.md`. Extract the main buy box default ATC label.

**Step 2:** Read `sticky-atc-copy.md`. Extract the sticky bar ATC label.

**Step 3:** Read `mini-cart-copy.md`. Extract the mini-cart primary action label ("Checkout") and any ATC-equivalent button.

**Step 4:** Perform ATC label consistency check:
```
Main buy box ATC: "[label]"
Sticky bar ATC:   "[label]"
All match: [Y/N]
```
- Main and sticky MUST be identical
- Mini-cart uses "Checkout" (different context -- this is expected and correct)

**Step 5:** Perform price consistency check across touchpoints:
```
Main buy box price:  "$[XX.XX]"
Sticky bar price:    "$[XX.XX]"
Mini-cart item price: "$[XX.XX]"
All match: [Y/N]
```
- The same product at the same configuration must show the same price everywhere

**Step 6:** Perform offer consistency check:
- No contradicting subscription offers (e.g., main says "Save 25%" but mini-cart says "Save 20%")
- No contradicting shipping claims (e.g., micro-trust says "Free Shipping" but mini-cart says "Shipping: $5.99")
- No contradicting guarantee terms across touchpoints

**Step 7:** Check for confusing action labels:
- Mini-cart "Checkout" and "Continue Shopping" are distinct actions: Y/N
- No buttons that could be confused with each other: Y/N
- Disabled states guide user to correct action: Y/N

**Step 8:** Write `cta-consistency-check.md`.

## Quality Criteria
- ATC label is identical on main buy box and sticky bar
- Prices are identical across all touchpoints for the same product configuration
- No contradicting offer claims (savings percentages, shipping, guarantee)
- Action labels are clear and distinct (no ambiguity)
- File exists after execution

## Failure Modes
- ATC labels differ between main and sticky -> HALT; align sticky to match main
- Prices differ between touchpoints -> HALT; reconcile all to pricing-copy.md (source of truth)
- Subscription savings inconsistency -> HALT; reconcile all to subscription-copy.md (source of truth)
- Mini-cart shows different price than main buy box -> Most likely a configuration issue; reconcile to pricing-copy.md
