# 3.3 — Anti-Slop Scanner

**Skill:** LP-10-social-proof-writer
**Position:** Layer 3, Microskill 3
**Purpose:** Systematically scan ALL proof copy elements against the 5-category forbidden word/phrase list from LP-10-AGENT.md. A single instance of any forbidden element = FAIL. This is the final quality gate before package assembly — no AI telltales, no generic proof language, no credential inflation in the final output.
**Effort Level:** low
**Dependencies:** All Layer 2 outputs that were generated
**Output:** `anti-slop-scan.md`

---

## Input Schema

```yaml
input:
  elements_to_scan:
    testimonial_intros: "[from testimonial-intros.md — if generated]"
    formatted_testimonials: "[from formatted-testimonials.md — if generated]"
    review_section_copy: "[from review-section-copy.md — if generated]"
    before_after_captions: "[from before-after-captions.md — if generated]"
    expert_endorsements: "[from expert-endorsements.md — if generated]"
    proof_transitions: "[from proof-transitions.md — if generated]"
    volume_signals: "[from volume-signals.md — if generated]"
```

---

## Execution Logic

1. **Load the 5-category forbidden list** from LP-10-AGENT.md:

   **Category 1 — Generic Proof Intros:**
   "But don't just take our word for it" (literal), "See what our customers are saying", "Hear from real people", "Real results from real people", "Success stories that speak for themselves"

   **Category 2 — Vague Testimonial Language:**
   "Great product!" (standalone), "Love it!", "Changed my life" (without specifics of HOW), "Amazing results" (without specifying WHAT), "I feel so much better" (without specifying what changed), "Best purchase ever"

   **Category 3 — AI Telltales:**
   transformative, game-changing, revolutionary, groundbreaking, unprecedented, life-changing (without specifics), unlock, empower, journey (metaphorical), holistic, synergy

   **Category 4 — Fake Urgency in Proof:**
   "These results won't last forever", "Hurry before spots run out" (in a proof section), "Limited testimonials available"

   **Category 5 — Credential Inflation:**
   "World-renowned" (without verification), "Leading expert" (without named institution), "Top doctor" (without ranking source), "Acclaimed researcher" (without specific publications)

2. **Scan each Layer 2 output independently.** For each file:
   - Check every word, phrase, and sentence against all 5 categories
   - Record ANY match with: the forbidden element, the file it was found in, and the category

3. **Check for near-misses** — words not on the list but carrying similar signals:
   - "Empower" / "empowering" (in ANY proof copy)
   - "Unlock" / "unlocking" (in ANY proof copy)
   - "Elevate" / "elevated" (in proof copy framing)
   - "Seamless" / "seamlessly"
   - "Robust"
   - "Leverage" (as a verb)

4. **Determine scan result:**
   - Zero forbidden elements AND zero near-misses = **PASS**
   - Zero forbidden elements but near-misses found = **PASS WITH FLAG**
   - One or more forbidden elements found = **FAIL** (require revision)

5. **If FAIL:** For each forbidden element found, provide a specific replacement suggestion.

## Quality Criteria
- Every generated Layer 2 output scanned — no files skipped (except those not generated)
- All 5 categories checked explicitly (each category listed in output, even if "None found")
- Near-miss check conducted in addition to the forbidden list
- FAIL result includes specific replacement suggestions, not just identification
- Scan result is binary: any forbidden element = FAIL. No "minor" or "contextually acceptable" exceptions.
- Compound phrases scanned as units ("But don't just take our word for it" — not word-by-word)

## Failure Modes
- Scanning only some Layer 2 outputs ("I checked the testimonials and transitions") -> ALL generated outputs must be scanned. Expert endorsements are the most common location for credential inflation.
- Marking a forbidden word as "acceptable in this context" -> No context exceptions. The word list is absolute for proof copy. Remove and replace.
- Missing compound phrases (scanning "take" and "our" and "word" individually instead of the full phrase) -> Scan for multi-word phrases as complete units.
- Near-miss found but not flagged -> Near-misses should be noted even on a PASS result for human awareness.
- Skipping Category 5 for expert endorsement copy -> Category 5 specifically targets expert framing. Expert endorsements are the PRIMARY location to check for credential inflation.
