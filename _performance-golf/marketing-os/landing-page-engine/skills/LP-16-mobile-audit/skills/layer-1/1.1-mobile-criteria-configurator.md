# 1.1 — Mobile Criteria Configurator

> **Skill:** LP-16-mobile-audit
> **Layer:** 1 (Audit Planning)
> **Model:** sonnet
> **Input:** `brief-load.md`, `benchmark-load.md`
> **Output:** `mobile-criteria.md`

---

## PURPOSE

Configure the mobile audit criteria for all 7 dimensions based on the page type (Type A, Type B, or Hybrid). Each dimension needs type-specific thresholds, scoring bands, and evidence requirements. This configurator ensures Layer 2 auditors know exactly what to evaluate and at what thresholds.

---

## PROCEDURE

1. **Read** `brief-load.md` to determine page type.
2. **Read** `benchmark-load.md` to load benchmark thresholds.
3. **Configure** each of the 7 dimensions with type-specific criteria.
4. **Write** the configured criteria to `mobile-criteria.md`.

### Configuration Logic by Page Type

#### Type A (Long-Form Sales Page)
- Load time target: **< 3 seconds** (maximum acceptable: 4 seconds)
- Sticky CTA bar: **Recommended** (score ceiling 7 if absent, not automatic FAIL)
- Content collapse: **Required** for any section > 500 words
- CTA frequency: **3+ placements** across the page
- Above-fold CTA: May be in deck copy below hero but must be within first viewport
- Anchor links: **Recommended** for pages > 10 sections
- Back-to-top: **Recommended** for pages > 5 screen heights

#### Type B (Ecomm/PDP)
- Load time target: **< 2 seconds** (maximum acceptable: 3 seconds)
- Sticky ATC bar: **REQUIRED** (score ceiling 4 if absent — this is a FAIL)
- Content collapse: **Required** for ingredient/science sections > 500 words
- CTA frequency: Sticky bar provides continuous visibility
- Above-fold CTA: CTA + price **MUST** be above fold
- Anchor links: **Not required** (sticky bar handles navigation)
- Back-to-top: **Not required**

#### Hybrid (Type B above-fold + Type A education below)
- Load time target: **< 2 seconds** (applies Type B standard because above-fold is product-focused)
- Sticky ATC bar: **REQUIRED** (same as Type B)
- Content collapse: **Required** for education sections below fold > 500 words
- CTA frequency: Sticky bar + 2+ inline CTAs in education section
- Above-fold CTA: CTA + price **MUST** be above fold (Type B standard)
- Anchor links: **Recommended** if education section > 8 sub-sections
