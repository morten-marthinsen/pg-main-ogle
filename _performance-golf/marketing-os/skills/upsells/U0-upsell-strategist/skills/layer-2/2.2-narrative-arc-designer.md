# 2.2 — Narrative Arc Designer

**Layer:** 2
**Model:** opus
**Depends on:** 2.1 (Congruence Thread Map)
**Produces:** `layer-2/2.2-narrative-arc.md`

---

## Purpose

Design the emotional and logical arc that carries the buyer from purchase through the entire upsell sequence. Each touchpoint should feel like the next natural beat in a story, not a standalone sales pitch.

## The Upsell Narrative Arc

```
PURCHASE → Celebration → Extension → Decision → Acknowledgment → Completion
```

### Beat Structure

| Beat | Position | Emotional State | Logical Frame |
|------|----------|----------------|---------------|
| **Celebration** | Post-purchase / Bump | Excitement, validation, relief | "You made a smart decision" |
| **Extension** | Upsell 1 intro | Curiosity, momentum | "There's one more thing that makes this work even better" |
| **Decision** | Upsell 1 CTA | Confidence, clarity | "Yes or no — either way, you're set" |
| **Acknowledgment** | Downsell intro | Understanding, zero pressure | "I completely understand — and here's an alternative" |
| **Completion** | Thank you | Satisfaction, anticipation | "Here's how to get started and get the most from this" |

## Execution Steps

1. Read the congruence thread map (2.1)
2. Read the sequence structure (Layer 1.4)
3. For each position in the sequence, define:
   - **Emotional target:** What should the buyer FEEL at this moment?
   - **Logical frame:** What's the rational argument for this step?
   - **Transition:** How do we move from this beat to the next?
   - **Tone direction:** Specific voice/energy guidance for the copywriter (U1/U2/U3)
4. Validate the arc: read all beats in sequence — is there emotional momentum?
