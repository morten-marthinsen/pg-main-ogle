# 0.3: Format Strategy Loader

## Purpose
Load the Format Strategy output from A03. Extract format assignments per hook (video/static/carousel), platform assignments, target lengths per ad concept, creative volume plan, and platform-specific constraints that will determine which script frameworks are viable.

## Inputs
- A03 FORMAT-STRATEGY.md (primary output from Format Strategy skill)
- A03 checkpoint file with format assignments

## Process

### Step 1: Locate and Read Format Strategy
```
LOCATE: [project]/A03-format-strategy/FORMAT-STRATEGY.md
READ entire file
VERIFY file contains format assignments section
```

### Step 2: Extract Per-Hook Format Assignments
```
FOR EACH hook in format assignments:
  EXTRACT:
    - hook_id (cross-reference with A02)
    - format_type (video | static | carousel)
    - platform_primary (TikTok | Meta | YouTube | Google Display | etc.)
    - platform_secondary (optional additional platforms)
    - target_length (6s | 15s | 30s | 60s | 90s | 2min | 3min)
    - aspect_ratio (9:16 vertical | 1:1 square | 16:9 horizontal)
    - sound_assumption (sound-on | sound-off | both)
```

### Step 3: Extract Creative Volume Plan
```
READ creative_volume_plan section
EXTRACT:
  - total_ad_concepts (hooks x formats x platforms)
  - concepts_by_length (how many 15s, 30s, 60s, etc.)
  - concepts_by_platform (how many TikTok, Meta, YouTube, etc.)
  - testing_priorities (which concepts are highest priority)
```

### Step 4: Extract Platform Constraints
```
FOR EACH platform in strategy:
  EXTRACT:
    - platform_name
    - allowed_formats (video, static, carousel)
    - allowed_lengths (range)
    - required_aspect_ratio
    - sound_requirement (on/off/both)
    - text_overlay_policy (required | optional | avoid)
    - skip_point (for YouTube pre-roll: 5-second skip)
```

### Step 5: Build Ad Concept Inventory
```
CREATE ad_concept_inventory:
  FOR EACH hook x format x platform x length combination:
    CREATE concept_object:
      concept_id: "[hook_id]-[platform]-[length]"
      hook_id: string
      format_type: string
      platform: string
      target_length: string
      aspect_ratio: string
      sound_assumption: string
```

## Output Schema
```yaml
format_strategy:
  total_ad_concepts: integer
  ad_concepts:
    - concept_id: string
      hook_id: string
      format_type: string
      platform_primary: string
      platform_secondary: [list or null]
      target_length: string
      aspect_ratio: string
      sound_assumption: string
  creative_volume_plan:
    concepts_by_length:
      6s: integer
      15s: integer
      30s: integer
      60s: integer
      90s: integer
      2min: integer
      3min: integer
    concepts_by_platform:
      TikTok: integer
      Meta: integer
      YouTube: integer
      Google_Display: integer
  platform_constraints:
    - platform: string
      allowed_formats: [list]
      allowed_lengths: [list]
      aspect_ratio: string
      sound_requirement: string
      text_overlay: string
  loading_status:
    format_assignments_loaded: integer
    ad_concepts_inventoried: integer
    warnings: [list or empty]
```

## Output File
- **Path:** `layer-0-outputs/0.3-format-strategy-loader.md`
- **Minimum size:** 1KB

## Quality Checks
1. Format Strategy file exists and was successfully read
2. Every selected hook from A02 has a format assignment
3. At least one target length specified per ad concept
4. Platform constraints documented for all target platforms
5. Creative volume plan totals are consistent with format assignments
6. No ad concepts with incompatible format-platform combinations (e.g., carousel on TikTok)

## Dependencies
- A03 (Format Strategy) must be complete
- A02 hooks must be loaded first (0.2 must run before this)
- FORMAT-STRATEGY.md must exist at standard output path
