# 2.4: Mechanism Module Briefs

## Purpose
Design the [MECHANISM] module content brief for each ad concept. For 60+ second ads: full mechanism integration with naming moment, metaphor anchor, "why this works" explanation. For 30s: compressed hint (10-15 words). For 15s: mechanism omitted (noted). Specify how mechanism serves hook's angle.

## Inputs
- Module Sequence Design (from 2.1)
- Mechanism Package (from 0.1)

## Process
For each concept length category (15s/30s/60s+), adapt mechanism integration appropriately. Full integration for 60s+, compressed hint for 30s, documented omission for 15s.

## Output File
- **Path:** `layer-2-outputs/2.4-mechanism-module-briefs.md`
- **Minimum size:** 5KB

## Dependencies
- 2.1 complete
- Mechanism from 0.1 loaded
