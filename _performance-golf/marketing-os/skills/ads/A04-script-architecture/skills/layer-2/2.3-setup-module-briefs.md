# 2.3: Setup Module Briefs

## Purpose
Design the [SETUP] module content brief for each ad concept. Specify: what problem/context to establish, how it connects to the hook's promise, root cause framing, awareness level bridging, word count target. Verify setup fulfills the hook's promise.

## Inputs
- Module Sequence Design (from 2.1)
- Hook Module Briefs (from 2.2)
- Root Cause Package (from 0.1)

## Process
Map hook promise to setup content. Integrate root cause framing. Design awareness-level bridge. Verify hook-to-setup coherence.

## Output File
- **Path:** `layer-2-outputs/2.3-setup-module-briefs.md`
- **Minimum size:** 5KB

## Dependencies
- 2.1, 2.2 complete
