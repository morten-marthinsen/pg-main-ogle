# 0.1: Campaign Brief Loader

## Purpose

Extract all format-strategy-relevant data from the Campaign Brief (Skill 09 output). This microskill identifies the total creative budget, timeline constraints, target platforms, audience demographics, and campaign objectives that will drive platform priority decisions, budget allocation, and production planning in Layers 1-3.

## Inputs

- Campaign Brief (Skill 09 output): `[project]/09-campaign-brief/campaign-brief-package.yaml`

## Process

1. **READ Campaign Brief completely**
   - Full read of the YAML package
   - Do NOT rely on memory or summaries

2. **EXTRACT budget data:**
   - Total creative budget (CRITICAL for Layer 3 budget allocation)
   - If not specified: FLAG for human input with BLOCKING status
   - Budget breakdown if provided (video budget, design budget, creator fees)
   - Budget constraints or caps

3. **EXTRACT timeline data:**
   - Campaign launch date (when do ads need to be live?)
   - Production deadline (when must creative be delivered?)
   - Campaign duration (ongoing, seasonal, time-limited)
   - Any milestone dates

4. **EXTRACT target platforms:**
   - Which platforms does the client want to run on?
   - If not specified: DEFAULT to Meta + TikTok (highest volume platforms for most verticals)
   - Platform priority if specified by client
   - Platform exclusions if any

5. **EXTRACT audience demographics:**
   - Age range (affects platform priority — TikTok skews younger, Meta spans widest)
   - Gender skew (affects creative treatment decisions)
   - Geographic targeting (US, international, specific regions)
   - Interest targeting (influences format and hook selection)
   - Awareness level (unaware, problem-aware, solution-aware, product-aware, most-aware)

6. **EXTRACT campaign objectives:**
   - Primary goal: awareness / consideration / conversion
   - Secondary goals if any
   - Success metrics (ROAS target, CPA target, brand lift)
   - Funnel stage focus (TOFU, MOFU, BOFU)

7. **EXTRACT platform-specific requirements:**
   - Any client-mandated constraints (must use UGC, no polished production, etc.)
   - Brand guidelines affecting format (tone, style, voice)
   - Competitor exclusions or considerations
   - Creative refresh cadence requirements

8. **DOCUMENT extraction completeness:**
   - If budget missing: CRITICAL FLAG
   - If platforms not specified: WARN and use defaults
   - If audience demographics sparse: WARN (affects platform priority logic)
