# 1.1: Platform Priority Analysis

## Purpose

Determine the platform priority order based on A01 competitive intelligence data, campaign objectives, and audience demographics. This establishes which platforms get primary focus, which are secondary, and which are tertiary. Platform priority drives budget allocation in Layer 3 and testing sequence planning.

## Inputs

- `layer-0-outputs/0.1-campaign-brief.md` — target platforms, objectives, audience demographics
- `layer-0-outputs/0.2-ad-intelligence.md` — platform-specific competitive intelligence
- `layer-0-outputs/0.0.1-vertical-profile.md` — vertical platform preference patterns

## Process

1. **LOAD target platforms** from Campaign Brief (0.1)
   - If specified by client: use those platforms
   - If defaulted: Meta + TikTok baseline

2. **CROSS-REFERENCE with A01 intelligence:**
   - **Market size signal:** Where are competitors spending most? (indicates viable market)
   - **Winner concentration:** Which platforms have highest concentration of winning ads?
   - **Opportunity gaps:** Which platforms have identified format/creative opportunities?

   Example analysis:
   ```
   Meta: 65% of competitor spend, 12/20 winning ads → High market validation
   TikTok: 25% of competitor spend, 6/20 winning ads → Growing platform
   YouTube: 10% of competitor spend, 2/20 winning ads → Underutilized opportunity OR poor fit
   ```

3. **CROSS-REFERENCE with audience demographics:**
   - **Age skew:**
     - TikTok: 18-34 dominant → Elevated priority for young demos
     - YouTube: 18-55+ wide span → Versatile for most demos
     - Meta: 25-65+ widest span → Safe bet for broad targeting
   - **If audience 55+:** TikTok deprioritized, Meta elevated
   - **If audience 18-24:** TikTok elevated, YouTube Shorts elevated

4. **CROSS-REFERENCE with campaign objectives:**
   - **Awareness:** TikTok + YouTube (reach + discovery algorithms)
   - **Consideration:** YouTube + Meta (education + engagement formats)
   - **Conversion:** Meta + Google (targeting precision + intent signals)

5. **CROSS-REFERENCE with vertical patterns (0.0.1):**
   - Does the vertical profile indicate platform preferences?
   - Are there vertical-specific performance patterns?

6. **CALCULATE platform priority score** (weighted formula):
   ```
   Platform Priority Score =
     (A01 Market Share × 0.25) +
     (A01 Winner Concentration × 0.25) +
     (Audience Demographic Fit × 0.20) +
     (Campaign Objective Alignment × 0.20) +
     (Opportunity Gap Bonus × 0.10)

   Score range: 0-10
   ```

7. **PRODUCE ranked platform priority list:**
   - Rank 1 (Primary): Highest scoring platform
   - Rank 2 (Secondary): Second highest
   - Rank 3 (Tertiary): Third highest
   - Each rank MUST have data-grounded rationale (cite specific numbers from A01)

8. **DOCUMENT priority rationale** for each platform:
   - Not generic statements like "Meta has the most users"
   - Specific data points: "Meta shows 65% competitor spend with 12/20 winning ads (60% win rate). Audience 35-55 fits Meta's demographic strength. Campaign objective is conversion, where Meta's targeting precision excels."

- [ ] Every rationale cites SPECIFIC A01 data (not generic "popular platform")
- [ ] Audience demographic fit explained (not assumed)
- [ ] Campaign objective alignment explicit
- [ ] Strategic implications documented (budget %, test allocation)
- [ ] Priority order is data-grounded, not arbitrary
