# 4.1: Visual Direction Package Assembler

## Purpose
Assemble VISUAL-DIRECTION-PACKAGE.md with ALL required sections (as defined in the Output Schema in AGENT.md). This is the PRIMARY DELIVERABLE of A05. Must include: executive summary, per-concept visual briefs (complete shot lists), color/typography systems, talent direction, product staging, motion graphics specs, B-roll strategy, sound-off strategy, platform adaptations, thumbnail specs, tool-specific production specs, coherence scorecard. MINIMUM SIZE: 80KB. This package is what A06 (Ad Arena) evaluates and A08 (Visual/Video Production) executes.

## Inputs
- ALL Layer 0 through Layer 3 outputs (every microskill output produced in the skill)
- Coherence scorecard from 2.5.3
- All tool-specific specs from Layer 3

## Process
1. READ all microskill outputs from Layers 0-3
2. READ coherence scorecard from 2.5.3
3. ASSEMBLE VISUAL-DIRECTION-PACKAGE.md following the output schema:

   **Section 1: Executive Summary**
   - Total concepts with visual direction
   - Treatment type distribution
   - Target platforms
   - Coherence scores summary
   - Production tools required

   **Section 2: Visual Style Strategy Overview**
   - Campaign visual identity (from 1.1-1.4)
   - Treatment type rationale per concept (from 1.1)
   - Color palette system with hex codes and rules (from 1.2)
   - Typography system (from 1.3)
   - Competitive visual positioning (from 1.4)

   **Section 3: Per-Concept Visual Briefs**
   FOR EACH concept:
   - Treatment type and target platform
   - COMPLETE shot list table (from 2.1) -- every shot with all 7 elements
   - Talent direction (from 2.2)
   - Product staging (from 2.3)
   - Motion graphics elements (from 2.4)
   - B-roll strategy (from 2.5)
   - Sound-off strategy (from 2.6)
   - Thumbnail specs if YouTube (from 2.7)

   **Section 4: Platform Adaptations**
   - Meta specifications (from 3.5 + 0.2)
   - TikTok specifications (from 3.5 + 0.2)
   - YouTube specifications (from 3.5 + 0.2)

   **Section 5: Tool-Specific Production Specs**
   - Midjourney prompts (from 3.1)
   - Arcads talent briefs (from 3.2)
   - ElevenLabs voice direction (from 3.3)
   - Stock footage queries (from 3.4)

   **Section 6: Coherence Scorecard**
   - Full scorecard from 2.5.3

   **Section 7: Variant Visual Treatments**
   - Alternative visual treatments per concept (for A09 visual swap testing)

   **Section 8: Production Priority Order**
   - Recommended production sequence

4. VERIFY minimum size: >= 80KB (81,920 bytes)
   - If under 80KB: sections are likely missing detail
   - DO NOT pad with filler -- expand shot lists, add detail to briefs
5. VERIFY all 8 sections populated (not empty, not placeholder text)
6. WRITE VISUAL-DIRECTION-PACKAGE.md to project output directory
