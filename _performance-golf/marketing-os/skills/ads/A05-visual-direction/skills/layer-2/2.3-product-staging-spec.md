# 2.3: Product Staging Specification

## Purpose
For each concept featuring the product: define product shot specifications (angle, lighting, surface, background, props), unboxing sequence direction (if applicable), product-in-use shots (application method, hands positioning, setting), before/after staging (lighting consistency, angle consistency, timing markers), and product detail shots (ingredient list, label, texture, packaging details). The product is the physical proof of the mechanism -- staging must make it look credible, desirable, and real.

## Inputs
- Shot lists from 2.1 (which shots feature the product)
- Brand asset details from 0.4 (product packaging, existing photography)
- Treatment type from 1.1 (treatment affects staging aesthetic)
- Color palettes from 1.2 (surface/background color coordination)

## Process
1. READ shot lists from 2.1
2. READ brand assets from 0.4 (product packaging details)
3. READ treatment types from 1.1
4. READ color palettes from 1.2
5. IDENTIFY all product shots across all scripts
6. FOR EACH concept featuring the product:
   a. DEFINE hero product shots:
      - Angle: camera angle relative to product (e.g., "30-degree above, slight left rotation")
      - Lighting: key light position, fill, accent (e.g., "morning light from camera-left, soft fill from right")
      - Surface: what the product sits on (e.g., "white marble countertop with natural veining")
      - Background: behind the product (e.g., "blurred greenery, bokeh with warm tones")
      - Props: supporting objects (e.g., "fresh ingredients arranged camera-right, glass of water camera-left")
   b. DEFINE unboxing sequence (if applicable):
      - Package arrival shot
      - Opening sequence (hands, angle, pacing)
      - First reveal moment
      - Product extraction and placement
   c. DEFINE product-in-use shots:
      - Application method (pouring, swallowing, applying, mixing)
      - Hands positioning (which hand, how held, distance from camera)
      - Setting for use (kitchen counter, bathroom mirror, bedside table)
      - Action pacing (speed of application/use)
   d. DEFINE before/after staging (if applicable):
      - Lighting consistency (identical setup for both shots)
      - Angle consistency (identical camera position)
      - Timing markers (e.g., "4 weeks later" text overlay)
      - Visual proof elements (measurements, scale, comparison reference)
   e. DEFINE detail shots:
      - Ingredient list (macro shot of label, legible)
      - Label design (brand logo, product name, key claims)
      - Texture (if supplement: capsule color/size, if topical: consistency/color)
      - Packaging details (seal, quality indicators)
7. WRITE to output file
