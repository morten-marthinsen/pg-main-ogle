# 1.3: Typography System

## Purpose
Design typography system for text overlays and motion graphics. Select: headline font (bold, high-impact), body font (clean, readable at small sizes), accent font (if needed). Define: size hierarchy, weight rules, color rules, animation rules. Typography must be platform-appropriate -- TikTok native fonts differ from YouTube lower thirds.

## Inputs
- Treatment type assignments from 1.1 (treatment affects typography style)
- Brand typography from 0.4 (brand fonts take precedence if specified)
- Platform constraints from 0.2 (text size requirements vary by platform)
- Color palettes from 1.2 (text colors must work with palette)

## Process
1. READ treatment type assignments from 1.1
2. READ brand typography standards from 0.4
3. READ platform constraints from 0.2
4. READ color palettes from 1.2
5. ESTABLISH typography hierarchy:
   a. **Headline font:** Bold, high-impact, readable at all sizes
      - For Polished: Premium serif or modern sans-serif (e.g., Playfair, Montserrat)
      - For UGC: Platform-native (TikTok caption style, Instagram story style)
      - For Hybrid: Clean sans-serif (e.g., Inter, DM Sans)
      - For Demo: Technical/clean sans-serif
      - For Testimonial: Warm, readable sans-serif
   b. **Body font:** Clean, readable at small mobile sizes (minimum 16pt equivalent)
      - Must contrast with headline font (if headline is serif, body is sans-serif)
   c. **Accent font:** Optional, for callouts, annotations, emphasis
6. DEFINE size hierarchy:
   - Hero text (hook/headline): [size range by platform]
   - Primary text (key points): [size range by platform]
   - Secondary text (supporting): [size range by platform]
   - Caption text: [size range by platform]
7. DEFINE weight rules:
   - When to use Bold, SemiBold, Regular, Light
   - Maximum weight levels per frame (avoid clutter)
8. DEFINE animation rules:
   - Text entrance methods (fade, slide, typewriter, pop)
   - Text duration (minimum display time for readability)
   - Platform-specific animation styles (TikTok: animated/playful, YouTube: clean/professional)
9. VERIFY platform appropriateness:
   - TikTok: Does typography feel native? (Not corporate)
   - Meta: Is text within 20% overlay limit at readable size?
   - YouTube: Are lower thirds professional enough?
10. WRITE to output file
