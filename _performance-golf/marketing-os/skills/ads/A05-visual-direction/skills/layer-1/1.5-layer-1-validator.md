# 1.5: Layer 1 Validator

## Purpose
Validate all Layer 1 outputs for completeness and correctness. Confirm: every script has a treatment type assigned, rationale provided for each assignment, no platform-treatment mismatches (e.g., polished on TikTok without documented strategic reason), color palettes are accessible (WCAG contrast ratios for text on background), typography is platform-appropriate. This is the final quality gate before Layer 2 begins.

## Inputs
- All Layer 1 outputs: 1.1 (treatment types), 1.2 (color palettes), 1.3 (typography), 1.4 (competitive positioning)
- Script inventory from 0.1 (for completeness cross-check)
- Platform constraints from 0.2 (for platform-appropriateness check)

## Process
1. READ all Layer 1 outputs (1.1, 1.2, 1.3, 1.4)
2. READ script inventory from 0.1
3. READ platform constraints from 0.2
4. VALIDATE treatment type completeness (from 1.1):
   a. Every script has a treatment type: [count assigned / count total]
   b. Every assignment has rationale: [Y/N per script]
   c. No unassigned scripts: [list any or "none"]
5. VALIDATE platform-treatment appropriateness:
   a. Check for mismatches:
      - Polished (Treatment 1) on TikTok without documented reason
      - UGC (Treatment 2) on YouTube pre-roll without documented reason
   b. Flag any mismatch with severity (BLOCK vs WARNING)
6. VALIDATE color palette accessibility (from 1.2):
   a. WCAG contrast ratio for text on background per concept
   b. All meet minimum 4.5:1: [Y/N per concept]
   c. Flag any failing ratios
7. VALIDATE typography platform-appropriateness (from 1.3):
   a. TikTok: typography feels native (not corporate)
   b. Meta: text sizes work within 20% overlay rule
   c. YouTube: lower thirds are professional
8. VALIDATE competitive positioning completeness (from 1.4):
   a. Both alignment and differentiation documented
   b. Whitespace analysis present
9. DETERMINE gate status:
   - PASS: All validations pass
   - FAIL: Any critical validation fails (list specific failures)
10. WRITE validation results to output file
