# 2.5.3: Coherence Validator

## Purpose
Aggregate alignment scores from 2.5.1 (visual-audio) and 2.5.2 (emotional arc). Flag any concept scoring below 7.0 on either dimension. For flagged concepts: identify the specific beat(s) where misalignment occurs, describe the misalignment, provide specific revision direction. Produce the COHERENCE SCORECARD. This is the quality gate that prevents visual-copy disconnect from reaching production.

## Inputs
- Visual-audio alignment analysis from 2.5.1 (VA scores per concept)
- Emotional arc alignment analysis from 2.5.2 (EA scores per concept)
- Shot lists from 2.1 (for revision reference)
- Script beats from 0.1 (for revision reference)

## Process
1. READ visual-audio alignment results from 2.5.1
2. READ emotional arc alignment results from 2.5.2
3. FOR EACH concept/script:
   a. EXTRACT VA alignment score (from 2.5.1)
   b. EXTRACT EA alignment score (from 2.5.2)
   c. CALCULATE overall coherence score:
      - Overall = (VA x 0.6) + (EA x 0.4)
      - VA weighted higher because visual-audio match is more critical than pacing
   d. DETERMINE status:
      - >= 7.0 overall AND >= 7.0 on both VA and EA: PASS
      - < 7.0 on either dimension: REVISE
4. FOR FLAGGED CONCEPTS (any below 7.0):
   a. IDENTIFY specific beats where misalignment occurs
   b. DESCRIBE the misalignment precisely
   c. PROVIDE specific revision direction:
      - Which shot(s) need to change
      - What the visual should be instead
      - How this fixes the alignment issue
   d. ESTIMATE revision complexity (minor tweak vs major rework)
5. BUILD COHERENCE SCORECARD (mandatory format from AGENT.md)
6. DETERMINE gate status:
   - PASS: All concepts >= 7.0 on both dimensions
   - FAIL: Any concept below 7.0 -- revision required
7. IF FAIL: concepts must be revised and re-checked (maximum 3 revision rounds before human escalation)
8. WRITE to output file
