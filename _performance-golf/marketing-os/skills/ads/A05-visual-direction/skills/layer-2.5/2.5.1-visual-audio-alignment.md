# 2.5.1: Visual-Audio Alignment

## Purpose
For each script, read the audio column and visual brief side-by-side. Check: Does the visual at each beat MATCH what the audio is saying? Does the visual AMPLIFY the audio's emotional trajectory? Are there any contradictions (audio says X, visual shows Y)? Are there any missed opportunities (audio delivers a powerful claim, visual shows nothing relevant)? Score each concept on a 1-10 visual-copy alignment scale. This prevents the #3 degradation pattern: visual-copy disconnect.

## Inputs
- Shot lists from 2.1 (the visual column)
- Script beat structures from 0.1 (the audio column)
- Two-column AV format from A04 script package

## Process
1. READ shot lists from 2.1
2. READ script beat structures with audio content from 0.1
3. FOR EACH script:
   a. ALIGN visual and audio beat-by-beat:
      - For each audio beat, identify the corresponding visual shot(s)
      - Map: audio line --> visual shot number --> visual description
   b. CHECK ALIGNMENT at each beat:
      - **Match:** Visual directly illustrates what the audio says
        * Example: Audio "ancient Himalayan compound" + Visual "monastery ingredients" = MATCH
      - **Amplification:** Visual adds emotional/informational depth beyond audio
        * Example: Audio "thousands have tried" + Visual "scrolling testimonial wall" = AMPLIFICATION
      - **Contradiction:** Visual shows something that conflicts with audio
        * Example: Audio "all-natural ingredients" + Visual "lab with chemicals" = CONTRADICTION
      - **Missed opportunity:** Audio makes a strong claim, visual is neutral/generic
        * Example: Audio "clinical results prove..." + Visual "talking head" = MISSED OPPORTUNITY
   c. DOCUMENT each beat alignment:
      - Beat number, audio content, visual content, alignment type, notes
   d. SCORE overall visual-audio alignment (1-10):
      - 10: Every beat is matched or amplified, zero contradictions
      - 7-9: Most beats aligned, minor missed opportunities
      - 4-6: Some contradictions or significant missed opportunities
      - 1-3: Major contradictions, visual disconnected from audio
   e. FLAG specific beats requiring revision:
      - Contradictions: MUST be fixed
      - Missed opportunities: SHOULD be fixed
4. COMPILE alignment analysis for all scripts
5. WRITE to output file
