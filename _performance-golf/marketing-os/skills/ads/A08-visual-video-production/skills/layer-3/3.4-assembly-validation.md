# 3.4: Assembly Validation

## Purpose
Verify every assembled ad unit across all platforms. Check visual-copy coherence (text overlay matches script), audio-visual sync, duration compliance, platform spec compliance, and file format compliance. This is the final quality gate before output packaging.

## Inputs
- 3.1 Static Ad Assembly (all assembled static ads)
- 3.2 Video Ad Assembly (all assembled video ads)
- 3.3 Multi-Platform Rendering (all platform-specific renders)
- A07 AD-COPY-FINAL/ (for visual-copy coherence check)
- Platform-Specific Technical Requirements from AGENT.md

## Process
```
FOR EACH assembled and rendered ad unit:

VALIDATION CHECKLIST:

1. VISUAL-COPY COHERENCE
   - Text overlays match A07 copy exactly (no words changed, no typos)
   - CTA text matches A07 CTA copy exactly
   - Headline text matches A07 headline exactly
   - If video: key phrases appear at correct timing marks
   Verdict: [PASS/FAIL]

2. AUDIO-VISUAL SYNC (video only)
   - Talking head lip sync is accurate
   - B-roll cuts match voiceover content (visual matches narration)
   - Music fades and voiceover timing are correct
   - No audio/video desync or drift
   Verdict: [PASS/FAIL]

3. DURATION COMPLIANCE (video only)
   - Duration matches target (+/- 2 seconds)
   - Duration is within platform maximum
   - End card is present for 3-5 seconds
   Verdict: [PASS/FAIL]

4. PLATFORM SPEC COMPLIANCE
   - Resolution meets minimum per platform
   - Aspect ratio is exact
   - File format is correct (JPG/PNG for static, MP4 H.264 for video)
   - File size is under platform maximum
   - Audio codec and bitrate meet platform requirements
   Verdict: [PASS/FAIL]

5. SAFE ZONE COMPLIANCE
   - All text within platform safe zones
   - CTA not in platform UI overlap area
   - No critical visual information in unsafe areas
   Verdict: [PASS/FAIL]

6. OVERALL COMPOSITION
   - No obvious defects (blank frames, missing elements, corrupted files)
   - Brand consistency (logo visible, colors consistent)
   - Professional appearance (would not be rejected by platform review)
   Overall score: [1-10] (minimum 7.0)

OVERALL AD UNIT VERDICT:
  ALL checks pass = READY FOR A09
  ANY check fails = FIX before packaging (return to assembly)
```
