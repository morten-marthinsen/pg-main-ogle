# 0.2: Copy Production Package Loader

## Purpose
Load A07 AD-COPY-FINAL/ containing all finalized scripts with timing marks, text overlays, and CTA copy per variant per platform. This is the source of truth for all text content in produced assets. A08 does NOT modify any copy -- it executes the copy as-is.

## Inputs
- A07 AD-COPY-FINAL/ directory (from upstream A07 -- Copy Production skill)
- All finalized scripts per concept, per variant, per platform

## Process
```
STEP 1: LOCATE COPY PRODUCTION PACKAGE
  - Search project directory for AD-COPY-FINAL/ directory
  - IF not found: HALT -- cannot proceed without finalized copy
  - Log directory path and file count

STEP 2: LOAD AND PARSE
  - FOR EACH copy file in AD-COPY-FINAL/:
    - EXTRACT concept ID (C01, C02, etc.)
    - EXTRACT variant ID (V01, V02, etc.)
    - EXTRACT platform target
    - EXTRACT script text (full verbatim text)
    - EXTRACT timing marks (when specific words/phrases appear on screen)
    - EXTRACT text overlay content (exact text for visual overlays)
    - EXTRACT CTA copy (exact button/banner text)
    - EXTRACT voiceover script (if different from display text)
    - EXTRACT emotional direction markers (if embedded in script)

STEP 3: VALIDATE COMPLETENESS
  - Every winning concept must have finalized copy
  - Every variant must have platform-specific copy
  - All text overlay content must be specified
  - All CTA copy must be specified
  - Flag any missing scripts

STEP 4: WRITE OUTPUT
  - Create per-microskill output file documenting all loaded copy
```
