# 2.2: CTA Variant Generation

## Purpose
For each concept, generate 2-3 CTA variants. Each CTA MUST use a DIFFERENT psychological lever from the CTA Lever Taxonomy. "Click now" vs "Tap now" vs "Order now" are NOT different variants -- they all use the same lever (Direct Command). Each CTA must match the platform's CTA conventions and meet the word count budget for the CTA module.

## Inputs
- Base copy scripts (from Layer 1: 1.2, 1.3, 1.4)
- CTA Lever Taxonomy (defined in AGENT.md)
- Platform constraints (from 0.5-format-strategy-loader.md)
- Word count limits

## Process
```
FOR EACH concept (C-001, C-002, C-003):

STEP 1: IDENTIFY the base CTA.
  - What CTA lever does the base copy use?
  - What is the exact CTA text?
  - What platform-native CTA format does it use?

STEP 2: SELECT 2-3 DIFFERENT levers for variants.
  CTA Lever Taxonomy:
    - Urgency: fear of missing out, time pressure
      Example: "Order now -- sale ends Friday"
    - Risk Reversal: remove purchase anxiety
      Example: "Try it for 60 days. Full refund if you're not thrilled."
    - Low-Friction: reduce commitment threshold
      Example: "Take the free quiz" / "Get your free sample"
    - Social Proof: bandwagon, safety in numbers
      Example: "Join 50,000 customers"
    - Curiosity Continuation: open loop, next-step curiosity
      Example: "See what your results could look like"
    - Direct Command: simple instruction, no friction
      Example: "Tap the link below"
    - Value Restatement: remind of value before asking
      Example: "Get all 3 bonuses plus free shipping -- tap below"

  Selection criteria:
    a. Each variant MUST use a different lever than the base AND each other
    b. Lever must be appropriate for: product type, price point, awareness level
    c. Lever must match platform-native CTA conventions

STEP 3: WRITE each CTA variant.
  - Write the CTA copy using the selected lever's psychology
  - Match platform-native CTA language:
    - TikTok: "Link in bio" / "Comment [keyword]"
    - YouTube: "Click the link below" / "Subscribe for more"
    - Meta: CTA button text ("Shop Now", "Learn More", "Get Offer") + caption CTA
    - Google: Headline CTA, description CTA
  - Verify word count within CTA module budget
  - Include any matching VISUAL column changes (CTA button text, URL overlay, etc.)

STEP 4: CLASSIFY each CTA variant.
  - Lever type (from taxonomy)
  - Platform-native format: [Y/N]
  - Word count: [exact]
  - Flow from body/proof module: [natural/forced]

STEP 5: VERIFY lever diversity.
  - No two CTAs in the same concept use the same lever
  - "Click now" vs "Tap now" is NOT diversity (both Direct Command)
```
