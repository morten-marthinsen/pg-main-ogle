# 2.2: Hook Generation: Curiosity & Information Gap (Category A)

## Purpose

Generate 3-5 hooks per angle using Category A types (A1-A6: Question, Curiosity Gap, Surprising Fact, Cliffhanger, Belief Reversal, Secret/Insider). These hooks create information gaps that compel the viewer to keep watching.

## Inputs

- 2.1-hook-type-matrix.md (angles mapped to Category A types)
- 0.3-hook-taxonomy.md (verbatim examples for A1-A6 — MUST be in active context)
- 0.4-soul-md.md (voice constraints)
- 0.0.1-vertical-profile.md (vertical-specific patterns)

## Process

1. **Load verbatim examples from taxonomy**
   - For each type being used (A1-A6), load verbatim examples
   - HOLD in active context during generation
   - Do NOT summarize — exact token sequences needed

2. **For each angle mapped to Category A types:**
   Generate 3-5 hooks using the mapped types

3. **Each hook MUST be:**
   - **Classified:** Tagged with specific type (A1, A2, etc.)
   - **Concise:** 5-15 words for text hooks
   - **Attributed:** Tagged to source angle
   - **Platform-tagged:** Which platforms this suits
   - **Voice-aligned:** Matches Soul.md constraints if loaded

4. **Hook format:**
   ```
   ### Hook C-001
   - **Text:** "[Hook text 5-15 words]"
   - **Type:** [A1/A2/A3/A4/A5/A6]
   - **Angle:** Angle [number] -- "[angle statement]"
   - **Platform:** [Meta / TikTok / YouTube / etc.]
   - **Word Count:** [number]
   - **Visual Direction Notes:** [if applicable]
   ```

5. **Apply vertical-specific patterns**
   - Health: "weird food" hooks (A6), doctor reveals (A6)
   - Golf: distance secrets (A6), pro techniques (A2)
   - Finance: hidden opportunity (A6), contrarian takes (A5)
