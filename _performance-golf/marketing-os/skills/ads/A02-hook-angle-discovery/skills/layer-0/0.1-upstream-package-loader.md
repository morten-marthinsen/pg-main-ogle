# 0.1: Upstream Package Loader

## Purpose

Load all required upstream packages from CopywritingEngine skills (09-Brief, 01-Research, 03-Root Cause, 04-Mechanism, 05-Promise, 06-Big Idea, 07-Offer, 02-Proof) to ground angle and hook generation in campaign strategy and market intelligence.

## Inputs

- Campaign Brief (Skill 09) — REQUIRED
- Research Handoff (Skill 01) — REQUIRED (pain quotes, hope quotes, language patterns)
- Root Cause (Skill 03) — REQUIRED
- Mechanism (Skill 04) — REQUIRED
- Promise (Skill 05) — REQUIRED
- Big Idea (Skill 06) — REQUIRED (critical — hooks LEAD to the big idea)
- Offer (Skill 07) — OPTIONAL (for proof-based angles)
- Proof Inventory (Skill 02) — OPTIONAL (for proof-based angles)

## Process

1. **Verify required packages exist**
   - Campaign Brief: BLOCKING if missing
   - Research Handoff: BLOCKING if missing
   - Root Cause: BLOCKING if missing
   - Mechanism: BLOCKING if missing
   - Promise: BLOCKING if missing
   - Big Idea: BLOCKING if missing

2. **Load required packages**
   - Read each package file
   - Extract relevant sections for hook/angle discovery:
     - Campaign Brief: target audience, awareness level, platforms, goals
     - Research: pain quotes (top 10-15), hope quotes (top 10-15), language patterns
     - Root Cause: root cause statement, surface problem, reframe technique
     - Mechanism: mechanism name, mechanism concept, how it works (simplified)
     - Promise: primary promise, promise calibration level
     - Big Idea: big idea statement (1-2 sentences), emotional core, FSSIT candidates

3. **Load optional packages if available**
   - Offer: offer structure, value stack, pricing (for proof-based angles)
   - Proof: strongest proof elements by category (for proof-based angles)

4. **Extract hook-relevant intelligence**
   - From Research: What language do prospects use? What pain is most intense? What hope is most vivid?
   - From Strategic outputs: What is the campaign's strategic foundation?
   - From Big Idea: What is the destination hooks must lead TO?

5. **Validate big idea clarity**
   - Big idea statement must be 1-2 sentences maximum
   - Big idea must be distinct from hooks (hooks lead TO it, not state it)
   - Flag if big idea is too vague or too complex for hook generation
