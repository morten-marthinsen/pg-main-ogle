# 0.2: Ad Intelligence Loader

## Purpose

Load A01 AD-INTELLIGENCE-HANDOFF.md containing competitive landscape analysis, winning hooks from competitors, hook type distribution analysis, and opportunity gaps to inform strategic angle selection and hook type targeting.

## Inputs

- A01 AD-INTELLIGENCE-HANDOFF.md (from Skill A01)

## Process

1. **Load AD-INTELLIGENCE-HANDOFF.md**
   - Path: `[project]/A01-ad-intelligence/AD-INTELLIGENCE-HANDOFF.md`
   - If file doesn't exist → HALT (A01 must run before A02)

2. **Extract competitive landscape**
   - Competitor list with positioning summaries
   - Hook patterns competitors are using
   - Messaging themes saturated in market
   - Messaging themes underexploited

3. **Extract hook type distribution**
   - Which of 32 hook types competitors use most
   - Hook type monoculture patterns (everyone using same types)
   - Hook types NOT being used (whitespace opportunity)

4. **Extract winning hooks**
   - Specific competitor hooks with performance data
   - Hook-to-outcome patterns (what worked)
   - Hook-to-angle patterns (what strategic territory worked)

5. **Extract opportunity gaps**
   - Underused hook types
   - Unexploited strategic angles
   - Platform-specific gaps
   - Uncrowded messaging territory

6. **Synthesize competitive intelligence for hook generation**
   - What should we AVOID (saturated patterns)?
   - What should we EXPLOIT (whitespace)?
   - What can we DIFFERENTIATE from (unique angles)?
