# 0.5: Input Validator

## Purpose

Verify all required inputs are loaded and valid before proceeding to Layer 1 (Angle Discovery). This is the final checkpoint before angle/hook generation begins.

## Inputs

- All Layer 0 outputs (0.0.1, 0.1, 0.2, 0.3, 0.4)

## Process

1. **Verify required packages loaded**
   - Campaign Brief: MUST exist
   - Research Handoff: MUST exist
   - Root Cause: MUST exist
   - Mechanism: MUST exist
   - Promise: MUST exist
   - Big Idea: MUST exist
   - Ad Intelligence Handoff (A01): MUST exist
   - Hook Taxonomy: MUST be loaded

2. **Verify big idea clarity**
   - Big idea statement exists and is 1-2 sentences
   - Big idea is NOT a hook (hooks lead TO it, don't state it)
   - Big idea provides clear destination for hooks

3. **Verify market intelligence loaded**
   - Minimum 10 pain quotes from research
   - Minimum 10 hope quotes from research
   - Competitive landscape understood
   - Opportunity gaps identified

4. **Verify hook taxonomy loaded**
   - All 32 types available
   - Examples loaded for reference
   - Performance benchmarks available

5. **Check optional inputs**
   - Vertical profile: loaded or defaulted to universal
   - Soul.md: loaded or voice constraints absent
   - Proof inventory: available or not (affects proof-based angles)
   - Offer: available or not (affects proof-based angles)

6. **Validate readiness for angle discovery**
   - Do we have enough strategic foundation to derive angles?
   - Do we have enough research intelligence to ground angles?
   - Do we have clear competitive differentiation strategy?

## Output File

- **Path:** `layer-0-outputs/0.5-input-validation.md`
- **Minimum size:** 1KB

## Quality Checks

- **IF all_required_inputs is false → Gate 0 FAILS**
- Big idea MUST be 1-2 sentences (not a paragraph, not vague)
- Minimum 10 pain quotes AND 10 hope quotes
- Hook taxonomy MUST have exactly 32 types

## Dependencies

- 0.0.1 output exists
- 0.1 output exists
- 0.2 output exists
- 0.3 output exists
- 0.4 output exists (even if Soul.md missing, output confirms status)

## Gate 0 Blocking

**IF THIS VALIDATION FAILS → LAYER 1 IS BLOCKED**

The gate file `GATE_0_COMPLETE.yaml` will NOT be created unless this validation passes.
