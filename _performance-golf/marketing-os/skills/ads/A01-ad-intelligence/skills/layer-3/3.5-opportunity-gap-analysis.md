# 3.5: Opportunity Gap Analysis

**Version:** 1.0
**Layer:** 3 (Intelligence Synthesis)
**Type:** Strategic Analysis
**Dependencies:** 3.1, 3.2, 3.3 complete; Skill 01 Research output (if available)
**Model:** opus

---

## Purpose

Identify actionable opportunity gaps in the competitive ad landscape. This includes underused hook types with high performance potential, format gaps, messaging gaps (pain points from Skill 01 research that no competitor addresses in ads), and visual style gaps. Each gap is ranked by estimated impact and paired with a specific recommended action for this campaign.

This is the highest-value synthesis output -- it tells the campaign team WHERE TO PLAY for maximum differentiation and impact. Vague observations like "there are opportunities" are worthless. Each gap must be specific, evidence-backed, and actionable.

## Process

### Step 1: Hook Type Gap Analysis

```
FROM 3.2 hook type distribution:

  IDENTIFY hook type gaps:

  TYPE 1 — HIGH-PERFORMANCE UNDERUSED:
    Hook types with winner_representation_ratio > 1.5 BUT total_pct < 10%
    These types work well but few competitors use them.
    → STRONG opportunity signal

  TYPE 2 — BENCHMARK-SUPPORTED ABSENT:
    Hook types with 0% representation in scraped ads BUT positive benchmark data
    from AD-HOOK-TAXONOMY.md in comparable markets.
    → Whitespace opportunity -- no competition at all

  TYPE 3 — CATEGORY IMBALANCE:
    Hook categories (A-J) where total representation is < 5%
    An entire category is being ignored by competitors.
    → Broad angle opportunity

  FOR EACH gap:
    gap:
      gap_type: "hook_type_gap"
      hook_type_id: string
      hook_type_name: string
      current_usage_pct: float
      winner_representation: float
      benchmark_data: string  # from taxonomy
      evidence: string  # specific data supporting this gap
      estimated_impact: string  # HIGH / MEDIUM / LOW
      impact_rationale: string
      recommended_action: string
        # Specific action — not "consider using this hook type"
        # e.g., "Create 3 test ads using D3 (Testimonial/UGC) hook on TikTok,
        #        targeting the 'gut health skeptic' persona identified in research"
```

### Step 2: Format Gap Analysis

```
FROM 3.3 format-platform intelligence:

  IDENTIFY format gaps:

  TYPE 1 — PLATFORM FORMAT MISMATCH:
    Formats that are known to perform well on a platform (per platform data)
    but are underused by competitors in this vertical.
    e.g., "Carousel is 15% of Meta ads, but only 2% in this vertical"

  TYPE 2 — VIDEO LENGTH GAP:
    Video length buckets that are underrepresented vs. platform norms.
    e.g., "6s bumper ads are 0% of competitors but are a proven format on YouTube"

  TYPE 3 — CROSS-PLATFORM OPPORTUNITY:
    Platforms where competitors are sparse but market potential exists.
    e.g., "Only 2 of 12 competitors run TikTok ads — low competition"

  FOR EACH gap:
    gap:
      gap_type: "format_gap"
      format: string
      platform: string  # if platform-specific
      current_usage_pct: float
      benchmark_reference: string
      evidence: string
      estimated_impact: string
      impact_rationale: string
      recommended_action: string
```

### Step 3: Messaging Gap Analysis (Cross-Reference with Skill 01 Research)

```
IF Skill 01 Research FINAL_HANDOFF.md is available:

  LOAD research output and extract:
    - Top pain points (what audience says hurts most)
    - Top hope themes (what audience wants most)
    - Root cause beliefs (what audience thinks causes the problem)
    - Solutions tried (what audience has tried and failed)
    - Competitor mechanisms (what competitor products claim)
    - Villain analysis (who/what audience blames)
    - Verbatim audience quotes (exact language patterns)

  CROSS-REFERENCE with ad copy patterns from 2.4:

  IDENTIFY messaging gaps:

  TYPE 1 — UNADDRESSED PAIN POINTS:
    Pain points from research that appear in zero competitor ad copy.
    The audience talks about this pain, but no ad addresses it.
    → Strong messaging opportunity

  TYPE 2 — UNUSED AUDIENCE LANGUAGE:
    Specific phrases and words the audience uses that no competitor mirrors.
    Audience says "I feel like my body is fighting against me"
    but no ads use this language.
    → Voice/language opportunity

  TYPE 3 — AWARENESS LEVEL MISMATCH:
    Audience is at awareness level X (from research) but competitors target level Y.
    e.g., "Audience is mostly problem-aware but competitors run solution-aware ads"
    → Awareness targeting opportunity

  TYPE 4 — EMOTIONAL ANGLE GAPS:
    Emotional themes from research (fear, frustration, hope, identity) that
    no competitor touches in their ad creative.
    → Emotional resonance opportunity

  FOR EACH gap:
    gap:
      gap_type: "messaging_gap"
      sub_type: string  # pain_point / language / awareness / emotional
      research_evidence: string  # specific data from Skill 01
      ad_landscape_evidence: string  # what competitors ARE doing instead
      gap_description: string  # what is missing
      estimated_impact: string
      impact_rationale: string
      recommended_action: string

IF Skill 01 Research NOT available:
  NOTE: "Messaging gap analysis limited — Skill 01 Research not available.
         Format and hook type gaps are complete. Messaging gaps require
         Skill 01 output for cross-reference."
  PROCEED with hook and format gaps only.
```

### Step 4: Visual Style Gap Analysis

```
FROM 2.3 visual style analysis and 3.3 format-platform intelligence:

  IDENTIFY visual style gaps:

  TYPE 1 — PRODUCTION QUALITY GAP:
    If market is 90% polished, UGC is a gap (and vice versa).
    Production quality counter-positioning.

  TYPE 2 — STYLE x PLATFORM MISMATCH:
    Visual styles that underperform on one platform but could work on another.
    e.g., "Before/after imagery banned on Meta but performing well on TikTok"

  TYPE 3 — COMPETITOR VISUAL MONOTONY:
    If all competitors use the same visual style, ANY different style is differentiation.
    e.g., "All 12 competitors use polished production. Zero UGC."

  FOR EACH gap:
    gap:
      gap_type: "visual_style_gap"
      current_market_pattern: string
      gap_description: string
      evidence: string
      estimated_impact: string
      impact_rationale: string
      recommended_action: string
```

### Step 5: Opportunity Ranking

```
AGGREGATE all identified gaps (hook, format, messaging, visual).

RANK by estimated impact using these criteria:
  1. Size of the gap (larger gap = more whitespace = higher impact)
  2. Performance evidence (benchmark data supporting the opportunity)
  3. Alignment with campaign brief (does this gap align with our target audience?)
  4. Execution feasibility (can we actually produce this type of ad?)
  5. Competitive moat (how long before competitors copy us if this works?)

PRODUCE ranked opportunity list:
  Top 5 = highest priority for this campaign
  Remaining = secondary opportunities

MINIMUM: 5 identified opportunities.
IF fewer than 5 found after all analysis: re-examine data with fresh perspective.
The threshold exists because a competitive market ALWAYS has gaps.
```

## Quality Checks

1. **Minimum 5 opportunities identified** -- fewer than 5 means analysis is insufficient.
2. **All 4 gap types analyzed** -- hook type, format, messaging (if research available), visual style.
3. **Evidence is specific** -- every gap cites specific data (percentages, counts, quotes), not vague claims.
4. **Recommended actions are actionable** -- "Create 3 test ads using X hook type on Y platform" not "consider this opportunity."
5. **Impact assessments have rationale** -- HIGH/MEDIUM/LOW with specific reasoning.
6. **Ranking considers multiple criteria** -- not just size of gap but performance evidence and campaign alignment.
7. **Skill 01 cross-reference attempted** -- if research exists, messaging gaps are analyzed. If not, the limitation is documented.
8. **Gaps are DIFFERENT from each other** -- no duplicative entries (same gap phrased differently).
