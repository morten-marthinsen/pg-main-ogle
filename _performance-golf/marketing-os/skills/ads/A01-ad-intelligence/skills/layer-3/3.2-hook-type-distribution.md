# 3.2: Hook Type Distribution

**Version:** 1.1
**Layer:** 3 (Intelligence Synthesis)
**Type:** Distribution Analysis
**Dependencies:** Layer 2 complete (LAYER_2_COMPLETE.yaml exists), AD-HOOK-TAXONOMY.md loaded
**Model:** opus
**Updated:** 2026-02-27 (v1.1 — impression-weighted distribution support)

---

## Purpose

Calculate the prevalence of each of the 32 hook types across all scraped ads. Identify overused (saturated) types, underused (opportunity) types, and types with disproportionate TIER_WINNER representation (high-performance types). Cross-reference with AD-HOOK-TAXONOMY.md benchmark data. Calculate per-competitor hook type diversity. This analysis directly feeds opportunity gap identification (3.5) and the top-10 hook recommendations (Section 10 of the handoff).

## Process

### Step 1: Full 32-Type Distribution Table

```
FOR EACH of the 32 hook types in the taxonomy:

  CALCULATE:
    total_count: number of ads classified as this type
    total_pct: percentage of all scraped ads
    tier_winner_count: number of TIER_WINNER ads with this type
    tier_winner_pct: percentage of this type's ads that are TIER_WINNER
    winner_representation_ratio: (tier_winner_pct for this type) / (overall tier_winner_pct)
      # > 1.0 = over-represented in winners (good performance signal)
      # < 1.0 = under-represented in winners (weaker performance)
      # = 0 = no winners with this type
    platforms: which platforms this type appears on
    top_brands: which brands use this type most

  ASSIGN saturation_assessment:
    "oversaturated" — top 5 by frequency AND winner_representation_ratio < 1.5
      (everyone uses it, it doesn't particularly outperform)
    "high_performance" — winner_representation_ratio > 2.0
      (disproportionately represented in long-running ads)
    "balanced" — neither oversaturated nor underused, reasonable frequency
    "underused" — bottom 10 by frequency BUT has benchmark data suggesting potential
    "absent" — zero ads use this type (0%)
    "underused_high_potential" — underused AND winner_representation_ratio > 1.5
      (the few ads using it perform well — strong opportunity signal)
```

### Step 2: Saturation Analysis

```
IDENTIFY top 5 most prevalent hook types:
  Sort by total_count descending → take top 5

  FOR EACH:
    - type_id and name
    - count and percentage
    - winner representation ratio
    - Is this type oversaturated? (high frequency + low winner ratio = saturated)
    - What does saturation mean for our campaign?
      → "Entering this space means competing against [X]% of all ads"
    - Verbatim examples from scraped ads (2-3 examples)
```

### Step 3: Opportunity Identification

```
IDENTIFY bottom 10 least used hook types:
  Sort by total_count ascending → take bottom 10

  FOR EACH:
    - type_id and name
    - count and percentage (may be 0%)
    - AD-HOOK-TAXONOMY.md benchmark data (does this type perform well in other markets?)
    - winner_representation_ratio (if any ads exist)
    - Opportunity assessment:
      "strong_opportunity" — underused AND benchmarks show high performance elsewhere
      "moderate_opportunity" — underused, some benchmark support
      "low_opportunity" — underused for good reason (niche, compliance restricted, etc.)
      "unknown_opportunity" — insufficient data to assess

  IDENTIFY hook types with highest winner_representation_ratio:
    These are the "hidden winners" — types that appear rarely but when they do,
    they tend to be long-running ads. Strong signal for A02 hook generation.
```

### Step 4: Per-Competitor Hook Diversity

```
FOR EACH competitor brand:

  CALCULATE:
    total_hook_types_used: integer  # out of 32
    hook_diversity_score: float
      # Simpson's Diversity Index or simple count-based
      # 1-5 types = "narrow" (1.0)
      # 6-10 types = "moderate" (2.0)
      # 11-15 types = "diverse" (3.0)
      # 16+ types = "highly diverse" (4.0+)
    dominant_type: string  # most-used hook type
    dominant_type_pct: float  # what % of their ads use it
    strategy_characterization: string
      # e.g., "Heavily reliant on authority hooks (60%), narrow strategy"
      # e.g., "Diverse hook strategy across 14 types, no single type > 15%"

  IDENTIFY:
    most_diverse_brand: string  # uses the most hook types
    most_narrow_brand: string  # relies on fewest hook types
    unique_type_users: list  # brands using types that NO other competitor uses
```

### Step 5: Cross-Reference with Taxonomy Benchmarks

```
IF AD-HOOK-TAXONOMY.md contains performance benchmark data:

  FOR EACH hook type:
    COMPARE:
      market_frequency: our scraped data (% of ads using this type)
      benchmark_performance: from taxonomy (conversion rate, CTR, engagement data)

    IDENTIFY mismatches:
      "HIGH BENCHMARK + LOW MARKET FREQUENCY" → Strong opportunity
        Competitors are ignoring a proven hook type
      "LOW BENCHMARK + HIGH MARKET FREQUENCY" → Potential waste
        Competitors are overusing an underperforming hook type
      "HIGH BENCHMARK + HIGH MARKET FREQUENCY" → Competitive necessity
        Must compete here, but differentiation is harder
      "LOW BENCHMARK + LOW MARKET FREQUENCY" → Avoid
        Low potential, market correctly ignoring it
```

### Step 6: Impression-Weighted Distribution (Tool-Assisted Scan Only)

```
IF impression_scores available (2.7 output exists):

  CALCULATE impression-weighted hook distribution:
    FOR EACH of the 32 hook types:
      count_weighted: SUM of performance_score for all ads with this hook type
        # Instead of counting each ad as 1, count it as its performance_score (0.0-1.0)
        # High-impression, long-running ads contribute more to the type's weight
      pct_weighted: count_weighted / SUM of all performance_scores

    COMPARE count-based vs impression-weighted distributions:
      FOR EACH hook type:
        count_rank: rank by total_count
        weighted_rank: rank by count_weighted
        rank_delta: count_rank - weighted_rank
          # Positive delta = type ranks HIGHER when weighted by impressions
          #   → Its ads over-index on performance relative to frequency
          # Negative delta = type ranks LOWER when weighted
          #   → Many ads of this type, but they underperform

    IDENTIFY "Performance-Weighted Risers":
      Hook types with rank_delta >= 3
      These are types that appear moderately frequent but whose ads have
      disproportionately high impression/duration scores.
      Strong signal for A02: "this type WORKS, even if it's not the most common"

    IDENTIFY "Performance-Weighted Fallers":
      Hook types with rank_delta <= -3
      These are types that are common by count but underperform by impressions.
      Signal: competitors use this type a lot but it doesn't produce winners.

  OUTPUT impression_weighted_hook_distribution:
    Same 32-type table with additional columns:
    - count_weighted: float
    - pct_weighted: float
    - weighted_rank: integer
    - rank_delta: integer
    - performance_assessment: "riser" / "faller" / "stable"

ELSE (no impression data):
  SKIP this step. Count-based distribution (Steps 1-5) is the sole output.
  LOG: "No impression data available — using count-based distribution only."
```

## Quality Checks

1. **All 32 hook types in distribution table** -- even types with 0% representation must appear.
2. **Winner representation ratios calculated** -- every type has a ratio (or explicit "N/A" if no TIER_WINNER ads exist).
3. **Saturation assessments use defined categories** -- oversaturated/high_performance/balanced/underused/absent/underused_high_potential.
4. **Competitor diversity scores cover all brands** -- no brands skipped.
5. **Benchmark cross-reference attempted** -- if taxonomy has benchmarks, the cross-reference is complete.
6. **Key findings are actionable** -- "Hook type X is underused" includes a specific recommendation for this campaign.
7. **Examples are verbatim** -- hook text examples are exact text from scraped ads.
