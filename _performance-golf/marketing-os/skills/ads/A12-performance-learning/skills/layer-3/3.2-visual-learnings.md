# 3.2: Visual Learnings

**Version:** 1.0
**Layer:** 3 (Learning Extraction)
**Type:** Learning Extraction
**Model:** opus
**Dependencies:** Layer 2 outputs (GATE_2 must be PASS), 2.5 element-attribution
**Output:** layer-3-outputs/3.2-visual-learnings.md

---

## Purpose

Extract actionable learnings about visual style performance. Did UGC outperform polished as expected? By how much? On which platforms? Which visual elements correlated with higher hook rates? Which with higher conversion rates? Were there visual-hook combinations that significantly outperformed? WHY did winning visuals win (hypothesis: authenticity trust? pattern interrupt? product visibility? emotional resonance?)? Generate specific visual direction recommendations for the next campaign.

## Process

### Step 1: Aggregate Visual Style Performance

```
GROUP variants by visual_style (UGC, polished, mixed, animated, static, talking_head, etc.):

  FOR EACH visual style:
    variant_count: integer
    avg_hook_rate: float
    avg_CPA: float (spend-weighted)
    avg_ROAS: float (spend-weighted)
    avg_CTR: float
    avg_ThruPlay_rate: float (for video styles)
    win_rate: winners / total * 100
    total_spend: float

  RANK visual styles by CPA (best to worst)
  RANK visual styles by hook_rate (best to worst)
```

### Step 2: UGC vs Polished Deep Analysis

```
IF both UGC and polished variants exist with sufficient data:

  COMPARE:
    | Metric | UGC | Polished | Delta | Winner |
    |--------|-----|----------|-------|--------|
    | Hook Rate | [X]% | [X]% | [X]% | [style] |
    | CPA | $[X] | $[X] | [X]% | [style] |
    | ROAS | [X]x | [X]x | [X]% | [style] |
    | CTR | [X]% | [X]% | [X]% | [style] |
    | ThruPlay | [X]% | [X]% | [X]% | [style] |
    | Win Rate | [X]% | [X]% | [X]% | [style] |

  PLATFORM BREAKDOWN:
    On Meta: UGC [X]% better/worse CPA than polished
    On TikTok: UGC [X]% better/worse CPA than polished
    On YouTube: [if applicable]

  HYPOTHESIS for UGC performance:
    - Authenticity trust (users trust "real people" over produced content)
    - Pattern interrupt (UGC looks different from typical ads in feed)
    - Platform algorithm preference (Meta/TikTok may favor native-looking content)
    - Lower production cost allows more variant testing
```

### Step 3: Visual-Hook Synergy Analysis

```
FROM 2.5 element-attribution synergy data:

  IDENTIFY visual-hook combinations that significantly outperformed:
    [visual_style] + [hook_type] = [X]% better CPA than either element alone

  IDENTIFY visual-hook conflicts:
    [visual_style] + [hook_type] = worse than either element alone

  SPECIFIC PATTERNS TO INVESTIGATE:
    - UGC visual + testimonial hook (expected synergy)
    - Polished visual + authority hook (expected synergy)
    - UGC visual + polished script (potential conflict -- mismatched production value)
    - Animated visual + data hook (potential synergy -- visual data presentation)
```

### Step 4: Extract Visual Learnings

```
FOR EACH actionable learning:

  FORMAT (MANDATORY -- all 6 fields required):

  ### Learning V-[ID]: [Title]

  **WHAT:** [Specific observation with data]
  **WHY:** [Causal hypothesis about visual performance]
  **HOW:** [Actionable recommendation for visual direction]
  **CONFIDENCE:** [HIGH/MEDIUM/LOW with basis]
  **SCOPE:** [Vertical-specific/Universal] [Campaign-specific/Generalizable]
  **PROPAGATION TARGET:** [Which engine file(s) to update]

MINIMUM: 2 visual learnings per campaign. Target: 3-4.
```

### Step 5: Generate Visual Direction Recommendations

```
SYNTHESIZE into next campaign visual strategy:

  VISUAL STYLE ALLOCATION:
    1. [style] -- [X]% of variants -- [rationale]
    2. [style] -- [X]% of variants -- [rationale]
    ...

  VISUAL-HOOK PAIRINGS TO PRIORITIZE:
    - [visual] + [hook_type] -- strong synergy evidence
    - [visual] + [hook_type] -- moderate synergy evidence

  VISUAL-HOOK PAIRINGS TO AVOID:
    - [visual] + [hook_type] -- conflict evidence

  PLATFORM-SPECIFIC VISUAL RECOMMENDATIONS:
    Meta: [recommendations]
    TikTok: [recommendations]
    [etc.]

  VISUAL TESTS TO RUN:
    - [untested visual style or combination] -- establish baseline data
```

8. Visual tests for next campaign identified
