# 3.3: Platform Learnings

**Version:** 1.0
**Layer:** 3 (Learning Extraction)
**Type:** Learning Extraction
**Model:** opus
**Dependencies:** Layer 2 outputs (GATE_2 must be PASS)
**Output:** layer-3-outputs/3.3-platform-learnings.md

---

## Purpose

Extract actionable learnings per platform. Which platform delivered the best ROAS? Which the best CPA? Were there platform-specific creative patterns (hooks that worked on TikTok but not Meta, or vice versa)? How did platform-specific constraints (sound-off on Meta, vertical format on TikTok) affect performance? Generate platform-specific creative recommendations for next campaign.

## Process

### Step 1: Per-Platform Aggregate Performance

```
FOR EACH platform in the campaign:

  CALCULATE platform-level aggregates:
    total_variants: integer
    total_spend: float
    total_conversions: integer
    total_impressions: integer
    platform_CPA: float
    platform_ROAS: float
    platform_CTR: float
    platform_avg_hook_rate: float
    platform_avg_CPM: float (cost to reach audience)
    winners: count
    performers: count
    losers: count
    insufficient: count
    win_rate: float

  RANK platforms by:
    1. ROAS (best to worst)
    2. CPA (best to worst)
    3. Win rate (best to worst)
    4. Hook rate (best to worst -- creative effectiveness)
    5. CPM (lowest to highest -- audience cost)

  PRODUCE platform comparison table:
    | Platform | Spend | CPA | ROAS | Hook Rate | CTR | CPM | Win Rate | Variants |
```

### Step 2: Platform-Specific Creative Patterns

```
FOR EACH platform:
  ANALYZE which creative elements performed best ON THAT PLATFORM:

  HOOK TYPES BY PLATFORM:
    Which hook types had highest hook rate on this platform?
    Which had best CPA?
    How does this compare to other platforms?
    Example: "D3 (UGC testimonial) hook rate on Meta: 34% vs on TikTok: 28%"

  VISUAL STYLES BY PLATFORM:
    Which visual styles performed best on this platform?
    Example: "UGC CPA on TikTok: $19 vs polished CPA on TikTok: $31"

  FORMAT BY PLATFORM:
    Which ad formats performed best?
    Example: "Short video (<15s) CPA on TikTok: $22 vs long video (>30s): $35"

  IDENTIFY PLATFORM-SPECIFIC WINNERS:
    Hooks that worked on one platform but NOT another:
    "A2 (curiosity) achieved $20 CPA on Meta but $45 CPA on TikTok"
    This reveals platform-audience-creative interactions.
```

### Step 3: Platform Constraint Analysis

```
ANALYZE how platform-specific constraints affected performance:

  META:
    - Sound-off default: Did variants with text overlay outperform those without?
    - Feed vs Stories vs Reels placement: Performance by placement?
    - Image vs video: Performance comparison on Meta specifically?

  TIKTOK:
    - Vertical-first format: Did vertically-native content outperform repurposed?
    - Sound-on default: Did music/narration quality affect performance?
    - Trending format alignment: Did trend-aligned content outperform?

  YOUTUBE:
    - Skip behavior: How many seconds before skip? Hook rate implications?
    - Long-form tolerance: Did longer formats perform differently?

  GOOGLE:
    - Responsive vs static: Performance comparison?
    - Headline variants: Which message angles performed?
```

### Step 4: Competitive Benchmark Comparison

```
FROM 0.4 benchmarks:
  COMPARE campaign platform performance vs competitive benchmarks:

  | Platform | Our CTR | Bench CTR | Delta | Our CPA | Bench CPA | Delta | Our CPM | Bench CPM |
  |----------|---------|-----------|-------|---------|-----------|-------|---------|-----------|

  IDENTIFY:
    - Platforms where we outperform the vertical average
    - Platforms where we underperform
    - Platform opportunity gaps (high potential, currently underperforming)
```

### Step 5: Extract Platform Learnings

```
FOR EACH actionable learning:

  FORMAT (MANDATORY):

  ### Learning P-[ID]: [Title]

  **WHAT:** [Specific platform performance observation with data]
  **WHY:** [Hypothesis about platform-specific performance]
  **HOW:** [Platform-specific recommendation for next campaign]
  **CONFIDENCE:** [HIGH/MEDIUM/LOW]
  **SCOPE:** [Platform-specific is always the scope here]
  **PROPAGATION TARGET:** [Vertical profile, A10 platform scoring, AD-ENGINE-CLAUDE.md]

MINIMUM: 1 platform learning per platform used. Target: 2-3 per platform.
```

### Step 6: Platform Strategy Recommendations

```
SYNTHESIZE into next campaign platform strategy:

  PLATFORM PRIORITIZATION:
    1. [platform] -- primary -- [rationale from CPA/ROAS data]
    2. [platform] -- secondary -- [rationale]
    3. [platform] -- test/expand -- [rationale]

  PER-PLATFORM CREATIVE STRATEGY:
    [platform]:
      - Prioritize hook types: [types]
      - Prioritize visual styles: [styles]
      - Optimal format: [format and length]
      - Budget allocation: [X]% of total

  CROSS-PLATFORM CREATIVE REUSE GUIDANCE:
    - Creative that transfers well: [elements that work across platforms]
    - Creative that does NOT transfer: [platform-specific elements]
    - Platform-native content requirement: [percentage per platform]
```
