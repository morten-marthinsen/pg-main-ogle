## Execution Steps

### Step 1: Map the Emotional Journey

Design the emotional journey from the lead through the close, informed by E5 architecture (lead = emotional sale, argument = logical sale, close = financial sale):

```yaml
emotional_arc:
  opening_emotion: string       # What the prospect feels at the lead (curiosity, recognition, alarm, intrigue)
  lead_transition: string       # How the lead shifts emotion (from X to Y)
  story_emotion: string         # What the story makes them feel (empathy, hope, outrage, identification)
  argument_emotion: string      # What the logical argument produces (belief, understanding, confidence, vindication)
  reveal_emotion: string        # What the product/mechanism reveal produces (excitement, relief, desire, clarity)
  offer_emotion: string         # What the offer section produces (urgency, certainty, commitment, anticipation)
  close_emotion: string         # What the close produces (decisiveness, fear-of-missing, conviction)
```

Each emotion must be:
- Specific to THIS campaign (not generic)
- Connected to the lead type and story type selected
- Grounded in the market psychology from research

### Step 2: Define Emotional Intensity Curve

Map the emotional intensity across the campaign (1-10 scale):

```yaml
intensity_curve:
  - section: "Lead"
    emotion: string
    intensity: integer  # 1-10
    function: string    # E5 function: "emotional sale"
  - section: "Story"
    emotion: string
    intensity: integer
    function: string
  - section: "Root Cause Narrative"
    emotion: string
    intensity: integer
    function: string    # "logical sale begins"
  - section: "Mechanism Narrative"
    emotion: string
    intensity: integer
    function: string
  - section: "Product Introduction"
    emotion: string
    intensity: integer
    function: string    # "reveal"
  - section: "Offer"
    emotion: string
    intensity: integer
    function: string    # "financial sale"
  - section: "Close"
    emotion: string
    intensity: integer
    function: string    # "action"
```

The curve should show:
- High opening intensity (grab attention)
- Story peak (emotional engagement)
- Brief dip during logical argument (recalibrate — building belief, not excitement)
- Steady rise through reveal and offer
- Peak at close (maximum urgency/conviction)

### Step 3: Establish Tone Direction

```yaml
tone_direction:
  primary_tone: string          # The dominant voice
  primary_justification: string # WHY this tone — connected to mechanism, Big Idea, market
  secondary_tone: string        # The supporting voice
  secondary_justification: string
  avoid: [string]               # Tones that would undermine the campaign
  avoid_justification: [string] # WHY each avoidance — specific to this campaign
  calibration_notes: string     # Specific guidance for tone calibration
```

Tone options (select from, do not invent new categories):
- **Authoritative:** Expert voice, confident assertions, clinical precision
- **Empathetic:** Understanding, shared experience, warmth
- **Conspiratorial:** Insider knowledge, secrets, "what they don't want you to know"
- **Scientific:** Evidence-first, research-driven, measured claims
- **Conversational:** Casual, direct, person-to-person
- **Urgent:** Time-pressured, consequences-focused, action-demanding
- **Passionate:** Emotionally invested, mission-driven, conviction
- **Matter-of-fact:** Straightforward, no-nonsense, let-the-data-speak

### Step 4: Cross-Reference with Vault Patterns

Compare the designed arc against vault_intelligence.emotional_arc_patterns:
- Do elite controls in this niche follow a similar intensity curve?
- Does the tone direction match successful campaigns with similar mechanism types?
- Are there vault patterns that suggest a different emotional approach?

### Step 5: Validate Arc Against Campaign Elements

- Does the opening emotion match the lead type recommendation?
- Does the story emotion match the story type recommendation?
- Does the argument emotion build belief in the campaign thesis?
- Does the reveal emotion connect to the mechanism's nature?
- Does the offer emotion connect to the promise?
- Does the close emotion create action without contradicting the primary tone?

## Quality Gate

- [ ] All 7 emotional stages defined (opening through close)
- [ ] Each emotion is specific to THIS campaign (not generic)
- [ ] Intensity curve mapped across all sections
- [ ] Primary and secondary tones selected with justification
- [ ] At least 2 tone avoidances specified with justification
- [ ] Calibration notes provide specific guidance
- [ ] Arc validated against lead type and story type recommendations
- [ ] Vault patterns cross-referenced
- [ ] Emotional arc covers the full E5 framework (emotional sale -> logical sale -> financial sale)
- [ ] No anti-slop violations in arc descriptions

---

## Anti-Patterns

- **Generic arc:** "Start with curiosity, build to excitement, close with urgency" — this could describe any campaign. Must be specific.
- **Flat intensity:** All sections at similar intensity — the campaign should have peaks and valleys
- **Tone contradiction:** Primary tone "scientific" but lead type is vulnerability story — tonal disconnect
- **Missing avoidances:** Not specifying what tones to AVOID is as important as specifying what tones to use
