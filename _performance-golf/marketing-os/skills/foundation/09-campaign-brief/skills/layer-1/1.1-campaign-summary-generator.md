# Layer 1.1 — Campaign Summary Generator

**Skill:** 09 — Campaign Brief
**Microskill:** 1.1 — Campaign Summary Generator
**Layer:** 1
**Model:** opus
**Purpose:** Generate the one-page Campaign Summary from all 8 upstream packages — the "war room view" that distills the complete strategic position into a single readable page.

---

## Input

| Source | File | Required |
|--------|------|----------|
| 0.1-upstream-loader | `0.1-upstream-context.json` | Yes |
| 0.4-input-validator | `0.4-validation-report.json` (must be PASS) | Yes |

## Output

| File | Format | Min Size |
|------|--------|----------|
| `1.1-campaign-summary.json` | JSON | 3KB |

## Execution Steps

### Step 1: Distill Each Element to Its Essence

For each upstream element, write a single clear sentence that captures its core:

```yaml
element_summaries:
  market_context: string       # 2-3 sentences: who is the audience, what's their situation
  avatar_headline: string      # 1 sentence: who specifically are we talking to
  root_cause: string           # 1 sentence: the real reason they're stuck
  villain: string              # 1 sentence: who/what is to blame
  mechanism: string            # 1 sentence: the named mechanism and what it does
  promise: string              # 1 sentence: what the prospect gets
  big_idea: string             # 1 sentence: the central compelling concept
  offer_headline: string       # 1 sentence: what they're buying
  structure_headline: string   # 1 sentence: how the argument flows
```

**Constraint:** Each sentence must use terminology EXACTLY as it appears in the upstream packages. No synonym substitution. If the mechanism is called "The Neural Reset Protocol," call it that — not "the brain reset method."

### Step 2: Write the Full Connecting Paragraph

Write a 4-6 sentence paragraph that connects ALL elements into one coherent narrative:

```yaml
full_paragraph: string  # 4-6 sentences connecting ALL elements
```

**The full_paragraph is the acid test:** If you cannot write a coherent paragraph connecting all elements, there IS an incoherence problem. The paragraph is not decoration — it is the diagnostic.

**Structure of the paragraph:**
1. Open with the audience situation and root cause
2. Introduce the villain and why existing solutions failed
3. Present the Big Idea frame that recontextualizes the problem
4. Introduce the mechanism as the solution the Big Idea reveals
5. State the promise as the outcome the mechanism delivers
6. Connect to the offer as the vehicle for accessing the mechanism

### Step 3: Distill the One-Sentence Campaign Thesis

Extract or refine the campaign thesis from structure-package.json into a single sentence:

```yaml
one_sentence_thesis: string  # The single overarching claim the entire campaign proves
```

The thesis must:
- Be expressible in one sentence (hard limit: 2 sentences)
- Connect root cause -> mechanism -> promise
- Be provable through evidence and education
- Sound like a prosecutor's opening statement, not a sales pitch

### Step 4: Validate Summary Against Upstream

Cross-check every element summary against its upstream source:
- No element should contradict its upstream package
- No element should add claims not present in upstream
- Terminology must match exactly
- If any contradiction is found, flag it explicitly

## Quality Gate

- [ ] All 9 element summaries populated with specific, non-generic content
- [ ] Full paragraph connects ALL elements without contradiction
- [ ] Full paragraph is 4-6 sentences (not shorter, not longer)
- [ ] One-sentence thesis distilled from structure package
- [ ] Terminology matches upstream packages exactly — no synonym substitution
- [ ] No anti-slop violations in any generated text
- [ ] Paragraph reads as a natural, compelling narrative (not a list of bullet points)
- [ ] Any contradictions between elements flagged explicitly

---

## Anti-Patterns

- **Disconnected bullet points:** Summary reads like a list with no narrative thread — FAIL
- **Synonym substitution:** Changing "Neural Reset Protocol" to "brain reset method" — REJECT
- **Generic filler:** "This campaign targets health-conscious consumers" when the avatar is specific — REJECT
- **Thesis as sales pitch:** "You need to try this breakthrough" — this is selling, not prosecuting

## Handoff

- **1.3-campaign-thesis-validator.md** (thesis for validation)
- **4.1-campaign-brief-assembler.md** (summary included in final brief)
