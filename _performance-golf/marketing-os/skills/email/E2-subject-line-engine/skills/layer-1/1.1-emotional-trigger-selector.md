# 1.1-emotional-trigger-selector.md

> **Layer:** 1 (Formula Selection)
> **Skill:** E2-subject-line-engine
> **Purpose:** Identify which emotional triggers fit this specific email content
> **Outputs:** `emotional_triggers` object with ranked trigger list and evidence

---

## FUNCTION

Analyze the email body loaded in Layer 0 and identify the dominant emotional triggers present in the content. These triggers drive formula selection in 1.2 -- each formula category works best with specific emotional triggers. The goal is to match the SL's emotional pull to the email's actual emotional content so the reader's feeling upon opening aligns with their feeling while reading.

---

## EMOTIONAL TRIGGER TAXONOMY

| Trigger Code | Trigger | Description | Best Formula Categories |
|--------------|---------|-------------|------------------------|
| ET-CUR | Curiosity | Reader needs to know something. Information gap created. | SL-THE, SL-CUR, SL-PAR, SL-ATK |
| ET-FER | Fear/Warning | Something bad could happen. Threat awareness. | SL-CMD, SL-WHY, SL-URG |
| ET-DES | Desire/Aspiration | Reader wants a specific outcome. Benefit pull. | SL-HOW, SL-NUM, SL-PRF |
| ET-OUT | Outrage/Injustice | Something is wrong and the reader should care. | SL-WHY, SL-PRV, SL-CT |
| ET-IDE | Identity Challenge | Reader's self-concept is engaged or threatened. | SL-PRV, SL-VS, SL-CON |
| ET-SOC | Social Proof | Others have achieved or validated something. | SL-PRF, SL-NAM, SL-NUM |
| ET-HUM | Humor/Entertainment | Reader is entertained, amused, delighted. | SL-POP, SL-EDG, SL-SIM, SL-BPM |
| ET-AUT | Authority/Credibility | Reader respects the source or information. | SL-NAM, SL-THE, SL-HOW |
| ET-VUL | Vulnerability/Confession | Writer reveals something personal. Trust-building. | SL-CON, SL-WHY |
| ET-SCR | Scarcity/Urgency | Window is closing. Time pressure. | SL-URG, SL-CMD |

## EXECUTION PROTOCOL

```
EXECUTE 1.1-emotional-trigger-selector:
  INPUT: email_context (from Layer 0)
  OUTPUT: emotional_triggers object

  1. READ email body text completely
     DO NOT skim -- read every sentence

  2. SCAN for emotional trigger signals:
     FOR each paragraph in email body:
       a. What emotion does this paragraph evoke?
       b. Is the emotion explicit (stated) or implicit (induced)?
       c. How strong is the signal? (1-10)
       d. MAP to trigger code from taxonomy above

  3. ANALYZE opening hook specifically:
     The first 3 lines set the email's emotional tone.
     EXTRACT: What emotion does the opening create?
     This is typically the DOMINANT trigger.

  4. ANALYZE bridge/CTA section:
     The bridge shifts emotion toward action.
     EXTRACT: What emotion drives the CTA?
     This may be a SECONDARY trigger.

  5. RANK triggers by:
     a. Frequency (how many paragraphs evoke this trigger)
     b. Intensity (how strongly the trigger is activated)
     c. Position (triggers in opening/closing weighted higher)

  6. SELECT top 3-5 triggers
     These will drive formula matching in 1.2

  7. DOCUMENT trigger-to-content evidence
     FOR each selected trigger:
       Quote specific text from the email that activates this trigger
       This evidence is used for SL-body alignment checking later

  8. ASSEMBLE emotional_triggers object
  9. RETURN emotional_triggers
```

## TRIGGER-CONTENT ALIGNMENT PRINCIPLE

The SL's emotional trigger MUST match the email's dominant emotion. Mismatched triggers destroy trust:

| Mismatch | Problem |
|----------|---------|
| SL promises curiosity, email delivers lecture | Reader feels tricked |
| SL signals urgency, email has no deadline | Reader feels manipulated |
| SL implies humor, email is serious | Reader feels confused |
| SL promises confession, email is instructional | Reader feels misled |

**Rule:** The SL's emotional promise must be DELIVERED by the email body. This is checked via SL-body alignment in Layer 2 and Arena.

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | No triggers detected | HALT -- Email may lack emotional content. Review E1 output. |
| DEGRADED | Only 1 trigger detected | WARN -- Limited formula options. May need to generate from fewer categories. |
| DEGRADED | Opening emotion unclear | WARN -- Use body-level triggers as fallback. |
