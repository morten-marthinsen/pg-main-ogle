# 0.2 Teaching Loader

## Purpose
Load and internalize hook psychology and virality teachings that inform hook generation.

---

## Input Requirements
- Validated inputs from 0.1
- Access to teaching files

---

## Execution Protocol

### Step 1: Load Kane 3-Second Hook Framework
**File:** `teachings/virality/kane-3-second-hook.yaml`

**Extract:**
- 3-second attention threshold
- Visual + verbal hook synergy
- Pattern interrupt principles
- Hook-to-payoff ratio (promise what you deliver)

**Key insight:** The hook IS the content decision. If hook fails, nothing else matters.

---

### Step 2: Load Berger STEPPS Framework
**File:** `teachings/virality/berger-stepps-framework.yaml`

**Extract:**
- Social Currency (make people look good sharing)
- Triggers (top of mind → tip of tongue)
- Emotion (high-arousal emotions drive sharing)
- Public (make private behavior observable)
- Practical Value (useful information spreads)
- Stories (narrative containers)

**Application to hooks:** Hooks that trigger high-arousal emotions (awe, anger, anxiety) + practical value perform best.

---

### Step 3: Load Cialdini Influence Principles
**File:** `teachings/audience-psychology/cialdini-influence-principles.yaml`

**Extract:**
- Reciprocity (giving value in hook)
- Scarcity (limited time/access)
- Authority (credibility signals)
- Consistency (aligning with their identity)
- Liking (relatability)
- Consensus (social proof)

**Application to hooks:** Most effective hook categories leverage 2+ principles.

---

### Step 4: Load Heath "Made to Stick" Framework
**File:** `teachings/content-strategy/heath-made-to-stick.yaml`

**Extract:**
- Simple (core message, no clutter)
- Unexpected (break patterns)
- Concrete (specific, tangible)
- Credible (believable)
- Emotional (make people care)
- Stories (narrative format)

**SUCCESs framework application:** Best hooks are Unexpected + Concrete + Emotional.
