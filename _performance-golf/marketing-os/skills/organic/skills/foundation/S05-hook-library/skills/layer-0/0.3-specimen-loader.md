# 0.3 Specimen Loader

## Purpose
Load hook taxonomy and viral hook specimens to inform generation patterns.

---

## Input Requirements
- Teaching summary from 0.2
- Access to specimen library

---

## Execution Protocol

### Step 1: Load Hook Taxonomy
**File:** `specimens/hooks/hook-taxonomy.json`

**Extract complete 7-category structure:**

```yaml
hook_taxonomy:
  curiosity_hooks:
    description: "Create information gaps that demand closure"
    mechanisms:
      - "Open loop (incomplete story)"
      - "Unexpected juxtaposition"
      - "Impossible/unlikely statement"
      - "Hidden knowledge reveal"
    examples: []

  fear_hooks:
    description: "Activate loss aversion and threat detection"
    mechanisms:
      - "Warning/danger signal"
      - "Loss framing"
      - "Mistake exposure"
      - "Future threat"
    examples: []

  desire_hooks:
    description: "Promise specific valuable outcomes"
    mechanisms:
      - "Result promise"
      - "Secret/shortcut"
      - "Transformation preview"
      - "Status/identity upgrade"
    examples: []

  social_proof_hooks:
    description: "Leverage tribal belonging and validation"
    mechanisms:
      - "Numbers/statistics"
      - "Authority endorsement"
      - "Trend signaling"
      - "Community reference"
    examples: []

  identity_hooks:
    description: "Trigger self-recognition and belonging"
    mechanisms:
      - "'You' direct address"
      - "Tribe identification"
      - "Struggle recognition"
      - "Aspiration alignment"
    examples: []

  controversy_hooks:
    description: "Create tension through unexpected positions"
    mechanisms:
      - "Contrarian take"
      - "Myth busting"
      - "Hot take"
      - "Challenge to authority"
    examples: []

  story_hooks:
    description: "Open narrative loops with conflict/tension"
    mechanisms:
      - "'I was wrong about...'"
      - "Transformation beginning"
      - "Conflict introduction"
      - "Stakes establishment"
    examples: []
```

**This taxonomy is UNIVERSAL. All hooks fit into these 7 categories.**

---

### Step 2: Load Platform-Specific Viral Specimens
**Source:** Specimen library filtered by primary platform from PSF

**Extract for each high-performing hook:**
- Hook text (exact wording)
- Platform (where it performed)
- Format (video/carousel/thread/etc)
- Performance indicators (views, engagement)
- Category (which of 7 categories)
- What made it work (pattern analysis)

**Load at least 10 platform-specific specimens.**

---

### Step 3: Load Competitor Hooks (if provided)
If user provided competitor_hooks in validation:

**Extract:**
- Creator name
- Hook used
- Performance (if known)
- Category classification
- Learning (what to adapt, what to avoid)

---

### Step 4: Pattern Recognition Across Specimens
Identify patterns:

**High-performing hook patterns:**
- Most common categories on primary platform
- Most common mechanisms within categories
- Language patterns (specificity, numbers, emotion words)
- Structure patterns (question vs. statement, first vs. second person)

**Platform-specific insights:**
- Hook length patterns (character count or word count)
- Visual vs. verbal emphasis
- Trending hook formats
