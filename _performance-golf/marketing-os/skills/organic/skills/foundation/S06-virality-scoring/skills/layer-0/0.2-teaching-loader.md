# 0.2 Teaching Loader

## Purpose
Load the foundational teachings on virality science that define the scoring dimensions. All dimensions MUST be derived from established virality research, not invented.

---

## Input Requirements

Required teaching files:
- `teachings/virality/berger-stepps-framework.yaml`
- `teachings/virality/kane-viral-content-model.yaml`
- `teachings/content-strategy/heath-made-to-stick.yaml`
- `teachings/audience-psychology/cialdini-influence-principles.yaml`

---

## Execution Protocol

### Step 1: Load Berger STEPPS Framework

Extract the 6 STEPPS principles:
- **S**ocial Currency
- **T**riggers
- **E**motion
- **P**ractical Value
- **P**ublic
- **S**tories

Document how each principle translates to scorable dimensions.

---

### Step 2: Load Kane Viral Content Model

Extract Kane's viral content elements:
- Hook strength (first 3 seconds)
- Pattern interrupt
- Emotional resonance
- Value delivery
- Platform optimization

---

### Step 3: Load Heath "Made to Stick" Framework

Extract SUCCESs principles:
- **S**imple
- **U**nexpected
- **C**oncrete
- **C**redible
- **E**motional
- **S**tories

Map to content scoring dimensions (specificity, emotional resonance, story structure).

---

### Step 4: Load Cialdini Influence Principles

Extract relevant principles for virality:
- Social proof (shareability indicator)
- Authority (credibility dimension)
- Scarcity (urgency/exclusivity)

---

### Step 5: Synthesize Teaching-Backed Dimensions

Cross-reference all 4 teaching sources to identify the 8 core dimensions supported by research:

1. **Hook Strength** (Kane)
2. **Emotional Resonance** (Berger, Heath, Cialdini)
3. **Practical Value** (Berger, Heath)
4. **Social Currency** (Berger, Cialdini)
5. **Story Structure** (Berger, Heath)
6. **Platform Fit** (Kane)
7. **Specificity** (Heath)
8. **Voice Alignment** (brand integrity check)

Each dimension MUST cite which teaching(s) support it.

---
