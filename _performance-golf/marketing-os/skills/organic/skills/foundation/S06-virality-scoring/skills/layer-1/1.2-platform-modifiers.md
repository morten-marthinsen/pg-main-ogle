# 1.2 Platform Modifiers

## Purpose
Define platform-specific adjustments to dimension scores and bonus/penalty points based on the primary platform's algorithm priorities and cultural norms.

---

## Input Requirements

From Layer 0:
- `0.4-upstream-synthesis.yaml` (primary platform identified)

From Layer 1:
- `1.1-dimensions.yaml` (base dimensions)

---

## Execution Protocol

### Step 1: Identify Primary Platform

Extract from upstream synthesis:
- Primary platform name
- Algorithm priorities
- Cultural fit requirements
- Technical specs

---

### Step 2: Define Dimension Adjustments

For the primary platform, specify which dimensions get weighted higher/lower:

**Pattern:**
```yaml
dimension_adjustments:
  [dimension_name]: +X  # Increase importance
  [dimension_name]: -X  # Decrease importance
```

**Platform-Specific Guidance:**

**Instagram Reels:**
- hook_strength: +5 (hooks matter MORE)
- platform_fit: +3 (format critical)
- story_structure: -2 (less time for story)

**TikTok:**
- hook_strength: +5 (first second critical)
- emotional_resonance: +3 (emotion drives TikTok)
- platform_fit: +3

**YouTube Shorts:**
- practical_value: +3 (audience wants learning)
- hook_strength: +3

**YouTube Long-form:**
- story_structure: +5 (story critical for retention)
- practical_value: +5 (must deliver value)
- hook_strength: +3 (first 30 sec, not 1 sec)

**LinkedIn:**
- practical_value: +5 (value is currency)
- social_currency: +5 (professional image matters)
- specificity: +3

**X (Twitter):**
- hook_strength: +5 (first line is everything)
- social_currency: +3 (RT value)
- specificity: +3

---

### Step 3: Define Bonus Points

Platform-specific bonus opportunities (add to final score):

```yaml
bonus_points:
  [behavior]: +X points
  [behavior]: +X points
```

**Examples by platform:**
- Instagram: trending_audio: +5, duet_potential: +3, save_worthy: +5
- TikTok: trend_aligned: +5, loop_friendly: +5, sound_trend: +3
- YouTube Long: chapter_worthy: +3, evergreen_topic: +5, thumbnail_title_synergy: +5
- LinkedIn: thought_leadership: +5, document_format: +3
- X: thread_depth: +3, quote_tweet_worthy: +5

---

### Step 4: Define Penalty Points

Platform-specific violations (subtract from final score):

```yaml
penalties:
  [violation]: -X points
  [violation]: -X points
```

**Examples by platform:**
- Instagram: recycled_watermark: -10, over_60_seconds: -3
- TikTok: too_polished: -3, no_hook: -10, promotional_feel: -5
- YouTube: slow_opening: -5, filler_content: -10
- LinkedIn: salesy_tone: -5, external_link: -3
- X: link_in_main: -5

---

## Output

**File:** `layer-1/1.2-modifiers.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"
primary_platform: "[platform]"

dimension_adjustments:
  hook_strength: +5
  emotional_resonance: +3
  platform_fit: +3
  # [only dimensions with adjustments]

bonus_points:
  trending_audio: +5
  duet_potential: +3
  save_worthy: +5
  share_to_story_friendly: +3

penalties:
  recycled_watermark: -10
  over_60_seconds: -3
  no_text_overlay: -2

modifier_examples:
  - content: "Hook uses trending audio + clear text overlay"
    base_score: 72
    bonuses_applied: [trending_audio: +5]
    final_score: 77

  - content: "Reposted from TikTok with watermark visible"
    base_score: 68
    penalties_applied: [recycled_watermark: -10]
    final_score: 58
```

**Minimum size:** 1KB
