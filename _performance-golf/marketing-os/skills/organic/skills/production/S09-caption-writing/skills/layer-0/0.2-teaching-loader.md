# 0.2 Teaching Loader

**Layer:** 0 (Foundation)
**Model:** Claude Haiku
**Upstream:** 0.1 Input Validator
**Downstream:** 0.3 Specimen Loader

---

## PURPOSE

Load relevant teaching frameworks that inform caption strategy: Berger's STEPPS (virality), Cialdini's influence principles, and Miller's StoryBrand framework (clear messaging). These teachings guide hook construction, CTA design, and engagement optimization.

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  validation_result: "outputs/layer-0/0.1-validation-result.json"
  platform: "Instagram" | "TikTok" | "LinkedIn" | "X" | "YouTube"
  content_function: "Educate" | "Entertain" | "Inspire" | "Convert" | "Nurture"

teaching_files:
  - "../../teaching-frameworks/berger-stepps.md"        # Social Currency, Triggers, Emotion, Public, Practical Value, Stories
  - "../../teaching-frameworks/cialdini-influence.md"   # Reciprocity, Commitment, Social Proof, Authority, Liking, Scarcity
  - "../../teaching-frameworks/miller-storybrand.md"    # Clear one-liner, problem/solution, guide positioning
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Read all 3 teaching framework files
2. Extract principles relevant to caption writing:
   - **Berger STEPPS:** Social Currency (status signaling), Emotion (high-arousal emotions drive shares), Practical Value (actionable takeaways), Triggers (memory cues)
   - **Cialdini:** Social Proof (testimonials in captions), Scarcity (limited spots/time), Authority (credentials in bio-driven captions)
   - **Miller StoryBrand:** Clear one-liner hooks, problem → solution structure, guide positioning (not hero)
3. Map teaching principles to platform + content_function:
   - Educate → Practical Value (Berger), Authority (Cialdini)
   - Entertain → Emotion (Berger), Social Currency (Berger)
   - Inspire → Emotion (Berger), Stories (Berger)
   - Convert → Scarcity (Cialdini), Clear CTA (Miller)
   - Nurture → Liking (Cialdini), Stories (Berger)
4. Generate teaching reference summary with platform-specific applications
5. Output `0.2-teaching-references.md`

**NEVER:**
- Load generic "social media tips" instead of specific teaching frameworks
- Skip mapping teachings to platform + content_function
- Cite teachings without explaining application to captions
- Overload with all teachings (select 3-5 most relevant principles)

---

## OUTPUT

**File:** `outputs/layer-0/0.2-teaching-references.md`

**Format:**
```markdown
# Teaching References — S09 Caption Writing

**Platform:** Instagram
**Content Function:** Educate
**Selected Teachings:** 5 principles

---

## 1. Practical Value (Berger STEPPS)
**Principle:** Content that provides actionable value gets saved and shared.
**Caption Application:** Lead with specific takeaway ("3 ways to..."), use numbered lists, promise screenshottable insight.

## 2. Emotion — High Arousal (Berger STEPPS)
**Principle:** Awe, excitement, humor drive engagement more than sadness/contentment.
**Caption Application:** Use exclamation points, power words ("game-changer", "breakthrough"), emoji for emotional punctuation.

## 3. Social Currency (Berger STEPPS)
**Principle:** People share content that makes them look smart/helpful/in-the-know.
**Caption Application:** Position reader as insider ("Not many people know this..."), early adopter, solution provider.

## 4. Authority (Cialdini)
**Principle:** Expertise signals increase trust and action.
**Caption Application:** Lead with credential if unfamiliar ("As a [role] who [accomplishment]..."), cite research, use domain-specific terminology.

## 5. Clear One-Liner (Miller StoryBrand)
**Principle:** Clarity > cleverness. Reader should understand value in 5 seconds.
**Caption Application:** First sentence = problem solved or value delivered. No jargon before hook.

---

**Next Layer:** Apply these principles in hook strategy (1.3) and body writing (2.2).
```

**Minimum Size:** 800 chars (5 principles with applications)
