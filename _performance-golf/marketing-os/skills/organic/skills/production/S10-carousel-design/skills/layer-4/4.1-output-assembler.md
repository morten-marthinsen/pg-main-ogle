# 4.1 Output Assembler

## Purpose
Assemble final carousel outputs in standardized format with virality scoring and design brief. Validate all requirements met before handoff.

---

## Input Requirements

From Layer 2.5:
- `arena/[carousel-id]-arena-synthesis.md` (final selected carousel)
- `checkpoints/ARENA_COMPLETE.yaml` (Arena verification)

From Layer 2:
- `2.4-[carousel-id]-visual-direction.md` (per-slide visual specs)

From Layer 1:
- `1.4-save-optimization.yaml` (save strategy)

---

## Execution Protocol

### Step 1: Load Arena-Selected Carousels

For each carousel, load the human-selected final version:

```yaml
final_carousels:
  - carousel_id: "carousel_1"
    arena_file: "arena/carousel-1-arena-synthesis.md"
    selected_hybrid: "Hybrid A"
    slide_count: [N]
    platform: "[platform]"
```

**MUST:** Verify `selected_hybrid` matches ARENA_COMPLETE.yaml. If mismatch → HALT.

### Step 2: Apply Virality Scoring

Score each final carousel using VSF:

```yaml
virality_scoring:
  carousel_id: "carousel_1"

  dimensions:
    ea_emotional_activation:
      score: [1-10]
      rationale: "[Why this score]"

    sc_social_currency:
      score: [1-10]
      rationale: "[Why this score]"

    pi_pattern_interrupt:
      score: [1-10]
      rationale: "[Why this score]"

    pf_platform_fit:
      score: [1-10]
      rationale: "[Why this score]"

    sh_shareability:
      score: [1-10]
      rationale: "[Why this score]"

  total_score: [calculated]
  threshold: 60
  status: PASS | FAIL
```

**CRITICAL:** If total_score < 60 → carousel requires revision before output
**CRITICAL:** If total_score < 40 → carousel is BLOCKED from production

### Step 3: Compile Design Package

For each carousel, assemble complete design package:

```yaml
design_package:
  carousel_id: "carousel_1"
  platform: "[Instagram | LinkedIn]"
  template_type: "[Educational Listicle | How-To Guide | Myth-Busting | Framework | Before/After]"
  slide_count: [N]
  aspect_ratio: "[1:1 | 4:5 | variable]"

  slides:
    - slide_number: 1
      slide_type: "cover"
      headline: "[Final hook text]"
      subheadline: "[Supporting line if any]"
      body_copy: ""
      visual_direction:
        layout: "[layout type]"
        color_palette: ["#HEX1", "#HEX2", "#HEX3"]
        typography: "[style]"
        imagery_notes: "[guidance]"

    # ... all slides

  design_brief:
    overall_style: "[style]"
    brand_colors: ["#HEX1", "#HEX2"]
    typography_system: "[fonts]"
    visual_consistency: "[pattern]"
    accessibility: "[notes]"
```

### Step 4: Validate Completeness

```yaml
validation_checklist:
  carousel_id: "carousel_1"

  structural:
    - cover_slide_present: true
    - cta_slide_present: true
    - all_slides_have_visual_direction: true
    - character_counts_within_limits: true
    - slide_count_within_platform_range: true

  quality:
    - arena_complete: true
    - arena_selection_verified: true
    - virality_score_met: true
    - anti_slop_passed: true
    - slides_standalone: true

  design:
    - design_brief_complete: true
    - brand_colors_specified: true
    - typography_specified: true
    - layout_per_slide: true

  all_checks_passed: true
```

**If any check fails → INCOMPLETE → fix before output**

### Step 5: Generate Campaign Summary

```yaml
campaign_summary:
  campaign_name: "[Campaign]"
  total_carousels: [count]

  carousels_by_platform:
    instagram: [count]
    linkedin: [count]

  carousels_by_template:
    educational_listicle: [count]
    how_to_guide: [count]
    myth_busting: [count]
    framework: [count]
    before_after: [count]

  average_virality_score: [score]
  average_slide_count: [count]

  design_requirements:
    total_unique_slides: [count]
    graphics_needed: [count]
    brand_assets_required: [list]
```

---

## Output

**File:** `outputs/[campaign]-carousels-assembled.yaml`

**Minimum size:** 5KB+ (depending on carousel count)

---

## Quality Gates

- [ ] All carousels from Arena loaded and verified
- [ ] Virality scores >= 60 for all carousels
- [ ] Design packages complete per slide
- [ ] Validation checklist passed for all carousels
- [ ] Campaign summary generated
- [ ] `arena_selection_verified: true` confirmed

---

## Handoff

Pass complete carousel package to: **S14 Content Assembly** and **4.2 Platform Export**
