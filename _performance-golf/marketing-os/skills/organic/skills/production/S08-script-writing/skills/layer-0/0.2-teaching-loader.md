# 0.2 Teaching Loader

## Purpose
Load all required teaching files that inform script writing principles. These teachings establish theoretical frameworks for hooks, retention, virality, and audience psychology.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (must show PASS status)

---

## Execution Protocol

### Step 1: Identify Required Teachings

Based on ORGANIC-ENGINE-CLAUDE.md and S08 requirements:

```yaml
required_teachings:
  content_strategy:
    - teachings/content-strategy/hormozi-content-to-leads.yaml
    - teachings/content-strategy/miller-storybrand.yaml

  virality:
    - teachings/virality/kane-hook-point-system.yaml
    - teachings/virality/berger-stepps-framework.yaml
    - teachings/virality/heath-made-to-stick.yaml

  psychology:
    - teachings/audience-psychology/eyal-hook-model.yaml
    - teachings/audience-psychology/cialdini-influence-principles.yaml

  retention:
    - teachings/retention/video-retention-principles.yaml # if exists
```

**CRITICAL:** Minimum 4 teachings must be loaded. Cannot proceed with fewer.

---

### Step 2: Load Each Teaching File

For each teaching:

```yaml
teaching_load:
  - path: "[full path to teaching file]"
    exists: true/false
    loaded: true/false
    key_frameworks_extracted: []
    application_to_scripts: |
      [How this teaching applies to S08]
```

**If teaching file doesn't exist:** Document as missing, continue with others. If < 4 total loaded → FAIL

---

### Step 3: Extract Key Frameworks

For each loaded teaching, extract the core frameworks relevant to script writing:

**Example:**
```yaml
kane_hook_point:
  framework: "Hook Point System"
  key_principles:
    - "First 3 seconds determine retention"
    - "Pattern interrupt + curiosity gap"
    - "Hook must be platform-native"
  application: |
    Use for 1.3-hook-strategy and 2.1-hook-drafting.
    Every script must have hook scored against Kane principles.

berger_stepps:
  framework: "STEPPS (Social Currency, Triggers, Emotion, Public, Practical Value, Stories)"
  key_principles:
    - "Social currency: make people look good for sharing"
    - "Practical value: useful information is shared"
    - "Stories: wrap message in narrative"
  application: |
    Use for virality scoring in Layer 4.
    SC and PV dimensions directly map to STEPPS.
```

---

### Step 4: Create Teaching Reference Guide

Synthesize teachings into quick-reference format for downstream microskills:

```yaml
teaching_synthesis:
  hook_principles:
    - "[Principle from Kane]"
    - "[Principle from Eyal]"

  retention_principles:
    - "[Principle about pattern interrupts]"
    - "[Principle about pacing]"

  virality_principles:
    - "[STEPPS elements]"
    - "[Made to Stick principles]"

  storytelling_principles:
    - "[Storybrand framework]"
    - "[Narrative structure]"
```
