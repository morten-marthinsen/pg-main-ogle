# 1.3 Hook Strategy

## Purpose
Define hook strategy for each script by selecting hook types from HLF (Hook Library) and referencing specimen hook patterns.

---

## Input Requirements

From Layer 0:
- `0.3-specimens-loaded.yaml` (hook patterns)
- `0.4-cbf-context.yaml` (HLF reference)

From Layer 1:
- `1.1-platform-analysis.yaml` (hook windows per platform)
- `1.2-format-selection.yaml` (script formats)

---

## Execution Protocol

### Step 1: Load Hook Library

If HLF exists, load hook categories:

```yaml
hlf_hooks:
  curiosity_hooks:
    - "Did you know [surprising fact]..."
    - "The [adjective] truth about [topic]..."

  authority_pain_hooks:
    - "Stop [common mistake]..."
    - "If you're [pain state], this is why..."

  transform_identity_hooks:
    - "How I went from [before] to [after]..."
    - "The [method] that changed everything..."

  # ... other categories from HLF
```

---

### Step 2: Analyze Specimen Hook Patterns

Extract hook patterns from loaded specimens:

```yaml
specimen_hook_analysis:
  tiktok_patterns:
    - pattern: "Question hook"
      examples:
        - "[Exact hook from specimen 1]"
        - "[Exact hook from specimen 2]"
      timing: "0-1s"
      effectiveness: "High scroll-stop"

    - pattern: "Contrarian statement"
      examples:
        - "[Example]"
      timing: "0-2s"
      effectiveness: "High engagement"

  instagram_patterns:
    - pattern: "Visual + text overlay hook"
      examples: []
      timing: "0-3s"
      effectiveness: "High save rate"
```

---

### Step 3: Select Hook Type Per Script

For each script, select hook type and draft hook strategy:

```yaml
hook_strategy_per_script:
  - script_id: "script_1"
    platform: "TikTok"
    content_pillar: "[Pillar from request]"
    content_function: "Awareness"

    hook_selection:
      primary_type: "Curiosity Hook"
      hlf_reference: "[Hook category]"
      specimen_pattern: "Question hook"
      reasoning: |
        Curiosity hooks perform best for awareness on TikTok.
        Specimen TT_001 shows 85% completion with question hook.

    hook_requirements:
      timing: "0-1 second"
      visual_component: "Pattern interrupt visual"
      text_overlay: "Bold question text"
      audio: "Trending sound or original"

    backup_hook_types:
      - "Contrarian statement"
      - "Transformation hook"
```

---

### Step 4: Define Hook Success Criteria

Establish what makes a successful hook for each script:

```yaml
hook_success_criteria:
  - script_id: "script_1"
    criteria:
      - "Stops scroll in first 0.5 seconds"
      - "Creates curiosity gap"
      - "Matches specimen pattern effectiveness"
      - "Platform-native (not generic)"
      - "Aligns with content pillar"

  scoring_dimensions:
    pattern_interrupt: /10
    curiosity_gap: /10
    platform_fit: /10
    specimen_alignment: /10
```
