# 0.3 Specimen Loader

## Purpose
Load specimen scripts that demonstrate excellent execution. Specimens provide concrete examples of what "great" looks like for each platform and format.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (platform list)
- `0.2-teachings-loaded.yaml` (context)

---

## Execution Protocol

### Step 1: Identify Specimen Requirements

Based on script requests from 0.1, determine which specimens to load:

```yaml
specimen_requirements:
  by_platform:
    tiktok:
      min_specimens: 3
      formats: ["viral TikToks", "educational TikToks"]

    instagram:
      min_specimens: 3
      formats: ["Reels", "video posts"]

    youtube:
      min_specimens: 2
      formats: ["Shorts", "long-form intros"]

  by_content_function:
    awareness: 2
    engagement: 2
    conversion: 1
```

**CRITICAL:** Minimum 5 total specimens required (Law 3: Specimens first, then generate)

---

### Step 2: Load Platform-Specific Specimens

For each target platform, load relevant specimens:

```yaml
specimen_inventory:
  - specimen_id: "SPEC_TT_001"
    platform: "TikTok"
    title: "[Title/description]"
    path: "[path to specimen file]"
    format: "Viral hook pattern"
    performance:
      views: [count]
      engagement_rate: [%]
    key_elements:
      hook: "[Hook pattern used]"
      structure: "[Script structure]"
      cta: "[CTA type]"
    learnings: |
      [What makes this specimen effective]
```

---

### Step 3: Analyze Specimen Patterns

Extract patterns from loaded specimens:

```yaml
pattern_analysis:
  hook_patterns:
    tiktok:
      - pattern: "Question hook"
        example: "[Exact hook from specimen]"
        timing: "0-1 second"

      - pattern: "Contrarian statement"
        example: "[Example]"
        timing: "0-2 seconds"

    instagram:
      - pattern: "Visual hook + text overlay"
        example: "[Example]"
        timing: "0-3 seconds"

  body_patterns:
    value_density: "[Insights per second]"
    pattern_interrupts: "[Frequency]"
    pacing: "[Fast/Medium/Slow examples]"

  cta_patterns:
    soft_ctas: []
    medium_ctas: []
    hard_ctas: []
```

---

### Step 4: Create Specimen Reference Matrix

Build quick-reference for Layer 2 generation:

```yaml
specimen_reference:
  by_platform:
    tiktok:
      best_hooks: []
      typical_length: "15-30 seconds"
      retention_tactics: []
      top_specimens: ["SPEC_TT_001", "SPEC_TT_003"]

    instagram:
      best_hooks: []
      typical_length: "20-45 seconds"
      retention_tactics: []
      top_specimens: []

  by_function:
    awareness:
      specimens: []
      common_structures: []

    engagement:
      specimens: []
      common_structures: []
```

---

### Step 5: Document Specimen-Teaching Connections

Link specimen patterns to teaching frameworks:

```yaml
specimen_teaching_alignment:
  - specimen: "SPEC_TT_001"
    teaching: "Kane Hook Point"
    alignment: |
      This specimen demonstrates Kane's "pattern interrupt" principle.
      Hook stops scroll in 0.5 seconds with unexpected visual.

  - specimen: "SPEC_IG_002"
    teaching: "Berger STEPPS - Social Currency"
    alignment: |
      This Reel makes viewers look smart/cool for sharing.
      Educational value wrapped in entertaining format.
```
