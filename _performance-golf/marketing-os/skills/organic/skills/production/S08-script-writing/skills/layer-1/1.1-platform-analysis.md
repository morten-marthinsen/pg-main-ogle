# 1.1 Platform Analysis

## Purpose
Analyze each target platform's script requirements, algorithm preferences, and format specifications. Platform-native scripts require platform-specific structures.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (script requests with platforms)
- `0.4-cbf-context.yaml` (platform strategy)

---

## Execution Protocol

### Step 1: Identify Unique Platforms

Extract unique platforms from script requests:

```yaml
platforms_to_analyze:
  - TikTok
  - Instagram
  - YouTube
  # ... others as specified
```

---

### Step 2: Define Platform-Specific Requirements

For EACH platform, document specifications:

```yaml
platform_specs:
  tiktok:
    algorithm_signals:
      - "Completion rate (watch time %)"
      - "Re-watches (loop count)"
      - "Engagement in first hour"
      - "Shares > saves > comments > likes"

    format_requirements:
      aspect_ratio: "9:16 (vertical)"
      max_length: "10 minutes (practical: 15-60s for virality)"
      hook_window: "0.5-1 second"

    content_preferences:
      tone: "Casual, authentic, trend-aware"
      pacing: "Fast cuts, high energy"
      text_overlays: "Large, bold, reinforcing (not repeating) audio"
      sound: "Trending sounds boost discovery"

    script_structure:
      hook: "Immediate pattern interrupt, no intro"
      body: "Dense value, no filler, pattern interrupt every 3-5s"
      cta: "Brief, integrated into content flow"

    retention_tactics:
      - "Visual hooks (first frame = thumbnail)"
      - "Fast pacing (shot changes every 1-3s)"
      - "Curiosity gaps throughout"
      - "Loop potential (end ties to beginning)"

  instagram:
    algorithm_signals:
      - "Save rate (highest weight)"
      - "Shares to stories/DMs"
      - "Completion rate"
      - "Profile visits from Reel"

    format_requirements:
      aspect_ratio: "9:16 (vertical) or 4:5 (feed)"
      max_length: "90 seconds (Reels)"
      hook_window: "1-3 seconds"

    content_preferences:
      tone: "Polished but authentic"
      pacing: "Slightly slower than TikTok"
      text_overlays: "Clean, on-brand typography"
      caption_integration: "Caption complements (not repeats) video"

    script_structure:
      hook: "Strong visual + verbal hook (thumbnail matters)"
      body: "Educational or entertaining value"
      cta: "Save/share triggers embedded"

    retention_tactics:
      - "Thumbnail optimization (hook frame)"
      - "Value density (screenshot-worthy)"
      - "Strategic pacing (keeps attention)"
      - "Caption reinforcement"

  youtube_shorts:
    algorithm_signals:
      - "Watch time + completion rate"
      - "CTR to main channel"
      - "Engagement rate"

    format_requirements:
      aspect_ratio: "9:16 (vertical)"
      max_length: "60 seconds"
      hook_window: "1-3 seconds"

    content_preferences:
      tone: "Title + thumbnail synergy"
      pacing: "Clear value proposition"
      cross_promotion: "Reference main channel content"

    script_structure:
      hook: "Title/thumbnail work together"
      body: "Self-contained value"
      cta: "Subscribe or watch more"

  youtube_longform:
    algorithm_signals:
      - "Watch time (absolute + %)"
      - "CTR (title + thumbnail)"
      - "Engagement (likes, comments, shares)"
      - "Session time (keeps viewers on YouTube)"

    format_requirements:
      aspect_ratio: "16:9 (horizontal)"
      min_length: "8+ minutes (for mid-roll ads)"
      hook_window: "0-30 seconds (cold open)"

    content_preferences:
      tone: "Professional, authoritative"
      pacing: "Varies by niche (fast for most)"
      chapters: "Timestamp-friendly structure"

    script_structure:
      cold_open: "0-30s: Most compelling moment"
      intro: "30-90s: Value promise + authority"
      body: "3-7 sections with re-engagement hooks"
      close: "30-60s: Summary + CTA"

    retention_tactics:
      - "Cold open (best moment first)"
      - "Pattern interrupts every 30-60s"
      - "Open loops (teasing upcoming content)"
      - "B-roll variety"
```

---

### Step 3: Cross-Reference with CBF Platform Strategy

Validate platform specs against CBF requirements:

```yaml
cbf_alignment:
  - platform: "TikTok"
    cbf_posting_freq: "[from CBF]"
    cbf_format_priorities: []
    cbf_length_preferences: "[from CBF]"
    spec_alignment: "Aligned/Misaligned"
    notes: "[Any conflicts or adjustments needed]"
```

---

### Step 4: Create Platform Requirement Matrix

Build decision matrix for script planning:

```yaml
platform_matrix:
  hook_requirements:
    tiktok: "0.5-1s, immediate interrupt"
    instagram: "1-3s, visual + verbal"
    youtube_shorts: "1-3s, title synergy"
    youtube_longform: "Cold open 0-30s"

  optimal_lengths:
    tiktok: "15-30s (viral sweet spot)"
    instagram: "20-45s (Reels)"
    youtube_shorts: "30-60s"
    youtube_longform: "8-15 min"

  cta_styles:
    tiktok: "Brief, in-video"
    instagram: "Save/share triggers"
    youtube_shorts: "Subscribe prompt"
    youtube_longform: "Multiple CTAs throughout"

  production_notes:
    tiktok: "Raw/authentic > polished"
    instagram: "Polished but authentic"
    youtube_shorts: "Clear value, quick"
    youtube_longform: "Chapter structure, B-roll"
```
