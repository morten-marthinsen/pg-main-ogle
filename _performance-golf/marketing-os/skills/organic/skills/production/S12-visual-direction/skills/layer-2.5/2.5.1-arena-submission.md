# Microskill 2.5.1: Arena Submission
Skill: S12 Visual Direction
Layer: 2.5
Model: claude-opus-4.6
Version: 1.0

## Purpose

Prepare thumbnail concepts for 7-persona Arena evaluation. Package the primary concept (from 2.1) plus alternatives (from 1.2 strategy) for comparative evaluation.

## Input

**Required gate:** `GATE_2_BRIEFS_COMPLETE`

**Required files:**
- `s12-2.1-thumbnail-brief.yaml` — primary concept (Concept A)
- `s12-1.2-thumbnail-strategy.md` — alternative concepts (B, C)
- `s12-0.3-specimens.yaml` — performance benchmarks

## Process

### MUST
1. **Package Concept A (primary)**
   - Full brief from 2.1
   - Rationale for selection
   - Expected performance (scroll-stop mechanism, platform fit)

2. **Develop Concept B brief**
   - From alternative in 1.2 strategy
   - Same level of detail as Concept A
   - Different scroll-stop approach

3. **Develop Concept C brief**
   - From second alternative in 1.2 strategy
   - Same level of detail
   - Third unique approach

4. **Frame for Arena evaluation**
   - Evaluation criteria: scroll-stop effectiveness, brand alignment, platform performance, differentiation
   - Performance context: platform benchmarks from specimens
   - Human decision goal: select strongest concept or identify hybrid opportunity

### NEVER
- Submit only 1 concept (Arena needs real alternatives)
- Vary detail level between concepts (unfair comparison)
- Skip performance framing (Arena needs evaluation context)

## Output

**File:** `s12-2.5.1-arena-package.yaml`

```yaml
arena_submission:
  skill: S12
  content_id: "retirement-40-3bucket-video"
  submission_type: "Thumbnail concepts"

  concepts:
    - concept_id: "A"
      name: "Authority + Mechanism"
      brief: "[Full brief from 2.1]"
      rationale: "Strongest content promise alignment, proven pattern"

    - concept_id: "B"
      name: "Result-Focused"
      brief: "[Developed brief]"
      rationale: "Emotional aspiration, outcome clarity"

    - concept_id: "C"
      name: "Data Visualization"
      brief: "[Developed brief]"
      rationale: "Pattern interrupt, differentiation"

  evaluation_criteria:
    - scroll_stop_effectiveness: "Will it stop scrolling? (pattern interrupt, contrast, emotional trigger)"
    - brand_alignment: "Does it match brand guidelines and voice?"

    platform_average_ctr: "4.8% (YouTube finance vertical)"
    high_performer_threshold: "8%+ CTR"
    specimen_patterns: "Face+text: 67% of high-performers, High contrast: 82%"
```
