# Microskill 2.5.3: Synthesis
Skill: S12 Visual Direction
Layer: 2.5
Model: claude-opus-4.6
Version: 1.0

## Purpose

Generate 3+ hybrid thumbnail concepts combining strongest elements from Arena evaluation, then capture human selection of final concept.

## Input

**Required file:** `s12-2.5.2-critique-report.yaml`

## Process

### MUST
1. **Analyze Arena results**
   - Identify strengths of each original concept
   - Identify weaknesses flagged by personas/Critic
   - Look for complementary strengths (A's strength + B's strength)

2. **Generate 3+ hybrid concepts**
   - Hybrid Alpha: [Concept combination description]
   - Hybrid Beta: [Different combination]
   - Hybrid Gamma: [Third alternative]
   - Each hybrid: full brief with rationale for combination

3. **Present to human**
   - Original Concepts A/B/C
   - Hybrid Concepts Alpha/Beta/Gamma
   - Arena insights summary
   - Recommendation (with uncertainty flagged)

4. **Capture human selection**
   - Selected concept ID
   - Any modifications requested
   - Rationale for selection

### NEVER
- Generate <3 hybrids (human needs real options)
- Skip human selection (Arena informs, human decides)
- Proceed without captured selection

**File:** `s12-2.5.3-synthesis-results.yaml`
