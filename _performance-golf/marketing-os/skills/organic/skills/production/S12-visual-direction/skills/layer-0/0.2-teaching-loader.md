# Microskill 0.2: Teaching Loader
Skill: S12 Visual Direction
Layer: 0
Model: claude-haiku-4
Version: 1.0

## Purpose

Load visual design principles, platform-specific visual conventions, and thumbnail best practices from teaching files. This provides the knowledge foundation for visual direction strategy.

## Input

**Required gate:** `GATE_0_INPUT_VALID`

Teaching files to load:
- `skills/organic/teaching/visual-design-principles.md` — core design principles (composition, color theory, typography)
- `skills/organic/teaching/thumbnail-strategy.md` — scroll-stopping thumbnail techniques
- `skills/organic/teaching/platform-visual-conventions.md` — platform-specific visual requirements
- `skills/organic/teaching/text-overlay-best-practices.md` — readable text on visuals

## Process

### MUST
1. **Load core visual design principles**
   - Read `visual-design-principles.md`
   - Extract: composition rules, color theory basics, visual hierarchy principles, typography fundamentals
   - Document key principles for Layer 1 consumption

2. **Load thumbnail strategy teaching**
   - Read `thumbnail-strategy.md`
   - Extract: scroll-stop mechanisms, pattern interrupts, contrast techniques, emotional triggers in visuals
   - Document thumbnail frameworks

3. **Load platform-specific conventions**
   - Read `platform-visual-conventions.md`
   - Extract per-platform: dimensions, aspect ratios, text-safe zones, UI element positions, visual trends
   - Create quick-reference table for Layer 1

4. **Load text overlay best practices**
   - Read `text-overlay-best-practices.md`
   - Extract: font size minimums, contrast ratios, readability rules, platform-specific text limits
   - Document text overlay constraints

5. **Create consolidated teaching context**
   - Synthesize all teaching into single reference document
   - Organize by topic for easy Layer 1 lookup
   - Include page references for deep dives

### NEVER
- Skip teaching files ("I know design principles")
- Rely on pre-training knowledge over teaching files
- Load only subset of teaching files
- Modify or editorialize teaching content
- Proceed if teaching files are missing (escalate instead)
