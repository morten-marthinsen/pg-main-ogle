# 0.3 Specimen Loader

## Purpose
Load high-performing thread specimens to calibrate quality standards and learn structural patterns.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (platform confirmed)
- `0.2-teaching-reference.yaml` (frameworks loaded)

---

## Execution Protocol

### Step 1: Identify Relevant Specimens

Based on platform from validation:

```yaml
x_twitter_threads:
  directory: "specimens/threads/x-twitter/"
  categories:
    - listicle-threads/ # "N lessons from X"
    - story-threads/ # Narrative with lessons
    - breakdown-threads/ # Analysis of examples
    - framework-threads/ # Teaching methodology

  minimum_performance:
    views: 100000
    engagement_rate: 5%
    tier: "viral" or "strong"

linkedin_threads:
  directory: "specimens/threads/linkedin/"
  categories:
    - long-form-posts/ # Single 1500+ word posts
    - multi-post-series/ # Connected posts
    - professional-insights/ # Industry analysis

  minimum_performance:
    views: 10000
    engagement_rate: 3%
    comments: 50+
```

### Step 2: Load Specimens (Minimum 5, Target 10+)

For each specimen:
```yaml
specimen_id: "THREAD_001"
title: "[Thread title/topic]"
platform: "[X_TWITTER or LINKEDIN]"
thread_type: "[Type]"
tweet_count: [N]

performance:
  views: [N]
  likes: [N]
  comments: [N]
  shares: [N]
  engagement_rate: [%]
  tier: "[viral/strong/solid]"

structure_analysis:
  hook_pattern: "[How it hooks]"
  body_structure: "[How content flows]"
  cta_type: "[How it closes]"

  standout_elements:
    - "[What makes this work]"
    - "[What makes this work]"

content_sample:
  tweet_1: |
    [Exact text]

  best_body_tweet: |
    [Exact text]

  cta_tweet: |
    [Exact text]
```

### Step 3: Pattern Extraction

Analyze loaded specimens for patterns:

```yaml
common_patterns:
  hook_structures:
    - pattern: "[N] [things] I learned from [experience]:"
      specimens_using: ["THREAD_001", "THREAD_003"]
      effectiveness: "High for listicle threads"

    - pattern: "In [year], I was [bad state]. Here's what changed:"
      specimens_using: ["THREAD_002"]
      effectiveness: "High for story threads"

  body_techniques:
    - technique: "One insight per tweet, 2-3 sentences"
      specimens_using: ["THREAD_001", "THREAD_004"]

    - technique: "Numbered points with brief explanation"
      specimens_using: ["THREAD_001", "THREAD_003"]

  cta_patterns:
    - pattern: "Found this helpful? RT the first tweet."
      specimens_using: ["THREAD_001", "THREAD_002", "THREAD_004"]
      conversion_type: "Engagement"

    - pattern: "Follow @handle for more threads like this"
      specimens_using: ["THREAD_003"]
      conversion_type: "Audience growth"
```

## Output

**File:** `layer-0/0.3-specimen-library.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"
platform: "[X_TWITTER or LINKEDIN]"
specimens_loaded: [count]

specimens:
  - specimen_id: "THREAD_001"
    title: "[Title]"
    performance: {...}
    structure_analysis: {...}
    content_sample: {...}

  # ... repeat for all specimens

pattern_library:
  hook_structures: []
  body_techniques: []
  cta_patterns: []

quality_calibration:
  viral_tier_characteristics:
    - "[Common trait]"
    - "[Common trait]"

  strong_tier_characteristics:
    - "[Common trait]"

  avoid_patterns:
    - pattern: "[What doesn't work]"
      reason: "[Why it fails]"
```

**Minimum size:** 3KB (more with >5 specimens)

- [ ] Pattern extraction complete
- [ ] Quality calibration documented
