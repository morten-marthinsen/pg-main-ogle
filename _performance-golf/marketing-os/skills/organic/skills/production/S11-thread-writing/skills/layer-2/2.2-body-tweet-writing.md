# 2.2 Body Tweet Writing

## Purpose
Write tweets 2 through N-1 — the value delivery tweets that build on the hook and lead to the CTA.

---

## Input Requirements

From Layer 0:
- `0.4-cbf-context.yaml` (voice, key points)
- Thread brief (key_points array)

From Layer 1:
- `1.2-thread-type-selection.yaml` (body structure)
- `1.4-thread-count-planning.yaml` (tweet allocation)

From Layer 2:
- `2.1-hook-tweet-draft.yaml` (hook established)

---

## Execution Protocol

### Step 1: Map Content to Tweets

Allocate key points to specific tweets:

```yaml
content_mapping:
  total_body_tweets: [N minus hook minus CTA]

  tweet_2:
    purpose: "[Context OR Point 1]"
    key_point: "[From brief]"
    approach: "[How to deliver this]"

  tweet_3:
    purpose: "[Point 1 or 2]"
    key_point: "[From brief]"
    approach: "[How to deliver this]"

  # ... continue for all body tweets

  tweet_N_minus_1:
    purpose: "[Final point or transition]"
    key_point: "[From brief]"
    approach: "[How to deliver this]"
```

### Step 2: Write Each Body Tweet

For each tweet, apply structure:

**Tweet Structure Options:**

```yaml
tip_tweet:
  format: "[Number]. [Tip headline]\n\n[Brief explanation or example]"
  character_budget: "50 chars headline, rest for body"

story_beat_tweet:
  format: "[Narrative moment]\n\n[What it meant or what happened]"
  character_budget: "Full 280 for story flow"

insight_tweet:
  format: "[Core insight]\n\n[Evidence or example]\n\n[Implication]"
  character_budget: "80/100/100"

comparison_tweet:
  format: "[Old way/wrong way]\n\nvs.\n\n[New way/right way]"
  character_budget: "130/130 split"
```

**Per Tweet:**
```yaml
tweet_N:
  number: [N]
  type: "[tip/story_beat/insight/comparison]"

  content: |
    [Tweet text with line breaks]

  character_count: [N/280 or N/3000]

  standalone_value:
    test: "[What value does this tweet provide alone?]"
    passes: [Yes/No]

  no_filler_test:
    test: "[If removed, would thread be incomplete?]"
    passes: [Yes/No]

  structure_variety:
    previous_tweet_structure: "[Type]"
    this_tweet_structure: "[Type]"
    different: [Yes/No]
```

### Step 3: Pacing and Transitions

Ensure momentum:

```yaml
pacing_analysis:
  intensity_flow:
    - tweet: 2
      intensity: "[High/Medium/Low]"
      reasoning: "[Why this intensity here]"

    - tweet: 3
      intensity: "[High/Medium/Low]"
      reasoning: "[Why this intensity here]"

    # ... for all tweets

  transition_tweets:
    - between_tweets: "[N and N+1]"
      transition_phrase: "[Optional transition like 'But here's the thing...']"
      necessary: [Yes/No]

  engagement_boosters:
    - tweet: [N]
      booster_type: "[Question/Controversial statement/Personal story]"
      reasoning: "[Why here]"
```

### Step 4: Voice Consistency

Apply BVF across all tweets:

```yaml
voice_check:
  tone_consistency:
    target_tone: "[From BVF]"
    tweets_matching: [N/total]
    adjustments_needed: ["[Tweet N needs...]"]

  power_words_used:
    - word: "[Power word]"
      tweet: [N]

  banned_words_avoided:
    violations: [None OR list]
    remediation: "[If any violations]"

  anti_voice_scan:
    patterns_detected: [None OR list]
    remediation: "[If any patterns]"
```

## Output

**File:** `layer-2/2.2-body-tweets-draft.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"
total_body_tweets: [N]

body_tweets:
  - tweet_number: 2
    type: "[Structure type]"
    content: |
      [Tweet text]
    character_count: [N]
    standalone_value: true
    no_filler: true

  - tweet_number: 3
    type: "[Structure type]"
    content: |
      [Tweet text]
    character_count: [N]
    standalone_value: true
    no_filler: true

  # ... continue for all body tweets

pacing_analysis:
  intensity_flow: []
  transition_tweets: []
  engagement_boosters: []

voice_check:
  tone_consistency: {...}
  power_words_used: []
  banned_words_avoided: {...}
  anti_voice_scan: {...}

quality_confirmation:
  all_tweets_standalone_valuable: true
  no_filler_tweets: true
  structure_variety_maintained: true

```

**Minimum size:** 2KB (scales with tweet count)
