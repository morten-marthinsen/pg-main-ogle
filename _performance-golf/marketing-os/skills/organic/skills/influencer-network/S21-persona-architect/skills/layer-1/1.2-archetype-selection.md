# 1.2 Archetype Selection

## Purpose
Assign one of 7 archetypes to each persona: Educator, Curator, Storyteller, Provocateur, Entertainer, Connector, Analyst. Justify selection based on niche requirements and audience needs.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Niche ownership map (from 1.1)
- Teaching summary (archetype definitions)
- Upstream context (audience preferences, platform priorities)

## Execution Protocol

### MUST
- Assign ONE primary archetype per persona
- Justify archetype selection based on: niche content needs, audience preferences, platform fit, competitive differentiation
- Verify archetype diversity across network (avoid too many of same type)
- Document how archetype informs content approach
- Consider platform-archetype fit (e.g., Entertainer thrives on TikTok, Analyst on LinkedIn)
- Allow secondary archetype traits where appropriate (e.g., "Educator with Storyteller elements")

### NEVER
- Assign multiple equal-weight archetypes (must have one PRIMARY)
- Choose archetype based on creator preference rather than audience needs
- Create network with all similar archetypes
- Ignore platform-archetype compatibility
- Use archetype labels without understanding their content implications

## Output
**Filename:** `archetype-assignments.md`

**Format:** Markdown archetype selection document

**Minimum Size:** 3KB

**Required Sections:**
1. Archetype Assignments (persona, niche, primary archetype, secondary traits if any)
2. Selection Rationale (why this archetype fits this niche and audience)
3. Network Diversity Check (distribution of archetypes across personas)
4. Platform Fit Analysis (how archetype aligns with platform priorities)
5. Content Approach Implications (how archetype shapes content style)

## Quality Gates
- [ ] One primary archetype per persona
- [ ] Archetype selection justified with specific audience/niche reasons
- [ ] Network shows archetype diversity (not all same type)
- [ ] Platform compatibility assessed for each archetype
- [ ] Content approach implications documented (not just labels)

## Handoff
Feeds into: 1.3 Identity Concepts (archetype shapes personality development)
