# 0.3 Upstream Loader

## Purpose
Load upstream dependencies from earlier organic marketing skills: AIF (audience intelligence), BVF (brand voice framework), PSF (platform selection framework). These provide constraints and context for persona development.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- `audience-intelligence-framework.yaml` (from S01)
- `brand-voice-framework.yaml` (from S02)
- `platform-selection-framework.yaml` (from S03)
- Upstream completion status

## Execution Protocol

### MUST
- Verify all 3 upstream files exist and are complete
- Extract audience segments from AIF (demographics, psychographics, pain points, desires)
- Extract brand voice constraints from BVF (tone boundaries, forbidden language, required voice traits)
- Extract platform selection priorities from PSF (which platforms, why, audience presence data)
- Create persona development constraints from upstream data (what personas MUST respect)
- Map audience segments to potential persona niches (who will each persona serve?)

### NEVER
- Create personas that contradict brand voice constraints from BVF
- Develop for platforms not prioritized in PSF without justification
- Ignore audience segment data when assigning persona niches
- Proceed if any upstream file is missing or incomplete

## Output
**Filename:** `upstream-context-summary.md`

**Format:** Markdown context summary

**Minimum Size:** 4KB

**Required Sections:**
1. Audience Segments (from AIF) — list all segments with key traits
2. Brand Voice Constraints (from BVF) — boundaries personas must respect
3. Platform Priorities (from PSF) — platforms to focus on, audience data per platform
4. Persona Development Constraints (synthesis of all 3 frameworks)
5. Segment-to-Niche Mapping (preliminary mapping of audience segments to persona opportunities)

## Quality Gates
- [ ] All 3 upstream files loaded successfully
- [ ] Audience segments extracted with sufficient detail (not just labels)
- [ ] Brand voice constraints are specific and actionable
- [ ] Platform priorities include audience presence justification
- [ ] Persona constraints are synthesis, not just copy-paste

## Handoff
Feeds into: All Layer 1 microskills (context for all persona development decisions)
