# 4.1 Account Plan Assembler

## Purpose
Assemble complete account strategy per persona × platform combination. Integrate all Layer 1 outputs into comprehensive, actionable account plans. Minimum 15KB per plan.

## Model Assignment
**Model:** claude-3-5-sonnet-20241022

## Input Requirements
- All Layer 1 outputs (1.1 through 1.6)
- Platform selections, profile optimization, content calendar, growth strategy, cross-promotion, audience targeting
- Persona Bibles (for voice and identity context)

## Execution Protocol

### MUST
- Create one Account Plan file per persona × platform combination
- Include 8 required sections per plan: (1) Overview, (2) Profile Optimization, (3) Content Calendar, (4) Growth Strategy, (5) Cross-Promotion, (6) Audience Targeting, (7) Metrics & KPIs, (8) 90-Day Roadmap
- Integrate all Layer 1 content (not just copy-paste, synthesize into cohesive action plan)
- Ensure each plan is minimum 15KB (comprehensive and actionable)
- Create internal consistency (all sections align with persona voice and platform constraints)
- Include quick-start checklist for immediate implementation
- Add troubleshooting section for common obstacles

### NEVER
- Create plans under 15KB (indicates insufficient tactical detail)
- Copy-paste Layer 1 files without synthesis
- Create structural inconsistencies across plans
- Skip 90-day roadmap (critical for execution)
- Omit metrics and KPIs section
- Forget platform-specific adaptations

## Output
**Filename:** `account-plan-[persona-name]-[platform].md` (one file per persona × platform)

**Format:** Markdown comprehensive account plan

**Minimum Size:** 15KB per file

**Required Sections (8):**
1. Overview (persona, platform, strategic rationale)
2. Profile Optimization (bio, links, photos, pinned content — ready to implement)
3. Content Calendar (posting frequency, content mix, timing, content types)
4. Growth Strategy (hashtags, engagement tactics, collaborations, milestones)
5. Cross-Promotion (how this persona engages with network, frequency, guidelines)
6. Audience Targeting (ideal follower profile, where to find them, engagement approach)
7. Metrics & KPIs (what to track, targets, optimization triggers)
8. 90-Day Roadmap (month 1/2/3 priorities, week-by-week if possible)
9. Quick-Start Checklist (immediate action items)
10. Troubleshooting (common obstacles and solutions)

## Quality Gates
- [ ] One complete plan per persona × platform created
- [ ] Each plan minimum 15KB
- [ ] All 8 required sections present
- [ ] Quick-start checklist and troubleshooting included
- [ ] Internal consistency verified (sections align)
- [ ] Platform-specific adaptations throughout

## Handoff
Feeds into: 4.2 Network Sync Check (plans provide input for conflict verification)
