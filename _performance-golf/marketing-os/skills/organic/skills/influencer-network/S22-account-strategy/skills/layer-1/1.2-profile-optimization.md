# 1.2 Profile Optimization

## Purpose
Design bio copy, profile specifications, link strategy, and visual hierarchy per persona × platform. Optimize for discoverability, credibility, and conversion to follow.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Platform selections (from 1.1)
- Persona Bibles (voice, identity, niche)
- Teaching summary (profile optimization formulas, platform specs)

## Execution Protocol

### MUST
- Write bio copy per persona × platform respecting character limits (Twitter 160, Instagram 150, LinkedIn 2000)
- Use persona voice from Bible while optimizing for platform conventions
- Include niche keywords for discoverability
- Design link strategy: what link(s) to include, link-in-bio tool or direct link
- Specify profile photo direction per platform (dimensions, style consistency)
- Define header/cover image strategy (where applicable)
- Create pinned content or highlight strategy (platform-specific)
- Optimize username/handle for searchability and brand consistency

### NEVER
- Exceed platform character limits
- Write generic bios that don't signal niche expertise
- Ignore persona voice in favor of "optimization"
- Skip link strategy (critical conversion point)
- Use identical bio copy across platforms without adaptation
- Forget about profile photo specifications

## Output
**Filename:** `profile-optimization-specs.md`

**Format:** Markdown profile specifications

**Minimum Size:** 6KB

**Required Sections:**
1. Per-Persona × Platform Bio Copy (complete bio text respecting limits)
2. Link Strategy (which links, link-in-bio vs direct, what's being optimized for)
3. Profile Photo Specifications (dimensions, style direction per platform)
4. Header/Cover Image Strategy (where applicable)
5. Pinned Content / Highlights Strategy (platform-specific features)
6. Username/Handle Optimization (searchability, consistency across network)

## Quality Gates
- [ ] Bio copy written for each persona × platform combination
- [ ] All bio copy within platform character limits
- [ ] Persona voice maintained while optimizing for discovery
- [ ] Link strategy defined per persona
- [ ] Profile photo specs include platform-specific dimensions
- [ ] Pinned content strategy included for applicable platforms

## Handoff
Feeds into: 1.3 Content Calendar Framework (profile attracts audience, content retains them)
