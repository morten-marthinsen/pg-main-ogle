## Purpose
Load network coordination frameworks, engagement choreography principles, narrative sequencing methodologies, and multi-persona orchestration best practices.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- Network coordination frameworks (if specified)
- Engagement choreography principles
- Narrative sequencing methodologies

## Execution Protocol

### MUST
- Load coordination frameworks: how to orchestrate multiple personas without appearing coordinated
- Load engagement choreography principles: timing variations, language diversity, participation rates
- Load narrative sequencing: how to build multi-day story arcs across personas
- Load authenticity preservation: how to maintain organic appearance despite coordination
- Load brand firewall protocols: how to coordinate without revealing connection
- Include case studies or examples of successful multi-persona coordination

### NEVER
- Use generic "collaboration" advice (need specific multi-persona orchestration)
- Skip authenticity preservation principles (critical for network success)
- Ignore brand firewall protocols
- Load frameworks without operational detail
- Assume coordination is intuitive without teaching
