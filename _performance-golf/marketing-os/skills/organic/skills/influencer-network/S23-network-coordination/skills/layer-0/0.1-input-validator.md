# 0.1 Input Validator

## Purpose
Validate all S21 Persona Bibles and S22 Account Plans exist and are complete before network coordination begins. Ensure cross-promotion map and network overview available.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- S21 Persona Bibles (all persona-bible-[name].md files)
- S21 Network Overview (network-overview.md)
- S22 Account Strategy Plans (all account-plan-[persona]-[platform].md files)
- S22 Cross-Promotion Map (cross-promotion-map.md)
- S22 Network Sync Report (network-sync-report.md)

## Execution Protocol

### MUST
- Verify all Persona Bibles from S21 exist and are minimum 30KB
- Confirm Network Overview exists with differentiation matrix and brand firewall rules
- Verify all Account Plans from S22 exist and are minimum 15KB per persona × platform
- Check Cross-Promotion Map exists with frequency limits and authenticity guidelines
- Validate Network Sync Report shows READY status (not ISSUES)
- Flag missing or incomplete inputs with specific requirements

### NEVER
- Proceed with missing Persona Bibles or Account Plans
- Skip Network Sync Report verification
- Ignore ISSUES status in sync report
- Continue if Cross-Promotion Map missing
- Accept incomplete brand firewall rules

## Output
**Filename:** `input-validation-report.md`

**Format:** Markdown validation report

**Minimum Size:** 2KB

**Required Sections:**
1. Persona Bible Status (all present, size-validated)
2. Network Overview Status (PASS/FAIL + gaps)
3. Account Plan Status (all persona × platform combinations present)
4. Cross-Promotion Map Status (PASS/FAIL)
5. Network Sync Report Status (READY or ISSUES)
6. Overall Readiness (READY / INCOMPLETE)

## Quality Gates
- [ ] All Persona Bibles and Account Plans accounted for
- [ ] Network Sync Report shows READY status
- [ ] Cross-Promotion Map complete with authenticity guidelines
- [ ] Brand firewall rules loaded
- [ ] Overall readiness decision is binary
- [ ] If INCOMPLETE, execution MUST halt

## Handoff
Feeds into: 0.2 Teaching Loader (if READY) or escalates to human (if INCOMPLETE)
