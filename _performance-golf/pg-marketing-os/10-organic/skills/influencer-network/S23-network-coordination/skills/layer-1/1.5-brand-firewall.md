# 1.5 Brand Firewall

## Purpose
Ensure personas never reveal parent brand connection. Create firewall protocols for coordination that maintains persona independence and network authenticity.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Network Overview (brand firewall rules from S21)
- All coordination plans (1.1-1.4)
- Persona Bibles (identity, voice)

## Execution Protocol

### MUST
- Audit all coordination plans for brand reveals (mentions, links, common phrases)
- Define forbidden content: what personas must NEVER say, link to, or reference
- Create discovery narratives: how personas "found" each other (authentic backstories)
- Design independence signals: how personas demonstrate they're not part of network
- Verify cross-promotion frequency stays under threshold (<2 per persona per week)
- Establish firewall monitoring: how to detect breaches in ongoing operations
- Create escalation protocol: what to do if firewall breach occurs

### NEVER
- Allow any persona to mention parent brand name
- Permit links to parent brand properties
- Use parent brand taglines or distinctive phrases
- Create coordination patterns that reveal common management
- Skip discovery narratives (critical for cross-promotion authenticity)
- Ignore cross-promotion frequency thresholds

## Output
**Filename:** `brand-firewall-protocol.md`

**Format:** Markdown firewall enforcement document

**Minimum Size:** 4KB

**Required Sections:**
1. Firewall Audit (all coordination plans checked for brand reveals)
2. Forbidden Content List (what personas must never say/link/reference)
3. Discovery Narratives (how personas "found" each other authentically)
4. Independence Signals (how personas demonstrate they're not part of network)
5. Cross-Promotion Frequency Limits (enforcement of <2 per persona per week)
6. Firewall Monitoring Protocol (ongoing breach detection)
7. Escalation Protocol (breach response procedures)

## Quality Gates
- [ ] All coordination plans audited for brand reveals
- [ ] Forbidden content list explicit and comprehensive
- [ ] Discovery narratives created for all cross-persona references
- [ ] Independence signals designed into coordination
- [ ] Cross-promotion frequency verified under threshold
- [ ] Monitoring and escalation protocols operational
