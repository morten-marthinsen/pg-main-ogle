# 1.2 Engagement Choreography

## Purpose
Plan organic-looking engagement across personas: comments, shares, reactions. Must appear authentic, not coordinated. Design timing variations, language diversity, realistic participation rates.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Network calendar sync (from 1.1)
- Cross-promotion map (from S22)
- Persona Bibles (voice for authentic engagement)
- Teaching summary (choreography principles)

## Execution Protocol

### MUST
- Design engagement timing: minimum 4-hour delay after original post, spread over 24-72 hours
- Create language variation guidelines: how each persona engages in their voice, zero copy-paste
- Define realistic participation rates: 10-30% cross-promotion (most content standalone)
- Plan engagement types: comments (adds value), shares (with persona spin), reactions, quotes
- Include "missed opportunities" for realism (not every persona engages with every post)
- Design variation patterns: sometimes Persona A engages first, sometimes last, sometimes not at all
- Verify engagement doesn't reveal coordination (no simultaneous, no identical language)

### NEVER
- Create simultaneous engagement within 2-hour windows
- Use identical or near-identical language across personas
- Design 100% participation (too perfect, signals coordination)
- Skip language variation guidelines
- Forget "missed opportunities" for realism
- Create predictable engagement patterns

## Output
**Filename:** `engagement-choreography.md`

**Format:** Markdown engagement playbook

**Minimum Size:** 6KB

**Required Sections:**
1. Engagement Timing Guidelines (delays, spread patterns, variation)
2. Language Variation Rules (per-persona engagement voice, zero copy-paste)
3. Participation Rate Design (10-30% cross-promotion, realistic patterns)
4. Engagement Type Mix (comments, shares, reactions, quotes — when to use each)
5. Missed Opportunities Strategy (intentional non-engagement for realism)
6. Authenticity Verification (timing check, language check, pattern detection)

## Quality Gates
- [ ] Engagement timing minimum 4-hour delay, spread over 24-72 hours
- [ ] Language variation guidelines prevent identical phrasing
- [ ] Participation rate 10-30% (most content standalone)
- [ ] Engagement types diverse (not just shares)
- [ ] Missed opportunities designed for realism
- [ ] Authenticity verified (no coordination signals)
