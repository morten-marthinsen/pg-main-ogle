# 1.1 Niche Ownership

## Purpose
Define what specific niche each persona OWNS. No overlap allowed. Map niches to audience segments from AIF to ensure full audience coverage without duplication.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Upstream context summary (audience segments, brand constraints)
- Teaching summary (archetype framework)
- Competitive context (existing influencer landscape)
- Number of personas to create (typically 3-7)

## Execution Protocol

### MUST
- Define one primary niche per persona
- Ensure zero overlap between persona niches (different sub-topics, audiences, or angles)
- Map each niche to specific audience segments from AIF
- Verify all major audience segments have a persona assigned
- Justify why each niche deserves its own persona (distinct enough to warrant separate identity)
- Consider competitive gaps (niches underserved by existing influencers)
- Document niche boundaries clearly (what's IN, what's OUT)

### NEVER
- Create overlapping niches that would cause persona competition
- Assign same audience segment to multiple personas
- Create niches too narrow to sustain content production
- Create niches too broad to establish clear ownership
- Ignore competitive landscape when defining niches

## Output
**Filename:** `niche-ownership-map.md`

**Format:** Markdown niche definition document

**Minimum Size:** 4KB

**Required Sections:**
1. Niche Definitions (one per persona: name, scope, boundaries, target sub-audience)
2. Audience Segment Mapping (which segments each niche serves)
3. Competitive Positioning (how each niche differs from existing influencers)
4. Coverage Verification (confirm all audience segments covered)
5. Differentiation Matrix (table showing how niches don't overlap)

## Quality Gates
- [ ] Each niche has clear boundaries (IN/OUT scope defined)
- [ ] Zero overlap between niches verified in differentiation matrix
- [ ] All audience segments from AIF assigned to at least one niche
- [ ] Competitive gaps identified and exploited
- [ ] Each niche justified as distinct enough for separate persona

## Handoff
Feeds into: 1.2 Archetype Selection (niches inform archetype choices)
