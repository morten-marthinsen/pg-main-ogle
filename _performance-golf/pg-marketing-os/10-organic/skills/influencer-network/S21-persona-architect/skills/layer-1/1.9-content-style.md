# 1.9 Content Style

## Purpose
Define preferred content formats, posting rhythm, signature content types, recurring series concepts for each persona. Shapes how persona shows up consistently on platforms.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Voice architecture (from 1.8)
- Archetype assignments (from 1.2)
- Platform priorities (from upstream context)
- Niche ownership map (from 1.1)

## Execution Protocol

### MUST
- Define preferred content formats per platform: text, video, carousel, threads, stories, reels, etc.
- Specify posting rhythm: frequency, day-of-week patterns, time-of-day preferences
- Create signature content types: 3-5 recurring formats this persona is known for
- Design recurring series concepts: weekly features, monthly themes, ongoing narratives
- Consider archetype-format alignment (Educator = how-to, Provocateur = hot takes)
- Ensure content style fits niche and audience expectations
- Differentiate content approaches across personas

### NEVER
- Create identical content styles across personas
- Ignore platform-format norms (e.g., LinkedIn carousels vs TikTok short video)
- Define content style without considering production feasibility
- Skip recurring series (critical for audience expectation and retention)
- Create posting rhythm that's unsustainable

## Output
**Filename:** `content-style.md`

**Format:** Markdown content style specifications

**Minimum Size:** 5KB

**Required Sections:**
1. Per-Persona Content Format Preferences (by platform)
2. Posting Rhythm Specifications (frequency, timing patterns)
3. Signature Content Types (3-5 recurring formats per persona)
4. Recurring Series Concepts (weekly/monthly features)
5. Content Style Differentiation (how personas differ in content approach)

## Quality Gates
- [ ] Each persona has platform-specific format preferences
- [ ] Posting rhythm is specific (not "post regularly")
- [ ] Minimum 3 signature content types per persona
- [ ] Recurring series concepts include at least 2 per persona
- [ ] Content style differentiation across personas verified
