# 0.2 Teaching Loader

## Purpose
Load platform strategy teachings, growth frameworks, profile optimization best practices, and content calendar methodologies. Provides tactical foundation for creating effective platform-specific strategies.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- Platform strategy frameworks (if specified)
- Growth tactics libraries
- Profile optimization guides per platform
- Content calendar best practices

## Execution Protocol

### MUST
- Load platform-specific conventions: character limits, image specs, posting best practices
- Load growth frameworks: hashtag strategy, engagement tactics, algorithm understanding
- Load profile optimization principles: bio formulas, link strategies, visual hierarchy
- Load content calendar methodologies: posting frequency research, timing optimization, content mix ratios
- Create teaching summary that operationalizes frameworks for account strategy development
- Include platform-specific examples (not just generic social media advice)

### NEVER
- Use generic "post regularly" advice without platform specifics
- Load outdated platform conventions (verify current specs)
- Skip algorithm understanding (critical for organic growth)
- Assume same strategy works across all platforms
- Ignore platform demographic differences

## Output
**Filename:** `teaching-summary.md`

**Format:** Markdown teaching reference

**Minimum Size:** 4KB

**Required Sections:**
1. Platform Specifications (character limits, image dimensions, posting mechanics per platform)
2. Growth Frameworks (hashtag strategy, engagement tactics, algorithm insights)
3. Profile Optimization Principles (bio formulas, link strategies, visual hierarchy)
4. Content Calendar Methodologies (frequency research, timing, content mix)
5. Platform-Specific Examples (actual account examples demonstrating principles)

## Quality Gates
- [ ] All major platforms covered (minimum: Twitter/X, Instagram, LinkedIn, TikTok)
- [ ] Platform specifications current and accurate
- [ ] Growth frameworks include tactical depth (not generic advice)
- [ ] Profile optimization includes platform-specific formulas
- [ ] Content calendar methodologies backed by research or data

## Handoff
Feeds into: 0.3 Upstream Loader and all Layer 1 microskills (teaching reference throughout)
