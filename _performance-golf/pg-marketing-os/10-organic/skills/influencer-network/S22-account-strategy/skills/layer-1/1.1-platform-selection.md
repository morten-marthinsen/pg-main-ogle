# 1.1 Platform Selection

## Purpose
Select platforms per persona based on archetype fit, audience presence, content format match, and niche requirements. Not all personas need to be on all platforms.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Upstream context summary (persona summaries with platform fit signals)
- Teaching summary (platform specifications and audience demographics)
- PSF platform priorities

## Execution Protocol

### MUST
- Evaluate each persona against each major platform (Twitter/X, Instagram, LinkedIn, TikTok, YouTube)
- Score platform fit across 5 dimensions: archetype compatibility, audience presence, content format match, niche relevance, growth potential
- Select 2-4 platforms per persona (focus over breadth)
- Justify why platforms were selected AND why others were rejected
- Consider production capacity (don't select platforms requiring formats persona can't sustain)
- Differentiate platform selections across personas (avoid identical platform portfolios)
- Prioritize platform order (primary platform vs. secondary)

### NEVER
- Put all personas on all platforms (dilutes focus)
- Select platforms without archetype-format fit analysis
- Ignore audience presence data from PSF
- Choose platforms based on creator preference rather than persona fit
- Create identical platform selections across personas

## Output
**Filename:** `platform-selection-per-persona.md`

**Format:** Markdown platform assignments

**Minimum Size:** 4KB

**Required Sections:**
1. Per-Persona Platform Selections (2-4 platforms each with priority ranking)
2. Selection Rationale (why these platforms for this persona across 5 dimensions)
3. Rejection Rationale (why other platforms NOT selected)
4. Network Platform Distribution (matrix showing which personas on which platforms)
5. Production Capacity Assessment (can these platform commitments be sustained?)

## Quality Gates
- [ ] Each persona assigned 2-4 platforms with priority ranking
- [ ] Selection justified across 5 fit dimensions
- [ ] Rejection rationale documented (not just positive selections)
- [ ] Platform distribution shows differentiation (not identical across personas)
- [ ] Production capacity assessed and realistic

## Handoff
Feeds into: 1.2 Profile Optimization (selected platforms need profile strategies)
