# 1.2 Platform Adaptation Matrix

## Purpose
Define how atomic units transform into platform-native content. Specify adaptation rules for each platform to ensure rebuilding (not resizing).

---

## Input Requirements

From Layer 1:
- `1.1-atomization.yaml` (atomic units)

From Layer 0:
- `0.2-upstream-content.yaml` (platform requirements, voice)

---

## Execution Protocol

### Step 1: Define Platform-Specific Adaptation Rules

For each active platform, specify:

**Instagram:**
- Reel adaptations (hook, pacing, captions, music, CTA)
- Carousel adaptations (slide structure, design, caption)
- Story adaptations (format, interactive elements)

**TikTok:**
- Native restyle (energy matching, hook optimization, text styling)
- Series episode structure
- Trend incorporation opportunities

**YouTube:**
- Shorts from long-form (extraction, vertical crop/re-record)
- Community post format
- Premiere clip teasers

**LinkedIn:**
- Text post from video (problem→insight→lesson→question)
- Carousel format (professional styling, clean design)
- Article from long-form

**X/Twitter:**
- Thread structure (hook tweet, content tweets, CTA tweet)
- Single tweet extraction
- Quote tweet expansion

### Step 2: Map Atomic Units to Platform Formats

For each high-potential atomic unit (strength >= 7):
- Which platforms are best fit?
- Which format on each platform?
- What adaptation is required?

### Step 3: Specify Transformation Requirements

For each platform/format combination:
- Hook transformation
- Pacing adjustments
- Text styling (platform-native)
- CTA adaptation
- Audio/music requirements

### MUST Define

- Adaptation rules for 3+ platforms
- Transformation requirements for each format
- Platform-native styling requirements

### NEVER Allow

- "Same content, different platform" (cross-posting)
- "Minor edits" (must be rebuilt)
- "Resize for [platform]" (must adapt)

## Output

**File:** `layer-1/1.2-platform-adaptation-matrix.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"

platform_adaptations:
  instagram:
    formats:
      reel:
        transformation_requirements:
          hook: "Re-cut for first 0.5-1 second impact"
          pacing: "Faster cuts, more visual variety"
          captions: "Instagram native text style"
          music: "Trending or original audio"
          cta: "Follow for more / Save this / Link in bio"
        output_length: "7-90 seconds"

      carousel:
        transformation_requirements:
          slide_1: "Hook slide (question or bold statement)"
          slides_2_to_n: "One point per slide, visual hierarchy"
          final_slide: "CTA + summary"
          design: "Consistent brand styling"
        output_slides: "5-10 slides"

  tiktok:
    formats:
      native_video:
        transformation_requirements:
          energy: "Match TikTok energy (usually higher)"
          hook: "First 0.5 seconds critical"
          format: "Native text styling"
          trends: "Incorporate trending sounds if relevant"
        output_length: "15-60 seconds"

  youtube:
    formats:
      shorts:
        transformation_requirements:
          extract: "Best 30-60 second moments"
          hook: "Front-load the value"
          cta: "Watch full video (link in description)"
          vertical: "Crop or re-record for 9:16"
        output_length: "30-60 seconds"

  linkedin:
    formats:
      text_post:
        transformation_requirements:
          extract: "Key insight or story"
          format: "Text with line breaks"
          hook: "First 2 lines must hook (preview cut-off)"
          structure: "Problem → Insight → Lesson → Question"
        output_length: "150-300 words"

  x_twitter:
    formats:
      thread:
        transformation_requirements:
          tweet_1: "Hook + promise"
          tweets_2_to_n: "One point per tweet"
          final_tweet: "Summary + CTA"
        output_tweets: "5-15 tweets"

atomic_unit_platform_mapping:
  - atomic_unit_id: "[ID]"
    unit_type: "hook"
    best_platforms:
      - platform: "instagram"
        format: "reel"
        transformation: "High-energy opening, faster pacing"
      - platform: "tiktok"
        format: "native_video"
        transformation: "Even higher energy, trending sound"

  # [Repeat for high-potential atomic units]

adaptation_summary:
  platforms_covered: [Count]
  formats_per_platform: {}
  transformation_types: [List unique transformations]
```

**Minimum size:** 3KB

- [ ] All rules specify rebuilding (not resizing)
- [ ] High-potential atomic units mapped to platforms
