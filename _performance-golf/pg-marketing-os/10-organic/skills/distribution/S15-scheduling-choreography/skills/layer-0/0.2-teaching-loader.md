## Execution Protocol

### Step 1: Load Platform Timing Teachings

Platform-specific optimal posting windows (2026 data):

**Instagram:**
- Weekday Tier 1: 6:00-7:00 AM, 12:00-1:00 PM, 7:00-9:00 PM
- Reels best: 12:00-1:00 PM, 7:00-8:00 PM
- Stories best: 8:00-9:00 AM, 5:00-6:00 PM
- First hour critical: 5%+ engagement = algorithm push

**TikTok:**
- Weekday Tier 1: 7:00-9:00 AM, 12:00-3:00 PM, 7:00-11:00 PM
- First 30 minutes critical for initial push
- Optimal frequency: 1-3 posts/day, 3hr minimum gap

**YouTube:**
- Shorts: 12:00-3:00 PM, 7:00-10:00 PM weekdays
- Long-form: 2:00-4:00 PM weekdays (publish before evening viewing)
- First 24 hours critical for long-form momentum

**LinkedIn:**
- Weekday only: 7:00-8:00 AM, 10:00-11:00 AM, 5:00-6:00 PM
- Weekend engagement limited
- First 90 minutes critical for algorithmic push

**X/Twitter:**
- Weekday: 8:00-10:00 AM, 12:00-1:00 PM, 5:00-6:00 PM
- First 30 minutes determines reach trajectory
- Optimal frequency: 3-5 posts/day

### Step 2: Load Cross-Platform Cascade Best Practices

**Cascade Rules:**
- Never post same content simultaneously across platforms
- Minimum 30min delay between platforms
- Primary platform first, secondary 2-4hr later
- Tertiary platform 18-24hr later
- Each platform requires native adaptation

### Step 3: Load Momentum Stacking Teachings

**Viral Response Protocol:**
- If content exceeds 3x average performance in first hour → activate amplification
- Within 30 min: increase engagement, post supporting story
- Within 2 hours: create follow-up content
- Within 24 hours: create Part 2 or response video
