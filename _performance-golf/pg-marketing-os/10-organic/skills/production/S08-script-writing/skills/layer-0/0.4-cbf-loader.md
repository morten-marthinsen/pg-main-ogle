# 0.4 CBF Loader

## Purpose
Extract relevant context from Campaign Brief File (CBF) for script writing. The CBF contains strategic direction that constrains and guides all content production.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (CBF path validated)

---

## Execution Protocol

### Step 1: Load Full CBF

Read the complete Campaign Brief File:

```yaml
cbf_source:
  path: "[project]/S07-campaign-brief/outputs/[campaign]-CBF.yaml"
  campaign_name: "[name]"
  date_created: "[date]"
```

---

### Step 2: Extract Core Campaign Context

Pull high-level campaign information:

```yaml
campaign_context:
  brand_name:
  campaign_objective:
  target_audience:
    primary_demographic:
    psychographic_profile:
    pain_points: []
    desires: []

  brand_voice:
    tone:
    energy_level:
    signature_phrases: []
    banned_words: []
```

---

### Step 3: Extract Content Strategy

Pull content pillar and function distribution:

```yaml
content_strategy:
  content_pillars:
    - pillar_name: "[name]"
      pillar_description: "[description]"
      topics: []
      audience_need: "[what need this addresses]"

  content_functions:
    awareness: [%]
    engagement: [%]
    conversion: [%]
    community: [%]

  content_types_prioritized: []
```

---

### Step 4: Extract Platform Strategy

Pull platform-specific requirements:

```yaml
platform_strategy:
  primary_platforms: []

  per_platform:
    tiktok:
      posting_frequency:
      optimal_times:
      format_priorities: []
      hook_requirements:
      length_preferences:

    instagram:
      posting_frequency:
      optimal_times:
      format_priorities: []
      hook_requirements:
      length_preferences:

  # ... repeat for each platform
```

---

### Step 5: Extract Funnel Integration

Pull CTA strategy and conversion goals:

```yaml
funnel_integration:
  primary_conversion_goal:

  cta_strategy:
    soft_ctas:
      - action: "Save this"
        context: "Educational/value posts"
      - action: "Follow for more"
        context: "All content"

    medium_ctas:
      - action: "Link in bio"
        context: "Awareness → Lead generation"

    hard_ctas:
      - action: "Book call"
        context: "Conversion posts only"

  funnel_stages:
    awareness_goal:
    engagement_goal:
    conversion_goal:
```

---

### Step 6: Extract Hook Library Reference

If CBF references Hook Library from S05:

```yaml
hook_library_ref:
  hlf_path: "[project]/S05-hook-library/outputs/[campaign]-HLF.yaml"
  hook_categories_available: []
  recommended_hooks_per_pillar: {}
```

---

### Step 7: Create Script-Writing Context Package

Synthesize CBF data into actionable context for script generation:

```yaml
script_context:
  messaging_constraints:
    core_message: "[Central message]"
    supporting_messages: []
    brand_voice_requirements: []

  platform_requirements:
    tiktok: "[Requirements summary]"
    instagram: "[Requirements summary]"

  cta_guidance:
    - script_function: "Awareness"
      recommended_cta: "Soft (save/follow)"
    - script_function: "Conversion"
      recommended_cta: "Hard (link/book)"

  content_pillar_mapping:
    pillar_1: "[Pillar name → topics → hook types]"
    pillar_2: "[...]"
```

---

```yaml
  primary_conversion_goal:
  cta_strategy: {...}

```

**Minimum size:** 2KB
