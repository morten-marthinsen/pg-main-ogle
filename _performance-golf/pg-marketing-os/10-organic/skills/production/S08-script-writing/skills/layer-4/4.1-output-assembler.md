# 4.1 Output Assembler

## Purpose
Assemble final script outputs in standardized format with virality scoring and production specifications. Validate all requirements met before handoff.

---

## Input Requirements

From Layer 2.5:
- `arena/[script-id]-arena-synthesis.md` (final selected scripts)
- `checkpoints/ARENA_COMPLETE.yaml` (Arena verification)

From Layer 2:
- `2.4-[script-id]-visual-direction.md` (visual specs)

---

## Execution Protocol

### Step 1: Load Arena-Selected Scripts

For each script, load the human-selected final version:

```yaml
final_scripts:
  - script_id: "script_1"
    arena_file: "arena/script-1-arena-synthesis.md"
    selected_hybrid: "Hybrid A"
    final_script:
      hook: "[Final hook text]"
      body: "[Final body text]"
      cta: "[Final CTA text]"
      visual_direction: "[Final visual notes]"
```

---

### Step 2: Apply Virality Scoring

Score each final script using VSF (Virality Scoring Framework):

```yaml
virality_scoring:
  script_id: "script_1"

  dimensions:
    ea_emotional_activation:
      score: 8/10
      rationale: "Hook creates surprise, body delivers aha moments"

    sc_social_currency:
      score: 7/10
      rationale: "Valuable insights make sharer look smart"

    pi_pattern_interrupt:
      score: 9/10
      rationale: "Visual hook + fast cuts every 5s"

    pf_platform_fit:
      score: 9/10
      rationale: "Perfectly matches TikTok algorithm signals"

    sh_shareability:
      score: 8/10
      rationale: "Save-worthy value + loop structure"

  total_score: 82/100
  threshold: 60
  status: PASS

  improvement_areas:
    - "Social currency could be higher with stat/proof"
```

**CRITICAL:** If total_score < 60 → script requires revision
**CRITICAL:** If total_score < 40 → script is BLOCKED

---

### Step 3: Compile Production Package

For each script, create complete production package:

```yaml
production_package:
  script_id: "script_1"
  platform: "TikTok"
  format: "Short-form 25s"

  script_sections:
    hook:
      timing: "0:00-0:01"
      verbal: "[Exact words to say]"
      visual: "[Shot-by-shot direction]"
      text_overlay: "[Exact text, styling, timing]"
      audio: "[Sound/music notes]"

    value_section_1:
      timing: "0:01-0:10"
      verbal: "[Script]"
      visual: "[Direction]"
      pattern_interrupt: "0:05 - Visual cut"
      text_overlay: "[Text]"

    value_section_2:
      timing: "0:10-0:22"
      verbal: "[Script]"
      visual: "[Direction]"
      pattern_interrupt: "0:15 - Text emphasis"
      text_overlay: "[Text]"

    cta:
      timing: "0:22-0:25"
      verbal: "[CTA script]"
      visual: "[End card/loop]"
      text_overlay: "[Text]"

  production_specs:
    total_length: "25 seconds"
    aspect_ratio: "9:16"
    pacing: "Fast"
    energy: "High"
    broll_needed:
      - "[B-roll requirement 1]"
      - "[B-roll requirement 2]"
    graphics_needed:
      - "[Graphic requirement 1]"
    sound_requirements: "[Trending sound or original]"

  metadata:
    content_pillar: "[Pillar]"
    content_function: "[Function]"
    target_audience: "[From CBF]"
    brand_voice_notes: "[From BVF]"
```

---

### Step 4: Validate Completeness

Check all requirements:

```yaml
validation_checklist:
  script_id: "script_1"

  structural_requirements:
    - hook_present: true
    - hook_timing_0_3_seconds: true
    - body_complete: true
    - cta_clear: true
    - visual_direction_all_sections: true
    - timing_annotations_accurate: true

  quality_requirements:
    - arena_complete: true
    - virality_score_met: true  # >= 60
    - platform_native: true
    - anti_slop_passed: true

  production_requirements:
    - aspect_ratio_specified: true
    - broll_identified: true
    - graphics_identified: true
    - sound_requirements_noted: true

  all_checks_passed: true
```

**If any check fails → INCOMPLETE → fix before output**

---

### Step 5: Generate Summary Report

Create campaign-level summary:

```yaml
campaign_summary:
  campaign_name: "[Campaign]"
  total_scripts: [count]

  scripts_by_platform:
    tiktok: [count]
    instagram: [count]
    youtube: [count]

  scripts_by_function:
    awareness: [count]
    engagement: [count]
    conversion: [count]

  average_virality_score: [score]

  scripts_above_80: [count]
  scripts_60_79: [count]
  scripts_below_60: [count]

  production_requirements_summary:
    total_broll_shots: [count]
    total_graphics: [count]
    trending_sounds_needed: [count]

  estimated_production_time: "[hours]"
```

---

## Output

**File:** `outputs/[campaign]-scripts.yaml`

**Format:**
```yaml
campaign_name: "[Campaign]"
date_created: "[ISO 8601]"
version: "1.0"

scripts:
  - script_id: "script_1"
    platform: "[platform]"
    format: "[format]"
    target_length: "[seconds]"

    script_sections: {...}

    production_specs: {...}

    virality_score:
      ea: 8/10
      sc: 7/10
      pi: 9/10
      pf: 9/10
      sh: 8/10
      total: 82/100

    arena_notes:
      selected_hybrid: "[Hybrid X]"
      strongest_personas: []

    metadata: {...}

  # ... all scripts

campaign_summary: {...}

validation:
  all_scripts_complete: true
  all_virality_scores_met: true
  all_arena_complete: true
  ready_for_production: true
```

**Minimum size:** 10KB+ (depending on script count)

---

## Quality Gates

- [ ] All scripts from Arena loaded
- [ ] Virality scores >= 60 for all scripts
- [ ] Production packages complete
- [ ] Validation checklist passed for all scripts
- [ ] Campaign summary generated

**If any script fails validation → INCOMPLETE → remediate before final output**

---

## Handoff

Pass complete script package to: **S14 Content Assembly** and **4.2 Execution Log**
