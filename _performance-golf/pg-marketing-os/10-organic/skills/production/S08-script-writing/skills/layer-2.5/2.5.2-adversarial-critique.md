# 2.5.2 Adversarial Critique

## Purpose
Conduct adversarial critique where each persona challenges the others' perspectives. Rounds 2 and 3 incorporate this competition to prevent convergence.

---

## Input Requirements

From Layer 2.5:
- `[script-id]-arena-round-1.yaml` (Round 1 results)

---

## Execution Protocol

### Step 1: Identify Convergence Risks

Analyze Round 1 for signs of convergence:

```yaml
convergence_check:
  score_variance: [Calculate std deviation of scores]
  # Low variance (< 1.5) = potential convergence

  perspective_diversity:
    unique_suggestions: [Count distinct revision ideas]
    # < 5 unique suggestions = convergence risk

  criticism_depth:
    average_critique_length: [words]
    # < 50 words per persona = surface-level

  convergence_detected: true/false
  severity: [1-10]
```

**If convergence detected:** Flag for aggressive adversarial critique.

---

### Step 2: Conduct Adversarial Round

Each persona attacks weaknesses in others' approaches:

```yaml
adversarial_critique:
  round: 2

  volume_machine_attacks:
    targets: ["Brand Purist", "Storyteller"]
    critique: |
      "Brand Purist wants perfection that will never ship.
      Storyteller adds narrative fluff that kills pacing.
      This script needs to be DONE and TESTED, not perfect."

  brand_purist_attacks:
    targets: ["Volume Machine", "Algorithm Hacker"]
    critique: |
      "Volume Machine sacrifices quality for quantity.
      Algorithm Hacker chases trends with no brand coherence.
      One great piece > ten mediocre pieces."

  virality_engineer_attacks:
    targets: ["Community Builder", "Storyteller"]
    critique: |
      "Community Builder ignores viral mechanics.
      Storyteller's narrative arc kills retention.
      Data says: hook + value + loop = viral. Story = optional."

  # ... all personas attack each other
```

---

### Step 3: Run Round 2 with Revisions

Each persona revises the script based on adversarial feedback:

```yaml
round_2_revisions:
  - persona: "Volume Machine"
    revised_script:
      hook: "[Revised hook emphasizing speed]"
      body: "[Tighter, faster pacing]"
      cta: "[Quick, direct]"
    score_justification: |
      "Incorporated Value Architect's depth but kept pacing.
      Rejected Brand Purist's over-polishing."

  - persona: "Value Architect"
    revised_script:
      hook: "[Revised hook with deeper promise]"
      body: "[More tactical value]"
      cta: "[Value-reinforcing CTA]"
    score_justification: |
      "Added tactical depth Virality Engineer requested.
      Maintained educational core against Volume Machine's rush."

  # ... all 7 personas revise
```

---

### Step 4: Score Round 2 Revisions

Re-evaluate after adversarial revisions:

```yaml
round_2_results:
  average_score: 8.2/10  # Should improve from Round 1
  score_improvement: +0.6

  persona_scores:
    volume_machine: 8/10
    value_architect: 9/10
    virality_engineer: 8/10
    community_builder: 8/10
    brand_purist: 8/10
    algorithm_hacker: 9/10
    storyteller: 8/10

  diversity_maintained: true
  convergence_risk: low

  breakthrough_insights:
    - "[Insight from adversarial tension]"
    - "[Unexpected synthesis idea]"
```

---

### Step 5: Run Round 3 (Final Round)

Repeat adversarial process for Round 3:

**CRITICAL:** 3 rounds are MANDATORY. Cannot skip Round 3.

---

## Output

**File:** `arena/[script-id]-arena-round-2.yaml` and `arena/[script-id]-arena-round-3.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"
script_id: "[id]"
round: 2

convergence_check:
  convergence_detected: false
  severity: 3/10

adversarial_critique:
  [persona]_attacks: {...}

revised_scripts:
  - persona: "[persona]"
    revised_script: {...}
    score_justification: |

round_results:
  average_score: 8.2/10
  persona_scores: {...}
  diversity_maintained: true
  breakthrough_insights: []

ready_for_round_3: true  # or synthesis if Round 3
```

**Minimum size:** 4KB per round

---

## Quality Gates

- [ ] Round 2 completed with all 7 personas
- [ ] Adversarial critique conducted
- [ ] Round 3 completed with all 7 personas
- [ ] Score improvement from Round 1
- [ ] Diversity maintained (no convergence)

**If convergence detected at severity 8+ → HALT → re-run with stronger adversarial prompts**

---

## Handoff

Pass Round 2 and 3 results to: **2.5.3 Synthesis**
