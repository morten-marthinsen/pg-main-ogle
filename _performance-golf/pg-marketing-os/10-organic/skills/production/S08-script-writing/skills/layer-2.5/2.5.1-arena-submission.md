# 2.5.1 Arena Submission

## Purpose
Submit each draft script to the 7-persona Arena system for competitive evaluation. Mode: `generative_full_draft`. Arena is MANDATORY per Law 2.

---

## Input Requirements

From Layer 2:
- `2.1-[script-id]-hook-draft.md` (all hook options)
- `2.2-[script-id]-body-draft.md` (body content)
- `2.3-[script-id]-cta-draft.md` (CTA options)
- `2.4-[script-id]-visual-direction.md` (visual specs)

---

## Execution Protocol

### Step 1: Assemble PRE-ARENA Script

Combine all Layer 2 outputs into complete script for Arena:

```markdown
# SCRIPT: [Title] - PRE-ARENA

## Metadata
- Platform: [platform]
- Format: [format]
- Target Length: [length]

---

## HOOK (0:00-0:XX)
[Selected hook from 2.1]

## BODY
[Complete body from 2.2]

## CTA (XX:XX-XX:XX)
[Selected CTA from 2.3]

## VISUAL DIRECTION
[Complete visual notes from 2.4]
```

---

### Step 2: Submit to 7 Organic Personas

The 7 organic personas evaluate the script:

1. **Volume Machine** — More content > perfect content. Speed and consistency.
2. **Value Architect** — Give away premium value. Educational depth.
3. **Virality Engineer** — Data-driven format optimization. Algorithm hacking.
4. **Community Builder** — 1,000 true fans > 1M passive. Engagement focus.
5. **Brand Purist** — Less but better. Every post is premium.
6. **Algorithm Hacker** — Platform mechanics. Trend surfing.
7. **Storyteller** — Everything wrapped in narrative.

**Each persona evaluates:**
```yaml
persona_evaluation:
  persona_name: "[Persona]"
  overall_score: [1-10]

  strengths:
    - "[What this persona likes]"

  weaknesses:
    - "[What this persona would change]"

  suggested_revisions:
    hook: "[Specific hook revision]"
    body: "[Specific body revision]"
    cta: "[Specific CTA revision]"

  philosophy_alignment: |
    [How well script aligns with this persona's approach]
```

---

### Step 3: Document Round 1 Results

After all 7 personas evaluate:

```yaml
round_1_results:
  script_id: "[id]"
  round: 1

  persona_scores:
    volume_machine: 7/10
    value_architect: 8/10
    virality_engineer: 6/10
    community_builder: 7/10
    brand_purist: 9/10
    algorithm_hacker: 8/10
    storyteller: 8/10

  average_score: 7.6/10

  consensus_strengths:
    - "[What most personas agreed was strong]"

  consensus_weaknesses:
    - "[What most personas agreed needs work]"

  divergent_perspectives:
    - persona_1: "[Unique perspective]"
    - persona_2: "[Contradictory perspective]"
```

---

### Step 4: Prepare for Round 2

**CRITICAL:** 2 rounds + audience evaluation are MANDATORY. Do not skip.

Round 2 will incorporate learning from Round 1.

---

**Format:**
```yaml
timestamp: "[ISO 8601]"

round: 1

submitted_script:
  hook: "[hook text]"
  body: "[body text]"
  cta: "[cta text]"

  - persona: "Volume Machine"
    score: 7/10
    strengths: []
    weaknesses: []
    suggested_revisions: {...}

  # ... all 7 personas

round_1_results:
  average_score: 7.6/10
  consensus_strengths: []
  consensus_weaknesses: []
  divergent_perspectives: []

ready_for_round_2: true
```

**Minimum size:** 3KB
