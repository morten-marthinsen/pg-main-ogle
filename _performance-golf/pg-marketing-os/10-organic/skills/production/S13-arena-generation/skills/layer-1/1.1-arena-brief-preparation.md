# Microskill 1.1: Arena Brief Preparation
Skill: S13 Arena Generation | Layer: 1 | Model: claude-opus-4.6 | Version: 1.0

## Purpose
Prepare the evaluation context and criteria for Arena execution. This frames what personas will evaluate and how.

## Input
Required gate: `GATE_0_INPUT_VALID`
Required files: s13-0.2-personas.yaml, s13-0.3-content-draft.md, s13-0.4-brand-context.yaml

## Process
### MUST
1. Define evaluation criteria specific to content type
2. Set scoring rubric (0-10 scale, what each score means)
3. Frame evaluation questions for each persona lens
4. Define success threshold
5. Document Arena objectives

### NEVER
- Use generic evaluation criteria (must be content-type specific)
- Skip scoring rubric definition
- Proceed without clear objectives

## Output
**File:** `s13-1.1-arena-brief.yaml`

```yaml
arena_brief:
  content_id: "thread-flagship-2026-03-05"
  content_type: thread
  evaluation_mode: generative_full_draft

  evaluation_criteria:
    hook_strength: "Does tweet 1 stop scrolling?"
    value_density: "Are takeaways clear and actionable?"
    narrative_flow: "Does thread flow tweet-to-tweet?"
    share_ability: "Will audience retweet?"
    brand_alignment: "Voice matches brand?"
    algorithm_signals: "Engagement hooks present?"
    story_resonance: "Memorable moments?"

  scoring_rubric:
    9_10: "Exceptional — publish-ready, reference-quality"
    7_8: "Strong — minor tweaks only"
    5_6: "Acceptable — substantive revision needed"
    3_4: "Weak — major rework required"
    0_2: "Fails — restart from concept"

  persona_evaluation_questions:
    volume_machine: "Can I sustain this quality weekly?"
    value_architect: "What's the tangible takeaway?"
    virality_engineer: "Will people share this?"
    community_builder: "Does this deepen connection?"
    brand_purist: "Is this on-brand?"
    algorithm_hacker: "Will the algorithm reward this?"
    storyteller: "Is this a memorable story?"

  success_threshold: "7.0+ composite score across personas"

  arena_objectives:
    - "Identify content weaknesses before launch"
    - "Surface alternative approaches"
    - "Validate brand alignment"
    - "Generate hybrid improvements"
```

## Quality Gates
- [ ] Content-specific criteria defined
- [ ] Scoring rubric clear
- [ ] Persona questions framed
- [ ] Objectives documented
- [ ] File written

## Handoff
