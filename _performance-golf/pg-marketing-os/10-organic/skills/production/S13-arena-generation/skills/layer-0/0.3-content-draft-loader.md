# Microskill 0.3: Content Draft Loader
Skill: S13 Arena Generation | Layer: 0 | Model: claude-haiku-4 | Version: 1.0

## Purpose
Load the content draft to be evaluated, extracting key elements for Arena context.

## Input
Required gate: `GATE_0_INPUT_VALID`
Content draft file from S08-S12

## Process
### MUST
1. Load complete content draft
2. Extract: content type, length, core promise, structure
3. Identify evaluation dimensions (hook strength, value density, narrative flow, etc.)
4. Package for Layer 1 consumption

### NEVER
- Load partial draft
- Editorialize or modify content
- Skip structural analysis
