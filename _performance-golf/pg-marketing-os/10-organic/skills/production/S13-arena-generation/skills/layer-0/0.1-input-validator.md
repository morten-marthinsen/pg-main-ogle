# Microskill 0.1: Input Validator
Skill: S13 Arena Generation
Layer: 0
Model: claude-haiku-4
Version: 1.0

## Purpose
Validate that content draft exists, is complete, and ready for Arena evaluation. Verify all required context is present.

## Input
- Content draft file from S08-S12 (video script, carousel, thread, post)
- `campaign-brief-foundation.yaml`
- Project infrastructure (CLAUDE.md, PROJECT-STATE.md, PROGRESS-LOG.md)

## Process

### MUST
1. **Verify content draft exists**
   - Check for content file from S08-S12
   - Identify content type: video, carousel, thread, post
   - Verify content is complete (not empty, not placeholder)

2. **Verify CBF exists**
   - Check for campaign-brief-foundation.yaml
   - Verify brand context section exists

3. **Check project infrastructure**
   - Verify CLAUDE.md, PROJECT-STATE.md, PROGRESS-LOG.md exist
   - If missing: create or escalate

4. **Check for stale Arena artifacts**
   - Scan for existing s13-*.yaml or arena_results.yaml
   - If found: flag as stale, recommend archiving

5. **Validate Arena necessity**
   - Check if content is high-stakes (primary video, lead carousel, flagship thread)
   - If low-stakes: flag for human decision on Arena necessity

### NEVER
- Proceed without complete content draft
- Assume content type (verify explicitly)
- Ignore stale artifacts

## Output
**File:** `s13-0.1-validation-report.yaml`

```yaml
validation_status: PASS | FAIL
timestamp: 2026-03-05T14:00:00Z

content_draft:
  exists: true

  files: []

arena_necessity:
  content_stakes: high  # high | medium | low
  recommendation: "MANDATORY — flagship thought leadership thread"

blocking_issues: []
warnings: []
gate_eligible: true
```

## Quality Gates
- [ ] Content draft verified
- [ ] Content type identified
- [ ] CBF exists
- [ ] Infrastructure complete
- [ ] Stale artifacts flagged (if any)
- [ ] File written

**BLOCKING GATE:** Create `GATE_0_INPUT_VALID` if validation passes.
