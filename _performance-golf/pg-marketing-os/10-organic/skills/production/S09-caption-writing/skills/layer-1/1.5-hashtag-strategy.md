# 1.5 Hashtag Strategy

**Layer:** 1 (Strategy)
**Model:** Claude Sonnet
**Upstream:** 1.4 CTA Strategy

---

## PURPOSE

Build platform-appropriate hashtag set from Platform Strategy File (PSF), mixing branded, niche, and trending hashtags. Define hashtag placement (end vs inline), count (platform-specific), and function (discovery, community, trending). Ensure hashtags align with content pillars and target audience.

## EXECUTION PROTOCOL

**MUST:**
1. Load **Platform Strategy File (PSF)** hashtag recommendations:
   - Branded hashtags (owned by brand)
   - Niche hashtags (vertical-specific, 10K-500K posts)
   - Community hashtags (audience-focused)
   - Trending hashtags (temporary, high volume)

2. Define **platform-specific hashtag count**:
   - Instagram: 5-10 hashtags (sweet spot, not 30 like 2019)
   - TikTok: 3-5 hashtags (fewer, more targeted)
   - LinkedIn: 3-5 hashtags (professional, not spammy)
   - X/Twitter: 1-3 hashtags (space-constrained, inline)
   - YouTube: 3-5 hashtags (in description, first 3 appear above title)

3. Design **hashtag mix ratio** (based on current platform algorithms, 2026):
   - 1-2 Branded (e.g., #YourBrandName, #YourMethod)
   - 4-6 Niche (e.g., #nutritioncoach, #fatlossjourney, #macrotracking)
   - 1-2 Trending/Broad (e.g., #fitnessmotivation, #healthylifestyle)
   - 0-1 Community (e.g., #fitfam, #gymmemes — only if authentic)

4. Map **hashtags to content pillars**:
   - Education content → #howtips, #didyouknow, #mythbusting
   - Transformation content → #beforeandafter, #results, #journey
   - Community content → #clientwin, #testimonial, #community

5. Determine **hashtag placement**:
   - **End of Caption (Standard):** After CTA, separated by line break + dot/dash
   - **Inline (Alternative):** For X/Twitter where hashtags are part of sentence
   - **First Comment (Instagram Option):** Keep caption clean, hashtags in first comment

6. Research **current trending hashtags** in vertical (2026 trends):
   - Check target platform's trending/explore page
   - Identify 2-3 relevant trending hashtags
   - Avoid jumping on unrelated trends

7. Generate hashtag strategy document with 8-12 hashtag options for Layer 2 selection

**NEVER:**
- Use 30 hashtags (Instagram 2019 strategy — no longer works)
- Use only mega hashtags (10M+ posts — too competitive, low discovery)
- Use only branded hashtags (no discovery potential)
- Copy competitor hashtags without relevance check
- Use banned/restricted hashtags (reduces reach)
- Hashtag spam (all trending, no branded/niche)

## HASHTAG RESEARCH NOTES (2026)

**Current Trends in Fitness/Nutrition Vertical:**
- "Macro tracking" gaining popularity vs "calorie counting"
- "Nutrition myths" performing well (educational, not prescriptive)
- Mega hashtags (#fitness, #health) declining in effectiveness
- Niche hashtags (50K-500K) = sweet spot for discovery

**Banned/Restricted Hashtags to Avoid:**
- #weightloss (sometimes restricted)
- #diet (sometimes shadowbanned)
- **Strategy:** Use alternative phrasing (#fatlossjourney, #nutritioncoach)

**Hashtag Performance Tracking:**
- Use Instagram Insights to track which hashtags drive reach
- Replace underperforming hashtags monthly
- Test new niche hashtags quarterly
