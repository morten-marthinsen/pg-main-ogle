# 1.2 Caption Structure Strategy

**Layer:** 1 (Strategy)
**Model:** Claude Sonnet
**Upstream:** 1.1 Platform Caption Analysis
**Downstream:** 1.3 Hook Strategy

---

## PURPOSE

Define the structural blueprint for the caption: hook/body/CTA architecture, length strategy (short vs long-form), line break placement, and section transitions. This blueprint guides Layer 2 writing execution.

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  platform_analysis: "outputs/layer-1/1.1-platform-caption-analysis.md"
  content_function: "Educate" | "Entertain" | "Inspire" | "Convert" | "Nurture"
  content_type: "Reel" | "Static Post" | "Story" | "Carousel" | "Video"
  cbf_extraction: "outputs/layer-0/0.4-cbf-extraction.yaml"
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Define **3-part structure** (universal across platforms):
   - **Hook (15-125 chars):** Stops scroll, creates curiosity/value before truncation
   - **Body (100-1500 chars):** Delivers on hook promise, provides value/story/insight
   - **CTA (20-80 chars):** Action prompt matched to content_function intensity
2. Determine **length strategy** based on content_function + content_type:
   - **Short (200-500 chars):** Entertainment content, Reels, quick tips
   - **Medium (500-1200 chars):** Most educational content, storytelling
   - **Long (1200-2000 chars):** Deep dives, case studies, comprehensive guides
3. Plan **line break placement**:
   - Frequency: Every 1-2 sentences (or ~40-60 chars)
   - Function: Create visual "paragraphs", improve scannability
   - Platform consideration: Instagram/LinkedIn support native breaks, TikTok limited
4. Design **section transitions**:
   - Hook → Body: Seamless (no "In this post..." meta-narration)
   - Body → CTA: Natural (not jarring shift from story to hard sell)
   - Use emoji or spacing to signal transition
5. Map **formatting elements**:
   - Emoji usage: Count and function (bullets, emphasis, celebration)
   - List structure: If applicable (1., 2., 3. or 🔹 bullets)
   - Spacing: Blank lines between sections
6. Generate structural blueprint document

**NEVER:**
- Use identical structure for all content types (Reel caption ≠ carousel caption)
- Ignore platform truncation points in hook length
- Skip line break strategy (wall of text kills engagement)
- Design structure without CTA (every caption needs action prompt)
- Use overly complex structures (more than 3 main sections = confusing)

## OUTPUT

**File:** `outputs/layer-1/1.2-caption-structure-strategy.md`

**Format:**
```markdown
# Caption Structure Strategy — Instagram Educational Reel

**Platform:** Instagram
**Content Type:** Reel (Educational)
**Content Function:** Educate
**Length Strategy:** Medium (700-900 chars)

---

## 3-PART STRUCTURE

### HOOK (60-80 chars)
**Function:** Stop scroll with surprising insight or bold claim before "...more" (125 char truncation)

**Structure:**
- First sentence = complete thought (no cliffhanger that requires "more")
- Deliver micro-value or create curiosity gap
- Use power words or specificity

**Example Pattern (from specimens):**
"This nutrition myth is costing you results. ⚠️"
[72 chars — complete thought, creates curiosity, emoji emphasis]

---

### BODY (500-700 chars)
**Function:** Deliver on hook promise, provide educational value, build credibility

**Structure:**
1. **Expand Hook (100-150 chars):** Reveal the myth or problem
   - Line break after hook
   - 1-2 sentences explaining the issue

2. **Deliver Value (300-400 chars):** The truth or solution
   - Use numbered list OR bullet emojis (🔹) for scannability
   - 3-5 specific, actionable points
   - Line break between each point

3. **Credibility Signal (100-150 chars):** Why this matters
   - Brief "why" or "what this means for you"
   - Optional: Personal experience or client result
   - Line break before CTA

**Formatting:**
- Line break every 1-2 sentences
- Use 🔹 emoji as visual bullets for list items
- One blank line between major sections

---

### CTA (60-80 chars)
**Function:** Medium intensity — encourage saves and engagement

**Placement:** After body, before hashtags

**Structure:**
- One blank line before CTA
- CTA line: "Save this to remember next time you're meal planning."
- Optional engagement prompt: "What myth have you heard? 👇"

**Intensity:** Medium (not asking for purchase/DM, but more than "let me know thoughts")

---

## LENGTH STRATEGY

**Target Range:** 700-900 chars total
- Hook: 60-80 chars
- Body: 500-700 chars
- CTA: 60-80 chars
- Hashtags: 80-120 chars (separate section)

**Rationale:**
- Reel captions are read WHILE watching video (can't be too long)
- Educational content needs enough space for value delivery
- Medium length optimizes for saves (longer than entertainment, shorter than deep-dive posts)

---

## LINE BREAK STRATEGY

**Frequency:** Every 1-2 sentences (avg 50 chars)

**Function:**
- Create visual "breathing room"
- Improve mobile readability (80% of Instagram views)
- Signal section transitions

**Placement:**
- After hook (1 line break)
- Between list items (1 line break)
- Before CTA (2 line breaks for visual separation)

---

## EMOJI USAGE

**Count:** 4-6 emojis
**Function:**
- Hook emoji (⚠️): Emphasis or tone
- Body emojis (🔹 x3): Visual bullets for list
- CTA emoji (👇): Engagement prompt

**Avoid:** Random emoji, overuse (10+), emoji that confuse message
```
