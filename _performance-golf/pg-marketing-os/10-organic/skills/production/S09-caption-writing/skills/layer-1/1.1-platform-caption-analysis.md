# 1.1 Platform Caption Analysis

**Layer:** 1 (Strategy)
**Model:** Claude Sonnet
**Upstream:** 0.4 CBF Loader
**Downstream:** 1.2 Caption Structure Strategy

---

## PURPOSE

Analyze target platform's caption best practices, character limits, truncation points, formatting norms, and algorithmic preferences. This analysis guides all downstream caption decisions (length, formatting, hashtag usage, CTA placement).

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  platform: "Instagram" | "TikTok" | "LinkedIn" | "X" | "YouTube"
  specimen_references: "outputs/layer-0/0.3-specimen-references.md"
  cbf_extraction: "outputs/layer-0/0.4-cbf-extraction.yaml"
  content_function: "Educate" | "Entertain" | "Inspire" | "Convert" | "Nurture"
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Document platform-specific caption constraints:
   - **Character Limit:** Hard maximum (e.g., Instagram 2200, X 280)
   - **Truncation Point:** Where caption gets cut with "...more" (Instagram 125 chars, TikTok 70 chars)
   - **Optimal Length Range:** Based on content_function (educational longer, entertainment shorter)
2. Analyze formatting norms from specimen references:
   - **Line Break Support:** Native (Instagram, LinkedIn) vs limited (TikTok, X)
   - **Line Break Frequency:** Average from specimens (every 40 chars, every sentence, etc.)
   - **Emoji Usage Norms:** Count, placement, function
3. Document hashtag strategy for platform:
   - **Hashtag Count:** Platform sweet spot (Instagram 5-10, TikTok 3-5, LinkedIn 3-5, X 1-3)
   - **Hashtag Placement:** End of caption vs inline
   - **Hashtag Mix:** Branded/niche/trending ratio
4. Identify CTA placement norms:
   - End of caption (most common)
   - Mid-caption (for longer posts)
   - First comment (some platforms)
5. Analyze algorithmic preferences from recent platform behavior:
   - Engagement signals prioritized (saves, shares, comments)
   - Content that gets suppressed (external links in caption vs bio)
6. Generate platform analysis document with actionable guidelines

**NEVER:**
- Use outdated platform limits (verify current constraints)
- Ignore truncation points (critical for hook effectiveness)
- Apply Instagram norms to all platforms (each has unique culture)
- Skip emoji/formatting analysis from specimens
- Recommend practices that violate platform TOS

## CHARACTER CONSTRAINTS

| Metric | Value | Implication |
|--------|-------|-------------|
| Hard Limit | 2,200 chars | Can write long-form captions |
| Truncation Point | 125 chars ("...more") | Hook MUST deliver value before this |
| Optimal Range (Educate) | 1,200-1,800 chars | Detailed enough for value, not overwhelming |
| First Line Importance | CRITICAL | Determines whether user taps "...more" |

---

## FORMATTING NORMS

**Line Breaks:**
- Native support: YES (press return)
- Frequency from specimens: Every 1-2 sentences (avg 45 chars between breaks)
- Strategy: Use line breaks to create visual "paragraphs" — easier to scan

**Emoji Usage:**
- Count: 3-7 per caption (educational content)
- Function: Visual bullets (• → 🔹), emphasis (💡), celebration (✨)
- Placement: Start of line for bullets, inline for emphasis
- Avoid: Emoji overload (10+), random emoji without purpose

---

## HASHTAG STRATEGY

**Count:** 5-10 hashtags
- 1-2 branded (e.g., #YourBrandName, #YourSignatureMethod)
- 4-6 niche (e.g., #nutritioncoach, #healthymealprep)
- 1-2 trending/broad (e.g., #fitnessmotivation, #healthylifestyle)

**Placement:** End of caption (after line break or dot separator)

**Current Algorithm Notes (2026):**
- Hashtags still drive discovery but less than 2023
- Niche hashtags (10K-500K posts) perform better than mega hashtags (10M+)
- Branded hashtags build community

---

## CTA PLACEMENT & INTENSITY

**Placement:** End of caption (after main content, before hashtags)

**Intensity by Content Function:**
- Educate → Medium CTA ("Save this for later", "Share with someone who needs this")
- Entertain → Soft CTA ("Which one are you? Comment below")
- Inspire → Soft CTA ("Tag someone who inspires you")
- Convert → Hard CTA ("Link in bio to join", "DM me 'READY' to start")
- Nurture → Soft CTA ("What's your biggest question? Let's chat in comments")

---

## ALGORITHMIC PREFERENCES (2026)

**Prioritized Signals:**
1. **Saves** — Highest weight (indicates valuable content)
2. **Shares** — Strong signal (DM shares + story reshares)
3. **Comments** — Especially multi-turn conversations
4. **Time on Post** — How long user reads caption

**Suppressed Content:**
- External links in caption (use link in bio instead)
- Hashtag spam (>15 hashtags)
- Reposted content without original value-add

**Best Practices:**
- Ask save-worthy questions ("Save this to remember...")
- Encourage DM shares ("Send this to a friend who...")
- Spark comment conversations (open-ended questions)

---

## PLATFORM-SPECIFIC CAPTION STRATEGIES

**Instagram Reels:**
- Caption is secondary to video hook (first 3 seconds)
- Keep caption 300-800 chars (viewers watching video)
- CTA: "Follow for more" or "Save for later"

**Instagram Static Posts:**
- Caption is PRIMARY value (especially carousels)
- Can go long (1,200-2,000 chars) for educational content
- CTA: "Swipe for more" (carousel) or "Save this post"

**Instagram Stories:**
- Caption via text overlay on image (not separate field)
- Use polls, questions, sliders for engagement
- CTA: "Swipe up" (if >10K followers) or "Link in bio"

---

**Next Layer:** Use this analysis to design caption structure (1.2) and hook strategy (1.3).
