# 4.1 Output Assembler

## Purpose
Assemble final caption outputs in standardized format with platform-specific formatting validated and engagement scoring applied. Validate all requirements met before handoff.

---

## Input Requirements

From Layer 2.5:
- `arena/[caption-id]-arena-synthesis.md` (final selected captions)
- `checkpoints/ARENA_COMPLETE.yaml` (Arena verification)

From Layer 2:
- `2.4-platform-formatting.md` (platform-specific formatting specs)

From Layer 1:
- `1.5-hashtag-strategy.md` (hashtag sets)

---

## Execution Protocol

### Step 1: Load Arena-Selected Captions

For each caption, load the human-selected final version:

```yaml
final_captions:
  - caption_id: "caption_1"
    arena_file: "arena/caption-1-arena-synthesis.md"
    selected_hybrid: "Hybrid A"
    final_caption:
      hook: "[Final hook text]"
      body: "[Final body text]"
      cta: "[Final CTA text]"
      hashtags: "[Final hashtag set]"
```

---

### Step 2: Validate Platform Compliance

For each caption, verify platform-specific constraints:

```yaml
platform_validation:
  caption_id: "caption_1"
  platform: "Instagram"

  checks:
    - character_count: [count]
      limit: 2200
      status: PASS/FAIL
    - hook_before_truncation: true/false
      truncation_point: 125  # Instagram fold
    - hashtag_count: [count]
      limit: 30
      status: PASS/FAIL
    - line_breaks_correct: true/false
    - emoji_usage_appropriate: true/false
    - cta_present: true/false
```

**CRITICAL:** If hook does not land before platform truncation point, caption requires revision.

---

### Step 3: Compile Caption Package

For each caption, create complete output:

```yaml
caption_package:
  caption_id: "caption_1"
  platform: "Instagram"
  content_type: "Reel"
  content_function: "Awareness"

  caption_text: |
    [Complete formatted caption text including line breaks]

  components:
    hook: "[Hook text — must appear before fold]"
    body: "[Body text]"
    cta: "[CTA text]"
    hashtags: "[Hashtag set]"

  metadata:
    word_count: [count]
    character_count: [count]
    hashtag_count: [count]
    content_pillar: "[Pillar]"
    brand_voice_notes: "[From BVF]"
    arena_persona: "[Selected persona]"
```

---

### Step 4: Validate Completeness

```yaml
validation_checklist:
  caption_id: "caption_1"

  structural_requirements:
    - hook_present: true
    - hook_before_fold: true
    - body_complete: true
    - cta_clear: true
    - hashtags_included: true
    - platform_formatted: true

  quality_requirements:
    - arena_complete: true
    - platform_native: true
    - anti_slop_passed: true
    - voice_alignment: true

  all_checks_passed: true
```

**If any check fails, fix before output.**

---

### Step 5: Generate Summary

```yaml
campaign_summary:
  campaign_name: "[Campaign]"
  total_captions: [count]

  captions_by_platform:
    instagram: [count]
    tiktok: [count]
    youtube: [count]
    linkedin: [count]
    x: [count]

  captions_by_function:
    awareness: [count]
    engagement: [count]
    conversion: [count]
    community: [count]

  all_platform_compliant: true
  all_arena_complete: true
```

---

## Output

**File:** `outputs/[campaign]-captions.yaml`

- [ ] All captions from Arena loaded

- [ ] Caption packages complete
- [ ] Validation checklist passed for all captions

## Handoff

Pass complete caption package to: **S14 Content Assembly** and **4.2 Execution Log**
