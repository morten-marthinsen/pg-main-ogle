# 0.4 CBF Loader

**Layer:** 0 (Foundation)
**Model:** Claude Haiku
**Upstream:** 0.3 Specimen Loader
**Downstream:** 1.1 Platform Caption Analysis

---

## PURPOSE

Extract caption-relevant context from Campaign Brief File (CBF): platform plan, voice summary, hook strategy, content pillars, and brand positioning. This context ensures captions align with broader content strategy and brand voice.

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  caption_brief: "caption-brief.yaml"
  campaign_brief_file: "CBF path from caption_brief"

cbf_sections_to_extract:
  - platform_strategy_file (PSF reference)
  - brand_voice_file (BVF reference)
  - hook_library_file (HLF reference)
  - content_pillars
  - brand_positioning
  - target_audience
  - conversion_goal
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Read `caption-brief.yaml` to get CBF path
2. Load CBF and extract:
   - **Platform Plan:** Which platforms prioritized, posting frequency, content mix
   - **Brand Voice File (BVF):** Load full BVF for voice alignment check
   - **Hook Library File (HLF):** Load for hook strategy selection
   - **Content Pillars:** 3-5 recurring themes (e.g., Education, Behind-Scenes, Social Proof, Transformation, Community)
   - **Target Audience:** Demographics, psychographics, pain points
   - **Conversion Goal:** Awareness, Consideration, Conversion stage
3. Extract voice descriptor from BVF:
   - Voice attributes (e.g., "conversational expert", "bold disruptor")
   - Anti-voice (forbidden tones/phrases)
   - Pacing notes (short punchy vs longer contemplative)
4. Map content pillar to caption content_function
5. Generate CBF extraction summary for Layer 1 strategy development

**NEVER:**
- Skip loading BVF (voice alignment mandatory)
- Ignore content pillars (caption must fit strategic themes)
- Proceed without target audience context
- Use generic brand voice assumptions

## OUTPUT

**File:** `outputs/layer-0/0.4-cbf-extraction.yaml`

**Format:**
```yaml
cbf_extraction:

  platform_strategy:
    platform: "Instagram"
    posting_frequency: "5x per week"
    content_mix:
      - Educational Reels (40%)
      - Behind-Scenes Stories (30%)
      - Social Proof Posts (20%)
      - Community Engagement (10%)

  brand_voice:
    voice_file_path: "../../voice/fitness-coach-bvf.md"
    voice_descriptor: "Empowering Coach — motivational but not pushy, science-backed but accessible, celebrates small wins"
    anti_voice:
      - "Drill sergeant aggressive"
      - "Pseudo-science claims"

    pacing: "Mix of short punchy lines and longer storytelling. Line breaks every 1-2 sentences."

  content_pillars:
    - pillar: "Education"
      caption_angle: "Myth-busting, science-backed tips, how-to guides"
    - pillar: "Transformation"
      caption_angle: "Client results, before/after context, journey stories"
    - pillar: "Community"
      caption_angle: "User-generated content features, group challenges, engagement prompts"

    demographics: "Women 25-45, busy professionals, family-focused"
    psychographics: "Want results without extreme diets, value sustainability, skeptical of fads"

      - "No time for long workouts"

**Next Layer:** Use this context in platform analysis (1.1), hook strategy (1.3), and voice alignment throughout writing.
```

**Minimum Size:** 800 chars (comprehensive CBF extraction)
