# 0.3 Specimen Loader

**Layer:** 0 (Foundation)
**Model:** Claude Haiku
**Upstream:** 0.2 Teaching Loader
**Downstream:** 0.4 CBF Loader

---

## PURPOSE

Load platform-specific caption specimens (minimum 5) that represent high-performing captions in the target vertical. These specimens provide concrete examples of hook structures, formatting, CTA integration, and voice patterns that work on the specified platform.

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  platform: "Instagram" | "TikTok" | "LinkedIn" | "X" | "YouTube"
  vertical: "From CBF or inferred from content"
  content_function: "Educate" | "Entertain" | "Inspire" | "Convert" | "Nurture"

specimen_directory: "../../caption-specimens/"
  # Organized by platform: instagram/, tiktok/, linkedin/, x/, youtube/
  # Each platform folder contains vertical-specific caption files

minimum_specimens: 5
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Navigate to `../../caption-specimens/{platform}/`
2. Load minimum 5 caption specimens matching platform
3. If vertical-specific specimens exist, prioritize those (e.g., fitness, finance, B2B SaaS)
4. For each specimen, extract:
   - **Hook structure:** How first 70-125 chars create curiosity/value
   - **Line break pattern:** Frequency and placement
   - **CTA integration:** Soft/Medium/Hard, placement (inline vs end)
   - **Hashtag strategy:** Count, branded vs trending, placement
   - **Emoji usage:** Frequency, function (punctuation vs emphasis)
   - **Character count:** Total length, platform optimization
5. Generate specimen reference document with pattern analysis
6. Tag specimens by content_function for relevance filtering

**NEVER:**
- Load fewer than 5 specimens
- Use specimens from different platform (Instagram specimen for TikTok caption)
- Skip pattern extraction (copying specimens without analysis)
- Load only same content_function (need diverse examples)
- Include specimens without clear performance indicators
