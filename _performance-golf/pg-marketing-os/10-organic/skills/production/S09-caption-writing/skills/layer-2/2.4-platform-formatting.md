## EXECUTION PROTOCOL

**MUST:**
1. Load **caption with CTA** from 2.3
2. Load **hashtag strategy** from 1.5 (8-12 hashtag options)
3. Select **7-9 hashtags** (Instagram) based on:
   - 1 Branded (always include)
   - 4-6 Niche (highest relevance to content)
   - 1-2 Trending/Broad (optional)
4. Format **hashtag placement**:
   - Add 1 blank line after CTA
   - Add separator (• or —)
   - Add hashtags in single line (Instagram) OR multi-line (LinkedIn)
5. Verify **total character count**:
   - Instagram: <2200 chars (hard limit)
   - TikTok: <4000 chars
   - LinkedIn: <3000 chars
   - X: <280 chars
   - YouTube: <5000 chars
6. Apply **platform-specific formatting adjustments**:
   - Instagram: Line breaks maintained, emoji moderate
   - TikTok: Shorten if too long (focus on hook + CTA)
   - LinkedIn: Professional spacing, emoji minimal
   - X: Compress to fit 280 chars (or split into thread)
7. Generate **final PRE-ARENA caption** file
8. Label: "PRE-ARENA — PLATFORM FORMATTED"
9. Create checkpoint file: `pre-arena-caption-draft.md`

**NEVER:**
- Exceed platform character limit (caption will be truncated/rejected)
- Use 20+ hashtags (2019 Instagram strategy, now penalized)
- Place hashtags mid-caption (breaks readability)
- Skip hashtag separator (dots or dashes improve visual clarity)
- Use banned/restricted hashtags (check platform policies)
