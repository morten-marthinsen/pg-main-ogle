# 4.2 Platform Export

## Purpose
Generate per-platform, per-carousel files ready for design handoff. Each carousel gets its own standalone file with all slides, visual direction, and production specs.

---

## Input Requirements

From 4.1:
- `outputs/[campaign]-carousels-assembled.yaml` (master assembly)

---

## Execution Protocol

### Step 1: Create Per-Carousel Files

For each carousel in the assembled package, create a standalone file:

**File:** `outputs/[carousel-id]-[platform].md`

Each file MUST contain:
1. **Header** — carousel ID, platform, template type, slide count
2. **Per-Slide Spec** — headline, body copy, visual direction, character count
3. **Design Brief** — style, colors, typography, consistency notes
4. **Production Notes** — accessibility, brand compliance, file format specs

### Step 2: Platform-Specific Formatting

**Instagram Export:**
```yaml
platform_specs:
  format: "Carousel Post"
  aspect_ratio: "1:1 (1080x1080) or 4:5 (1080x1350)"
  file_format: "PNG or JPG"
  max_slides: 10
  text_safe_zone: "10% margin all sides"
  font_minimum: "24pt for body, 36pt for headlines"
```

**LinkedIn Export:**
```yaml
platform_specs:
  format: "Document Upload (PDF)"
  aspect_ratio: "Variable (recommend 4:3 or 16:9)"
  file_format: "PDF"
  max_slides: 20
  text_safe_zone: "15% margin"
  font_minimum: "18pt for body, 28pt for headlines"
```

### Step 3: Generate Slide Index

Create master index file listing all carousel exports:

```yaml
carousel_index:
  campaign_name: "[Campaign]"
  total_files: [count]
  files:
    - file: "[carousel-1]-instagram.md"
      platform: "Instagram"
      slides: 7
      template: "Educational Listicle"
    - file: "[carousel-2]-linkedin.md"
      platform: "LinkedIn"
      slides: 12
      template: "Framework Breakdown"
```

**File:** `outputs/CAROUSEL-INDEX.md`

---

## Output

**Files:**
- `outputs/[carousel-id]-[platform].md` (one per carousel)
- `outputs/CAROUSEL-INDEX.md` (master index)

**Minimum:** 1 carousel file + index

---

## Quality Gates

- [ ] One file per carousel generated
- [ ] Platform specs included in each file
- [ ] All slides present in each file
- [ ] Visual direction complete per slide
- [ ] Index file lists all exports
- [ ] Character counts verified per platform limits

---

## Handoff

All carousel export files feed into: **S14 Content Assembly** and **4.3 Validation**
