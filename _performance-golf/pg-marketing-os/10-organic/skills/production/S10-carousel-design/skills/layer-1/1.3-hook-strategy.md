# 1.3 Hook Strategy

## Purpose
Design cover slide hook strategy that stops scroll AND creates curiosity gap for swipe-through.

## Input Requirements
- Carousel brief (topic)
- CBF hook strategy
- Platform requirements
- Teaching context (SUCCESs — Unexpected principle)

## Execution Protocol

**MUST:**
- Design cover slide hook with dual function: (1) stop scroll, (2) create curiosity gap
- Select hook type from Hook Library Framework (HLF):
  - Pattern Interrupt (unexpected angle)
  - Bold Claim (contrarian position)
  - Problem Amplification (pain awareness)
  - Curiosity Gap (information withholding)
  - Authority Signal (credibility + insight)
- Score hook strength on 10-point scale (target ≥8.0):
  - Scroll-stop power (0-5)
  - Curiosity gap strength (0-5)
- Design value promise that justifies swipe-through
- Ensure hook aligns with content pillars from CBF
- Test hook against "So what?" question
- Plan visual support for hook (image/icon/color contrast)

**NEVER:**
- Use generic hooks ("X Tips You Need", "You Won't Believe")
- Create hooks that fully satisfy curiosity (no swipe incentive)
- Mislead with clickbait (content must deliver on hook)
- Ignore scroll-stop requirement (pretty but passive hooks fail)
- Skip hook scoring

## Output
**Filename:** `hook-strategy.yaml`

**Format:**
```yaml
primary_hook:
  text: string
  hook_type: string (from HLF)
  value_promise: string
  scoring:
    scroll_stop_power: 0-5
    curiosity_gap_strength: 0-5
    total_score: 0-10
  visual_support: string
  alignment_with_pillars: [list]
alternative_hooks:
  - text: string
    hook_type: string
    score: 0-10
    rationale: string
hook_test_results:
  so_what_answer: string
  swipe_incentive: string
  scroll_stop_mechanism: string
timestamp: ISO8601
```

**Minimum size:** 1.5KB

## Quality Gates
- Primary hook score ≥8.0
- Minimum 2 alternative hooks provided
- Hook type from HLF documented
- Visual support specified
- "So what?" test passed

## Handoff
Provides hook strategy to 2.1 Cover Slide Writing and 2.5.1 Arena Submission.
