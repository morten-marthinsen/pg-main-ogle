# 2.5.3 Synthesis

## Purpose
Generate 3+ hybrid carousels pulling strongest elements from Arena submissions and present to human for selection.

## Input Requirements
- Arena Round 1 submissions (7 carousels)
- Adversarial critique (2 rounds)
- Emerging patterns (consensus strengths/weaknesses)
- Original scoring (4 dimensions)

## Execution Protocol

**MUST:**
- Analyze critique patterns to identify:
  - Strongest cover slides (hook + curiosity gap)
  - Best body slide structures (screenshot-worthiness + standalone quality)
  - Most effective CTAs (save/engagement mechanics)
  - Superior visual direction choices
- Generate minimum 3 hybrid carousels by combining strongest elements:
  - **Hybrid A:** Best overall balance across all 4 scoring dimensions
  - **Hybrid B:** Maximize save-worthiness (optimize for screenshot/reference value)
  - **Hybrid C:** Maximize engagement potential (optimize for comments/shares)
  - **Hybrid D+:** Additional hybrids exploring divergent approaches
- Each hybrid must be COMPLETE carousel (cover + body + CTA + visual direction)
- Score each hybrid on same 4 dimensions (Hook 30%, Save 30%, Platform 20%, Engagement 20%)
- Document composition: which elements from which personas
- Create comparison table for human decision
- Present to human with recommendation
- After human selection, create checkpoint: `ARENA_COMPLETE`
- Document selected carousel as FINAL

**NEVER:**
- Generate fewer than 3 hybrids
- Create incomplete hybrids (missing slides)
- Skip scoring of hybrids
- Proceed without human selection
- Forget to document element sources

## Output
**Filename:** `arena-synthesis-FINAL.yaml`

**Format:**
```yaml
hybrid_count: 3+
hybrids:
  - hybrid_id: "Hybrid A"
    strategy: "Balanced excellence"
    complete_carousel:
      cover_slide:
        headline: string
        subhead: string
        source: "Persona X cover"
        visual_direction: {}
      body_slides:
        - slide_number: N
          headline: string
          supporting_lines: [list]
          source: "Persona Y slide N"
          visual_direction: {}
      cta_slide:
        save_prompt: string
        follow_cta: string
        engagement_q: string
        source: "Persona Z CTA"
        visual_direction: {}
    scoring:
      hook_strength: 0-10
      save_worthiness: 0-10
      platform_fit: 0-10
      engagement_potential: 0-10
      weighted_total: 0-10
    composition_notes: string
  - hybrid_id: "Hybrid B"
    [...same structure]
comparison_table:
  - metric: string
    hybrid_a: string
    hybrid_b: string
    hybrid_c: string
recommendation: string (which hybrid and why)
human_selection: "Hybrid [X]"
selection_rationale: string (from human)
checkpoint: "ARENA_COMPLETE"
timestamp: ISO8601
```

**Minimum size:** 6KB (3+ complete carousels with full documentation)

## Quality Gates
- Minimum 3 hybrids generated
- All hybrids are complete carousels (not fragments)
- All hybrids scored on 4 dimensions
- Element sources documented per hybrid
- Human selection captured
- ARENA_COMPLETE checkpoint created

## Handoff
Provides FINAL selected carousel (ARENA_COMPLETE) to Layer 4 Package Assembly for client delivery.
