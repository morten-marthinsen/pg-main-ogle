# 0.2 Teaching Loader

## Purpose
Load viral content frameworks and teaching materials that inform carousel design strategy.

## Input Requirements
- Teaching file paths in organic skills protocols directory
- Validation report showing PASS status

## Execution Protocol

**MUST:**
- Load Heath brothers SUCCESs framework (Simple, Unexpected, Concrete, Credible, Emotional, Stories)
- Load Jonah Berger STEPPS (Social currency, Triggers, Emotion, Public, Practical value, Stories)
- Load Donald Miller StoryBrand framework for narrative structure
- Extract key principles applicable to carousel design
- Note practical value optimization techniques
- Document screenshot-worthiness criteria

**NEVER:**
- Skip teaching files
- Proceed without extracting actionable principles
- Load generic social media advice instead of specified frameworks
- Ignore platform-specific shareability principles

## Output
**Filename:** `teaching-context.yaml`

**Format:**
```yaml
frameworks_loaded:
  - name: "SUCCESs"
    source: "Heath & Heath"
    key_principles: [list]
    carousel_applications: [list]
  - name: "STEPPS"
    source: "Jonah Berger"
    key_principles: [list]
    carousel_applications: [list]
  - name: "StoryBrand"
    source: "Donald Miller"
    key_principles: [list]
    carousel_applications: [list]
shareability_criteria: [list]
save_triggers: [list]
timestamp: ISO8601
```

**Minimum size:** 2KB

## Quality Gates
- All 3 frameworks loaded
- Minimum 5 carousel-specific applications per framework
- Save triggers documented (minimum 8)

## Handoff
Provides framework context to 1.2 Slide Architecture and 1.4 Save Optimization.
