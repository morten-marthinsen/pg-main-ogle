# 2.4 Visual Direction

## Purpose
Specify visual direction PER SLIDE including layout, color, typography, and imagery.

## Input Requirements
- Complete slide content (cover, body, CTA) PRE-ARENA
- Specimen analysis (visual patterns)
- CBF brand constraints
- Platform requirements
- Save optimization strategy

## Execution Protocol

**MUST:**
- Create visual direction for EACH slide individually
- Specify 4 components per slide:
  1. **Layout:** Centered/left-aligned/split-screen/custom
  2. **Color Palette:** Background + text colors + accent colors (specify hex codes)
  3. **Typography:** Headline font size/weight + body font size + hierarchy
  4. **Imagery:** Photo/illustration/abstract/icon/none + placement notes
- Vary layout across slides (minimum 3 different layouts in carousel)
- Ensure high contrast for screenshot-worthiness:
  - Text-to-background contrast ratio ≥4.5:1
  - Key information highly legible
- Apply brand constraints from CBF
- Design for platform aesthetic:
  - Instagram: Bold, colorful, high visual impact
  - LinkedIn: Professional, clean, trust-building
- Create visual variety without chaos (maintain cohesion)
- Note any animation/transition suggestions (optional)

**NEVER:**
- Use same layout for all slides
- Specify low-contrast combinations
- Ignore brand guidelines
- Create platform-inappropriate aesthetics
- Skip typography specifications
- Forget hex codes for colors

## Output
**Filename:** `visual-direction.yaml`

**Format:**
```yaml
overall_aesthetic: string
brand_compliance: [constraints applied]
per_slide_direction:
  - slide_number: 1
    layout_type: string
    color_palette:
      background: "#hexcode"
      headline_text: "#hexcode"
      body_text: "#hexcode"
      accent: "#hexcode"
    typography:
      headline_size: string
      headline_weight: string
      body_size: string
      hierarchy_notes: string
    imagery:
      type: "photo | illustration | abstract | icon | none"
      placement: string
      description: string
    contrast_ratio: number
    screenshot_optimized: true/false
  - slide_number: 2
    [...same structure]
layout_variety_check:
  unique_layouts_used: number
  distribution: [list]
cohesion_strategy: string
animation_suggestions: [optional list]
timestamp: ISO8601
```

**Minimum size:** 3KB (for 5-10 slides)

## Quality Gates
- Visual direction specified for ALL slides
- Minimum 3 different layouts used
- All contrast ratios ≥4.5:1
- Hex codes provided for all colors
- Typography hierarchy specified
- Brand compliance verified

## Handoff
Provides complete visual specifications to 2.5.1 Arena Submission and downstream to design/production.
