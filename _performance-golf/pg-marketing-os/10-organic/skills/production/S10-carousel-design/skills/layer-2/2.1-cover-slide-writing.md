# 2.1 Cover Slide Writing

## Purpose
Write the cover slide (THE most critical slide) that stops scroll and drives swipe-through.

## Input Requirements
- Hook strategy (primary hook + alternatives)
- CBF voice summary
- Platform requirements (character limits)
- Slide architecture

## Execution Protocol

**MUST:**
- Write cover slide with 3 components:
  1. **Hook:** Primary hook from 1.3 (scroll-stopper)
  2. **Curiosity Gap:** Information withholding that creates swipe incentive
  3. **Value Promise:** Clear benefit of full carousel
- Respect platform character limits:
  - Instagram: <100 char headline, optional <50 char subhead
  - LinkedIn: <150 char headline
- Apply voice from CBF (tone, language level, key phrases)
- Design visual direction for cover:
  - Background (solid color/gradient/image)
  - Text placement (centered/left/custom)
  - Visual element (icon/image/pattern)
  - Color contrast requirements
- Generate 3-5 cover slide variations
- Label all versions PRE-ARENA
- Score each variation on hook strength (use 1.3 scoring)

**NEVER:**
- Exceed character limits
- Reveal too much (kill curiosity gap)
- Use generic value promises ("Learn more inside")
- Skip visual direction specifications
- Generate fewer than 3 variations
- Forget PRE-ARENA label

## Output
**Filename:** `cover-slide-PRE-ARENA.yaml`

**Format:**
```yaml
version_count: 3-5
variations:
  - version_id: "V1"
    headline: string (<limit)
    subhead: string (optional)
    hook_type: string
    curiosity_gap: string
    value_promise: string
    visual_direction:
      background: string
      text_placement: string
      visual_element: string
      color_scheme: [colors]
    scoring:
      scroll_stop: 0-5
      curiosity_gap: 0-5
      total: 0-10
  - version_id: "V2"
    [...same structure]
best_performer_prediction: "version_id"
rationale: string
voice_alignment: string
timestamp: ISO8601
```

**Minimum size:** 2.5KB

## Quality Gates
- Minimum 3 variations generated
- All variations within character limits
- Visual direction specified for each
- All versions labeled PRE-ARENA
- Hook strength scored per variation

## Handoff
Provides cover slide variations to 2.5.1 Arena Submission as critical evaluation component.
