# Microskill 4.5: Distinctiveness Pass (Organic)

**Layer:** 4 (Fix Application)
**Purpose:** Detect whether content assembly has drifted organic posts toward generic "social media marketing" language that could belong to any brand
**Sequence:** Runs AFTER content package assembly (4.1), platform file creation (4.2), and execution log (4.3)

---

## WHY THIS PASS EXISTS

Organic content assembly optimizes for platform specs, posting cadence, and format compliance — all pushing copy toward standard social media templates. This pass checks whether each post carries the campaign's distinctive voice and looks native to its target platform rather than cross-posted from a template.

**References:**
- `~system/humanization-specimens/SIGNATURE-MOVES.md` (Organic section)
- `~system/humanization-specimens/engine-packs/organic-pack.md`
- Campaign Soul.md
