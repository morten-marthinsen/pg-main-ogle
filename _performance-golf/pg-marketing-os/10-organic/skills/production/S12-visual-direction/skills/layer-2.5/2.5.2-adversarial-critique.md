# Microskill 2.5.2: Adversarial Critique
Skill: S12 Visual Direction
Layer: 2.5
Model: claude-opus-4.6
Version: 1.0

## Purpose

Execute Arena protocol — 7 organic personas evaluate 3 thumbnail concepts across 2 rounds with adversarial Critic challenging convergence.

## Input

**Required file:** `s12-2.5.1-arena-package.yaml`

**Required protocol:** `~system/protocols/ARENA-CORE-PROTOCOL.md`

## Process

### MUST
1. **Load Arena protocol and personas**
   - Read ~system/protocols/ARENA-CORE-PROTOCOL.md for full procedure
   - Load 7 organic personas: Volume Machine, Value Architect, Virality Engineer, Community Builder, Brand Purist, Algorithm Hacker, Storyteller

2. **Execute 2-round + audience evaluation Arena**
   - Round 1: All 7 personas evaluate 3 concepts, score + rationale
   - Critic: Challenge Round 1 scores, identify bias/convergence
   - Round 2: Personas respond to critique, re-evaluate
   - Critic: Challenge Round 2, push for differentiation
   - Round 2 (FINAL): Final scores, forced ranking

3. **Document all evaluations**
   - Per-persona scores (0-10) for each concept
   - Rationale for scores
   - Critic interventions and persona responses
   - Evolution across rounds

### NEVER
- Skip rounds (2 rounds mandatory)
- Allow persona convergence without Critic challenge
- Proceed without human selection
