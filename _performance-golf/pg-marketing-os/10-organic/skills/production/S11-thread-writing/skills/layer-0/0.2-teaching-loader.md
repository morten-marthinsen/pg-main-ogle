# 0.2 Teaching Loader

## Purpose
Load relevant teachings on virality, audience psychology, and platform mechanics that inform thread quality.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (confirms execution can proceed)

---

## Execution Protocol

### Step 1: Load Core Teachings

Required teachings for thread writing:

```yaml
virality_frameworks:
  - file: "teachings/virality/berger-stepps-framework.yaml"
    relevance: "Social currency, emotion, practical value"
    key_concepts: ["STEPPS", "Social Currency", "Emotion", "Public"]

  - file: "teachings/virality/heath-success-framework.yaml"
    relevance: "SUCCESs framework for sticky ideas"
    key_concepts: ["Simple", "Unexpected", "Concrete", "Credible", "Emotional", "Stories"]

audience_psychology:
  - file: "teachings/audience-psychology/cialdini-influence-principles.yaml"
    relevance: "Persuasion triggers in thread content"
    key_concepts: ["Reciprocity", "Social Proof", "Authority", "Scarcity"]
```

### Step 2: Extract Key Principles

For each teaching, extract:
- Core framework components
- Application to thread writing
- Common violations to avoid
- Quality benchmarks

### Step 3: Create Teaching Reference

Compile loaded teachings into quick-reference format:

```yaml
teaching_reference:
  virality_principles:
    - principle: "Emotional Activation"
      source: "Berger STEPPS"
      application: "Hook tweet must trigger emotion: surprise, anger, awe, anxiety"

    - principle: "Social Currency"
      source: "Berger STEPPS"
      application: "Thread makes reader look smart/informed when sharing"

    - principle: "Practical Value"
      source: "Berger STEPPS"
      application: "Actionable insights, not just interesting facts"

  stickiness_principles:
    - principle: "Simple"
      source: "Heath SUCCESs"
      application: "One core message per tweet, no complexity"

    - principle: "Unexpected"
      source: "Heath SUCCESs"
      application: "Pattern interrupt in hook, contrarian angles"

    - principle: "Concrete"
      source: "Heath SUCCESs"
      application: "Specific numbers, examples, not abstractions"

  persuasion_principles:
    - principle: "Social Proof"
      source: "Cialdini"
      application: "Reference popularity, others doing it"

    - principle: "Authority"
      source: "Cialdini"
      application: "Credibility signals in hook or tweet 2"
```
