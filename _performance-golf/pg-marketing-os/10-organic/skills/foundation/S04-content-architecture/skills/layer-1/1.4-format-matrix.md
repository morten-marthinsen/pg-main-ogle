# 1.4 Format Matrix

## Purpose
Define all content formats in use with platform-specific variations and pillar fit scores. This is Phase B (after concept approval).

---

## Input Requirements
- Approved pillar concepts (from 1.1 + CONCEPT_APPROVED.yaml)
- Function mapping from 1.2
- Platform context from PSF

---

## Execution Protocol

### Step 1: Identify Formats from PSF
Extract optimal formats from PSF:
- Primary platform formats
- Secondary platform formats
- Creator-preferred formats (from business context)

**Minimum 3 formats required.**

---

### Step 2: Define Format Variations
For each format, define platform-specific variations:

```yaml
format_matrix:
  short_form_video:
    platforms: ["Instagram Reels", "TikTok", "YouTube Shorts"]
    length_range: "15-90 seconds"
    production_time: "1-3 hours"
    frequency: "[Per week]"

    variations:
      - name: "Educational Quick Hit"
        description: "Single concept, fast delivery, high value"
        use_case: "Awareness and engagement"

      - name: "Behind-the-Scenes"
        description: "Process reveal, authentic moments"
        use_case: "Community and engagement"

      - name: "Problem-Solution"
        description: "Hook with problem, quick solution, CTA"
        use_case: "Conversion"

    pillar_fit:
      - pillar: "[Pillar name from approved concepts]"
        fit_score: [1-10]
        rationale: ""
```

---

### Step 3: Score Pillar Fit
For each format, score fit (1-10) with each pillar:
- **8-10:** Excellent fit, prioritize this format for this pillar
- **5-7:** Good fit, use occasionally
- **1-4:** Poor fit, avoid or use rarely

**Scoring criteria:**
- Does format match pillar content depth?
- Does format suit pillar topics?
- Does platform audience expect this pillar in this format?

---

### Step 4: Define Production Requirements
For each format, specify:
- Length range (time/word count/slides)
- Production time estimate
- Frequency target (per week/month)
- Equipment/tools needed
- Skill level required

---

### Step 5: Create Format-to-Function Map
Map which formats serve which functions best:

| Format | Awareness | Engagement | Conversion | Community |
|--------|-----------|------------|------------|-----------|
| Short video | High | High | Medium | Medium |
| Carousel | High | Medium | High | Low |
| Thread | Medium | High | Medium | High |
