# 1.5 Competitor Audience Analysis

## Purpose
Understand who else has their attention and WHY. Identifies gaps to exploit and patterns to avoid.

---

## Input Requirements
- competitive_context from input validation (0.1)
- platform-behavior.yaml (from 1.3) to know where to look

---

## Execution Protocol

### Step 1: Identify Competitors to Analyze
```
FROM competitive_context.direct_competitors:
  SELECT ≥3 competitors for deep analysis

CRITERIA for selection:
  - Similar audience size (relatable, not aspirational only)
  - Active on same platforms
  - Selling similar products/services
  - Strong engagement (proof they're doing something right)

IF <3 direct competitors available:
  ADD adjacent_competitors or aspirational_comparisons to reach 3 minimum
```

### Step 2: Analyze Each Competitor Account
```
FOR EACH competitor (≥3 required):

DOCUMENT:
  name: "[Account name]"
  platform: "[Primary platform]"
  follower_count: [Number]
  engagement_rate: [% if calculable]

  why_they_follow:
    - Value provided (education/entertainment/inspiration)
    - Content format they excel at
    - Personality/voice appeal
    - Community they've built
    - Results they showcase

  content_style:
    - Format preferences (video/carousel/text)
    - Posting frequency
    - Content themes
    - Hook patterns
    - Voice characteristics

  gap:
    - What's missing from their content
    - Complaints in comments
    - Unfulfilled audience needs
    - Topics they avoid
    - Formats they underutilize
```

### Step 3: Cross-Competitor Pattern Analysis
```
AFTER analyzing all competitors:

IDENTIFY patterns across ALL:
  - Common value propositions
  - Common content formats
  - Common audience pain points addressed
  - Common audience pain points IGNORED

SYNTHESIZE:
  why_they_follow: "[What ALL successful competitors provide]"
```

### Step 4: Gap Identification
```
GAPS TO EXPLOIT:

UNDERSERVED NEEDS:
  - What the audience wants but no competitor provides
  - Pain points consistently ignored
  - Questions repeatedly asked but not answered

FORMAT GAPS:
  - Content formats audience engages with elsewhere but competitors don't use
  - Platform opportunities (they're all on X but audience is on TikTok)

TOPIC GAPS:
  - Adjacent topics audience cares about but no one covers
  - Depth gaps (everyone does surface, no one goes deep)

VOICE GAPS:
  - Personality/tone not represented
  - Demographic representation missing
  - Perspective/angle unused
```

---

## Output
- **File:** `competitor-audience.yaml`
- **Format:** YAML
- **Minimum size:** 2KB

```yaml
competitors_they_follow:
  accounts:
    - name: "[Account name]"
      platform: "[Platform]"
      follower_count: [number]
      engagement_rate: "[% if known]"
      why_they_follow:
        - "[Value 1]"
        - "[Value 2]"
      content_style:
        format_preferences: []
        posting_frequency: "[Pattern]"
        voice_characteristics: "[Description]"
      gap:
        missing_content: "[What they don't provide]"
        audience_complaints: "[What followers want]"
    # MINIMUM 3 REQUIRED

  cross_competitor_patterns:
    why_they_follow: "[Common value across all competitors]"
    common_formats: []
    common_pain_points_addressed: []
    common_pain_points_ignored: []

  gaps_to_exploit:
    underserved_needs:
      - need: "[What's missing]"
        evidence: "[How we know]"
    format_gaps:
      - gap: "[Format/platform opportunity]"
        rationale: "[Why it's a gap]"
    topic_gaps:
      - gap: "[Topic/depth opportunity]"
        rationale: "[Why it matters]"
    voice_gaps:
      - gap: "[Personality/perspective missing]"
        opportunity: "[How we differentiate]"

verification:
  competitors_analyzed: [count]
  minimum_met: true | false  # MUST be true (≥3)
```

---

## Quality Gates
- [ ] ≥3 competitor accounts analyzed
- [ ] Each competitor has why_they_follow populated with ≥2 items
- [ ] Each competitor has gap identified
- [ ] gaps_to_exploit has ≥1 entry in each category (underserved/format/topic/voice)
- [ ] Cross-competitor patterns synthesized

**CRITICAL:** If competitors_analyzed < 3, this is a BLOCKING FAILURE.
