# 0.1 Input Validator

## Purpose
Validate that all 6 foundation outputs from S01-S06 exist and contain the necessary data for campaign brief synthesis.

---

## Input Requirements

Required files from upstream skills:
- `outputs/[campaign]-AIF.yaml` (S01: Audience Intelligence)
- `outputs/[campaign]-PSF.yaml` (S02: Platform Strategy)
- `outputs/[campaign]-BVF.yaml` (S03: Brand Voice)
- `outputs/[campaign]-CAF.yaml` (S04: Content Architecture)
- `outputs/[campaign]-HLF.yaml` (S05: Hook Library)
- `outputs/[campaign]-VSF.yaml` (S06: Virality Scoring)

---

## Execution Protocol

### Step 1: File Existence Check

Verify all 6 foundation files exist:

```yaml
file_check:
  AIF: [exists: true/false]
  PSF: [exists: true/false]
  BVF: [exists: true/false]
  CAF: [exists: true/false]
  HLF: [exists: true/false]
  VSF: [exists: true/false]
```

**BLOCKER:** If ANY file missing, HALT and return to foundation skills.

---

### Step 2: Chain Validation

Verify each file passed its own gate:

**From AIF (Gate G01):**
- `target_segment` (not empty)
- `primary_pain` (defined)
- `primary_desire` (defined)
- `language_to_use` (at least 5 phrases)

**From PSF (Gate G02):**
- `primary_platform` (defined)
- `posting_cadence` (defined)
- `optimal_times` (at least 2)
- `content_formats` (at least 2)

**From BVF (Gate G03):**
- `voice_dimensions` (complete)
- `tone_spectrum` (defined)
- `signature_phrases` (at least 3)

**From CAF (Gate G04):**
- `content_pillars` (at least 1, target 3-5)
- `content_function_mix` (percentages defined)
- `funnel_connection` (defined)

**From HLF (Gate G05):**
- `hook_types` (at least 3)
- `platform_specific_hooks` (at least 5)
- `hook_templates` (at least 3)

**From VSF (Gate G06):**
- `dimensions` (all 8 defined)
- `benchmarks.minimum_publish` (defined)
- `benchmarks.target` (defined)
- `quick_score.must_haves` (at least 5)

---

### Step 3: Completeness Check

For each file, verify it's not a stub or incomplete:

```python
def check_completeness(file_data):
    """
    Verify foundation file is complete enough for synthesis.
    """
    issues = []

    # Check file size (proxy for completeness)
    if file_size < 3KB:
        issues.append("File suspiciously small — may be incomplete")

    # Check for placeholder values
    placeholders = ["TBD", "TODO", "[name]", "[description]", "N/A"]
    if any(placeholder in str(file_data) for placeholder in placeholders):
        issues.append("File contains placeholder values")

    return issues
```
