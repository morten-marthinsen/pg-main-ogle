# 0.3 Upstream Loader

## Purpose
Load S19 PAR, learning flags, hypotheses, and current system baselines to enable Layer 1 learning categorization and updates.

## Assigned Model
`claude-haiku-4` (Layer 0 loading)

## Input
- S19 PAR file
- S19 learning flags (1.5-learning-flags.json)
- S19 hypotheses (1.6-hypotheses.json)
- S19 pattern detection (1.4-pattern-detection.json)
- Current system baselines (specimen counts, teaching versions)

## Process

### MUST
1. **Load S19 PAR** — Full performance analysis report with executive summary, patterns, variance analysis
2. **Load learning flags** — All flags with categories, priorities, confidence, evidence
3. **Load hypotheses** — Testable hypotheses with test designs and success criteria
4. **Load pattern detection** — Detected patterns with sample sizes and confidence
5. **Load current system baselines:**
   - Specimen library counts per category
   - Teaching file versions
   - Gate threshold values
   - Last learning capture date
6. **Extract key metrics** — Content analyzed count, variance distribution, pattern count
7. **Identify pending learnings** — Any learnings from previous campaigns marked "deferred"
8. **Output upstream package** — Ready for Layer 1 processing

### NEVER
- Proceed without full PAR
- Skip hypotheses loading
- Ignore pending learnings from previous campaigns
- Omit current baselines (needed for before/after comparison)
