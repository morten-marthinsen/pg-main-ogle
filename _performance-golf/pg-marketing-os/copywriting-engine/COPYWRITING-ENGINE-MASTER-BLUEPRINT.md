## CORE INNOVATIONS

### RSF Framework (Resonant Surprise Framework)

The RSF Framework captures the meta-architecture of elite persuasion:

| Component | What It Captures | Integration |
|-----------|------------------|-------------|
| **R - Relatable Problem** | Primary problem in prospect language, emotional amplifiers, specificity, relatability triggers | root_cause_architecture, villain_profile, lead |
| **S - Specific Solution** | Core mechanism, proof points, differentiators, tactical breakdown | mechanism, proof_inventory, story_architecture |
| **F - Freedom Promise** | Primary transformation, specific outcomes, freedom language, timelines | promise_architecture, offer, close |

All TIER1 extractions include RSF analysis. See `VAULT_V3_SCHEMA.md` section 3 for full documentation.

### Arena Layer 2.5 (Multi-Perspective Generation)

The Arena Layer transforms single-perspective generation into multi-perspective competition, positioned between Layer 2 (drafting) and Layer 3 (validation) in narrative skills.

**The 6 Arena Personas:**

| Persona | Editorial Lens | Key Question |
|---------|---------------|--------------|
| **Makepeace** | Flow & Architecture | "Does it pull the reader forward?" |
| **Halbert** | Entertainment & Personality | "Would they stop scrolling for this?" |
| **Schwartz** | Sophistication Calibration | "Is this fresh for THIS market?" |
| **Ogilvy** | Credibility & Clarity | "Would a skeptic accept this?" |
| **Craig Clemens** | Scientific Clarity | "Would a 12-year-old understand this?" |
| **Bencivenga** | Proof-First Authority | "Can every element be proven?" |

**7 Judging Criteria:**
- Issue Resolution (20%)
- Voice Preservation (20%)
- Flow Enhancement (15%)
- Clarity Improvement (15%)
- Slop Elimination (10%)
- Brevity (10%)
- Threading Preservation (10%)

**Minimum Scores:** Overall ≥8.5, Voice ≥7.0, Threading ≥7.0, Issue ≥7.0

### Dual-System Specimen Architecture

**SYSTEM 1 (ACTIVE): Type-Indexed Structural Pattern Loading**
- All 6 personas load the SAME type-matched specimens from `0.2.6-curated-gold-specimens.md`
- Specimens indexed by TYPE (story type, issue type, proof type), not by persona
- Each persona applies their EDITORIAL LENS to the same foundation

**SYSTEM 2 (FUTURE): Persona Voice Loading**
- Will load actual copy from each specific writer
- Not yet implemented — requires extraction of correctly-attributed specimens

## PHASE 1: STRATEGIC CLARITY (Skills 01-06)

**Purpose:** Build the strategic foundation before generating any copy.

**Operating Principle:** World-class copywriters achieve clarity BEFORE generation.

```
01-RESEARCH
│   Market intelligence, avatar, competition, Schwartz stage
│   Output: Market Psychology Snapshot, Competitive Audit, Belief Map
│
├──→ 02-PROOF-INVENTORY
│       Extract, classify, score all proof elements
│       Output: Ranked proof catalog, promise ceiling, knockout proof
│
├──→ 03-ROOT-CAUSE [ARENA LAYER]
│       Why the problem REALLY persists (not what they think)
│       Output: Surface vs real cause, villain, reframe technique
│
└──→ 04-MECHANISM [ARENA LAYER]
        The unique solution pathway — HOW it works
        Output: Mechanism name, type, analogy, 7+ features, scorecard
            │
            ├──→ 05-PROMISE [ARENA LAYER]
            │       What we can credibly claim (constrained by proof)
            │       Output: Primary promise, campaign thesis, variations
            │
            └──→ 06-BIG-IDEAS [ARENA LAYER]
                    Central compelling concept synthesized from all above
                    Output: Big Idea statement, headlines, leads
```
