## STRATEGIC APPLICATIONS

### Template Patterns to Extract

#### 1. Price Psychology Template
```
[PREMIUM PRODUCT] for just [DAILY COST] per day
- Break monthly price into daily amount
- Compare to minimal daily expense (coffee, parking, snack)
- Contrast against expensive alternatives (doctor visits, multiple supplements)
- Emphasize "pennies a day" language
```

**Example:** "$97/month" becomes "just $3.23 per day—less than a cup of coffee"

#### 2. Mechanism Reframe Template
```
What They Think: [COMMON BELIEF]
What It Really Is: [SCIENTIFIC REFRAME]
Why Nothing Worked: [SPECIFIC REASON + NOT YOUR FAULT]

Backed by: [AUTHORITATIVE SOURCE]
```

**Example:**
- Think: Vitamins come in pills
- Really: Liquid absorbs at 98% vs pills at 10-20%
- Why: You were using wrong delivery system (not your fault)
- Source: Physicians Desk Reference

#### 3. Compilation Dossier Story Template
```
[5 CHARACTERS] representing [5 DIFFERENT PROBLEMS]
Each discovers [SOLUTION]
Each achieves [SPECIFIC OUTCOME]
All now part of [LARGE NUMBER] success stories

Result: Universal appeal—every prospect sees their problem
```

#### 4. Proof Sequence Template
```
1. Headline: Credibility stack (numbers, authority, validation)
2. Lead: Rapid-fire testimonials (diverse benefits)
3. Mechanism: Scientific validation (authoritative source)
4. Body: Extended stories (emotional depth)
5. Close: Risk reversal (guarantee)
```

#### 5. Testimonial Specificity Template
```
[NAME] achieved [SPECIFIC NUMBER] in [SPECIFIC TIMEFRAME]
- NOT: "John lost weight"
- YES: "John lost 47 pounds in 90 days"

Add authority when possible:
- "Registered nurse Joan..."
- "Engineer Mike who tried everything..."
```
