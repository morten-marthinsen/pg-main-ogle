# 1.2 Testimonial Extraction

## Purpose
Extract individual proof elements from testimonial files, capturing the full richness of customer stories, specific results, and attribution data. Testimonials are the primary source of SOCIAL proof.

---

## Input Requirements

From 1.1 Source Parsing:
- Parsed testimonial file segments
- Delimiter positions
- Attribution markers identified

---

## Testimonial Anatomy

A complete testimonial contains up to 7 extractable components:

```
TESTIMONIAL COMPONENTS:
├── QUOTE TEXT (Required)
│   └── The actual testimonial content
│
├── SPECIFIC RESULT (High value)
│   └── Numbers, timeframes, measurable outcomes
│   └── "Lost 27 pounds in 6 weeks"
│
├── BEFORE STATE (Context)
│   └── What their situation was before
│   └── "I had tried everything..."
│
├── AFTER STATE (Transformation)
│   └── What changed for them
│   └── "Now I feel confident at the beach"
│
├── ATTRIBUTION (Credibility)
│   ├── Full name
│   ├── Location (city, state)
│   ├── Photo indicator
│   └── Credentials if relevant
│
├── MEDIA TYPE (Format)
│   └── Written | Video | Audio | Screenshot
│
└── TIMESTAMP (Recency)
    └── Date of testimonial if available
```

---

## Execution Protocol

### Step 1: Segment Analysis

For each testimonial segment from parsing:

```
ANALYZE SEGMENT:
1. Identify quote boundaries
2. Scan for specific results (numbers, timeframes)
3. Look for transformation language (before→after)
4. Extract attribution
5. Determine media type
6. Note any timestamps
```

### Step 2: Specific Result Extraction

This is the MOST VALUABLE component. Extract with precision:

```
RESULT PATTERNS TO CAPTURE:

QUANTITY RESULTS:
├── "lost [X] pounds" → weight_loss
├── "made $[X]" → money_earned
├── "saved [X] hours" → time_saved
├── "[X] strokes off my game" → improvement_metric
└── "[X]% improvement" → percentage_gain

TIMEFRAME RESULTS:
├── "in [X] days/weeks/months"
├── "within [X] weeks"
├── "after just [X] days"
└── "in the first [X] weeks"

COMPARATIVE RESULTS:
├── "more than I lost on [other solution]"
├── "faster than anything else"
├── "for the first time in [X] years"
└── "when nothing else worked"
```

**Extraction Example:**
```
Input: "I lost 27 pounds in just 6 weeks! Nothing else worked for me
       in 15 years of trying." - Sarah M., Dallas TX

Extracted:
  specific_results:
    - type: "weight_loss"
      value: "27 pounds"
      timeframe: "6 weeks"
      qualifier: "nothing else worked in 15 years"
```

### Step 3: Transformation Narrative Extraction

Look for before/after story arc:

```
BEFORE SIGNALS:
├── "I used to..."
├── "Before I found..."
├── "I was struggling with..."
├── "I had tried everything..."
└── "I was about to give up..."

AFTER SIGNALS:
├── "Now I..."
├── "I finally..."
├── "For the first time..."
├── "I can't believe..."
└── "My life has changed..."
```

**Extraction Example:**
```
Input: "I used to dread looking in the mirror. Now I actually look forward
       to getting dressed in the morning. My confidence is through the roof!"

Extracted:
  transformation:
    before_state: "dreaded looking in mirror"
    after_state: "looks forward to getting dressed"
    emotional_shift: "confidence through the roof"
```

### Step 4: Attribution Quality Assessment

Score the attribution:

| Attribution Level | Components | Credibility Score |
|------------------|------------|-------------------|
| **Full** | Name + Location + Photo | 10 |
| **Strong** | Name + Location | 8 |
| **Partial** | First name + Initial | 6 |
| **Weak** | First name only | 4 |
| **Anonymous** | "A customer said..." | 2 |

### Step 5: Media Type Classification

```
MEDIA TYPE INDICATORS:

VIDEO:
├── "[VIDEO]", "Video testimonial"
├── "Watch [Name] explain"
├── Timestamp references
└── Visual description cues

AUDIO:
├── "[AUDIO]", "Voice recording"
├── "Hear [Name] describe"
└── Podcast/interview format

SCREENSHOT:
├── Social media format (Facebook, Instagram layout)
├── "[Screenshot]" tag
├── Platform indicators (blue checkmark, post format)

WRITTEN:
├── Standard quote format
├── Letter/email format
└── Default if no media indicators
```

## Output: Extracted Testimonial Element

```yaml
proof_element:
  id: "PROOF-001"
  category: "social"
  sub_type: "testimonial_specific_result"  # or testimonial_transformation, etc.
  sub_type_id: "SOC-02"

  raw_text: |
    "I lost 27 pounds in just 6 weeks! Nothing else worked for me
    in 15 years of trying. This is the first program that actually
    delivered on its promise." - Sarah M., Dallas TX

  extracted_components:
    quote_text: "I lost 27 pounds in just 6 weeks! Nothing else worked..."
    specific_result:
      type: "weight_loss"
      value: "27 pounds"
      timeframe: "6 weeks"
      comparative: "nothing else worked in 15 years"
    transformation:
      before_state: "15 years of failed attempts"
      after_state: "program delivered on promise"
      emotional_shift: null
    attribution:
      name: "Sarah M."
      location: "Dallas TX"
      photo: false
      attribution_level: "strong"
    media_type: "written"
    timestamp: null

  preliminary_scores:
    specificity: 9  # Exact numbers
    credibility: 8  # Named + location
    relevance: null  # Set in later layer
    novelty: null    # Set in later layer
    emotional_impact: 7  # Has comparison, moderate emotion

  source_metadata:
    file: "testimonials.md"
    segment_id: 1
    line_range: "15-18"
```

---

## Sub-Type Classification Guide

Match to most appropriate SOCIAL sub-type:

| If testimonial has... | Sub-type | ID |
|----------------------|----------|-----|
| Full before/after story arc | testimonial_transformation | SOC-01 |
| Specific numbers/results | testimonial_specific_result | SOC-02 |
| On-camera video | testimonial_video | SOC-03 |
| Voice recording | testimonial_audio | SOC-04 |
| Text only, basic praise | testimonial_written | SOC-05 |
| No name attribution | testimonial_anonymous | SOC-06 |
| Famous person | testimonial_celebrity | SOC-07 |

---

## Quality Gates

Before element is complete:

- [ ] Quote text captured completely (not truncated)
- [ ] Specific results extracted if present (don't miss numbers!)
- [ ] Attribution level assessed
- [ ] Media type classified
- [ ] Sub-type assigned
- [ ] Preliminary scores set (specificity, credibility, emotional_impact)

---

## High-Value Testimonial Indicators

Prioritize testimonials with these signals:

```
HIGH-VALUE SIGNALS:
├── Exact numbers (27 lbs, $10,000, 3 weeks)
├── Comparison to failed alternatives ("tried everything")
├── Timeframe specified ("in just 6 weeks")
├── Emotional transformation language
├── Full attribution with location
├── Video/audio format
└── "First time in X years" language
```

---

## Common Extraction Errors

**Error: Missing Specific Results**
```
BAD: Extract only "Great product, changed my life!"
GOOD: Look for hidden specifics - even vague ones like "in weeks"
```

**Error: Losing Attribution**
```
BAD: "Lost 30 pounds!" (no attribution)
GOOD: Always capture who said it, even if anonymous
```

**Error: Merging Multiple Testimonials**
```
BAD: Combining two testimonials into one element
GOOD: One element per person, even if on same topic
```

- **1.7 Element Scoring** (for full scoring)

Also pass Testimonial Cluster data (testimonials grouped by similar outcomes) for ranking use.
