## CALCULATION PROCESS

### Step 1: Determine Total Word Count

```
FROM assembly_context:
  total_target_words: [from structure-package]

OR calculate from section totals:
  sum of all section word counts
```

### Step 2: Calculate Actual Proportions

```
FOR each section:
  actual_words = [section word count]
  actual_percentage = (actual_words / total_words) * 100
```

### Step 3: Compare to Benchmarks

```
FOR each section:
  format = [from campaign-brief]
  target_range = [from benchmark table]

  IF actual_percentage < target_range.min:
    status = UNDER
    adjustment = target_range.min - actual_percentage

  IF actual_percentage > target_range.max:
    status = OVER
    adjustment = actual_percentage - target_range.max

  IF actual_percentage within target_range:
    status = OK
    adjustment = 0
```

---

## ADJUSTMENT RECOMMENDATIONS

### For Over-Length Sections

| Over By | Recommendation |
|---------|----------------|
| 1-5% | Light trim — remove redundancy |
| 5-10% | Moderate trim — cut weakest paragraphs |
| >10% | Significant trim — consider splitting or major revision |

### For Under-Length Sections

| Under By | Recommendation |
|----------|----------------|
| 1-5% | Light expansion — add detail |
| 5-10% | Moderate expansion — add examples or proof |
| >10% | Significant expansion — may need additional content |
