# 1.3 Promise Type Variations

## Purpose
Generate promise variations across different psychological frames. The same core transformation can be framed as a transformation, improvement, relief, capability, or prevention promise — each resonates differently with different prospect mindsets.

## The Five Promise Types

```
PROMISE TYPE HIERARCHY:

TRANSFORMATION → What you BECOME
"Transform from overweight and tired to slim and energetic"
- Appeals to: Identity change seekers
- Emotional driver: Hope, aspiration
- Best for: High-hope markets

IMPROVEMENT → What gets BETTER
"Significantly improve your energy and metabolism"
- Appeals to: Incremental progress seekers
- Emotional driver: Optimism, realistic hope
- Best for: Moderate-skepticism markets

RELIEF → What goes AWAY
"End your constant cravings and food obsession"
- Appeals to: Pain-focused prospects
- Emotional driver: Frustration, desire for escape
- Best for: High-pain markets

CAPABILITY → What you can now DO
"Finally be able to eat without gaining weight"
- Appeals to: Action-oriented prospects
- Emotional driver: Freedom, empowerment
- Best for: Limitation-aware markets

PREVENTION → What you AVOID
"Prevent the weight regain that follows every diet"
- Appeals to: Risk-averse prospects
- Emotional driver: Fear, caution
- Best for: High-fear, burned markets
```

## Execution Protocol

### Step 1: Identify Core Outcome

```python
def extract_core_outcome(promise):
    """
    Extract the fundamental outcome from any promise.
    """
    # Strip framing language to get core
    framing_words = [
        'transform', 'improve', 'end', 'stop', 'gain', 'lose',
        'finally', 'never again', 'prevent', 'avoid'
    ]

    core = promise
    for word in framing_words:
        core = core.replace(word, '').strip()

    return {
        'core_outcome': core,
        'original_promise': promise
    }
```

### Step 2: Generate Type Variations

```python
def generate_type_variations(core_outcome, root_cause, villain):
    """
    Generate promises across all 5 psychological frames.
    """
    variations = {}

    # TRANSFORMATION promises
    variations['transformation'] = [
        f"Transform your {extract_topic(core_outcome)} completely",
        f"Become someone who {positive_identity_version(core_outcome)}",
        f"Experience a total {core_outcome} transformation",
        f"Go from {negative_state} to {positive_state}"
    ]

    # IMPROVEMENT promises
    variations['improvement'] = [
        f"Significantly improve your {core_outcome}",
        f"Get better {core_outcome} than you've had in years",
        f"See real, measurable improvement in your {core_outcome}",
        f"Experience {core_outcome} like you did {timeframe} ago"
    ]

    # RELIEF promises
    variations['relief'] = [
        f"End your {problem_version(core_outcome)} once and for all",
        f"Finally stop dealing with {pain_version(core_outcome)}",
        f"Free yourself from {villain}",
        f"Escape the {frustration} that comes with {problem}"
    ]

    # CAPABILITY promises
    variations['capability'] = [
        f"Finally be able to {action_version(core_outcome)}",
        f"Gain the ability to {capability}",
        f"Do what you've never been able to do: {action}",
        f"Unlock the freedom to {desire}"
    ]

    # PREVENTION promises
    variations['prevention'] = [
        f"Prevent {negative_consequence} from ever happening",
        f"Avoid the {fear} that comes from {problem}",
        f"Protect yourself from {villain}",
        f"Stop {negative_outcome} before it starts"
    ]

    return variations
```

## Type-Specific Templates

### Transformation Templates
```
1. "Transform from [negative state] to [positive state]"
2. "Become [positive identity]"
3. "Experience a complete [topic] transformation"
4. "Go from [before] to [after]"
5. "Finally become the [identity] you were meant to be"
```

### Improvement Templates
```
1. "[Significantly/Dramatically] improve your [topic]"
2. "Get better [results] than [comparison]"
3. "See [measurable] improvement in your [topic]"
4. "Experience [topic] like you did [past reference]"
5. "[X]% better [results] than [current state]"
```

### Relief Templates
```
1. "End your [problem] once and for all"
2. "Finally stop [negative experience]"
3. "Free yourself from [villain/constraint]"
4. "Escape the [frustration] of [problem]"
5. "Never [suffer/struggle/deal with] [problem] again"
```

### Capability Templates
```
1. "Finally be able to [desired action]"
2. "Gain the [ability/freedom/power] to [action]"
3. "Do what you've never been able to do"
4. "Unlock [capability] without [struggle]"
5. "[Action] without [negative consequence]"
```

### Prevention Templates
```
1. "Prevent [negative outcome] from [happening/returning]"
2. "Avoid the [consequence] that comes from [cause]"
3. "Protect yourself from [threat/villain]"
4. "Stop [problem] before it [worsens/returns]"
5. "Never let [fear] happen to you again"
```
