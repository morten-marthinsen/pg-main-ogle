## Schwartz Stage → Promise Strategy

```yaml
stage_strategies:

  stage_1:
    name: "First to Market"
    description: "No one has made this claim before"
    promise_strategy: "Direct Promise"
    promise_carries_weight: true
    mechanism_emphasis: none

    framing_formula: "[Product] will [promise]"
    example: "This pill will help you lose weight"

    what_works:
      - Simple, direct claims
      - Bold promises
      - Minimal proof needed
      - No mechanism explanation required

    what_to_avoid:
      - Over-explaining
      - Complex mechanisms
      - Heavy proof stacking

  stage_2:
    name: "Competition Enters"
    description: "Others are making similar claims"
    promise_strategy: "Bigger/Better Promise"
    promise_carries_weight: true
    mechanism_emphasis: light

    framing_formula: "Now get [bigger/better/faster] results"
    example: "Lose weight 2X faster than any other pill"

    what_works:
      - Amplified claims (more, faster, easier)
      - Comparative language
      - Light mechanism mention
      - Social proof

    what_to_avoid:
      - Same promise as competitors
      - Mechanism-heavy messaging

  stage_3:
    name: "Crowded Market"
    description: "Claims are becoming saturated"
    promise_strategy: "Mechanism + Promise"
    promise_carries_weight: shared with mechanism
    mechanism_emphasis: heavy

    framing_formula: "The [mechanism] way to [promise]"
    example: "The Gut Reset Protocol for lasting weight loss"

    what_works:
      - Lead with HOW it works
      - Unique mechanism positioning
      - Mechanism makes promise credible
      - Strong proof supporting mechanism

    what_to_avoid:
      - Bare promises without mechanism
      - Generic mechanism descriptions

  stage_4:
    name: "Sophisticated Market"
    description: "Prospects have heard it all"
    promise_strategy: "Unique Mechanism Promise"
    promise_carries_weight: secondary to mechanism
    mechanism_emphasis: unique angle required

    framing_formula: "The ONLY way to [promise] is [unique mechanism]"
    example: "The ONLY way to end cravings is to defeat the Bad Bugs"

    what_works:
      - Unique, novel mechanism angle
      - "Only" or exclusive positioning
      - Mechanism IS the differentiator
      - Heavy authority proof

    what_to_avoid:
      - Generic promises
      - "Me too" mechanism claims

  stage_5:
    name: "Exhausted Market"
    description: "Prospects are skeptical of everything"
    promise_strategy: "Identification + Promise"
    promise_carries_weight: secondary to identification
    mechanism_emphasis: buried (identity-wrapped)

    framing_formula: "For [identity] who want [promise]"
    example: "For busy moms who've tried everything and still can't lose weight"

    what_works:
      - Lead with prospect identity
      - Empathy and understanding
      - Promise wrapped in identification
      - Heavy social proof

    what_to_avoid:
      - Hard sell
      - Bold claims upfront
      - Aggressive mechanism pushing
```

## Execution Protocol

### Step 1: Assess Current Promise Framing

```python
def assess_framing(promise):
    """
    Determine current framing type of a promise.
    """
    framing_indicators = {
        'direct': {
            'patterns': ['will help', 'can get', 'lets you'],
            'mechanism_present': False
        },
        'bigger_better': {
            'patterns': ['faster', 'easier', 'more', 'better than', '2x', '3x'],
            'mechanism_present': False
        },
        'mechanism_led': {
            'patterns': ['with the', 'using the', 'through the', 'protocol', 'system'],
            'mechanism_present': True
        },
        'unique_mechanism': {
            'patterns': ['only way', 'the only', 'unlike', 'different from'],
            'mechanism_present': True
        },
        'identification': {
            'patterns': ['for those who', 'if you\'ve', 'for anyone who'],
            'mechanism_present': 'buried'
        }
    }

    promise_lower = promise.lower()

    for framing_type, indicators in framing_indicators.items():
        if any(p in promise_lower for p in indicators['patterns']):
            return framing_type

    return 'direct'  # Default
```

### Step 2: Determine Required Framing

```python
def get_required_framing(schwartz_stage):
    """
    Get required framing for market stage.
    """
    stage_to_framing = {
        1: 'direct',
        2: 'bigger_better',
        3: 'mechanism_led',
        4: 'unique_mechanism',
        5: 'identification'
    }

    return stage_to_framing.get(schwartz_stage, 'mechanism_led')
```

### Step 3: Reframe for Stage

```python
def reframe_for_stage(promise, current_framing, required_framing, context):
    """
    Adjust promise framing for target stage.
    """
    mechanism = context.get('mechanism_name', '[mechanism]')
    identity = context.get('target_identity', 'people')
    villain = context.get('villain', None)

    reframing_templates = {
        ('direct', 'mechanism_led'): [
            f"The {mechanism} way to {extract_core_promise(promise)}",
            f"With the {mechanism}, {promise}",
            f"{promise} — through the {mechanism}"
        ],
        ('direct', 'unique_mechanism'): [
            f"The ONLY way to {extract_core_promise(promise)} is with the {mechanism}",
            f"Unlike anything else, the {mechanism} lets you {extract_core_promise(promise)}"
        ],
        ('mechanism_led', 'unique_mechanism'): [
            f"The ONLY way to {extract_core_promise(promise)} is with the {mechanism}",
            f"What makes the {mechanism} the ONLY solution that can {extract_core_promise(promise)}"
        ],
        ('any', 'identification'): [
            f"For {identity} who want to {extract_core_promise(promise)}",
            f"If you've tried everything to {extract_core_promise(promise)}...",
            f"For anyone who's struggled to {extract_core_promise(promise)}"
        ]
    }

    # Get appropriate templates
    key = (current_framing, required_framing)
    if key not in reframing_templates:
        key = ('any', required_framing)

    templates = reframing_templates.get(key, [])

    reframed_versions = []
    for template in templates:
        reframed = template.format(
            promise=promise,
            mechanism=mechanism,
            identity=identity
        )
        reframed_versions.append(reframed)

    # Add villain version for stage 4+ if villain exists
    if required_framing in ['unique_mechanism', 'identification'] and villain:
        reframed_versions.append(
            f"The ONLY way to {extract_core_promise(promise)} is to defeat {villain}"
        )

    return {
        'original': promise,
        'current_framing': current_framing,
        'required_framing': required_framing,
        'reframed_versions': reframed_versions,
        'recommended': reframed_versions[0] if reframed_versions else promise
    }
```

## Stage-Specific Adjustments

```yaml
stage_adjustments:

  stage_1_adjustments:
    add:
      - Direct benefit language
      - Simple promise statements
    remove:
      - Mechanism jargon
      - Complex explanations
    emphasis:
      - Promise itself

  stage_2_adjustments:
    add:
      - Comparative language ("2x faster")
      - Amplification words ("even more", "now you can")
    remove:
      - Generic claims
    emphasis:
      - Superiority over alternatives

  stage_3_adjustments:
    add:
      - Mechanism name
      - "How it works" language
      - "The [mechanism] way to..."
    remove:
      - Bare promises
    emphasis:
      - Mechanism as enabler of promise

  stage_4_adjustments:
    add:
      - "Only" / "unique" language
      - Novel mechanism framing
      - Villain/enemy framing
    remove:
      - Generic mechanism descriptions
      - "Me too" language
    emphasis:
      - Mechanism uniqueness

  stage_5_adjustments:
    add:
      - Identity markers
      - Empathy language
      - "If you've tried everything..."
    remove:
      - Hard sell language
      - Bold upfront claims
    emphasis:
      - Identification before promise
```
