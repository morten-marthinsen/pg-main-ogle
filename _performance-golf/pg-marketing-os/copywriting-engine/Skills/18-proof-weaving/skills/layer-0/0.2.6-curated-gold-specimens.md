**Strategy 1: Proof-First Architecture (Whitaker Pattern)**
```
PROVE it (strongest proof) → Scale it (thousands → 40K) → Explain it → CTA
```
- Use when: Proof is dramatic enough to lead
- Example: National TV, celebrity validation, impossible transformation

**Strategy 2: Authority → Clinical → Social (Sinatra Pattern)**
```
Authority credentials → Proprietary framework → Clinical data → Scale social proof → Risk reversal
```
- Use when: Authority figure is central to brand
- Example: MD critiquing own profession, cardiologist notebook

**Strategy 3: Scale Cascade (Individual → Pattern → Scale)**
```
One person (Ken) → Thousands ("not alone") → 40,000 patients → "You too"
```
- Use when: Building from single story to mass validation
- Example: "One lucky guy" objection handling

**Strategy 4: Testimonial Parade (IVL Pattern)**
```
25%+ of copy = testimonials in waves
First wave (5-8 testimonials) → Mechanism break → Second wave (5-8) → Offer → Third wave (5-8)
```
- Use when: High-volume testimonial inventory available
- Example: Green Energizer (35+ testimonials)

**Strategy 5: Wave Pattern (Alternating Density)**
```
Heavy proof (lead) → Moderate (body) → Heavy (proof section) → Light (offer) → Moderate (close)
```
- Use when: Standard long-form structure
- Example: Most health magalogs
