## LOADING PROTOCOL

```
EXECUTE 0.2-vault-intelligence-loader:

  1. LOCATE headline-vault-intelligence.json
     IF not found:
       WARN: "Vault intelligence not built yet"
       SET vault_available = false
       RETURN empty patterns with fallback flag

  2. LOAD pattern_type_frequency
     VALIDATE: frequencies sum to 1.0 ± 0.01

  3. LOAD format_pattern_frequency
     VALIDATE: frequencies sum to 1.0 ± 0.01

  4. LOAD power_words
     EXTRACT: top_50 and emotion-indexed words

  5. LOAD schema_distance_benchmarks
     EXTRACT: averages, optimal ranges, niche-specific

  6. LOAD gold_specimens
     FILTER: top 20 by quality_score
     INDEX: by pattern_type for quick retrieval

  7. LOAD niche_patterns
     MATCH: to campaign niche from upstream_context

  8. ASSEMBLE vault_patterns object
  9. RETURN vault_patterns
```
