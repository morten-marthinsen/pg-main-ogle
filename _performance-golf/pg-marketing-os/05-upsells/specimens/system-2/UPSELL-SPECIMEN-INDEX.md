The engine should ALWAYS explore done-for-you as the first upsell option during U0 Strategy.

3. **Performance Golf dominates the specimen set (14 of 28 specimens).** They have the most refined and consistent upsell structure, making their patterns the strongest signal for golf. Their formula: congratulate -> stick-the-sale -> congruence bridge -> mechanism education -> proof -> bonus stack -> price anchor -> guarantee -> urgency CTA.

4. **US-21 and US-26 prove the CAIRO template is reusable across verticals.** Same copywriter (Jay Deiboldt), same structure, same survey → discovery → test group arc, same $3.23/day coffee comparison, same "click out / strike out" language — just different vertical content.
