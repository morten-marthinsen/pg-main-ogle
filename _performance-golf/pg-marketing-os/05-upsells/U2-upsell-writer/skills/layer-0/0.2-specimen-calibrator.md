# 0.2 — Specimen Calibrator

**Version:** 1.1
**Skill:** U2 — 1-Click Upsell Writer
**Layer:** 0
**Microskill:** 0.2 — Specimen Calibrator
**Purpose:** Load and analyze upsell specimens from the TIER1 vault to calibrate CAIRO proportions, tone patterns, price presentation patterns, and CTA language for this specific upsell.

---

## INPUT

- `UPSELL-specimen-index.md` (specimen registry with US-01 through US-28)
- `layer-0/0.1-input-context.md` (for position, vertical, and format filtering)
- TIER1 extraction vault files (filtered by vertical match)

## PROCESS

1. **Load the specimen index** from `UPSELL-specimen-index.md`. This contains metadata for all indexed upsell specimens (US-01 through US-20+).
2. **Filter specimens by three criteria** (in priority order):
   - **(a) Position match:** Select specimens that match the upsell position (Upsell 1 vs Upsell 2). Position is the strongest filter — an Upsell 2 specimen teaches Upsell 2 psychology regardless of vertical.
   - **(b) Vertical match:** From position-matched specimens, prefer those matching the campaign vertical (golf, health, finance, personal-dev, technology). Performance Golf specimens (14 of 18 total) are the strongest signal for golf vertical campaigns.
   - **(c) Format match:** From the remaining set, prefer video script specimens for video script output and text page specimens for text page output. Format is the weakest filter — CAIRO structure is format-agnostic.
3. **Select 3-5 calibration specimens** from the filtered set. If fewer than 3 match all three criteria, relax format match first, then vertical match. Never relax position match.
4. **Extract CAIRO proportions** from each selected specimen:
   - Count words per CAIRO section (C, A, I, R, O)
   - Calculate percentage of total for each section
   - Note any missing or merged sections (some specimens combine I+R)
   - Compute average proportions across all selected specimens
5. **Extract tone patterns** from each specimen:
   - Opening tone (celebratory, conspiratorial, warm, urgent)
   - Transition language between CAIRO sections
   - Register (conversational, authoritative, intimate, professional)
   - Energy trajectory (builds, steady, peaks-at-intrigue, peaks-at-offer)
6. **Extract price presentation patterns:**
   - Anchor type (retail price, value-of-components, competitor price, cost-of-problem)
   - Discount framing (percentage off, dollar savings, payment plan, per-day)
   - Guarantee presentation (mirror FE, separate guarantee, implied, explicit)
   - Specimen data: "two easy payments of $97" outperformed $97 flat and $149 flat by $38/order in split tests
7. **Extract CTA language patterns:**
   - YES button text (product name in button? price in button? action verb?)
   - NO link text (how specimens phrase the decline option)
   - Placement (after guarantee? after price? after both?)
   - Check for guilt-trip patterns to AVOID (flag any specimens using manipulative NO text)
8. **Synthesize calibration targets** — combine extracted patterns into recommended ranges:
   - Target CAIRO proportions (with +-15% flexibility per section)
   - Recommended tone register and trajectory
   - Recommended price presentation approach
   - Recommended CTA phrasing template
9. **Flag Performance Golf bias** — if 14 of 18 specimens are Performance Golf, note that calibration is golf-heavy. For non-golf campaigns, weight the remaining 4 specimens more heavily and apply vertical-appropriate adjustments.
9.5. **Apply Goff specimen recommendations for non-golf verticals:**
   - Credit repair / info product: US-21 (CAIRO gold standard — the specimen CAIRO was derived from)
   - Real estate / biz opp: US-22 (DFY website upsell with coaching)
   - Health / supplement: US-23 (size upgrade), US-24 (info→supplement with 17 ingredients)
   - Diet / weight loss: US-25 (completeness pack — 5-component stack)
   - Dating / self-improvement: US-26 (DFY messaging templates — structurally identical to US-21)
   - Finance / newsletter: US-27 (subscription upgrade with Legacy Program)
   - Survival / prepper: US-28 (massive quantity upgrade with warehouse template)
   - For done-for-you upsells in ANY vertical: US-21 and US-26 are the primary calibration specimens (same CAIRO template, proven 30-50% conversion)
10. **Write output file.**

## CONSTRAINTS

1. Never select fewer than 3 specimens. If the filtered set is too small, relax filters in order: format first, then vertical. Never relax position match.
2. Performance Golf is now 50% of specimens (14/28). For non-golf campaigns, weight Goff specimens (US-21 through US-28) more heavily — they cover 8 distinct non-golf verticals. US-21 and US-26 are vertical-agnostic CAIRO gold standards. Do not let golf-specific patterns (e.g., "shave 7 strokes") contaminate non-golf calibration.
3. CAIRO proportions are TARGETS, not mandates. They inform Layer 2 generation but do not override the word count ranges in the AGENT.md proportion table.
4. If a specimen is missing a CAIRO section (e.g., no distinct Intrigue section), note this but do not use it to justify skipping Intrigue in the output. All 5 CAIRO sections are mandatory regardless of specimen patterns.
5. CTA language extraction must flag any guilt-trip NO options found in specimens. These are anti-patterns to AVOID, not patterns to replicate.
6. Specimen calibration informs generation — it does not replace the AGENT.md structure. Where specimens conflict with AGENT.md constraints, AGENT.md wins.
