### Failure Mode 2: Partial Index Created Without Detection
- **Symptom:** Downstream microskills reference patterns not in index
- **Cause:** Section parsing fails silently for some sections
- **Prevention:** Section-by-section count validation in Step 16
- **Recovery:** Re-index with explicit section-presence verification

### Failure Mode 3: Control Examples Not Linked to Patterns
- **Symptom:** Patterns exist as abstract rules without concrete examples
- **Cause:** Control references stripped during indexing
- **Prevention:** Every pattern MUST include at least one control example
- **Recovery:** Re-index with mandatory control linkage

- **Symptom:** Draft generation produces vaguely similar but inauthentic language
- **Cause:** Indexer summarized instead of preserving exact quotes
- **Prevention:** Transition language must be preserved CHARACTER-FOR-CHARACTER
