# Microskill 0.1: Upstream Loader

**Layer:** 0 (Foundation)
**Purpose:** Load assembled-draft.md and supporting packages from 19-campaign-assembly

---

## INPUTS TO LOAD

### Primary Input (REQUIRED)

From 19-campaign-assembly:

| File | Purpose | Failure Action |
|------|---------|----------------|
| `assembled-draft.md` | Complete assembled copy | STOP: Cannot edit without copy |
| `campaign-assembly-package.json` | Assembly metadata, threading counts, callbacks | WARN: Limited context |

### Secondary Inputs (RECOMMENDED)

From upstream skills:

| File | Purpose | If Missing |
|------|---------|------------|
| `campaign-brief-package.json` | Tone, avatar, sophistication level | Use defaults |
| `structure-package.json` | Section proportions, format type | Use assembled structure |
| `mechanism-package.json` | Mechanism name, framework | Extract from copy |
| `root-cause-package.json` | Root cause anchor, villain | Extract from copy |
| `promise-package.json` | Primary promise | Extract from copy |

---

## LOADING PROCESS

### Step 1: Load Primary Input

```yaml
primary_load:
  assembled_draft:
    file: "assembled-draft.md"
    status: LOADED | MISSING
    word_count: "[count]"
    sections_present: [list]

  assembly_package:
    file: "campaign-assembly-package.json"
    status: LOADED | MISSING
    threading_summary: "[from package]"
    callback_registry: "[from package]"
    quality_scores: "[from package]"
```

### Step 2: Load Secondary Inputs

```yaml
secondary_load:
  campaign_brief:
    status: LOADED | MISSING
    tone: "[if loaded]"
    avatar: "[if loaded]"
    sophistication: "[if loaded]"

  structure:
    status: LOADED | MISSING

  mechanism:
    status: LOADED | MISSING
    mechanism_name: "[if loaded]"
    framework_name: "[if loaded]"

  root_cause:
    status: LOADED | MISSING
    anchor_phrase: "[if loaded]"
    villain: "[if loaded]"

  promise:
    status: LOADED | MISSING

```

### Step 3: Extract Missing Elements

If secondary inputs missing, extract from assembled copy:

```yaml
extracted_elements:
  mechanism_name: "[found in copy]"
  root_cause_anchor: "[found in copy]"
  primary_promise: "[found in copy]"

```
