# Microskill 3.1: Structural Compliance Audit

**Layer:** 3 (5-Tier Evaluation)
**Purpose:** Verify copy structure meets required framework requirements

---

## TIER 1 OF 5: STRUCTURAL COMPLIANCE

This tier evaluates whether the copy contains all required structural elements in the correct sequence.

---
