# Skill 13.4.4: Final Narrative Assembler

**Version:** 1.0
**Skill:** 10 — Root Cause Narrative
**Layer:** 4 (Validation & Assembly)
**Position:** Final microskill — assembles the complete root-cause-narrative-package.json
**Type:** Assembly & Output
**Dependencies:** ALL upstream microskill outputs — every Layer 0-4 output
**Output:** root-cause-narrative-package.json

---

## PURPOSE

Assemble the complete root-cause-narrative-package.json — the final output of Skill 13. This package combines metadata, the confirmed root cause, the narrative strategy (classification, framing, arc, sequence), all seven narrative sections with their prose text, countersells, the threading guide, evidence integration data, the reframe stack, the full concatenated narrative text, validation scores across all dimensions, and downstream handoffs for Skills 11, 12, 13, and 14. The overall weighted average score must be >= 7.0 to pass. If the score falls below 7.0, specific remediation is recommended.

---

## IDENTITY

**This microskill IS:**
- The final assembler that compiles all outputs into a single deliverable package
- A score calculator that computes weighted averages across all quality dimensions
- A downstream handoff packager that prepares data for Skills 11, 12, 13, and 14
- A quality gate that enforces the minimum 7.0 overall score

**This microskill is NOT:**
- A validator — validation happened in 4.1, 4.2, and 4.3; this assembles validated outputs
- A writer — it concatenates and packages; all writing happened in Layers 2-3
- A modifier — if the score is below 7.0, it recommends remediation; it does not fix anything itself
- An optional step — every Skill 10 run must end with this assembler producing the package

### Step 10: Calculate Validation Scores

```
CALCULATE scores across all dimensions:

  narrative_persuasion: float (1-10)
    Based on: emotional arc execution, three-part delivery clarity,
              reframe stack effectiveness, overall narrative flow
    WEIGHT: 0.20

  three_part_clarity: float (1-10)
    Based on: all three parts delivered clearly, false belief → root cause
              contrast is sharp, counter-intuitive element present
    WEIGHT: 0.15

  emotional_arc_execution: float (1-10)
    Based on: phase targets hit, peak moment achieved, ending state correct,
              intensity curve shaped (not flat)
    WEIGHT: 0.15

  evidence_integration: float (1-10)
    Based on: evidence naturalness scores, evidence density vs. benchmark,
              pattern diversity
    WEIGHT: 0.15

  countersell_effectiveness: float (1-10)
    Based on: minimum 2 present, root cause used as weapon, specific competitors
              named, placement natural
    WEIGHT: 0.10

  threading_completeness: float (1-10)
    Based on: anchor phrase quality, repetition points comprehensive,
              downstream notes complete, emphasis level appropriate
    WEIGHT: 0.10

  vault_pattern_comparison: float (1-10)
    Based on: vault_comparison_report.overall_comparison_score
    WEIGHT: 0.15

  CALCULATE overall_weighted_average:
    = (narrative_persuasion * 0.20) +
      (three_part_clarity * 0.15) +
      (emotional_arc_execution * 0.15) +
      (evidence_integration * 0.15) +
      (countersell_effectiveness * 0.10) +
      (threading_completeness * 0.10) +
      (vault_pattern_comparison * 0.15)

  MINIMUM REQUIRED: 7.0
```

### Step 13: Evaluate Pass/Fail and Assemble Final Package

```
EVALUATE overall pass:

  CONDITIONS for PASS:
    1. overall_weighted_average >= 7.0
    2. critical_rules_report.all_rules_pass == true
    3. anti_slop_report.pass == true
    4. vault_comparison_report.pass == true
    5. full_narrative_text is non-empty and complete

  IF all conditions met:
    SET package_status = "PASS"
    LOG: "Root cause narrative package PASSES all quality gates.
          Overall score: [score]/10. All 10 rules pass. Zero slop violations.
          Vault comparison: [score]/10."

  IF overall_weighted_average < 7.0:
    SET package_status = "FAIL"
    IDENTIFY lowest_dimensions: [dimensions below 7.0, sorted ascending]
    FOR each low dimension:
      GENERATE remediation_recommendation:
        dimension: string
        current_score: float
        target_score: 7.0
        recommendation: string  # specific action to raise score
    LOG: "Root cause narrative package FAILS. Score: [score]/10.
          Lowest dimensions: [list]. Remediation required."

  IF critical_rules not all pass:
    SET package_status = "FAIL"
    LOG: "Critical rules violations: [failing rules]. MUST be fixed."

  IF anti_slop violations > 0:
    SET package_status = "FAIL"
    LOG: "Anti-slop violations: [count]. MUST be fixed."

```

## QUALITY CRITERIA

1. **Overall score >= 7.0** — weighted average across all dimensions must meet minimum threshold
2. **All 10 critical rules pass** — zero tolerance for rule violations
3. **Anti-slop passes** — zero violations
4. **Vault comparison passes** — comparison score >= 6.0 with differentiation >= 5.0
5. **Full narrative text present** — all 7 phases concatenated with transitions
6. **All downstream handoffs complete** — mechanism, body copy, and close handoffs all populated
7. **Threading guide complete** — anchor phrase, repetition points, language bank all present
8. **Package schema complete** — all required fields populated; no null required fields

---

## FAILURE MODES

| Failure | Severity | Detection | Handling |
|---------|----------|-----------|----------|
| Overall score < 7.0 | HIGH | Weighted average below threshold | Identify lowest dimensions; recommend specific remediation |
| Critical rules not all pass | CRITICAL | One or more rules fail | Cannot assemble; return to relevant Layer for fix |
| Anti-slop violations present | HIGH | Violation count > 0 | Cannot assemble; return to Layer 3 for language fix |
| Full narrative text empty | CRITICAL | Concatenation produced empty string | Check all Layer 2-3 outputs; something fundamental missing |
| Missing downstream handoff data | MODERATE | Handoff field null or empty | Populate from available data; flag if insufficient |
| Threading guide incomplete | MODERATE | Anchor phrase or repetition points missing | Return to 3.4 for completion |
| Package write fails | LOW | File system error | Retry; log error |
