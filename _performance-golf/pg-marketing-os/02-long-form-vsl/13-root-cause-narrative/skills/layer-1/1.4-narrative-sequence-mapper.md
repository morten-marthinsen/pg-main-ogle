# Skill 13.1.4: Narrative Sequence Mapper

**Version:** 1.0
**Skill:** 10 — Root Cause Narrative
**Layer:** 1 (Strategic Classification)
**Position:** Fourth classification microskill — maps the 7-phase universal sequence to the specific campaign
**Type:** Structural Planning & Allocation
**Dependencies:** communication_type_classification.json (from 1.1), framing_pattern_selection.json (from 1.2), emotional_arc_design.json (from 1.3), checkpoint_response.json (from 0.4), upstream_data.json (from 0.1), tier1_intelligence_index.json (from 0.2)
**Output:** narrative_sequence_map.json

---

## PURPOSE

Map the 7-phase universal root cause sequence to the specific campaign by allocating word counts per phase, selecting the appropriate format variation (VSL, magalog, or advertorial), aligning phases with CPB chunks from the structure package, and defining transition types between every phase. This is the structural blueprint that Layer 2 writers build against — it tells them exactly how much to write for each phase and how each phase connects to the next.

---

## IDENTITY

**This microskill IS:**
- A structural planner that converts the universal 7-phase sequence into a campaign-specific blueprint
- A word count allocator that distributes the total word budget across phases according to TIER1 proportions
- A format adapter that adjusts phase sequencing for VSL, magalog, or advertorial formats
- A transition planner that selects entry, exit, and inter-phase transitions from the TIER1 catalog

**This microskill is NOT:**
- A writer — it allocates and plans; writers execute in Layer 2
- An emotional designer (that is 1.3) — it incorporates the emotional arc but doesn't design it
- A content classifier (that is 1.1 and 1.2) — it maps structure, not strategy
- A replacement for the structure package — it extends structure-package.json for this specific section

## PROCESS

### Step 1: Determine Total Word Count Budget

```
DETERMINE total_word_count:

  IF word_count_budget available from structure package:
    total_word_count = structure.word_count_budget
    source = "structure_package"
  ELSE:
    DETERMINE from format_type:
      IF format_type == "vsl": total_word_count = 2500  # root cause section of VSL
      IF format_type == "magalog": total_word_count = 1800  # tighter in print
      IF format_type == "advertorial": total_word_count = 1200  # most concise
      IF format_type == "email_sequence": total_word_count = 800  # per email
      IF format_type == "landing_page": total_word_count = 1000
      IF format_type == "unknown": total_word_count = 2000  # safe default
    source = "format_default"

  LOG: "Total word count budget: [total_word_count] words (source: [source])"
```

### Step 2: Allocate Word Counts Per Phase

```
USING TIER1 universal_sequence percentage ranges:

  Phase 1 — Problem Agitation / Symptom Identification: 5-10%
  Phase 2 — False Belief Installation: 3-5%
  Phase 3 — Authority Establishment: 5-10%
  Phase 4 — Root Cause Reveal: 10-15%
  Phase 5 — Failure Explanation: 5-8%
  Phase 6 — Villain Deep Dive: 8-12%
  Phase 7 — Mechanism Bridge: 10-15%

ADJUST percentages based on classification:

  FOR communication_type:
    IF scientific_explanation:
      INCREASE Phase 3 (authority) to upper range (10%)
      INCREASE Phase 4 (reveal) to upper range (15%)
      Phase 6 (villain) to lower range (8%)
    IF conspiracy_reveal OR conspiracy_expose:
      INCREASE Phase 6 (villain) to upper range (12%)
      Phase 3 (authority) to lower range (5%)
    IF direct_confrontation:
      INCREASE Phase 1 (agitation) to upper range (10%)
      INCREASE Phase 2 (false belief) to upper range (5%)
      Phase 6 (villain) to mid range (10%)
    IF data_presentation:
      INCREASE Phase 4 (reveal) to upper range (15%)
      Phase 1 (agitation) to lower range (5%)
    IF not_your_fault:
      INCREASE Phase 5 (failure explanation) to upper range (8%)
      INCREASE Phase 4 (reveal) to upper range (15%)
    IF metaphor_driven:
      INCREASE Phase 4 (reveal) to upper range (15%)
      INCREASE Phase 7 (bridge) to upper range (15%)

  FOR emotional_arc peak_moment:
    INCREASE the peak phase by 2-3% (needs room for emotional impact)
    DECREASE non-peak phases proportionally

  FOR framing:
    IF villain_driven primary: Phase 6 to upper range
    IF science_driven primary: Phase 3-4 to upper range
    IF conspiracy_driven primary: Phase 6 to upper range
    IF inversion_driven primary: Phase 2 and Phase 5 to upper range

NORMALIZE: Total percentages must sum to approximately 56-80%
  (remaining 20-44% is for structure elements outside root cause:
   sub-headers, proof callouts, visual breaks in magalog, etc.)

CALCULATE word counts:
  FOR each phase:
    word_count = ROUND(total_word_count * adjusted_percentage)

VALIDATE: Sum of phase word counts <= total_word_count
  IF sum exceeds budget: proportionally reduce non-peak phases
```

### Step 3: Select Format Variation

```
DETERMINE format_variation:

  IF format_type == "vsl" OR "landing_page":
    format_variation = "vsl_sequence"
    Characteristics:
      - Linear progression through all 7 phases
      - Longer transitions between phases (viewer can't skip ahead)
      - Heavier emotional build (audio/visual medium)
      - Phase 1-2 can be more extended (hold attention through agitation)

  IF format_type == "magalog":
    format_variation = "magalog_sequence"
    Characteristics:
      - Tighter prose (reader can scan)
      - Sub-headers mark phase transitions
      - Visual callout boxes for key evidence
      - Phase 3 (authority) may use credential sidebar instead of prose
      - Phase 6 (villain) may use "exposé" formatting

  IF format_type == "advertorial":
    format_variation = "advertorial_sequence"
    Characteristics:
      - Most compressed — editorial voice throughout
      - Phases 1-2 may be combined into a single section
      - Phase 3 woven into journalistic credibility (not overt credential stack)
      - News-style formatting affects word choice
      - Phase 7 transitions to "article continues" format

  IF format_type == "email_sequence":
    format_variation = "email_sequence"
    Characteristics:
      - Phases may span multiple emails
      - Each email has its own micro-arc
      - Cliffhangers between emails
      - Phase 4 (reveal) is the key email

  IF format_type == "unknown":
    format_variation = "vsl_sequence"  # safest default
```

### Step 4: Align Phases with CPB Chunks

```
FROM upstream_data.structure.relevant_cpb_chunks:

  FOR each CPB chunk assigned to root cause:
    MAP chunk to specific phase(s):
      chunk.content_summary → MATCH to phase purpose

  EXAMPLE alignment:
    CPB Chunk "symptom_naming" → Phase 1
    CPB Chunk "false_belief_challenge" → Phase 2
    CPB Chunk "credibility_stack" → Phase 3
    CPB Chunk "root_cause_presentation" → Phase 4
    CPB Chunk "failure_reframe" → Phase 5
    CPB Chunk "villain_section" → Phase 6
    CPB Chunk "mechanism_introduction" → Phase 7

  IF no relevant CPB chunks available:
    LOG: "No CPB chunk alignment available. Phases will follow default sequence."
    structure_aligned = false
  ELSE:
    structure_aligned = true
    LOG: "[count] CPB chunks aligned to [count] phases"

  IDENTIFY gaps:
    Phases without CPB chunk assignment
    CPB chunks that don't match any phase (may belong to different skill)
```

### Step 5: Map Transition Types

```
FROM tier1_intelligence_index.transition_patterns:

MAP entry transition (into root cause section):
  SELECT from 6 entry patterns based on what precedes root cause in flow:
    1. "but_heres_the_real_problem" — after problem/story section
    2. "credential_to_revelation" — after credibility is established earlier
    3. "data_cascade" — when data leads into root cause
    4. "personal_paradox" — when authority's personal story leads to discovery
    5. "what_they_dont_want_you_to_know" — when conspiracy frame opens
    6. "question_sequence" — when questions build tension into reveal

  SELECTION LOGIC:
    IF communication_type == "conspiracy_reveal": prefer "what_they_dont_want_you_to_know"
    IF communication_type == "scientific_explanation": prefer "credential_to_revelation"
    IF communication_type == "direct_confrontation": prefer "but_heres_the_real_problem"
    IF communication_type == "data_presentation": prefer "data_cascade"
    IF previous section is story/personal: prefer "personal_paradox"

MAP inter-phase transitions (Phase N → Phase N+1):
  FOR each phase boundary (6 total):
    Phase 1→2: "symptom_to_belief" — "You think the problem is... You've been told..."
    Phase 2→3: "belief_to_authority" — "But one person saw through the lie..."
    Phase 3→4: "authority_to_reveal" — "And what [authority] discovered changes everything..."
    Phase 4→5: "reveal_to_failure" — "That's why everything you've tried has failed..."
    Phase 5→6: "failure_to_villain" — "And it's not an accident. Here's who's responsible..."
    Phase 6→7: "villain_to_mechanism" — "But now that you know the truth, here's what you can do..."

  ADJUST transitions based on arc:
    IF emotional arc peak at Phase 4: strengthen Phase 3→4 transition (build anticipation)
    IF emotional arc peak at Phase 6: strengthen Phase 5→6 transition (build outrage)

MAP exit transition (out of root cause section):
  SELECT from 4 exit patterns based on what follows:
    1. "root_cause_to_mechanism" — most common; leads to mechanism narrative
    2. "root_cause_to_villain_deep_dive" — when villain continues in next section
    3. "root_cause_to_social_proof" — when proof section follows
    4. "root_cause_to_historical_precedent" — when historical evidence section follows

  SELECTION LOGIC:
    IF next skill is 14-mechanism-narrative: select "root_cause_to_mechanism"
    IF villain deep dive extends beyond root cause section: select "root_cause_to_villain_deep_dive"
    DEFAULT: "root_cause_to_mechanism"
```

## QUALITY CRITERIA

1. **All 7 phases mapped** — every phase must have word count, emotional target, and transition
2. **Word counts sum to budget** — total phase word counts cannot exceed the total word count budget
3. **Percentages follow TIER1 ranges** — no phase should be wildly outside its TIER1 percentage range without explicit rationale
4. **Peak moment phase gets adequate word count** — the phase containing the emotional peak must have enough words to deliver impact
5. **Format variation matches campaign** — VSL, magalog, and advertorial have meaningfully different sequences
6. **All transitions defined** — entry, 6 inter-phase, and exit transitions all specified
7. **Phase 7 connects to Skill 11** — the mechanism bridge must explicitly reference the handoff point
8. **Structure alignment attempted** — CPB chunks mapped where available; alignment status documented

---

## FAILURE MODES

| Failure | Severity | Detection | Handling |
|---------|----------|-----------|----------|
| Word count budget unavailable | LOW | No budget from structure or format | Use format-based defaults; flag as estimated |
| Format type unknown | LOW | format_type = "unknown" | Default to vsl_sequence; flag in output |
| No CPB chunks available | MODERATE | relevant_cpb_chunks empty | Continue without alignment; flag structure_aligned = false |
| Phase word counts exceed budget | MODERATE | Sum > total_word_count | Proportionally reduce non-peak phases |
| Transition pattern not found | LOW | Selected transition not in TIER1 catalog | Use closest match; document adaptation |
| Peak phase word count too low | MODERATE | Peak phase allocated < 10% | Redistribute from non-peak phases to ensure peak has room |
| Entry transition conflicts with flow | LOW | Previous section doesn't match expected entry pattern | Adapt entry transition; may need generic "but here's the real problem" |

    intensity: medium
    transition: "But Dr. Steven Gundry — Yale honors, 16 years chairman of cardiothoracic surgery..."

    emotional_target: "shock and vindication"
    intensity: PEAK

    transition: "And what he discovered about lectins changes everything..."

  Phase 5: "Failure Explanation" — 175 words (7%)
    emotional_target: "righteous frustration"
    intensity: high
    transition: "That's exactly why every diet you've tried has failed..."

  Phase 6: "Villain Deep Dive" — 275 words (11%)
    emotional_target: "directed anger"
    intensity: high
    villain_name: "Food industry / USDA"
    transition: "And the food industry has known about this since 1950..."

  Phase 7: "Mechanism Bridge" — 300 words (12%)
    emotional_target: "determined hope"
    intensity: medium_high
    exit_transition: "root_cause_to_mechanism"

transition_map:
  entry: "credential_to_revelation"
  inter_phase:
    1→2: "symptom_to_belief"
    2→3: "belief_to_authority"

    5→6: "failure_to_villain"
