# Layer 2.5: Arena Persona Generation — Close Skill

**Version:** 2.1
**Parent Skill:** 17-close
**Position:** After Layer 2 (Draft Generation), Before Layer 3 (Refinement & Polish)
**Type:** Multi-Perspective Creative Generation with Human Selection Gate
**Reference:** `~system/protocols/ARENA-PERSONA-PANEL.md` (Master Specification v1.3)

> **Arena Mode:** `generative_full_draft` — Competitors write COMPLETE pieces from scratch using upstream packages. Layer 2 draft = reference material, not template. See `~system/protocols/ARENA-CORE-PROTOCOL.md` for 3-round execution protocol.

---

## PURPOSE

Generate multiple complete close drafts through 7 competitors (6 legendary copywriter personas + The Architect), each bringing their distinctive voice and philosophy to the FINAL PERSUASIVE PUSH of the promotion. The close is where desire becomes decision, where consideration becomes commitment. This layer ensures the close passes through diverse closing styles before the human selects the winning approach for refinement.

**Why Arena for Close:**
- Close copy has the highest variance in effectiveness — small differences in approach create massive conversion differences
- The 6 personas each approach closing with radically different philosophies (institutional vs. street-smart, proof-first vs. emotion-first, elegant vs. breathless)
- Makepeace's 7 closing themes can be executed in dramatically different ways by each persona
- CTA repetition requires variety — persona variation ensures genuinely different emotional appeals
- P.S. strategy benefits from diverse creative approaches to "highest-read real estate"
- Human selection at Gate 2.5 ensures the close matches the promotion's established voice

### Generation Requirements

Each persona MUST generate a complete close including:

1. **Benefit Summary** — "You get" format with 5+ items (persona-styled interpretation)
2. **Guarantee Close** — Contract/promise framing (persona voice)
3. **Closing Theme Execution** — Selected theme(s) in persona's distinctive style
4. **CTA Sequence** — 6-10+ asks with persona-appropriate emotional variety
5. **Action Instructions** — Step-by-step with persona voice
6. **Future Pacing/Crossroads** — Dream vs. nightmare in persona's style
7. **P.S. Section** — Using Makepeace techniques with persona interpretation
8. **Cialdini Integration** — Subtle commitment/consistency woven in persona voice

**Word Count Target:** 800-1,500 words per candidate (close is substantial but focused)

## 7 JUDGING CRITERIA

All candidates scored against these weighted criteria totaling 100%.

### Criterion 1: CTA Repetition Effectiveness (20%) — CRITICAL

**What It Measures:** Whether the 6-10+ CTAs build cumulative persuasive momentum through variety rather than mechanical repetition.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | 8+ CTAs, each with genuinely different emotional appeal, phrase, and supporting reason. Creates feeling: "NOW is the moment." No two CTAs could be confused. |
| 7-8 | 6-7 CTAs with clear variety. Most use different emotions/reasons. Repetition builds momentum. Minor overlap in 1-2 CTAs. |
| 5-6 | 6 CTAs present but variety is limited. Same emotional appeals recycled. Feels slightly mechanical. |
| 3-4 | Fewer than 6 CTAs OR most CTAs use identical/similar phrasing. Feels patronizing or robotic. |
| 1-2 | Minimal CTAs, nearly identical language. No emotional variety. Reader feels badgered, not persuaded. |

**Critical Threshold:** Score < 7 triggers REJECT. Makepeace methodology mandates 6+ varied asks.

**Variety Dimensions:**
- Emotional appeal (confidence, fear, urgency, greed, pride, belonging)
- Phrase structure (imperative, question, conditional, invitation)
- Supporting reason (benefit, consequence, exclusivity, guarantee, scarcity)
- Placement (embedded, standalone, sidebar, P.S.)

---

### Criterion 2: Guarantee Confidence Positioning (15%)

**What It Measures:** Whether the guarantee is presented as bold confidence expression rather than refund policy.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | "You must be absolutely delighted" framing. Guarantee has branded name. Company character statement present. Written as contract/promise, not exit option. Reader feels PROTECTED, not suspicious. |
| 7-8 | Strong confidence framing with branded guarantee. Minor moments that could be stronger. Overall positions guarantee as commitment. |
| 5-6 | Guarantee present and named, but framing leans toward "risk-free" rather than confidence. Adequate but not powerful. |
| 3-4 | Generic guarantee language creeping in. "Satisfaction guaranteed" type phrasing. Feels like policy, not promise. |
| 1-2 | Uses "if not satisfied, return for refund" or equivalent. Guarantee undermines rather than builds confidence. |

**Banned Phrases (Auto-Fail):**
- "If not satisfied, return for refund"
- "Satisfaction guaranteed" (unbranded)
- "Money-back guarantee" (unbranded)
- "Risk-free" (without confidence framing)

**Excellence Markers:**
- Branded guarantee name (e.g., "Triple-Back-Guarantee," "Iron-Clad Promise")
- "In the very unlikely event" framing
- "We don't deserve your money if..." company character
- "Rush" for refund speed
- Guarantee strengthens perception of product quality

---

### Criterion 3: Benefit Summary Impact (15%)

**What It Measures:** Whether the "You get" benefit summary re-ignites desire after the price presentation.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | 7+ "You get" bullets, each connecting benefit to strong reason why. Summarizes everything (product, bonuses, guarantee, access). Re-ignites desire powerfully. Reader thinks: "All of THAT for THIS price?" |
| 7-8 | 5-6 "You get" bullets with clear benefit-reason connections. Good summary coverage. Solid desire re-ignition. |
| 5-6 | 5 bullets present but reasons are weak or missing. Summary incomplete. Moderate impact. |
| 3-4 | Fewer than 5 bullets OR "You get" format not used consistently. Benefits listed without emotional connection. |
| 1-2 | Minimal benefit summary. No "You get" format. Fails to re-ignite desire. |

**"You Get" Formula:**
```
"You get [specific benefit] — [strong reason why this changes your life]"
```

---

### Criterion 4: Future Pacing/Crossroads Sincerity (15%)

**What It Measures:** Whether the dream vs. nightmare comparison and crossroads feel genuine rather than fabricated.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | Vivid, specific future states that feel genuinely helpful. Crossroads clarifies real options without manipulation. Decision ownership present. Reader thinks: "This person genuinely wants to help me decide." |
| 7-8 | Strong future pacing with niche-specific details. Crossroads feel mostly sincere. Minor moments of generic language. |
| 5-6 | Future pacing present but somewhat generic. Crossroads feel slightly forced. Adequate but not compelling. |
| 3-4 | Generic "imagine your life" language. Crossroads feel obviously manipulative. Reader senses fabrication. |
| 1-2 | No real future pacing. Crossroads feel cynical or fear-mongering. Damages trust. |

**Sincerity Test:**
- Would a genuinely helpful advisor say this?
- Is the negative path a REAL consequence or invented fear?
- Is decision ownership respected ("I can't make this decision for you")?
- Are the two paths specific to THIS offer and THIS reader's situation?

---

### Criterion 5: P.S. Section Impact (15%)

**What It Measures:** Whether the P.S. maximizes the "second-most-read section" real estate with powerful closing ammunition.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | P.S. uses 2+ Makepeace techniques. Each P.S. could standalone as powerful sell. Best ammo saved for P.S. Reader who only reads headline + P.S. gets compelling close. |
| 7-8 | P.S. uses 1-2 strong techniques. Good selling impact. Minor opportunities missed. |
| 5-6 | P.S. present but underutilizes the real estate. Generic bonus mention or weak technique application. |
| 3-4 | P.S. wastes real estate on administrative text ("P.S. Don't forget..."). Minimal selling impact. |
| 1-2 | No P.S. OR P.S. damages the close (confusion, contradictions, weak finish). |

**6 Makepeace P.S. Techniques:**
1. Fast-reply bonus (urgency)
2. Call-in bonus (phone orders)
3. Additional FREE bonus revealed
4. Most powerful testimonials
5. Additional reasons to act NOW
6. Guarantee restatement

---

### Criterion 6: Closing Theme Execution (10%)

**What It Measures:** How effectively the selected Makepeace closing theme(s) are executed.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | Theme executed with mastery. If crossroads: sincere and vivid. If logical restatement: airtight. If reasons why: compelling. Theme and secondary themes weave together seamlessly. |
| 7-8 | Theme clearly present and well-executed. Minor moments where theme could be stronger. Secondary themes integrated. |
| 5-6 | Theme recognizable but execution is average. Template-feeling rather than masterful. |
| 3-4 | Theme present but weak execution. Doesn't land. Theme feels tacked on rather than structural. |
| 1-2 | Theme not executed OR wrong theme for the situation. Close lacks cohesive structure. |

**7 Makepeace Themes (Reference):**
1. Crossroads (three options)
2. Logical restatement (facts chain)
3. Reasons why (condensed selling points)
4. Don't go it alone (expert guide essential)
5. Simple restatement (benefits + offer + guarantee)
6. But wait there's more (value additions)
7. USP close (exclusivity)

---

### Criterion 7: Cialdini Integration Subtlety (10%)

**What It Measures:** Whether commitment/consistency psychology is woven in subtly rather than heavy-handedly.

**Scoring Rubric:**

| Score | Description |
|-------|-------------|
| 9-10 | "Since you've read this far..." type nudges that feel natural, not manipulative. Reader doesn't consciously notice psychology but feels acting is consistent with everything they've agreed to. Masterful subtlety. |
| 7-8 | Cialdini present and mostly subtle. One or two moments slightly noticeable. Overall effective without feeling manipulative. |
| 5-6 | Cialdini present but somewhat obvious. Reader might notice the psychology. Not offensive but not elegant. |
| 3-4 | Cialdini heavy-handed. "You've already agreed that..." type language feels pushy. Damages trust. |
| 1-2 | No Cialdini integration OR so obvious it backfires. Reader feels manipulated. |

**Subtle Integration Examples:**
- "Since you've read this far, you must agree that..."
- "You've seen the evidence... the only question is..."
- "Everything you've learned points to one conclusion..."
- Implicit rather than explicit agreement references

```

                 (P.S. Impact × 0.15) + (Theme Execution × 0.10) +
```

## ARENA EXECUTION PROTOCOL

**See `~system/protocols/ARENA-CORE-PROTOCOL.md` for the complete 3-round execution protocol.**

> **Effort Level:** All generation phases use `effort: max`. Critique uses `effort: high`. See ~system/SYSTEM-CORE.md Effort Protocol.
> **Agent Team Mode:** When Agent Teams enabled, each persona runs as a separate teammate agent with full-draft generation in its own 200K context. See `~system/protocols/ARENA-CORE-PROTOCOL.md` v2.0 Agent Team Execution Mode.

This skill uses `arena_mode: generative_full_draft`:
- **Competitors write COMPLETE pieces from scratch** — NOT variations of a Layer 2 draft
- Layer 2 draft output = reference material and structural guide, NOT a template
- Upstream packages (root cause, mechanism, promise, big idea, structure) are the primary input
- Competitors are NOT constrained to follow the Layer 2 draft's specific approach
- **7 competitors** (6 personas + The Architect) generating independently
- **Adversarial critique** before scoring (The Critic identifies ONE weakest element per output)
- **Targeted revision** (each competitor fixes their identified weakness)
- **3 rounds** of competition with learning briefs between rounds
- **Post-arena synthesis** (Layer 2.6) creating 2-3 phrase-level hybrids
- **Human selection** from 9-10 candidates (7 pure + 2-3 hybrids)

### Full-Draft Mode Specifics
- Each competitor generates their OWN complete version from the upstream strategic packages
- The Layer 2 draft is available as ONE reference among many, not THE template
- Competitors may take radically different approaches (different structure, different emphasis, different tone)
- This produces TRUE creative diversity, not minor variations

### What Stays Skill-Specific (Below)
- 7 judging criteria with weights (used by both The Critic and the Judge)
- Persona generation instructions for this skill
- Critique-specific guidance for this skill
- Quality thresholds
- Anti-slop enforcement
- Input/output requirements

### Critique-Specific Guidance

**What The Critic should particularly target in Close Arena:**
- Fewer than 6 CTAs or identical CTA phrases
- Guarantee written as policy instead of confidence/contract
- Benefit summary under 5 "You get" items
- P.S. section that doesn't advance the close
- Crossroads/future pacing that feels fabricated

## GATE 2.5 SPECIFICATION

**Gate Type:** HUMAN_SELECT (BLOCKING)

**Entry Condition:** Gate 2 passes (all close sections drafted)

**Exit Condition:** Human explicitly selects a candidate

**Constraints:**
- No default selection permitted
- No timeout bypass permitted
- Human MUST actively choose
- Selection logged with rationale

**Failure Handling:**
- If all candidates < 8.5 threshold: Present all with warning, allow selection or regeneration
- If human requests regeneration: Execute with specific direction, return to judging
- If human requests hybrid: Validate element compatibility before assembly

## ANTI-SLOP ENFORCEMENT (CLOSE-SPECIFIC)

### Banned Close Phrases (All Candidates)

**CTA Poison:**
- "Click here" (naked)
- "Order now" (without urgency reason)
- "Buy today" (without consequence)
- "Don't wait" (without specific cost of waiting)
- "Take action" (vague)
- "Act now" (without justification)

**Guarantee Poison:**
- "If not satisfied, return for refund"
- "Satisfaction guaranteed" (unbranded)
- "Money-back guarantee" (unbranded)
- "Risk-free" (without confidence framing)
- "No risk" (naked)

**P.S. Poison:**
- "P.S. Don't forget..."
- "P.S. Remember..."
- "P.S. One more thing..." (unless adding real value)

**Future Pacing Poison:**
- "Imagine a life where..."
- "Picture yourself..." (without vivid specifics)
- Generic "dream" and "nightmare" without niche details

**Urgency Poison:**
- "Limited time offer" (without specifics)
- "Act now" (without consequence)
- "Hurry" (without deadline/supply reason)
- "Final chance" (without credibility)

### Validation Check

```
FOR each candidate:
    SCAN for banned phrases
    IF found:
        FLAG phrase location
        REQUIRE replacement before scoring
        DO NOT score candidate with poison phrases present
```

## MAKEPEACE ELEMENT COMPLETENESS AUDIT

Before any candidate advances past Arena, verify ALL 6 Makepeace elements:

| Element | Requirement | Audit Check |
|---------|-------------|-------------|
| 1. Benefit Summary | "You get" format, 5+ items | Count bullets, verify "You get" starts each |
| 2. Guarantee | Contract/promise framing, branded name | Check for banned phrases, verify branded name |
| 3. Ask for Sale | 6-10+ CTAs with variety | Count CTAs, assess emotional variety |
| 4. Action Instructions | Step-by-step, theme woven in | Verify explicit steps, benefit connection |
| 5. P.S. Section | Uses Makepeace techniques, real value | Identify technique(s), assess impact |
| 6. Sidebars | Instructions provided if format supports | Verify sidebar guidance included |

**Audit Result:**
- All 6 present and valid → Candidate eligible
- Any element missing → FLAG and require completion before selection
