> **Inputs:** Power-enhanced + specificity-enhanced candidates from 3.1, 3.2
> **Outputs:** Complete headline UNITS (headline + subheadline)

---

## FUNCTION

Generate SUBHEADLINES that complete each headline candidate. The subheadline is NOT an afterthought — it often does the HEAVY LIFTING that the headline can't do. Together, headline and subheadline form a complete promise unit.

**Key Principle:** The headline captures attention. The subheadline builds the promise. They work as a team, not as primary/secondary.

---

## SUBHEADLINE FUNCTIONS

### Function 1: PROOF INJECTION
Add credibility elements the headline couldn't fit.

```
HEADLINE: "Why Your Diet Keeps Failing"
SUBHEADLINE: "Harvard researchers discover the enzyme that blocks fat burning — and how to shut it off"

WHAT SUB DOES: Adds Harvard authority, mechanism tease, benefit hint
```

### Function 2: CURIOSITY EXTENSION
Expand the curiosity gap opened by headline.

```
HEADLINE: "The 17-Cent Circulation Miracle"
SUBHEADLINE: "How a WWII accident discovered by Navy doctors is now saving thousands from heart surgery"

WHAT SUB DOES: Adds story hook, authority, stakes
```

### Function 3: SPECIFICITY DELIVERY
Provide the specific numbers/claims headline teased.

```
HEADLINE: "Finally: A Golf Swing That Works for Seniors"
SUBHEADLINE: "Add 30 yards to your drive and eliminate slices in just 15 minutes — even if you're over 60 and haven't played in years"

WHAT SUB DOES: Delivers specific promise, qualifies audience, reduces risk perception
```

### Function 4: OBJECTION PREEMPTION
Address the skepticism the headline triggers.

```
HEADLINE: "I HAVEN'T HAD A GLASS OF WATER IN 20 YEARS"
SUBHEADLINE: "Sure, drinking tons of water is great if you want to raise your stroke risk, bring on kidney failure, and encourage early Alzheimer's."

WHAT SUB DOES: Flips objection into proof, creates schema distance payoff
```

### Function 5: EMOTIONAL AMPLIFICATION
Intensify the emotional response headline started.

```
HEADLINE: "Had Enough?"
SUBHEADLINE: "Escape the health nags forever — eat steak, eggs, real butter, and enjoy life guilt-free because everything they told you about health is JUNK MEDICINE backed by bad science"

WHAT SUB DOES: Delivers the promise the 2-word headline couldn't fit, amplifies liberation emotion
```

## EXECUTION PROTOCOL

### Step 1: Analyze Headline Gaps

```
FOR each refined headline:

  IDENTIFY what headline is MISSING:
    - Credibility/proof? → Need proof injection sub
    - Curiosity resolution? → Need extension sub
    - Specific promise? → Need specificity sub
    - Believability? → Need objection preemption sub
    - Emotional intensity? → Need amplification sub

  DETERMINE subheadline function priority
```

### Step 2: Match Subheadline Type

```
HEADLINE TYPE → SUBHEADLINE TYPE mapping:

  IF headline = curiosity question
    SUB = proof injection OR specificity delivery

  IF headline = bold claim
    SUB = objection preemption OR proof injection

  IF headline = story hook
    SUB = curiosity extension OR emotional amplification

  IF headline = warning
    SUB = specificity delivery OR emotional amplification

  IF headline = very short (1-5 words)
    SUB = MUST carry most of the promise weight
```

### Step 3: Generate Subheadline

```
FOR each headline:

  WRITE subheadline that:
    1. COMPLETES what headline started (not repeats)
    2. ADDS information headline couldn't fit
    3. MAINTAINS emotional trajectory
    4. CREATES momentum into body copy

  LENGTH GUIDELINES:
    - VSL: 10-20 words (can be spoken)
    - Print: 15-40 words (can be dense)
    - Email: No subheadline (subject line only)

  STRUCTURE OPTIONS:
    - Single sentence
    - Two short sentences
    - List preview (3 things you'll discover...)
    - Quote format (attributed statement)
```

### Step 4: Unit Coherence Check

```
READ headline + subheadline as SINGLE UNIT:

  CHECK:
    [ ] They flow naturally together
    [ ] Subheadline doesn't repeat headline
    [ ] Combined unit delivers complete promise
    [ ] Emotional trajectory is consistent
    [ ] Reader has CLEAR reason to continue reading

  IF checks fail:
    - Rewrite subheadline
    - Adjust word choice for flow
    - Ensure complementary, not redundant
```

## SUBHEADLINE PATTERN LIBRARY

### Pattern S1: Authority + Mechanism + Benefit
```
[AUTHORITY] discovers the [MECHANISM] that [BENEFIT] — and [HOW TO ACCESS]

EXAMPLE: "Harvard researchers discover the enzyme that blocks fat burning — and how to shut it off in 48 hours"
```

### Pattern S2: Story Hook + Stakes
```
How [UNEXPECTED ELEMENT] is now [HIGH-STAKES OUTCOME]

EXAMPLE: "How a WWII accident discovered by Navy doctors is now saving thousands from heart surgery"
```

### Pattern S3: Promise Stack
```
[BENEFIT 1] and [BENEFIT 2] in just [TIME] — even if [OBJECTION ADDRESSED]

EXAMPLE: "Add 30 yards and eliminate slices in just 15 minutes — even if you haven't played in years"
```

### Pattern S4: Objection Flip
```
Sure, [CONVENTIONAL WISDOM] is great if you want to [IRONIC BAD OUTCOME]

EXAMPLE: "Sure, drinking tons of water is great if you want to raise your stroke risk"
```

### Pattern S5: Numbers + Credibility Stack
```
[SPECIFIC NUMBER] [AUTHORITY] now [ACTION] because [REASON]

EXAMPLE: "Why 2.3 million Americans have abandoned their doctors' advice and are using this instead"
```

**UNIT ANALYSIS:**

**UNIT ANALYSIS:**

- This pattern REQUIRES heavy subheadline

## ANTI-PATTERNS (FORBIDDEN)

1. **Repetition**
   - Headline: "Lose Weight Fast"
   - Subheadline: "A fast way to lose weight"
   - PROBLEM: Subheadline adds nothing

2. **Contradiction**
   - Headline: "The SAFE Way to Invest"
   - Subheadline: "Why risky investments are your best bet"
   - PROBLEM: Conflicting messages

3. **Orphan Headline**
   - Writing headline without considering subheadline
   - They must be designed as a UNIT

4. **Subheadline Overload**
   - 50+ word subheadline trying to do everything
   - Pick ONE function and execute it well
