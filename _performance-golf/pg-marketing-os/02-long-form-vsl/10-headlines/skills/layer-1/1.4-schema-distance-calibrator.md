# 1.4-schema-distance-calibrator.md

> **Layer:** 1 (Architecture)
> **Skill:** 10-headlines
> **Purpose:** Set target schema distance and identify expectation to violate
> **Outputs:** `schema_calibration` with target and violation strategy

---

## FUNCTION

Calibrate the optimal schema distance for this headline — how much to violate reader expectations without causing confusion. Identify specific expectations to violate and design the resolution pathway.

---

## SCHEMA DISTANCE FRAMEWORK (RSF)

### Scale Definition

| Score | Description | Reader Response | Risk Level |
|-------|-------------|-----------------|------------|
| 1-3 | **Conventional** | "Same old thing" | Low attention capture |
| 4-5 | **Moderate** | "That's interesting" | Safe but forgettable |
| 6-7 | **Strong** | "Wait, what? Tell me more!" | **OPTIMAL ZONE** |
| 8 | **Significant** | "That's really surprising" | May need quick resolution |
| 9-10 | **Extreme** | "That doesn't make sense" | Confusion risk |

### Optimal Range: 6-7

Headlines in the 6-7 range achieve:
- Immediate attention capture
- Sufficient intrigue to continue reading
- Plausible enough to engage rather than dismiss
- Memorable differentiation from competitors

---

## CALIBRATION PROTOCOL

### Step 1: Inherit Target from Big Idea

```
CHECK upstream_context.big_idea.rsf_scores.schema_distance

IF available:
  INHERIT as starting_target
  NOTE: "Target inherited from Big Idea RSF analysis"

IF not available:
  SET starting_target = 6.5 (default optimal)
  NOTE: "Using default optimal target"
```

### Step 2: Adjust for Niche Tolerance

Different niches have different tolerance for schema violations:

```
NICHE_ADJUSTMENTS:
  health:
    - Audiences skeptical of "miracle" claims
    - Medical credibility required
    - ADJUST: -0.5 (more credible feel)

  financial:
    - Audiences expect bold predictions
    - Contrarian thinking valued
    - ADJUST: +0.5 (can push further)

  golf:
    - Audiences skeptical but hopeful
    - Specific claims valued
    - ADJUST: 0 (stay in optimal zone)

  self_help:
    - Audiences open to new paradigms
    - Transformation language expected
    - ADJUST: +0.5 (can push further)

APPLY adjustment to starting_target
```

### Step 3: Identify Expectation to Violate

```
COMMON VIOLATIONS BY NICHE:

health:
  - "Doctors usually know best" → Doctor MISSED something
  - "Medical procedures are superior" → Natural works BETTER
  - "Symptoms require drugs" → Root cause is different
  - "Healing takes time" → Results in [short timeframe]

financial:
  - "Markets are rational" → Hidden manipulation
  - "Diversification is safe" → Everything correlated
  - "Recovery is real" → Smoke and mirrors
  - "Past performance matters" → New paradigm coming

golf:
  - "Improvement takes practice" → Single fix works instantly
  - "Equipment matters most" → Body mechanics override
  - "Pro methods are complex" → Simple adjustment
  - "Age limits improvement" → Works at any age
```

### Step 4: Design Resolution Visibility

High schema distance requires visible resolution:

```
IF target_distance >= 7:
  REQUIRE: Clear resolution pathway visible in subheadline or deck
  EXAMPLE: "Better Than Botox?" + "A cream developed for stretch marks..."
           (The subheadline shows HOW this could be true)

IF target_distance >= 8:
  REQUIRE: Authority anchor in headline or immediate subheadline
  EXAMPLE: "Cardiologist reveals..." establishes credibility immediately

IF target_distance >= 9:
  WARN: High confusion risk
  REQUIRE: Immediate credibility + mechanism hint
  RECOMMEND: Consider reducing to 7-8 range
```

## FAILURE MODES

| Failure | Diagnosis | Remediation |
|---------|-----------|-------------|
| Distance too low (1-4) | Won't capture attention | Increase violation boldness |
| Distance too high (9-10) | Confusion risk | Add credibility anchor, soften claim |
| No clear violation | Sounds conventional | Identify what competitors DON'T say |
| Resolution not visible | Gap too abstract | Make subheadline explain "how" |
| Dismissal risk high | Claim unbelievable | Add authority or reduce distance |
