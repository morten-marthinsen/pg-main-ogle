# Layer 0.2.5 — Specimen Decomposer (Structural Pattern Extraction)

**Version:** 1.0
**Layer:** 0 (Foundation)
**Type:** Pattern Extraction
**Purpose:** Decompose Gold headline specimens into reusable structural patterns (rhythm, word choice, emotional triggers, format templates) that serve as building blocks during headline generation

---

## Why Decomposed Patterns Matter

Gold specimens loaded verbatim by 0.2.6 reshape the LLM's probability distribution through attention-weighted pattern transfer. But verbatim specimens alone are not enough. Decomposition extracts the **transferable structural DNA** -- the patterns that make specimens work -- so generation can recombine elements across specimen boundaries.

Without decomposition, generation tends to either:
- Copy specimens too closely (plagiarism risk)
- Ignore specimens entirely (no pattern transfer)

Decomposed patterns sit between these extremes: they provide **structural scaffolding** without dictating specific words.

## Decomposition Dimensions

Each Gold specimen is decomposed across 7 structural dimensions:

### Dimension 1: Rhythm Pattern

The syllable and stress pattern of the headline.

```
RHYTHM CLASSIFICATION:

  STACCATO:
    Short words, punchy rhythm, 1-2 syllables dominant.
    Example: "Blood in the Streets"
    Pattern: STRESS-rest-rest-STRESS (4 words, 4 syllables)

  ROLLING:
    Longer phrases, building momentum, clause-over-clause.
    Example: "Shield Your Wealth From Certain Annihilation And Grow Up To
             10 Times Richer In The Coming SLAUGHTER of The Stock Market"
    Pattern: Compound clause + "And" bridge + escalating clause

  CONVERSATIONAL:
    Natural speech rhythm, as if spoken aloud.
    Example: "They laughed when I called this Wall Street darling...
             'AMAZON.BOMB'"
    Pattern: Narrative voice + dramatic pause + punchline

  INTERROGATIVE:
    Question cadence that forces internal answer.
    Example: "Will This Be The End of America As We Know It?"
    Pattern: Question word + escalating stakes + identity anchor
```

### Dimension 2: Word Choice Architecture

The vocabulary strategy used to create impact.

```
WORD CHOICE PATTERNS:

  POWER WORD CLUSTERS:
    Violence/danger words: blood, slaughter, annihilation, bomb, savage
    Specificity words: numbers, dates, names, places
    Emotional trigger words: laughed, shocked, panicked, secret
    Authority words: doctor, scientist, physicist, world-famous

  CAPITALIZATION STRATEGY:
    SELECTIVE CAPS: Key words only (SLAUGHTER, BOMB, NINE TIMES)
    ALL CAPS: Entire headline for maximum impact
    Mixed case: Strategic emphasis on 1-3 words

  QUALIFIER PLACEMENT:
    Front-loaded: "World-Famous Trading Wizard RICHARD DENNIS"
    Embedded: "the most savage FINANCIAL SHOCK of the century"
    Trailing: "starting with just $400"
```

### Dimension 3: Emotional Trajectory

The emotional arc within the headline unit (headline + subheadline).

```
EMOTIONAL ARC PATTERNS:

  FEAR-TO-HOPE:
    Headline creates fear/threat → Subheadline offers escape/profit
    Example: "Shield Your Wealth From Certain Annihilation" →
             "And Grow Up To 10 Times Richer"

  RIDICULE-TO-VINDICATION:
    Headline establishes mockery → Resolution proves mocker wrong
    Example: "They laughed when I called this..." →
             "But the laughing stopped when the stock dropped 59%"

  CURIOSITY-TO-PROOF:
    Headline poses question → Subheadline stacks proof/authority
    Example: "Will This Be The End of America?" →
             Authority stack (Christie, 60 Minutes, Jim Rogers, Whitney)

  PROBLEM-TO-ACCESS:
    Headline names the desire → Subheadline promises access
    Example: "Discover the Amazing Health Secret" →
             "Dr. Rosenbaum's Natural Health Breakthroughs"

  IMPOSSIBILITY-TO-PROOF:
    Headline states something seemingly impossible → Subheadline proves it
    Example: "$400 → $200 million" →
             Dad quote humanizing the achievement
```

### Dimension 4: Specificity Element Placement

Where and how concrete details are deployed.

```
SPECIFICITY PATTERNS:

  TRIPLE STACK:
    3 specific numbers in rapid succession.
    Example: "59%, 45 days, 53%" (Amazon.Bomb)
    Example: "10 times, 7 crises, 53 million, 72 years" (Shield)

  RANGE ANCHOR:
    Believable range instead of single number.
    Example: "230 to 270 yards" (not "270 yards")
    Effect: Increases credibility by acknowledging variance

  TIME COMPRESSION:
    Specific short timeframe creating urgency.
    Example: "in just 30 minutes", "in just 7 days", "in just 45 days"
    Placement: Almost always at END of headline

  PRICE ANCHOR:
    Tiny price creating accessibility.
    Example: "$400" (starting investment), "17-Cent" (daily cost)
    Effect: Lowers perceived barrier

  GEOGRAPHIC ANCHOR:
    Specific location adding believability.
    Example: "Chicago scientist", "Japan's #1"
    Effect: Transforms generic claim into specific discovery
```

### Dimension 5: Promise Architecture

How the core promise is structured.

```
PROMISE STRUCTURES:

  DUAL PROMISE (Defense + Offense):
    "Shield Your Wealth" (defense) + "Grow 10 Times Richer" (offense)
    Template: [PROTECT from X] AND [GAIN Y]

  TRANSFORMATION ARC:
    "$400 → $200 million"
    Template: [STARTING STATE] → [END STATE]

  CONTRARIAN REVERSAL:
    "Vardon Grip is fundamentally flawed"
    Template: [SACRED COW] is [WRONG/FLAWED] — [BETTER ALTERNATIVE]

  SECRET ACCESS:
    "World-Famous Trading Wizard Reveals All"
    Template: [AUTHORITY] [REVEALS/SHARES] [GUARDED KNOWLEDGE]

  ASPIRATIONAL IDENTITY:
    "Longest-Living People on Earth"
    Template: [DESIRABLE GROUP] + [THEIR SECRET]
```

### Dimension 6: Format Template

The structural skeleton of the headline unit.

```
FORMAT TEMPLATES:

  QUESTION + AUTHORITY STACK:
    [Big Question]?
    [Authority 1 quote]
    [Authority 2 quote]
    [Authority 3 quote]
    [Warning/Disclaimer]

  THEY-LAUGHED NARRATIVE:
    They [mocked] when I [bold action]...
    [Dramatic name/label]
    ...But [vindication with specific proof]!
    [System/method credibility line]

  METAPHOR + PROMISE:
    [Visceral Metaphor]
    How To [Specific Promise]... Even as [Obstacle/Threat]

  DUAL-PROMISE COMPOUND:
    [Protect/Defend from THREAT] And [Gain/Grow BENEFIT]
    [Specificity stack] + [insider positioning]

  AUTHORITY REVEAL:
    [Title/Credential] [FAMOUS NAME] [Reveals/Shares] [All/Everything]!
    [Personal quote/anecdote humanizing the authority]

  GEOGRAPHIC SECRET:
    Discover [Benefit] of [Geographic/Cultural Group]
    [Doctor/Expert] + [Insider tease] + [Endorsement quote]

  SPECIFIC PROMISE + AUTHORITY:
    Finally... [new method] lets [target audience]
    [achieve specific result] in just [timeframe]
    [Authority credential] teaches [mechanism description]

  OUTSIDER SCIENTIST:
    [Location] [scientist/expert type] develops [new thing]...
    now [anyone] can [dual benefit] in just [timeframe]
    [Credential] proves [sacred cow] is [flawed]
```

### Dimension 7: Schema Distance Mechanism

How the headline creates surprise while remaining comprehensible.

```
SCHEMA VIOLATION TYPES:

  IDENTITY THREAT:
    Violates "my country/group is safe" assumption.
    Example: "End of America As We Know It" (distance: 9/10)

  AUTHORITY INVERSION:
    Non-expert proves experts wrong.
    Example: "Chicago scientist" beats golf pros (distance: 8/10)

  SACRED COW ATTACK:
    Challenges universally accepted practice.
    Example: "Vardon Grip is fundamentally flawed" (distance: 8/10)

  IMPOSSIBILITY CLAIM:
    States something that seems impossible but is documented.
    Example: "$400 → $200 million" (distance: 7/10)

  CATEGORY VIOLATION:
    Solution comes from unexpected category.
    Example: Stretch mark cream beats Botox (distance: 7/10)

  PARADOX RESOLUTION:
    Two opposing things are both true.
    Example: "Shield AND Grow" — safety + aggression (distance: 8/10)
```

## Pattern Recombination Rules

When generating headlines, use decomposed patterns as recombination building blocks:

```
RECOMBINATION PROTOCOL:

  1. SELECT a Promise Architecture that matches the Big Idea
  2. SELECT an Emotional Arc that matches the audience's emotional state
  3. SELECT a Format Template that can contain the promise + arc
  4. CALIBRATE Schema Distance to 6-7 range (optimal)
  5. INJECT Specificity Elements from campaign research
  6. APPLY Rhythm Pattern appropriate to format (print = rolling, VSL = conversational)
  7. DEPLOY Word Choice Architecture matching niche conventions

VALID RECOMBINATIONS (examples):
  - Contrarian Reversal + Ridicule→Vindication + They-Laughed + Sacred Cow Attack
  - Dual Promise + Fear→Hope + Metaphor + Promise + Paradox Resolution
  - Transformation Arc + Impossibility→Proof + Authority Reveal + Impossibility Claim

INVALID RECOMBINATIONS:
  - Aspirational Identity + Fear→Hope (identity aspiration does not pair with fear)
  - Secret Access + Staccato (secrets need room to breathe -- use rolling or conversational)
  - Contrarian Reversal + Geographic Secret (outsider reversal contradicts geographic tradition)
```

## Failure Modes

| Failure | Remediation |
|---------|-------------|
| Gold specimens not loaded from 0.2.6 | HALT -- cannot decompose what is not loaded. Run 0.2.6 first. |
| Specimen count < 3 | WARN -- limited decomposition diversity. Proceed but flag low pattern coverage. |
| Decomposition produces duplicate patterns across specimens | Expected for common patterns. Log frequency but do not deduplicate -- frequency IS data. |
| Pattern dimension unclear for a specimen | Classify as closest match. Note ambiguity. Do not leave dimensions empty. |
| Recombination matrix too large | Constrain to top 3 combinations per Big Idea type. Generation skills will filter further. |
