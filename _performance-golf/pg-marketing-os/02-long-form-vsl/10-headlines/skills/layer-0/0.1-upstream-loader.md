# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** 10-headlines
> **Purpose:** Load all upstream packages required for headline engineering
> **Outputs:** `upstream_context` object for downstream skills

---

## FUNCTION

Load and parse all upstream skill packages that inform headline generation. Validate presence and structure of required fields. Assemble unified context object.

---

## REQUIRED UPSTREAM PACKAGES

| Package | Source Skill | Critical Fields |
|---------|--------------|-----------------|
| `campaign-brief-package.json` | 09-campaign-brief | `big_idea`, `creative_direction`, `avatar`, `coherence_scores` |
| `big-idea-package.json` | 06-big-idea | `central_idea`, `big_idea_type`, `mechanism_angle`, `rsf_scores` |
| `mechanism-package.json` | 04-mechanism | `name`, `uniqueness_claim`, `credibility_anchors` |
| `promise-package.json` | 05-promise | `primary_promise`, `promise_ceiling`, `proof_pairings` |
| `root-cause-package.json` | 03-root-cause | `three_part_structure`, `villain`, `reframe` |

---

## LOADING PROTOCOL

```
FOR each required package:
  1. LOCATE: Find package in outputs/ directory of source skill
  2. PARSE: Load JSON and validate structure
  3. EXTRACT: Pull critical fields for headline use
  4. VALIDATE: Confirm non-null values for required fields
  5. STORE: Add to upstream_context object

IF any package missing:
  FAIL with specific error identifying missing package
  LOG remediation: "Package [X] not found. Ensure skill [Y] has completed successfully."

IF any critical field null/empty:
  WARN but continue with degraded mode flag
  LOG: "Field [X] in package [Y] is empty. Headline quality may be reduced."
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | big-idea-package missing | Cannot proceed — return to 06-big-idea |
| CRITICAL | campaign-brief-package missing | Cannot proceed — return to 09-campaign-brief |
| DEGRADED | mechanism-package missing | Proceed with Big Idea only, flag quality risk |
| DEGRADED | promise-package missing | Use Big Idea for promise, flag quality risk |
| DEGRADED | root-cause missing | Proceed without enemy angle, flag quality risk |

---

## EXECUTION

```
EXECUTE 0.1-upstream-loader:
  INPUT: skill output directories
  OUTPUT: upstream_context object

  1. Load campaign-brief-package.json
     EXTRACT: big_idea, creative_direction, avatar, coherence

  2. Load big-idea-package.json
     EXTRACT: central_idea, big_idea_type, mechanism_angle, rsf_scores

  3. Load mechanism-package.json
     EXTRACT: name, uniqueness_claim, credibility_anchors

  4. Load promise-package.json
     EXTRACT: primary_promise, promise_ceiling, promise_type

  5. Load root-cause-package.json
     EXTRACT: enemy, reframe, three_part_structure

  6. ASSEMBLE upstream_context
  7. SET loading_status
  8. RETURN upstream_context
```
