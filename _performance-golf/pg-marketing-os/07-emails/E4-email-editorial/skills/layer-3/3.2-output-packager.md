# Layer 3.2 -- Output Packager

## Purpose

Package the final revised email sequence and the editorial report into the two primary deliverables: `revised-sequence.yaml` (machine-readable sequence with all emails and metadata) and `EDITORIAL-REPORT.md` (human-readable report showing the editorial process, before/after comparisons, and quality improvements). These are the final outputs of the entire Email Pipeline.

---

## Input Requirements

```yaml
from_all_layers:
  loaded_sequence:
    campaign_meta: object  # from 0.1
    emails: [object]       # original emails from 0.1

  quality_baseline:
    per_email_scores: [object]  # from 0.2
    campaign_level: object       # from 0.2
    summary: object              # from 0.2

  audit_results:
    structural_audit: object   # from 1.1
    voice_consistency: object  # from 1.2
    pitch_ratio_check: object  # from 1.3
    sl_body_alignment: object  # from 1.4

  issue_clustering: object     # from 2.1
  revision_execution: object   # from 2.2

  rescore_results:
    per_email_comparison: [object]  # from 3.1
    campaign_level_comparison: object
    aggregate_metrics: object
    threshold_verification: object
```

## Execution Protocol

### Step 1: Verify Gate 3 Prerequisites

```
VERIFICATION:
  [ ] rescore_results.threshold_verification.all_emails_above_7_5 == true
  [ ] rescore_results.threshold_verification.all_campaign_criteria_pass == true
  [ ] rescore_results.threshold_verification.any_regressions == false (or regressions fixed)
  [ ] rescore_results.threshold_verification.gate_3_result == "PASS"

IF any check fails:
  HALT -- Cannot package. Return to appropriate layer to fix.
```

### Step 2: Build revised-sequence.yaml

Assemble the final sequence file:

```yaml

  generated_at: "[ISO timestamp]"

  campaign_meta:
    campaign_name: "[from E3]"
    campaign_type: "[from E3]"
    total_emails: [integer]

  emails:
    FOR EACH email in the revised sequence:
      - position: [integer]
        subject_line: "[final subject line]"
        body: "[full revised email body text -- COMPLETE, not truncated]"

        baseline_score: [float -- from 0.2]
        final_score: [float -- from 3.1]

        issues_found: [integer]
```

## Failure Modes

| Failure | Remediation |
|---------|-------------|
| Gate 3 prerequisites not met | HALT -- cannot package. Return to the layer that failed. |
| revised-sequence.yaml truncated (not all emails) | Re-write. Verify email count matches expected. |
| EDITORIAL-REPORT.md missing sections | Re-write with all required sections. |
| Human requests changes after review | Return to Layer 2 for specific fixes. Rescore. Re-package. |
| File write fails | Retry. If persistent, log error and attempt alternate path. |
