# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** EC-01-feature-naming
> **Purpose:** Load strategy + campaign brief + product research
> **Outputs:** `upstream_context` object with strategy scope and product data

---

## FUNCTION

Load the e-commerce strategy from EC-00 and the campaign brief (or standalone brief) that contains product capability data. Extract the feature research scope that EC-00 defined and the raw product data needed for capability mapping.

---

## LOADING PROTOCOL

### Step 1: Load Strategy

```
LOCATE: ecomm-strategy.yaml from EC-00 output
EXTRACT:
  feature_research_scope:
    hero_candidates: [list]
    supporting_candidates: [list]
    technical_features: [list]
    research_gaps: [list]
  section_map: [active sections that need features]
  word_budgets: [per-section budgets for feature density guidance]
  page_type: [pdp|collection|bundle]

IF strategy not found:
  FAIL -- "EC-00 strategy required. Execute EC-00 first."
```

### Step 2: Load Product Data

```
LOCATE: campaign-brief-package.json OR standalone brief
EXTRACT:
  product.features: [all listed features/capabilities]
  product.ingredients: [if applicable]
  product.technologies: [if applicable]
  product.materials: [if applicable]
  product.certifications: [if applicable]
  product.differentiators: [vs competitors]
  competitive_landscape: [what competitors offer]

IF product data missing:
  FAIL -- "Product capability data required for feature naming."
```

### Step 3: Merge and Deduplicate

```
MERGE strategy scope + product data:
  COMBINE hero_candidates from strategy with features from brief
  DEDUPLICATE overlapping entries
  FLAG capabilities in brief but NOT in strategy scope (potential additions)
  FLAG capabilities in strategy scope but NOT in brief (research gaps)
```

---

## OUTPUT STRUCTURE

```json
{
  "upstream_context": {
    "strategy_scope": {
      "hero_candidates": ["Feature A", "Feature B"],
      "supporting_candidates": ["Feature C", "Feature D"],
      "technical_features": ["Spec 1"],
      "research_gaps": ["Need info on X"]
    },
    "product_data": {
      "features": ["Feature A", "Feature B", "Feature C"],
      "ingredients": [],
      "technologies": [],
      "materials": [],
      "differentiators": []
    },
    "merged_capability_list": ["Capability 1", "Capability 2"],
    "loading_status": {
      "strategy_loaded": true,
      "product_data_loaded": true,
      "total_capabilities": 8
    }
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | EC-00 strategy not found | HALT -- execute EC-00 first |
| CRITICAL | No product capability data | HALT -- provide product brief |
| DEGRADED | Research gaps identified | Proceed, flag for human input |
| INFO | Strategy scope and brief don't fully overlap | Merge both, use union |
