# 1.3-highlights-selector.md

> **Layer:** 1 (Hero Architecture)
> **Skill:** EC-02-hero-value-prop
> **Purpose:** Select top 3-5 features for TLDR product highlights
> **Outputs:** `highlights_selection` object with selected features and rationale

---

## FUNCTION

Select 3-5 features from the EC-01 feature package to serve as the Product Highlights TLDR in the hero section. These are the "quick answers" that a scanning prospect sees immediately -- they must address the most common purchase questions.

---

## SELECTION PROTOCOL

### Step 1: Identify Purchase Questions

```
FOR the product and audience, IDENTIFY top purchase questions:
  "What does this product do?" -> Hero feature answers this
  "What's different about it?" -> Key differentiator feature
  "Does it actually work?" -> Proof-backed feature
  "Is it worth the price?" -> Value-supporting feature
  "What's in it / what's it made of?" -> Material/ingredient feature

MAP each question to the best-fit feature from EC-01 feature package
```

### Step 2: Rank Candidates

```
FOR each feature in EC-01 package:
  SCORE for highlights:
    purchase_question_match: [1-10] Does it answer a top purchase question?
    scan_friendliness: [1-10] Can the highlight be understood in 2 seconds?
    hero_reinforcement: [1-10] Does it reinforce the hero feature story?
    word_economy: [1-10] Can it be expressed in 10-20 words?

  RANK by total score
  SELECT top 3-5
```

### Step 3: Write Highlight Format

```
FOR each selected highlight:
  FORMAT:
    feature_name: [from EC-01, exact name]
    highlight_text: [10-20 word answer to the purchase question]

  CONSTRAINTS:
    - Use exact feature name from EC-01
    - 10-20 words per highlight
    - Answer format: "[Feature Name] -- [what it means for you]"
    - Total highlights: 50-100 words
    - Hero feature MUST be highlight #1
```

---

## OUTPUT STRUCTURE

```json
{
  "highlights_selection": {
    "total_selected": 4,
    "highlights": [
      {
        "rank": 1,
        "feature_name": "HyperSpeed Face",
        "highlight_text": "HyperSpeed Face delivers up to 15 more yards of carry -- even on mishits.",
        "purchase_question": "What does this product do?",
        "word_count": 14
      }
    ],
    "total_word_count": 56,
    "hero_feature_is_first": true
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Fewer than 3 features available | Use all available features |
| CRITICAL | Hero feature not selected as #1 | Force hero feature to position 1 |
| WARNING | Highlights exceed 100 words total | Compress individual highlights |
| INFO | More than 5 strong candidates | Select 5, note alternatives |
