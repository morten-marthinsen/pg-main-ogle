# 2.1-headline-subhead-generator.md

> **Layer:** 2 (Hero Generation)
> **Skill:** EC-02-hero-value-prop
> **Purpose:** Generate headline + subhead candidates
> **Outputs:** 5+ headline/subhead pairs

---

## FUNCTION

Generate 5+ headline/subhead pairs using the selected headline approaches from Layer 1. Each pair must work together -- the headline stops the scroll, the subhead expands with specificity. Ecom headlines are SHORT (5-12 words).

---

## GENERATION PROTOCOL

### Step 1: Generate Using Primary Approach (3+ pairs)
```
USING headline_approach.primary:
  GENERATE 3+ headline/subhead pairs
  EACH must:
    - Headline: 5-12 words, stops the scroll
    - Subhead: 10-20 words, expands headline (does NOT repeat)
    - Reference hero feature (name or concept)
    - Include at least 1 specific element (number, outcome, timeframe)

  ANTI-PATTERN: Subhead that restates headline in different words
    BAD: H: "More Distance. More Accuracy." S: "Get extra yards with better precision."
    GOOD: H: "More Distance. More Accuracy." S: "The HyperSpeed Face adds 15 yards while keeping your ball in the fairway."
```

### Step 2: Generate Using Secondary Approach (2+ pairs)
```
USING headline_approach.secondary:
  GENERATE 2+ headline/subhead pairs
  DIFFERENT ANGLE from primary approach
  This ensures diversity in hero options
```

### Step 3: Quality Screen All Pairs
```
FOR each pair:
  VERIFY:
    [ ] Headline word count: 5-12
    [ ] Subhead word count: 10-20
    [ ] Subhead EXPANDS (adds new info), does not REPEAT
    [ ] At least 1 specific element present
    [ ] Hero feature referenced (name or concept)
    [ ] No slop words (revolutionary, unlock, discover)
    [ ] Would stop a scrolling prospect

  IF any check fails: REVISE or DISCARD
```

### Step 4: Score Pairs
```
FOR each pair:
  SCORE:
    stopping_power: [1-10]
    subhead_expansion: [1-10] (how much NEW info subhead adds)
    specificity: [1-10]
    hero_feature_integration: [1-10]
    word_economy: [1-10]
    pair_score = average

  RANK pairs by score
  TOP 5 proceed to Arena
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Fewer than 5 viable pairs | Generate additional pairs |
| CRITICAL | All headlines exceed 12 words | Compress -- brevity is the discipline |
| WARNING | Subheads repeat headline content | Rewrite subheads with NEW information |
| WARNING | No specificity in any pair | Add numbers, outcomes, or timeframes |
