# 4.1-assembly-validator.md

> **Layer:** 4 (Validation)
> **Skill:** EC-05-ecomm-assembly
> **Purpose:** Cross-section consistency final check
> **Outputs:** `validation_result` with consistency report

---

## FUNCTION

Final cross-section consistency validation: verify tone consistency, voice consistency, feature name consistency across all assembled sections, and word budget compliance for the complete page.

---

## VALIDATION PROTOCOL

### Step 1: Feature Name Consistency (Full Page)
```
FROM feature-package.json, LOAD exact name registry
SCAN entire assembled page for feature references
CHECK every reference matches registry exactly
FLAG any paraphrasing, shortening, or invention
```

### Step 2: Tone Consistency
```
EVALUATE tone across all sections:
  [ ] Consistent voice (not formal in one section, casual in another)
  [ ] Consistent perspective (not "you" in some, "we" in others randomly)
  [ ] Consistent energy level (not high-energy hero, flat BTF)
  [ ] No sudden style shifts between sections
```

### Step 3: Total Word Count
```
SUM all section word counts
CHECK against EC-00 total page budget
IF over by > 10%: WARN
IF over by > 20%: FAIL
```

### Step 4: No Cross-References (Final Check)
```
SCAN assembled page for any cross-section references
("as mentioned above", "see below", "building on that")
IF found: FAIL -- these should have been caught in EC-03 but verify here
```

---

## OUTPUT STRUCTURE

```json
{
  "validation_result": {
    "overall_status": "PASS|FAIL",
    "feature_consistency": { "status": "PASS", "violations": 0 },
    "tone_consistency": { "status": "PASS", "issues": [] },
    "total_word_count": { "actual": 3150, "budget": 3200, "status": "PASS" },
    "cross_references": { "found": 0, "status": "PASS" }
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Feature name violations | Correct names in assembled copy |
| CRITICAL | Cross-references found | Remove and add built-in context |
| WARNING | Tone inconsistency | Flag for EC-06 editorial attention |
| WARNING | Word count over budget | Flag for EC-06 to compress |
