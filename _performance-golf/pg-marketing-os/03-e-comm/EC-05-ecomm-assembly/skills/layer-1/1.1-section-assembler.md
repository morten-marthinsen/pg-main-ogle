# 1.1-section-assembler.md

> **Layer:** 1 (Assembly)
> **Skill:** EC-05-ecomm-assembly
> **Purpose:** Assemble sections in priority order
> **Outputs:** `assembled_page` object with all sections in order

---

## FUNCTION

Assemble all copy sections into a single page document following the priority order from EC-00 strategy. Hero section first (from EC-02), then BTF sections (from EC-03), with micro-scripts placed at their mapped sections (from EC-04).

---

## ASSEMBLY PROTOCOL

### Step 1: Place Hero Section
```
INSERT hero copy (EC-02) as the first section
INCLUDE: headline, subhead, value prop, CTA, highlights, carousel copy
```

### Step 2: Place BTF Sections in Priority Order
```
FOR each section in assembly_order (excluding hero):
  LOCATE section copy from section-copy-package.json
  INSERT section in priority position
  ATTACH micro-script reference if one is mapped to this section
```

### Step 3: Place Micro-Script References
```
FOR each micro-script in micro-scripts.json:
  LOCATE mapped section
  INSERT script reference AFTER section copy:
    [VIDEO: script_type | duration | script_id]
```

### Step 4: Verify Completeness
```
CHECK:
  [ ] Hero section present
  [ ] All active BTF sections present
  [ ] Sections in correct priority order
  [ ] No duplicate sections
  [ ] No missing sections
  [ ] Micro-scripts placed at mapped sections
```

---

## OUTPUT STRUCTURE

```json
{
  "assembled_page": {
    "total_sections": 14,
    "assembly_order_followed": true,
    "sections": [
      { "position": 1, "section_number": 1, "section_name": "Hero", "source": "EC-02" },
      { "position": 2, "section_number": 4, "section_name": "Product Highlights", "source": "EC-03" }
    ],
    "micro_scripts_placed": 6,
    "completeness": "ALL_PRESENT"
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Section copy missing for an active section | HALT -- return to EC-03 |
| CRITICAL | Hero section missing | HALT -- return to EC-02 |
| WARNING | Micro-script mapped to non-existent section | Skip that micro-script, log warning |
