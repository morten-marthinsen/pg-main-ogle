# 1.2-feature-thread-checker.md

> **Layer:** 1 (Assembly)
> **Skill:** EC-05-ecomm-assembly
> **Purpose:** Verify hero feature carried through all sections
> **Outputs:** `thread_check_result` with feature mention map

---

## FUNCTION

Verify that the hero feature from EC-01 appears not just in the hero section but is threaded through the product page -- appearing in at least 2 BTF sections. The hero feature is the anchor of the product story; it must be reinforced throughout.

---

## CHECKING PROTOCOL

### Step 1: Identify Hero Feature
```
FROM feature-package.json:
  hero_feature_name = feature_hierarchy.hero[0].name
```

### Step 2: Scan All Sections
```
FOR each section in assembled_page:
  SCAN copy text for hero_feature_name (exact match)
  LOG:
    section_number
    mention_count
    context (how it's referenced)
```

### Step 3: Evaluate Thread Strength
```
ASSESS:
  hero_section_mention: [Y/N] (must be Y)
  btf_mention_count: [number of BTF sections mentioning hero feature]
  total_mentions: [across entire page]

  IF hero_section_mention == N:
    FAIL -- "Hero feature must appear in hero section"
  IF btf_mention_count < 2:
    WARN -- "Hero feature should appear in at least 2 BTF sections"
    SUGGEST sections where hero feature could be naturally referenced
  IF btf_mention_count >= 2:
    PASS -- "Hero feature thread is continuous"
```

---

## OUTPUT STRUCTURE

```json
{
  "thread_check_result": {
    "hero_feature": "HyperSpeed Face",
    "status": "PASS|WARN|FAIL",
    "hero_section_mention": true,
    "btf_mentions": [
      { "section_number": 4, "context": "Product highlights" },
      { "section_number": 10, "context": "How it works" },
      { "section_number": 14, "context": "Review references feature" }
    ],
    "btf_mention_count": 3,
    "total_mentions": 4,
    "thread_strength": "CONTINUOUS"
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Hero feature absent from hero section | Return to EC-02 |
| WARNING | Fewer than 2 BTF mentions | Suggest sections for natural inclusion |
| INFO | Strong thread (3+ BTF mentions) | Best case |
