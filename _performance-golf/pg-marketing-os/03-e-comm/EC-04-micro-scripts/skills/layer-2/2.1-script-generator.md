# 2.1-script-generator.md

> **Layer:** 2 (Script Generation)
> **Skill:** EC-04-micro-scripts
> **Purpose:** Generate scripts per section mapping
> **Outputs:** `generated_scripts` object with narration text per script

---

## FUNCTION

Write the narration/copy for each assigned micro-script within its duration budget. Each script follows its type-specific structure and references the appropriate features and section content.

---

## GENERATION PROTOCOL

### Step 1: Generate per Script Type

```
FOR each assigned script:

  HERO VIDEO (30-60s):
    0-5s: Hook -- attention-grabbing opening (problem or promise)
    5-15s: Product introduction + hero feature name
    15-35s: Top 2-3 benefits with visual proof references
    35-50s: Social proof element (customer count, review quote)
    50-60s: CTA -- where to buy, risk reversal hint

  FEATURE EXPLAINER (15-30s):
    0-5s: Feature name introduction
    5-15s: What it does (mechanism/function)
    15-25s: Why it matters (specific benefit)
    25-30s: Proof point or CTA

  UGC SCRIPT (15-30s):
    0-5s: Customer introduction (name, context)
    5-15s: Their experience/problem
    15-25s: Their result with the product
    25-30s: Recommendation

  HOW-TO (30-60s):
    0-5s: "Here's how to use [Product]"
    5-40s: Step-by-step demonstration (3-5 steps)
    40-55s: Expected result / what to notice
    55-60s: CTA

  COMPARISON (15-30s):
    0-5s: "Most [category] products do X"
    5-15s: "But [Product] does Y because of [Feature]"
    15-25s: Visual proof of difference
    25-30s: CTA
```

### Step 2: Word Count Enforcement

```
FOR each script:
  CHECK narration word count against word_budget
  IF over budget: COMPRESS -- cut lowest-value words
  IF significantly under: ADD detail or extend benefit statement
  TOLERANCE: +/- 10% of word budget
```

### Step 3: Feature Name Enforcement

```
FOR each script that references a feature:
  USE exact EC-01 feature name
  DO NOT paraphrase, shorten, or substitute
```

```json
      "feature_names_used": ["HyperSpeed Face"],

```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Narration exceeds word budget by >10% | Compress narration |
| WARNING | Feature name paraphrased | Correct to exact EC-01 name |
| WARNING | CTA missing from script | Add CTA to final 5 seconds |
