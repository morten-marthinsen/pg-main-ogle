# 0.2-section-mapping-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** EC-04-micro-scripts
> **Purpose:** Load section-to-script mapping
> **Outputs:** `script_mapping` object linking sections to script types

---

## FUNCTION

Define the mapping between page sections and script types. Each eligible section maps to a specific type of micro-script (hero video, feature explainer, UGC script, how-to, comparison).

---

## MAPPING PROTOCOL

### Step 1: Define Script Type Library
```
SCRIPT TYPES:
  hero_video:
    duration: 30-60s
    purpose: Product overview/demonstration
    section: Hero Carousel (1)
    content: Product intro + hero feature + top benefits + CTA

  feature_explainer:
    duration: 15-30s
    purpose: Single feature deep-dive
    sections: Product Highlights (4), Ingredients (7), Why It Works (10)
    content: Feature name + what it does + why it matters + proof

  ugc_script:
    duration: 15-30s
    purpose: Customer testimonial framework
    section: UGC Carousel (5)
    content: Customer setup + experience + result + recommendation

  how_to:
    duration: 30-60s
    purpose: Usage demonstration
    section: How to Use (13)
    content: Setup + steps + tips + expected result

  comparison:
    duration: 15-30s
    purpose: Side-by-side demonstration
    section: Comparison Chart (11)
    content: Category standard + this product difference + proof
```

### Step 2: Map Sections to Script Types
```
FOR each script_eligible_section:
  ASSIGN script_type based on section purpose
  SET default duration based on type
  IDENTIFY content sources (feature package, section copy)
```

---

## OUTPUT STRUCTURE

```json
{
  "script_mapping": [
    {
      "section_number": 1,
      "section_name": "Hero Carousel",
      "script_type": "hero_video",
      "default_duration": 45,
      "content_sources": ["feature_package.hero", "section_copy.hero"]
    }
  ],
  "total_scripts_planned": 6,
  "total_duration_planned": 195
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| INFO | Section doesn't need a script | Skip that section |
| WARNING | Too many scripts (>8) | Prioritize by section priority |
