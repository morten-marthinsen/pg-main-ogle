# 1.2-duration-budget-setter.md

> **Layer:** 1 (Script Architecture)
> **Skill:** EC-04-micro-scripts
> **Purpose:** Set duration per script (15-60s)
> **Outputs:** `duration_budgets` object with seconds per script

---

## FUNCTION

Set the duration budget for each assigned script. Duration determines word count (at ~2.5 words/second) and visual complexity.

---

## BUDGETING PROTOCOL

### Step 1: Set Duration by Script Type
```
DEFAULT DURATIONS:
  hero_video: 45s (range: 30-60s)
  feature_explainer: 20s (range: 15-30s)
  ugc_script: 20s (range: 15-30s)
  how_to: 45s (range: 30-60s)
  comparison: 20s (range: 15-30s)
```

### Step 2: Adjust for Content Depth
```
FOR each script:
  IF feature is complex (multiple components): ADD 10s
  IF feature is simple (single mechanism): USE default or SUBTRACT 5s
  IF section has strong visual potential: LEAN toward longer duration
  IF section has limited visual material: LEAN toward shorter duration

  ENFORCE: 15s minimum, 60s maximum
```

### Step 3: Calculate Word Budgets
```
FOR each script:
  word_budget = duration_seconds * 2.5
  EXAMPLE: 45s script = 112 words maximum narration
  EXAMPLE: 20s script = 50 words maximum narration
```

---

## OUTPUT STRUCTURE

```json
{
  "duration_budgets": [
    {
      "script_id": "MS-001",
      "section_number": 1,
      "script_type": "hero_video",
      "duration_seconds": 45,
      "word_budget": 112,
      "rationale": "Hero needs full product overview with feature demo"
    }
  ],
  "total_duration": 195,
  "total_word_budget": 487
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Any script exceeds 60s | Reduce to 60s or split into two scripts |
| WARNING | Total duration exceeds 5 minutes | Reduce script count or individual durations |
