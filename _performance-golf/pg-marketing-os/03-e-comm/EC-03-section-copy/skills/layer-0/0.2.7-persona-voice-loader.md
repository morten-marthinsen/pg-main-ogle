### Selection Priority for This Skill

Since this skill generates SECTION COPY across multiple section types (problem, mechanism, offer, proof, product), specimen selection must balance breadth with depth. Unlike skills that generate one type of copy (e.g., headlines), section copy requires voice calibration across persuasion, proof, mechanism explanation, and offer framing.

**Key challenge:** Section copy is SCANNED, not READ. Specimens from long-form masters must be read with an eye toward how their voice COMPRESSES — which phrases, rhythms, and techniques survive when paragraphs become 2-3 sentences and sections become standalone islands.
