# 4.2-feature-consistency-checker.md

> **Layer:** 4 (Validation)
> **Skill:** EC-03-section-copy
> **Purpose:** Feature names from EC-01 used correctly?
> **Outputs:** `consistency_result` object with PASS/FAIL

---

## FUNCTION

Verify that every feature name reference in section copy matches the exact name from the EC-01 feature package. No paraphrasing, no shortening, no synonym substitution. Feature names are brand assets -- they must be consistent across the entire page.

---

## CHECKING PROTOCOL

### Step 1: Build Feature Name Registry

```
FROM feature-package.json:
  BUILD registry of exact feature names:
    ["HyperSpeed Face", "Dynamic Loft Control", "Triple-Layer Cushioning", ...]

  INCLUDE name variants that are acceptable:
    With "the": "the HyperSpeed Face" -> OK
    With "our": "our HyperSpeed Face" -> OK (if brand voice supports)
    Standalone: "HyperSpeed Face" -> OK
```

### Step 2: Scan All Section Copy

```
FOR each section's copy:
  SCAN for feature references:
    IDENTIFY every mention of a product feature, technology, or ingredient
    COMPARE against registry

  DETECT violations:
    PARAPHRASE: "the high-speed face" instead of "HyperSpeed Face"
    SHORTENING: "HyperSpeed" alone when "HyperSpeed Face" is the name
    SYNONYM: "speed-optimized face technology" instead of "HyperSpeed Face"
    INVENTION: A feature name that doesn't exist in EC-01 package
```

### Step 3: Report Violations

```
FOR each violation:
  REPORT:
    section_number: [where the violation is]
    violation_type: [paraphrase|shortening|synonym|invention]
    found_text: [what was written]
    correct_name: [exact EC-01 name]
    location_in_section: [approximate position]

  SEVERITY:
    Paraphrase: HIGH -- must fix
    Shortening: MEDIUM -- may be acceptable in some contexts
    Synonym: HIGH -- must fix
    Invention: CRITICAL -- must remove or add to EC-01 package
```

---

## OUTPUT STRUCTURE

```json
{
  "consistency_result": {
    "overall_status": "PASS|FAIL",
    "total_feature_references": 24,
    "exact_matches": 24,
    "violations": 0,
    "violation_details": [],
    "feature_usage_map": {
      "HyperSpeed Face": { "count": 6, "sections": [1, 4, 9, 10, 14, 16] },
      "Dynamic Loft Control": { "count": 3, "sections": [4, 10, 15] }
    }
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Invented feature name found | Remove or add to EC-01 package |
| CRITICAL | Multiple paraphrasing violations | Return to section copy for correction |
| WARNING | Single shortening violation | May be acceptable -- human judgment |
| INFO | All names match exactly | Proceed to packaging |
