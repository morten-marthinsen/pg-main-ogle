# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** EC-06-ecomm-editorial
> **Purpose:** Load assembled e-commerce copy from EC-05
> **Outputs:** `assembled_copy` object with full page copy and section metadata

---

## FUNCTION

Load the assembled e-commerce copy from EC-05, including all section content, design notes, word counts, feature references, and the page builder handoff document. This is the primary input that editorial will revise.

---

## LOADING PROTOCOL

### Step 1: Load Assembled Copy
```
LOCATE: ecomm-copy-assembled.md from EC-05
EXTRACT:
  all_sections: [full copy text per section]
  section_metadata: [name, priority, word count, template, proof elements]
  design_notes: [layout, mobile, visuals per section]
  total_word_count: [page total]
  total_sections: [count]
```

### Step 2: Load Page Builder Handoff
```
LOCATE: page-builder-handoff.yaml from EC-05
EXTRACT:
  page_spec: [page type, scroll depth, mobile priority sections]
  cta_placements: [section numbers]
  video_placements: [section numbers with micro-scripts]
```

### Step 3: Load Feature Package
```
LOCATE: feature-package.json from EC-01
EXTRACT:
  feature_registry: [all exact feature names]
  feature_hierarchy: [hero, supporting, technical]
  micro_scripts: [per feature, for consistency reference]
```

### Step 4: Build Editorial Working Copy
```
FOR each section in assembled copy:
  BUILD editorial_section:
    section_number: [N]
    section_name: [name]
    priority: [from EC-00]
    current_copy: [full text from EC-05]
    current_word_count: [actual]
    word_budget: [from EC-00]
    design_notes: [current notes]
    proof_elements: [list of proofs in this section]
    feature_names_used: [list of EC-01 names referenced]
    editorial_status: "pending_audit"
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | EC-05 assembled copy not found | HALT -- execute EC-05 |
| CRITICAL | EC-01 feature package not found | HALT -- execute EC-01 |
| WARNING | Page builder handoff missing | Proceed -- editorial can work without it |
