### Selection Priority for This Skill

Since this skill uses `editorial_revision` arena mode, load a **mix of teaching and selling specimens** for voice reference during revision. Editorial doesn't generate from scratch — it EDITS existing copy. Voice specimens serve as CALIBRATION for quality judgment: "Does this section SOUND like it should?"

For ecom editorial specifically, specimens serve a dual purpose:
1. **Voice reference** — Does the revised copy maintain the quality standard these masters represent?
2. **Craft detection** — Would this master catch the issue the audit flagged? What would they tighten, strengthen, or cut?

## Editorial-Specific Specimen Application

Unlike generative skills where specimens guide CREATION, editorial specimens guide JUDGMENT:

| Editorial Function | Specimen Application |
|-------------------|---------------------|
| Scan Optimization Audit | Would this master's copy scan this poorly? If their copy is tighter, this section needs work |
| Proof Density Review | Does this section have the proof density of a Bencivenga letter or an Ogilvy ad? If not, proof is thin |
| Feature Consistency Check | How consistently do these masters use their proprietary terms? That's the standard |
| DR Principles Audit | How do these masters handle offers, guarantees, and CTAs? That's the quality bar |
| Design-Copy Integration | How do these masters write copy that implies layout? (Ogilvy's ads are inseparable from their design) |
| Mobile-First Review | Which specimen patterns survive mobile compression? Short sentences, bold phrases, clear hierarchy |
