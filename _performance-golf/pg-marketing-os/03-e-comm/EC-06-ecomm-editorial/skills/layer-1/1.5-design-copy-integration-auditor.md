# 1.5-design-copy-integration-auditor.md

> **Layer:** 1 (Audit)
> **Skill:** EC-06-ecomm-editorial
> **Purpose:** Audit design note presence, specificity, and copy-layout integration
> **Outputs:** Design-copy integration audit report with section-by-section assessment

---

## FUNCTION

Verify that every section has actionable design notes and that the copy is written with its intended layout in mind. E-comm copy is inseparable from layout — a 3-column feature grid needs 3 equal-length descriptions, not prose. A page builder should be able to implement any section from copy + design notes alone, without asking questions.

---

## AUDIT PROTOCOL

### Check 1: Design Notes Presence
```
FOR each section:
  CHECK:
    [ ] Layout spec present (single-column, two-column, card grid, carousel, etc.)
    [ ] Mobile behavior defined (stack, carousel, accordion, hide, reorder, etc.)
    [ ] Visual media specified (product shot, lifestyle image, icon set, video embed, etc.)

  IF any of the 3 specs missing: FLAG
  IF all 3 missing: CRITICAL FLAG
```

### Check 2: Design Notes Specificity
```
FOR each section with design notes present:
  EVALUATE specificity:
    VAGUE: "nice layout" / "good design" / "make it look professional" → FLAG
    GENERIC: "two-column layout" (without specifying which column gets what) → FLAG
    SPECIFIC: "two-column: left column = product image (60%), right column = copy (40%)" → PASS

  SPECIFICITY TEST:
    "Could a designer implement this section from these notes alone,
     without asking a single question?"
    YES → PASS
    NO → FLAG with what's missing
```

### Check 3: Copy-Layout Fit
```
FOR each section:
  CHECK copy-layout compatibility:
    [ ] If layout = card grid: all cards have similar copy length
    [ ] If layout = carousel: each slide has standalone copy
    [ ] If layout = two-column: copy fits the specified column width
    [ ] If layout = icon + text: text is brief enough for icon-adjacent display
    [ ] If layout = full-width: copy is structured with headers/bullets for scanability

  IF copy doesn't fit the specified layout: FLAG with specific mismatch
```

### Check 4: Image/Video Integration
```
FOR each section with visual media:
  CHECK:
    [ ] Copy references or complements the visual (not competing with it)
    [ ] If video is specified: video placement and copy placement don't overlap
    [ ] If before/after images: copy references the comparison
    [ ] If product shots: copy describes what the image shows (not duplicating)

  IF copy and visuals are disconnected: FLAG
```

## MINIMUM THRESHOLDS

- Design notes present: 100% of sections
- Design notes specific (passes designer test): >= 80% of sections
- Copy-layout fit: >= 90% of sections
- Any section with CRITICAL missing notes: revision mandatory
