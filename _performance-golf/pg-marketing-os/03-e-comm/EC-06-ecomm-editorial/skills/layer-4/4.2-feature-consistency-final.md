# 4.2-feature-consistency-final.md

> **Layer:** 4 (Scoring & Packaging)
> **Skill:** EC-06-ecomm-editorial
> **Purpose:** Final feature naming consistency pass across entire page
> **Outputs:** `consistency_result` object with deviation report

---

## FUNCTION

Perform the final, definitive feature naming consistency check across the entire editorially revised page. This is the last chance to catch feature name deviations before the page goes to the page builder. Every single reference to a product feature must match the EC-01 registry exactly -- no paraphrasing, no shortening, no invention.

---

## CONSISTENCY PROTOCOL

### Step 1: Load Feature Registry
```
FROM 0.1-upstream-loader:
  LOAD feature_registry: [all exact feature names from EC-01]
  LOAD feature_hierarchy: [hero, supporting, technical]
  LOAD micro_scripts: [per-feature reference text]

BUILD exact_match_list:
  FOR each feature in registry:
    exact_name: [the locked name]
    common_variations: [known paraphrases to watch for]
    section_appearances: [which sections should reference this feature]
```

### Step 2: Full Page Scan
```
FOR each section in revised page copy:
  SCAN for any word or phrase that:
    - Matches a feature name exactly (GOOD)
    - Partially matches a feature name (FLAG)
    - Paraphrases a feature name (FLAG)
    - Shortens a feature name (FLAG)
    - Introduces a feature name not in registry (FLAG)

  TRACK:
    exact_matches: [count]
    deviations: [list with location and type]
```

### Step 3: Deviation Classification
```
DEVIATION TYPES:
  PARAPHRASE: Feature name restated differently
    Example: "HyperSpeed Face" -> "the high-speed face"
    Action: REPLACE with exact name

  SHORTENING: Feature name abbreviated
    Example: "ForgeSteel Construction" -> "ForgeSteel"
    Action: REPLACE with full exact name

  INVENTION: New feature name not in registry
    Example: "PowerDrive Technology" (doesn't exist in EC-01)
    Action: REMOVE or replace with correct feature name

  INCONSISTENT_FORMAT: Right name, wrong formatting
    Example: "hyperSpeed face" (wrong capitalization)
    Action: CORRECT formatting to match registry
```

### Step 4: Generate Correction List
```
FOR each deviation:
  DOCUMENT:
    section_number: [where found]
    current_text: [what's written]
    correct_text: [exact EC-01 name]
    deviation_type: [paraphrase/shortening/invention/format]
    action: [specific correction]
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Any INVENTION deviation found | Remove invented name, replace with registry name |
| CRITICAL | 3+ deviations of any type | Systematic name correction pass |
| WARNING | 1-2 PARAPHRASE or SHORTENING deviations | Correct specific instances |
| INFO | Zero deviations | Perfect consistency confirmed |
