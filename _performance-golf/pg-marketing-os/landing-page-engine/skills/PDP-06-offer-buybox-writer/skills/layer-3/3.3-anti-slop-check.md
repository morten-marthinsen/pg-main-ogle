# 3.3 — Anti-Slop Check

**Skill:** PDP-06-offer-buybox-writer
**Position:** Layer 3, Microskill 3
**Purpose:** Scan every piece of buy box copy for generic AI language. Zero tolerance. No "revolutionary", "game-changing", "unlock", "transform", "journey", "elevate", "supercharge", "harness the power", "cutting-edge", "state-of-the-art", or any other telltale AI phrasing. Buy box copy must be functional, specific, and factual.
**Effort Level:** low
**Dependencies:** All Layer 1 and Layer 2 output files
**Output:** `anti-slop-check.md`

---

## Input Schema

```yaml
input:
  pricing_copy: "[path to pricing-copy.md]"
  variant_copy: "[path to variant-copy.md]"
  subscription_copy: "[path to subscription-copy.md]"
  quantity_copy: "[path to quantity-copy.md]"
  short_description_copy: "[path to short-description-copy.md]"

  sticky_atc_copy: "[path to sticky-atc-copy.md]"

```

## Execution Logic

**Step 1:** Define the forbidden word list:

```
FORBIDDEN WORDS AND PHRASES:
  Tier 1 (absolute ban -- zero tolerance):
    revolutionary, game-changing, game changer, cutting-edge, state-of-the-art,
    groundbreaking, transformative, transform, unlock, unleash, supercharge,
    elevate, harness, harness the power, next-level, next level,
    empower, empowering, journey, seamless, seamlessly,
    robust, leverage (as verb), synergy, paradigm, holistic,
    disruptive, innovative (without evidence), premium quality (as standalone),
    best in class, world-class, second to none

  Tier 2 (flagged -- replace with specifics):
    high quality, top quality, amazing, incredible, unbelievable,
    powerful, ultimate, perfect, guaranteed results,
    100% satisfaction (without specific guarantee terms),
    trusted by thousands (without specific numbers)
```

**Step 2:** Read each copy file and scan for forbidden words/phrases.

**Step 3:** For each file, document:
- File name
- Any forbidden words found (Tier 1 or Tier 2)
- Exact line/context where each word appears
- Severity (Tier 1 = must fix, Tier 2 = should fix)

**Step 4:** Pay special attention to:
- `short-description-copy.md` (highest risk -- prose copy, most likely to have AI telltales)
- `micro-trust-copy.md` (medium risk -- could have generic trust language)
- `mini-cart-copy.md` (low risk -- mostly structural, but cross-sell line could slip)
- Pricing/ATC files (lowest risk -- mostly numbers and standard labels)

**Step 5:** For every Tier 1 violation, provide a specific rewrite suggestion:
```
Violation: "revolutionary formula"
Rewrite:   "clinically studied formula" or "[specific ingredient] formula"

Violation: "unlock better sleep"
Rewrite:   "supports deep, uninterrupted sleep"

Violation: "premium quality ingredients"
Rewrite:   "Third-Party Tested ingredients" or "GMP-certified formula"
```

**Step 6:** Count total violations:
- Tier 1 count: must be 0 for PASS
- Tier 2 count: should be 0, flagged but not blocking

**Step 7:** Write `anti-slop-check.md`.

## Quality Criteria
- Every copy file scanned (no files skipped)
- Tier 1 violations = 0 for PASS
- Every violation has a specific rewrite suggestion (not "make it less generic")
- Context provided for each violation (so the rewrite targets the right line)
- No false positives (e.g., "Add to Cart" is not flagged just because it contains "transform" as a substring)
- File exists after execution

## Failure Modes
- Tier 1 violations found -> FAIL; provide rewrites; offending files must be updated before Layer 4
- Tier 2 flags found -> PASS with warnings; document suggested improvements
- Copy file missing -> Cannot scan; flag as Layer 1/2 incompletion (separate from slop check)
- False positive in scan -> Verify word is used in the forbidden sense, not legitimate context (e.g., "transform" in "data transform" would be a false positive, but this is buy box copy so unlikely)
