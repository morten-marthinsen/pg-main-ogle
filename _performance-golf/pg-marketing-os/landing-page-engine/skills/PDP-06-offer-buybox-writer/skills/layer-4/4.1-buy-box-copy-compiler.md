# 4.1 — Buy Box Copy Package Compiler

**Skill:** PDP-06-offer-buybox-writer
**Position:** Layer 4, Microskill 1
**Purpose:** Assemble all buy box copy from Layer 1 and Layer 2 outputs into a single structured JSON file (pdp-offer-buy-box.json) that downstream skills (PDP-07, LP-15, PDP-16) can consume. This is the primary deliverable of PDP-06.
**Effort Level:** low
**Dependencies:** All Layer 1 outputs, All Layer 2 outputs, 3.1 buy-box-copy-audit.md (must PASS), 3.2 cta-consistency-check.md (must PASS), 3.3 anti-slop-check.md (must PASS)
**Output:** `pdp-offer-buy-box.json`

## Output Schema

The output IS the JSON structure defined in Step 3 above, written to `pdp-offer-buy-box.json`.

---

## Quality Criteria
- JSON is valid and parseable
- All Layer 1 and Layer 2 copy is included (nothing dropped)
- Prices are internally consistent (same price appears same everywhere)
- ATC labels are internally consistent
- audit_results section confirms all audits passed
- File exists after execution

## Failure Modes
- Any Layer 3 audit is FAIL -> HALT, do not assemble package. Fix failures first.
- Copy file missing from Layer 1 or 2 -> HALT, Layer 3 should have caught this
- JSON validation error -> Fix structure and re-write
- Data inconsistency between sections -> Reconcile using source-of-truth files (pricing-copy.md for prices, atc-copy.md for labels)
