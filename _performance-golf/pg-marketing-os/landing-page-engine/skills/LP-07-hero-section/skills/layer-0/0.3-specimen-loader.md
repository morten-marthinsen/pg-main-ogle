# 0.3 — Specimen Loader

**Skill:** LP-07-hero-section
**Position:** Layer 0, Microskill 3
**Purpose:** Load type-matched hero section specimens before any generation begins. Specimens are verbatim examples of elite headlines, leads, and above-fold copy that function as structural attractors during generation — the same principle as the CopywritingEngine's 0.2.6 curated gold specimens.
**Effort Level:** low
**Dependencies:** `packages-loaded.md` (0.1), `page-brief.json` (page type + vertical)
**Output:** `specimens-loaded.md`

---

## Input Schema

```yaml
input:
  page_type: "[type_a | type_b | hybrid]"
  vertical: "[from page-brief.json]"
  lead_type_pending: "[will be confirmed in Layer 1 — use for pre-loading if known from LP-03]"
  headline_type_pending: "[from LP-03 above-fold blueprint]"
```

---

## Specimen Loading Protocol

**Step 1:** Check for vertical-specific specimens in:
```
landing-page-engine/specimens/long-form-sales-pages/[vertical]/
landing-page-engine/specimens/ecomm-pdp/[vertical]/
```

**Step 2:** If vertical-specific specimens directory exists and has files → Load from there.

**Step 3:** If directory is empty or doesn't exist → Use FALLBACK specimens from the swipe vault reference list below.

**Step 4:** Load minimum quantities:
- Type A: 3 headline examples + 2 lead opening examples (first 150 words each)
- Type B: 3 product title + short description examples + 2 key benefits sets

## FALLBACK SPECIMEN REFERENCE (Use When Specimen Directory Empty)

These are reference examples from the research vault (Trello research board). Load the conceptual pattern, not verbatim text, when actual specimens are unavailable.

### Type A — Long-Form Headline Specimens by Vertical

**Health/Supplements:**
- Hormonal Harmony: Problem-first headline targeting women 40+ with specific symptom language
- SANE Solution Leaky Brain: Discovery headline with authority framing and mechanism hint
- BioTrust Ageless Multi-Collagen: Benefit + mechanism hint headline
- Be-Serene VSL: Curiosity/revelation headline with unexpected cause framing

**Info Products:**
- Todd Brown E5 VSL: How-to discovery headline with specific mechanism name
- Profits Unlimited (Agora): Warning/revelation headline with financial specifics

**Pattern to extract from each:** Headline structure, specificity level, curiosity gap technique, awareness stage targeting.

### Type A — Lead Opening Specimens

**Story Lead examples:**
- Classic health DR opening: Drop into a specific scene (doctor's office, experiment, unexpected moment)
- Financial DR opening: Drop into a market event or personal crisis moment

**Question Lead examples:**
- "If you've tried [X] and still [problem], keep reading..."
- "Are you one of the [number] [demographic] who [problem]?"

**Agitation Lead examples:**
- Vivid description of the worst moment of the problem
- The feeling of failure after trying "everything"

### Type B — Product Title + Description Specimens

**Health/Supplements DTC:**
- Magnesium Breakthrough: Named-product + benefit expansion below in description
- Athletic Greens / AG1: Product name + one-sentence total benefit claim
- Seed Daily Symbiotic: Clinical name + consumer-friendly benefit statement

**Pattern to extract:** How benefit is communicated alongside product name; how specificity is achieved in 10–15 words.

## Quality Criteria
- Minimum 3 headline specimens for Type A
- Minimum 2 lead opening specimens for Type A
- Minimum 3 product title examples for Type B
- All specimens loaded before any generation begins (this is the gate)
- Each specimen has an identified "key technique" — AI must extract what makes it work, not just copy it

## Failure Modes
- Both specimen directory AND FALLBACK examples unavailable → Proceed with generation but flag: "No specimens loaded — generation will rely on pattern knowledge only. Quality ceiling may be lower."
- Specimens loaded from wrong vertical → Re-load from correct vertical. Cross-vertical contamination degrades quality.
