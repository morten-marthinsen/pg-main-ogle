# 2.5 — Micro-CTA Writer

> Version: 1.0
> Layer: 2
> Model: sonnet
> Input: cta-type-plan.md, primary-cta-candidates.md, cta-architecture-loaded.md
> Output: micro-cta-candidates.md

**Skill:** LP-14-cta-optimizer
**Position:** Layer 2, Microskill 5
**Purpose:** Write inline/contextual micro-CTAs for placement between content sections. These are NOT button CTAs — they are text-based calls to action embedded in the copy flow. They serve as soft conversion prompts for readers who are convinced mid-page and do not want to scroll to the bottom. Primarily for Type A and Hybrid pages.
**Effort Level:** medium
**Dependencies:** `cta-type-plan.md` (1.1), `primary-cta-candidates.md` (2.1), `cta-architecture-loaded.md` (0.2)
**Output:** `micro-cta-candidates.md`

## Execution Logic

1. **Check page type.** Micro-CTAs are primarily for Type A and Hybrid pages. Type B pages typically have fewer sections and use button CTAs at natural positions instead. If Type B, generate 1-2 optional micro-CTAs and note they may not be used.

2. **Identify micro-CTA insertion points.** Micro-CTAs go between content sections where a reader might be ready to act. Common insertion points:
   - After the mechanism explanation (reader now understands WHY)
   - After the first major proof block (reader now BELIEVES)
   - After a particularly powerful testimonial (reader now RELATES)
   - Before the offer reveal (building anticipation)
   - After a benefit stack (reader sees the full value)

3. **For each insertion point, write 1-2 micro-CTA variants:**

   **Micro-CTA Types:**

   **Type M1: Inline Text CTA**
   - Format: 1 sentence embedded in copy flow, often italicized or set apart
   - Pattern: "If you're ready to [benefit], [CTA action] below."
   - Example: *"If you've seen enough and you're ready to start sleeping deeper tonight, click the button below to get your first bottle."*

   **Type M2: Parenthetical CTA**
   - Format: Parenthetical aside within a section
   - Pattern: "([CTA action] if you're ready — or keep reading for [what comes next].)"
   - Example: *(Ready to try it? Get your bottle below — or keep reading to see the clinical results.)*

   **Type M3: Bridge CTA**
   - Format: Transitional sentence between sections that includes a soft CTA
   - Pattern: "Before I show you [next section content], I want to make sure you know you can [CTA action] right now."
   - Example: "Before I share what Dr. [Name] discovered in his clinic, I should mention — you can get started with this formula right now by clicking below."

4. **Rules for micro-CTAs:**
   - Maximum 2 sentences each
   - Must feel natural in the copy flow (not a jarring interruption)
   - Must reference the same single offer as the primary CTA
   - Must be skippable — the reader who is not ready should be able to flow past without friction
   - Should use softer language than button CTAs ("if you're ready" / "whenever you'd like")
   - Do NOT include urgency language in micro-CTAs (save urgency for terminal CTA)

5. **Generate 4-6 micro-CTAs** for different insertion points.

6. **Write `micro-cta-candidates.md`** with all micro-CTA variants.

## Quality Criteria
- 4-6 micro-CTAs generated for Type A/Hybrid (1-2 for Type B)
- Each micro-CTA is 1-2 sentences maximum
- Each feels natural in the copy flow — not a jarring interruption
- All reference the same single offer
- No urgency language in micro-CTAs
- Mix of M1, M2, M3 types for variety
- Soft language used ("if you're ready" / "whenever you'd like") — not pushy

## Failure Modes
- Micro-CTAs that sound like banner ads -> Must feel like part of the copy, not an advertisement inserted into the flow.
- More than 2 sentences -> Cut. Micro-CTAs that are too long break the reading flow.
- Urgency in micro-CTAs -> Save urgency for terminal CTA and P.S. Micro-CTAs are soft prompts.
- All micro-CTAs identical in structure -> Use different types (M1, M2, M3) for variety. Same sentence pattern repeated 4 times feels robotic.
- Micro-CTAs for Type B that feel out of place -> Type B has a shorter, scan-oriented format. 1-2 optional micro-CTAs max, or skip entirely if they feel forced.
