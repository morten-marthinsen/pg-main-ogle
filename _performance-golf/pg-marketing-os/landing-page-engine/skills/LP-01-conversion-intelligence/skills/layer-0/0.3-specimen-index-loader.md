# 0.3: Specimen Index Loader

> **Skill:** LP-01-conversion-intelligence
> **Layer:** 0
> **Position:** 3 of 3
> **Purpose:** Load specimen-index.md into working memory and produce a structured lookup table for Layer 1 specimen matching.
> **Effort Level:** haiku
> **Dependencies:** specimen-index.md exists at `landing-page-engine/specimens/specimen-index.md`
> **Output:** `specimen-index-load.md`

---

## INPUT

```yaml
input:
  specimen_index_path: "landing-page-engine/specimens/specimen-index.md"
```

---

## EXECUTION

1. **Read specimen-index.md** in full. This is the sole registry of all landing page specimens.

2. **Extract the collection summary tables:**
   - By Vertical: Which verticals have specimens, how many, which IDs
   - By Page Type: Which page types have specimens, how many, which IDs
   - Specimen Gap: What's noted as missing

3. **For each specimen entry, extract:**
   - Specimen ID (e.g., LP-SPEC-01)
   - Name/Brand
   - Vertical
   - Page Type (Type A, Type B, or variant)
   - Platform
   - Section count
   - Proof Density (LOW, MEDIUM, HIGH, EXTREME)
   - Key Pattern (the primary learning from this specimen)
   - Specimen Value (LOW, MEDIUM, HIGH)
   - File path to raw specimen

4. **Build a lookup table** organized for fast matching:
   - By page type (all Type A specimens, all Type B specimens)
   - By vertical (all Golf specimens, all Health/Supplement specimens, etc.)
   - By proof density (EXTREME, HIGH, MEDIUM, LOW)

5. **Write specimen-index-load.md** with the complete lookup structure.

---

## QUALITY CRITERIA

- All specimens from specimen-index.md captured (count matches)
- Lookup tables correctly categorize each specimen
- No specimen appears in wrong page type or vertical category
- Gap section accurately reflects specimen-index.md gaps
- File paths to raw specimen files documented
- File exists after execution

---

## FAILURE MODES

- **specimen-index.md not found:** HALT — this file is required. It should exist at landing-page-engine/specimens/.
- **Specimen count mismatch:** If extracted count differs from header count, flag discrepancy but proceed.
- **Specimen has missing fields:** Note the gap in the lookup table (e.g., "proof_density: not_specified") but include the specimen.
