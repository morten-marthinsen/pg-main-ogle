# 1.4: Element Impact Ranker

> **Skill:** LP-01-conversion-intelligence
> **Layer:** 1
> **Position:** 4 of 4
> **Purpose:** Rank page elements by their impact and relevance to this specific page type + vertical build, using element-taxonomy.md frequency data and conversion-data-reference.md lift data.
> **Effort Level:** sonnet
> **Dependencies:** `brief-extract.md` (from 0.1), `data-reference-load.md` (from 0.2), element-taxonomy.md
> **Output:** `element-impact-ranking.md`

---

## INPUT

```yaml
input:
  brief_extract: "brief-extract.md"
  data_reference_load: "data-reference-load.md"
  element_taxonomy: "LandingPageEngine/element-taxonomy.md"
  conversion_data_reference: "LandingPageEngine/conversion-data-reference.md"
  page_type: "[type_a | type_b | hybrid]"
  primary_vertical: "[vertical_id]"
  traffic_temperature: "[cold | warm | hot]"
  awareness_stage: "[1-5]"
```

---

## EXECUTION

1. **Read element-taxonomy.md** in full. This contains 64 elements across 8 functional categories with frequency data from 11 specimens.

2. **Read the PRD element taxonomy** (landing-page-engine-master-prd.md, Element Taxonomy section) for the Required/Optional/Recommended designations per page type.

3. **Filter elements by page type:**
   - For Type A: include all elements marked "Required" for Type A in the PRD
   - For Type B: include all elements marked "Required" for Type B in the PRD
   - For Hybrid: include Required elements from both types, noting which zone (above-fold vs below-fold)

4. **Score each element using this composite formula:**

   **Frequency Score (0-3):**
   - UNIVERSAL (7+/11 specimens): 3 points
   - COMMON (3-6/11): 2 points
   - SPECIALIZED (1-2/11): 1 point

   **Impact Score (0-3):**
   - Quantified lift in conversion-data-reference.md: 3 points
   - "Significant" or "High" impact noted without specific number: 2 points
   - No impact data available: 1 point (baseline)

   **Page-Type Relevance (0-2):**
   - Required for this page type per PRD: 2 points
   - Recommended/Optional for this page type: 1 point
   - Not applicable to this page type: 0 points

   **Vertical Relevance (0-2):**
   - Element observed in specimens from this vertical: 2 points
   - Element common in adjacent verticals: 1 point
   - Element not associated with this vertical: 0 points

   **Total Score: 0-10**

5. **Rank all relevant elements by total score** (highest first).

6. **Mark each element as "required" or "recommended"** for this specific build based on page type + vertical.

7. **Write the top 15 elements** (minimum 8 required) in ranked order. Include all elements scoring 5+ and all elements marked "Required" in the PRD for this page type.

---

## QUALITY CRITERIA

- At minimum 8 elements ranked (per AGENT.md GATE_1 requirement)
- Every frequency score matches element-taxonomy.md data
- Impact data citations trace to conversion-data-reference.md
- Required/Optional designations match PRD element taxonomy for the correct page type
- Vertical-specific elements identified (e.g., Ingredient Breakdown for health-supplements)
- No elements invented — all must appear in element-taxonomy.md or PRD
- Scoring is consistent and reproducible

---

## FAILURE MODES

- **element-taxonomy.md not found:** HALT — required reference file.
- **Page type has no PRD column:** This should not happen (PRD covers Type A and Type B). If encountered, use frequency data alone and flag.
- **Vertical has no specimens in element-taxonomy.md:** Set Vertical Relevance to 1 for all elements (no data to differentiate) and note the gap.
- **Fewer than 8 elements relevant:** This is unlikely given 64 elements in taxonomy. If it occurs, expand inclusion criteria (lower the score threshold).
