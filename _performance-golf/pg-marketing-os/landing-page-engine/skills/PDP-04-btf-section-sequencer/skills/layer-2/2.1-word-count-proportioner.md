# 2.1 — Word Count Proportioner

**Skill:** PDP-04-btf-section-sequencer
**Position:** Layer 2, Microskill 1
**Purpose:** Assign a specific word count target to every section in the locked BTF sequence. Every section gets ONE number — not a range — as its target. Targets are selected from the PDP Word Count Benchmark Table based on page length tier (Minimum/Standard/Extended). PDP word counts are significantly lower than Type A — a PDP BTF is 1,200-6,500 words total, not 4,000-18,000.
**Effort Level:** medium
**Dependencies:** `optional-decisions.md` (1.4 — final locked section list), `pdp-reference-loaded.md` (0.3 — word count benchmarks), `brief-verified.md` (0.1)
**Output:** `word-count-plan.md`

---

## Input Schema

```yaml
input:
  final_sections_list: "[from optional-decisions.md — locked section list]"
  word_count_benchmarks: "[PDP Word Count Benchmark Table from pdp-reference-loaded.md]"
  awareness_stage: "[1-5 from brief]"
  traffic_temperature: "[cold | warm | hot — from brief or inferred]"
  price_point: "[from brief]"
  page_length_preference: "[minimum | standard | extended | not_specified]"
  ingredient_count: "[from brief — for What's in It section multiplication]"
```

---

## Execution Logic

**Step 1: Determine page length tier**

```
IF page_length_preference specified -> use that tier
ELSE:
  Cold traffic + Awareness stage 1-2 -> standard
  Cold traffic + Awareness stage 3 -> standard
  Warm traffic + Awareness stage 3-4 -> minimum to standard
  Hot traffic + Awareness stage 4-5 -> minimum
  Price point > $75 -> shift one tier up (minimum -> standard, standard -> extended)
  Price point < $25 -> shift one tier down (standard -> minimum)
```

**Step 2: Load PDP Word Count Benchmark Table from pdp-reference-loaded.md**

Reference table (from PDP-04-AGENT.md):

| Section | Minimum | Standard | Extended |
|---------|--------|---------|---------|
| Product Highlights (TLDR) | 75-125 | 125-200 | 200-300 |
| UGC Video Carousel | 30-50 | 50-100 | 100-150 |
| The Expert | 75-125 | 125-250 | 250-400 |
| What's in It (per ingredient) | 50-100 each | 100-175 each | 175-300 each |
| What to Expect | 100-175 | 175-300 | 300-500 |
| The Problem | 75-150 | 150-250 | 250-400 |
| The Solution | 75-150 | 150-250 | 250-400 |
| Why It Works | 100-175 | 175-300 | 300-450 |
| Comparison Chart | 50-100 | 100-175 | 175-250 |
| Cost Savings | 50-100 | 100-175 | 175-250 |
| How It's Made | 75-125 | 125-200 | 200-350 |
| How to Use | 50-100 | 100-175 | 175-250 |
| Reviews & Ratings | 200-400 | 400-700 | 700-1,200 |
| FAQ (5-7 items) | 250-400 | 400-600 | 600-900 |
| Identity Matching | 75-125 | 125-200 | 200-300 |
| The Proof (Data) | 100-175 | 175-300 | 300-450 |

**Step 3: For each section, select a SINGLE target from the appropriate tier range**
- Minimum tier: pick the midpoint of the Minimum range
- Standard tier: pick the midpoint of the Standard range
- Extended tier: pick the midpoint of the Extended range
- **What's in It special handling:** multiply per-ingredient target by actual ingredient_count from brief

**Step 4: Sum all targets to get total BTF word count estimate**

**Step 5: Sanity check totals against PDP benchmarks**
- BTF total must fall within: Minimum (1,200-2,200), Standard (2,200-3,800), Extended (3,800-6,500)
- No single section should exceed 25% of total BTF word count
- If total outside range for selected tier: adjust proportions (scale sections up or down proportionally)

**Step 6: Write word count plan**

## Quality Criteria
- Every section has a single specific target number — not a range
- Total word count within PDP benchmark range for selected tier
- No section exceeds 25% of total BTF word count
- What's in It correctly calculated as per-ingredient target x ingredient count
- Tier selection rationale documented with specific reasoning
- Adjustments from benchmarks documented

## Failure Modes
- Section list is incomplete (fewer sections than locked list in 1.4) -> HALT. This is a data integrity failure. Reconcile with optional-decisions.md.
- Targets produce total outside PDP range -> Redistribute proportionally until total is within range.
- What's in It ingredient count unknown -> Use 4 as default (standard supplement), but flag warning.
- Any section target is 0 or empty -> HALT. Every section must have a specific word count target.
- Range given instead of target -> HALT. "150-250" is not acceptable. Pick ONE number (e.g., 200).
