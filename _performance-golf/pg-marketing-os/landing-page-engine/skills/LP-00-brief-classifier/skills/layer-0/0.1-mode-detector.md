# 0.1 — Mode Detector

**Skill:** LP-00-brief-classifier
**Position:** Layer 0, Microskill 1
**Purpose:** Scan project folder for CopywritingEngine packages and declare operating mode before any other work begins.
**Effort Level:** low
**Dependencies:** Project folder path
**Output:** `mode-declaration.md`

---

## Input Schema

```yaml
input:
  project_folder: "[path to project folder]"
  product_name: "[product name from user trigger]"
```

---

## Execution Logic

**Step 1:** Search for CopywritingEngine output packages in the following locations (in order of preference):

```
Priority 1: ./outputs/[product-name]/
  → 09-campaign-brief/outputs/campaign-brief.json
  → 07-offer/outputs/offer-package.json
  → 05-promise/outputs/promise-output.json
  → 19-campaign-assembly/outputs/assembled-draft.md
  → 01-research/outputs/ (any file)

Priority 2: ./clients/[client-name]/[project]/
  → Same package pattern
```

**Step 2:** Tally found packages. Declare operating mode:
- **Downstream Mode:** ≥1 CopywritingEngine output package found
- **Standalone Mode:** 0 packages found

**Step 3:** List all found packages with paths. List all missing standard packages.

**Step 4:** Note any assembled-draft.md — if this exists, the full page copy may already be drafted and this run is producing a page structure brief to guide implementation.

---

## Output Schema

`mode-declaration.md`:

```markdown
# Mode Declaration — [Product Name]

## Operating Mode: [DOWNSTREAM | STANDALONE]

## Packages Found
- [package name]: [full path]
- [package name]: [full path]
[or: "No CopywritingEngine packages found"]

## Packages Missing (Standard)
- campaign-brief.json: [found | not found]
- offer-package.json: [found | not found]
- promise-output.json: [found | not found]
- assembled-draft.md: [found | not found]
- research outputs: [found | not found]

## Assembled Draft Available
[Y — path: [path] | N]

## Mode Rationale
[1 sentence explaining why this mode was declared]
```

---

## Quality Criteria
- Mode declared as one of two values: `downstream` or `standalone`
- All found packages listed with valid paths
- File exists after execution

## Failure Modes
- Project folder not provided → Request folder path from user
- Ambiguous folder structure → List all potential locations searched
