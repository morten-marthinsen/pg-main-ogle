## EXECUTION

1. Read `conversion-data-reference.md`. Extract:

   **Baseline Conversion Rates:**
   - Median CVR all industries: 6.6%
   - Industry-specific rates (health/supplements, info products, financial, ecommerce, B2B)
   - "Good" threshold: 10%+

   **Element-Level Impact Data (A/B Test Calibration):**
   - Personalized CTAs: +202% (HubSpot)
   - Video on landing pages: +86% (VWO)
   - Long-form vs short-form: +220% (KlientBoost)
   - Multiple offers vs single: -266% (KlientBoost)
   - Mobile-responsive: +25.2% (VWO)
   - Load delay: -7% per second (Google)
   - Simple copy (5th-7th grade): +56% (VWO)
   - Personalized headline: +15-30% (Multiple)

   **A/B Test Priority Data (from conversion-data-reference.md):**
   - Headline: Very high variance, P1
   - Hero image: High variance, P1
   - Above-fold layout: High variance, P1
   - CTA button color: Medium-High, P2
   - Price display format: Medium-High, P2
   - Guarantee placement: Medium, P2
   - Social proof type: Medium, P2
   - FAQ position: Medium, P3
   - Page length: Medium, P3
   - Trust badge design: Low-Medium, P3

   **Testing Timeline Guidelines:**
   - Low traffic (<100/day): 4+ weeks, 100 min conversions
   - Medium (100-500/day): 2-3 weeks, 200 min conversions
   - High (500-2,000/day): 1-2 weeks, 300 min conversions
   - Very High (2,000+/day): 1 week, 500+ min conversions

   **Offer Type Lift Data:**
   - Soft offer: +20-50% vs hard
   - Free + shipping: highest trial rate
   - Limited offer: +15-30%
   - Limited-time: +15-25%
   - One-time offer: +30-60% AOV

   **Guarantee Data:**
   - 30-day: ~10-15% return rate
   - 60-day: ~8-12% return rate
   - 90-day: ~7-10% return rate
   - 365-day: ~5-8% return rate

2. Read `cross-page-pattern-analysis.md`. Extract:

   **Specimen-Derived Test Patterns:**
   - Headline type distribution (result/outcome most common at 4/11)
   - Proof density distribution (bimodal — extreme or zero)
   - CTA count ranges (2 to 100+)
   - CTA text consistency patterns (7/11 consistent, 4/11 varying)
   - Offer model patterns (trial, subscription, bundle, no-pricing, premium)
   - Section sequence patterns by page type (Type A skeleton vs Type B skeleton)
   - Above-fold CTA placement (DTC-specific pattern)
   - Proof placement timing (above fold vs section 2-3 vs deep)

   **Competitive Gap Patterns:**
   - Zero curiosity/secret/number headlines in dataset
   - Zero before/after proof in dataset
   - Only 1 charity tie-in (underutilized)
   - Performance guarantees as B2B pattern

3. Compile all extracted data into `benchmark-load.md`.
