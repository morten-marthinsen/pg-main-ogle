# 0.3 — Template Loader

**Skill:** LP-04-section-sequence
**Position:** Layer 0, Microskill 3
**Purpose:** Load the full section template for the classified page type from the AGENT.md BASE SECTION TEMPLATES. Type A = 18 sections. Type B = 15 sections. Hybrid = 3-zone template. This raw template is what Layer 1 will refine (select, validate mandatory, decide optional). The template loader does NOT make decisions — it loads the raw material.
**Effort Level:** low
**Dependencies:** `brief-verified.md` (0.1 — specifically `page_type.classification`)
**Output:** `template-loaded.md`

---

## Input Schema

```yaml
input:
  page_type: "[type_a | type_b | hybrid — from brief-verified.md]"
```

---

## Execution Logic

**Step 1:** Read the `page_type` from `brief-verified.md`.

**Step 2:** Load the corresponding base template from LP-04-AGENT.md `BASE SECTION TEMPLATES` section:

- **Type A** → Load all 18 sections with section ID, name, persuasion function, and mandatory flag [M]/[O]
  - Total mandatory: 12–14 (depends on product type conditionals)
  - Estimated word count range: 4,000–15,000
- **Type B** → Load all 15 sections with section ID, name, persuasion function, and mandatory flag [M]/[O]
  - Total mandatory: 10–12
  - Estimated word count range: 1,500–4,000
- **Hybrid** → Load the 3-zone template:
  - Zone 1 (Type B): Above-fold through Section 3
  - Zone 2 (Type A): Sections 4–8 (education zone)
  - Zone 3 (Mixed): Sections 9–14 (conversion zone)

**Step 3:** Count mandatory vs. optional sections. Record the exact counts.

**Step 4:** Write the complete loaded template to `template-loaded.md`. This file is consumed by 1.1 (Template Selector) and 1.2 (Mandatory Section Validator).

---

## Quality Criteria
- Template matches the page type exactly — no mixing Type A sections into a Type B load
- Every section from the AGENT.md template table is included — zero omissions
- Mandatory/optional flags accurately transcribed
- Conditional mandatory sections flagged with their conditions
- Hybrid zone split clearly delineated if applicable

## Failure Modes
- Page type not recognized (not type_a, type_b, or hybrid) → HALT. This should have been caught in 0.1 but double-check here.
- AGENT.md template section not found or empty → HALT. The template is the raw material — without it, no sequence can be constructed.
- Hybrid template loaded but zone boundaries unclear → Flag for human clarification before Layer 1 proceeds.
