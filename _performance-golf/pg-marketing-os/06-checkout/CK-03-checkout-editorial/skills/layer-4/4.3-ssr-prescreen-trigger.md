# SSR Pre-Screen Trigger — Checkout

**Purpose:** Invoke SSR pre-screening on the final checkout copy output. This is the LAST validation step before output packaging.

**Protocol:** Load and follow `~system/protocols/SSR-PRESCREEN-PROTOCOL.md`

---

## TRIGGER SEQUENCE

1. **Verify prerequisites:**
   - CK-03 editorial output is COMPLETE
   - E2E verification report exists (if applicable for this engine)
   - Project tier is Full or Standard (skip entirely for Quick tier)

2. **Identify stimulus:**
   - Load: `~outputs/[project-code]/checkout/editorial-final.*`
   - This is the complete checkout page copy after editorial (CK-03 output)

3. **Load audience personas:**
   - Source: `~outputs/[project-code]/audience-agent-personas.json`
   - If personas not available: flag to human "SSR pre-screening skipped — no audience personas constructed" and proceed to output packaging

4. **Invoke SSR pipeline per protocol:**
   - Panel size: per tier (Full = 100, Standard = 50)
   - Reaction model: Gemini 3.1 Pro
   - Embedding model: text-embedding-3-small
   - Generate reactions → embed → score → analyze

5. **Write report:**
   - Output: `~outputs/[project-code]/checkout/ssr-prescreen-report.yaml`
   - Include: overall PI, segment breakdown, diagnostic analysis, recommendation

6. **Present recommendation to human:**
   - Display: GO / REVISE / KILL with rationale
   - If REVISE: show revision targets with affected segments
   - If homogenization flags fired: include warning
   - **BLOCKING:** Wait for human acknowledgment before proceeding to output packaging

---

## ENGINE-SPECIFIC NOTES

- Short-form copy — checkout pages are functional, not persuasive.
- Trust-focused reactions: security signals, friction points, abandonment risk.
- Lower the KILL threshold by 0.3 per SSR protocol (lower PI expected for checkout copy).

---

## ANTI-DEGRADATION

- Do NOT run on incomplete or unedited output
- Do NOT skip segment breakdown (mandatory for Full and Standard)
- Do NOT auto-act on KILL recommendation — human decides
- Do NOT substitute models without recalibration
