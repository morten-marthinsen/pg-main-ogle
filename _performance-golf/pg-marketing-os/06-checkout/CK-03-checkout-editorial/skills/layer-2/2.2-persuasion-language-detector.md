# 2.2-persuasion-language-detector.md

> **Layer:** 2 (Audit Pass 2)
> **Skill:** CK-03-checkout-editorial
> **Purpose:** No persuasion language outside the order bump?
> **Outputs:** `persuasion_scan_result` object with violation list and cleanup recommendations

---

## FUNCTION

Scan every piece of checkout copy for persuasion, sales, or emotional manipulation language. Checkout is a trust and friction-reduction zone -- the buyer has already decided to purchase. Persuasion language on checkout raises anxiety rather than reducing it. The ONLY exception is the order bump area, which is the single designated persuasion moment on the entire checkout page.

---

## PERSUASION DETECTION PROTOCOL

```
EXECUTE CK-03/2.2-persuasion-language-detector:
  INPUT: checkout_copy_elements (from 0.1), strategy_reference (from 0.2)
  OUTPUT: persuasion_scan_result object

  STEP 1: DEFINE SCAN BOUNDARY
    - Order bump area: persuasion language IS acceptable here
    - Everywhere else on checkout: persuasion language is BANNED
    - Load order_bump_boundary from strategy reference
    - All scan results tagged with location (in_bump vs. outside_bump)

  STEP 2: SCAN TRUST COPY FOR PERSUASION
    FOR each trust copy element:
      SCAN for:
        - Emotional manipulation: "imagine," "picture this," "what if"
        - Urgency language: "limited time," "act now," "don't miss," "hurry"
        - Sales framing: "exclusive deal," "special offer," "best value"
        - Superlatives without proof: "best," "most trusted," "leading"
        - Benefit re-selling: "you'll love," "amazing results," "incredible"
        - Scarcity tactics: "only X left," "selling fast," "almost gone"

      IF found outside order bump:
        FLAG as violation with exact phrase and location

  STEP 3: SCAN MICRO-COPY FOR PERSUASION
    FOR each micro-copy element:
      SCAN for:
        - Persuasive button text: "Claim Your Spot!", "Get Instant Access!", "Yes! I Want This!"
        - Benefit labels: "Your Amazing Product" instead of product name
        - Emotional helper text: "Don't miss this incredible opportunity"
        - Urgency in error messages: "Hurry! Fix this to complete your order"

      IF found:
        FLAG as violation with exact phrase and location

  STEP 4: SCAN ORDER SUMMARY FOR PERSUASION
    FOR order summary elements:
      SCAN for:
        - Benefit-heavy descriptions (exceeding informational purpose)
        - Urgency in pricing display
        - Sales language in savings callout
        - EXCEPTION: order bump checkbox and copy block ARE allowed persuasion

      IF found outside order bump area:
        FLAG as violation

  STEP 5: CLASSIFY VIOLATIONS
    a. HARD VIOLATION: explicit sales/urgency language -> MUST be removed
    b. SOFT VIOLATION: borderline promotional tone -> FLAG for human review
    c. ACCEPTABLE: within order bump boundary -> NOT a violation

  STEP 6: GENERATE CLEANUP RECOMMENDATIONS
    FOR each hard violation:
      - Provide removal or replacement recommendation
      - "Claim Your Spot!" -> "Complete Order"
      - "Don't miss this amazing deal" -> [DELETE -- not checkout copy]
      - "Best value on the internet" -> [DELETE -- not trust language]

  IF any hard violations found:
    FAIL -- Violations must be resolved before packaging.
    Provide specific replacement for each violation.
```

---

## PERSUASION LANGUAGE REFERENCE

### BANNED Outside Order Bump

| Category | Banned Phrases | Why |
|----------|---------------|-----|
| Urgency | "Limited time," "act now," "hurry," "don't miss," "today only" | Creates anxiety, not trust |
| Scarcity | "Only X left," "selling fast," "almost gone" | Manipulation at point of commitment |
| Emotional | "Imagine," "picture this," "you'll love," "life-changing" | Sale is already made |
| Sales framing | "Special offer," "exclusive deal," "best value," "incredible price" | Checkout is trust, not selling |
| Superlatives | "Best," "most trusted," "#1," "leading" (without proof) | Vague claims reduce trust |
| Pressure CTAs | "Claim it now!", "Yes! Send me [product]!", "Get instant access!" | Functional CTA outperforms hype |

### ALLOWED In Order Bump Only

| Category | Allowed Phrases | Why |
|----------|----------------|-----|
| Benefit statement | "Add [product] for [benefit]" | Order bump IS the persuasion moment |
| Value framing | "Just $X more" | Price anchoring for add-on |
| Social proof | "Most popular add-on" | Social validation for bump |

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Hard persuasion violation outside order bump | REMOVE -- replace with neutral or delete |
| CRITICAL | Persuasive CTA button text | REWRITE -- use functional action text |
| DEGRADED | Soft violation (borderline tone) | FLAG for human review |
| DEGRADED | Order bump boundary unclear | Default: checkbox + copy block area only |
