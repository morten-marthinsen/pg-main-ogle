# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** CK-01-trust-security-copy
> **Purpose:** Load checkout strategy and guarantee details from CK-00
> **Outputs:** `upstream_context` object with trust architecture plan and guarantee details

---

## FUNCTION

Load the checkout-strategy.yaml output from CK-00 and extract the trust architecture plan, guarantee details, density targets, and placement specifications. Also load the offer-package.json to cross-reference guarantee terms for accuracy verification.

---

## LOADING PROTOCOL

```
EXECUTE CK-01/0.1-upstream-loader:
  INPUT: CK-00 output directory, offer-package output directory
  OUTPUT: upstream_context object

  STEP 1: LOAD CHECKOUT STRATEGY
    - Locate checkout-strategy.yaml from CK-00 outputs
    - Parse YAML and validate structure
    - Extract: trust_architecture section (all 5 categories, placements, density targets)
    - Extract: flow_map section (for placement context)
    - Extract: funnel_classification (for trust emphasis calibration)

  STEP 2: LOAD GUARANTEE DETAILS
    - From checkout-strategy.yaml: guarantee signal details
    - From offer-package.json: guarantee type, duration, conditions
    - Cross-reference: strategy guarantee must match offer package
    - Flag any discrepancy for resolution

  STEP 3: LOAD DENSITY TARGETS
    - Extract density target (should be "3+ per viewport")
    - Extract mobile density plan
    - Extract desktop density plan

  STEP 4: ASSEMBLE CONTEXT
    - Combine trust architecture, guarantee details, density targets
    - Include flow map for placement reference
    - Set loading status

  IF checkout-strategy.yaml missing:
    HALT -- "Cannot generate trust copy without checkout strategy. Run CK-00 first."

  IF trust_architecture section missing from strategy:
    HALT -- "Strategy exists but trust architecture is missing. Return to CK-00."
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | checkout-strategy.yaml missing | HALT -- run CK-00 first |
| CRITICAL | Trust architecture section absent | HALT -- return to CK-00 |
| CRITICAL | Guarantee terms mismatch between strategy and offer package | FLAG -- resolve discrepancy before proceeding |
| DEGRADED | Density targets not explicit | Default to "3+ per viewport", flag for confirmation |
| DEGRADED | Flow map incomplete | Proceed with available sections, flag gaps |
