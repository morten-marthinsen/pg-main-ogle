## Key Decisions Log

1. **8 scripts, not 6** — Split feature highlights into 3 individual scripts (SwingScan, Coach, Adaptive) rather than one combined reel. Each highlight can be used independently as social content.
2. **Hero video is 45s** — Long enough to build the full argument (problem > mechanism > system > CTA) without exceeding attention threshold. Keynote-warmed traffic already knows the story; cold traffic needs the 45-second version.
3. **Founder invitation is Brixton on camera** — Per Soul.md: insider-to-insider, founder energy. This script IS Brixton's voice. Must feel like a real person sharing something, not a produced ad.
4. **Comparison script uses section header** — "They measure. They prescribe. We diagnose." is the comparison section's header, repurposed as the closing line. Creates section-to-script consistency.
5. **No UGC script** — Pre-launch product has no real user footage. Placeholder UGC section in EC-03 is text-only. UGC scripts will be added post-launch when real testimonial footage exists.
6. **No expert/coaching team script** — Coach names and credentials not yet populated (GAP-02). Script deferred until coach profiles are finalized.
7. **Problem/vindication script stays short (25s)** — The emotional weight is in the stat and the reframe, not in length. The section copy in EC-03 carries the full argument.
8. **All scripts close on micro-scripts** — Every script ends with a product-level micro-script or CTA. Reinforces linguistic cohesion across the page.
