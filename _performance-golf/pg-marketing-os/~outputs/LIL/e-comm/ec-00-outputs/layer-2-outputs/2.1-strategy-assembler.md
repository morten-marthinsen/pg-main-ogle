**Page flow note:** ATF sections (Hero, Copy Block, CTA Box) capture high-intent visitors. BTF follows the Structure skill's education arc: Problem > Mechanism > Authority > Differentiation > Timeline > Ease > Offer > Guarantee > FAQ > Use Cases. Product Highlights sits between ATF and the narrative BTF sections as a scan-speed bridge — visitors who scroll past ATF but are not yet committed to reading get fast answers.

### Design Framework Seeds

| Section | Design Direction |
|---------|-----------------|
| Hero Carousel | Full-width carousel with 10-thumbnail story. Mobile: swipe with visible thumbnails. First thumbnail has video play overlay (Grant introduction). Product photography: child using product, parent-child lifestyle, Skills Mat in use, component breakdown. |
| Copy Block | Two-column (desktop): text left, product image right. Mobile: full-width stacked. Premium brand feel. Short — 3-4 lines max. |
| CTA Box | Single SKU card. $149 price. "Get the Lil' Legends Starter Set" CTA. Trust badges below (365-day guarantee, free shipping if applicable, PG brand). NO urgency countdown. |
| Product Highlights | Icon + label + expansion accordion. Mobile: accordion with first item expanded. 4-5 highlights. |
| Problem/Solution | Full-width, warm/empathetic color palette (NOT dark/aggressive like DR). Parent-child imagery. Mobile: stack vertically. |
| Why It Works | Visual-forward mechanism section. Shape sorter comparison visual. Component demonstration images. Yellow/white ball color matching visual. Full-width with generous imagery. |
| Expert Credibility | Grant portrait/video centered. Credentials displayed naturally (not badge-heavy). Photo of Grant with product and/or with his own kid. |
| What to Expect Timeline | 3-phase horizontal timeline (desktop), vertical (mobile). Day 1 / Week 1 / Month 1. Warm illustration or photography per phase. |
| Comparison Chart | 3-column table. "Plastic Toy Clubs" / "Heavy Junior Sets" / "Lil' Legends Starter Set." LIL column highlighted. Check/X icons. Mobile: horizontal scroll or responsive condensation. |
| How to Use | 3-step numbered cards. Horizontal (desktop), stacked (mobile). "1. Unroll the mat. 2. Say seven words. 3. Watch them play." |
| Offer/Pricing | Full offer card. Component value stack. $306.99 value / $149 price. Bonus list. Identity-driven CTA: "Give your kid the game." |
| Guarantee | Centered, badge/seal visual. "365-Day: Your Kid Has Fun or It's Free." Standalone section, clean and confident. |
| FAQ | Accordion with first question expanded. 8-10 Q&A pairs. Mobile: native accordion. |
| Use Cases | 3-4 card grid (desktop), carousel (mobile). Each card: audience label + 1-2 sentence scenario + "this is for you" framing. |
