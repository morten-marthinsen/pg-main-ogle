## Type Decision

```yaml
mechanism_type_decision:
  primary_type: "SYSTEM"
  secondary_type: "SCIENTIFIC"

  primary_rationale: |
    LIL's mechanism is a SYSTEM — 4 components working together on a single
    learning principle. "System" captures the multi-part, integrated nature
    of the product. Each piece teaches one thing. Together, they form a complete
    golf learning environment.

    Critical: The system is named by its OPERATING PRINCIPLE (visual-tactile
    learning), not by its components. P90X = "Muscle Confusion" (principle),
    not "Varied Exercise Routine" (system description).

  secondary_rationale: |
    SCIENTIFIC grounding gives the system credibility. Piaget, developmental
    psychology, 90% brain development by age 5 — this isn't made-up theory.
    It's established science applied to a product category for the first time.

    The scientific element supports the system — it explains WHY the system
    works. But the mechanism IS the system, not the science.

  hybrid_approach: |
    SYSTEM × SCIENTIFIC: A learning system grounded in developmental science.
    The science explains WHY toddlers learn through shapes and colors.
    The system shows HOW the equipment uses that principle.
    This maps perfectly to the SCIENCE_CONCEPTUAL emphasis strategy.
```

## Type → Emphasis Alignment

| Element | Type Contribution | Emphasis Allocation |
|---------|-------------------|-------------------|
| **Root Cause** (25%) | System: the old approach was wrong because it assumed coaching | State quickly — already established |
| **Science/Conceptual** (50%) | Scientific: WHY visual-tactile learning works (Piaget, developmental science) | PRIMARY — explain the learning principle |
| **Delivery/Tangible** (25%) | System: HOW each of the 4 components uses that principle | Show the system in action |

This alignment is natural. The SYSTEM type gives us the "what" (4 components working together). The SCIENTIFIC type gives us the "why" (developmental science). The emphasis strategy says spend 50% on the "why" — which is exactly the science behind the system.
