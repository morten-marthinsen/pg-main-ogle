## Core Offer Definition

### What They're Buying

**The Lil' Legends Start Set — The See & Swing System**

A 3-club toddler golf system (ages 2-5) with integrated Skills Mat and carry bag. Every piece teaches one golf fundamental through shapes and colors — no verbal coaching needed. Drive, Hit, and Putt.

### Positioning

**System, not product.** This is not "3 toddler golf clubs." It is a self-teaching golf system where the equipment teaches through visual-tactile cues and the parent's only role is to play alongside their child.

| Positioning Axis | What It Is | What It Is NOT |
|-----------------|------------|---------------|
| Product type | Developmental golf system | Toy / Junior club set / Teaching aid |
| Teaching approach | Self-teaching through shapes & colors | Parent-coached / Instruction-dependent |
| Parent's role | Golf buddy | Golf coach / Instructor / Teacher |
| Category | NEW — "developmental golf system" | Existing: toy (Tier 3) or equipment (Tier 1-2) |

### Physical Product Value Ratio Adaptation

**Important note:** The Hormozi 10:1 value-to-price ratio is designed for info-products and services where perceived value can be dramatically inflated (courses, coaching, etc.). For a physical product, the customer can see and hold what they're buying. A 10:1 ratio on a $149 physical product would require claiming $1,490 in value — which would be unbelievable and counterproductive.

**Adapted standard for physical product e-comm:**
- Stated component value: 2-3x actual price (believable "you'd pay $X for each piece separately")
- Competitive comparison anchoring: "Charlie Golf charges $190-338 for clubs alone — no mat, no bag, no teaching system"
- Failure-cycle math: "3 broken sets = $60-120 wasted + frustrated kid"
- Developmental value reframe: "One 30-min private golf lesson for a toddler = $50-75. This system replaces every lesson."

**Effective value perception target: 3-5x price** through combined component value + competitive comparison + failure-cycle math + developmental reframe.
