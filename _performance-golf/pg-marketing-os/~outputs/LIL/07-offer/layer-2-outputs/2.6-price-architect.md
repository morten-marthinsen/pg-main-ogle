## Price Architecture

### Recommended Price: $149.00

**Position within research sweet spot:** $129-$179 range. $149 is the strategic midpoint.

### Price Selection Rationale

| Factor | Analysis | Impact on Price |
|--------|----------|----------------|
| **Research sweet spot** | $129-$179 per market intelligence | $149 = center of range |
| **Competitive positioning** | Tot Clubs $75, TLG $100-130, Charlie Golf $190-338 | Above starter competitors, well below premium. "System" justifies premium over clubs-only. |
| **WTP ceiling** | ~$150-200 IF it works, lasts, includes everything | $149 sits at the lower end of WTP ceiling — accessible |
| **Commodity cluster** | Tour X, Precise, Fore!, Alien all at $139.99 | $149 is $10 above commodity cluster — signals premium without being in the undifferentiated price band |
| **System vs. clubs** | Competitors sell clubs only. LIL sells 3 clubs + mat + bag. | System bundle justifies premium over clubs-only competitors |
| **Serial failure math** | 3 broken sets = $60-120 wasted | $149 < worst-case serial failure cost + frustrated kid |
| **Father's Day gift price** | Gift buyers at this price point: comfortable, not extravagant | $149 = "thoughtful premium gift" — not cheap, not scary |
| **Psychological pricing** | $149 vs. $150 — stays in the "$1XX" mental bracket | Sub-$150 feels meaningfully different from $150+ |

## Anchor Cascade (4-Level)

### Anchor 1: Competitor Comparison
```
"Charlie Golf charges $190-338 for clubs alone.
No teaching system. No Skills Mat. No carry bag.

The Lil' Legends Start Set: 3 clubs + Skills Mat + carry bag + the See & Swing System.
$149."
```
**Anchor effect:** LIL is positioned as MORE product for LESS money than the premium competitor.

### Anchor 2: Component Value Math
```
"If you bought each piece separately:

3 See & Swing clubs (with Focal Sphere, Walk-In Frame, Face Forward): $199 value
Skills Mat with 8 visual training guides: $49 value
Child-sized carry bag: $29 value
PG1 App — 30-day free trial: $29.99 value

Total value: $306.99

The Lil' Legends Start Set: $149"
```
**Anchor effect:** 2x stated value for price paid. Component breakdown makes the math tangible.

### Anchor 3: Serial Failure Math
```
"The average parent in our research bought 2-4 sets before finding one that worked.

Set 1: $25 plastic from Amazon. Broke in a week.
Set 2: $35 'upgraded' plastic. Broke in a month.
Set 3: $40 metal junior set. Too heavy. Kid gave up after 3 swings.

Total spent: $100. Total result: a kid who thinks golf is frustrating.

One Lil' Legends Start Set: $149. Works from day one."
```
**Anchor effect:** Reframes $149 as LESS than the alternative (serial failure + frustrated kid). Proven by research: serial purchase pattern documented across 2-4 products.

### Anchor 4: Private Lesson Comparison
```
"One 30-minute private golf lesson for a toddler: $50-75 (if you can even find one).
Two lessons: $100-150.
Three lessons: $150-225.

The Lil' Legends Start Set replaces every lesson — because the equipment IS the lesson.
$149. Use it hundreds of times."
```
**Anchor effect:** Reframes the product as replacing a recurring expense. $149 = less than 3 lessons, unlimited use.
