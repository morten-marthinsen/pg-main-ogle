## Key Decisions Log

1. **DSI is putter-level, not per-feature.** Not every feature gets a superlative. Only features with genuine "only" claims get superlative micro-scripts.
2. **Push/pull language correction:** "Pushing" and "pulling" are negative putting terms (miss directions). The RS1 physics: you pull the weight back on backstroke, then let gravity pull the putter through. Takes human error out.
3. **Don't compare against Gemini 70+ micro-scripts** — those are unapproved. Create fresh from the DSI.
4. **Bill Schley framework updates for EC-01 and EC-04** — deferred until after this manual process is complete. Learnings from this session will inform system updates.
5. **Micro-script avg length:** ~5.2 words per Schley. Must pass the "bubba at the bar" repeat test.
