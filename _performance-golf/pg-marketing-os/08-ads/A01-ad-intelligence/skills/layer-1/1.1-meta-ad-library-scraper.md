# 1.1: Meta Ad Library Scraper

**Version:** 1.0
**Layer:** 1 (Scraping)
**Type:** Platform Scraping
**Dependencies:** Layer 0 complete, Apify MCP available
**Output:** meta_ad_data.json

---

## Purpose

Scrape Facebook/Meta Ad Library for each competitor brand using Apify actor. Target: 200+ ads minimum. Extract complete ad data including copy text, headlines, CTAs, image/video URLs, format type, estimated run duration, and all available metadata.

---

## Inputs

```yaml
input:
  from_layer_0:
    competitor_list: [string]  # 10+ brands
    vertical_profile:
      platform_priorities: [platforms]
  from_0.2:
    apify_actor_id: string  # for Meta Ad Library scraper
```

---

## Process

### Step 1: Identify Apify Actor

```
USE actor from 0.2 validation OR search for best available:
  RECOMMENDED: "apify/facebook-ads-library-scraper"

FETCH actor details:
  mcp__apify__fetch-actor-details(actorId="[actor-id]")

REVIEW input schema to understand parameters
```

### Step 2: Per-Competitor Scraping

```
FOR EACH competitor in competitor_list:

  CALL Apify actor:
    mcp__apify__call-actor(
      actorId="[actor-id]",
      input={
        "searchQuery": "[competitor brand name]",
        "country": "[target country from brief — default US]",
        "adType": "all",
        "mediaType": "all",
        "maxResults": 50  # per competitor — adjust based on actor limits
      }
    )

  WAIT for run completion:
    mcp__apify__get-actor-run(runId="[run-id]")
    Check status every 10 seconds
    Max wait: 5 minutes per competitor

  RETRIEVE results:
    mcp__apify__get-actor-output(runId="[run-id]")

  EXTRACT per ad:
    - ad_id: unique identifier
    - brand_name: competitor brand
    - platform: "meta" (Facebook/Instagram)
    - ad_copy_text: full body text
    - headline: headline text
    - cta_text: call-to-action text
    - image_url: URL if static image
    - video_url: URL if video
    - format_type: enum[video, image, carousel, stories]
    - page_name: advertiser page name
    - start_date: when ad started running (if available)
    - last_seen_date: when ad was last active (if available)
    - estimated_run_duration_days: calculated from dates

  STORE all extracted ads

  RATE LIMIT: Wait 5-10 seconds between competitors
```

### Step 3: Pagination Handling

```
IF actor supports pagination:
  FOR EACH competitor:
    ITERATE through ALL pages of results
    DO NOT stop at first page
    Track: pages_scraped per competitor

LOG per competitor:
  - Competitor name
  - Ads found
  - Pages scraped
  - Any errors encountered
```

### Step 4: Data Quality Checks

```
FOR EACH scraped ad:
  VERIFY required fields populated:
    - ad_copy_text OR headline (at least one must exist)
    - format_type identified
    - brand_name matches competitor

  FLAG incomplete ads but DO NOT discard
  (Incomplete data is still data — Layer 2 will handle gaps)
```

### Step 5: Volume Tracking

```
CALCULATE:
  - Total ads scraped across all competitors
  - Ads per competitor (min/max/average)
  - Percentage with video vs static
  - Percentage with all required fields

UPDATE PROJECT-STATE.md:
  Meta ads: [count] / 200 target

IF total < 200:
  LOG: "Meta scraping yielded [X] ads. Target: 200."
  → Will trigger expansion in 1.5 (Raw Data Validator)
```

1. All competitors were scraped (not just first 5)
2. Total ads >= 20 (absolute minimum — will likely need expansion if < 200)

4. At least 80% of ads have ad_copy_text OR headline populated
5. Format type identified for at least 90% of ads
6. Run duration estimated for ads with date data

## Error Handling

```
LEVEL 1: Transient failure (timeout, rate limit)
  → Wait 30 seconds
  → Retry up to 3 times per competitor
  → LOG each retry

LEVEL 2: Actor not found / changed
  → Search for alternative: mcp__apify__search-actors(query="facebook ad library")
  → Use best alternative
  → Document switch in execution log

LEVEL 3: Platform blocked / inaccessible
  → LOG the failure with error details
  → Attempt Firecrawl direct scrape as fallback (next microskill will try)
  → Continue with other competitors
  → Report what was/wasn't accessible

LEVEL 4: All methods fail
  → Comprehensive failure log
  → Meta ads = 0
  → Continue to other platforms (TikTok, Google)
  → Gate 1 will assess total across all platforms
```
