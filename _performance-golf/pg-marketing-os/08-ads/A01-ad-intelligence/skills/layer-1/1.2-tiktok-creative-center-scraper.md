# 1.2: TikTok Creative Center Scraper

**Version:** 1.0
**Layer:** 1 (Scraping)
**Type:** Platform Scraping
**Dependencies:** Layer 0 complete, Apify OR Firecrawl available
**Output:** tiktok_ad_data.json

---

## Purpose

Scrape TikTok Creative Center top ads for the vertical using Apify actor (if available) OR Firecrawl direct scraping. Target: 150+ ads minimum. Extract script/text overlay, hook text (first 3 seconds), format details, engagement signals, and advertiser names.

---

```yaml
input:
  from_layer_0:
    vertical_profile:
      platform_priorities: [platforms]
    product_category: string
  from_0.2:
    apify_available: boolean
    firecrawl_available: boolean
```

---

## Process

### Step 1: Select Scraping Method

```
IF Apify available:
  SEARCH for TikTok actor:
    mcp__apify__search-actors(query="tiktok creative center ads")
  
  IF actor found:
    USE Apify actor method (similar to 1.1 Meta pattern)
  ELSE:
    FALLBACK to Firecrawl

IF Apify unavailable OR no actor found:
  USE Firecrawl direct scraping
```

### Step 2: Firecrawl Scraping (if needed)

```
TARGET URL: https://ads.tiktok.com/business/creativecenter/inspiration/topads/pc/en
  (Adjust region/vertical filters as needed)

mcp__firecrawl-mcp__firecrawl_scrape(
  url="[TikTok Creative Center URL]",
  formats=["markdown"],
  waitFor=5000  # TikTok pages load dynamically
)

PARSE markdown output:
  - Identify ad entries (structured patterns in markdown)
  - Extract per ad: hook text, video length, advertiser, engagement metrics
  - Handle pagination (construct next-page URLs manually)
```

### Step 3: Data Extraction

```
FOR EACH ad scraped:
  EXTRACT:
    - ad_id: generated or platform ID
    - platform: "tiktok"
    - script_text: full text/voiceover transcript (if available)
    - hook_text: first 3 seconds text overlay OR opening line
    - video_length_seconds: duration
    - format_style: enum[ugc, polished, animation, product_demo]
    - advertiser_name: brand or account name
    - engagement_signals: views, likes, shares (if available)
    - industry_category: TikTok's category classification

  STORE extracted data
```

### Step 4: Volume Tracking

```
UPDATE running count:
  TikTok ads scraped: [count]
  Target: 150

UPDATE PROJECT-STATE.md

IF count < 150:
  LOG deficit
  → Expansion will be attempted in 1.5
```

## Quality Checks

1. Total ads >= 15 (absolute minimum)
2. Hook text captured for >= 80% of ads
3. Video length identified for >= 90% of ads
4. Advertiser names present for >= 70% of ads
5. No duplicate entries

## Error Handling
