# 3.1: Competitive Landscape Map

**Version:** 1.0
**Layer:** 3 (Intelligence Synthesis)
**Type:** Strategic Analysis
**Dependencies:** Layer 2 complete (LAYER_2_COMPLETE.yaml exists)
**Model:** opus

---

## Purpose

Map all competitor brands by ad volume, platform focus, creative strategy (dominant hook types, formats, visual styles), and estimated market position. Identify the top 3 market leaders and characterize their creative strategies. This produces the competitive intelligence foundation that downstream skills use to understand the competitive landscape and find differentiation opportunities.

## Process

### Step 1: Per-Competitor Profile Construction

```
FOR EACH competitor brand (10+ brands):

  AGGREGATE data from Layer 2 outputs:

  brand_profile:
    brand_name: string
    brand_url: string  # if available from scraping
    total_ads_found: integer
    platforms_active:
      - platform: string
        ad_count: integer
    dominant_hook_types:
      - type_id: string
        type_name: string
        count: integer
        pct_of_brand_ads: float
      # Top 3 hook types for this brand
    dominant_formats:
      - format: string
        count: integer
        pct: float
      # Top 2 formats for this brand
    dominant_visual_style: string
    tier_distribution:
      TIER_WINNER: integer
      TIER_PERFORMER: integer
      TIER_TESTING: integer
      TIER_NEW: integer
    brand_longevity_score: float  # from 2.5
    creative_strategy_notes: string
      # 2-3 sentence description of what defines this brand's ad approach
```

### Step 2: Market Position Estimation

```
FOR EACH brand, estimate market position based on:

  SIGNALS:
    1. Ad volume (more ads = more investment = likely larger player)
    2. Platform breadth (more platforms = more sophisticated operation)
    3. TIER_WINNER count (more winners = better creative performance)
    4. Hook type diversity (more types = more testing/sophistication)
    5. Creative production quality (polished vs. UGC ratio)

  ASSIGN market_position:
    "leader" — Top 3 by ad volume + strong TIER_WINNER representation
    "challenger" — Mid-range ad volume, actively testing, growing
    "niche" — Low ad volume, focused on specific platform or creative approach
    "new_entrant" — Recently appeared, few ads, mostly TIER_NEW/TIER_TESTING

  JUSTIFICATION: Each position assignment must include specific data points.
    GOOD: "Leader — 85 ads across 3 platforms, 12 TIER_WINNER ads, uses 8 hook types"
    BAD: "Leader — seems like a big player"
```

### Step 3: Top 3 Leader Analysis

```
IDENTIFY top 3 market leaders (by composite score: volume + winners + platform breadth).

FOR EACH leader, produce deep creative strategy analysis:

  leader_analysis:
    brand_name: string
    why_leader: string  # specific evidence for market leadership
    creative_strategy:
      hook_strategy: string
        # What hook types do they rely on? Are they diversified or narrowly focused?
      format_strategy: string
        # What formats dominate? Video length preferences? Platform-specific approaches?
      visual_strategy: string
        # What defines their visual identity? UGC, polished, mixed?
      messaging_themes: string
        # What messages and angles recur across their ads?
      winning_pattern: string
        # What characterizes their longest-running (TIER_WINNER) ads specifically?
    vulnerabilities: string
      # Where are they weak? What are they NOT doing?
      # e.g., "Zero presence on TikTok" or "Only uses 2 hook types — rigid"
```

### Step 4: Market-Level Analysis

```
PRODUCE market-level intelligence:

  market_analysis:
    competitive_intensity:
      total_brands: integer
      total_ads: integer
      ads_per_brand_avg: float
      intensity_score: string  # "low" (<20 ads/brand), "moderate" (20-50), "high" (50+)
      intensity_note: string

    leader_vs_challenger:
      leader_ad_share: float  # % of total ads from top 3
      challenger_ad_share: float  # % from middle tier
      niche_ad_share: float  # % from niche players
      concentration: string  # "concentrated" (leaders > 60%), "fragmented" (leaders < 30%)

    platform_competition:
      - platform: string
        brands_active: integer
        total_ads: integer
        most_competitive_hook_types: [string]
        competitive_density: string  # "crowded" / "moderate" / "sparse"

    creative_homogeneity:
      score: float  # 1-10 (1 = all brands do the same thing, 10 = highly diverse)
      note: string  # What makes the market homogeneous or diverse?
```

## Quality Checks

1. **All 10+ competitors profiled** -- every brand from the competitor list has a profile. No brands skipped.
2. **Market positions have evidence** -- every position assignment cites specific data (ad count, winner count, platform count).
3. **Top 3 leaders have deep analysis** -- each leader has a full creative strategy breakdown with vulnerabilities.
4. **Market-level metrics calculated** -- competitive intensity, concentration, homogeneity scores present.
5. **Key findings are campaign-relevant** -- findings connect to implications for THIS campaign, not generic observations.
6. **Creative strategy notes are specific** -- "Uses video" is insufficient; "85% video, predominantly 15-30s UGC format with authority hooks" is required.
