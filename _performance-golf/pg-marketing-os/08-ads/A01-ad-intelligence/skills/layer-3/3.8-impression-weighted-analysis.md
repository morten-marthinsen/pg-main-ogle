# 3.8: Impression-Weighted Analysis

**Version:** 1.0
**Layer:** 3 (Intelligence Synthesis)
**Type:** Weighted Distribution Analysis
**Dependencies:** Layer 2 complete (LAYER_2_COMPLETE.yaml exists), 2.7 (Impression Scorer), 3.2 (Hook Type Distribution)
**Model:** opus
**Mode:** Executes in Tool-Assisted Scan mode. Skipped when no impression data is available.

---

## Purpose

Generate impression-weighted versions of the key distribution analyses from Layer 3. While 3.2 (Hook Type Distribution) counts each ad equally, this microskill weights ads by their dual-signal performance score. This reveals which hook types, formats, and visual styles are associated with HIGH-PERFORMING ads, not just frequently used ones.

The key insight: a hook type used in 5% of ads but associated with 15% of impression-validated winners is a stronger signal than a hook type used in 30% of ads but associated with only 10% of winners.

## Process

### Step 1: Check Impression Data Availability

```
IF 2.7 output shows scoring_skipped = true:
  LOG: "No impression data available. Skipping impression-weighted analysis."
  OUTPUT: empty output file with skip reason
  EXIT
```

### Step 2: Impression-Weighted Hook Distribution

```
FOR EACH of the 32 hook types:

  CALCULATE weighted metrics:
    weighted_count: sum of performance_score for all ads with this hook type
    weighted_pct: weighted_count / total_weighted_count (across all types)
    impression_validated_count: count of impression-validated ads with this type
    raw_count: unweighted count (same as 3.2)
    raw_pct: unweighted percentage (same as 3.2)

  CALCULATE performance ratio:
    performance_ratio = weighted_pct / raw_pct
      > 1.0 = hook type over-represented in high-performing ads (STRONG SIGNAL)
      < 1.0 = hook type under-represented in high-performing ads (WEAK SIGNAL)
      = 1.0 = hook type performance matches its frequency (NEUTRAL)

  COMPARE with 3.2 count-based distribution:
    IDENTIFY hooks where count-based and impression-weighted tell DIFFERENT stories
    These divergences are the highest-value intelligence:
      "High frequency + Low performance" = oversaturated, underperforming
      "Low frequency + High performance" = hidden winners, strong opportunity
```

### Step 3: Impression-Weighted Format Distribution

```
FOR EACH format type (video, static, carousel, etc.):
  CALCULATE same weighted metrics as Step 2
  IDENTIFY format types that outperform/underperform their frequency
```

### Step 4: Key Findings

```
PRODUCE actionable findings:

1. TOP 3 IMPRESSION-WEIGHTED HOOK TYPES
   Hook types with highest performance_ratio
   These should be prioritized in A02 hook generation

2. TOP 3 IMPRESSION-WEIGHTED FORMATS
   Formats associated with highest-performing ads

3. DIVERGENCE ALERTS
   Cases where count-based and impression-weighted analyses disagree
   These are the most valuable intelligence outputs

4. HIDDEN WINNERS
   Hook types with low frequency but high performance ratio
   Direct input to opportunity gap analysis (3.5)
```

## Quality Checks

1. **All 32 hook types in weighted distribution** — even types with 0 weighted count.
2. **Performance ratios calculated correctly** — weighted_pct / raw_pct.
3. **Divergence alerts are actionable** — each alert includes a specific insight.
4. **Hidden winners identified** — low-frequency, high-performance-ratio types highlighted.
5. **Comparison with 3.2 explicit** — count-based vs. impression-weighted differences documented.
6. **Graceful skip when no data** — clear skip message, not an error.
