# 0.4: Input Validator

**Version:** 1.0
**Layer:** 0 (Foundation)
**Type:** Validation
**Dependencies:** 0.0.1, 0.1, 0.2, 0.3 complete
**Output:** layer_0_validation.yaml

---

## Purpose

Validate that ALL Layer 0 inputs are present and above minimum thresholds. This is the final Layer 0 microskill — it aggregates all previous validation checks and makes the GATE_0 pass/fail decision. If any input is missing or below threshold, Layer 1 is BLOCKED.

---

## Inputs

```yaml
input:
  from_0.0.1:
    vertical_profile_loaded: boolean
  from_0.1:
    campaign_brief_loaded: boolean
    competitor_count_from_brief: integer
  from_0.2:
    firecrawl_available: boolean
    apify_available: boolean
  from_0.3:
    competitor_list_count: integer
```

---

## Process

### Step 1: Verify All Microskills Executed

```
CHECK file existence:
  ✓ layer-0-outputs/0.0.1-vertical-profile-loader.yaml EXISTS
  ✓ layer-0-outputs/0.1-campaign-brief-loader.yaml EXISTS
  ✓ layer-0-outputs/0.2-mcp-tool-validator.yaml EXISTS
  ✓ layer-0-outputs/0.3-competitor-list-builder.yaml EXISTS

IF any file missing:
  → HALT: "Layer 0 microskill [X] did not produce output file"
  → Cannot validate without all inputs
```

### Step 2: Validate Each Input

```
VERTICAL PROFILE:
  IF vertical_profile_loaded = true:
    ✓ PASS
  ELSE:
    ✓ WARN: "Using universal configuration (no vertical-specific optimization)"
    ✓ PASS (not blocking)

CAMPAIGN BRIEF:
  IF campaign_brief_loaded = true:
    ✓ PASS
  ELSE:
    ✗ FAIL: "Campaign Brief not loaded. Execute Skill 09 first."
    → BLOCKING

MCP TOOLS:
  IF firecrawl_available = true AND apify_available = true:
    ✓ PASS
  ELSE IF firecrawl_available = true OR apify_available = true:
    ✓ WARN: "One MCP tool unavailable. Scraping will use fallback methods."
    ✓ PASS (degraded capability but not blocking)
  ELSE:
    ✗ FAIL: "Both MCP tools unavailable. Cannot scrape ads."
    → BLOCKING

COMPETITOR LIST:
  IF competitor_list_count >= 10:
    ✓ PASS
  ELSE IF competitor_list_count >= 5:
    ✓ WARN: "Only [X] competitors found. Scraping coverage may be limited."
    ✓ PASS (degraded coverage but not blocking)
  ELSE:
    ✗ FAIL: "Fewer than 5 competitors. Insufficient for competitive intelligence."
    → BLOCKING
```

### Step 3: Gate Decision

```
IF all PASS (no blocking failures):
  → GATE_0: PASS
  → Can proceed to Layer 1
  → Create LAYER_0_COMPLETE.yaml

IF any BLOCKING failure:
  → GATE_0: FAIL
  → HALT execution
  → List blocking issues
  → Provide resolution steps
```
