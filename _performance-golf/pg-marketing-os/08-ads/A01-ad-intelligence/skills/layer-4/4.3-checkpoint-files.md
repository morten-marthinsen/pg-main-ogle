# 4.3: Checkpoint Files Generator

**Version:** 1.0
**Layer:** 4 (Output Packaging)
**Type:** Verification / Gate
**Dependencies:** 4.1 and 4.2 complete
**Model:** sonnet

---

## Purpose

Create all final checkpoint YAML files, verify every output file exists with sizes, and produce LAYER_4_COMPLETE.yaml -- the final gate that marks the A01 skill as complete. This microskill performs the terminal verification: does the AD-INTELLIGENCE-HANDOFF.md exist at 100KB+, do all 10 sections have content, and do all per-microskill output files exist?

This is the LAST microskill in the A01 pipeline. When LAYER_4_COMPLETE.yaml is created with status: PASS, the skill is done.

---

## Inputs

```yaml
input:
  from_4.1:
    handoff_file: "AD-INTELLIGENCE-HANDOFF.md"
  from_4.2:
    execution_log: "execution-log.md"
  from_all_layers:
    all_output_files: [string]  # complete list of every output file expected
    all_checkpoint_files: [string]  # LAYER_0_COMPLETE.yaml through LAYER_3_COMPLETE.yaml
```

## Process

### Step 1: Verify All Prior Checkpoint Files Exist

```
CHECK for existence of each checkpoint file:

  REQUIRED checkpoints:
    [ ] checkpoints/LAYER_0_COMPLETE.yaml
    [ ] checkpoints/LAYER_1_COMPLETE.yaml
    [ ] checkpoints/LAYER_2_COMPLETE.yaml
    [ ] checkpoints/LAYER_3_COMPLETE.yaml

  FOR EACH checkpoint:
    exists: boolean
    contains_status_pass: boolean

  IF any checkpoint missing:
    HALT: "Cannot complete Layer 4 — [LAYER_X_COMPLETE.yaml] does not exist.
           Layer [X] was not properly validated."

  IF any checkpoint does not contain status: PASS:
    HALT: "Checkpoint [file] exists but status is not PASS.
           Layer was not properly passed."
```

### Step 2: Verify AD-INTELLIGENCE-HANDOFF.md

```
VERIFY handoff file:

  file_exists: boolean
  file_path: string
  file_size_bytes: integer
  file_size_kb: float

  IF NOT exists:
    HALT: "AD-INTELLIGENCE-HANDOFF.md does not exist. Run 4.1 first."

  IF file_size_kb < 100:
    HALT: "AD-INTELLIGENCE-HANDOFF.md is [X]KB. Minimum: 100KB.
           Re-run 4.1 with chunked assembly protocol."

  VERIFY all 10 sections present:
    sections_found: [string]
    sections_missing: [string]

    SEARCH for each section header:
      "## Section 1: Executive Summary"
      "## Section 2: Competitor Landscape"
      "## Section 3: Hook Type Distribution"
      "## Section 4: Format Distribution"
      "## Section 5: Visual Style Analysis"
      "## Section 6: Winning Ad Specimens"
      "## Section 7: Emerging Trends"
      "## Section 8: Opportunity Gaps"
      "## Section 9: Platform-Specific Intelligence"
      "## Section 10: Recommended Hooks"

    IF any section missing:
      HALT: "Section [X] not found in handoff file. Re-run 4.1."

  VERIFY no empty sections:
    FOR EACH section:
      content_length: integer  # characters between this header and next
      is_empty: boolean  # < 100 characters = effectively empty

    IF any section empty:
      HALT: "Section [X] is empty or near-empty. Re-run 4.1."
```
