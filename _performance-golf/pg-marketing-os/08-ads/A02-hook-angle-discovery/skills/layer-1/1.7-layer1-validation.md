# 1.7: Layer 1 Validator

## Purpose

Verify Layer 1 output meets all quality thresholds before proceeding to Layer 2 (Hook Generation). This is the gate that blocks hook generation if angle discovery is incomplete or weak.

## Inputs

- 1.6-ranked-angles.md (primary validation target)
- All Layer 1 outputs (1.1-1.5)

## Process

1. **Verify angle count threshold**
   - Total unique angles >= 10
   - If < 10 → HALT, return to 1.1-1.5 and expand

2. **Verify source attribution completeness**
   - EVERY angle must cite specific sources
   - Research quotes, data points, or upstream skill references
   - If any angle lacks attribution → HALT, add attribution

3. **Verify no angle IS the big idea**
   - Each angle must be an ENTRY POINT, not the destination
   - Test: Does the angle state the complete big idea? → If yes, FAIL
   - Angles lead TO big idea, don't state it

4. **Verify category diversity**
   - Minimum 3 of 5 categories represented
   - Categories: Pain, Mechanism, Identity, Proof, Competitive Gap
   - If < 3 → HALT, mine underrepresented categories

5. **Verify big idea connection**
   - EVERY angle must explain how it leads to big idea
   - Connection must be logical and clear
   - If any angle lacks connection → HALT, add connection

6. **Calculate Layer 1 readiness**
   - All thresholds met → PASS
   - Any threshold failed → FAIL (blocking)

## Output File

- **Path:** `layer-1-outputs/1.7-layer1-validation.md`
- **Minimum size:** 2KB

## Quality Checks

**IF ANY CHECK FAILS → GATE 1 FAILS**

1. Angle count < 10 → Return to 1.1-1.5, expand angle mining
2. Missing attribution → Add source attribution to all angles
3. Any angle IS big idea → Reframe as entry point
4. Categories < 3 → Mine underrepresented categories
5. Missing big idea connection → Add connection statements

## Dependencies

- 1.6-ranked-angles.md MUST exist
- All Layer 1 angle mining outputs (1.1-1.5) MUST exist

## Gate 1 Blocking

**IF THIS VALIDATION FAILS → LAYER 2 IS BLOCKED**

The gate file `GATE_1_COMPLETE.yaml` will NOT be created unless ALL checks pass.
