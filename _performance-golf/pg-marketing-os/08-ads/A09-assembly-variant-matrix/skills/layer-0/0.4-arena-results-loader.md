# 0.4: Arena Results Loader

## Purpose

Load AD-ARENA-RESULTS.md from A06 (Arena/Concept Selection) and extract winning concept IDs, Arena scores, per-concept scoring breakdowns, human selection notes, and any priority guidance. This data drives concept numbering consistency and testing priority assignment in Layer 3.

## Inputs

- **A06 Arena Results:** `[project]/A06-arena-concept-selection/AD-ARENA-RESULTS.md` -- the Arena competition results from A06
- **A06 Concept Selection:** Any additional human selection notes or concept approval files from A06

## Process

```
STEP 1: LOCATE ARENA RESULTS FILE
  - Verify AD-ARENA-RESULTS.md exists at expected path
  - IF file does not exist: HALT -- A06 has not completed
  - IF file exists: read entire file

STEP 2: EXTRACT WINNING CONCEPT IDS
  - Identify the winning concepts selected from Arena competition
  - Record concept numbering (C1, C2, C3, etc.)
  - Verify at least 3 winning concepts exist (Gate 0 requirement)
  - Record which round each concept won (Round 1, 2, or 3)

STEP 3: EXTRACT ARENA SCORES
  For each winning concept:
    a. Overall Arena score (weighted composite)
    b. Per-criterion scoring breakdown (all 7 Arena judging criteria)
    c. Round-by-round score progression (R1, R2, R3)
    d. Which competitor generated the winning version
    e. Whether the final output is a pure competitor output or a hybrid

STEP 4: EXTRACT HUMAN SELECTION NOTES
  - Record any human selection reasoning or notes
  - Record any priority guidance from human ("test C1 first because...")
  - Record any concept-specific constraints or instructions
  - Record approval status (confirmed selected vs tentative)

STEP 5: BUILD ARENA-SCORES.yaml
  - Write structured Arena data to output file
  - Include per-concept scores normalized to 0-10 scale (for priority algorithm)
  - Include human selection notes
  - Include concept numbering that MUST be used consistently downstream

STEP 6: VALIDATE COMPLETENESS
  - Verify at least 3 winning concepts
  - Verify all concepts have Arena scores
  - Verify concept numbering is unique and sequential
```
