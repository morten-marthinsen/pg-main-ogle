# 3.2: CTA Swap Groups

## Purpose
For each ad concept, verify the 3 CTA variants designed in Layer 2.6 are truly independent. Verify each CTA works with the same [PROOF] module exit. Design CTA swap points. Add 1-2 additional CTA variants if concept supports them.

## Inputs
- CTA module briefs (from 2.6)
- Offer package (from 0.1)

## Process
Test CTA independence (can swap without changing proof exit). Design swap transition points. Generate 1-2 additional CTA variants (e.g., quiz CTA, free sample CTA).

## Output File
- **Path:** `layer-3-outputs/3.2-cta-swap-groups.md`
- **Minimum size:** 3KB

## Dependencies
- 2.6 complete
