# 0.2: Hook-Angle Matrix Loader

## Purpose
Load the complete Hook-Angle Matrix from A02 output. Extract all human-selected hooks (8-10 minimum), hook type classifications, angle attributions, platform recommendations, scoring data, and any human notes that constrain script architecture.

## Inputs
- A02 HOOK-ANGLE-MATRIX.md (primary output from Hook Generation skill)
- A02 checkpoint file with hook selection confirmation

## Process

### Step 1: Locate and Read Hook-Angle Matrix
```
LOCATE: [project]/A02-hook-generation/HOOK-ANGLE-MATRIX.md
READ entire file
VERIFY file is not empty
VERIFY file contains "HUMAN_SELECTED" section
```

### Step 2: Extract Selected Hooks
```
FOR EACH hook in HUMAN_SELECTED section:
  EXTRACT:
    - hook_id (unique identifier)
    - hook_text (verbatim hook copy)
    - hook_type (from AD-HOOK-TAXONOMY.md classification)
    - angle_id (which angle this hook expresses)
    - angle_statement (the angle in plain language)
    - platform_recommendations (which platforms suit this hook)
    - composite_score (if available)
    - human_notes (any selection rationale or constraints)

VERIFY: hook_count >= 8
IF hook_count < 8: FLAG as insufficient -- A02 must provide minimum 8 selected hooks
```

### Step 3: Build Hook Inventory
```
CREATE hook_inventory as array of hook objects
SORT by angle (group hooks by shared angle)
COUNT hooks per angle
IDENTIFY primary angle (most hooks)
IDENTIFY secondary angles
```

### Step 4: Extract Hook Type Distribution
```
COUNT hooks by type code (A1-A6, B1-B4, C1-C4, D1-D3, E1-E3, F1-F4, G1-G4, H1-H3, I1-I2, J1)
CALCULATE hook_type_distribution percentages
IDENTIFY dominant hook type
```

### Step 5: Platform Recommendation Aggregation
```
FOR EACH hook with platform_recommendations:
  EXTRACT platform affinities (TikTok, Meta, YouTube, etc.)
  AGGREGATE across all hooks
  IDENTIFY which platforms have strongest hook support
```

## Output Schema
```yaml
hook_angle_matrix:
  total_selected_hooks: integer
  hooks:
    - hook_id: string
      hook_text: string
      hook_type: string
      angle_id: string
      angle_statement: string
      platform_recommendations: [list]
      composite_score: float
      human_notes: string or null
  angle_distribution:
    - angle_id: string
      angle_statement: string
      hook_count: integer
      percentage: float
  hook_type_distribution:
    - type_code: string
      type_name: string
      count: integer
      percentage: float
  platform_affinity:
    - platform: string
      hook_count_recommended: integer
  loading_status:
    hooks_loaded: integer
    minimum_met: boolean
    warnings: [list or empty]
```

## Output File
- **Path:** `layer-0-outputs/0.2-hook-angle-matrix-loader.md`
- **Minimum size:** 1KB

## Quality Checks
1. Hook-Angle Matrix file exists and was successfully read
2. At least 8 selected hooks extracted (minimum threshold)
3. Every hook has a hook_type classification
4. Every hook has an angle_id attribution
5. Platform recommendations present for majority of hooks
6. No duplicate hook_ids in the inventory

## Dependencies
- A02 (Hook Generation) must be complete with human selection
- HOOK-ANGLE-MATRIX.md must exist at standard output path
