# 2.7: AV Format Blueprints

## Purpose
Assemble the complete AV (Audio-Visual) two-column blueprint for each ad concept. Left column: visual intent per module. Right column: audio intent per module. Module boundaries clearly marked. Transition types specified at each seam.

## Inputs
- All module briefs (2.2, 2.3, 2.4, 2.5, 2.6)

## Process
Combine all module briefs into unified two-column AV format. Specify visual intent per module (what the visual DOES, not shot-level detail). Specify audio intent per module. Mark module boundaries with transition types.

## Output File
- **Path:** `layer-2-outputs/2.7-av-format-blueprints.md`
- **Minimum size:** 8KB

## Dependencies
- 2.2, 2.3, 2.4, 2.5, 2.6 all complete
