# 1.2: Framework Analysis Matrix

## Purpose
For each ad concept, analyze the 5 selection variables (hook type, platform, length, vertical, awareness level) and score each framework's fit (1-10). Produce a scored matrix per concept.

## Inputs
- Ad Concept Inventory (from 1.1)
- Script Structures (from 0.4 - held in active context)
- Vertical Profile (from 0.0.1)
- Strategic Context (from 0.1)

## Process

### Step 1: Load Required Context
```
READ: layer-1-outputs/1.1-ad-concept-inventory.md
READ: layer-0-outputs/0.4-script-structures-loader.md
READ: layer-0-outputs/0.0.1-vertical-profile-loader.md
READ: layer-0-outputs/0.1-upstream-strategic-loader.md
HOLD: All 8 framework definitions in active context
```

### Step 2: For Each Ad Concept, Extract 5 Selection Variables
```
FOR EACH concept in ad_concept_inventory:
  VARIABLE 1: HOOK TYPE (from concept.hook_type)
    - Pain hooks (C1-C4) → prefer PAS, BAB
    - Authority hooks (B1-B4) → prefer AIDA, Edutainment
    - Curiosity hooks (A1-A6) → prefer Hook-Body-CTA, Edutainment
    - Transformation hooks (D1-D3) → prefer BAB, Story
    - etc.
  
  VARIABLE 2: PLATFORM (from concept.platform_primary)
    - TikTok → prefer Hook-Body-CTA, UGC-DR, Fast-Paced Viral
    - Meta → prefer PAS, BAB, Hook-Body-CTA, UGC-DR
    - YouTube → prefer AIDA, Story, Edutainment
    - etc.
  
  VARIABLE 3: AD LENGTH (from concept.target_length)
    - 6s → Bumper only (not a full framework)
    - 15s → Hook-Body-CTA, Fast-Paced Viral, PAS compressed
    - 30s → PAS, BAB, Hook-Body-CTA
    - 60s → AIDA, PAS, UGC-DR (first length for full mechanism)
    - 2-3min → Story, Edutainment, extended structures
  
  VARIABLE 4: VERTICAL (from vertical_profile)
    - Health → AIDA, UGC-DR, PAS
    - Golf → BAB, Hook-Body-CTA, Edutainment
    - Finance → AIDA, PAS
    - Personal Dev → Story, Edutainment
    - Technology → Hook-Body-CTA, BAB
  
  VARIABLE 5: AWARENESS LEVEL (from strategic_context.campaign_brief)
    - Unaware → Fast-Paced Viral, Story
    - Problem-Aware → PAS, BAB
    - Solution-Aware → AIDA, Edutainment
    - Product-Aware → UGC-DR, Hook-Body-CTA
    - Most-Aware → Hook-Body-CTA with CTA focus
```

### Step 3: Score All 8 Frameworks Against Each Concept
```
FOR EACH concept:
  FOR EACH of 8 frameworks:
    SCORE framework fit on scale 1-10:
      - Check hook type compatibility (does this framework suit this hook type?)
      - Check platform compatibility (is this framework platform-native?)
      - Check length compatibility (can this framework execute at this length?)
      - Check vertical preference (is this framework common in this vertical?)
      - Check awareness compatibility (does this framework match awareness level?)
    
    CALCULATE composite_score = average of 5 variable scores
    
    DOCUMENT rationale for score:
      - Why this score was assigned
      - Which variables favor this framework
      - Which variables disfavor this framework
```

### Step 4: Create Scored Matrix Per Concept
```
FOR EACH concept:
  CREATE framework_scores:
    concept_id: string
    frameworks:
      - framework_code: "PAS"
        composite_score: float (1-10)
        variable_scores:
          hook_type_fit: float
          platform_fit: float
          length_fit: float
          vertical_fit: float
          awareness_fit: float
        rationale: string
      - framework_code: "AIDA"
        [same structure]
      [... all 8 frameworks]
    
    RANK frameworks by composite_score (highest first)
```

## Output Schema
```yaml
framework_analysis:
  total_concepts_analyzed: integer
  concept_analyses:
    - concept_id: string
      five_variables:
        hook_type: string
        platform: string
        ad_length: string
        vertical: string
        awareness_level: string
      framework_scores:
        - framework_code: string
          composite_score: float
          variable_scores: object
          rationale: string
      ranked_frameworks: [list of codes, highest score first]
```

## Output File
- **Path:** `layer-1-outputs/1.2-framework-analysis-matrix.md`
- **Minimum size:** 5KB

## Quality Checks
1. Every ad concept has analysis for all 8 frameworks
2. Every framework score has 5 variable scores (none missing)
3. Composite scores are within 1-10 range
4. Frameworks are ranked by score (descending)
5. Rationale explains the score (not empty)

## Dependencies
- 1.1 (Ad Concept Inventory) must be complete
- 0.4 (Script Structures) must be loaded in active context
