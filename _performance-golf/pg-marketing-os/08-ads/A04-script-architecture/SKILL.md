# A04 — Script Architecture

**Pipeline Position:** After A03 (Format Strategy). Feeds A05 (Visual Direction) and A07 (Copy Production).

---

## PURPOSE

Design modular script structures — the architectural blueprints that A07 fills with
copy. The script serves the hook AND the big idea; the hook promise must be fulfilled
by the body.

**Three Laws:**
1. The script serves the hook AND the big idea
2. Word count is physics, not suggestion (derived from speech rate)
3. Modules are independent, scripts are not (swappable modules, coherent flow)
