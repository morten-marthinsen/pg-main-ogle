# 1.2: Engagement Scorer

**Version:** 1.0
**Layer:** 1 (Individual Variant Scoring)
**Type:** Judgment / Prediction
**Model:** opus
**Dependencies:** 0.1 (Variant Matrix), 0.3 (Arena Scores), 0.5 (Benchmarks)
**Output:** engagement_scores.md

---

## Purpose

Score EVERY variant on Dimension 2: Engagement Prediction. Engagement predicts whether a user who stops scrolling will CONTINUE WATCHING/READING past the first 3 seconds. This is the bridge between scroll-stop and conversion. Every score must cite evidence from A06 Arena scores, script structure analysis, word count compliance, and platform nativeness assessment.

---

## Process

### Step 1: Load All Evidence Sources

```
READ all input files
FOR EACH variant, COLLECT available evidence:

EVIDENCE SOURCE A: Arena Overall Score (from 0.3)
  LOOK UP variant's parent concept in arena_scores
  EXTRACT overall_weighted_score (1-10)
  This is the strongest single predictor of engagement quality

EVIDENCE SOURCE B: Arena Sub-Scores (from 0.3)
  EXTRACT:
    - mechanism_clarity: float (clear mechanism = sustained attention)
    - visual_copy_coherence: float (visual-copy alignment = engagement)
    - platform_nativeness: float (native-feeling = less skip-resistance)

EVIDENCE SOURCE C: Script Structure Analysis
  ANALYZE variant's body_text against AD-SCRIPT-STRUCTURES.md:
    - IDENTIFY which of 8 frameworks the script follows (if any)
    - CHECK pacing: does the script maintain momentum?
    - CHECK transitions: are they smooth or jarring?
    - CHECK information density: too dense = overwhelming, too sparse = boring
  IF follows proven framework: +0.5
  IF pacing/transitions are smooth: +0.5
  IF no recognizable structure: -0.5

EVIDENCE SOURCE D: Word Count Compliance
  CHECK variant's ad_length against target for its format type:
    - 15-second video: 35-50 words
    - 30-second video: 70-90 words
    - 60-second video: 140-170 words
    - Static ad: 20-40 words primary text
  CALCULATE deviation:
    - Within range: neutral
    - 10-20% over: -0.5 (attention fatigue risk)
    - 20%+ over: -1.0 (significant engagement drop)
    - Under range: neutral (brevity can be good) unless core info missing

EVIDENCE SOURCE E: Platform Nativeness
  ASSESS whether variant feels native to its target platform:
    - Meta: clean, professional, clear CTA
    - TikTok: raw, authentic, creator-style
    - YouTube: informational, longer-form friendly
    - Google: direct, benefit-led, keyword-aligned
  IF Arena platform_nativeness score exists: use it
  IF not: assess from copy style and format alignment
```

### Step 2: Score Each Variant

```
FOR EACH variant in the inventory:

  1. START with Arena overall_weighted_score as baseline (if available)
     IF no Arena score: START at 5.0 (neutral baseline)

  2. ADJUST for mechanism clarity:
     - Arena mechanism_clarity >= 8: +0.5 (clear mechanisms sustain attention)
     - Arena mechanism_clarity <= 4: -0.5 (confusing mechanisms lose audience)

  3. ADJUST for visual-copy coherence:
     - Arena visual_copy_coherence >= 8: +0.5
     - Arena visual_copy_coherence <= 4: -0.5

  4. ADJUST for script structure:
     - Follows proven framework: +0.5
     - No structure / chaotic flow: -0.5

  5. ADJUST for word count compliance:
     - Within range: 0
     - 10-20% over: -0.5
     - 20%+ over: -1.0

  6. ADJUST for platform nativeness:
     - Arena score >= 8 or clearly native: +0.5
     - Arena score <= 4 or clearly non-native: -0.5

  7. CLAMP final score to 1-10 range

  8. DETERMINE confidence range and level:
     - Use same methodology as 1.1 (count evidence sources, ±1 to ±2)

  9. BUILD evidence chain citing specific data
```

### Step 3: Apply Scoring Scale

```
SCORING SCALE:

9-10: HIGH predicted engagement (70%+ of scroll-stoppers will complete/engage)
      Evidence: high Arena overall score + proven framework + platform-native +
      strong visual-copy coherence

7-8:  MODERATE-HIGH predicted engagement (50-70%)
      Evidence: solid Arena score + reasonable structure + platform-appropriate

5-6:  MODERATE predicted engagement (30-50%)
      Evidence: adequate but not exceptional — some structural weaknesses
      or platform mismatch

3-4:  LOW predicted engagement (15-30%)
      Evidence: weak structure OR platform mismatch OR over-length OR
      poor visual-copy coherence

1-2:  VERY LOW predicted engagement (<15%)
      Evidence: multiple engagement barriers — poor structure + non-native +
      over-length + unclear mechanism
```

### Step 4: Verify Differentiation

```
AFTER scoring all variants:
  CHECK score spread and distribution
  SAME differentiation rules as 1.1: reject if all within 1.5-point range
```

6. Platform nativeness assessed for every variant
7. Scores are meaningfully differentiated
8. Variants without Arena scores have LOW confidence noted
