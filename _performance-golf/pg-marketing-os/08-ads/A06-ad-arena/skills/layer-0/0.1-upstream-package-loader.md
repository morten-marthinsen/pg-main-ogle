# 0.1: Upstream Package Loader

## Purpose
Load Campaign Brief (Skill 09), Research Handoff (Skill 01), Root Cause (Skill 03), Mechanism (Skill 04), Promise (Skill 05), Big Idea (Skill 06), and Ad Intelligence (A01). Extract key strategic elements that provide campaign context for persona evaluations: big idea statement, mechanism name, root cause anchor, target audience, awareness level, competitive landscape.

## Inputs
- Campaign Brief (Skill 09): `[project]/09-campaign-brief/campaign-brief.json`
- Research Handoff (Skill 01): `[project]/01-research/FINAL_HANDOFF.md`
- Root Cause (Skill 03): `[project]/03-root-cause/root-cause-package.yaml`
- Mechanism (Skill 04): `[project]/04-mechanism/mechanism-package.json`
- Promise (Skill 05): `[project]/05-promise/promise-output.json`
- Big Idea (Skill 06): `[project]/06-big-idea/big-idea-output.json`
- Ad Intelligence (A01): `[project]/A01-ad-intelligence/AD-INTELLIGENCE-PACKAGE.md`

## Process
```
STEP 1: Verify all upstream files exist
  - CHECK each file path
  - IF any missing: Flag as BLOCKING (which file, which skill)
  - IF all present: Proceed to loading

STEP 2: Load Campaign Brief (Skill 09)
  - EXTRACT: target_audience (demographics, psychographics)
  - EXTRACT: awareness_level (Schwartz stage)
  - EXTRACT: campaign_objective (lead gen, purchase, retargeting, etc.)
  - EXTRACT: campaign_tone/voice direction
  - EXTRACT: compliance_requirements

STEP 3: Load Research Handoff (Skill 01)
  - EXTRACT: key pain points (top 5)
  - EXTRACT: key desires (top 5)
  - EXTRACT: market sophistication level
  - EXTRACT: competitive landscape summary

STEP 4: Load Strategic Packages (Skills 03-06)
  - FROM Root Cause: root_cause_statement, anchor_phrase, villain
  - FROM Mechanism: mechanism_name, mechanism_explanation, simplification_metaphor
  - FROM Promise: primary_promise, proof_ceiling
  - FROM Big Idea: big_idea_statement, FSSIT_elements

STEP 5: Load Ad Intelligence (A01)
  - EXTRACT: competitive ad landscape
  - EXTRACT: winning ad patterns in this niche
  - EXTRACT: platform performance benchmarks

STEP 6: Compile strategic context summary for persona evaluation
```
