# 0.4: Visual Direction Loader

## Purpose
Load A05 VISUAL-DIRECTION-PACKAGE.md. For each script, load the corresponding visual direction: treatment type (Talking Head, B-Roll+VO, Text-on-Screen, Screen Recording, Mixed), shot list, mood, reference ads, and aspect ratio.

## Inputs
- A05 VISUAL-DIRECTION-PACKAGE.md: `[project]/A05-visual-direction/VISUAL-DIRECTION-PACKAGE.md`
- Script inventory from 0.3-script-package-loader output

## Process
```
STEP 1: Load VISUAL-DIRECTION-PACKAGE.md
  - READ full file
  - IDENTIFY all visual directions

STEP 2: For EACH script (from 0.3 output):
  - MATCH script to its corresponding visual direction
  - IF no visual exists for a script: Flag as INCOMPLETE
  - IF visual exists: Extract full direction

STEP 3: For EACH matched visual direction, extract:
  - Visual ID (V-001, V-002, etc.)
  - Associated Script ID (S-XXX)
  - Treatment type (Talking Head, B-Roll+VO, Text-on-Screen, Screen Recording, Mixed)
  - Shot list (shot-by-shot breakdown with shot type, subject, action, duration)
  - Mood / visual tone
  - Reference ads (if specified)
  - Aspect ratio (9:16 vertical, 1:1 square, 16:9 horizontal)
  - Key visual moments (opening frame, transition points, CTA visual)
  - Production level (UGC raw, semi-produced, fully produced)

STEP 4: Validate script-to-visual mapping completeness
  - Every script should have a visual direction
  - Flag any gaps

STEP 5: Compile visual inventory for concept assembly
```
