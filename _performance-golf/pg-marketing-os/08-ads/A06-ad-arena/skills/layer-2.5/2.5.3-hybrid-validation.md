# 2.5.3: Hybrid Validation

## Purpose
Validate each hybrid concept against four checks: (a) coherence -- do all improvements work together as one concept? (b) consistency -- does the concept feel like ONE ad, not a committee? (c) threshold -- does the hybrid score >= 8.0 weighted? (d) differentiation -- does it meaningfully differ from the best pure Round 2 (FINAL) evaluation? Discard hybrids that fail coherence or don't improve on the best pure version.

## Inputs
- 2.5.2-hybrid-concept-generation output (hybrid concepts with scores)
- 2.14-round3-final-scoring output (best pure Round 2 (FINAL) scores for comparison)

## Process
```
STEP 1: Load hybrid concepts and pure Round 2 (FINAL) scores
  - READ: layer-2.5-outputs/2.5.2-hybrid-concept-generation.md
  - READ: layer-2-outputs/2.14-round3-final-scoring.md

STEP 2: FOR EACH hybrid concept:

  CHECK A: Coherence
    - Do the combined improvements work together?
    - Does improving the hook conflict with the script structure?
    - Does the visual direction still match the improved script?
    - Is the mechanism clarity maintained after integration?
    - Coherence score: [1-10] -- must be >= 7 to pass

  CHECK B: Consistency
    - Does the hybrid feel like ONE unified ad concept?
    - Or does it feel like suggestions from different people stitched together?
    - Would a producer receive this and think "clear vision" or "confused direction"?
    - Consistency score: [1-10] -- must be >= 7 to pass

  CHECK C: Threshold
    - Weighted score >= 8.0?
    - Scroll-Stop Power >= 7.0?
    - Platform Nativeness >= 6.5?
    - No criterion below 5.0?

  CHECK D: Differentiation
    - Does this hybrid score HIGHER than the best pure Round 2 (FINAL) evaluation?
    - Is it meaningfully different in approach/character from the pure version?
    - IF hybrid scores LOWER than best pure: Discard (no value added)

STEP 3: PASS/FAIL each hybrid
  - PASS: Coherence >= 7, Consistency >= 7, Threshold met, Differentiates from pure
  - FAIL: Any check below threshold -> discard with reason

STEP 4: Verify minimum 2 passing hybrids remain
  - IF fewer than 2 pass: Attempt to generate additional hybrids
  - IF still fewer than 2 after retry: Present what exists with note

STEP 5: Compile final hybrid inventory for human selection
```
