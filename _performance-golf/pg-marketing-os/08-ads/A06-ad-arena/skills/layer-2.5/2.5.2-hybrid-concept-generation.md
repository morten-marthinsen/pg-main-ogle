# 2.5.2: Hybrid Concept Generation

## Purpose
For each concept that met or nearly met thresholds, generate 2-3 IMPROVED versions by combining the best non-conflicting improvement suggestions across all 7 personas. Each hybrid must be a coherent, integrated concept -- not a Frankenstein of unrelated suggestions. Score each hybrid against all 7 criteria.

## Inputs
- 2.5.1-improvement-decomposition output (best-suggestion sets per concept, conflict maps)
- 1.3-concept-summary-generator output (original concept summaries)
- 2.14-round3-final-scoring output (scores -- to identify qualifying concepts)

## Process
```
STEP 1: Identify qualifying concepts for synthesis
  - Concepts that MET threshold (>= 8.0 weighted): Synthesize to create even better versions
  - Concepts NEAR threshold (7.0-7.9 weighted): Synthesize to push over threshold
  - Concepts FAR below threshold (< 7.0): Skip synthesis -- structural issues

STEP 2: FOR EACH qualifying concept, generate 2-3 hybrid variants:

  HYBRID VARIANT A: "Best-of-All" hybrid
    - Take the best non-conflicting suggestion for EVERY aspect
    - Reconstruct the concept incorporating all best suggestions
    - This is the "maximum improvement" version

  HYBRID VARIANT B: "Strongest-Lens" hybrid
    - Identify the persona whose overall evaluation was strongest
    - Use that persona's improvement recommendations as the BASE
    - Layer in the single highest-significance suggestion from each other persona
    - This is the "coherent single-vision + selective enhancement" version

  HYBRID VARIANT C (optional): "Gap-Targeted" hybrid
    - Identify the weakest criterion across all personas
    - Build hybrid focused on maximizing that specific criterion
    - Accept minor tradeoffs on strong criteria to fix the weak one

STEP 3: For EACH hybrid, WRITE a coherent improved concept description
  - NOT a spliced list of suggestions
  - WRITE as one integrated concept (hook + script + visual)
  - Must READ as a unified ad concept, not committee output

STEP 4: Score each hybrid against all 7 criteria
  - Use same scoring rubrics as Arena rounds
  - Score with evidence
  - Calculate weighted total
  - Check threshold (>= 8.0 to advance)

STEP 5: Verify hybrids are meaningfully different from each other
  - If all hybrids converge to same concept: only 1 unique hybrid
  - Need minimum 2 meaningfully different hybrids
```
