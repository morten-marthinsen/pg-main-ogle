# 2.7: Round 2 Adversarial Critique

## Purpose
Same Critic protocol as Round 1. The Critic checks if previous-round weaknesses were addressed. If a material weakness exists, identifies the SINGLE most impactful NEW weakness for each persona evaluation (may differ from Round 1 weakness). Maps to specific criterion. Provides actionable fix direction. If no element materially underperforms, reports `no_material_weakness` with a `strength_note`.

## Inputs
- 2.6-round2-persona-evaluation output (all 7 Round 2 persona evaluations per concept)
- 2.2-round1-adversarial-critique output (previous round critiques for comparison)

## Process
```
STEP 1: Load Round 2 evaluations and Round 1 critiques
  - READ: layer-2-outputs/2.6-round2-persona-evaluation.md
  - READ: layer-2-outputs/2.2-round1-adversarial-critique.md (for comparison)

STEP 2: FOR EACH concept:
  FOR EACH persona evaluation (7 total):

    A. CHECK if Round 1 weakness was addressed
      - Compare Round 1 critique to Round 2 evaluation
      - Was the specific weakness fixed? [Y/N]

    B. ASSESS whether a material weakness exists
      - With Round 1 weakness (hopefully) fixed, is there NOW a material weakness?
      - If yes, identify the SINGLE most impactful one — may be a DIFFERENT criterion than Round 1
      - Or it may be the SAME criterion if not adequately fixed
      - If no element materially underperforms, report no_material_weakness

    C. MAP to specific criterion
    D. PROVIDE actionable fix direction (specific, not vague)
    E. CITE evidence from the Round 2 evaluation
    F. ASSESS severity (1-10)

CRITICAL RULES:
  - Round 2 critique must be DIFFERENT from Round 1 if weakness was fixed
  - If Round 1 weakness was NOT fixed, critique the same weakness with updated evidence
  - Critic remains adversarial -- but honest. If learning genuinely resolved weaknesses, report no_material_weakness
  - The fact that scores improved does NOT mean critique is unnecessary -- but it may mean no_material_weakness is the correct assessment
```
