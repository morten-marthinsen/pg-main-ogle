# 2.6: Round 2 Persona Evaluation

## Purpose
All 7 personas receive the Learning Brief from Round 1. Each persona re-evaluates each concept incorporating learned techniques. Key rule: Absorb TECHNIQUES, not PERSPECTIVE. Each persona notes which techniques they integrated and how.

## Inputs
- 2.5-round1-learning-brief output (Learning Brief with techniques and persona-specific guidance)
- 1.3-concept-summary-generator output (concept summaries)
- 0.5-persona-specimen-validator output (specimens still in active context)
- Round 1 winner evaluations per concept (from 2.1 output)
- Round 1 critiques per persona (from 2.2 output)

## Process
```
STEP 1: Load Learning Brief and previous round context
  - READ: layer-2-outputs/2.5-round1-learning-brief.md
  - RECALL: Specimens still in active context
  - RECALL: Own Round 1 critique (what weakness was identified)

STEP 2: FOR EACH concept:
  FOR EACH persona (7 total):

    A. ABSORB Learning Brief techniques:
      - Read persona-specific guidance from Learning Brief
      - Identify which winner technique to integrate
      - Plan HOW to integrate without losing own perspective

    B. RE-EVALUATE concept with integrated techniques (effort: max)
      - Fresh evaluation, not a patch on Round 1
      - Apply absorbed technique to own evaluative lens
      - Reference loaded specimens (still in context)
      - Address own Round 1 weakness

    C. DOCUMENT technique integration:
      - "Technique absorbed: [description]"
      - "How integrated: [specific application to my evaluation]"
      - "Perspective preserved: [what I maintained from my lens]"

    D. SCORE against 7 criteria (fresh scores)
      - All 7 criteria scored independently
      - Evidence provided for each
      - Weighted total calculated

    E. GENERATE updated improvement recommendation

CRITICAL RULES:
  - Absorb TECHNIQUES, not PERSPECTIVE
  - DR Strategist learning from UGC Native's authenticity insights
    does NOT abandon conversion focus -- integrates authenticity
    INTO the conversion evaluation
  - Each persona must NOTE which techniques they absorbed
  - Scores should reflect genuine improvement from learning, not inflation
  - Specimens MUST still be referenced
```
