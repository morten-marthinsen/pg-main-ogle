# 2.11 — Audience Evaluation

**Purpose:** Deploy audience agents on Round 1 outputs for blind evaluation across 8 dimensions.

**Reference:** `~system/protocols/AUDIENCE-AGENT-PROTOCOL.md`

---

## Inputs
- All 7 revised Round 1 ad variant outputs
- Project audience research (from 00-deep-research)
- Audience agent constructor (AUDIENCE-AGENT-PROTOCOL.md)

## Process
1. Construct 5-7 audience agents from project's audience research
2. Each agent evaluates ALL 7 Round 1 outputs BLIND (no persona attribution)
3. Evaluation across 8 dimensions:
   - Believability, Emotional resonance, Objection activation, Purchase intent
   - Clarity of promise, Mechanism credibility, Proof sufficiency, Personal relevance
4. Each agent produces per-output scores and qualitative feedback
5. Pass all evaluations to Analyst (Step 2.12)

## Output
- `audience-evaluation-raw.yaml` -- All agent evaluations with scores and feedback
