# 1.4: Base Copy -- Concept 3

## Purpose
Write the complete base copy for Concept 3 (third Arena-approved concept). Fill every module of the A04 script architecture with actual words. Produce a complete AV-format script within word count limits, with specific visual column entries and persona-calibrated voice. Same protocol as 1.2 applied to C-003. If more than 3 concepts are approved, create additional microskill files (1.5-base-copy-concept-4.md, 1.6-base-copy-concept-5.md, etc.) following this same template.

## Inputs
- Concept mini-brief for C-003 (from 1.1)
- Persona voice specimens (from 0.7 -- held in active context)
- Soul.md constraints (if loaded)
- AD-SCRIPT-STRUCTURES.md (framework reference)

## Process
```
STEP 1: READ the concept mini-brief for C-003 from 1.1 output.
  - Internalize: strategic angle, framework, word budgets, visual direction
  - NOTE: C-003 may have a DIFFERENT framework, different platforms,
    and different visual treatment. Do not carry over assumptions from C-001 or C-002.

STEP 2: READ the A04 script architecture for this concept.
  - Identify every module
  - Identify the framework
  - Note word count budget per module

STEP 3: READ the A05 visual direction for this concept.
  - Visual treatment type, visual cues, copy-visual complementarity

STEP 4: PLAN the word budget per module for this concept's target length.

STEP 5: WRITE each module as the assigned framework dictates.
  - Use persona voice specimens as statistical attractors
  - Respect Soul.md voice constraints
  - Write AUDIO column and VISUAL column

STEP 6: VERIFY word count (AUDIO column only).
  - IF over limit: compress
  - IF under limit by more than 20%: expand

STEP 7: VERIFY visual column specificity.

NOTE: If A06 approved more than 3 concepts, this microskill template
is replicated for each additional concept (1.5, 1.6, etc.).
The validator (1.5 or the next available number after all concept files)
runs AFTER all concept base copies are complete.
```
