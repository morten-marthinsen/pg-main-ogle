# 1.3: Base Copy -- Concept 2

## Purpose
Write the complete base copy for Concept 2 (second Arena-approved concept). Fill every module of the A04 script architecture with actual words. Produce a complete AV-format script within word count limits, with specific visual column entries and persona-calibrated voice. Same protocol as 1.2 applied to C-002.

## Inputs
- Concept mini-brief for C-002 (from 1.1)
- Persona voice specimens (from 0.7 -- held in active context)
- Soul.md constraints (if loaded)
- AD-SCRIPT-STRUCTURES.md (framework reference)

## Process
```
STEP 1: READ the concept mini-brief for C-002 from 1.1 output.
  - Internalize: strategic angle, framework, word budgets, visual direction
  - NOTE: C-002 may have a DIFFERENT framework, different platforms,
    and different visual treatment than C-001. Do not carry over assumptions.

STEP 2: READ the A04 script architecture for this concept.
  - Identify every module: [HOOK], [SETUP], [MECHANISM], [PROOF], [CTA]
  - Identify the framework (PAS, AIDA, BAB, etc.)
  - Note word count budget per module

STEP 3: READ the A05 visual direction for this concept.
  - Identify visual treatment type
  - Note specific visual cues
  - Understand copy-visual complementarity for THIS concept

STEP 4: PLAN the word budget per module for this concept's target length.

STEP 5: WRITE each module as the assigned framework dictates.
  - Use persona voice specimens as statistical attractors
  - Respect Soul.md voice constraints
  - Write AUDIO column with dialogue/voiceover
  - Write VISUAL column with specific shot descriptions

STEP 6: VERIFY word count (AUDIO column only).
  - IF over limit: compress
  - IF under limit by more than 20%: expand

STEP 7: VERIFY visual column specificity.
  - Every row MUST specify shot type, subject, and action
```
