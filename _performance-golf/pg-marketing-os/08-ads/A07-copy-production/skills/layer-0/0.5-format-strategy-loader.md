# 0.5: Format Strategy Loader

## Purpose
Load A03 FORMAT-STRATEGY.md. Extract platform assignments per concept, target ad lengths per platform, and platform-specific constraints (sound-on/off, vertical/horizontal, character limits, CTA button options).

## Inputs
- A03 FORMAT-STRATEGY.md (from Format Strategy skill output)

## Process
```
STEP 1: LOCATE A03 Format Strategy.
  - Expected path: [project]/A03-format-strategy/FORMAT-STRATEGY.md
  - IF file does not exist: HALT -- cannot do platform adaptations without platform assignments

STEP 2: READ FORMAT-STRATEGY.md completely.

STEP 3: EXTRACT per concept:
  a. Platform assignments (which platforms this concept targets)
  b. Target ad lengths per platform
  c. Primary platform (where the base copy is written for)

STEP 4: EXTRACT platform-specific constraints:
  a. Sound-on vs sound-off defaults
  b. Aspect ratio (vertical 9:16, horizontal 16:9, square 1:1)
  c. Character limits (for text-based ad components)
  d. CTA button options (platform-native CTAs)
  e. Caption requirements
  f. Maximum ad length

STEP 5: CREATE platform constraint matrix for easy reference during Layer 2.

STEP 6: WRITE extracted data to output file.
```
