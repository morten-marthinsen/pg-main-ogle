# 0.2: Arena Results Loader

## Purpose
Load A06 AD-ARENA-RESULTS.md. Extract winning concept IDs, concept descriptions, Arena scores per criterion, human selection notes, and any human edits to selected concepts. This is the PRIMARY input -- copy is written for THESE concepts ONLY.

## Inputs
- A06 AD-ARENA-RESULTS.md (from Ad Arena skill output)

## Process
```
STEP 1: LOCATE A06 Arena Results.
  - Expected path: [project]/A06-ad-arena/AD-ARENA-RESULTS.md
  - IF file does not exist: HALT -- cannot write copy without approved concepts

STEP 2: READ AD-ARENA-RESULTS.md completely.

STEP 3: EXTRACT for each winning concept:
  a. Concept ID (C-001, C-002, C-003, etc.)
  b. Concept name/description
  c. Strategic angle (the persuasion approach)
  d. Arena scores (per-criterion breakdown)
  e. Overall Arena score
  f. Human selection notes (why the human reviewer chose this concept)
  g. Human edits (any modifications made during selection)
  h. Concept strengths (from Arena judge feedback)
  i. Concept weaknesses / areas to watch (from Arena critique)

STEP 4: VERIFY minimum concept count.
  - At least 2 Arena-approved concepts required
  - IF fewer than 2: HALT -- A06 must produce at least 2 approved concepts

STEP 5: NOTE any human-modified elements.
  - These override Arena defaults
  - Copy must reflect human edits, not pre-edit versions

STEP 6: WRITE extracted data to output file.
```
