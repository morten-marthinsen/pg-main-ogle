# 0.3: Hook-Angle Matrix Loader

## Purpose

Load the 8-10 human-selected hooks from the A02 Hook-Angle Matrix. This microskill extracts each hook's ID, text, type classification, category, source angle, platform tags, human notes (binding constraints), big idea alignment score, and compliance safety score. These hooks are what Layer 1-2 will map to platforms and formats.

## Inputs

- HOOK-ANGLE-MATRIX.md from A02: `[project]/A02-hook-angle-matrix/HOOK-ANGLE-MATRIX.md`

## Process

1. **READ complete HOOK-ANGLE-MATRIX.md**
   - Full read required — every selected hook must be captured
   - Do NOT summarize or skip any hooks

2. **IDENTIFY the selected hooks section**
   - Should contain 8-10 hooks marked as SELECTED by human
   - If fewer than 8: WARN but proceed
   - If more than 10: WARN (may indicate selection error)

3. **FOR EACH selected hook, EXTRACT:**
   - **Hook ID:** Unique identifier (e.g., H-001, H-002)
   - **Hook text:** Full verbatim text (not summarized)
   - **Hook type code:** From 32-type taxonomy (e.g., A6, B3, C2, D1)
   - **Hook category:** Letter designation A through J
   - **Source angle:** Which angle from A02 generated this hook
   - **Angle ID:** Identifier for the source angle
   - **Platform tags:** If A02 scoring included platform recommendations (optional)
   - **Human notes:** Any binding constraints or directions from human selector (CRITICAL)
   - **Big idea alignment score:** Numerical score from A02 scoring
   - **Compliance safety score:** Numerical score from A02 compliance check

4. **BUILD working hook table** for Layer 1-2 processing:
   - Table format with all extracted fields
   - Organized by Hook ID for easy reference
   - Human notes prominently displayed (these are binding)

5. **VALIDATE extraction:**
   - All 8-10 hooks have complete data (no missing fields)
   - Hook types are valid taxonomy codes
   - Scores are numerical
   - If any human notes exist, they are captured verbatim

6. **HOLD in context** for Layer 1-2 consumption
   - Layer 1 will assess each hook against each platform
   - Layer 2 will assign formats to each hook-platform pair
   - Human notes are BINDING — they constrain format decisions
