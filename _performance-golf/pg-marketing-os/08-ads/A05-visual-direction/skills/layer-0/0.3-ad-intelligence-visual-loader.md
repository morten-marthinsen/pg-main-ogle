# 0.3: Ad Intelligence Visual Loader

## Purpose
Load A01 AD-INTELLIGENCE-HANDOFF.md -- specifically the Visual Style Analysis section. Extract competitor visual styles (UGC vs polished ratios per platform), color patterns, text overlay patterns, and winning visual specimens (visual descriptions of top 20 ads). This competitive intelligence shapes differentiation strategy in Layer 1.

## Inputs
- A01 AD-INTELLIGENCE-HANDOFF.md (REQUIRED -- A05 cannot execute without this)

## Process
1. READ A01 AD-INTELLIGENCE-HANDOFF.md from the project outputs directory
2. LOCATE the Visual Style Analysis section
3. EXTRACT competitor visual styles:
   - UGC vs polished ratio per platform (e.g., "Meta: 70% UGC, 30% polished")
   - Dominant treatment types in the competitive landscape
   - Treatment types competitors are NOT using (whitespace opportunities)
4. EXTRACT color patterns:
   - Dominant colors used by competitors
   - Color differentiation opportunities
5. EXTRACT text overlay patterns:
   - Average text overlay frequency
   - Common text overlay styles (font types, sizes, animation)
   - Text overlay placement patterns
6. EXTRACT winning visual specimens:
   - Visual descriptions of top 20 performing competitor ads
   - What visual elements correlated with high performance
   - What visual elements correlated with poor performance
7. EXTRACT any additional visual intelligence:
   - Thumbnail patterns (for YouTube)
   - Pacing patterns (cut frequency, shot duration averages)
   - Talent patterns (demographics, presentation style)
8. BUILD competitive visual intelligence summary
9. WRITE to output file
