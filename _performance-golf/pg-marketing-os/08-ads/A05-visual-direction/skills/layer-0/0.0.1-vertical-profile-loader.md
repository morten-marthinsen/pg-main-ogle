# 0.0.1: Vertical Profile Loader

## Purpose
Load ad-specific vertical configuration from `ad-verticals/`. Extract vertical-specific color palettes, visual style preferences, compliance constraints (e.g., health: no before/after weight loss on Meta), and anti-slop visual patterns. This is the FIRST microskill to execute -- everything downstream depends on vertical context.

## Inputs
- Vertical profile file from `ad-verticals/[vertical].md`
- Project brief (to identify which vertical applies)

## Process
1. IDENTIFY the campaign vertical from the project brief (golf, health, finance, personal-dev, technology)
2. READ the corresponding vertical profile file from `ad-verticals/[vertical].md`
3. EXTRACT the following vertical-specific configurations:
   - Color palettes (primary, secondary, accent -- with hex codes)
   - Visual style preferences (which treatment types are default/preferred for this vertical)
   - Compliance constraints (platform-specific visual restrictions for this vertical)
   - Anti-slop visual patterns (visual descriptions that are banned for this vertical)
   - Persona panel configuration (if vertical overrides default Arena panel)
4. IF no vertical is specified in the project brief:
   - Default to universal configuration
   - Log warning: "No vertical specified -- using universal visual configuration"
5. WRITE extracted configuration to output file
