# 1.2: Color Palette Designer

## Purpose
Design color palette for each concept. Start with vertical-specific palette (from Color Psychology reference section), adapt for brand guidelines, differentiate between concepts where possible. Output: primary color, secondary color, accent color, text color, background color -- with hex codes and usage rules per concept.

## Inputs
- Treatment type assignments from 1.1 (treatment type affects color approach)
- Vertical color palette from 0.0.1 (vertical-specific color psychology)
- Brand colors from 0.4 (brand guidelines must be respected)
- Competitive color patterns from 0.3 (differentiation opportunities)
- Color Psychology by Vertical section from AGENT.md

## Process
1. READ treatment type assignments from 1.1
2. READ vertical color palette from 0.0.1
3. READ brand color guidelines from 0.4
4. READ competitive color patterns from 0.3
5. ESTABLISH campaign-level color foundation:
   a. START with vertical-specific palette as base
   b. MERGE with brand colors (brand colors take precedence where they exist)
   c. IDENTIFY colors competitors are overusing (avoid or use strategically)
6. FOR EACH concept:
   a. SELECT primary color (the dominant visual color)
   b. SELECT secondary color (supporting color, contrasts with primary)
   c. SELECT accent color (CTAs, highlights, attention-drawing elements)
   d. SELECT text color (must meet WCAG contrast ratio on backgrounds)
   e. SELECT background color (clean, not competing with content)
   f. ADJUST palette based on treatment type:
      - Polished: Richer, more controlled palette
      - UGC: More natural, less "designed" palette
      - Hybrid: Balanced approach
      - Demonstration: Neutral backgrounds to let product shine
      - Testimonial: Warm, inviting tones
   g. DOCUMENT usage rules for each color
7. VERIFY WCAG contrast ratios:
   - Text on background: minimum 4.5:1 ratio
   - Important text: minimum 7:1 ratio
8. DIFFERENTIATE between concepts where strategic (avoid making all concepts look identical)
9. WRITE to output file
