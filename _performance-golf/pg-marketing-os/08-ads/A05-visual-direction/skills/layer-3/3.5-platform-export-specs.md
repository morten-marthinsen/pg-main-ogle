# 3.5: Platform Export Specifications

## Purpose
For each target platform, compile final technical specifications: aspect ratio and resolution (exact pixel dimensions), file format and codec requirements, frame rate, maximum file size, text safe zone coordinates (pixel-level), audio specs (bit rate, channels, loudness normalization), platform-specific ad format requirements (Meta responsive formats, YouTube companion banners, TikTok Spark Ads specs). These are the production-ready technical specs A08 needs for final export.

## Inputs
- Platform constraints from 0.2 (platform requirements loaded from A03)
- Format strategy from A03 (platform assignments per concept)
- Script inventory from 0.1 (which concepts target which platforms)

## Process
1. READ platform constraints from 0.2
2. READ format strategy details from A03
3. READ script inventory from 0.1
4. FOR EACH target platform:
   a. COMPILE video/image specifications:
      - Aspect ratio: [exact ratio, e.g., 4:5]
      - Resolution: [exact pixels, e.g., 1080x1350]
      - File format: [MP4, MOV, etc.]
      - Codec: [H.264, H.265, etc.]
      - Frame rate: [24fps, 30fps, 60fps]
      - Maximum file size: [in MB]
      - Bitrate: [minimum/recommended]
   b. COMPILE safe zone coordinates:
      - Top safe: [pixels from top where platform UI overlays]
      - Bottom safe: [pixels from bottom where CTA/controls overlay]
      - Left safe: [pixels from left edge]
      - Right safe: [pixels from right edge where engagement icons overlay]
      - Text zone: [pixel coordinates of safe text area]
   c. COMPILE audio specifications:
      - Format: [AAC, MP3, etc.]
      - Sample rate: [44.1kHz, 48kHz]
      - Channels: [mono, stereo]
      - Loudness normalization: [LUFS target]
      - Bitrate: [minimum]
   d. COMPILE platform-specific ad format requirements:
      - Meta: responsive format variants, carousel specs, collection ad specs
      - YouTube: companion banner specs, pre-roll requirements, Shorts specs
      - TikTok: Spark Ads specs, TopView specs, In-Feed specs
      - Google Display: responsive display specs, image sizes
   e. DOCUMENT delivery requirements:
      - Upload format per platform
      - Naming convention (if required)
      - Metadata requirements (captions, alt text, etc.)
5. CREATE concept-to-platform delivery matrix:
   - Which concepts export to which platforms
   - Which format variants needed per concept
6. WRITE to output file
