# 0.3: Arena Results Loader

## Purpose
Load A06 AD-ARENA-RESULTS.md containing winning concepts, human selections, and critique notes. This provides context on WHY concepts won -- what made them effective -- which informs production decisions about emphasis, tone, and visual priority.

## Inputs
- A06 AD-ARENA-RESULTS.md (from upstream A06 -- Ad Arena skill)

## Process
```
STEP 1: LOCATE ARENA RESULTS
  - Search project directory for AD-ARENA-RESULTS.md
  - IF not found: HALT -- cannot proceed without knowing which concepts won
  - Log file path and size

STEP 2: LOAD AND PARSE
  - READ the full Arena results document
  - EXTRACT:
    - List of winning concepts (concept IDs)
    - Human selection notes (why each concept was chosen)
    - Critique notes per concept (what the Critic identified)
    - Scoring data (which criteria scored highest/lowest)
    - Any human notes about production direction
    - Hybrid selections (if Synthesizer hybrids were chosen)

STEP 3: DERIVE PRODUCTION IMPLICATIONS
  - FOR EACH winning concept:
    - What visual elements were praised? (prioritize these in production)
    - What weaknesses were noted? (ensure production compensates)
    - What emotional register was rated highest? (match in voiceover/music)
    - What hook type won? (affects voice selection, pacing, visual style)

STEP 4: WRITE OUTPUT
  - Create per-microskill output file with arena context
```
