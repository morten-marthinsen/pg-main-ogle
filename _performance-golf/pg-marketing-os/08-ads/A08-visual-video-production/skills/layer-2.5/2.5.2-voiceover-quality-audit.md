# 2.5.2: Voiceover Quality Audit

## Purpose
Review every generated voiceover against its A07 script and emotional direction. Evaluate on three dimensions: Brief Fidelity (script accuracy + emotional match), Technical Compliance (duration, format), and Production Quality (audio clarity, naturalness). Make a binary PASS/FAIL determination per voiceover.

## Inputs
- All generated voiceovers from Layer 2.2
- A07 AD-COPY-FINAL/ (verbatim script text -- for accuracy verification)
- A05 Visual Direction (emotional register per concept)
- Audio Specifications from AGENT.md (format, sample rate, normalization)

## Process
```
FOR EACH generated voiceover:

DIMENSION 1: BRIEF FIDELITY (Weight: 40%)
  Does the voiceover match script and emotional direction?
  - Script accuracy: [1-10]
    Every word from A07 present? No dropped words? No added words?
  - Emotional register match: [1-10]
    Does the voice FEEL what the script intends?
    Warning script = authoritative delivery?
    Testimonial script = natural, relatable delivery?
  - Pacing match: [1-10]
    Does the pacing match the brief direction?
    Varied where emotional beats require variation?
  Minimum: 7.0 average

DIMENSION 2: TECHNICAL COMPLIANCE (Weight: 30%)
  Does the voiceover meet technical specifications?
  - Duration match: [PASS/FAIL -- within +/- 5% of target]
  - File format: [PASS/FAIL -- WAV master + MP3 delivery]
  - Sample rate: [PASS/FAIL -- 44.1kHz+]
  - Peak level: [PASS/FAIL -- normalized to -1dB]
  Minimum: ALL PASS

DIMENSION 3: PRODUCTION QUALITY (Weight: 30%)
  Is the voiceover production-ready?
  - Audio clarity (no pops, clicks, background noise): [1-10]
  - Naturalness (no robotic cadence, no uncanny valley): [1-10]
  - Pronunciation (all words pronounced correctly): [PASS/FAIL]
  Minimum: 6.0 average, AND PASS on pronunciation

OVERALL VERDICT:
  ALL three dimensions pass = PASS
  ANY dimension fails = FAIL -> regeneration queue
```
