# 3.3: Multi-Platform Rendering

## Purpose
Render each assembled ad unit across all target platform aspect ratios and specifications. Ensure proper recomposition (not simple cropping) when adapting between aspect ratios. Verify safe zones, text positions, and CTA placement per platform.

## Inputs
- 3.1 Static Ad Assembly (assembled static ads)
- 3.2 Video Ad Assembly (assembled video ads)
- 0.4 Format Strategy (platform targets per concept)
- Platform-Specific Technical Requirements from AGENT.md

## Process
```
FOR EACH assembled ad unit:

  DETERMINE required aspect ratios from A03 Format Strategy:
    Meta Feed: 1:1 (1080x1080) AND/OR 4:5 (1080x1350)
    Meta Stories/Reels: 9:16 (1080x1920)
    TikTok: 9:16 (1080x1920)
    YouTube Pre-Roll: 16:9 (1920x1080)
    YouTube Shorts: 9:16 (1080x1920)

  RENDERING RULES:
    1. DO NOT simply crop a 16:9 master to 9:16
       -> Recompose: adjust subject framing, text positions, safe zones
    2. Text overlays must be repositioned per aspect ratio
       -> 16:9 text positions do NOT work at 9:16
    3. Safe zones differ per platform
       -> TikTok: bottom 150px reserved for platform UI
       -> Meta Stories: top 250px and bottom 250px are unsafe
    4. CTA positions adjust per platform convention
       -> TikTok: CTA overlaps with "Shop Now" button area
       -> YouTube: CTA should not overlap with skip button

  IDEAL APPROACH:
    If assets were generated natively per aspect ratio at Layer 2:
      -> Use the native asset (already correct framing)
    If assets were generated at a single aspect ratio:
      -> Reframe using intelligent crop (keep subject centered)
      -> Adjust text overlay positions
      -> Verify readability at new dimensions
      -> Log that this is a reframe (not native)

  EXPORT each variant per platform:
    - Correct format (JPG/PNG for static, MP4 for video)
    - Correct resolution per platform spec table
    - Correct file size (under platform maximum)
    - Apply naming convention with platform identifier
```

## Quality Checks
- Every assembled ad has renders for all target platforms
- No simple cropping (recomposition used for aspect ratio changes)
- Text overlays repositioned per platform (not scaled from one ratio)
- Platform safe zones respected (TikTok bottom, Meta Stories top/bottom)
- CTA placement follows platform conventions
- All renders meet platform technical specifications
- Reframes are logged (vs. native renders)
