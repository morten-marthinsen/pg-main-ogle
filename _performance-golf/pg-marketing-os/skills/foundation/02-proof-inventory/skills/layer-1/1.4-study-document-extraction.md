# 1.4 Study Document Extraction

## Purpose
Extract proof elements from research documents, academic papers, clinical trials, and scientific studies. Studies provide the highest-credibility AUTHORITY and MECHANISM proof—extract with precision to preserve citation integrity.

---

## Input Requirements

From 1.1 Source Parsing:
- Parsed study document segments
- Section markers identified
- Methodology indicators flagged

---

## Study Quality Hierarchy

Not all studies are equal. Classify before extraction:

```
STUDY QUALITY TIERS:

TIER 1 (Highest):
├── Meta-analysis
├── Systematic review
├── Randomized controlled trial (RCT)
├── Double-blind placebo-controlled
└── Peer-reviewed + high-impact journal

TIER 2 (Strong):
├── Cohort study
├── Case-control study
├── University research
├── Independent lab testing
└── Peer-reviewed journal

TIER 3 (Moderate):
├── Cross-sectional study
├── Single-arm trial
├── Government data reports
├── Industry-funded but disclosed
└── Conference presentations

TIER 4 (Weak):
├── Case studies/reports
├── Observational only
├── Company-funded, undisclosed
├── Non-peer-reviewed
└── Pilot studies only
```

---

## Execution Protocol

### Step 1: Study Metadata Extraction

Capture full citation details:

```yaml
study_metadata:
  title: "[Full study title]"
  authors:
    - name: "[Author name]"
      credentials: "[PhD, MD, etc.]"
      institution: "[University/organization]"
  publication:
    journal: "[Journal name]"
    volume_issue: "[Vol X, Issue Y]"
    year: 2023
    doi: "[DOI if available]"
  study_type: "RCT | cohort | meta-analysis | ..."
  quality_tier: 1-4
  peer_reviewed: true | false
  funding_disclosed: true | false
  funding_source: "[If disclosed]"
```

### Step 2: Methodology Extraction

Methodology determines credibility:

```
METHODOLOGY ELEMENTS:

SAMPLE:
├── Sample size (n=X)
├── Population description
├── Inclusion/exclusion criteria
├── Demographic breakdown if relevant

DESIGN:
├── Study design type
├── Randomization method
├── Blinding (single, double, none)
├── Control group description
├── Duration of study

MEASUREMENT:
├── Primary outcome measures
├── Secondary outcome measures
├── Measurement instruments used
└── Statistical methods
```

**Extraction Example:**
```yaml
methodology:
  sample_size: 847
  population: "Adults with BMI > 30"
  randomized: true
  blinding: "double-blind"
  control_group: "placebo"
  duration: "12 weeks"
  primary_outcome: "weight change in kg"
```

### Step 3: Results Extraction

Extract ALL quantifiable results:

```
RESULT TYPES TO CAPTURE:

PRIMARY RESULTS:
├── Main finding statement
├── Primary outcome data
├── Statistical significance (p-value)
├── Confidence interval
└── Effect size

SECONDARY RESULTS:
├── Secondary outcome data
├── Subgroup analyses
├── Safety data
└── Adherence rates

COMPARATIVE RESULTS:
├── Treatment vs. control
├── Improvement percentages
├── Number needed to treat (NNT)
└── Relative/absolute risk reduction
```

**Extraction Example:**
```yaml
results:
  primary_finding:
    outcome: "weight loss"
    treatment_group: "4.7 kg average loss"
    control_group: "0.8 kg average loss"
    difference: "3.9 kg (95% CI: 3.2-4.6)"
    p_value: "<0.001"
    statistical_significance: true

  secondary_findings:
    - outcome: "waist circumference"
      change: "-4.2 cm"
      p_value: "<0.01"

  sample_sizes:
    treatment: 423
    control: 424
    completed: 789 (93.2% retention)
```

### Step 4: Mechanism Proof Extraction

Studies often explain HOW something works—this is MECHANISM proof:

```
MECHANISM ELEMENTS:

BIOLOGICAL MECHANISM:
├── Pathway described
├── Molecular targets
├── Biomarker changes
└── Physiological process

LOGICAL MECHANISM:
├── Cause-effect chain
├── Process steps
├── Intervening variables
└── Theoretical framework
```

**Example:**
```yaml
mechanism_proof:
  mechanism_described: true
  mechanism_type: "biological"
  pathway: "Inhibits alpha-glucosidase, reducing glucose absorption"
  supporting_data: "40% reduction in postprandial glucose spike"
```

### Step 5: Quote-Worthy Excerpts

Identify statements that can be directly quoted:

```
QUOTABLE ELEMENTS:
├── Abstract conclusions
├── Author interpretation statements
├── "We found that..." statements
├── Comparison conclusions
└── Clinical significance statements
```

**Example:**
```yaml
quotable_excerpts:
  - excerpt: "Our findings suggest that [compound] may be an effective
             adjunct therapy for weight management in adults with obesity."
    source: "Conclusion section"
    author: "lead author"
    usable_as: "expert statement"
```

### Step 6: Limitation Acknowledgment

Capture limitations for credibility calibration:

```yaml
limitations_stated:
  - "Short duration (12 weeks)"
  - "Single geographic region"
  - "Self-reported dietary adherence"

credibility_adjustment:
  base_tier: 1
  adjusted_tier: 2
  reason: "Short duration limits long-term claims"
```

## Output: Extracted Study Elements

For each study, generate multiple proof elements:

```yaml
study_extraction:
  source_id: "SRC-003"
  study_type: "RCT"
  quality_tier: 1

  extracted_elements:
    # AUTHORITY element - the study itself
    - id: "PROOF-030"
      category: "authority"
      sub_type: "study_clinical_trial"
      sub_type_id: "AUT-02"
      raw_text: "A randomized controlled trial (n=847) published in
                 Journal of Clinical Nutrition..."
      citation: "Smith et al., J Clin Nutr, 2023;45(3):112-120"
      quality_tier: 1
      peer_reviewed: true
      preliminary_scores:
        credibility: 9
        specificity: 8

    # DATA element - specific statistic
    - id: "PROOF-031"
      category: "data"
      sub_type: "data_exact_statistic"
      sub_type_id: "DAT-01"
      raw_text: "Treatment group lost an average of 4.7 kg compared
                 to 0.8 kg in placebo (p<0.001)"
      statistic: "4.7 kg vs 0.8 kg"
      p_value: "<0.001"
      citation: "Smith et al., 2023"
      preliminary_scores:
        credibility: 9
        specificity: 10

    # MECHANISM element - how it works
    - id: "PROOF-032"
      category: "mechanism"
      sub_type: "mechanism_study"
      sub_type_id: "MEC-01"
      raw_text: "The compound inhibits alpha-glucosidase, reducing
                 intestinal glucose absorption by 40%"
      mechanism_pathway: "alpha-glucosidase inhibition"
      supporting_data: "40% reduction"
      citation: "Smith et al., 2023"
      preliminary_scores:
        credibility: 9
        specificity: 8

  citation_record:
    full_citation: "Smith J, et al. Effects of [compound] on weight
                    management: A randomized controlled trial.
                    J Clin Nutr. 2023;45(3):112-120."
    doi: "10.xxxx/xxxxx"
    accessible: true

  verification_status: "verified"
```

## Quality Gates

Before study extraction complete:

- [ ] Full citation captured (authors, journal, year, volume)
- [ ] Study quality tier assigned
- [ ] Methodology details extracted (sample size, design, duration)
- [ ] All statistics captured with p-values/confidence intervals
- [ ] Mechanism explanation extracted if present
- [ ] Limitations noted
- [ ] Multiple proof elements generated per study

---

## Citation Integrity Rules

**NEVER:**
- Fabricate study details
- Misrepresent findings
- Claim peer review without verification
- Ignore stated limitations
- Extrapolate beyond study scope

**ALWAYS:**
- Preserve exact statistics
- Note confidence intervals
- Capture author caveats
- Flag unverified citations
- Distinguish correlation from causation

---

## Handoff

Pass Extracted Study Elements to:
- **1.5 Category Classification** (for verification)
- **1.6 Sub-Type Matching** (for refinement)
- **1.7 Element Scoring** (for full scoring)

Pass Citation Record to output for downstream copy use.

Flag incomplete citations for Layer 3 Discovery verification.
