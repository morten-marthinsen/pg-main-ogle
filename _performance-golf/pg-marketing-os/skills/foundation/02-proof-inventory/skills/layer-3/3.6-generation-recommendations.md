# 3.6 Generation Recommendations

## Purpose
For gaps that cannot be filled through Discovery, recommend specific proof to CREATE. This includes testimonial collection strategies, study commissioning, demonstration production, and guarantee design. Prioritize by impact/effort ratio.

---

## Input Requirements

From 3.1 Discovery Routing:
- Gaps routed to generation
- Gaps unfilled after discovery

From Discovery Results (3.2-3.5):
- Gaps still missing after discovery
- Partial fills needing augmentation

From Context:
- Client capabilities/resources
- Timeline constraints
- Budget indicators

---

## Generation Decision Matrix

```
WHEN TO GENERATE vs WHEN TO WORK WITHOUT:

CRITICAL (Must Generate)
├── Schwartz Stage 3+ without mechanism proof
├── High-ticket offer without strong social proof
├── Safety-sensitive niche without authority backing
└── No guarantee in competitive market

HIGH PRIORITY (Should Generate)
├── Weak testimonials needing specificity
├── Missing demonstration proof
├── Data gaps for lead statistics
└── No expert endorsement

MEDIUM PRIORITY (Nice to Have)
├── More testimonial variety
├── Stronger before/after visuals
├── Additional data points
└── Extended guarantee options

LOW PRIORITY (Can Work Around)
├── Minor category gaps
├── Stylistic proof preferences
└── "Nice to have" additions
```

def calculate_impact_effort_score(option, severity, schwartz_stage):
    """
    Score generation option by impact/effort ratio.
    """

    IMPACT_SCORES = {'very_high': 10, 'high': 8, 'medium': 5, 'low': 3}
    EFFORT_SCORES = {'low': 1, 'medium': 2, 'high': 4}

    impact = IMPACT_SCORES.get(option['impact'], 5)
    effort = EFFORT_SCORES.get(option['effort'], 2)

    # Base ratio
    score = impact / effort

    # Severity multiplier
    severity_mult = {'critical': 1.5, 'high': 1.25, 'medium': 1.0, 'low': 0.75}
    score *= severity_mult.get(severity, 1.0)

    # Stage adjustments
    if schwartz_stage >= 3 and option.get('method', '').startswith('mechanism'):
        score *= 1.3  # Mechanism proof more valuable at Stage 3+

    return score

### Step 2: Prioritization Algorithm

```python
def prioritize_generation_plan(recommendations, constraints):
    """
    Prioritize generation recommendations into actionable plan.
    """

    # Extract all best recommendations
    all_actions = []
    for rec in recommendations:
        if rec['best_recommendation']:
            all_actions.append({
                'gap_id': rec['gap_id'],
                'category': rec['category'],
                'severity': rec['severity'],
                'action': rec['best_recommendation']
            })

    # Sort by: severity (critical first), then score
    all_actions.sort(
        key=lambda x: (
            -severity_to_num(x['severity']),
            -x['action']['score']
        )
    )

    # Categorize into tiers
    plan = {
        'immediate': [],     # Do now
        'short_term': [],    # Do within 2 weeks
        'medium_term': [],   # Do within 1-2 months
        'long_term': []      # Do if resources allow
    }

    for action in all_actions:
        timeline = action['action'].get('timeline', '')
        effort = action['action'].get('effort', '')
        severity = action['severity']

        if severity == 'critical' or timeline == 'immediate':
            plan['immediate'].append(action)
        elif severity == 'high' or effort == 'low':
            plan['short_term'].append(action)
        elif effort == 'medium':
            plan['medium_term'].append(action)
        else:
            plan['long_term'].append(action)

    return plan
```
