# 3.4 Expert Quote Discovery

## Purpose
Find expert quotes, professional endorsements, and authoritative statements that can fill AUTHORITY gaps, strengthen mechanism credibility, and add third-party validation. Focus on credentialed experts whose statements can be ethically cited.

---

## Input Requirements

From 3.1 Discovery Routing:
- Gaps routed to expert discovery
- Mechanism/topic requiring expert validation
- Credibility needs

From Context:
- Niche/industry
- Mechanism being promoted
- Claims requiring expert backing

---

## Expert Source Hierarchy

```
EXPERT CREDIBILITY TIERS:

TIER 1 - Academic/Research Experts
├── University professors (named institution)
├── Research scientists with publications
├── Medical doctors (MDs) with specialization
├── PhDs in relevant field
└── Published authors in peer-reviewed journals

TIER 2 - Clinical/Practice Experts
├── Board-certified specialists
├── Licensed practitioners (verified)
├── Clinical researchers
├── Hospital/institution affiliated
└── Professional association leaders

TIER 3 - Industry Experts
├── Industry analysts
├── Trade publication editors
├── Professional certifications
├── Recognized thought leaders
└── Conference speakers

TIER 4 - Practitioner/Applied Experts
├── Experienced practitioners
├── Book authors (popular press)
├── Podcast hosts/media personalities
├── Coaches/consultants with credentials
└── Product formulators/creators

TIER 5 - General Authorities (Use Carefully)
├── News commentators
├── Bloggers with following
├── Social media influencers
└── Celebrity endorsements
```

### Step 1: Expert Need Analysis

```python
def analyze_expert_need(gap, mechanism):
    """
    Determine what type of expert authority is needed.
    """

    EXPERT_NEEDS = {
        'mechanism_validation': {
            'description': 'Expert confirming how the mechanism works',
            'ideal_expert': 'Researcher or scientist in relevant field',
            'quote_type': 'Explanation or confirmation of mechanism',
            'tier_required': [1, 2]
        },
        'safety_confirmation': {
            'description': 'Expert confirming safety',
            'ideal_expert': 'Medical doctor or toxicologist',
            'quote_type': 'Safety statement or endorsement',
            'tier_required': [1, 2]
        },
        'efficacy_endorsement': {
            'description': 'Expert confirming it works',
            'ideal_expert': 'Clinical practitioner or researcher',
            'quote_type': 'Endorsement or recommendation',
            'tier_required': [1, 2, 3]
        },
        'credibility_boost': {
            'description': 'General expert backing for authority',
            'ideal_expert': 'Any credentialed expert',
            'quote_type': 'General positive statement',
            'tier_required': [1, 2, 3, 4]
        },
        'problem_validation': {
            'description': 'Expert confirming the problem is real/serious',
            'ideal_expert': 'Researcher or practitioner',
            'quote_type': 'Problem statement or warning',
            'tier_required': [1, 2, 3]
        }
    }

    # Analyze gap to determine need
    gap_text = gap.get('description', '').lower()

    for need_type, info in EXPERT_NEEDS.items():
        if matches_need_type(gap_text, need_type):
            return {
                'need_type': need_type,
                'description': info['description'],
                'ideal_expert': info['ideal_expert'],
                'quote_type': info['quote_type'],
                'min_tier': max(info['tier_required']),
                'search_strategy': build_expert_search(need_type, mechanism)
            }

    return {
        'need_type': 'general_authority',
        'description': 'General expert backing',
        'min_tier': 4
    }
```

### Step 3: Expert Validation Protocol

```python
def validate_expert(expert_data):
    """
    Verify expert credentials and quote authenticity.
    """

    validation = {
        'expert_id': generate_id(),
        'is_valid': False,
        'credential_score': 0,
        'relevance_score': 0,
        'quote_strength_score': 0,
        'tier': None,
        'warnings': []
    }

    # CREDENTIAL VERIFICATION
    credential_score = 5  # Base

    # Check credential indicators
    if has_md_or_phd(expert_data):
        credential_score += 2
    if has_institutional_affiliation(expert_data):
        credential_score += 2
    if has_publications(expert_data):
        credential_score += 1
    if is_board_certified(expert_data):
        credential_score += 1

    # Deductions
    if credentials_unverifiable(expert_data):
        credential_score -= 2
        validation['warnings'].append('Credentials could not be verified')
    if potential_conflict_of_interest(expert_data):
        credential_score -= 1
        validation['warnings'].append('Potential conflict of interest')

    validation['credential_score'] = max(1, min(10, credential_score))

    # DETERMINE TIER
    validation['tier'] = assign_expert_tier(expert_data, credential_score)

    # RELEVANCE CHECK
    relevance = 5  # Base

    if expert_specialty_matches(expert_data):
        relevance += 3
    if expert_has_published_on_topic(expert_data):
        relevance += 2

    validation['relevance_score'] = max(1, min(10, relevance))

    # QUOTE STRENGTH
    quote_strength = 5  # Base

    if is_direct_endorsement(expert_data['quote']):
        quote_strength += 2
    if is_mechanism_explanation(expert_data['quote']):
        quote_strength += 2
    if is_specific_not_vague(expert_data['quote']):
        quote_strength += 1
    if is_recent_statement(expert_data):
        quote_strength += 1

    validation['quote_strength_score'] = max(1, min(10, quote_strength))

    # Overall validity
    avg_score = (validation['credential_score'] +
                 validation['relevance_score'] +
                 validation['quote_strength_score']) / 3

    validation['is_valid'] = avg_score >= 5 and validation['tier'] <= 4
    validation['overall_score'] = avg_score

    return validation


def assign_expert_tier(expert_data, credential_score):
    """
    Assign credibility tier based on credentials.
    """

    if credential_score >= 9:
        return 1  # Academic/Research
    elif credential_score >= 7:
        return 2  # Clinical/Practice
    elif credential_score >= 5:
        return 3  # Industry
    elif credential_score >= 3:
        return 4  # Practitioner
    else:
        return 5  # General
```

### Step 4: Quote Formatting

```python
def format_expert_quote(expert_data, validation):
    """
    Format expert quote for use in copy.
    """

    formatted = {
        'raw_quote': expert_data['quote'],
        'expert_attribution': None,
        'copy_versions': {},
        'citation': None
    }

    # Build attribution
    name = expert_data['name']
    credentials = expert_data.get('credentials', '')
    institution = expert_data.get('institution', '')

    # Full attribution
    attr_parts = [name]
    if credentials:
        attr_parts.append(credentials)
    if institution:
        attr_parts.append(institution)

    formatted['expert_attribution'] = {
        'full': ', '.join(attr_parts),
        'short': f"{name}, {credentials}" if credentials else name,
        'name_only': name
    }

    # Copy versions
    quote = expert_data['quote']
    full_attr = formatted['expert_attribution']['full']

    # Version 1: Quote with full attribution
    formatted['copy_versions']['full_attribution'] = (
        f'"{quote}" — {full_attr}'
    )

    # Version 2: Attribution-first
    formatted['copy_versions']['attribution_first'] = (
        f"According to {full_attr}, \"{quote}\""
    )

    # Version 3: Embedded authority
    formatted['copy_versions']['embedded'] = (
        f"{full_attr} explains: \"{quote}\""
    )

    # Version 4: For Tier 1-2 experts - authority emphasis
    if validation['tier'] <= 2:
        formatted['copy_versions']['authority_emphasis'] = (
            f"As {full_attr} puts it: \"{quote}\""
        )

    # Citation
    source = expert_data.get('source', '')
    year = expert_data.get('year', '')

    formatted['citation'] = {
        'source': source,
        'year': year,
        'full': f"{name} quoted in {source}, {year}" if source else None
    }

    return formatted
```

## Expert Stacking Strategy

```
EXPERT PROOF STACKING FOR MAXIMUM CREDIBILITY:

1. TIER 1 ACADEMIC EXPERT
   "According to Dr. [Name] at [University]..."
   → Establishes scientific/institutional credibility

2. TIER 2 CLINICAL EXPERT
   "Practitioners like Dr. [Name] confirm..."
   → Shows real-world application

3. TIER 3-4 PRACTITIONER
   "I've recommended this to hundreds of patients..."
   → Personal endorsement, social proof hybrid

NEVER lead with Tier 4-5 experts.
Always pyramid from highest to lower tier.
```

## Ethical Considerations

```
QUOTE USAGE ETHICS:

✓ DO:
- Use publicly available quotes
- Maintain full context of statement
- Include accurate attribution
- Link to original source when possible

✗ DO NOT:
- Take quotes out of context
- Combine separate quotes
- Imply endorsement of specific product if quote is general
- Use quotes from experts who have since retracted statements
- Fabricate credentials or affiliations
```
