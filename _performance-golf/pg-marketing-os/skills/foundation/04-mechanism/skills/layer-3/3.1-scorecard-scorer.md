# Skill 3.1: Scorecard Scorer

**Version:** 1.0
**Layer:** 3 (Validation & Selection)
**Type:** Aggregation
**Dependencies:** All Layer 2 outputs (2.1-2.12)
**Output:** scorecard_results.json

---

## Purpose

Aggregate all 12 dimension scores into a unified scorecard per mechanism candidate. Calculate averages, identify strengths and weaknesses, and rank candidates by overall performance.

---

## Input Schema

```yaml
input:
  layer_2_outputs:
    image_strength: from 2.1
```

## Scoring Logic

### Step 1: Collect Final Scores

```
FOR each candidate:

  FOR each dimension (2.1-2.12):
    USE the FINAL score (post-optimization if optimization was applied)
    IF optimization was applied but score didn't improve:
      USE the post-optimization score (may still be below threshold)
```

### Step 2: Calculate Aggregates

```
FOR each candidate:

  average = sum(all_12_scores) / 12
  median = middle value when sorted
  lowest_dimension = dimension with min score
  highest_dimension = dimension with max score
  dimensions_below_5 = [dim for dim in scores if score < 5]
  dimensions_above_8 = [dim for dim in scores if score > 8]
```

### Step 3: Identify Profiles

```
FOR each candidate:

  strength_cluster = top 3 scoring dimensions
  weakness_cluster = bottom 3 scoring dimensions

  DETERMINE archetype based on pattern:

  "THE EXPLAINER" — high simplicity + image + belief compatibility
  "THE PROVOCATEUR" — high virality + visceral + doomsday
  "THE SCIENTIST" — high proof + differentiation + thesis cohesion
  "THE ACTIVATOR" — high ease of use + super power + embedded benefits
  "THE BALANCED" — no dimension below 6, no standout cluster

  Archetype informs how the mechanism will be deployed in copy
```

### Step 4: Rank Candidates

```
SORT candidates by average score descending

FOR each candidate:
  viable = (average ≥ 7.0) AND (no dimension < 5)

RANK with viable status
```

---

## Quality Criteria

1. All 12 dimensions scored for all candidates (no gaps)
2. Aggregates calculated correctly
3. Profile archetype assigned to each candidate
4. Ranking is clear and includes viability assessment
