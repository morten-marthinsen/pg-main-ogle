# Skill 1.1: Mechanism Type Selector

**Version:** 1.0
**Layer:** 1 (Mechanism Ideation)
**Type:** Strategic Analysis
**Dependencies:** upstream_context.json, vault_patterns.json, teachings_framework.json
**Output:** type_candidates.json

---

## Purpose

Evaluate which mechanism TYPES best fit the diagnosed root cause, Big Idea direction, and market context. Not all types work for all situations — a scientific mechanism requires different proof than a system mechanism. This skill narrows the field before naming begins.

---

## Mechanism Types

These types classify the SOURCE OF POWER — where the mechanism derives its persuasive authority. Assign a PRIMARY type and optionally a SECONDARY type when two clear power sources are present.

```
7 MECHANISM TYPES (source of power):

1. SCIENTIFIC — Power from research, studies, biological/chemical processes
   Best when: Root cause is physiological/neurological, clinical proof available
   Examples: "Cortisol Reset Protocol", "EDTA Chelation", "Sirtuin Gene Activation"

2. TECHNICAL — Power from specialized expertise, precision, proprietary analysis
   Best when: Root cause is technical/analytical, audience respects insider knowledge
   Examples: "The X Pattern", "Weiss Rating System", "Power Plane", "Triple Coil"

3. NATURAL — Power from nature, organic processes, ancestral/traditional wisdom
   Best when: Audience distrusts pharma/synthetic, root cause is lifestyle/environmental
   Examples: "Okinawan Longevity Secret", "Ancient Seed Activation", "Caveman Diet"

4. COMPOUND — Power from a specific combination, blend, or synergistic formula
   Best when: Root cause is deficiency/imbalance, the COMBINATION is the innovation
   Examples: "The 4-3-1 Formula", "7 Forms of Magnesium", "Budwig Formula"

5. BEHAVIORAL — Power from a specific action, movement, drill, or habit change
   Best when: Root cause is wrong action/technique, audience wants something to DO
   Examples: "Simple Strike", "Counter-Slice Sequence", "6-Second Stretch", "The Turn"

6. SEQUENCE — Power from a chain reaction or cascade effect
   Best when: One trigger initiates automatic downstream effects
   Examples: "Metabolic Cascade", "The Domino Effect", "Sleep Switch"

7. SYSTEM — Power from a structured multi-component methodology
   Best when: Root cause is lack of structure/consistency, audience wants organized framework
   Examples: "3-Phase Swing System", "Ask Method", "Product Launch Formula"
```

## Selection Logic

### Step 1: Analyze Root Cause Compatibility

```
FROM root_cause_context:

MAP reframe_technique to compatible types:
  blame_removal → SCIENTIFIC, SEQUENCE (externalize cause to biological/chain reaction)
  enemy_reveal → SYSTEM, COMPOUND (structured counter-attack, specific alternative blend)
  false_solution_expose → BEHAVIORAL, COMPOUND (different action, different formula)
  scientific_discovery → SCIENTIFIC, SEQUENCE (evidence-based, cascade mechanism)
  conspiracy_reveal → NATURAL, SYSTEM (suppressed natural truth + actionable framework)
  contrarian_truth → BEHAVIORAL, NATURAL (unconventional action/approach)

MAP emphasis_level to type characteristics:
  heavy → needs STRONG explanatory power (SCIENTIFIC, SEQUENCE, SYSTEM)
  medium → needs clear differentiation (TECHNICAL, COMPOUND, BEHAVIORAL)
  light → needs memorability over depth (NATURAL, BEHAVIORAL)
```

### Step 2: Analyze Audience Compatibility

```
FROM audience_context:

ASSESS sophistication:
  Stage 3 (Solution Aware) → NATURAL, SEQUENCE (novelty/cascade wins)
  Stage 4 (Product Aware) → BEHAVIORAL, COMPOUND, SYSTEM (specificity wins)
  Stage 5 (Most Aware) → SCIENTIFIC, TECHNICAL (proof/precision wins)

ASSESS belief systems:
  Trust science → SCIENTIFIC, SEQUENCE
  Trust nature → NATURAL, COMPOUND
  Trust expertise → TECHNICAL, SYSTEM
  Trust action → BEHAVIORAL, SYSTEM
  Skeptical of all → need strongest proof regardless of type
```

### Step 3: Analyze Market Saturation

```
FROM vault_patterns.type_distribution + upstream_context.saturated_mechanisms:

IDENTIFY overused types in this niche:
  - Which types do competitors use most?
  - Which types has the audience seen too often?

IDENTIFY fresh types:
  - Which types are rare in this niche?
  - Cross-niche: which types work elsewhere but aren't used here?

BIAS toward fresh types when fit_score is comparable
```

### Step 4: Analyze Proof Feasibility

```
FOR each type:
  ASSESS what proof is needed:
    SCIENTIFIC → studies, credentials, clinical data, peer review
    TECHNICAL → expert demonstrations, precision metrics, track records
    NATURAL → tradition, longevity evidence, ancestral precedent
    COMPOUND → ingredient research, synergy studies, combination logic
    BEHAVIORAL → before/after demonstrations, biomechanics, kinesthetic proof
    SEQUENCE → cause-effect chain evidence, cascade documentation
    SYSTEM → results, case studies, framework validation

  ASSESS what proof is available:
    - From market research
    - From product/client knowledge
    - From scientific literature
    - From testimonials/case studies

  SCORE proof_feasibility (1-10):
    Can we actually PROVE this type of mechanism?
```

### Step 5: Score and Rank

```
FOR each of 7 types:
  CALCULATE fit_score:
    root_cause_alignment (weight: 0.30)
    audience_compatibility (weight: 0.25)
    market_freshness (weight: 0.20)
    proof_feasibility (weight: 0.15)
    big_idea_alignment (weight: 0.10)

  COMPILE rationale string explaining score

RANK by fit_score descending
SET primary_recommendation = highest scorer
SET secondary_recommendation = second highest
FLAG types_to_avoid = score < 4 with reasons
```

---

## Quality Criteria

1. All 7 types evaluated (none skipped)
2. Primary recommendation has clear rationale connecting root cause → type → proof
3. Types to avoid have specific reasons (not just "low score")
4. Cross-niche insight provided for innovation potential
5. No type recommended without proof feasibility ≥ 6
