# 3.5 Validation Synthesis

## Purpose
Compile all validation scores into a FINAL VALIDATION REPORT that determines whether the expressed root cause is ready for copy integration. This is the Layer 3 output—the quality-assured root cause package.

---

## Input Requirements

From Layer 3 microskills:
- 3.1 Truth Validation Report
- 3.2 Mechanism Alignment Report
- 3.3 Proof Availability Report
- 3.4 Audience Resonance Report

---

## Execution Protocol

### Step 1: Compile All Validation Scores

```
VALIDATION SCORE SUMMARY:
┌────────────────────────────────────────┐
│ 3.1 Truth Validation                   │
│     Score: [X/10]                       │
│     Status: [passed | marginal | failed]│
├────────────────────────────────────────┤
│ 3.2 Mechanism Alignment                │
│     Score: [X/10]                       │
│     Status: [strong | acceptable | weak]│
├────────────────────────────────────────┤
│ 3.3 Proof Availability                 │
│     Score: [X/10]                       │
│     Status: [strong | adequate | weak]  │
├────────────────────────────────────────┤
│ 3.4 Audience Resonance                 │
│     Score: [X/10]                       │
│     Status: [strong | adequate | weak]  │
└────────────────────────────────────────┘
```

---

### Step 2: Calculate Composite Score

Weight the scores by importance:

```
COMPOSITE SCORE CALCULATION:

Truth Validation (25%):        [X] × 0.25 = [Y]
Mechanism Alignment (25%):     [X] × 0.25 = [Y]
Proof Availability (20%):      [X] × 0.20 = [Y]
Audience Resonance (30%):      [X] × 0.30 = [Y]

COMPOSITE VALIDATION SCORE: [Sum] / 10
```

**Weight Rationale**:
- Resonance highest (30%): If audience doesn't connect, nothing else matters
- Truth & Alignment equal (25% each): Foundation of ethical, effective copy
- Proof lower (20%): Can sometimes work around proof gaps with other elements

---

### Step 3: Identify Weak Points

List any validation dimension scoring below threshold:

```
WEAK POINT ANALYSIS:
┌─────────────────┬─────────┬───────────┬──────────────────────┐
│ Dimension       │ Score   │ Threshold │ Action Required      │
├─────────────────┼─────────┼───────────┼──────────────────────┤
│ Truth           │ [X/10]  │ 6.0       │ [None | Adjust | Block] │
│ Alignment       │ [X/10]  │ 7.0       │ [None | Adjust | Block] │
│ Proof           │ [X/10]  │ 6.0       │ [None | Adjust | Block] │
│ Resonance       │ [X/10]  │ 7.0       │ [None | Adjust | Block] │
└─────────────────┴─────────┴───────────┴──────────────────────┘

CRITICAL FAILURES: [List any dimension below minimum]
ADJUSTMENT NEEDED: [List dimensions needing work]
```

---

### Step 4: Make Go/No-Go Decision

```
VALIDATION DECISION MATRIX:

APPROVED (Green Light):
├── Composite score: 7.5+
├── No dimension below minimum threshold
└── All critical requirements met
→ Proceed to Layer 4 Output

CONDITIONALLY APPROVED (Yellow Light):
├── Composite score: 6.0-7.4
├── No more than 1 dimension below threshold
├── Clear path to fixing weak point
└── Truth score above 6.0
→ Proceed with documented adjustments

REVISION REQUIRED (Orange Light):
├── Composite score: 5.0-5.9
├── 2+ dimensions below threshold
├── Fixable with expression adjustment
└── Truth score above 5.0
→ Return to Layer 2 for expression revision

BLOCKED (Red Light):
├── Composite score below 5.0
├── OR Truth score below 5.0
├── OR Critical alignment failure
└── OR Fundamental resonance failure
→ Return to Layer 1 for re-derivation

DECISION: [APPROVED | CONDITIONAL | REVISION | BLOCKED]
```

## Quality Gates

Before passing to Layer 4:

- [ ] Composite score is 6.0+ (absolute minimum)
- [ ] Truth validation passed (score 6.0+)
- [ ] Decision is APPROVED or CONDITIONAL
- [ ] Any conditional requirements documented
- [ ] Full copy block is complete and polished
