# 4.6 Copy Integration Guide

## Purpose
Provide specific guidance for integrating promises into copy structure. This microskill bridges promise development with actual copywriting—ensuring the copywriter knows exactly how, where, and when to deploy each promise element.

---

## Input Requirements

From 4.1-4.5:
- Primary promise with variations
- Supporting promises with variations
- Usage matrix
- Proof pairings

From 08-structure (if available):
- Copy outline
- Section requirements
- CBP segment assignments

From Context:
- Copy format (VSL, sales letter, email sequence)
- Total length target
- Platform requirements

## Integration Framework

```yaml
integration_principles:

  promise_density:
    description: "How often promises appear per section"
    rule: "At least one promise element per 200-300 words"
    variation: "Increase density in opening and close"

  promise_progression:
    description: "How promises evolve through copy"
    pattern:
      - opening: "Hook promise (intrigue)"
      - thesis: "Full promise (commitment)"
      - body: "Supporting promises (expansion)"
      - proof: "Promise + proof (validation)"
      - offer: "Promise tied to product (connection)"
      - close: "Promise callback (resolution)"

  variation_rotation:
    description: "Don't repeat exact wording"
    rule: "Each promise mention uses different variation"
    exception: "Exact repetition only for deliberate callback/anchor"

  proof_proximity:
    description: "Proof should follow promise quickly"
    rule: "Proof within 50-100 words of claim"
    exception: "Thesis can stand alone before mechanism section"
```

## Execution Protocol

### Step 1: Generate Section-Specific Instructions

```python
def generate_integration_instructions(
    promises,
    usage_matrix,
    proof_pairings,
    copy_format,
    section_outline
):
    """
    Generate specific integration instructions per section.
    """
    instructions = {}

    format_guide = get_format_guide(copy_format)

    for section in section_outline:
        section_name = section['name']
        section_config = format_guide['sections'].get(section_name, {})

        # Get assigned promise for this section
        position_match = map_section_to_position(section_name)
        assigned = usage_matrix.get(position_match, {})

        if not assigned:
            instructions[section_name] = {
                'status': 'NO_PROMISE_ASSIGNED',
                'recommendation': 'Use supporting promise or omit'
            }
            continue

        promise = get_promise_by_id(assigned['primary_choice']['promise_id'], promises)
        variation = assigned['primary_choice'].get('variation', {})
        proof = proof_pairings.get(promise['promise_id'], {}).get('position_proof', {}).get(position_match)

        instructions[section_name] = {
            'promise_to_use': variation.get('text', promise['statement']),
            'variation_type': variation.get('type', 'statement'),
            'proof_to_deploy': proof,
            'word_count_target': section_config.get('word_count', 'varies'),
            'integration_format': section_config.get('format'),
            'what_comes_before': section.get('previous_section'),
            'what_comes_after': section.get('next_section'),
            'example': generate_integration_example(
                promise, variation, proof, section_config
            ),
            'warnings': generate_integration_warnings(promise, section_name),
            'alternatives': assigned.get('alternatives', [])
        }

    return instructions


def generate_integration_example(promise, variation, proof, config):
    """
    Generate a specific integration example.
    """
    promise_text = variation.get('text', promise['statement'])
    proof_text = proof.get('deployment', '[PROOF HERE]') if proof else '[PROOF HERE]'
    format_style = config.get('format', 'statement')

    examples = {
        'hook': f"What if {promise_text.lower()}? {proof_text}",
        'thesis': f"Here's the truth: {promise_text}",
        'benefit_bullet': f"✓ {promise_text}",
        'testimonial_setup': f"[NAME] wanted to {extract_desire(promise_text)}. Here's what happened...",
        'future_pace': f"Imagine {promise_text.replace('you can', 'yourself').replace('You can', 'yourself')}...",
        'close': f"This is your chance to {promise_text.lower()}. Click below."
    }

    return examples.get(format_style, f"[{format_style.upper()}]: {promise_text}")


def generate_integration_warnings(promise, section):
    """
    Generate warnings for integration challenges.
    """
    warnings = []

    # Check compliance requirements
    if promise.get('requires_disclaimer'):
        warnings.append({
            'type': 'COMPLIANCE',
            'warning': f"Disclaimer required: {promise.get('disclaimer_text')}",
            'action': 'Include disclaimer near this claim'
        })

    # Check proof requirements
    if promise.get('proof_strength', 5) < 7 and section in ['lead', 'thesis']:
        warnings.append({
            'type': 'PROOF_STRENGTH',
            'warning': 'Promise has moderate proof strength in high-visibility section',
            'action': 'Consider using softer language or stronger proof'
        })

    # Check specificity for platform
    if section in ['ad_creative', 'email_subject'] and len(promise['statement'].split()) > 15:
        warnings.append({
            'type': 'LENGTH',
            'warning': 'Promise too long for this format',
            'action': 'Use shorter variation'
        })

    return warnings
```

### Step 2: Generate Transition Language

```python
def generate_transitions(section_instructions, copy_format):
    """
    Generate transition language between promise mentions.
    """
    transitions = {}

    sections = list(section_instructions.keys())

    for i, section in enumerate(sections[:-1]):
        next_section = sections[i + 1]

        current_promise = section_instructions[section].get('promise_to_use')
        next_promise = section_instructions[next_section].get('promise_to_use')

        if current_promise and next_promise:
            if promises_related(current_promise, next_promise):
                # Reinforcing transition
                transitions[f"{section}_to_{next_section}"] = {
                    'type': 'reinforcement',
                    'options': [
                        f"And that's not all...",
                        f"But here's where it gets even better...",
                        f"Which brings us to..."
                    ]
                }
            else:
                # Bridging transition
                transitions[f"{section}_to_{next_section}"] = {
                    'type': 'bridge',
                    'options': [
                        f"Now, you might be wondering...",
                        f"Here's what this means for you...",
                        f"Let me show you how..."
                    ]
                }

    return transitions
```

### Step 3: Compile Integration Checklist

```python
def compile_integration_checklist(instructions, promises, copy_format):
    """
    Create checklist for copywriter to verify integration.
    """
    checklist = {
        'pre_writing': [],
        'during_writing': [],
        'post_writing': []
    }

    # Pre-writing checks
    checklist['pre_writing'] = [
        "Confirm primary promise and variations available",
        "Confirm proof pairings for each position",
        "Review compliance requirements",
        "Review word count targets per section"
    ]

    # During-writing checks
    for section, inst in instructions.items():
        checklist['during_writing'].append({
            'section': section,
            'check': f"Use '{inst.get('variation_type', 'statement')}' variation of promise",
            'reminder': inst.get('integration_format')
        })

    # Post-writing checks
    checklist['post_writing'] = [
        "Verify no exact promise repetition (use variations)",
        "Verify proof follows each major claim within 100 words",
        "Verify disclaimers present where required",
        "Verify promise progression (hook → thesis → support → close)",
        "Verify callback to opening promise in close"
    ]

    return checklist
```
