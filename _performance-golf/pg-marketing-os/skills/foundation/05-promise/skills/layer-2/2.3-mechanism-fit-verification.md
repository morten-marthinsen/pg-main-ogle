# 2.3 Mechanism Fit Verification

## Purpose
Verify that the mechanism can actually deliver what the promise claims. A promise without mechanism backing is just hype. This microskill ensures every promise has a credible "how" behind it.

---

## Input Requirements

From 04-mechanism:
- mechanism_name
- mechanism_type (process, system, protocol, compound, principle)
- mechanism_capabilities (what it actually does)
- villain_defeat_action
- E5_scorecard

From 2.1 Proof Ceiling Application:
- Ceiling-calibrated promises

From 2.2 Schwartz Stage Calibration:
- Stage-calibrated promises

---

## Mechanism-Promise Alignment Framework

```yaml
alignment_types:

  direct_alignment:
    description: "Mechanism directly causes the promised result"
    strength: "strongest"
    example:
      mechanism: "Gut microbiome rebalancing"
      promise: "Reduce cravings"
      connection: "Gut bacteria directly influence hunger signals"
    requirement: "Must have proof of direct causation"

  indirect_alignment:
    description: "Mechanism enables conditions for promised result"
    strength: "moderate"
    example:
      mechanism: "Gut microbiome rebalancing"
      promise: "Lose weight"
      connection: "Reduced cravings → less eating → weight loss"
    requirement: "Must explain the chain of causation"

  parallel_alignment:
    description: "Mechanism addresses root cause that creates multiple benefits"
    strength: "moderate-strong"
    example:
      mechanism: "Gut microbiome rebalancing"
      promise: "More energy AND weight loss"
      connection: "Root cause (gut imbalance) causes both problems"
    requirement: "Root cause link must be established"

  weak_alignment:
    description: "Connection exists but is stretched"
    strength: "weak"
    example:
      mechanism: "Gut microbiome rebalancing"
      promise: "Better relationships"
      connection: "Weak—too many steps between mechanism and promise"
    action: "Downgrade promise or don't make claim"

  no_alignment:
    description: "Mechanism cannot credibly deliver this promise"
    strength: "none"
    action: "Remove promise from candidate pool"
```

## Execution Protocol

### Step 1: Map Mechanism Capabilities

```python
def map_mechanism_capabilities(mechanism_data):
    """
    Extract what the mechanism can actually do.
    """
    capabilities = {
        'direct_effects': [],
        'secondary_effects': [],
        'conditions_created': [],
        'villain_defeats': []
    }

    # From mechanism E5 scorecard
    if mechanism_data.get('e5_scorecard'):
        scorecard = mechanism_data['e5_scorecard']

        # Extract from "What It Does" section
        capabilities['direct_effects'] = scorecard.get('primary_actions', [])

        # Extract from "How It Works" section
        capabilities['secondary_effects'] = scorecard.get('downstream_effects', [])

    # From villain defeat action
    if mechanism_data.get('villain_defeat_action'):
        capabilities['villain_defeats'].append({
            'villain': mechanism_data.get('villain'),
            'action': mechanism_data['villain_defeat_action'],
            'result': mechanism_data.get('villain_defeat_result')
        })

    # From mechanism type
    mech_type = mechanism_data.get('mechanism_type')
    if mech_type == 'process':
        capabilities['conditions_created'].append('step-by-step pathway')
    elif mech_type == 'compound':
        capabilities['conditions_created'].append('ingredient synergy')
    elif mech_type == 'protocol':
        capabilities['conditions_created'].append('systematic approach')

    return capabilities
```

### Step 2: Verify Promise-Mechanism Fit

```python
def verify_mechanism_fit(promise, capabilities, mechanism_data):
    """
    Check if mechanism can deliver the promise.
    """
    promise_core = extract_core_promise(promise['statement'])

    fit_assessment = {
        'promise_id': promise['id'],
        'promise': promise['statement'],
        'alignment_type': None,
        'alignment_strength': 0,
        'connection_path': [],
        'gaps': [],
        'recommendation': None
    }

    # Check direct alignment
    for effect in capabilities['direct_effects']:
        if promise_matches_effect(promise_core, effect):
            fit_assessment['alignment_type'] = 'direct'
            fit_assessment['alignment_strength'] = 10
            fit_assessment['connection_path'] = [
                f"Mechanism ({mechanism_data['mechanism_name']})",
                f"→ Directly causes: {effect}",
                f"→ Delivers: {promise_core}"
            ]
            return fit_assessment

    # Check indirect alignment
    for effect in capabilities['secondary_effects']:
        if promise_matches_effect(promise_core, effect):
            fit_assessment['alignment_type'] = 'indirect'
            fit_assessment['alignment_strength'] = 7
            fit_assessment['connection_path'] = [
                f"Mechanism ({mechanism_data['mechanism_name']})",
                f"→ Creates condition: {capabilities['direct_effects'][0] if capabilities['direct_effects'] else 'primary action'}",
                f"→ Which leads to: {effect}",
                f"→ Delivers: {promise_core}"
            ]
            return fit_assessment

    # Check villain defeat alignment
    for defeat in capabilities['villain_defeats']:
        if promise_connects_to_villain_defeat(promise_core, defeat):
            fit_assessment['alignment_type'] = 'parallel'
            fit_assessment['alignment_strength'] = 8
            fit_assessment['connection_path'] = [
                f"Mechanism defeats: {defeat['villain']}",
                f"→ Via: {defeat['action']}",
                f"→ Enables: {promise_core}"
            ]
            return fit_assessment

    # No clear alignment found
    fit_assessment['alignment_type'] = 'weak_or_none'
    fit_assessment['alignment_strength'] = 2
    fit_assessment['gaps'].append(f"No clear path from mechanism to '{promise_core}'")
    fit_assessment['recommendation'] = 'review_or_remove'

    return fit_assessment


def promise_matches_effect(promise_core, effect):
    """
    Check if promise aligns with mechanism effect.
    """
    # Semantic matching (simplified)
    promise_keywords = extract_keywords(promise_core)
    effect_keywords = extract_keywords(effect)

    overlap = len(set(promise_keywords) & set(effect_keywords))
    return overlap >= 2 or semantic_similarity(promise_core, effect) > 0.7


def promise_connects_to_villain_defeat(promise_core, defeat):
    """
    Check if promise is delivered by defeating the villain.
    """
    # If defeating villain removes problem, and promise is relief from that problem
    villain_problems = get_problems_caused_by(defeat['villain'])
    promise_relief = extract_relief_target(promise_core)

    return promise_relief in villain_problems
```

### Step 3: Generate Connection Language

```python
def generate_connection_language(fit_assessment, mechanism_data):
    """
    Generate copy-ready language explaining how mechanism delivers promise.
    """
    alignment = fit_assessment['alignment_type']
    path = fit_assessment['connection_path']
    mechanism_name = mechanism_data['mechanism_name']

    templates = {
        'direct': [
            f"The {mechanism_name} directly {path[-1].split(': ')[-1]}",
            f"By {path[1].split(': ')[-1]}, the {mechanism_name} {path[-1].split(': ')[-1]}",
            f"Here's how the {mechanism_name} works: {path[1].split(': ')[-1]} — which means {path[-1].split(': ')[-1]}"
        ],
        'indirect': [
            f"When the {mechanism_name} {path[1].split(': ')[-1]}, it creates the conditions for {path[-1].split(': ')[-1]}",
            f"The {mechanism_name} triggers a cascade: first {path[1].split(': ')[-1]}, then {path[2].split(': ')[-1]}, and finally {path[-1].split(': ')[-1]}"
        ],
        'parallel': [
            f"By defeating {path[0].split(': ')[-1]}, the {mechanism_name} removes the barrier to {path[-1].split(': ')[-1]}",
            f"Once the {mechanism_name} {path[1].split(': ')[-1]}, {path[-1].split(': ')[-1]} becomes possible"
        ]
    }

    return {
        'recommended': templates.get(alignment, ["Connection unclear"])[0],
        'alternatives': templates.get(alignment, [])[1:],
        'copy_integration': f"Use when transitioning from mechanism to promise"
    }
```

### Step 4: Flag Mismatches

```python
def flag_mechanism_mismatches(fit_assessments):
    """
    Identify promises that don't fit the mechanism.
    """
    mismatches = []

    for assessment in fit_assessments:
        if assessment['alignment_strength'] < 5:
            mismatches.append({
                'promise_id': assessment['promise_id'],
                'promise': assessment['promise'],
                'issue': assessment['gaps'],
                'recommendation': assessment['recommendation'],
                'action_options': [
                    'Remove from candidate pool',
                    'Reframe promise to match mechanism capability',
                    'Add mechanism component to support this promise',
                    'Make promise secondary/implicit rather than primary'
                ]
            })

    return mismatches
```
