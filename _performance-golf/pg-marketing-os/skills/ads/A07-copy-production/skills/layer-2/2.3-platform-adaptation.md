# 2.3: Platform Adaptation

## Purpose
For each concept, rewrite for every assigned platform from A03. Platform adaptations are NOT reformats -- they are substantive rewrites. Each adaptation must demonstrate at least 3 substantive differences from the base: (a) different hook approach suited to platform, (b) different tone register, (c) different CTA format. Adaptations must also adjust word count for platform-specific lengths.

## Inputs
- Base copy scripts (from Layer 1: 1.2, 1.3, 1.4)
- Platform constraint matrix (from 0.5-format-strategy-loader.md)
- Platform assignments per concept (from 0.5)
- Persona voice specimens (for tone calibration)
- Word count limits per platform-length

## Process
```
FOR EACH concept, FOR EACH assigned platform from A03 (excluding primary platform):

STEP 1: IDENTIFY the adaptation requirements.
  - Source: base copy (primary platform, primary length)
  - Target: [platform] at [length]
  - Platform constraints: sound-on/off, aspect ratio, CTA options, caption limits

STEP 2: PLAN the 3+ substantive differences.

  DIFFERENCE 1: HOOK APPROACH
    - TikTok: Pattern interrupt in frame 1. Raw, immediate. "POV:" or direct address.
    - YouTube: Must deliver value before 5s skip. More structured.
    - Meta: Sound-off capable. Text overlay carries the hook. Visual-first.
    - Google: Search intent aligned. Keyword-conscious.

  DIFFERENCE 2: TONE REGISTER
    - TikTok: Conversational, peer-to-peer, UGC-native. Imperfect is better.
    - YouTube: More polished, authoritative. Longer attention span assumed.
    - Meta: Visual-forward. Copy is concise. Caption + text overlay economy.
    - Google: Professional, benefit-led. Shorter format.

  DIFFERENCE 3: CTA FORMAT
    - TikTok: "Link in bio" / "Comment [keyword]" / swipe-up gestures
    - YouTube: "Click the link below" / "Subscribe for more" / end screen
    - Meta: CTA button ("Shop Now", "Learn More", "Get Offer") + text
    - Google: Headline CTA, description CTA, sitelink CTAs

  DIFFERENCE 4: LENGTH ADJUSTMENT (if applicable)
    - 60s base -> 15s TikTok: REWRITE, not cut. Single most powerful element.
    - 30s base -> 2min YouTube: EXPAND with additional proof, deeper mechanism.

  DIFFERENCE 5: SOUND-ON vs SOUND-OFF
    - Meta: 85% sound-off. Text overlays must carry full message.
    - TikTok: Sound-on dominant. Music/voice is primary carrier.
    - YouTube: Sound-on assumed. Voiceover can carry.

STEP 3: REWRITE the script for the target platform.
  - This is a FULL REWRITE, not a reformat
  - Core angle/message stays the same
  - Execution is platform-native
  - Word count adjusted for target length
  - AV format with platform-appropriate visual column

STEP 4: VERIFY 3+ substantive differences.
  - List each difference explicitly
  - "Changed aspect ratio" is NOT a substantive difference
  - "Rewrote hook from informational to pattern-interrupt POV" IS substantive

STEP 5: VERIFY word count for target platform-length.
```
