# 2.5.2: Hook-Body Coherence Check

## Purpose
For every hook-body combination in the variant set, verify logical coherence. Does the hook promise something the body delivers? Does the setup module flow naturally from the hook? Score each pairing on a 3-point scale. LOW-rated pairings are flagged for revision or exclusion from the variant matrix.

## Inputs
- All hook swap variants (from 2.1)
- Base copy scripts (from Layer 1)
- Variant inventory (from 2.4)

## Process
```
FOR EACH concept:

STEP 1: IDENTIFY all hook-body combinations.
  - Base hook + base body (should always be HIGH)
  - Each hook swap + base body
  - Each hook swap + each CTA variant body (if body changes with CTA)
  - Total combinations = (1 base hook + N swap hooks) x body versions

STEP 2: For EACH combination, EVALUATE coherence on 3 dimensions:

  DIMENSION A: PROMISE DELIVERY
    - What does the hook promise (explicitly or implicitly)?
    - Does the body deliver on that promise?
    - Example FAIL: Hook says "3 foods destroying your gut" but body
      discusses energy levels -- PROMISE MISMATCH
    - Example PASS: Hook says "3 foods destroying your gut" and body
      names 3 gut-damaging foods

  DIMENSION B: NARRATIVE FLOW
    - Does the setup module flow naturally from the hook?
    - Is there a jarring topic shift between hook and setup?
    - Does the reader feel like they're reading one continuous piece?

  DIMENSION C: TONE ALIGNMENT
    - Does the hook's tone match the body's tone?
    - Casual hook + formal body = TONE MISMATCH
    - Urgent hook + relaxed body = TONE MISMATCH

STEP 3: RATE each combination:
  - HIGH: All 3 dimensions pass. Seamless connection.
  - MEDIUM: 1 minor issue. Acceptable but flagged for human review.
  - LOW: Promise mismatch, narrative break, or tone clash.
    Must be REVISED or EXCLUDED.

STEP 4: For LOW-rated combinations, SPECIFY:
  - Which dimension failed
  - What the specific mismatch is
  - Recommended fix (revise hook, revise body transition, or exclude)

STEP 5: PRODUCE the Hook-Body Coherence Matrix.
```
