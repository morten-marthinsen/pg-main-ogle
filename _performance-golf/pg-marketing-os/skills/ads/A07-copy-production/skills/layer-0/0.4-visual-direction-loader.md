# 0.4: Visual Direction Loader

## Purpose
Load A05 VISUAL-DIRECTION-PACKAGE.md. Extract visual treatment type per concept (Talking Head, B-Roll+VO, Text-on-Screen, Screen Recording, Mixed), shot list summaries, text overlay requirements, and visual cues that copy must reference or complement.

## Inputs
- A05 VISUAL-DIRECTION-PACKAGE.md (from Visual Direction skill output)

## Process
```
STEP 1: LOCATE A05 Visual Direction Package.
  - Expected path: [project]/A05-visual-direction/VISUAL-DIRECTION-PACKAGE.md
  - IF file does not exist: HALT -- cannot write AV format without visual column

STEP 2: READ VISUAL-DIRECTION-PACKAGE.md completely.

STEP 3: EXTRACT per concept:
  a. Visual treatment type (Talking Head, B-Roll+VO, Text-on-Screen,
     Screen Recording, Mixed)
  b. Shot list summary (key shots planned)
  c. Text overlay requirements (what text appears on screen)
  d. Visual cues the copy must reference or complement
  e. Visual style notes (lighting, color, mood, pace)
  f. What the visual SHOWS (so copy doesn't duplicate it)

STEP 4: IDENTIFY copy-visual complementarity requirements.
  - For each concept, note what the VISUAL column should show
    vs. what the AUDIO column should say
  - The visual and audio should COMPLEMENT, not duplicate

STEP 5: WRITE extracted data to output file.
```
