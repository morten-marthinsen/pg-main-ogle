# 3.4: Variant Matrix Calculator

## Purpose
Calculate total variant count per concept: hooks x bodies x CTAs x proof options. Aggregate across all concepts. Produce campaign-level variant matrix. Verify minimum variant count meets AD-ENGINE-CLAUDE.md thresholds (30+ target).

## Inputs
- Hook swap groups (from 3.1)
- CTA swap groups (from 3.2)
- Proof swap groups (from 3.3)

## Process
For each concept: multiply swap group sizes (hooks x CTAs x proofs). Aggregate across campaign. Verify total >= 30 variants. Calculate testing permutations.

## Output File
- **Path:** `layer-3-outputs/3.4-variant-matrix-calculator.md`
- **Minimum size:** 3KB

## Dependencies
- 3.1, 3.2, 3.3 all complete
