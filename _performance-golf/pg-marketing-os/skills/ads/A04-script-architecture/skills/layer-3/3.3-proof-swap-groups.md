# 3.3: Proof Swap Groups

## Purpose
For each ad concept, identify alternative proof elements. Design 2-3 proof swap options: (a) testimonial proof, (b) data/study proof, (c) demonstration proof. Verify each proof option works with same [MECHANISM] exit and [CTA] entry.

## Inputs
- Proof module briefs (from 2.5)
- Proof Inventory (Skill 02 if available)

## Process
Design alternative proof modules using different proof types. Test compatibility with mechanism exit and CTA entry transitions. Ensure word count budgets match.
