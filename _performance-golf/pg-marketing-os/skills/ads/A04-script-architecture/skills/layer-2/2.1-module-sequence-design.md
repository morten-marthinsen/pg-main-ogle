# 2.1: Module Sequence Design

## Purpose
For each ad concept, map the framework's structure to the 5 standard modules. Determine which modules appear at this length. Define module sequence order. Specify word count budget per module (MUST sum to target length).

## Inputs
- Framework Assignments (from 1.3)
- Script Structures (from 0.4)
- Format Strategy (from 0.3 - for target lengths)

## Process
Map each assigned framework to module presence pattern based on ad length. Calculate exact word count budgets per module that sum to target length. Verify totals match physics-based word count table.
