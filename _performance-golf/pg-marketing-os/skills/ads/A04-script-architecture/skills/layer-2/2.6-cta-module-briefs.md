# 2.6: CTA Module Briefs

## Purpose
Design the [CTA] module content brief for each ad concept. Specify 3 CTA variants per concept: (a) urgency CTA, (b) risk reversal CTA, (c) low-friction CTA. Each CTA must be specific to the offer and funnel stage. Match CTA style to platform norms. Word count budget per CTA.

## Inputs
- Module Sequence Design (from 2.1)
- Offer Package (from 0.1)
- Platform constraints (from 0.3)

## Process
For each concept, design 3 distinct CTA approaches with different emotional appeals. Ensure platform compatibility (e.g., TikTok "link in bio" vs Meta "Shop Now" button). Verify word budgets.

## Output File
- **Path:** `layer-2-outputs/2.6-cta-module-briefs.md`
- **Minimum size:** 5KB

## Dependencies
- 2.1 complete
- Offer from 0.1 loaded
