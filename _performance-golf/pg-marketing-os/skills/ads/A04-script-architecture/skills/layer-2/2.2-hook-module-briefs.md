# 2.2: Hook Module Briefs

## Purpose
Design the [HOOK] module content brief for each ad concept. Specify: the selected hook text (from A02), how it opens (text overlay, spoken, visual action), the transition point to [SETUP] (clean seam design).

## Inputs
- Module Sequence Design (from 2.1)
- Hook-Angle Matrix (from 0.2)

## Process
For each concept, extract hook text from A02, specify visual opening method, design audio delivery, create clean seam transition to Setup module.

## Output File
- **Path:** `layer-2-outputs/2.2-hook-module-briefs.md`
- **Minimum size:** 5KB

## Dependencies
- 2.1 (Module Sequence Design) complete
