# 0.1: Upstream Strategic Loader

## Purpose
Load all strategic foundation outputs from CopywritingEngine Skills 03-07. Extract the core strategic elements that will constrain and guide script architecture: big idea statement, mechanism name and explanation, root cause framing, calibrated promise, offer stack summary, and guarantee type.

## Inputs
- Campaign Brief (Skill 09 output)
- Root Cause Package (Skill 03 output)
- Mechanism Package (Skill 04 output)
- Promise Package (Skill 05 output)
- Big Idea Package (Skill 06 output)
- Offer Package (Skill 07 output)

## Process

### Step 1: Load Campaign Brief
```
LOCATE: [project]/09-campaign-brief/CAMPAIGN-BRIEF-PACKAGE.json
READ entire file
EXTRACT:
  - campaign_overview
  - target_audience
  - awareness_level
  - tone_direction
  - platform_priorities
```

### Step 2: Load Root Cause
```
LOCATE: [project]/03-root-cause/root-cause-package.yaml
READ entire file
EXTRACT:
  - surface_problem (what they think)
  - real_root_cause (what it really is)
  - why_nothing_worked (failure narrative)
  - reframe_technique used
  - emphasis_level
```

### Step 3: Load Mechanism
```
LOCATE: [project]/04-mechanism/mechanism-package.json
READ entire file
EXTRACT:
  - mechanism_name (branded name)
  - mechanism_explanation (plain language)
  - metaphor_anchor
  - why_unique (differentiation)
  - simplicity_score
```

### Step 4: Load Promise
```
LOCATE: [project]/05-promise/promise-output.json
READ entire file
EXTRACT:
  - primary_promise (calibrated claim)
  - promise_type
  - believability_calibration
  - proof_ceiling constraints
```

### Step 5: Load Big Idea
```
LOCATE: [project]/06-big-idea/big-idea-output.json
READ entire file
EXTRACT:
  - big_idea_statement
  - concept
  - angle
  - emotional_core
  - big_idea_type
```

### Step 6: Load Offer
```
LOCATE: [project]/07-offer/offer-package.json
READ entire file
EXTRACT:
  - core_offer (what they get)
  - price_point (if public)
  - guarantee_type (branded guarantee name)
  - bonus_count
  - value_stack_total
```

### Step 7: Normalize and Compile
```
CREATE strategic_context object with all extracted fields
VERIFY all required packages were loaded
FLAG any missing elements
```

## Output Schema
```yaml
strategic_context:
  campaign_brief:
    audience: string
    awareness_level: string
    tone: string
  root_cause:
    surface_problem: string
    real_root_cause: string
    failure_narrative: string
    reframe_technique: string
  mechanism:
    name: string
    explanation: string
    metaphor_anchor: string
    differentiation: string
  promise:
    primary_promise: string
    believability_level: string
  big_idea:
    statement: string
    concept: string
    angle: string
    emotional_core: string
  offer:
    core_offer: string
    guarantee_name: string
    bonus_count: integer
  loading_status:
    all_packages_loaded: boolean
    missing_packages: [list or empty]
    warnings: [list or empty]
```

## Output File
- **Path:** `layer-0-outputs/0.1-upstream-strategic-loader.md`
- **Minimum size:** 1KB

## Quality Checks
1. All 6 strategic packages (Campaign Brief, Root Cause, Mechanism, Promise, Big Idea, Offer) successfully loaded
2. No critical fields are null or empty
3. Mechanism name is present (required for 60+ second scripts)
4. Root cause and big idea are logically compatible
5. Promise is calibrated (not over-promising beyond believability)

## Dependencies
- Skills 03, 04, 05, 06, 07, 09 must have been executed and produced output packages
- All package files must be accessible at standard output paths
