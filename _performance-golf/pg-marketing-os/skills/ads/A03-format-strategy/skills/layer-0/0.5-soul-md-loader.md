# 0.5: Soul.md Loader

## Purpose

Load the project's Soul.md file if it exists. Soul.md contains voice register, emotional range, energy signature, anti-voice patterns, and pacing signature that constrain creative treatment selection in Layer 2. If Soul.md doesn't exist, the skill proceeds with a warning (Soul.md is optional for format strategy but recommended).

## Inputs

- `[project]/SOUL.md` — project-level taste constraint document (OPTIONAL)

## Process

1. **CHECK for Soul.md existence:**
   - Path: `[project]/SOUL.md`
   - If exists: proceed to load
   - If NOT exists: WARN "No Soul.md found. Creative treatment selection will lack taste constraints." and PROCEED

2. **IF Soul.md EXISTS, READ completely:**
   - Full read required
   - Do NOT summarize

3. **EXTRACT voice register:**
   - Voice tone descriptor (conversational, authoritative, educational, urgent, etc.)
   - Voice register constraints (formal/informal, technical/accessible, etc.)
   - This affects creative treatment selection (polished vs UGC, talking head style, etc.)

4. **EXTRACT energy signature:**
   - Energy level (high-energy, moderate, calm, intense)
   - Pacing preference (fast-cut, contemplative, steady)
   - This affects length and visual treatment recommendations

5. **EXTRACT emotional range:**
   - Emotional ceiling (maximum intensity allowed)
   - Emotional floor (minimum engagement required)
   - Emotional palette (which emotions are on-brand)
   - This affects creative treatment and platform selection

6. **EXTRACT anti-voice:**
   - Voice patterns to AVOID
   - Aesthetic patterns to AVOID
   - Treatments that violate brand voice
   - This creates negative constraints for Layer 2 treatment assignment

7. **EXTRACT pacing signature:**
   - Sentence length preferences
   - Visual cut rate (for video)
   - Information density preferences
   - This affects format recommendations (fast-paced viral vs story narrative)

8. **HOLD in context for Layer 2 consumption:**
   - Microskill 2.3 (creative treatment assignment) will reference Soul.md
   - If polished production violates Soul.md voice → UGC must be prioritized
   - If fast-paced energy violates Soul.md pacing → long-form narrative prioritized


```markdown
# Soul.md Loading Status

## Warning
No Soul.md found at [project]/SOUL.md.

Creative treatment selection (microskill 2.3) will lack taste constraints. Format decisions will be data-driven only, without brand voice considerations.

RECOMMENDATION: If brand voice/tone is important to this campaign, create Soul.md before proceeding to Layer 2.

## Soul.md Status
- Soul.md exists: NO
- Proceeding without taste constraints: YES
```
