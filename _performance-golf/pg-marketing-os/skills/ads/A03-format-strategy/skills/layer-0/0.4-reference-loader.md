# 0.4: Reference Document Loader

## Purpose

Load the two critical reference documents that contain all script frameworks, word count enforcement rules, visual treatment types, UGC templates, and platform-specific hook considerations. These references are the authoritative source for Layer 2 format assignment decisions.

## Inputs

- `./References/AD-SCRIPT-STRUCTURES.md` — 8 frameworks, length structures, visual treatments
- `./References/AD-HOOK-TAXONOMY.md` — 32 hook types, platform-specific considerations

## Process

### Part A: Load AD-SCRIPT-STRUCTURES.md

1. **READ complete AD-SCRIPT-STRUCTURES.md**
   - Full read required — contains authoritative framework definitions
   - Do NOT summarize

2. **EXTRACT Framework Selection Matrix:**
   - 8 frameworks: PAS, AIDA, BAB, Hook-Body-CTA, Story Narrative, Edutainment, UGC-DR, Fast-Paced Viral
   - For each framework: best length, best vertical, best platform, best awareness level
   - This matrix drives microskill 2.4 (framework recommendation)

3. **EXTRACT word count enforcement table (BINDING):**
   ```
   6 seconds = 15 words max (audio) / 3 lines max (visual)
   15 seconds = 40 words max / 6 lines max
   30 seconds = 75 words max / 10 lines max
   60 seconds = 160 words max / 16 lines max
   2-3 minutes = 450 words max / 30 lines max
   ```
   - These are HARD LIMITS, not suggestions
   - A04 receives these as non-negotiable constraints from A03
   - Scripts exceeding word count are RETURNED for compression

4. **EXTRACT script structures by vertical:**
   - Vertical-specific format patterns (if documented)
   - Vertical performance data for different structures

5. **EXTRACT visual treatment types:**
   - Talking Head
   - B-Roll + Voiceover
   - Text-on-Screen
   - Screen Recording
   - Mixed
   - Treatment definitions and use cases

6. **EXTRACT UGC script template categories:**
   - 5 types × 3 templates = 15 UGC templates
   - Template structure and characteristics

### Part B: Load AD-HOOK-TAXONOMY.md

1. **READ complete AD-HOOK-TAXONOMY.md**
   - Full read required — contains 32 hook type definitions and platform considerations

2. **EXTRACT Section 5: Platform-Specific Considerations:**
   - Facebook/Instagram considerations (what hook types work best, sound-off implications)
   - TikTok considerations (native aesthetic, sound-on bias, length constraints)
   - YouTube considerations (pre-roll vs Shorts, skip button strategy, longer-form hooks)

3. **EXTRACT hook type performance benchmarks:**
   - If documented: hook rate benchmarks by type
   - If documented: platform-specific hook type performance

4. **EXTRACT hook-platform compatibility notes:**
   - Which hook types are OPTIMAL for which platforms
   - Which hook types are POOR fits for which platforms
   - Sound dependency classifications by hook type

### Part C: Integration

1. **HOLD both references in active context:**
   - Layer 2 format assignment will cross-reference framework matrix
   - Layer 2 length assignment will use word count enforcement table
   - Layer 2 creative treatment assignment will use visual treatment types
   - Layer 1 hook-platform compatibility will use platform considerations

2. **CREATE quick-reference integration:**
   - Framework → Platform mapping
   - Hook Type → Platform compatibility mapping
   - Length → Word Count lookup table
