# 0.2: Ad Intelligence Loader

## Purpose

Load and quantify all format-related intelligence from the A01 Ad Intelligence Handoff. This microskill extracts format distribution data, winning format patterns, creative treatment distribution, length distribution, platform-specific intelligence, and opportunity gaps — all of which ground Layer 2 format assignments in competitive reality rather than arbitrary preferences.

## Inputs

- AD-INTELLIGENCE-HANDOFF.md from A01: `[project]/A01-ad-intelligence/AD-INTELLIGENCE-HANDOFF.md`

## Process

1. **READ complete AD-INTELLIGENCE-HANDOFF.md**
   - Full read required — this is the primary data source for format decisions
   - Do NOT rely on summaries or memory

2. **EXTRACT format distribution data:**
   - Video percentage across all analyzed ads
   - Static image percentage
   - Carousel percentage
   - Other formats (if any)
   - Calculate percentages to nearest 1%
   - Validate percentages sum to ~100%

3. **EXTRACT winning format patterns:**
   - What formats appear in the top 20 winning ads?
   - Calculate winning format distribution (may differ from overall distribution)
   - Identify format types with disproportionate success (e.g., 30% of ads are UGC, but 60% of winners are UGC)
   - Note any format underrepresentation in winners (saturation signal)

4. **EXTRACT creative treatment distribution:**
   - UGC percentage (handheld, natural, conversational style)
   - Polished production percentage (professional lighting, editing, studio)
   - Demonstration percentage (product demos, close-ups, before/after)
   - Talking head percentage (single person to camera)
   - Mixed treatment percentage (combination of above)
   - Other treatments
   - Calculate percentages to nearest 1%

5. **EXTRACT length distribution:**
   - 6-second ads: percentage
   - 15-second ads: percentage
   - 30-second ads: percentage
   - 60-second ads: percentage
   - 2-minute+ ads: percentage
   - Which lengths appear most in winners vs overall?

6. **EXTRACT platform-specific intelligence:**
   - Per-platform format patterns (Meta format distribution, TikTok format distribution, etc.)
   - Per-platform creative treatment patterns
   - Per-platform length patterns
   - Platform-specific insights (e.g., "UGC outperforms polished by 22% on TikTok")

7. **EXTRACT opportunity gaps related to FORMAT:**
   - Underused formats (present in market but low volume — opportunity)
   - Underused lengths (e.g., "No competitors running 60s+ on YouTube")
   - Underused creative treatments (e.g., "Zero demonstration ads despite product being visual")
   - Over-saturated formats (high volume, low differentiation — risk)

8. **QUANTIFY all metrics:**
   - All percentages must be calculated, not estimated
   - Create structured data objects for downstream consumption:
     ```
     format_distribution: { video: 65%, static: 28%, carousel: 7% }
     treatment_distribution: { ugc: 42%, polished: 31%, demo: 15%, talking_head: 8%, mixed: 4% }
     length_distribution: { 6s: 5%, 15s: 38%, 30s: 41%, 60s: 14%, 2min_plus: 2% }
     winning_format_pattern: "UGC video 15-30s dominates top 20 (14/20 ads)"
     ```

9. **HOLD quantified intelligence** for Layer 1-2 consumption
   - Layer 1 platform analysis will cross-reference this data
   - Layer 2 format assignment MUST validate against this data
   - Every format assignment must cite A01 data as rationale
