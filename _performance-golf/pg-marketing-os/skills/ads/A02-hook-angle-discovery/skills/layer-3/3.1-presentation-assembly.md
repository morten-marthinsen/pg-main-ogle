# 3.1: Presentation Assembly

## Purpose
Organize top 15-20 hooks (from Layer 2.5 scoring) by angle group for human scanning. Include all scores and rationale to facilitate informed selection.

## Inputs
- 2.5.7-composite-ranking.md (top 15-20 hooks above threshold)
- All individual scores (2.5.1-2.5.6)

## Process
1. Load top 15-20 hooks that passed composite threshold
2. Group by source angle for easier human comparison
3. Include full scoring breakdown for each hook
4. Add selection instructions

## Output Schema

## Output File
- **Path:** `layer-3-outputs/3.1-presentation.md`

- 15-20 hooks presented (minimum 15)
- Grouped by angle for comparison
- All scores included
- Instructions clear
