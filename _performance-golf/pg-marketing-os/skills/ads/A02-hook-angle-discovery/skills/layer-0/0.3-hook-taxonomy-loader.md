# 0.3: Hook Taxonomy Loader

## Purpose

Load `References/AD-HOOK-TAXONOMY.md` containing all 32 hook types across 10 categories with examples, performance benchmarks, and usage guidance. This taxonomy MUST be held in active context during all of Layer 2 (Hook Generation).

## Inputs

- `skills/ads/References/AD-HOOK-TAXONOMY.md`

## Process

1. **Load taxonomy file**
   - Path: `skills/ads/References/AD-HOOK-TAXONOMY.md`
   - If missing → HALT (taxonomy is REQUIRED for type classification)

2. **Extract all 32 types with examples**
   - Category A (A1-A6): Curiosity & Information Gap
   - Category B (B1-B4): Authority & Social Proof
   - Category C (C1-C4): Pain Point & Problem
   - Category D (D1-D3): Transformation & Results
   - Category E (E1-E3): Urgency & Scarcity
   - Category F (F1-F4): Engagement & Platform-Native
   - Category G (G1-G4): Visual & Sensory
   - Category H (H1-H2): Humor & Entertainment
   - Category I (I1-I2): List & Structure
   - Category J (J1-J3): Special Format

3. **Extract performance benchmarks**
   - Hook rate by type (where available)
   - Ranking of top performers
   - Platform-specific performance notes

4. **Extract usage guidance**
   - When to use each type
   - Awareness level fit
   - Platform fit
   - Niche fit

5. **Prepare for active context loading**
   - Confirm all 32 types extracted
   - Confirm examples are verbatim (not summarized)
   - This entire taxonomy will be loaded into context during Layer 2
