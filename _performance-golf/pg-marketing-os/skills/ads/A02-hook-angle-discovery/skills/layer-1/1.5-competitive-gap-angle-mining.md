# 1.5: Competitive Gap Angle Mining

## Purpose

Extract strategic angles from Ad Intelligence (A01) opportunity gaps. These angles exploit whitespace — hook types competitors aren't using, messaging angles competitors haven't claimed, uncrowded territory.

## Inputs

- Ad Intelligence Handoff (from 0.2-ad-intelligence.md)
- Opportunity gaps section
- Hook type distribution analysis
- Competitive landscape summary

## Process

1. **Load opportunity gaps from A01**
   - Underused hook types (whitespace in taxonomy)
   - Unexploited strategic angles (messaging not being used)
   - Platform-specific gaps (opportunities on target platforms)
   - Uncrowded territory (messaging themes with low competition)

2. **Identify exploitable whitespace**
   - Which gaps align with our strategic foundation?
   - Which gaps can we OWN (not just occupy)?
   - Which gaps create differentiation advantage?

3. **Derive competitive gap angles that:**
   - **Exploit underused hook types:**
     - "Nobody else is talking about X this way..."
     - "The one angle competitors are missing..."
   - **Claim uncrowded messaging territory:**
     - "While everyone focuses on [saturated], we address [whitespace]..."
   - **Differentiate on platform:**
     - "Every ad on [platform] says X. We say Y..."

4. **Plain language angle statements**
   - Focus on the STRATEGIC opportunity, not the gap itself
   - How does this angle position us uniquely?

5. **Source attribution**
   - Which competitive gap is being exploited?
   - Why is this gap an opportunity?
   - How does exploiting it differentiate us?
