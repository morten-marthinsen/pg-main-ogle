# 1.1: Concept Bundler

## Purpose
For each human-selected hook from A02, match it with: (a) the script architecture from A04 built for that hook, (b) the visual direction from A05 designed for that script. Create a concept bundle with all three elements presented together. Each bundle gets a unique Concept ID (C-001, C-002, etc.).

## Inputs
- 0.2-hook-matrix-loader output (selected hooks with IDs)
- 0.3-script-package-loader output (script architectures matched to hooks)
- 0.4-visual-direction-loader output (visual directions matched to scripts)

## Process
```
STEP 1: Load all three Layer 0 loader outputs
  - READ: layer-0-outputs/0.2-hook-matrix-loader.md
  - READ: layer-0-outputs/0.3-script-package-loader.md
  - READ: layer-0-outputs/0.4-visual-direction-loader.md

STEP 2: Create hook -> script -> visual chain for each hook
  - FOR EACH selected hook (H-XXX):
    - FIND: Script architecture matched to this hook (S-XXX)
    - FIND: Visual direction matched to this script (V-XXX)
    - IF script OR visual missing: Flag as INCOMPLETE
    - IF both present: Create concept bundle

STEP 3: Assign Concept IDs
  - C-001 for first complete concept
  - C-002 for second
  - Sequential numbering for all concepts

STEP 4: For EACH concept bundle, compile:
  - Concept ID: C-XXX
  - Hook (VERBATIM from A02):
    - Hook ID: H-XXX
    - Hook text: [exact text]
    - Hook type: [classification]
    - Hook angle: [angle]
  - Script Architecture (from A04):
    - Script ID: S-XXX
    - Framework: [PAS/AIDA/BAB/etc.]
    - Modular sections: [hook, setup, mechanism, proof, CTA]
    - Word count target: [count]
    - Duration: [seconds]
  - Visual Direction (from A05):
    - Visual ID: V-XXX
    - Treatment: [type]
    - Shot list: [shots]
    - Mood: [description]
    - Aspect ratio: [ratio]
  - Target platform: [platform]
  - Big idea alignment: [how this concept expresses the big idea]

STEP 5: Flag any incomplete bundles (missing any of the 3 elements)
```

## Output File
- **Path:** `layer-1-outputs/1.1-concept-bundler.md`
- **Minimum size:** 5KB

## Quality Checks
- Every selected hook has a complete bundle (hook + script + visual)
- Hook text is VERBATIM from A02 (not paraphrased)
- Script framework and modular sections present
- Visual direction has treatment type and shot list
- Target platform assigned per concept
- Concept IDs assigned sequentially
- Incomplete bundles explicitly flagged

## Dependencies
- Layer 0 complete (LAYER_0_COMPLETE.yaml must exist)
- 0.2, 0.3, 0.4 output files must exist
