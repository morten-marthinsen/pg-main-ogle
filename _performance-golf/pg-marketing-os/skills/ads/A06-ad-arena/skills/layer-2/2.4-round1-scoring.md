# 2.4: Round 1 Scoring

## Purpose
Score all revised evaluations against the 7 ad-specific criteria. Calculate weighted totals. Rank all 7 personas per concept. Produce cross-concept comparison. Check deal-breaker thresholds (Scroll-Stop >= 7.0, Platform Nativeness >= 6.5, no criterion below 5.0).

## Inputs
- 2.3-round1-targeted-revision output (revised evaluations with updated scores)
- 7 ad-specific judging criteria with weights (from AGENT.md)
- Quality thresholds: 8.0 weighted overall, 7.0 scroll-stop, 6.5 platform nativeness

## Process
```
STEP 1: Load all revised evaluations
  - READ: layer-2-outputs/2.3-round1-targeted-revision.md

STEP 2: FOR EACH concept:
  FOR EACH persona (7 total):

    A. COMPILE final Round 1 scores (post-revision):
      - Scroll-Stop Power (25%): [score]
      - Visual-Copy Coherence (15%): [score]
      - Mechanism Clarity (15%): [score]
      - Platform Nativeness (15%): [score]
      - Proof Integration (10%): [score]
      - CTA Strength (10%): [score]
      - Memorability (10%): [score]

    B. CALCULATE weighted total:
      = (SS x 0.25) + (VC x 0.15) + (MC x 0.15) + (PN x 0.15)
        + (PI x 0.10) + (CTA x 0.10) + (M x 0.10)

    C. CHECK deal-breaker thresholds:
      - Scroll-Stop Power >= 7.0? [Y/N]
      - Platform Nativeness >= 6.5? [Y/N]
      - Any criterion below 5.0? [Y/N]

STEP 3: RANK personas per concept
  - Sort by weighted total (descending)
  - Identify winner per concept

STEP 4: CROSS-CONCEPT comparison
  - Which concepts have the highest best-persona scores?
  - Which concepts have the most consistent scores across personas?
  - Which concepts have deal-breaker failures?

STEP 5: Identify scoring patterns
  - Which criterion has the widest variance across personas?
  - Which criterion is consistently weakest?
  - Which persona tends to score highest/lowest?
```
