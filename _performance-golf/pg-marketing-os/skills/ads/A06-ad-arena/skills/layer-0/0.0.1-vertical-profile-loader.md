# 0.0.1: Vertical Profile Loader

## Purpose
Load ad-specific vertical configuration from `ad-verticals/`. Extract platform priorities, hook patterns, compliance constraints, anti-slop rules, and vertical-specific scoring adjustments that configure the Arena evaluation for the campaign's vertical (health, golf, finance, personal-dev, technology).

## Inputs
- Campaign Brief (Skill 09) -- vertical identification
- `ad-verticals/[vertical].md` -- vertical-specific configuration file
- `AD-ENGINE-CLAUDE.md` -- master ad engine configuration

## Process
```
STEP 1: Identify campaign vertical from Campaign Brief
  - Extract vertical field (health | golf | finance | personal-dev | technology)
  - IF no vertical specified: Log warning, use universal defaults

STEP 2: Load vertical configuration file
  - READ: ad-verticals/[vertical].md
  - EXTRACT: platform_priorities (which platforms matter most for this vertical)
  - EXTRACT: hook_patterns (vertical-specific hook effectiveness patterns)
  - EXTRACT: compliance_constraints (health claims, financial disclaimers, etc.)
  - EXTRACT: anti_slop_rules (vertical-specific language to avoid)
  - EXTRACT: scoring_adjustments (if any criteria weights differ for this vertical)

STEP 3: Validate configuration completeness
  - All 5 config sections present?
  - Platform priorities non-empty?
  - Compliance constraints loaded (critical for health/finance)?

STEP 4: Write output file with loaded configuration
```
