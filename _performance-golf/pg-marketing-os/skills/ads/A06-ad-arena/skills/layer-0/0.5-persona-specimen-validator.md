# 0.5: Persona Specimen Validator

## Purpose
Validate `ad-persona-specimens/` directory structure. Verify >= 15 specimens exist per persona directory (01-dr-strategist through 06-visual-storyteller). Load 3-5 niche-matched specimens per persona into active context. Flag any persona with insufficient specimens as BLOCKING.

## Inputs
- `ad-persona-specimens/` directory with 6 persona subdirectories
- Campaign vertical (from 0.0.1 output)
- Campaign platform (from Campaign Brief)

## Process
```
STEP 1: Validate directory structure exists
  - CHECK: ad-persona-specimens/01-dr-strategist/ exists
  - CHECK: ad-persona-specimens/02-scroll-stopper/ exists
  - CHECK: ad-persona-specimens/03-ugc-native/ exists
  - CHECK: ad-persona-specimens/04-brand-builder/ exists
  - CHECK: ad-persona-specimens/05-data-scientist/ exists
  - CHECK: ad-persona-specimens/06-visual-storyteller/ exists
  - IF any directory missing: HALT -- create directory structure first

STEP 2: Count specimens per persona
  - FOR EACH persona directory:
    - COUNT total specimen files
    - IF count < 15: Flag as BLOCKING
    - IF count >= 15: Mark as PASS

STEP 3: Niche-match specimen selection (3-5 per persona)
  - FROM campaign vertical: Identify niche (health, golf, finance, etc.)
  - FROM campaign platform: Identify platform (TikTok, Meta, YouTube, Display)
  - FOR EACH persona:
    - SEARCH specimens matching campaign vertical (primary)
    - SEARCH specimens matching campaign platform (secondary)
    - SELECT 3-5 best-matched specimens
    - IF no vertical match: Select closest match + log gap
    - IF no platform match: Select closest match + log gap

STEP 4: Validate specimen content
  - FOR EACH selected specimen:
    - VERIFY contains verbatim ad copy
    - VERIFY contains platform identification
    - VERIFY contains vertical/niche tag
    - VERIFY contains performance data (if available)

STEP 5: Load selected specimens into active context
  - HOLD 3-5 specimens per persona for Arena evaluation
  - NOTE: The Architect (persona 7) does NOT load separate specimens
    -- references all specimens from other 6 personas
```
