# 0.2: Hook Matrix Loader

## Purpose
Load A02 HOOK-ANGLE-MATRIX.md. Extract human-selected hooks (8-10) with their angle attributions, type classifications, and scores. Verify human selection checkpoint was completed in A02.

## Inputs
- A02 HOOK-ANGLE-MATRIX.md: `[project]/A02-hook-angle-matrix/HOOK-ANGLE-MATRIX.md`
- A02 human selection checkpoint: `[project]/A02-hook-angle-matrix/checkpoints/HUMAN_SELECTION_COMPLETE.yaml`

## Process
```
STEP 1: Verify A02 human selection checkpoint exists
  - CHECK: HUMAN_SELECTION_COMPLETE.yaml exists in A02 checkpoints
  - IF NOT EXISTS: HALT -- "A02 human selection not completed. Cannot proceed."
  - IF EXISTS: Proceed to loading

STEP 2: Load HOOK-ANGLE-MATRIX.md
  - READ full file
  - EXTRACT all human-selected hooks (look for selection markers)

STEP 3: For EACH selected hook, extract:
  - Hook ID (H-001, H-002, etc.)
  - Hook text (VERBATIM -- do not paraphrase)
  - Angle attribution (what angle does this hook use)
  - Hook type classification (from AD-HOOK-TAXONOMY.md)
  - Hook score (from A02 evaluation)
  - Target platform (if specified)
  - Hook format (question, statement, statistic, story opener, etc.)

STEP 4: Validate minimum hook count
  - Minimum 3 selected hooks required for meaningful Arena evaluation
  - IF fewer than 3: HALT -- "Insufficient hooks selected in A02"

STEP 5: Compile hook inventory for concept assembly
```
