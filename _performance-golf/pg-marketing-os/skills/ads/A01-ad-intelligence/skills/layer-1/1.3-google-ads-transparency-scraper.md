# 1.3: Google Ads Transparency Center Scraper

**Version:** 1.0
**Layer:** 1 (Scraping)
**Type:** Platform Scraping
**Dependencies:** Layer 0 complete, Firecrawl available
**Output:** google_ad_data.json

---

## Purpose

Scrape Google Ads Transparency Center for each competitor brand. Target: 100+ ads minimum. Extract ad copy text, format (search/display/video), advertiser name, campaign type, and date range. Google Ads Transparency is typically scraped via Firecrawl direct method (less Apify actor support).

---

## Inputs

```yaml
input:
  from_layer_0:
    competitor_list: [string]
  from_0.2:
    firecrawl_available: boolean
```

---

## Process

### Step 1: Per-Competitor Scraping

```
BASE URL: https://adstransparency.google.com/?region=US

FOR EACH competitor:
  CONSTRUCT search URL:
    - Add advertiser name as query
    - Filter by region (default US)
    - Filter by date range (last 90 days)

  SCRAPE:
    mcp__firecrawl-mcp__firecrawl_scrape(
      url="[constructed URL]",
      formats=["markdown"],
      waitFor=3000
    )

  PARSE markdown results:
    - Identify ad entries
    - Extract: ad copy, format type, campaign type, dates
    - Handle "Show more" pagination
```

### Step 2: Data Extraction

```
FOR EACH ad:
  EXTRACT:
    - ad_id: generated
    - platform: "google"
    - ad_format: enum[search, display, video, shopping, unknown]
    - ad_copy_text: headline + description (search) OR body copy (display)
    - advertiser_name: official advertiser name from Google
    - campaign_type: brand, generic, competitor, etc. (if identifiable)
    - start_date: when ad campaign started
    - end_date: when ad campaign ended (or "active")
    - impressions_range: "1K-10K" format (if available)
```

### Step 3: Volume Tracking

```
UPDATE count:
  Google ads: [count] / 100 target

LOG per competitor:
  - Competitor name
  - Ads found
  - Date range covered
```
