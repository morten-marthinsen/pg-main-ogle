# 1.4: Additional Platform Scanner

**Version:** 1.0
**Layer:** 1 (Scraping)
**Type:** Conditional Platform Scraping
**Dependencies:** Layer 0 complete, vertical profile
**Output:** additional_platform_ad_data.json

---

## Purpose

CONDITIONAL execution: Only runs if vertical profile specifies LinkedIn, Pinterest, YouTube, or other platforms as relevant for this vertical. Target: 50+ ads if executed. Provides coverage for verticals where non-standard platforms matter (e.g., LinkedIn for B2B, Pinterest for e-commerce, YouTube for educational content).

---

## Inputs

```yaml
input:
  from_0.0.1:
    vertical_profile:
      additional_platforms: [string]  # e.g., ["linkedin", "pinterest"]
  from_layer_0:
    competitor_list: [string]
```

---

## Process

### Step 1: Execution Decision

```
IF vertical_profile.additional_platforms is empty OR null:
  → SKIP this microskill
  → LOG: "No additional platforms specified for this vertical"
  → CREATE empty output file with executed: false

ELSE:
  → PROCEED with scraping for specified platforms
```

### Step 2: Platform-Specific Scraping

```
FOR EACH platform in additional_platforms:

  CASE platform:

    "linkedin":
      - Search for LinkedIn ad library OR campaign examples
      - Use Firecrawl to scrape LinkedIn ad transparency (if available)
      - Extract: headline, body copy, CTA, format (single image, carousel, video)

    "pinterest":
      - Search Pinterest Ads Inspiration Gallery
      - Use Firecrawl direct scraping
      - Extract: pin description, image URL, CTA, product category

    "youtube":
      - Search YouTube ads via Google Ads Transparency (subset)
      - OR scrape competitor YouTube channels for recent video ads
      - Extract: video title, description, video length, CTA

    "reddit":
      - Scrape Reddit promoted posts (if vertical profile indicates relevance)
      - Extract: post title, body text, subreddit, engagement

    OTHER:
      - LOG: "Platform [X] scraping not yet implemented"
      - SKIP
```

### Step 3: Volume Tracking

```
UPDATE count per platform:
  LinkedIn: [count]
  Pinterest: [count]
  YouTube: [count]

TOTAL additional platform ads: [count]
Target: 50 (if any platforms specified)
```
