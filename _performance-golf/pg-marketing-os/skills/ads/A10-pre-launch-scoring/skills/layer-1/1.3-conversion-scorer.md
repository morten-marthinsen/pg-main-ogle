# 1.3: Conversion Scorer

**Version:** 1.0
**Layer:** 1 (Individual Variant Scoring)
**Type:** Judgment / Prediction
**Model:** opus
**Dependencies:** 0.1 (Variant Matrix), 0.3 (Arena Scores), 0.4 (Campaign Brief), 0.5 (Benchmarks)
**Output:** conversion_scores.md

---

## Purpose

Score EVERY variant on Dimension 3: Conversion Prediction. This predicts the probability that an engaged viewer will take the desired action (click, sign up, purchase). Conversion prediction is the LEAST reliable dimension for pre-launch scoring because actual conversion is determined by the full funnel (landing page, checkout, offer) — not just the ad creative. A10 predicts the AD's contribution to conversion, not the full funnel conversion rate. This inherently lower confidence MUST be documented.

## Process

### Step 1: Load All Evidence Sources

```
READ all input files
FOR EACH variant, COLLECT available evidence:

EVIDENCE SOURCE A: CTA Strength (from 0.3)
  LOOK UP variant's parent concept in arena_scores
  EXTRACT cta_strength score (1-10)
  ASSESS CTA specificity:
    - Specific ("Get your free copy now"): strong
    - Generic ("Learn more"): weak
    - Matched to funnel stage: bonus

EVIDENCE SOURCE B: Proof Integration (from 0.3)
  EXTRACT proof_integration score (1-10)
  Proof in the ad creative increases conversion confidence

EVIDENCE SOURCE C: Offer Alignment
  IF offer_package exists:
    CHECK: Does the variant's CTA connect to the actual offer?
    CHECK: Does the variant mention benefits that match the offer stack?
    IF misaligned: -1.0 (ad promises one thing, landing page delivers another)
  IF offer_package does NOT exist:
    SKIP this source — note as data gap

EVIDENCE SOURCE D: Funnel Stage Match (from 0.4)
  DETERMINE variant's implicit funnel stage:
    - Awareness ad (emotional hook, broad pain point): TOFU
    - Consideration ad (mechanism education, comparison): MOFU
    - Conversion ad (direct CTA, offer details, urgency): BOFU

  COMPARE to campaign target funnel stage:
    - Match: neutral
    - Mismatch: -1.0 (a TOFU ad targeting BOFU audiences wastes spend)
    - NOTE: Funnel mismatch is a SERIOUS conversion prediction penalty

EVIDENCE SOURCE E: Vertical Benchmarks (from 0.5)
  LOOK UP conversion benchmarks for this vertical and funnel stage:
    - Health supplements BOFU: 2-4% CVR typical
    - Info products TOFU: 0.5-1% CTR typical
    - Golf accessories MOFU: 1-2% CTR typical
  IF A12 historical data available: use calibrated benchmarks instead

EVIDENCE SOURCE F: A12 Historical Data (from 0.5, if available)
  IF previous campaigns in same vertical/product:
    LOOK UP actual conversion rates for similar variant types
    COMPARE A10 predictions vs. actuals for calibration adjustment
  IF not available: note as data gap, confidence stays MEDIUM or LOW
```

### Step 2: Score Each Variant

```
FOR EACH variant:

  1. START with Arena cta_strength as baseline (if available)
     IF no Arena score: START at 5.0

  2. ADJUST for proof integration:
     - Arena proof_integration >= 8: +0.5
     - Arena proof_integration <= 4: -0.5

  3. ADJUST for offer alignment:
     - Strong alignment: +0.5
     - Misalignment: -1.0
     - Data unavailable: 0

  4. ADJUST for funnel stage match:
     - Match: 0
     - Mismatch: -1.0

  5. CALIBRATE against vertical benchmarks:
     - If score implies conversion rate above vertical ceiling: reduce
     - If score implies conversion rate below vertical floor: verify evidence

  6. APPLY A12 calibration adjustment (if available):
     - If A10 was historically optimistic on conversion: reduce by adjustment factor
     - If A10 was historically pessimistic: increase by adjustment factor

  7. CLAMP to 1-10

  8. CONFIDENCE: Conversion prediction has INHERENTLY LOWER confidence
     than other dimensions. Maximum confidence level for conversion = MEDIUM
     (unless A12 historical data available AND shows strong correlation,
     in which case HIGH is permitted).

  9. BUILD evidence chain
```

### Step 3: Apply Scoring Scale

```
SCORING SCALE:

9-10: HIGH predicted conversion probability
      Evidence: strong CTA + integrated proof + matched funnel stage +
      positive historical data (if available) + vertical benchmark alignment

7-8:  MODERATE-HIGH conversion probability
      Evidence: solid CTA + some proof + correct funnel stage

5-6:  MODERATE conversion probability
      Evidence: adequate CTA + limited proof or slight funnel mismatch

3-4:  LOW conversion probability — CTA weak, proof missing, or funnel mismatch
      Evidence: multiple conversion barriers present

1-2:  VERY LOW — multiple conversion barriers present
      Evidence: weak CTA + no proof + wrong funnel stage + offer misalignment

CRITICAL NOTE: Express in output that conversion prediction has wider error
margins than other dimensions. An ad can have excellent creative quality
and still underperform on conversion due to landing page, checkout flow,
offer strength, or audience targeting — none of which A10 controls.
```

## Quality Checks

1. ALL variants scored (count must match total from 0.1)
2. Every score has a confidence range (WIDER than other dimensions)
3. Confidence level CAPPED at MEDIUM unless A12 data supports HIGH
4. Confidence disclosure statement included in output header
5. Funnel stage match checked for every variant
6. Offer alignment checked (or flagged as data gap)
7. CTA strength assessed (not just Arena score — specificity analyzed)
8. Scores are meaningfully differentiated
9. No variant scores imply conversion rates above vertical benchmarks without strong evidence
