# 0.2: Format Strategy Loader

## Purpose
Load A03 FORMAT-STRATEGY.md. Extract platform mappings, aspect ratio requirements, ad length assignments, format types (video/static/carousel), and platform-specific constraints (safe zones, text limits, sound-off requirements). This information constrains all visual decisions in Layers 1-3.

## Inputs
- A03 FORMAT-STRATEGY.md (REQUIRED -- A05 cannot execute without this)

## Process
1. READ A03 FORMAT-STRATEGY.md from the project outputs directory
2. EXTRACT platform mappings:
   - Which concepts target which platforms
   - Primary vs. secondary platform assignments
3. EXTRACT aspect ratio requirements per platform:
   - Meta: 4:5 (feed), 1:1 (carousel), 9:16 (Reels/Stories)
   - TikTok: 9:16 (mandatory)
   - YouTube: 16:9 (standard), 9:16 (Shorts)
   - Google Display: 1:1, 1.91:1, 4:5 (responsive)
4. EXTRACT ad length assignments per concept
5. EXTRACT format types (video, static image, carousel, etc.)
6. EXTRACT platform-specific constraints:
   - Safe zones (top, bottom, sides -- percentage and pixel coordinates)
   - Text overlay limits (Meta 20% rule, TikTok no limit)
   - Sound-off requirements (Meta: HIGH, TikTok: HIGH, YouTube: MEDIUM)
   - Caption requirements (Meta: MANDATORY, YouTube: RECOMMENDED)
7. BUILD platform constraint reference table
8. WRITE to output file
