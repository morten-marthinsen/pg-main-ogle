# 0.4: Brand Asset Loader

## Purpose
Load Campaign Brief (Skill 09) brand guidelines section. Extract brand colors, logo files, typography standards, existing product photography, approved imagery style, talent/spokesperson details, and product packaging details. These brand assets constrain all visual direction to maintain brand consistency.

## Inputs
- Campaign Brief from Skill 09 (REQUIRED -- specifically the brand guidelines section)

## Process
1. READ Campaign Brief from the project outputs directory
2. LOCATE the brand guidelines section
3. EXTRACT brand colors:
   - Primary brand color(s) with hex codes
   - Secondary brand colors with hex codes
   - Accent colors with hex codes
   - Color usage rules (where each color is used)
4. EXTRACT logo specifications:
   - Logo file references (primary, secondary, icon-only variants)
   - Logo minimum size requirements
   - Logo clear space rules
   - Logo color variants (full color, white, black)
5. EXTRACT typography standards:
   - Brand fonts (headline, body, accent)
   - Font weights and usage rules
   - Minimum font sizes for readability
6. EXTRACT existing visual assets:
   - Product photography (what exists, quality, formats)
   - Approved imagery style (what the brand looks like)
   - Talent/spokesperson details (if brand has existing talent)
7. EXTRACT product packaging details:
   - Package design (bottle, box, pouch, etc.)
   - Label design elements
   - Product color and texture
   - Distinguishing visual features
8. FLAG any missing brand elements that will constrain visual direction
9. WRITE to output file
