# 4.2: Per-Concept Brief Files

## Purpose
Create individual visual brief files in `visual-briefs/` directory -- one per script: `[script-id]-visual-brief.md`. These are the working files A08 (Visual/Video Production) will use for production. Each file is self-contained with all visual direction for that specific script -- shot list, talent direction, product staging, motion graphics, B-roll, sound-off strategy, tool-specific specs, and platform export specs. A production team should be able to work from a single file without referencing the master package.

## Inputs
- VISUAL-DIRECTION-PACKAGE.md from 4.1 (master package to extract per-concept content)
- All Layer 2 outputs (per-concept visual direction)
- All Layer 3 outputs (per-concept tool specs)
- Platform export specs from 3.5

## Process
1. READ VISUAL-DIRECTION-PACKAGE.md from 4.1
2. READ all Layer 2 and Layer 3 outputs
3. FOR EACH script/concept:
   a. CREATE file: `visual-briefs/[script-id]-visual-brief.md`
   b. POPULATE with ALL visual direction for that specific script:

      **Header:**
      - Script ID, Concept ID, Treatment Type, Target Platform(s)
      - Ad Length, Framework, Hook Type

      **Shot List:**
      - COMPLETE shot-by-shot table with all 7 elements
      - Audio alignment notes per shot

      **Talent Direction:**
      - Full talent brief (if applicable)
      - Arcads parameters (if AI talent)

      **Product Staging:**
      - All product shot specs (if applicable)

      **Motion Graphics:**
      - Elements needed for this specific script
      - Timing and animation specs

      **B-Roll:**
      - All B-roll shots with descriptions and sourcing info
      - Stock queries and AI prompts specific to this concept

      **Sound-Off Strategy:**
      - Text overlay narrative arc
      - Caption strategy
      - Visual proof elements

      **Thumbnail:**
      - Thumbnail specs (YouTube only)

      **Tool-Specific Specs:**
      - Midjourney prompts for this concept
      - Arcads brief for this concept (if applicable)
      - ElevenLabs spec for this concept (if applicable)
      - Stock footage queries for this concept

      **Platform Export:**
      - Technical specs for this concept's target platform(s)

      **Coherence Score:**
      - VA and EA alignment scores from Layer 2.5

   c. VERIFY file is self-contained:
      - A production team reading ONLY this file has everything needed
      - No references to "see master package for details"
4. COUNT files created and verify against total script count
5. WRITE assembly report to output file
