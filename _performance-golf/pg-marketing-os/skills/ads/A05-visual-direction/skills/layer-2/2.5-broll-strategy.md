# 2.5: B-Roll Strategy

## Purpose
For each concept, design the B-roll strategy: primary B-roll needs (product shots, lifestyle footage, environment shots, results visualization), secondary B-roll (supplementary footage adding context -- lab shots, ingredient close-ups, expert credentials, location establishing shots), B-roll pacing (when to cut to B-roll, duration per insert, frequency). For each B-roll shot: provide a specific description (not "nature footage" but "close-up of green tea leaves being picked by hand in morning mist, slow motion, warm golden light"), stock footage search queries, and AI generation prompts.

## Inputs
- Shot lists from 2.1 (where B-roll inserts are needed)
- Treatment type from 1.1 (treatment drives B-roll style)
- Script inventory from 0.1 (what the audio says during B-roll moments)
- Color palettes from 1.2 (B-roll must coordinate with palette)
- Product staging from 2.3 (product B-roll coordinates with hero shots)

## Process
1. READ shot lists from 2.1
2. READ treatment types from 1.1
3. READ script beat structures from 0.1
4. READ color palettes from 1.2
5. READ product staging from 2.3
6. FOR EACH concept:
   a. IDENTIFY B-roll insertion points in the shot list:
      - Where does the script benefit from visual variety?
      - Where does the audio discuss something not shown by talent?
      - Where does pacing need a visual break?
   b. CATEGORIZE B-roll needs:
      - **Primary B-roll:** Essential footage (product in use, results, key visual proof)
      - **Secondary B-roll:** Enhancement footage (lifestyle context, ingredient sourcing, manufacturing)
   c. FOR EACH B-roll shot:
      i. WRITE specific description (minimum 30 words per shot):
         - Subject: what is in frame (specific, not generic)
         - Action: what is happening (movement, process, change)
         - Lighting: natural, warm, cool, dramatic, etc.
         - Speed: real-time, slow motion, time-lapse
         - Color/mood: match to palette and emotional beat
      ii. The SPECIFICITY TEST: Can this description be swapped into a different campaign? If YES, it is too generic. REWRITE.
      iii. WRITE stock footage search queries:
         - Primary query: [main search term]
         - Secondary filters: [orientation, duration, style]
         - Alternative queries: [2-3 variations]
      iv. WRITE AI generation prompt (Midjourney/Flux):
         - Full prompt with style parameters
   d. DEFINE B-roll pacing:
      - Maximum consecutive seconds without B-roll: [X seconds]
      - Average B-roll insert duration: [X seconds]
      - B-roll frequency: [X inserts per 30 seconds]
7. ESTIMATE total B-roll footage needed per concept (in seconds)
8. WRITE to output file
