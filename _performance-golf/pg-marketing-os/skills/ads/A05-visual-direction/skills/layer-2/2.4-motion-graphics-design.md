# 2.4: Motion Graphics Design

## Purpose
Design motion graphics elements for each concept: lower thirds (name plates, credentials, locations), text animation styles (reveal method, duration, entrance/exit), data visualizations (charts, progress bars, comparison graphics), countdown timers / urgency elements, logo animations (brand reveal moments), screen recording overlays (cursor highlights, annotation callouts), transition graphics (branded wipes, slides). All elements must match the typography system from Layer 1.

## Inputs
- Shot lists from 2.1 (where motion graphics are needed)
- Typography system from 1.3 (fonts, sizes, animation rules)
- Color palettes from 1.2 (motion graphics colors)
- Treatment type from 1.1 (treatment affects motion graphics style)
- Platform constraints from 0.2 (size and placement limits)

## Process
1. READ shot lists from 2.1
2. READ typography system from 1.3
3. READ color palettes from 1.2
4. READ treatment types from 1.1
5. READ platform constraints from 0.2
6. DESIGN motion graphics element library:
   a. **Lower thirds:**
      - Name plate design (font, size, color, background, position)
      - Credential display (title, organization, years of experience format)
      - Location display (city/state format)
      - Animation: entrance method, duration, exit method
   b. **Text animation styles:**
      - Primary text reveal (for hook text, key claims)
      - Secondary text reveal (for supporting points)
      - Caption style (auto-caption format)
      - Animation timing: display duration = word count / reading speed + buffer
   c. **Data visualizations:**
      - Chart style (bar, line, pie -- match treatment aesthetic)
      - Progress bar design (color, animation speed, label)
      - Comparison graphic (split-screen, side-by-side, overlay)
      - Number animation (counting up/down for statistics)
   d. **Urgency elements:**
      - Countdown timer design (font, color, position, animation)
      - Limited stock indicator (style, placement)
      - Price display (original/sale formatting)
   e. **Logo animations:**
      - Brand reveal: when, how, duration
      - Persistent logo: position, size, opacity
      - End card: logo + CTA + website
   f. **Screen recording overlays (if applicable):**
      - Cursor highlight (circle, glow, arrow)
      - Annotation callout (box, arrow, text style)
      - Zoom-in region highlight
   g. **Transition graphics:**
      - Branded wipe direction and style
      - Color flash transitions
      - Slide transitions (direction, speed)
7. ADAPT all elements for each platform:
   - TikTok: bold, playful, large text
   - Meta: clean, within 20% text rule
   - YouTube: professional lower thirds, data visualizations
8. WRITE to output file
