# 2.7: Thumbnail Design

## Purpose
**YouTube concepts only.** For each YouTube ad, design thumbnail specifications: composition (layout, focal point, visual hierarchy), text (2-4 words maximum, font, color, size, position), subject (face close-up with expression, or compelling product/result image), background treatment (solid color, blurred image, split composition), A/B variants (at least 2 options per concept), and color contrast against YouTube UI (red play button, white background). YouTube is the ONLY platform where thumbnail is a separate creative decision -- and the thumbnail IS the scroll-stop mechanism.

## Inputs
- Shot lists from 2.1 (first frame reference, key visual moments)
- Scripts from 0.1 (content to reference in thumbnail text)
- Color palettes from 1.2 (thumbnail must coordinate with ad palette)
- Typography system from 1.3 (thumbnail text style)
- Platform specs from 0.2 (YouTube thumbnail dimensions: 1280x720)

## Process
1. READ shot lists from 2.1 (for YouTube concepts only)
2. READ script content from 0.1
3. READ color palettes from 1.2
4. READ typography from 1.3
5. FILTER: only YouTube concepts need thumbnails
6. FOR EACH YouTube concept:
   a. DESIGN thumbnail composition:
      - Layout: rule of thirds, subject placement, negative space
      - Focal point: what draws the eye first
      - Visual hierarchy: primary element > secondary > tertiary
      - Resolution: 1280x720 pixels
   b. DESIGN thumbnail text:
      - Words: 2-4 maximum (must be readable at mobile preview size)
      - Font: bold, high-contrast (from typography system or specific thumbnail font)
      - Color: contrasts with background AND YouTube UI (red/white)
      - Size: readable at 160x90 (smallest preview)
      - Position: not covering subject's face, within safe area
   c. DESIGN thumbnail subject:
      - Option A: Face close-up with exaggerated/emotional expression
        * Which expression (surprise, concern, excitement, curiosity)
        * Framing (how much of face visible)
      - Option B: Compelling product/result image
        * Product positioning
        * Result visualization
      - Note: Face close-ups outperform other styles on YouTube
   d. DESIGN background:
      - Solid color (complementary to YouTube UI)
      - Blurred image from ad content
      - Split composition (before/after, problem/solution)
      - Gradient or bokeh
   e. CHECK color contrast:
      - Against YouTube white background
      - Against YouTube red play button overlay
      - Against YouTube dark mode background
      - Thumbnail must POP in search results and suggested videos
   f. DESIGN A/B variants (minimum 2 per concept):
      - Variant A: [description]
      - Variant B: [description]
      - What is different between them (test hypothesis)
7. WRITE to output file

## Quality Checks
- Every YouTube concept has at least 2 thumbnail variants
- Thumbnail text is 2-4 words maximum (not full sentences)
- Text is readable at smallest preview size (160x90)
- Color contrast checked against YouTube UI (white, red, dark mode)
- Face close-ups with expression preferred (unless product demo is more compelling)
- A/B variants have a clear test hypothesis (not identical with minor tweaks)
