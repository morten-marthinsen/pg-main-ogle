# 2.2: Talent Direction Brief

## Purpose
For each concept requiring on-camera talent (Treatments 2, 3, 5 primarily): provide comprehensive talent description (age range, gender, ethnicity if relevant, energy type, wardrobe direction, grooming notes), performance direction (conversational vs authoritative, eye line, energy level, pacing, key emotional beats), and setting/environment direction. For AI talent (Arcads): map all direction to Arcads-compatible talent parameters.

## Inputs
- Shot lists from 2.1 (which scripts require talent, what they do)
- Treatment type assignments from 1.1 (treatment drives talent style)
- Soul.md visual constraints from 0.5 (energy signature affects talent direction)
- Brand asset info from 0.4 (existing talent/spokesperson if applicable)

## Process
1. READ shot lists from 2.1
2. READ treatment types from 1.1
3. READ Soul.md constraints from 0.5
4. READ brand assets from 0.4
5. IDENTIFY which concepts require on-camera talent:
   - Treatment 2 (UGC): ALWAYS requires talent
   - Treatment 3 (Hybrid): Usually requires talent
   - Treatment 5 (Testimonial): ALWAYS requires talent
   - Treatment 1 (Polished): Sometimes (authority/expert)
   - Treatment 4 (Demonstration): Sometimes (hands only or with presenter)
6. FOR EACH concept requiring talent:
   a. DEFINE talent profile:
      - Age range (e.g., "35-45")
      - Gender
      - Ethnicity/appearance (if relevant to target audience alignment)
      - Body type (if relevant)
      - Energy type (warm/authoritative/energetic/contemplative)
      - Wardrobe direction (specific: "olive linen button-down" not "casual clothes")
      - Grooming/styling (hair, makeup direction)
   b. DEFINE performance direction:
      - Delivery style: conversational, authoritative, excited, concerned, sincere
      - Eye line: direct to camera (UGC), slightly off-camera (interview style), varies
      - Energy level: 1-10 scale with description
      - Pacing: words per minute range, pause locations
      - Key emotional beats: where energy shifts in the script
      - Improvisation guidelines: what can/should feel unscripted
   c. DEFINE setting/environment:
      - Location type: kitchen, bathroom, outdoors, office, etc.
      - Specific details: "white quartz countertop, blurred plants in background"
      - Lighting setup: natural window, ring light, etc.
      - Props in scene: what objects are visible
   d. FOR AI TALENT (Arcads):
      - Map all above to Arcads-compatible parameters
      - Actor demographic parameters
      - Emotion settings
      - Camera angle preferences
      - Background/setting parameters
7. WRITE to output file
