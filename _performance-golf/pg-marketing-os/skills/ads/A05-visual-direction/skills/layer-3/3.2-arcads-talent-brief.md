# 3.2: Arcads Talent Brief

## Purpose
For each concept using AI talent (UGC, testimonial, hybrid treatments): translate talent direction into Arcads-compatible specifications. Include: actor demographic (age, gender, ethnicity), emotion setting (confident, excited, concerned, conversational), script text (verbatim from A04), delivery speed (words per minute), eye contact direction, wardrobe notes, background/setting, camera angle preference, brand safety flags. These briefs enable A08 to execute AI talent production without additional interpretation.

## Inputs
- Talent direction briefs from 2.2 (creative talent direction)
- Script text from 0.1 / A04 (verbatim dialogue/voiceover for Arcads)
- Treatment types from 1.1 (which concepts use AI talent)
- Platform specs from 0.2 (aspect ratio, framing requirements)

## Process
1. READ talent direction briefs from 2.2
2. READ script text from 0.1
3. READ treatment types from 1.1
4. READ platform specs from 0.2
5. IDENTIFY concepts requiring AI talent:
   - Treatment 2 (UGC): highly likely
   - Treatment 3 (Hybrid): likely
   - Treatment 5 (Testimonial): highly likely
   - Treatment 1 (Polished): sometimes
   - Treatment 4 (Demonstration): sometimes
6. FOR EACH concept requiring AI talent:
   a. TRANSLATE talent profile to Arcads parameters:
      - Actor age range: [Arcads age bracket]
      - Actor gender: [M/F/Any]
      - Actor ethnicity: [if specified for audience matching]
      - Actor appearance: [mapped to Arcads talent pool descriptors]
   b. TRANSLATE performance to Arcads emotion settings:
      - Primary emotion: [confident/excited/concerned/friendly/authoritative]
      - Emotion intensity: [1-10]
      - Delivery energy: [conversational/enthusiastic/serious/warm]
   c. PROVIDE verbatim script text:
      - Complete script for the talent to deliver
      - Pronunciation guides for brand names / technical terms
      - Emphasis markers (which words to stress)
   d. SPECIFY delivery parameters:
      - Words per minute: [range]
      - Pauses: [where to pause, duration]
      - Eye contact: [direct to camera / slight off-camera / varies]
   e. SPECIFY visual parameters:
      - Wardrobe notes: [specific description]
      - Background/setting: [description mapped to Arcads options]
      - Camera angle: [front/slight angle/profile]
      - Framing: [head and shoulders / waist up / etc.]
      - Aspect ratio: [from platform specs]
   f. SPECIFY brand safety flags:
      - Compliance requirements (health claims, financial disclaimers)
      - Content restrictions
   g. VALIDATE Arcads compatibility:
      - All required Arcads fields populated
      - Settings within Arcads capability range
7. WRITE to output file
