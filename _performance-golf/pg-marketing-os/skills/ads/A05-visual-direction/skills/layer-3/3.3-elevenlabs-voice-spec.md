# 3.3: ElevenLabs Voice Specification

## Purpose
For each concept requiring voiceover: define voice character description (age, gender, tone, energy, accent), delivery direction (conversational vs authoritative, pacing, emphasis words, pause locations), emotional arc mapping (start calm --> build intensity --> resolve), pronunciation notes for brand names, technical terms, product names, and reference audio descriptions. Map to ElevenLabs voice parameters where possible. These specs feed A08 for voiceover production.

## Inputs
- Script text from 0.1 / A04 (verbatim voiceover content)
- Treatment types from 1.1 (treatment affects voice style)
- Talent direction from 2.2 (performance direction informs VO direction)
- Soul.md constraints from 0.5 (energy signature affects voice)

## Process
1. READ scripts from 0.1
2. READ treatment types from 1.1
3. READ talent direction from 2.2
4. READ Soul.md constraints from 0.5
5. IDENTIFY concepts requiring voiceover:
   - Treatment 1 (Polished): Usually professional VO
   - Treatment 3 (Hybrid): Sometimes VO + on-camera
   - Treatment 4 (Demonstration): Often VO over product footage
   - Edutainment scripts: VO with motion graphics
   - Note: UGC (Treatment 2) rarely uses separate VO -- talent speaks directly
6. FOR EACH concept requiring voiceover:
   a. DEFINE voice character:
      - Age impression: [young adult / middle-aged / mature]
      - Gender: [male / female / neutral]
      - Tone: [warm / authoritative / conversational / energetic / serious]
      - Energy level: [calm / moderate / high]
      - Accent: [neutral American / British / regional / none]
      - Voice quality: [smooth / gravelly / clear / resonant]
   b. DEFINE delivery direction:
      - Style: conversational vs authoritative vs educational
      - Pacing: overall WPM range
      - Emphasis words: [list specific words to stress]
      - Pause locations: [mark pauses in script with duration]
      - Breathing: natural vs controlled
   c. MAP emotional arc:
      - Opening energy: [calm/curious/concerned]
      - Building phase: [where intensity increases]
      - Climax: [peak emotional moment -- voice should match]
      - Resolution: [confident/reassuring/urgent]
      - Map to specific script sections with timing
   d. DOCUMENT pronunciation notes:
      - Brand name: [phonetic spelling]
      - Product name: [phonetic spelling]
      - Technical terms: [phonetic spelling]
      - Ingredient names: [phonetic spelling]
   e. PROVIDE reference descriptions:
      - "Sounds like [celebrity/known voice], but [variation]"
      - Or: specific voice quality descriptors
   f. MAP to ElevenLabs parameters (where possible):
      - Voice selection recommendations
      - Stability setting (0-1)
      - Similarity boost (0-1)
      - Style exaggeration (0-1)
      - Speaker boost: [on/off]
7. WRITE to output file
