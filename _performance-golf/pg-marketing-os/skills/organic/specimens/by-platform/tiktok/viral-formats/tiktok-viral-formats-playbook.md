## 1. Format Performance Hierarchy

Not all formats are equal. Based on cross-specimen analysis, here is the performance hierarchy ranked by median views per post:

### Tier 1: Participation Formats (Median 80M+ views)
These formats invite mass replication, which creates exponential distribution.

| Format | Why Tier 1 | Key Metric |
|--------|-----------|------------|
| Dance Challenge | Every recreation is a new distribution node | Shares + Duets |
| Lip Sync / Expression | Mirror neuron activation = compulsive imitation | Duets + Loops |
| Trending Audio First-Mover | Algorithm prioritizes first high-quality use | Speed to publish |

**Teaching connection:** Berger's "Public" principle — when behavior is visible, it gets copied. Participation formats make imitation the entire point.

### Tier 2: Spectacle Formats (Median 40-80M views)
These formats generate awe, which drives share behavior.

| Format | Why Tier 2 | Key Metric |
|--------|-----------|------------|
| Practical Magic / Illusion | Cannot resolve on first watch = rewatch + share | Replay rate |
| Transition Edit | Transformation contrast triggers dopamine | Completion rate |
| Silent Comedy (Global) | Language-independent = global audience | Geographic spread |
| Challenge / Stunt | Extreme stakes = can't look away | Watch-through rate |
| Satisfying Process / ASMR | Sensory completion loop | Loop count |

**Teaching connection:** Heath brothers' "Unexpected" — the brain allocates attention to things that violate expectations. Spectacle formats are expectation-violation machines.

### Tier 3: Narrative Formats (Median 20-40M views)
These formats hold attention through story structure.

| Format | Why Tier 3 | Key Metric |
|--------|-----------|------------|
| Character Switch / Sketch | Each character re-hooks attention | Avg watch time |
| GRWM + Storytime | Dual-track (visual + story) compounds retention | Comments |
| POV Comedy | Viewer as protagonist = immersion | Saves + Shares |
| Mini Movie / Short Film | Complete narrative arc in 60 seconds | Replay + Saves |
| Storytime + Cooking | Emotional narrative + sensory visuals | Save rate |

**Teaching connection:** Heath brothers' "Stories" — narrative transport reduces counter-arguing. When someone is inside a story, they don't evaluate, they experience.

### Tier 4: Authority Formats (Median 10-25M views)
Lower raw views but highest conversion, save, and follow rates.

| Format | Why Tier 4 | Key Metric |
|--------|-----------|------------|
| Stitch + Educate | Borrowed audience + intellectual currency | Follows per view |
| Medical / Myth Busting | Authority + urgency = save behavior | Save rate |
| Nature / Science Narration | Curiosity gap + knowledge = share currency | Share rate |
| Financial Advice | Specific thresholds = self-selection | Save rate |
| Greenscreen Commentary | Low effort, high opinion, trend-riding | Speed to publish |

**Teaching connection:** Berger's "Practical Value" — people share things that help others. Authority formats position the sharer as a helpful curator.

### Tier 5: Identity Formats (Median 8-20M views)
Lower total views but extremely high engagement rate and community building.

| Format | Why Tier 5 | Key Metric |
|--------|-----------|------------|
| Text-On-Screen Confession | Faceless = low barrier, high save | Save-to-view ratio |
| Corporate Satire | Workplace = universal pain point | DM shares (dark social) |
| Hot Take / Contrarian | Binary opinions = comment wars | Comment count |
| Vulnerable Monologue | Trust building at scale | Follower conversion |
| Niche / Surreal / Oddity | Micro-community passion sharing | Engagement rate |

**Teaching connection:** Berger's "Social Currency" — people share content that makes them look good, smart, or aligned with their identity. Identity formats make sharing feel like self-expression.

## 2. Optimal Format Length by Type

The algorithm rewards completion rate, not absolute length. A 15-second video watched to the end beats a 3-minute video abandoned at 30 seconds.

| Format Type | Optimal Length | Why |
|-------------|---------------|-----|
| Dance / Lip Sync | 10-15 sec | Must be learnable in 3 watches |
| Transition / Magic | 10-20 sec | Surprise must happen before attention wavers |
| Comedy Sketch / POV | 20-45 sec | Setup + punchline. No padding. |
| Storytime / GRWM | 60-90 sec | Story needs arc but not filler |
| Educational / Stitch | 30-60 sec | One insight fully explained |
| Day-in-My-Life | 45-90 sec | 5-8 scenes max, each 8-12 seconds |
| Mini Movie | 45-60 sec | Full narrative arc compressed |
| ASMR / Satisfying | 30-60 sec | Process needs resolution |
| Series Content | 45-90 sec per part | Cliffhanger before natural end |
| Street Interview | 30-60 sec | 2-4 responses, best answers only |

### The Length-Performance Curve
```
Completion Rate vs. Length:

100% |*
     | *
 80% |  *
     |   *
 60% |    *
     |     * * *
 40% |          * *
     |              * * *
 20% |                    * * *
     |                          * * *
  0% +--+--+--+--+--+--+--+--+--+--+-->
     0  15 30 45 60 90 120 150 180 sec

Sweet spot: 15-60 seconds for most formats
Exception: Series content and storytime can go 90+ with strong hooks
```

### The 3-Second Rule (Kane)
Regardless of total length, the first 3 seconds determine everything:
- **0-1 sec:** Visual pattern interrupt (movement, contrast, face)
- **1-2 sec:** Hook text or audio that creates curiosity gap
- **2-3 sec:** Promise of payoff that earns the next 10 seconds

If they're not hooked by second 3, nothing else matters.

## 3. Audio Strategy: Original vs. Trending

### When to Use Trending Audio
- **Goal is maximum reach** — trending audio gets algorithm boost in discovery
- **Format is participation-based** — dances, lip syncs, transitions
- **Speed matters** — first 24 hours of a trend capture 70% of total views
- **Content is visual-first** — audio is secondary to what's happening on screen
- **Building initial audience** — trending audio introduces you to new viewers

### When to Use Original Audio
- **Building a personal brand** — your voice IS the content
- **Educational / authority content** — voiceover requires original
- **Storytime / narrative** — story needs your voice, not background music
- **Creating a template** — your sound becomes the trend others use
- **Established audience** — they follow for YOU, not for sounds

### Audio Strategy Matrix

| Your Goal | Audience Size | Audio Choice | Rationale |
|-----------|--------------|-------------|-----------|
| Grow followers fast | <10K | 80% trending, 20% original | Ride algorithm distribution |
| Build authority | Any | 80% original, 20% trending | Your voice = your brand |
| Go viral once | Any | Trending audio + visual hook | Maximize discovery surface |
| Build series | Any | Original (consistent voice) | Audience expects YOUR delivery |
| Cross-platform repurpose | Any | Original (you own it) | Trending audio = platform-locked |

### The Sound-to-Trend Pipeline
The most powerful play: create original audio that BECOMES a trend.
1. Post content with original sound
2. Sound gets used by 5-10 creators organically
3. Algorithm identifies it as emerging trend
4. Sound appears on trending page
5. Thousands of creators use it
6. Every use credits back to your profile

**Teaching connection:** Berger's "Triggers" — audio is the most powerful trigger mechanism on TikTok. A trending sound heard elsewhere triggers recall of associated content.

## 4. Text Overlay Best Practices

Text overlays are the most underrated tool on TikTok. They create a second engagement channel that operates independently from audio.

### The 4 Functions of Text Overlay

**1. Hook Text (First Frame)**
- Must be readable in under 1 second
- Creates curiosity gap or emotional reaction
- Examples: "Nobody talks about this", "Watch until the end", "I can't believe this worked"
- Placement: Top-center or center, large font, high contrast

**2. Context Text (Opening Seconds)**
- Sets up the scenario without spoken words
- Enables sound-off viewing (85% of TikTok is watched with sound off initially)
- Examples: "POV: You're...", "Day 47 of...", "Things that..."
- Placement: Center, medium font, background color bar

**3. Retention Text (Throughout)**
- Timed reveals that create micro-hooks within the video
- Each new text appearance re-hooks wandering attention
- Examples: Sequential list items, real-time commentary, reaction labels
- Placement: Varies — follow visual focus

**4. CTA Text (Final Frames)**
- Directs next action: follow, part 2, save for later, comment
- Must feel organic, not desperate
- Examples: "Part 2?", "Save this", "Follow for more [niche]"
- Placement: Bottom-center, clear font

### Text Technical Specs
- **Font size:** Minimum 24pt equivalent (must be readable on phone)
- **Duration:** Each text block visible for minimum 2 seconds
- **Contrast:** White text on dark overlay bar, or outline/shadow on all text
- **Safe zones:** Keep text away from bottom 15% (caption overlap) and top 10% (status bar)
- **Max words per frame:** 8-12 words (readers should finish before next cut)


## 6. Format Combination Strategies

The most viral content layers multiple formats. Single-format content caps at Tier performance. Combined formats break through.

### High-Performance Combinations

**GRWM + Storytime + Hot Take**
- Getting ready provides visual track
- Story provides narrative track
- Hot take provides opinion track
- Combined engagement: visual satisfaction + curiosity + tribal alignment
- Example: "Get ready with me while I explain why I quit my 6-figure job" (VF-001 structure)

**Stitch + Greenscreen + Educational**
- Stitch borrows audience
- Greenscreen shows evidence
- Educational framing earns saves
- Combined: borrowed audience + visual proof + practical value
- Example: "Let me show you why this viral finance tip will actually lose you money"

**POV + Character Switch + Trending Audio**
- POV creates immersion
- Characters create variety
- Trending audio creates discovery
- Combined: immersion + comedy + algorithm boost
- Example: "POV: Every person at Thanksgiving dinner, but make it [trending sound]"

**Transition + Before/After + Text Overlay**
- Transition creates spectacle
- Before/after creates transformation narrative
- Text overlay enables sound-off viewing
- Combined: visual dopamine + transformation satisfaction + accessibility
- Example: Text "Day 1 vs Day 365" over transition from messy room to organized space

## 7. Format Evolution & Platform Trends

### Format Lifecycle on TikTok

```
Phase 1: EMERGENCE (Week 1-2)
  - New format appears from niche creator
  - Early adopters with large followings replicate
  - Algorithm begins testing in discovery feeds

Phase 2: EXPLOSION (Week 2-4)
  - Major creators adopt format
  - Variations and remixes appear
  - Audio associated with format trends
  - Peak views per post for format

Phase 3: SATURATION (Week 4-8)
  - Everyone is doing it
  - Quality of entries declines
  - Audience fatigues on format
  - Algorithm reduces push

Phase 4: MUTATION (Week 8-16)
  - Format evolves into sub-variants
  - Niche-specific adaptations emerge
  - "This format but for [niche]" posts
  - Second wave smaller than first

Phase 5: ARCHIVE (Month 4+)
  - Format becomes "classic"
  - Occasional nostalgic revivals
  - Becomes template reference for new creators
  - Lives on as structural DNA in future formats
```

### Strategic Timing
- **Phase 1:** If you spot it, move IMMEDIATELY. First-mover advantage is extreme.
- **Phase 2:** Still worth joining. Focus on quality execution and niche adaptation.
- **Phase 3:** Only join if you have a genuinely novel twist. Otherwise, skip.
- **Phase 4:** Niche-specific mutations can outperform generic Phase 2 entries.
- **Phase 5:** Reference for format education, not for new content.

### Emerging Format Signals to Watch
1. A video from a <10K account gets 1M+ views using an unfamiliar structure
2. Multiple unrelated creators independently discover the same format
3. A new native feature launches (TikTok always boosts early adopters of new features)
4. A format from another platform (Reels, Shorts) has no TikTok equivalent yet

## 8. Format Testing Protocol

### The 3x3 Test
When launching a new format for your account:

1. **Create 3 variations** of the same format type
2. **Post them across 3 different times** (morning, afternoon, evening)
3. **Measure after 48 hours** (TikTok's algorithm takes ~48 hours to fully distribute)

### Metrics to Track Per Format

| Metric | What It Tells You | Threshold for "Working" |
|--------|-------------------|------------------------|
| Completion rate | Does the format hold attention? | >40% for 30-sec, >25% for 60-sec |
| Share rate | Does the format trigger sharing? | >2% of views |
| Save rate | Does the format have reference value? | >1% of views |
| Follow rate | Does the format attract subscribers? | >0.5% of views |
| Comment rate | Does the format trigger conversation? | >1% of views |
| Loop rate | Does the format earn replays? | >1.3x average views per viewer |

### When to Double Down vs. Pivot

**Double down when:**
- Completion rate exceeds 40% on a 30+ second video
- One video significantly outperforms your average (3x+ views)
- Comments contain "part 2?" or "more of this"
- Profile visits spike after posting this format

**Pivot when:**
- 3 attempts of same format all underperform your average
- Completion rate drops below 20%
- Engagement is all from existing followers (no new audience reach)
- The format consistently takes 3x the production time of alternatives
