# 0.2 Teaching Loader

## Purpose
Load monetization frameworks, creator economy models, pricing strategies, and audience readiness criteria. Provides foundation for designing effective, sustainable monetization.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- Monetization frameworks (if specified)
- Creator economy models and case studies
- Pricing strategies and psychology
- Audience readiness criteria

## Execution Protocol

### MUST
- Load creator economy monetization models: affiliates, sponsorships, digital products, memberships, consulting
- Load pricing strategies: value-based pricing, competitive benchmarking, psychological pricing, tiering
- Load audience readiness frameworks: follower thresholds, engagement benchmarks, trust signals
- Load product development principles: authority alignment, audience needs, delivery formats
- Load partnership frameworks: brand alignment, rate cards, disclosure requirements
- Include creator economy case studies showing monetization paths per archetype

### NEVER
- Use generic "make money online" advice (need creator-specific frameworks)
- Skip audience readiness criteria (critical for preventing premature monetization)
- Ignore pricing psychology (impacts conversion dramatically)
- Load frameworks without archetype-specific considerations
- Assume monetization is universal across creator types

## Output
**Filename:** `teaching-summary.md`

**Format:** Markdown teaching reference

**Minimum Size:** 5KB

**Required Sections:**
1. Creator Economy Monetization Models (5+ models with archetype fit)
2. Pricing Strategies (value-based, competitive, psychological, tiering)
3. Audience Readiness Frameworks (criteria for determining monetization timing)
4. Product Development Principles (authority alignment, needs assessment, formats)
5. Partnership Frameworks (brand alignment, rate cards, disclosures)
6. Creator Case Studies (successful monetization examples per archetype)

## Quality Gates
- [ ] Monetization models cover 5+ revenue streams
- [ ] Pricing strategies include value-based approach (not just competitive)
- [ ] Audience readiness criteria explicit (not vague)
- [ ] Product development principles include authority alignment
- [ ] Partnership frameworks address brand alignment and disclosure
- [ ] Case studies show monetization per archetype

## Handoff
Feeds into: 0.3 Upstream Loader and all Layer 1 microskills (teaching reference throughout)
