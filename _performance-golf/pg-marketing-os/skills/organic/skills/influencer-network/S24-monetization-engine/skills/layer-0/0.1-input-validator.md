# 0.1 Input Validator

## Purpose
Validate all S21 Persona Bibles, S22 Account Plans, and S23 Coordination Plan exist and are complete before monetization strategy begins.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- S21 Persona Bibles (all persona-bible-[name].md files)
- S21 Network Overview (network-overview.md)
- S22 Account Strategy Plans (all account-plan-[persona]-[platform].md files)
- S23 Network Coordination Plan (network-coordination-plan.md)
- S23 Network Dashboard (network-dashboard.md)

## Execution Protocol

### MUST
- Verify all Persona Bibles from S21 exist and are minimum 30KB
- Confirm all Account Plans from S22 exist and are minimum 15KB per persona × platform
- Validate Coordination Plan from S23 exists and is minimum 20KB
- Check Network Dashboard exists with performance metrics section
- Extract audience size and engagement data from dashboard/account plans
- Flag missing or incomplete inputs with specific requirements

### NEVER
- Proceed with missing Persona Bibles or Account Plans
- Skip Coordination Plan validation
- Ignore audience size and engagement data (critical for monetization readiness)
- Continue if Network Dashboard missing
- Accept incomplete upstream files

## Output
**Filename:** `input-validation-report.md`

**Format:** Markdown validation report

**Minimum Size:** 2KB

**Required Sections:**
1. Persona Bible Status (all present, size-validated)
2. Account Plan Status (all persona × platform combinations present)
3. Coordination Plan Status (PASS/FAIL)
4. Network Dashboard Status (performance data available)
5. Audience Data Summary (followers, engagement per persona)
6. Overall Readiness (READY / INCOMPLETE)

## Quality Gates
- [ ] All Persona Bibles and Account Plans accounted for
- [ ] Coordination Plan minimum 20KB verified
- [ ] Audience data extracted (followers, engagement per persona)
- [ ] Overall readiness decision is binary
- [ ] If INCOMPLETE, execution MUST halt

## Handoff
Feeds into: 0.2 Teaching Loader (if READY) or escalates to human (if INCOMPLETE)
