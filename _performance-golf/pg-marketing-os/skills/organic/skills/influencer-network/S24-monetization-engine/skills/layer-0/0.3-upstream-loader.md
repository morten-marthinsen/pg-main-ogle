# 0.3 Upstream Loader

## Purpose
Load S21 Persona Bibles, S22 Account Plans, S23 Coordination Plan, and audience performance data. Extract monetization-relevant context.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- All S21 Persona Bibles
- S21 Network Overview
- All S22 Account Strategy Plans
- S23 Network Coordination Plan
- S23 Network Dashboard (performance data)

## Execution Protocol

### MUST
- Load all Persona Bibles and extract: archetype, niche, authority areas, backstory credentials
- Extract audience data: follower counts, engagement rates, trust signals, content consistency
- Load Account Plans and extract: content calendar history, audience targeting, growth trajectory
- Load Coordination Plan and extract: cross-promotion patterns, network dynamics
- Create monetization readiness profile per persona (audience size, engagement, trust, authority)
- Identify audience segments per persona (from AIF if available)

### NEVER
- Proceed without loading all upstream files
- Skip audience performance data extraction
- Ignore authority areas from persona backstories
- Forget to assess trust signals (not just follower counts)
- Create generic summaries without monetization-relevant context

## Output
**Filename:** `upstream-context-summary.md`

**Format:** Markdown context summary

**Minimum Size:** 6KB

**Required Sections:**
1. Per-Persona Monetization Profiles (archetype, niche, authority, audience size, engagement)
2. Audience Performance Data (followers, engagement rates, trust signals, consistency)
3. Authority Areas (what each persona is qualified to teach/sell based on backstory)
4. Audience Segments (demographics, psychographics, purchasing power if known)
5. Network Dynamics (how personas interact — relevant for network-level products)
6. Monetization Readiness Summary (preliminary assessment per persona)

## Quality Gates
- [ ] All Persona Bibles and upstream plans loaded
- [ ] Audience performance data extracted (not just follower counts)
- [ ] Authority areas documented per persona
- [ ] Audience segments identified where available
- [ ] Preliminary readiness assessment per persona

## Handoff
Feeds into: All Layer 1 microskills (context for all monetization decisions)
