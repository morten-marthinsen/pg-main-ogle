# 0.3 Upstream Loader

## Purpose
Load S21 Persona Bibles, S22 Account Plans, and cross-promotion strategies. Extract coordination context needed for network orchestration.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- All S21 Persona Bibles
- S21 Network Overview
- All S22 Account Strategy Plans
- S22 Cross-Promotion Map
- S22 Network Sync Report

## Execution Protocol

### MUST
- Load all Persona Bibles and extract: voice, personality, content style, posting preferences
- Load Network Overview and extract: differentiation matrix, relationship map, brand firewall rules
- Load all Account Plans and extract: posting schedules, content calendars, growth tactics
- Load Cross-Promotion Map and extract: frequency limits, interaction types, authenticity guidelines
- Create coordination constraints from upstream data (what coordination must respect)
- Map network posting schedule (when each persona posts what)

### NEVER
- Proceed without loading all upstream files
- Skip relationship map from Network Overview (critical for coordination)
- Ignore posting schedules from Account Plans
- Forget cross-promotion constraints
- Create generic summaries without coordination-relevant details

## Output
**Filename:** `upstream-context-summary.md`

**Format:** Markdown context summary

**Minimum Size:** 6KB

**Required Sections:**
1. Per-Persona Coordination Profiles (voice, style, posting times, content types)
2. Network Differentiation Matrix (how personas differ — informs coordination)
3. Existing Posting Schedules (content calendar per persona × platform)
4. Cross-Promotion Baseline (frequency limits, interaction types from S22)
5. Brand Firewall Rules (what coordination must never reveal)
6. Coordination Constraints (synthesis of all upstream limitations)

## Quality Gates
- [ ] All Persona Bibles and Account Plans loaded
- [ ] Posting schedules extracted per persona
- [ ] Cross-promotion constraints documented
- [ ] Brand firewall rules explicit
- [ ] Coordination constraints synthesized (not just copy-paste)

## Handoff
Feeds into: All Layer 1 microskills (context for all coordination decisions)
