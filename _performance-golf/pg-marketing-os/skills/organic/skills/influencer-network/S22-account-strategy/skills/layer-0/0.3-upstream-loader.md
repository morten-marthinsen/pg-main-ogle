# 0.3 Upstream Loader

## Purpose
Load S21 Persona Bibles, Network Overview, and PSF (Platform Selection Framework). Extract persona-specific context that informs platform selection and account strategy decisions.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- All S21 Persona Bibles (persona-bible-[name].md)
- S21 Network Overview (network-overview.md)
- S03 Platform Selection Framework (PSF)
- S01 Audience Intelligence Framework (AIF) if available

## Execution Protocol

### MUST
- Load all Persona Bibles and extract key context: archetype, niche, voice, content style preferences
- Extract network differentiation matrix from Network Overview
- Load PSF platform priorities and audience presence data per platform
- Create per-persona summary highlighting platform fit signals (archetype, content format, audience location)
- Identify cross-promotion opportunities from Network Overview relationship map
- Extract brand firewall rules from Network Overview

### NEVER
- Proceed without loading all Persona Bibles
- Ignore Network Overview relationship map (critical for cross-promotion)
- Skip PSF audience data (needed for platform selection)
- Forget brand firewall rules
- Create generic persona summaries without platform fit signals

## Output
**Filename:** `upstream-context-summary.md`

**Format:** Markdown context summary

**Minimum Size:** 5KB

**Required Sections:**
1. Per-Persona Summaries (archetype, niche, voice, content preferences, platform fit signals)
2. Network Differentiation Matrix (how personas differ — informs strategy differentiation)
3. Platform Priorities from PSF (platforms, audience presence, strategic rationale)
4. Cross-Promotion Opportunities (from Network Overview relationship map)
5. Brand Firewall Rules (what personas must never reveal)

## Quality Gates
- [ ] All Persona Bibles loaded and summarized
- [ ] Platform fit signals extracted per persona (not just generic summaries)
- [ ] Network differentiation matrix included
- [ ] PSF platform priorities with audience data loaded
- [ ] Brand firewall rules documented

## Handoff
Feeds into: All Layer 1 microskills (context for all account strategy decisions)
