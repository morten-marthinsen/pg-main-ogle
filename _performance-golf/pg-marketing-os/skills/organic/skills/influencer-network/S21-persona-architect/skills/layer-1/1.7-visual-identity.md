# 1.7 Visual Identity

## Purpose
Define visual style for each persona: profile photo direction, color palette, typography, visual mood. NOT creating graphics — creating briefs for visual production.

## Model Assignment
**Model:** claude-opus-4-20250514

## Input Requirements
- Approved names and handles (from 1.6)
- Personality framework (from 1.5)
- Archetype assignments (from 1.2)
- Platform priorities (from upstream context)

## Execution Protocol

### MUST
- Define profile photo direction: style (professional headshot, candid, lifestyle, illustrated), setting, mood, clothing style
- Create color palette: 3-5 colors that reflect personality and niche
- Specify typography direction: font personality (modern, classic, bold, elegant), hierarchy approach
- Define overall visual mood: adjectives describing visual feel (polished, raw, warm, clinical, vibrant, minimalist)
- Ensure visual identity aligns with archetype and personality
- Consider platform norms (LinkedIn vs TikTok visual expectations)
- Differentiate visual identities across personas (avoid visual similarity)

### NEVER
- Create actual graphics (this is direction only)
- Use identical visual templates across personas
- Ignore platform visual conventions
- Create visual identity that contradicts personality
- Skip color psychology considerations

## Output
**Filename:** `visual-identity-briefs.md`

**Format:** Markdown visual direction document

**Minimum Size:** 4KB

**Required Sections:**
1. Per-Persona Visual Briefs (photo direction, color palette, typography, mood)
2. Color Psychology Rationale (why these colors for this persona)
3. Platform Visual Adaptation (how visual identity adapts per platform)
4. Visual Differentiation Matrix (how personas differ visually)
5. Production Handoff Notes (what designers need to know)

## Quality Gates
- [ ] Each persona has complete visual brief (photo, color, typography, mood)
- [ ] Color palettes include 3-5 specific colors with rationale
- [ ] Visual identity aligns with personality and archetype
- [ ] Visual differentiation across personas verified
- [ ] Platform adaptations considered
