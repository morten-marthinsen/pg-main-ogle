# 0.2 Teaching Loader

## Purpose
Load persona development teachings, archetype frameworks, and audience psychology principles. Provides theoretical foundation for creating believable, differentiated AI personas.

## Model Assignment
**Model:** claude-3-5-haiku-20241022

## Input Requirements
- Teaching library location (if specified)
- Archetype framework references
- Audience psychology frameworks
- Persona development best practices

## Execution Protocol

### MUST
- Load 7 archetype definitions: Educator, Curator, Storyteller, Provocateur, Entertainer, Connector, Analyst
- Load audience psychology principles: parasocial relationships, authority building, niche ownership
- Load persona differentiation frameworks: voice, visual identity, content style, backstory
- Load platform-specific persona considerations
- Create teaching summary that operationalizes frameworks for this specific project

### NEVER
- Use generic "personality types" without audience psychology grounding
- Load conflicting archetype definitions without resolving contradictions
- Skip platform-specific considerations
- Assume archetypes are self-explanatory without definitions

## Output
**Filename:** `teaching-summary.md`

**Format:** Markdown teaching reference

**Minimum Size:** 5KB

**Required Sections:**
1. 7 Archetype Definitions (purpose, traits, content approach, audience appeal)
2. Audience Psychology Principles (3+ frameworks)
3. Persona Differentiation Framework (how to create distinct personas)
4. Platform Considerations (archetype-platform fit)
5. Anti-Patterns (common persona development mistakes)

## Quality Gates
- [ ] All 7 archetypes defined with clear differentiation
- [ ] Audience psychology grounded in research/frameworks, not intuition
- [ ] Differentiation framework includes voice, visual, content, backstory
- [ ] Platform considerations address at least 3 major platforms
- [ ] Anti-patterns list includes minimum 5 common failures

## Handoff
Feeds into: 0.3 Upstream Loader and all Layer 1 microskills (teaching reference throughout)
