# 1.4 CTA Strategy

**Layer:** 1 (Strategy)
**Model:** Claude Sonnet
**Upstream:** 1.3 Hook Strategy
**Downstream:** 1.5 Hashtag Strategy

---

## PURPOSE

Map content_function to appropriate CTA type (Soft/Medium/Hard), define CTA placement strategy, and design action prompts that match content value and audience readiness. Ensure CTA doesn't break trust or feel jarring after educational/entertainment content.

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  content_function: "Educate" | "Entertain" | "Inspire" | "Convert" | "Nurture"
  conversion_goal: "From CBF extraction (awareness, consideration, conversion)"
  platform_analysis: "outputs/layer-1/1.1-platform-caption-analysis.md"
  structure_strategy: "outputs/layer-1/1.2-caption-structure-strategy.md"
  content_type: "Reel" | "Static Post" | "Carousel" | "Story"
```

## EXECUTION PROTOCOL

**MUST:**
1. Map **content_function to CTA intensity**:

   **SOFT CTA (Engagement-Focused):**
   - Content Functions: Entertain, Nurture
   - Goal: Comments, likes, relationship building
   - Examples: "Which one are you? 👇", "Tag someone who needs this", "What's your take?"

   **MEDIUM CTA (Value-Capture):**
   - Content Functions: Educate, Inspire
   - Goal: Saves, shares, follows
   - Examples: "Save this for later", "Share with someone who needs this", "Follow for more tips like this"

   **HARD CTA (Conversion-Focused):**
   - Content Functions: Convert
   - Goal: Link clicks, DMs, purchases
   - Examples: "Link in bio to get started", "DM me 'READY' for details", "Grab yours before they're gone"

2. Define **CTA placement**:
   - **End of Caption (Standard):** After body content, before hashtags
   - **Mid-Caption (Advanced):** For longer captions, CTA after list/value delivery, then additional context
   - **First Comment (Alternative):** For platforms that penalize links in captions

3. Design **CTA structure** (2 components):
   - **Primary Action (Required):** What you want user to do
   - **Engagement Prompt (Optional):** Secondary action for those not ready for primary

4. Ensure **value-to-ask ratio** is balanced:
   - Educational content (high value) → Can use Medium or Hard CTA
   - Entertainment content (moderate value) → Soft CTA only
   - Conversion content (low independent value) → Hard CTA but must deliver value first

5. Plan **CTA formatting**:
   - Spacing: 1-2 blank lines before CTA
   - Emoji: Optional for emphasis or engagement prompt
   - Length: 20-80 chars (concise, clear)

6. Generate CTA strategy document with 3-5 CTA options for Layer 2 testing

**NEVER:**
- Use Hard CTA after pure entertainment content (breaks trust)
- Skip CTA entirely (every caption needs action prompt)
- Use multiple competing CTAs ("Like, comment, share, save, follow, and DM me")
- Place CTA before value delivery (body → CTA, not CTA → body)
- Use vague CTAs ("Let me know what you think" when you want saves)

## CTA INTENSITY MAPPING

**Content Function:** Educate
**Value Delivered:** 3 nutrition myths debunked (high value)
**Audience Readiness:** Awareness stage (not ready for hard conversion)

**CTA Intensity:** MEDIUM
- Primary goal: Saves (indicates value, boosts algorithmic reach)
- Secondary goal: Follows (nurture for future conversion)
- Avoid: Hard CTA (too early in journey)

---

## CTA PLACEMENT

**Placement:** End of caption (standard placement)

**Structure:**
1. Body content delivers value (500-700 chars)
2. Blank line (visual separation)
3. CTA line 1: Save/share prompt (40-60 chars)
4. CTA line 2: Engagement prompt (optional, 30-50 chars)
5. Blank line + hashtag separator (• or —)
6. Hashtags (80-120 chars)

**Rationale:**
- End placement feels natural after value delivery
- Doesn't interrupt educational flow
- Standard placement = user expectation

## CTA OPTIONS (3 variants for Layer 2 testing)

### OPTION 1: SAVE-FOCUSED (Medium Intensity)
**Primary Action:** "Save this to remember next time you're meal planning. 🔖"
**Engagement Prompt:** "What myth surprised you most? 👇"

**Breakdown:**
- Primary: 63 chars, specific use case for save ("next time you're meal planning")
- Engagement: 37 chars, open-ended question, comment driver
- Total CTA: 100 chars

**Why This Works:**
- Gives concrete reason to save (not generic "save this")
- Engagement prompt maintains conversation
- Emoji signals action (🔖 = save, 👇 = comment below)

---

### OPTION 2: SHARE-FOCUSED (Medium Intensity)
**Primary Action:** "Send this to someone who's always asking for nutrition advice. 📲"
**Engagement Prompt:** (None — share is the action)

**Breakdown:**
- Primary: 68 chars, identifies specific person to share with
- No secondary prompt (share is enough friction)
- Total CTA: 68 chars

**Why This Works:**
- "Someone who's always asking" = easy to identify (vs vague "someone who needs this")
- DM shares are strong algorithmic signal
- Single clear action (not overwhelming)

---

### OPTION 3: FOLLOW-FOCUSED (Medium Intensity)
**Primary Action:** "Follow for science-backed nutrition tips that actually work. ✨"
**Engagement Prompt:** "Next up: Pre-workout nutrition myths. Stay tuned. 👀"

**Breakdown:**
- Primary: 66 chars, value promise for following (not just "follow me")
- Engagement: 57 chars, teases next content (retention)
- Total CTA: 123 chars

**Why This Works:**
- "Science-backed" + "actually work" = credibility + results (aligns with BVF)
- Tease next content = reason to follow now (not "I'll follow eventually")
- Emoji ✨ = value, 👀 = anticipation

## VALUE-TO-ASK RATIO

**Value Delivered:** HIGH
- 3 specific myths debunked
- Actionable nutrition guidance
- Science-backed (credibility)

**Ask Intensity:** MEDIUM (balanced)
- Not asking for purchase, DM, or personal info
- Save/share/follow = low friction
- Proportional to value delivered

**Balance Check:** ✅ PASS
- Educational content with Medium CTA = appropriate
- Could even support occasional Hard CTA ("Link in bio for meal plan") if value is exceptional

---

## CTA FORMATTING

**Spacing:**
```
[Last line of body content]

[Blank line]

[CTA Primary Action]
[CTA Engagement Prompt — optional]

[Blank line]
• (or —)
[Hashtags]
```

**Emoji Usage:**
- CTA Primary: 1 emoji (🔖📲✨💾)
- Engagement Prompt: 1 emoji (👇🤔💬)
- Total: 2 emojis in CTA section

**Length:**
- Single CTA: 40-80 chars
- CTA + Engagement: 80-150 chars total
- Never exceed 150 chars for CTA section (keeps focus)

---

## CTA SUCCESS CRITERIA

CTA is successful if:
1. **Matches Content Function:** Medium intensity for Educational content ✅
2. **Specific Action:** Not vague ("save this" > "let me know thoughts") ✅
3. **Proportional Ask:** Value delivered justifies the ask ✅
4. **Single Primary Action:** One clear thing you want user to do ✅
5. **Natural Transition:** Doesn't feel jarring after body content ✅

---

## PLATFORM-SPECIFIC CTA NOTES

**Instagram Reels:**
- "Follow" CTA performs well (users discovering via Explore)
- "Save" drives reach (algorithm boost)
- Avoid "Link in bio" for educational content (feels salesy)

**Instagram Carousels:**
- "Save" is king (swipe-through content = reference material)
- "Share" works for relatable/tag-worthy content
- "Swipe for more" in hook, save CTA at end

**Instagram Stories:**
- Use native features (polls, questions, sliders) as CTA
- "Swipe up" if >10K followers
- Time-sensitive CTAs work ("Next story in 5 seconds")
