**Layer:** 1 (Strategy)
**Model:** Claude Sonnet
**Upstream:** 1.2 Caption Structure Strategy

## PURPOSE

Select optimal hook types from Hook Library File (HLF) and adapt for caption format. Define how hook will work in text-only context (caption-first content like carousels) vs video-driven context (Reels where caption is secondary). Ensure hook delivers value BEFORE platform truncation point.

## EXECUTION PROTOCOL

**MUST:**
1. Load Hook Library File (HLF) — framework of proven hook types:
   - Question hooks ("Did you know...?", "What if I told you...?")
   - Stat/number hooks ("97% of people don't know...")
   - Contrarian hooks ("Everyone tells you X. Here's why they're wrong.")
   - Story hooks ("I used to believe X until...")
   - Listicle hooks ("3 things nobody tells you about...")
   - Pain/problem hooks ("Struggling with X? Here's why.")
   - Benefit/promise hooks ("How to X without Y")
2. Filter HLF hooks by content_function:
   - Educate → Stat, Listicle, Contrarian, Pain/Problem
   - Entertain → Story, Question, Contrarian
   - Inspire → Story, Benefit/Promise
   - Convert → Benefit/Promise, Pain/Problem
   - Nurture → Question, Story
3. Adapt hook for **text-only vs video-driven** context:
   - **Text-Only (Carousel, Static Post):** Hook must work independently, create curiosity to read on
   - **Video-Driven (Reel, TikTok):** Hook can reference video ("Watch till the end"), complements visual
4. Design hook to work **BEFORE truncation point**:
   - Instagram: Complete thought in 125 chars
   - TikTok: Complete thought in 70 chars
   - LinkedIn: Professional hook in 150 chars
   - X: Entire message in 280 chars (no truncation)
5. Select **3-5 hook types** to test in Layer 2 drafting
6. Define hook success criteria (curiosity gap, value delivery, platform fit)

**NEVER:**
- Use generic hooks ("Check this out!", "You need to see this")
- Create cliffhanger hooks that require "...more" tap (hook fails if user doesn't tap)
- Copy video script hooks verbatim for caption (different contexts)
- Ignore platform truncation point (Instagram hook at 200 chars = failure)
- Select only one hook type (need variety for Arena testing)

## SELECTED HOOK TYPES (5 types for Layer 2 testing)

### 1. CONTRARIAN HOOK
**HLF Pattern:** "Everyone tells you X. Here's why they're wrong."
**Adaptation for Caption:** Challenge common belief in first sentence

**Example Structure:**
"[Common belief] is actually hurting your results. ⚠️"
[68 chars with emoji]

**Why This Works:**
- Creates cognitive dissonance (stops scroll)
- Promises reveal of hidden truth
- Works before truncation point
- Complements video (video delivers the "why")

---

### 2. STAT/NUMBER HOOK
**HLF Pattern:** "X% of people don't know..."
**Adaptation for Caption:** Lead with surprising statistic

**Example Structure:**
"93% of [target audience] are making this mistake with [topic]. 😳"
[72 chars]

**Why This Works:**
- Specificity creates credibility
- High percentage = "this might be me"
- Emoji amplifies emotion
- Complete thought before truncation

---

### 3. LISTICLE HOOK
**HLF Pattern:** "X things nobody tells you about..."
**Adaptation for Caption:** Promise specific number of takeaways

**Example Structure:**
"3 myths about [topic] that are costing you [pain point]. 🔥"
[65 chars]

**Why This Works:**
- Odd number (3, 5, 7) performs better than even
- "Myths" signals contrarian value
- "Costing you" = pain amplification
- Sets expectation for list in body

---

### 4. PAIN/PROBLEM HOOK
**HLF Pattern:** "Struggling with X? Here's why."
**Adaptation for Caption:** Identify pain, promise solution

**Example Structure:**
"Not seeing [desired result]? It's not your fault. Here's why. 💡"
[70 chars]

**Why This Works:**
- Direct pain acknowledgment (empathy)
- "Not your fault" = removes shame, creates openness
- "Here's why" = promise of explanation
- Works as standalone or video complement

---

### 5. QUESTION HOOK
**HLF Pattern:** "What if I told you...?" or "Did you know...?"
**Adaptation for Caption:** Pose surprising question

**Example Structure:**
"What if [desired outcome] didn't require [common sacrifice]? 🤔"
[68 chars]

**Why This Works:**
- Open loop (brain seeks answer)
- Challenges assumption (doesn't require sacrifice)
- Emoji signals "think about this"
- Curiosity gap without cliffhanger

## TEXT-ONLY vs VIDEO-DRIVEN ADAPTATION

**This is a VIDEO-DRIVEN context (Reel):**

Hook strategy must account for:
1. **Video Hook (First 3 Seconds) Is Primary:** Caption hook is SECONDARY
2. **Caption Complements Video:** Shouldn't repeat video hook verbatim
3. **Caption Hook = Retention Aid:** Gives viewer reason to watch till end OR come back later (save)

**Caption Hook Techniques for Video Content:**
- Reference video: "The 3rd one surprised even me. 👀"
- Tease key insight: "If you only remember one thing from this video..."
- Save prompt in hook: "You'll want to save this for later. ⚡"

**For Text-Only Content (Carousel/Static):**
- Hook must work independently (no video to lean on)
- Cannot reference visual ("swipe to see" is CTA, not hook)
- Must create complete value/curiosity in text alone

---

## HOOK SUCCESS CRITERIA

Hook is successful if it achieves 2/3:
1. **Stops Scroll:** Surprises, challenges assumption, or promises specific value
2. **Works Before Truncation:** Complete thought in first 125 chars (Instagram)
3. **Aligns with Brand Voice:** Tone matches BVF (empowering, not aggressive; accessible, not dumbed-down)
