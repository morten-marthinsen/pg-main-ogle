# 2.1 Hook Drafting

**Layer:** 2 (Generative)
**Model:** Claude Opus
**Upstream:** 1.5 Hashtag Strategy (Layer 1 Complete)
**Downstream:** 2.2 Body Writing

---

## PURPOSE

Write caption hooks that stop scroll and deliver value BEFORE platform truncation point. Generate 8-10 hook variants using hook types from 1.3 strategy, ensuring each hook works as standalone thought in first 70-125 characters (platform-dependent).

---

## INPUT REQUIREMENTS

```yaml
required_inputs:
  hook_strategy: "outputs/layer-1/1.3-hook-strategy.md"
  platform_analysis: "outputs/layer-1/1.1-platform-caption-analysis.md"
  truncation_point: "Instagram 125 chars | TikTok 70 chars | LinkedIn 150 chars | X 280 chars"
  structure_strategy: "outputs/layer-1/1.2-caption-structure-strategy.md"
  bvf: "Brand Voice File from CBF extraction"
  teaching_references: "outputs/layer-0/0.2-teaching-references.md"
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Load **5 hook types** from 1.3 hook strategy
2. For each hook type, generate **2 variants** = 10 total hooks
3. Each hook MUST:
   - Deliver complete thought BEFORE truncation point (Instagram 125 chars, TikTok 70 chars)
   - Stop scroll via: surprise, specificity, curiosity gap, or immediate value
   - Align with BVF voice (load voice descriptor and anti-voice)
   - Work in platform context (text-only vs video-driven)
4. Apply **teaching principles** from 0.2:
   - Practical Value (Berger) → Promise specific takeaway
   - Emotion (Berger) → Use power words, high-arousal emotion
   - Social Currency (Berger) → Position reader as insider
   - Clarity (Miller) → Clear value in 5 seconds
5. Format each hook with:
   - Character count (verify < truncation point)
   - Hook type label
   - Rationale (why this hook works)
6. Score each hook on 3 criteria (1-10 scale):
   - Scroll-Stop Power (surprise, curiosity, boldness)
   - Truncation Safety (complete thought before "...more")
   - Voice Alignment (BVF match)
7. Select top 5 hooks for body writing in 2.2

**NEVER:**
- Write hooks that require "...more" tap to make sense (cliffhanger failure)
- Exceed platform truncation point (Instagram >125 chars, TikTok >70 chars)
- Use generic hooks ("You won't believe this!", "This is important")
- Ignore BVF voice (formal hook for casual brand, aggressive hook for empowering brand)
- Copy specimen hooks verbatim (adapt pattern, not text)
