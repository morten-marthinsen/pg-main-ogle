# 2.3 CTA Integration

**Layer:** 2 (Generative)
**Model:** Claude Opus
**Upstream:** 2.2 Body Writing
**Downstream:** 2.4 Platform Formatting

---

## PURPOSE

Integrate CTA from 1.4 strategy into caption body draft. Ensure CTA matches content_function intensity (Soft/Medium/Hard), feels natural after value delivery, and includes proper spacing/formatting. Select appropriate CTA from 3 options designed in Layer 1.

---

## INPUT REQUIREMENTS

```yaml

  body_draft: "outputs/layer-2/2.2-body-draft.md" (PRE-ARENA)

  cta_intensity: "Soft" | "Medium" | "Hard"
  structure_strategy: "outputs/layer-1/1.2-caption-structure-strategy.md"
```

---

## EXECUTION PROTOCOL

**MUST:**
1. Load **body draft** from 2.2 (hook + body, no CTA yet)
2. Load **3 CTA options** from 1.4 strategy
3. Select **1 CTA** that best matches:
   - Content function (Educate → Medium CTA)
   - Value delivered in body (high value = can ask more)
   - Platform norms (Instagram saves, TikTok engagement, LinkedIn thought leadership)
4. Format **CTA integration**:
   - Add 1-2 blank lines before CTA (visual separation from body)
   - CTA line 1: Primary action (40-80 chars)
   - CTA line 2 (optional): Engagement prompt (30-50 chars)
   - Total CTA section: 50-150 chars
5. Verify **CTA doesn't break trust**:
   - If body is educational/empowering, CTA should be Medium (save/share)
   - If body is conversion-focused, CTA can be Hard (link/DM)
   - If body is entertainment, CTA should be Soft (comment/engage)
6. Apply **CTA emoji** (1-2 max):
   - Save emoji: 🔖💾
   - Share emoji: 📲💬
   - Follow emoji: ✨🎯
   - Engagement emoji: 👇🤔
7. Generate **complete caption** (hook + body + CTA)
8. Update label from "PRE-ARENA" to "PRE-ARENA — CTA INTEGRATED"
9. Verify total character count still within platform limits

**NEVER:**
- Use Hard CTA after pure entertainment/nurture content (trust break)
- Skip CTA (every caption needs action prompt)
- Use multiple competing CTAs ("like, comment, share, save, follow" = overwhelming)
- Place CTA mid-body (interrupts value delivery)
- Copy CTA verbatim from specimens without adapting to this content

## CTA ANALYSIS

**CTA Selected:** Option 1 — Save-Focused (Medium Intensity)

**CTA Breakdown:**
- **Primary Action:** "Save this to remember next time you're meal planning. 🔖"
  - Length: 63 chars
  - Function: Drives saves (high algorithmic value on Instagram)
  - Specificity: "Next time you're meal planning" (concrete use case > generic "save this")
  - Emoji: 🔖 bookmark (signals save action)

- **Engagement Prompt:** "What's your go-to high-protein meal? Drop it below! 👇"
  - Length: 58 chars
  - Function: Drives comments (engagement signal)
  - Type: Open-ended question (not yes/no)
  - Emoji: 👇 points down to comments

**Total CTA Length:** 121 chars (within 50-150 target ✅)

**Spacing:** 2 blank lines before CTA (visual separation ✅)

---

## CTA INTENSITY VERIFICATION

**Content Value Delivered:** HIGH
- Specific protein targets (0.8-1g per lb)
- Clear explanation of why (muscle loss, metabolism)
- Actionable meal examples with gram counts
- Outcome promise (better energy in 2 weeks)

**CTA Intensity:** MEDIUM (Save + Engage)
- NOT asking for purchase, DM, or link click
- Save = low friction, high user value (can reference later)
- Engagement prompt = optional (doesn't feel pushy)

**Value-to-Ask Ratio:** ✅ BALANCED
- High educational value justifies Medium CTA
- Could even support occasional Hard CTA ("Link in bio for meal plan") but Medium is safer for pure educational content

---

## TRUST CHECK

**Trust Sequence:**
1. Hook delivers (protein stat → explanation ✅)
2. Body provides value (actionable targets + examples ✅)
3. CTA asks proportional action (save for later ✅)

**Trust Break Risks:**
- ❌ Would break trust: "DM me to buy my protein guide" (Hard CTA after free education)
- ❌ Would break trust: No CTA (missed opportunity for saves/engagement)
- ✅ This CTA: Proportional ask, enhances user experience (save for reference)

---
