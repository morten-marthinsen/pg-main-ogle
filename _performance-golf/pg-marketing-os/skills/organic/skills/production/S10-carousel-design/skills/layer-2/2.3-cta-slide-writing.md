# 2.3 CTA Slide Writing

## Purpose
Write final CTA slide that prompts save/share/follow and drives engagement.

## Input Requirements
- Carousel brief (content_function)
- Save optimization strategy
- Platform requirements
- Slide architecture
- CBF voice and brand

## Execution Protocol

**MUST:**
- Write final slide (Slide N) with 3 components:
  1. **Save/Share Prompt:** Explicit CTA ("Save this for later", "Share with someone who needs this")
  2. **Follow CTA:** Brand/account follow prompt with value proposition
  3. **Engagement Question:** Comment driver aligned with content_function
- Match CTA intensity to content_function:
  - **Educational:** "Save for reference" + "What's your biggest challenge with [topic]?"
  - **Promotional:** "Want more?" + "Follow for [value]" + "Ready to [action]?"
  - **Engagement:** "Tag someone" + "Share your experience in comments"
  - **Thought Leadership:** "Save this framework" + "How do you approach [topic]?"
- Design visual direction for CTA slide:
  - High contrast background
  - Clear hierarchy (save > follow > engage)
  - Optional brand element (logo/tagline)
- Generate 3 CTA slide variations
- Label all PRE-ARENA
- Respect character limits

**NEVER:**
- Skip save CTA (critical for algorithm)
- Use weak engagement questions ("What do you think?")
- Cram too many CTAs (dilutes action)
- Exceed character limits
- Forget PRE-ARENA label
- Make CTA slide text-heavy

## Output
**Filename:** `cta-slide-PRE-ARENA.yaml`

**Format:**
```yaml
variation_count: 3
variations:
  - version_id: "CTA-V1"
    save_prompt: string
    follow_cta: string
    engagement_question: string
    visual_direction:
      background: string
      hierarchy: [ordered list]
      brand_element: string
      color_scheme: [colors]
    cta_intensity: "soft | medium | strong"
    expected_action: "save | share | comment | follow"
  - version_id: "CTA-V2"
    [...same structure]
content_function_alignment: string
character_count_check:
  - version_id: string
    total_chars: number
    within_limit: true/false
timestamp: ISO8601
```

**Minimum size:** 1.5KB

## Quality Gates
- Minimum 3 CTA variations generated
- All include save prompt (non-negotiable)
- Engagement questions are specific (not generic)
- All within character limits
- Visual direction specified
- All labeled PRE-ARENA

## Handoff
Provides CTA slides to 2.5.1 Arena Submission as final carousel component.
