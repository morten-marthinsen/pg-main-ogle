# 2.5.1 Arena Submission

## Purpose
Submit complete carousel drafts to 7 organic content personas for competitive evaluation.

## Input Requirements
- Cover slide variations (PRE-ARENA)
- Body slides (PRE-ARENA)
- CTA slide variations (PRE-ARENA)
- Visual direction
- All Layer 0 and Layer 1 context

## Execution Protocol

**MUST:**
- Invoke 7 organic content personas:
  1. **The Educator:** Clarity and teaching effectiveness
  2. **The Viral Engineer:** Share/save mechanics optimization
  3. **The Platform Native:** Platform-specific best practices
  4. **The Storyteller:** Narrative flow and emotional engagement
  5. **The Designer:** Visual impact and screenshot-worthiness
  6. **The Strategist:** Alignment with content pillars and goals
  7. **The Audience Advocate:** Actual value to target audience
- Each persona generates COMPLETE carousel (all slides: cover + body + CTA)
- Each persona scores on 4 dimensions:
  - **Hook Strength (30%):** Scroll-stop + curiosity gap power
  - **Save-Worthiness (30%):** Reference value + screenshot utility
  - **Platform Fit (20%):** Technical compliance + platform aesthetic
  - **Engagement Potential (20%):** Comment/share/action drivers
- Document each persona's complete carousel with rationale
- Calculate composite scores (0-10 scale)
- Rank all 7 carousels by total score
- Create checkpoint: `ARENA_ROUND1_COMPLETE`

**NEVER:**
- Skip any of the 7 personas
- Allow personas to generate partial carousels
- Skip scoring rationale
- Proceed without complete round 1 submissions

## Output
**Filename:** `arena-round1-submissions.yaml`

**Format:**
```yaml
round: 1
persona_count: 7
submissions:
  - persona: "The Educator"
    complete_carousel:
      cover_slide: {headline, subhead, visual_direction}
      body_slides: [{slide_number, headline, supporting_lines}...]
      cta_slide: {save_prompt, follow_cta, engagement_q}
    scoring:
      hook_strength: 0-10 (weight 30%)
      save_worthiness: 0-10 (weight 30%)
      platform_fit: 0-10 (weight 20%)
      engagement_potential: 0-10 (weight 20%)
      weighted_total: 0-10
    rationale: string (why these choices)
  - persona: "The Viral Engineer"
    [...same structure for all 7]
rankings:
  - rank: 1
    persona: string
    score: number
  [...ranks 2-7]
checkpoint: "ARENA_ROUND1_COMPLETE"
timestamp: ISO8601
```

**Minimum size:** 8KB (comprehensive carousels from 7 personas)

## Quality Gates
- All 7 personas submitted
- Each submission is COMPLETE carousel (not fragments)
- All 4 scoring dimensions documented per persona
- Rankings calculated
- Checkpoint created

## Handoff
Provides 7 complete carousels to 2.5.2 Adversarial Critique for competitive evaluation.
