# 2.5.3 Synthesis

## Purpose
Synthesize the strongest elements from all 7 personas across 3 rounds into hybrid script options. Human selects the final version.

---

## Input Requirements

From Layer 2.5:
- `[script-id]-arena-round-1.yaml`
- `[script-id]-arena-round-2.yaml`
- `[script-id]-arena-round-3.yaml`

---

## Execution Protocol

### Step 1: Analyze Best Elements Across Rounds

Identify the strongest elements from each persona:

```yaml
best_elements:
  hook:
    volume_machine_r2: "Immediate, punchy hook with zero fat"
    value_architect_r3: "Hook that promises specific tactical value"
    virality_engineer_r1: "Pattern interrupt visual hook"

  body:
    community_builder_r2: "Conversational, inclusive language"
    storyteller_r3: "Narrative thread connecting sections"
    value_architect_r2: "Dense tactical value"

  cta:
    algorithm_hacker_r3: "CTA that triggers save behavior"
    brand_purist_r2: "Elegant, brand-aligned CTA"

  visual_direction:
    virality_engineer_r2: "Fast-cut pacing with pattern interrupts"
    brand_purist_r3: "Polished visual aesthetic"
```

---

### Step 2: Create Hybrid Variants

Build 3-5 hybrid scripts pulling from different persona combinations:

```yaml
hybrid_variants:
  hybrid_a_volume_value:
    name: "The Speed-Value Hybrid"
    composition:
      hook: "Volume Machine R2 (speed)"
      body: "Value Architect R3 (depth) + Virality Engineer R2 (pacing)"
      cta: "Algorithm Hacker R3 (save trigger)"

    script:
      hook: "[Combined hook text]"
      body: "[Combined body]"
      cta: "[Combined CTA]"

    rationale: |
      Combines Volume Machine's urgency with Value Architect's depth.
      Fast-paced but substantive. Algorithm-optimized CTA.

    predicted_performance:
      retention: 8/10
      engagement: 9/10
      brand_alignment: 7/10

  hybrid_b_viral_story:
    name: "The Viral Narrative"
    composition:
      hook: "Virality Engineer R1 (pattern interrupt)"
      body: "Storyteller R3 (narrative) + Community Builder R2 (inclusive)"
      cta: "Community Builder R3 (engagement)"

    script:
      hook: "[Combined hook text]"
      body: "[Combined body]"
      cta: "[Combined CTA]"

    rationale: |
      Viral mechanics wrapped in story. Human connection focus.

    predicted_performance:
      retention: 9/10
      engagement: 10/10
      brand_alignment: 8/10

  hybrid_c_premium:
    name: "The Premium Play"
    composition:
      hook: "Brand Purist R3 (elegant)"
      body: "Value Architect R3 (depth) + Brand Purist R2 (polish)"
      cta: "Brand Purist R2 (brand-aligned)"

    script:
      hook: "[Combined hook text]"
      body: "[Combined body]"
      cta: "[Combined CTA]"

    rationale: |
      Premium positioning. Quality over virality. Brand-first.

    predicted_performance:
      retention: 7/10
      engagement: 7/10
      brand_alignment: 10/10

  # ... 3-5 total hybrids
```

---

### Step 3: Present for Human Selection

Package hybrids for human review:

```markdown
# ARENA RESULTS: [Script Title]

## Summary
3 rounds complete. 7 personas competed. Synthesized into [N] hybrid options.

---

## Hybrid A: The Speed-Value Hybrid

### Full Script
[Complete script]

### Composition
- Hook: Volume Machine (speed + urgency)
- Body: Value Architect (depth) + Virality Engineer (pacing)
- CTA: Algorithm Hacker (save trigger)

### Predicted Performance
- Retention: 8/10
- Engagement: 9/10
- Brand Alignment: 7/10

### Best For
Fast-moving platforms (TikTok, IG Reels) where speed + value wins.

---

## Hybrid B: The Viral Narrative

[Repeat structure]

---

## Hybrid C: The Premium Play

[Repeat structure]

---

## Recommendation
Based on platform ([platform]), content function ([function]), and brand voice:

**Recommended:** Hybrid [X]

**Reasoning:** [Why this hybrid best fits strategic requirements]
```

---

### Step 4: Capture Human Selection

Document which hybrid the human selects:

```yaml
human_selection:
  selected_hybrid: "Hybrid A"
  selection_rationale: |
    [Why human chose this option]
  modifications_requested: |
    [Any adjustments human wants]
```
