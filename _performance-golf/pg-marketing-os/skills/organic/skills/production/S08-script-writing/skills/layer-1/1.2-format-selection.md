# 1.2 Format Selection

## Purpose
Determine the specific format for each script request based on platform, content function, and campaign strategy.

---

## Input Requirements

From Layer 0:
- `0.1-input-validation.yaml` (script requests)

From Layer 1:
- `1.1-platform-analysis.yaml` (platform specs)

---

## Execution Protocol

### Step 1: Map Requests to Formats

For each script request, select the optimal format:

```yaml
format_mapping:
  - script_id: "script_1"
    platform: "TikTok"
    request_format: "TikTok"  # from input
    selected_format: "TikTok-Short-Form-15-30s"
    structure_type: "Hook-Value-CTA"
    rationale: "[Why this format for this request]"

  - script_id: "script_2"
    platform: "Instagram"
    request_format: "Reel"
    selected_format: "Instagram-Reel-20-45s"
    structure_type: "Hook-Context-Value-CTA"
    rationale: "[...]"
```

---

### Step 2: Define Format Specifications

For each selected format, define precise specifications:

```yaml
format_specs:
  tiktok_short_15_30s:
    target_length: "15-30 seconds"
    section_breakdown:
      hook: "0-1s"
      value: "1-27s"
      cta: "27-30s"
    pattern_interrupt_frequency: "Every 3-5 seconds"
    visual_style: "Fast cuts, trending sounds"

  instagram_reel_20_45s:
    target_length: "20-45 seconds"
    section_breakdown:
      hook: "0-3s"
      context: "3-8s"
      value: "8-40s"
      cta: "40-45s"
    pattern_interrupt_frequency: "Every 5-8 seconds"
    visual_style: "Polished, save-worthy"
```

---

### Step 3: Validate Format-Content Fit

Ensure format matches content function:

```yaml
content_fit_validation:
  - script_id: "script_1"
    content_function: "Awareness"
    selected_format: "TikTok-Short-15-30s"
    fit_score: 9/10
    notes: "Short format perfect for awareness, quick value delivery"

  - script_id: "script_2"
    content_function: "Conversion"
    selected_format: "Instagram-Reel-20-45s"
    fit_score: 7/10
    notes: "Longer format allows for conversion messaging, save CTA"
```
