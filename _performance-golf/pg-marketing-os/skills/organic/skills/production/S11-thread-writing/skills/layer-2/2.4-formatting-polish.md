# 2.4 Formatting Polish

## Purpose
Apply platform-specific formatting, numbering, line breaks, and emphasis to complete thread before Arena.

---

## Input Requirements

From Layer 1:
- `1.1-platform-analysis.yaml` (platform formatting requirements)

From Layer 2:
- `2.1-hook-tweet-draft.yaml` (tweet 1)
- `2.2-body-tweets-draft.yaml` (tweets 2 to N-1)
- `2.3-cta-tweet-draft.yaml` (tweet N)

---

## Execution Protocol

### Step 1: Apply Numbering

Add tweet numbers based on platform:

```yaml
numbering_application:
  platform: "[X_TWITTER or LINKEDIN]"

  numbering_style:
    x_twitter:
      recommended: "1/" or "1." at start of each tweet
      total_count_shown: [Yes — "1/12"]

    linkedin:
      recommended: "Optional — often just bold headers"
      approach: "[Numbered OR headers OR neither]"

  applied_numbering:
    - tweet: 1
      prefix: "[1/12 or 1. or nothing]"

    - tweet: 2
      prefix: "[2/12 or 2. or nothing]"

    # ... for all tweets
```

### Step 2: Apply Line Breaks

Add visual breathing room:

```yaml
line_break_rules:
  after_hook_sentence: "Single line break"
  before_key_points: "Single line break"
  for_emphasis: "Single line on its own"

formatted_tweets:
  - tweet_number: 1
    original: |
      [Original text]

    formatted: |
      [Text with line breaks]

  - tweet_number: 2
    original: |
      [Original text]

    formatted: |
      [Text with line breaks]

  # ... for all tweets
```

### Step 3: Apply Emphasis

Add emphasis where appropriate:

```yaml
emphasis_application:
  all_caps:
    usage: "Sparingly for key words"
    examples:
      - tweet: [N]
        word: "[WORD]"
        reasoning: "[Why emphasized]"

  quotation_marks:
    usage: "For quotes and key terms"
    examples:
      - tweet: [N]
        phrase: '["Phrase"]'
        reasoning: "[Why emphasized]"

  emojis:
    platform: "[X_TWITTER or LINKEDIN]"

    x_twitter:
      recommendation: "Minimal or none"
      usage: "[If used, where]"

    linkedin:
      recommendation: "Strategic use, one per point max"
      usage: "[If used, where]"
```

### Step 4: Media Recommendations

Identify where visuals enhance thread:

```yaml
media_recommendations:
  platform: "[X_TWITTER or LINKEDIN]"

  recommended_media:
    - tweet: [N]
      media_type: "[Screenshot/Diagram/Chart/Photo]"
      description: "[What to show]"
      reasoning: "[Why this enhances]"

    - tweet: [N]
      media_type: "[Type]"
      description: "[What to show]"
      reasoning: "[Why this enhances]"

  total_media_pieces: [0-N]
```

### Step 5: Final Thread Assembly

Compile complete formatted thread:

```yaml
formatted_thread:
  platform: "[X_TWITTER or LINKEDIN]"
  total_tweets: [N]

  thread_copy:
    tweet_1: |
      [1/12]
      [Formatted hook with breaks and emphasis]

    tweet_2: |
      [2/12]
      [Formatted tweet with breaks and emphasis]

    # ... all tweets

    tweet_N: |
      [N/N]
      [Formatted CTA with breaks and emphasis]

  ready_to_post_format: |
    [All tweets concatenated, ready to copy-paste]
```
