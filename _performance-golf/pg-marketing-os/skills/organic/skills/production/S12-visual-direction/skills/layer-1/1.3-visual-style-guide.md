# Microskill 1.3: Visual Style Guide
Skill: S12 Visual Direction
Layer: 1
Model: claude-sonnet-4.5
Version: 1.0

## Purpose

Create a cohesive visual style guide for this specific content piece that aligns with brand guidelines while adapting to content type, platform, and strategic goals. This guide ensures all visual assets (thumbnail, video frames, carousel slides) feel like they belong to the same piece.

## Input

**Required files:**
- `s12-0.4-brand-guidelines.yaml` — brand baseline
- `s12-1.1-platform-analysis.yaml` — platform visual trends
- `s12-1.2-thumbnail-strategy.md` — selected thumbnail concept
- Content draft from S08-S11 — content tone and structure
- `campaign-brief-foundation.yaml` — campaign goals and audience

## Process

### MUST
1. **Define visual tone**
   - Extract brand voice from CBF (professional, playful, premium, etc.)
   - Extract content tone from draft (educational, inspirational, urgent, etc.)
   - Synthesize into visual tone for this piece:
     - Professional + Educational = Clean, structured, data-friendly
     - Playful + Inspirational = Colorful, dynamic, energetic
     - Premium + Urgent = Minimal, high-contrast, bold
   - Document 3-5 adjectives that describe the visual tone

2. **Establish color approach**
   - Start with brand color palette from brand guidelines
   - Adapt for content tone and platform:
     - If content is urgent/action-oriented: increase contrast, use accent colors
     - If content is calm/educational: use softer tones, more background space
     - If platform is mobile-first (TikTok, Instagram): ensure colors pop on small screen
   - Define primary, secondary, and accent color usage for this piece
   - Specify background treatment (solid, gradient, image overlay)

3. **Define typography system**
   - Start with brand typography from brand guidelines
   - Adapt for platform readability:
     - Mobile-first platforms: increase font sizes by 20-30%
     - Desktop-focused platforms: can use smaller, more refined typography
   - Define hierarchy for this piece:
     - Headline: font, size, weight, color
     - Subhead: font, size, weight, color
     - Body: font, size, weight, color (if applicable)
   - Specify text overlay treatment (solid, outlined, shadowed, background box)

4. **Establish imagery style**
   - Define photography/imagery approach for this piece:
     - Will it use faces? (authority/connection)
     - Will it use product shots? (tangibility)
     - Will it use lifestyle imagery? (aspiration)
     - Will it use abstract/graphic elements? (modern/clean)
   - Match imagery style to content promise and brand visual guidelines
   - Document mood: lighting, composition, color treatment

5. **Define graphic elements**
   - Specify any recurring graphic elements:
     - Dividers, borders, frames
     - Icons, shapes, patterns
     - Overlays, gradients, textures
   - Ensure consistency across all assets (thumbnail, frames, slides)
   - Keep brand-aligned (use brand shapes/patterns if specified)

6. **Create visual consistency rules**
   - Document specific rules for cohesion:
     - "All text overlays use 8px outline for contrast"
     - "All product shots on white background"
     - "All transitions use 15° gradient from primary to secondary color"
   - These rules ensure designer creates visually unified asset set

### NEVER
- Contradict brand guidelines without documented rationale
- Create style guide that works for thumbnail but not video frames/slides
- Define style so rigidly it prevents Arena variations
- Skip tone-to-visual translation (tone is not just colors)
- Ignore platform-specific adaptations

    color_adherence: "100% — all colors from brand palette"
    typography_adherence: "100% — Montserrat + Open Sans per brand guidelines"

    deviations: "None — style guide fully aligned with brand guidelines"
