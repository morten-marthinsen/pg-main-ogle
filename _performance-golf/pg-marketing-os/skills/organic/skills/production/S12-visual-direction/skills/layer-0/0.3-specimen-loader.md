# Microskill 0.3: Specimen Loader
Skill: S12 Visual Direction
Layer: 0
Model: claude-haiku-4
Version: 1.0

## Purpose

Load high-performing thumbnail and visual specimens for reference. These specimens provide concrete examples of effective visual direction across platforms and verticals.

## Input

**Required gate:** `GATE_0_INPUT_VALID`

**Required context:** Platform and content type from 0.1 validation report

Specimen files to load:
- `skills/organic/specimens/thumbnails/youtube/` — YouTube thumbnail examples
- `skills/organic/specimens/thumbnails/instagram/` — Instagram visual examples
- `skills/organic/specimens/thumbnails/tiktok/` — TikTok visual examples
- `skills/organic/specimens/carousels/` — Carousel slide examples
- `skills/organic/specimens/visual-patterns/` — Proven visual patterns by vertical

## Process

### MUST
1. **Load platform-specific specimens**
   - Read specimen files matching target platform from validation report
   - For each specimen, extract: visual composition, color palette, text overlay approach, performance context
   - Prioritize specimens from same vertical as current project

2. **Load content-type specimens**
   - If content type = video: prioritize thumbnail specimens
   - If content type = carousel: prioritize carousel slide specimens
   - If content type = static_post: prioritize feed post specimens

3. **Extract visual patterns**
   - Identify recurring patterns across high-performing specimens:
     - Composition approaches (rule of thirds, centered, asymmetric)
     - Color combinations (high contrast, monochrome, brand-aligned)
     - Text overlay styles (bold headline, minimal text, no text)
     - Imagery types (faces, products, abstract, lifestyle)

4. **Create platform + vertical matrix**
   - Organize specimens by: platform × vertical
   - Note which patterns perform well in target platform + vertical combination
   - Document any vertical-specific visual conventions

5. **Flag specimen gaps**
   - If target platform + vertical combination has < 3 specimens, flag gap
   - Note gap in output file for human awareness
   - Proceed with best-available specimens from adjacent categories

### NEVER
- Load specimens from only one platform (cross-reference for patterns)
- Skip performance context (WHY a visual worked matters as much as WHAT it looked like)
- Treat specimens as templates to copy (they're references for principles, not templates)
- Proceed without loading any specimens
- Ignore vertical differences in visual conventions
