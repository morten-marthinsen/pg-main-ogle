# Microskill 4.1: Output Assembler
Skill: S12 Visual Direction
Layer: 4
Model: claude-sonnet-4.5
Version: 1.0

## Purpose

Assemble the complete visual direction package with Arena-selected thumbnail, all asset briefs, and designer handoff documentation.

## Input

**Required gate:** `GATE_2.5_ARENA_COMPLETE`

**Required files:**
- `s12-2.5.3-synthesis-results.yaml` — Arena selection
- `s12-2.2-frame-direction.yaml` — video frames
- `s12-2.3-carousel-visuals.yaml` — carousel (if applicable)
- `s12-2.4-asset-specs.yaml` — additional assets
- All Layer 1 strategy files

## Process

### MUST
1. **Verify Arena selection is incorporated**
   - Read selected concept from synthesis results
   - Ensure final thumbnail brief uses selected concept (not pre-Arena draft)
   - Include any human modifications requested

2. **Compile all asset briefs**
   - Thumbnail (Arena-selected)
   - Video frames (if video content)
   - Carousel slides (if carousel content)
   - Additional assets (quote cards, end screens, etc.)

3. **Verify brand consistency**
   - Cross-check all assets against brand guidelines
   - Ensure color palette consistency
   - Ensure typography consistency
   - Ensure visual style consistency

4. **Create designer handoff document**
   - All asset briefs in one package
   - Production priorities (critical vs. optional)
   - File naming conventions
   - Approval criteria
   - Timeline expectations

5. **Package for S14 consumption**
   - Structured YAML format
   - All required schema fields
   - References to source files

### NEVER
- Use pre-Arena thumbnail brief (must use Arena selection)
- Package incomplete asset set
- Skip brand consistency verification
- Omit production priorities
