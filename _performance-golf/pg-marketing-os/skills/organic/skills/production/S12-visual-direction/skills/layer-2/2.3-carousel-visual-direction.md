# Microskill 2.3: Carousel Visual Direction
Skill: S12 Visual Direction
Layer: 2
Model: claude-opus-4.6
Version: 1.0

## Purpose

Create slide-by-slide visual direction for carousel content. Each slide must work independently AND flow as a sequence.

## Input

**Required files:**
- Carousel copy from S11
- `s12-1.3-style-guide.yaml`
- `s12-1.4-color-typography.yaml`

## Process

### MUST
1. **Map copy to slides**
   - Read carousel copy, identify natural slide breaks
   - 1 core idea per slide (avoid text overload)

2. **Design slide-by-slide**
   - Slide 1 (hook): maximum scroll-stop
   - Slides 2-N (body): progressive disclosure
   - Final slide (CTA): clear action
   - Each: composition, text, visuals, colors

3. **Ensure visual progression**
   - Consistent style across slides
   - Visual flow (color progression, layout variations)
   - Brand consistency maintained

4. **Specify platform adaptations**
   - Instagram: 1080×1080, account for UI safe zones
   - LinkedIn: 1200×627, professional tone emphasis

### NEVER
- Vary style wildly between slides
- Overload slides with text (mobile readability)
- Skip safe zone specs (Instagram UI obscures top/bottom)

## Output

**File:** `s12-2.3-carousel-visuals.yaml`

```yaml
carousel_visual_direction:
  content_id: "retirement-40-carousel"
  platform: instagram
  slide_count: 10

  slides:
    - slide: 1
      type: "Hook"
      copy: "Most people's retirement plan... [hook text]"
      composition: "Centered bold text, minimal visual"
      typography: "72px Montserrat Bold"
      background: "#1E40AF gradient to #10B981"
      text_color: "#FFFFFF"
      safe_zone: "80px top/bottom (Instagram UI)"

    - slide: 2
      type: "Problem"
      copy: "The average retirement age keeps rising..."
      composition: "Data visualization + supporting text"
      visual: "Line graph showing rising retirement age"
      typography: "H1: 64px, body: 28px"
      # [... more slides ...]
```
