# Microskill 2.4: Creative Asset Specs
Skill: S12 Visual Direction
Layer: 2
Model: claude-opus-4.6
Version: 1.0

## Purpose

Specify any additional creative assets needed — quote cards, teaser graphics, profile images, channel art, promotional graphics.

## Input

**Required files:**
- Content strategy from S04-S07
- `s12-1.3-style-guide.yaml`
- `s12-1.4-color-typography.yaml`

## Process

### MUST
1. **Identify required assets**
   - Beyond thumbnail/frames/carousel: what else is needed?
   - Quote cards (for social sharing)
   - Profile/cover images (if channel branding)
   - Teaser graphics (announcement posts)
   - End screens (YouTube)

2. **Specify each asset type**
   - Purpose and usage context
   - Dimensions and format
   - Visual direction (composition, colors, typography)
   - Consistency with main assets

3. **Prioritize by necessity**
   - Critical: assets blocking content launch
   - Important: assets enhancing performance
   - Optional: nice-to-have enhancements

### NEVER
- Skip asset specs if content strategy calls for them
- Contradict established visual style
- Forget platform-specific dimensions

## Output

**File:** `s12-2.4-asset-specs.yaml`

```yaml
creative_asset_specs:
  content_id: "retirement-40-3bucket-video"

  assets:
    - asset_id: "A01-quote-card"
      type: "Social quote card"
      purpose: "Share key insight on Instagram/Twitter"
      dimensions: "1080x1080px (Instagram)"
      quote_text: "Most people retire at 67. You can retire at 40."
      composition: "Centered quote, attribution below"
      typography: "48px Montserrat Bold"
      background: "Brand gradient (#1E40AF to #10B981)"
      priority: "Important"

    - asset_id: "A02-youtube-endscreen"
      type: "YouTube end screen"
      purpose: "Subscribe prompt + next video links"
      dimensions: "1920x1080px"
      elements: "Logo, Subscribe button, 2 video thumbnails"
      layout: "Logo center, buttons flanking"
      priority: "Critical"

    # [... more assets ...]
```
