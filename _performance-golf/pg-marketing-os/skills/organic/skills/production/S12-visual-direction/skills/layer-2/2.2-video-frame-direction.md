# Microskill 2.2: Video Frame Direction
Skill: S12 Visual Direction
Layer: 2
Model: claude-opus-4.6
Version: 1.0

## Purpose

Create visual direction for key video frames that support the script narrative. These frames provide visual reinforcement for spoken content — data visualizations, text overlays, section transitions.

## Input

**Required files:**
- Video script from S08/S09/S10
- `s12-1.3-style-guide.yaml` — visual consistency
- `s12-1.4-color-typography.yaml` — precise specs

## Process

### MUST
1. **Identify key frame moments from script**
   - Read video script, mark moments needing visual support
   - Frame types: section headers, data points, quotes, CTAs, transitions
   - Prioritize: which frames are critical vs. optional

2. **Design each key frame**
   - Composition: layout for frame type
   - Visual elements: text, graphics, imagery
   - Typography: specs from system
   - Color usage: maintain consistency with thumbnail

3. **Ensure frame-to-frame consistency**
   - Same style across all frames
   - Visual progression (not disconnected assets)
   - Brand-aligned throughout

4. **Specify timing and transitions**
   - When frame appears in video (timestamp)
   - Duration on screen
   - Transition type (cut, fade, slide)

### NEVER
- Create frames that contradict thumbnail style
- Overload frames with text (video is auditory medium)
- Skip timing specs (editor needs to know when/how long)

## Output

**File:** `s12-2.2-frame-direction.yaml`

```yaml
video_frame_direction:
  content_id: "retirement-40-3bucket-video"
  total_frames: 8

  frames:
    - frame_id: "F01-intro-title"
      timestamp: "0:03-0:07"
      type: "Title card"
      composition: "Centered headline on clean background"
      text: "The 3-Bucket System"
      typography: "Montserrat Bold 64px, #1E40AF"
      background: "#FFFFFF"
      notes: "Opening title, establishes system name"

    - frame_id: "F02-bucket-now"
      timestamp: "1:15-1:30"
      type: "Section header"
      composition: "Bucket graphic left, text right"
      visual: "Bucket #1 (Trust Blue) with NOW label"
      text: "BUCKET 1: Now Money"
      typography: "H1: 64px, H2: 36px"
      notes: "Introduces first bucket, visual anchor"

    # [... more frames ...]
```
