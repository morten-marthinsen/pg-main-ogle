# Microskill 1.2: Round 1 Competition
Skill: S13 Arena Generation | Layer: 1 | Model: claude-opus-4.6 | Version: 1.0

## Purpose
Execute Round 1 of Arena — all 7 personas evaluate content and provide scores + rationales.

## Input
s13-1.1-arena-brief.yaml, s13-0.3-content-draft.md, s13-0.2-personas.yaml

## Process
### MUST
1. Each persona evaluates content through their lens
2. Each persona provides: score (0-10), detailed rationale, strengths, weaknesses
3. All 7 personas MUST participate
4. Document per-persona evaluation in detail

### NEVER
- Skip any persona
- Provide generic rationales
- Allow scores without justification

## Output
**File:** `s13-1.2-round-1-scores.yaml`

```yaml
round_1:
  timestamp: 2026-03-05T15:00:00Z

  evaluations:
    - persona: "Volume Machine"
      score: 7
      rationale: "[Detailed evaluation through volume lens]"
      strengths: [list]
      weaknesses: [list]

    - persona: "Value Architect"
      score: 8
      rationale: "[Detailed evaluation through value lens]"
      strengths: [list]
      weaknesses: [list]

    # [... all 7 personas ...]

  convergence_detected: false  # or true if scores within 2 points
  average_score: 7.3
```

## Quality Gates
- [ ] All 7 personas evaluated
- [ ] Scores with detailed rationales
- [ ] Convergence flagged if present
- [ ] File written

## Handoff
Downstream: 1.3 round-1-critique challenges these evaluations
