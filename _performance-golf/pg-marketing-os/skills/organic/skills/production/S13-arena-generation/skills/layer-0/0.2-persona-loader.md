# Microskill 0.2: Persona Loader
Skill: S13 Arena Generation | Layer: 0 | Model: claude-haiku-4 | Version: 1.0

## Purpose
Load the 7 organic personas with their evaluation lenses, values, and critique focus for Arena execution.

## Input
Required gate: `GATE_0_INPUT_VALID`
Persona file: `skills/organic/persona-registry/organic-personas.md`

## Process
### MUST
1. Load all 7 personas: Volume Machine, Value Architect, Virality Engineer, Community Builder, Brand Purist, Algorithm Hacker, Storyteller
2. For each: extract lens, values, critique focus
3. Verify all 7 loaded completely

### NEVER
- Load subset of personas
- Skip persona characteristics
- Modify persona definitions

## Output
**File:** `s13-0.2-personas.yaml`

```yaml
personas_loaded: 7
timestamp: 2026-03-05T14:05:00Z

personas:
  - name: "Volume Machine"

    values: [practical_utility, immediate_application, clear_takeaway]
    critique_focus: "What's the tangible value?"

  # [... all 7 personas ...]
```
