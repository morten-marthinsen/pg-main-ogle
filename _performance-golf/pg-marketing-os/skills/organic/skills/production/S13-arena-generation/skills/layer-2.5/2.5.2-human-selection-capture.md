# Microskill 2.5.2: Human Selection Capture
Skill: S13 Arena Generation | Layer: 2.5 | Model: claude-opus-4.6 | Version: 1.0

## Purpose
Present Arena results + hybrids to human, capture selection decision with rationale.

## Input
s13-2.5.1-hybrids.yaml, s13-1.7-final-scores.yaml

## Process
### MUST
1. Present original content with Arena scores
2. Present all hybrids with predicted scores and rationales
3. Summarize key Arena insights
4. Provide recommendation (with uncertainty flagged)
5. Capture human selection: which content (original or hybrid), any modifications, rationale, timestamp
6. BLOCK if human unavailable (do NOT proceed without selection)

### NEVER
- Proceed without human selection
- Assume selection (Arena informs, human decides)
- Skip presentation of original content (must be an option)
- Omit Arena insights summary

## Output
**File:** `s13-2.5.2-selection.yaml`

```yaml
human_selection:
  timestamp: 2026-03-05T17:00:00Z
  decision_maker: "Anthony Flores"

  options_presented:
    - id: "original"
      composite_score: 6.6
    - id: "hybrid_alpha"
      predicted_score: 7.1
    - id: "hybrid_beta"
      predicted_score: 7.0
    - id: "hybrid_gamma"
      predicted_score: 7.4

  arena_insights_summary:
    - "Value density strong (8/10)"
    - "Story resonance weak (5/10)"
    - "Volume sustainability concerns"

  recommendation:
    suggested: "Hybrid Gamma"
    confidence: "Medium — comprehensive improvement but untested"

  selection:
    selected_id: "hybrid_alpha"
    modifications_requested: "Strengthen tweet 7 transition, maintain original CTA"
    rationale: "Narrative boost without over-complication. Gamma felt overworked."
    final_content: "[Modified Hybrid Alpha content]"
```

## Quality Gates
- [ ] All options presented
- [ ] Arena insights summarized
- [ ] Human selection captured
- [ ] Rationale documented
- [ ] Final content documented (with modifications if any)
- [ ] File written

**BLOCKING GATE:** Create `GATE_2.5_ARENA_COMPLETE` only after human selection captured.

