# Microskill 2.5.1: Synthesis Generation
Skill: S13 Arena Generation | Layer: 2.5 | Model: claude-opus-4.6 | Version: 1.0

## Purpose
Generate 3+ hybrid content variations combining original strengths + addressing Arena-identified weaknesses.

## Input
Required gate: `GATE_1_ARENA_ROUNDS_COMPLETE`
s13-1.7-final-scores.yaml, s13-0.3-content-draft.md

## Process
### MUST
1. Read synthesis opportunities from final scoring
2. Generate Hybrid Alpha: [approach]
3. Generate Hybrid Beta: [different approach]
4. Generate Hybrid Gamma: [third approach]
5. Each hybrid: full content, rationale tied to Arena insights, predicted scores
6. Minimum 3 hybrids (more if opportunities warrant)

### NEVER
- Generate <3 hybrids
- Create hybrids unconnected to Arena insights
- Skip rationale documentation
- Ignore consensus weaknesses
