# 0.2 Teaching Loader

## Purpose
Load performance analysis teaching files that define how to interpret metrics, detect patterns, and generate learning flags.

## Assigned Model
`claude-haiku-4` (Layer 0 loading)

## Input
- Teaching file registry
- Performance analysis domain teachings

## Process

### MUST
1. **Load metric interpretation teachings** — How to normalize cross-platform metrics, what variance thresholds matter
2. **Load pattern detection teachings** — Statistical significance thresholds, sample size requirements, correlation vs causation
3. **Load learning flag teachings** — Categories of learnings (teaching_update, specimen_addition, skill_refinement, gate_adjustment)
4. **Load root cause taxonomy** — Standard categories: hook failure, platform misfit, timing error, audience mismatch, algorithm shift
5. **Load confidence scoring guidelines** — How to assign confidence levels (high/medium/low) to findings
6. **Verify completeness** — All required teaching sections present
7. **Output loaded context** — Summary of teachings loaded for Layer 1

### NEVER
- Proceed without metric interpretation teachings
- Skip pattern detection thresholds
- Omit learning flag categories
- Load outdated teaching versions

2. Metric thresholds defined
3. Learning flag taxonomy present
