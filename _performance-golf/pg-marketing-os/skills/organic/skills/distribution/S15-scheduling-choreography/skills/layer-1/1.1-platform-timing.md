# 1.1 Platform Timing Intelligence

## Purpose
Analyze platform-specific timing data from PSF and teachings to create optimized posting schedule for each platform.

---

## Input Requirements

From Layer 0:
- `0.2-teachings-loaded.yaml` (platform timing best practices)
- `0.3-upstream-data.yaml` (platform constraints from PSF)

---

## Execution Protocol

### Step 1: Compile Platform Timing Matrix

For each platform in campaign:

```yaml
[platform]:
  optimal_windows:
    tier_1: "[Time ranges with highest engagement]"
    tier_2: "[Secondary windows]"
    tier_3: "[Acceptable windows]"

  format_specific_timing:
    [format_type]:
      best: "[Optimal time for this format]"
      rationale: "[Why this time works]"

  posting_frequency:
    optimal: "[Posts per day]"
    maximum: "[Upper limit]"
    minimum_gap: "[Hours between posts]"

  algorithm_notes:
    first_hour_critical: [true/false]
    velocity_threshold: "[Engagement % needed]"
    special_considerations: []
```

### Step 2: Content Type to Timing Mapping

Match content functions to optimal posting times:

```yaml
awareness_content:
  optimal_timing:
    instagram: "[Time]"
    tiktok: "[Time]"
    # [For each platform]
  rationale: "Peak traffic for discovery"

engagement_content:
  optimal_timing:
    # [Platform-specific]
  rationale: "When audience has time for deep engagement"

conversion_content:
  optimal_timing:
    # [Platform-specific]
  rationale: "Decision-making windows"

community_content:
  optimal_timing:
    # [Platform-specific]
  rationale: "Relaxed, conversational windows"
```

### Step 3: Identify Timing Conflicts

- Check for content competing for same time slots
- Identify platforms with limited optimal windows
- Flag potential over-posting risks

---

## Output

**File:** `layer-1/1.1-platform-timing.yaml`

**Format:**
```yaml
timestamp: "[ISO 8601]"

platform_timing_matrix:
  instagram:
    optimal_windows:
      tier_1: []
      tier_2: []
    format_timing:
      reels: {}
      carousels: {}
      stories: {}
    posting_frequency:
      optimal:
      maximum:
      minimum_gap:
    algorithm_notes:

  # [Repeat for all platforms]

content_function_timing:
  awareness:
    instagram: "[Time]"
    tiktok: "[Time]"
    # [etc]

  engagement:
    # [Platform times]

  conversion:
    # [Platform times]

  community:
    # [Platform times]

timing_conflicts:
  - conflict: "[Description]"
    affected_content: []
    resolution: "[Strategy]"
```

**Minimum size:** 2KB
