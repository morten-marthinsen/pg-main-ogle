## Purpose

Coordinate network of accounts to amplify content at critical moments. The difference between death and virality is often the first wave of engagement.

**Success Criteria:**
- Network activation sequence fully defined
- All comment content written and unique
- Video responses prepared (if planned)
- Organic appearance rules implemented
- No obvious coordination patterns

## Constraints

### MUST
- **MUST** vary timing between network engagements (3-20min gaps)
- **MUST** write unique comment content for each account
- **MUST** maintain independent content lives for network accounts
- **MUST** implement organic appearance rules

### NEVER
- **NEVER** have network accounts engage simultaneously (coordination signal)
- **NEVER** use identical comment text across accounts
- **NEVER** skip timing variation (pattern detection)
- **NEVER** make network activity obvious
