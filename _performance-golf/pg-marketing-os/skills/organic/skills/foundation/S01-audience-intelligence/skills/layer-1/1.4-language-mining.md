# 1.4 Language Mining

## Purpose
Capture HOW they talk — this is GOLD for content creation. Exact words, phrases, slang, jargon. This section is MANDATORY and cannot be skipped or synthesized from memory.

---

## Input Requirements
- brand_info.category (determines where to mine)
- psychographic-depth.yaml (informs which communities to target)

---

## Execution Protocol

### Step 1: Identify Mining Sources
```
PRIORITY SOURCES (MUST use actual audience quotes):
  - Reddit threads in relevant subreddits
  - Comment sections on competitor content
  - Amazon reviews of related products
  - Facebook group discussions
  - Twitter/X conversations with hashtags
  - YouTube comments on related videos
  - Customer support tickets (if accessible)
  - Forum posts (niche-specific forums)

MINIMUM: Mine 3+ different source types

NEVER synthesize language — MUST extract actual quotes.
```

### Step 2: Vocabulary Register Analysis
```
DETERMINE:
  - Formal vs casual language ratio
  - Technical term usage level
  - Profanity/casual speech patterns
  - Education level indicators

EXTRACT:
  - 10-20 specific words they use frequently
  - Contractions usage (don't vs do not)
  - Slang frequency
```

### Step 3: Common Phrases Collection
```
COLLECT ≥10 phrases they repeat:

EXAMPLES:
  - "I've tried everything"
  - "Nothing seems to work"
  - "I'm so done with..."
  - "Why is this so hard?"

CATEGORIZE by:
  - Frustration expressions
  - Hope/aspiration expressions
  - Identity expressions ("I'm the type of person who...")
  - Temporal markers ("I've been doing this for...")
```

### Step 4: Slang and Jargon Inventory
```
IDENTIFY:

INDUSTRY JARGON:
  - Technical terms specific to niche
  - Acronyms they use
  - Insider language

GENERATIONAL SLANG:
  - Age-specific language patterns
  - Platform-specific terms (TikTok-isms, Twitter-speak)

COMMUNITY TERMS:
  - In-group language
  - Tribe-specific vocabulary
  - Shared references
```

### Step 5: Pain Expression Mining (CRITICAL)
```
MINIMUM 3 REQUIRED — This is a GATE requirement.

HOW THEY DESCRIBE PROBLEMS:
  Extract EXACT phrases (verbatim quotes):
    - "[Exact quote 1]" — Source: [Reddit/YouTube/etc]
    - "[Exact quote 2]" — Source: [Platform]
    - "[Exact quote 3]" — Source: [Platform]

EMOTIONAL WORDS USED:
  - Frustration vocabulary ("annoying", "exhausting", "hopeless")
  - Intensity modifiers ("so", "really", "extremely")

QUESTIONS THEY ASK:
  - "How do I..." questions
  - "Why does..." questions
  - "What's the best way to..." questions
```

### Step 6: Desire Expression Mining
```
HOW THEY DESCRIBE GOALS:
  - "I want to..."
  - "I wish I could..."
  - "My goal is..."

SUCCESS LANGUAGE:
  - How they talk about winning
  - Achievement markers
  - Status symbols in their words

ASPIRATION LANGUAGE:
  - Future-state vocabulary
  - "Imagine if..."
  - "Someday I'll..."
```

## Output
- **File:** `language-mining.yaml`
- **Format:** YAML
- **Minimum size:** 3KB

```yaml
language_mining:
  vocabulary_register:
    formality_level: formal | casual | mixed
    technical_usage: high | medium | low
    specific_words: []  # 10-20 words

  phrases_they_use:
    common_expressions:
      - phrase: "[Exact phrase]"
        frequency: high | medium | low
        context: "[When they use it]"
    # ≥10 phrases

  slang_and_jargon:
    industry_jargon: []
    generational_slang: []
    community_terms: []

  pain_expressions:
    how_they_describe_problems:
      - quote: "[EXACT QUOTE — verbatim]"
        source: "[Platform/location]"
        context: "[What they were discussing]"
      # MINIMUM 3 REQUIRED
    emotional_words_used: []
    questions_they_ask: []

  desire_expressions:
    how_they_describe_goals: []
    success_language: []
    aspiration_language: []

mining_sources:
  - source_type: "[Reddit/YouTube/etc]"
    location: "[Specific subreddit/channel]"
    quotes_extracted: [count]
  # ≥3 source types

verification:
  total_verbatim_quotes: [count]
  minimum_pain_expressions_met: true | false  # MUST be true
```

## Quality Gates
- [ ] ≥3 pain expression quotes (verbatim, with sources)
- [ ] ≥3 mining source types used
- [ ] ≥10 common phrases collected
- [ ] All quotes are ACTUAL quotes (not AI-synthesized)
- [ ] Sources documented for every quote

**CRITICAL:** If pain_expressions count < 3, this is a BLOCKING FAILURE. Layer 1 cannot complete.
