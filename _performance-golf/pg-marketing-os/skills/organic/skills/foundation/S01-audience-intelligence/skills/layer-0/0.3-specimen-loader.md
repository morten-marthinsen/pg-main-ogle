# 0.3 Specimen Loader

## Purpose
Load growth trajectory case studies and competitor audience analysis examples from the relevant niche. Specimens provide pattern templates for analysis.

---

## Input Requirements
From 0.1 input validation:
- brand_info.category (determines which specimens to load)

---

## Execution Protocol

### Step 1: Identify Relevant Niche
```
MAP brand_info.category to specimen directory:
  IF category in [golf, sports, fitness]:
    directory: "specimens/organic/sports/"

  IF category in [health, wellness, supplements]:
    directory: "specimens/organic/health/"

  IF category in [business, entrepreneurship, coaching]:
    directory: "specimens/organic/business/"

  IF category in [personal-dev, mindset, productivity]:
    directory: "specimens/organic/personal-dev/"

  ELSE:
    directory: "specimens/organic/general/"
    LOG: "No exact category match — using general specimens"
```

### Step 2: Load Growth Trajectory Cases
```
LOAD 5+ case studies showing audience growth patterns:
  - Starting audience size
  - Growth tactics used
  - Time to milestones
  - Platform mix
  - Audience demographics evolution
```

### Step 3: Load Competitor Analysis Examples
```
LOAD 3+ competitor audience analysis specimens:
  - Audience demographics
  - Engagement patterns
  - Content strategy
  - Gaps/opportunities
```

### Step 4: Extract Patterns
```
FOR each specimen:
  IDENTIFY:
    - Audience pain/desire patterns
    - Language usage patterns
    - Platform behavior patterns
    - Growth inflection points
```

---

## Output
- **File:** `specimens-loaded.yaml`
- **Format:** YAML

```yaml
specimens_loaded:
  niche: [matched category]
  specimen_directory: [path]

  growth_trajectory_cases:
    - specimen_id: [id]
      creator: [name]
      audience_size: [start → end]
      key_tactics: []
    # 5+ entries

  competitor_analysis_examples:
    - specimen_id: [id]
      competitor: [name]
      audience_insights: [summary]
    # 3+ entries

  pattern_observations:
    common_pain_points: []
    common_desires: []
    common_platforms: []
    language_patterns: []

loading_summary:
  growth_cases_loaded: [count]
  competitor_examples_loaded: [count]
  total_specimens: [count]
```

---

## Quality Gates
- [ ] ≥5 growth trajectory cases loaded
- [ ] ≥3 competitor analysis examples loaded
- [ ] pattern_observations populated with at least 2 entries per category
- [ ] No loading errors
