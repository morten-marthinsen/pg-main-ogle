# 0.3 CopywritingEngine Loader

## Purpose
OPTIONAL: If this campaign is downstream from CopywritingEngine Skill 09 (Campaign Brief), load the campaign-brief-package.json to integrate long-form campaign strategy with organic content strategy.

---

## Input Requirements

Optional file (Mode A only):
- `CopywritingEngine/outputs/[project]/09-campaign-brief/campaign-brief-package.json`

---

## Execution Protocol

### Step 1: Check for CopywritingEngine Package

Look for campaign-brief-package.json:

```python
if file_exists("campaign-brief-package.json"):
    mode = "Mode A: Downstream Integration"
    load_package = true
else:
    mode = "Mode B: Organic Native"
    load_package = false
```

---

### Step 2: Load Package (Mode A only)

If package exists, extract:

```yaml
copywritingengine_context:
  product_name: "[name]"
  big_idea: "[the big idea]"
  mechanism: "[mechanism name]"
  promise: "[transformation promise]"
  root_cause: "[what they think vs what's real]"
  target_niche: "[niche]"
  campaign_type: "[VSL/Sales Letter/Email Sequence]"

  long_form_assets:
    - "[list of produced assets]"

  offer_structure:
    price_point: "[price]"
    guarantee: "[guarantee]"
    bonuses: [list]
```

---

### Step 3: Map to Organic Strategy

If Mode A, create mapping:

```yaml
organic_mapping:
  big_idea_to_pillar:
    big_idea: "[from CopywritingEngine]"
    maps_to_pillar: "[which CAF pillar]"
    content_approach: "[how to translate to social]"

  mechanism_to_content:
    mechanism: "[name]"
    content_series: "[recurring series explaining mechanism]"
    format: "[Reels/Carousels/Threads]"

  proof_to_social_proof:
    copywriting_proof: "[from proof inventory]"
    social_format: "[testimonial clips/data visualizations/case studies]"

  funnel_connection:
    organic_role: "[top of funnel awareness → long-form landing page]"
    cta_approach: "[how to drive to landing page]"
```

---

### Step 4: Extract Campaign Timing

If CopywritingEngine campaign has launch date:

```yaml
timing_integration:
  long_form_launch_date: "[date]"
  organic_pre_launch_period: "[X weeks before]"
  organic_post_launch_strategy: "[nurture and amplify]"
```

---

## Output

**File:** `layer-0/0.3-copywritingengine-context.yaml`

**Format:**

**If Mode A (package exists):**
```yaml
timestamp: "[ISO 8601]"
mode: "Mode A: Downstream Integration"
package_loaded: true

copywritingengine_context: {...}
organic_mapping: {...}
timing_integration: {...}

integration_notes: |
  Organic campaign will support CopywritingEngine long-form promotion.
  Content pillars should build awareness of mechanism and big idea.
  Funnel drives from organic platforms to landing page/VSL.
```

**If Mode B (no package):**
```yaml
timestamp: "[ISO 8601]"
mode: "Mode B: Organic Native"
package_loaded: false

note: |
  This is an organic-first campaign. No CopywritingEngine integration.
  Proceed with standard S07 synthesis from S01-S06 only.
```

---

## Quality Gates

- [ ] Mode determined (A or B)
- [ ] If Mode A: package loaded successfully
- [ ] If Mode A: mapping created between long-form and organic strategy
- [ ] If Mode B: confirmed no package exists

---

## Handoff

Pass context (with or without CopywritingEngine integration) to: **0.4 Upstream Synthesis**
