# 0.2 Foundation Loader

## Purpose
Load all 6 foundation files into working memory for synthesis. This step ensures all context is available before planning begins.

---

## Input Requirements

From 0.1:
- Validation report confirming all files exist and are complete

Foundation files:
- `outputs/[campaign]-AIF.yaml`
- `outputs/[campaign]-PSF.yaml`
- `outputs/[campaign]-BVF.yaml`
- `outputs/[campaign]-CAF.yaml`
- `outputs/[campaign]-HLF.yaml`
- `outputs/[campaign]-VSF.yaml`

---

## Execution Protocol

### Step 1: Load Each Foundation File

Read and parse all 6 YAML files into structured data.

**CRITICAL:** Do not just reference files. Load their actual content into memory for synthesis.

---

### Step 2: Extract Key Data Points

For each file, extract the MOST IMPORTANT insights:

**From AIF:**
```yaml
audience_key_data:
  target_segment: "[description]"
  primary_pain: "[pain]"
  primary_desire: "[desire]"
  emotional_triggers: [top 3]
  language_to_use: [top 10 phrases]
  scroll_stoppers: "[what makes them pause]"
  optimal_posting_times: [times]
```

**From PSF:**
```yaml
platform_key_data:
  primary_platform: "[platform]"
  secondary_platforms: [list]
  posting_cadence: "[frequency]"
  content_formats: [list]
  optimal_times: [times]
  algorithm_priorities: [list]
  cross_platform_strategy: "[how content moves]"
```

**From BVF:**
```yaml
voice_key_data:
  tone: "[e.g., conversational_authority]"
  energy_level: "[high/medium/low]"
  signature_phrases: [list]
  banned_words: [list]
  acceptable_range: "[variation allowed]"
```

**From CAF:**
```yaml
content_architecture_key_data:
  pillars: [list with descriptions]
  function_mix:
    awareness: [%]
    engagement: [%]
    conversion: [%]
    community: [%]
  series: [recurring formats]
  funnel_connection: "[how content drives revenue]"
```

**From HLF:**
```yaml
hook_key_data:
  primary_hook_types: [top 3-5]
  platform_specific_hooks: [top 10 hooks for primary platform]
  hook_templates: [proven patterns]
  testing_insights: "[what's been validated]"
```

**From VSF:**
```yaml
virality_key_data:
  minimum_publish: [score]
  target: [score]
  viral_threshold: [score]
  must_haves: [checklist]
  red_flags: [checklist]
  dimension_priorities: [top 3 dimensions]
```

---

### Step 3: Cross-Reference for Alignment

Check that all 6 foundation files are aligned:

**Platform consistency:**
- PSF primary platform matches HLF platform-specific hooks
- PSF formats match CAF content types
- VSF modifiers match PSF primary platform

**Audience-voice alignment:**
- BVF language matches AIF language_to_use
- BVF banned words align with AIF language_to_avoid

**Strategy coherence:**
- CAF function_mix makes sense for AIF audience stage
- HLF hook types align with AIF emotional triggers

**Flag any misalignments for human review.**
