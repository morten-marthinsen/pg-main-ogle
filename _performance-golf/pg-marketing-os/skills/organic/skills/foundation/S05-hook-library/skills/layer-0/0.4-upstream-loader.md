# 0.4 Upstream Loader

## Purpose
Load and synthesize all upstream outputs (AIF, PSF, BVF, CAF) to inform hook generation.

---

## Input Requirements
- Validated AIF, PSF, BVF, CAF files (from 0.1)
- Teaching summary (from 0.2)
- Specimen analysis (from 0.3)

---

## Execution Protocol

### Step 1: Extract from AIF (Audience Intelligence)
**Load:** `[project]/S01-audience-intelligence/outputs/[campaign]-AIF.yaml`

**Extract for hook generation:**
```yaml
audience_context:
  pain_language: []  # Exact phrases for pain-based hooks
  desire_language: []  # Exact phrases for desire-based hooks
  frustration_expressions: []  # For fear hooks
  aspiration_statements: []  # For identity hooks
  sophistication_level: ""  # Affects hook complexity
```

**CRITICAL:** Extract verbatim language patterns. Hooks must use audience's actual words.

---

### Step 2: Extract from PSF (Platform Strategy)
**Load:** `[project]/S02-platform-strategy/outputs/[campaign]-PSF.yaml`

**Extract:**
```yaml
platform_context:
  primary_platform: ""
  optimal_formats: []  # Formats that need hooks
  platform_constraints:
    max_hook_length: ""  # Character/word limits
    visual_requirements: ""  # If hooks need visual support
    sound_on_off: ""  # Audio strategy
```

---

### Step 3: Extract from BVF (Brand Voice)
**Load:** `[project]/S03-brand-voice/outputs/[campaign]-BVF.yaml`

**Extract:**
```yaml
voice_context:
  voice_attributes: []  # Must be reflected in hooks
  tone_range: ""  # Acceptable tones for hooks
  anti_voice: []  # FORBIDDEN patterns in hooks
  language_style: ""  # Formal/casual/technical/etc
```

**Create voice filter:** List of forbidden words/phrases from anti_voice.

---

### Step 4: Extract from CAF (Content Architecture)
**Load:** `[project]/S04-content-architecture/outputs/[campaign]-CAF.yaml`

**Extract:**
```yaml
pillar_context:
  pillars:
    - name: ""
      description: ""
      topic_clusters: []
      percentage: 0

  series:
    - name: ""
      hook_pattern: ""  # If series has defined hook pattern
      pillar: ""

  content_functions:
    awareness_percentage: 0
    engagement_percentage: 0
    conversion_percentage: 0
    community_percentage: 0
```

**Map pillars to hook categories:**
For each pillar, identify which hook categories are most appropriate.

---

### Step 5: Synthesize Hook Generation Strategy
Combine all inputs:

```yaml
hook_generation_strategy:
  priority_categories:
    - category: "[Most relevant to pillars]"
      rationale: "[Why this category for this campaign]"

  language_constraints:
    must_include: []  # Audience language from AIF
    must_avoid: []  # Anti-voice from BVF

  platform_requirements:
    format: ""
    length: ""
    structure: ""

  pillar_hook_distribution:
    pillar_1:
      name: ""
      hook_categories_to_emphasize: []
      hook_count_target: 5+
    # Repeat for all pillars

  voice_calibration_rules:
    - rule: ""
      source: "[BVF]"
```
