# 1.2 Template Generation

## Purpose
Generate reusable hook templates for each category with campaign-specific adaptations. Still Phase A (templates, not final hooks).

---

## Input Requirements
- Taxonomy application from 1.1
- Specimen analysis from 0.3
- Audience language from AIF
- CONCEPT_APPROVED.yaml checkpoint (REQUIRED)

---

---

### Step 2: Campaign-Specific Template Adaptation
For priority templates (from high-priority categories), add campaign specificity:

**Example:**
- **Generic:** "How I [achieved result] in [time frame]"
- **Campaign-adapted:** "How I [specific result using audience language from AIF] in [realistic timeframe]"

---

### Step 3: Voice Compatibility Notes
For each template, note:
- Which voice attributes it expresses
- Any anti-voice conflicts
- How to modify template to match BVF
