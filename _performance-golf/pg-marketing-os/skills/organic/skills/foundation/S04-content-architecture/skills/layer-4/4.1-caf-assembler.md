# 4.1 CAF Assembler

## Purpose
Compile all Layer 1 outputs into the complete Content Architecture File (CAF).

---

## Input Requirements
All Layer 1 outputs:
- 1.1-pillar-concepts.yaml
- 1.2-function-mapping.yaml
- 1.3-series-concepts.yaml
- 1.4-format-matrix.yaml
- 1.5-funnel-integration.yaml
- 1.6-calendar-framework.yaml

---

## Execution Protocol

### Step 1: Load All Layer 1 Outputs
Read all 6 Layer 1 files and verify completeness:
- [ ] Pillar concepts (3-5 pillars)
- [ ] Function mapping (4 functions, 100% total)
- [ ] Series concepts (≥1 series)
- [ ] Format matrix (≥3 formats)
- [ ] Funnel integration (≥1 primary CTA)
- [ ] Calendar framework (7 days populated)

**If any file incomplete → HALT. Return to Layer 1.**

---

### Step 2: Assemble CAF Structure
Compile into final CAF format:

```yaml
# CONTENT ARCHITECTURE FILE (CAF)
# Campaign: [Name]
# Created: [Date]
# Version: 1.0

campaign_name: "[From business context]"
date_created: "[ISO 8601]"
version: "1.0"

## CONTENT PILLARS
content_pillars:
  - name: "[Named pillar 1]"
    description: |
      [From 1.1]
    audience_need: |
      [From 1.1 — must be from AIF]
    topic_clusters:
      - cluster: "[From 1.1]"
        subtopics: []
    percentage_of_content: [From 1.1]
    business_connection: |
      [From 1.1]

  # Repeat for all pillars

## CONTENT FUNCTIONS
content_functions:
  awareness:
    percentage: [From 1.2]
    purpose: |
      [From 1.2]
    content_types:
      - type: ""
        pillar: ""
        frequency: ""
    success_metrics: []

  # Repeat for engagement, conversion, community

## CONTENT SERIES
content_series:
  - name: "[Named series 1]"
    format: "[From 1.3]"
    platform: "[From 1.3]"
    pillar: "[From 1.3]"
    function: "[From 1.3]"
    concept: |
      [From 1.3]
    structure:
      hook_pattern: |
      body_pattern: |
      close_pattern: |
    frequency: "[From 1.3]"
    example_episodes: [From 1.3]
    success_criteria: [From 1.3]

  # Repeat for all series

## FORMAT MATRIX
format_matrix:
  formats_in_use: ["[List all formats]"]
  format_details:
    short_form_video:
      platforms: []
      variations: []
      pillar_fit: []
    # Repeat for all formats

## FUNNEL INTEGRATION
funnel_integration:
  content_funnel:
    top_of_funnel: [From 1.5]
    middle_of_funnel: [From 1.5]
    bottom_of_funnel: [From 1.5]

  cta_strategy:
    primary_ctas: [From 1.5]
    soft_ctas: [From 1.5]

  lead_magnets: [From 1.5]

  offer_integration: [From 1.5]

## CALENDAR FRAMEWORK
calendar_framework:
  weekly_rhythm:
    monday: [From 1.6]
    tuesday: [From 1.6]
    # All 7 days

  posting_cadence:
    platform_1: [From 1.6]
    # All platforms

  content_batch_structure: [From 1.6]

  seasonal_adjustments: [From 1.6]

## SOURCE FILES
source_files:
  AIF: "[Path to AIF]"
  PSF: "[Path to PSF]"
  BVF: "[Path to BVF]"

## METADATA
metadata:
  skill_version: "S04 v1.0"
  created_by: "S04 Content Architecture Agent"
  layer_1_complete_timestamp: "[From checkpoint]"
  concept_approval_timestamp: "[From checkpoint]"
```

---

### Step 3: Validation Pass
Run final validation checks:

**Pillar validation:**
- [ ] 3-5 pillars present
- [ ] Each has audience_need from AIF
- [ ] Percentages sum to 100%

**Function validation:**
- [ ] All 4 functions present
- [ ] Percentages sum to 100%
- [ ] Content types mapped to pillars

**Series validation:**
- [ ] ≥1 series present
- [ ] Each has complete structure
- [ ] Each maps to pillar and function

**Format validation:**
- [ ] ≥3 formats defined
- [ ] Each has platform-specific variations
- [ ] Pillar fit scores present

**Funnel validation:**
- [ ] All 3 funnel stages defined
- [ ] ≥1 primary CTA present

**Calendar validation:**
- [ ] All 7 days populated
- [ ] ≥1 platform cadence defined

**If any validation fails → HALT. Fix in Layer 1.**

---

### Step 4: Write CAF to Output Location
Write complete CAF to:
```
[project]/S04-content-architecture/outputs/[campaign-name]-CAF.yaml
```

---

## Output
- **File:** `[campaign-name]-CAF.yaml`
- **Location:** `outputs/`
- **Format:** YAML (complete structure above)

- [ ] File written to outputs/ directory

Pass complete CAF to: **4.2 Execution Log**
