# 0.4 Upstream Loader

## Purpose
Load and synthesize all upstream outputs (AIF, PSF, BVF) to inform architecture decisions.

---

## Input Requirements
- Validated AIF, PSF, BVF files (from 0.1)
- Teaching summary (from 0.2)
- Specimen analysis (from 0.3)

---

## Execution Protocol

### Step 1: Extract from AIF (Audience Intelligence)
**Load:** `[project]/S01-audience-intelligence/outputs/[campaign]-AIF.yaml`

**Extract:**
```yaml
audience_context:
  primary_pain_points: []
  primary_desires: []
  language_patterns: []
  sophistication_level: ""
  demographic_context: {}
```

**Map each pain point and desire to potential pillar themes.**

---

### Step 2: Extract from PSF (Platform Strategy)
**Load:** `[project]/S02-platform-strategy/outputs/[campaign]-PSF.yaml`

**Extract:**
```yaml
platform_context:
  primary_platform: ""
  secondary_platforms: []
  optimal_formats: []
  posting_frequency: ""
  platform_constraints: []
```

**Note format and frequency constraints for architecture design.**

---

### Step 3: Extract from BVF (Brand Voice)
**Load:** `[project]/S03-brand-voice/outputs/[campaign]-BVF.yaml`

**Extract:**
```yaml
voice_context:
  voice_attributes: []
  tone_range: ""
  anti_voice: []
  content_style_guidelines: []
```

**Note voice constraints that affect pillar selection and series design.**

---

### Step 4: Synthesize Strategic Context
Combine all inputs to create strategic context for Layer 1:

```yaml
strategic_synthesis:
  pillars_must_address: []        # From AIF pain/desire mapping
  formats_to_prioritize: []        # From PSF optimal formats
  voice_constraints_on_topics: []  # From BVF anti-voice
  frequency_targets: {}            # From PSF cadence
  business_priorities: []          # From business context
```

---
