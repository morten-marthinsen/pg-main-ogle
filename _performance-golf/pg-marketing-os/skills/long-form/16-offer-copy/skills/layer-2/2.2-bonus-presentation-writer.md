## PURPOSE

The Bonus Presentation Writer creates the prose for the bonus section of the offer — the portion where the prospect learns about additional items they receive on top of the core product. Bonuses serve a specific psychological function: they transform the offer from "what you're buying" into "what you're GETTING." Each bonus is a gift, an act of generosity, an additional reason to buy. When presented correctly, bonuses create the feeling that the seller is going above and beyond — that the prospect is getting far more value than they are paying for.

Each bonus follows the D-F-W-B-P format with a stated dollar value. The stated value is not incidental — it contributes to the total retail value anchor that makes the actual price feel like a steal. Every bonus value, stacked on top of the core product value, builds the cumulative number that the price will later be contrasted against.

The bonus section must feel distinctly different from the deliverable stack. Deliverables are "what you're buying." Bonuses are "what you're getting FOR FREE." The transition from deliverables to bonuses is a pivotal emotional shift — from value to generosity, from expectation to surprise. The writer must nail this transition and maintain mounting excitement through each successive bonus.

**This microskill is NOT:**

## PROCESS

### Step 1: Write the Bonus Transition (SIN Chunk 4)

1. The bonus transition is a single pivotal sentence (or short paragraph) that shifts from the deliverable stack to the bonus section
2. This transition must:
   - Acknowledge what the prospect already gets ("With everything you've seen so far...")
   - Signal that there is MORE coming ("But I wanted to do even more for you...")
   - Create anticipation ("That's why I'm also including...")
3. Write the transition using TIER1 patterns:

**Pattern A: The Generosity Transition**
"And that's not all. When you grab [Product] today, I'm also including [N] special bonuses — completely free."

**Pattern B: The Above-and-Beyond Transition**
"Now, with everything you've seen so far, you already have everything you need to [promise]. But I wanted to go above and beyond. That's why I'm also including..."

**Pattern C: The Sweetener Transition**
"But here's where it gets really exciting. Because when you order [Product] today, you're also going to receive..."

**Pattern D: The Insider Transition**
"And because you're watching this today, I'm going to include something special that I normally charge for separately..."

4. Select the transition pattern that matches the campaign tone and product format
5. The transition must create a distinct emotional shift — from satisfaction ("I have everything") to excitement ("there's MORE")

### Step 2: Write Each Bonus in D-F-W-B-P Format

6. For each bonus, in the order mapped by 1.2, write the complete D-F-W-B-P sequence with stated dollar value:

**D (Deliverable) — The Bonus Introduction:**
- Include the bonus name and dollar value
- Frame as a gift, not a purchase
- Pattern: "The first bonus is [Bonus Name] — a $[value] value — and it's yours free today."
- Vary the introduction language for each bonus

TIER1 bonus introduction patterns:
- "Your first free bonus is [Name] — that's a $[value] value."
- "Bonus number one: [Name]. This normally sells for $[value], but today it's yours at no extra cost."
- "I'm also throwing in [Name] — a $[value] value — absolutely free."
- "And you also get [Name]. I could easily charge $[value] for this alone, but it's included free when you grab [Product] today."

**F (Feature) — What the Bonus Contains:**
- Specific contents: page count, chapter count, templates, tools, duration
- Format: "Inside, you'll find..." / "[Bonus] contains..." / "This is a..."

**W (Why) — Why This Bonus Matters:**
- Connect to a gap, pain, or question the prospect has
- Frame as solving a specific friction point
- "I'm including this because..." / "This matters because..."
- For medium/brief depth, the Why can be combined with the Feature sentence

**B (Benefit) — What the Prospect Gains:**
- Personal outcome from using this bonus
- "So you can..." / "Which means..." / "Giving you..."
- Must be a specific outcome, not "you'll get great results"

**P (Proof) — Evidence (if applicable):**
- Lighter proof for bonuses — brief reference is sufficient
- "Based on..." / "Used by..." / "The same approach that..."
- For brief depth, proof can be omitted

7. After presenting D-F-W-B-P for each bonus, include the value statement:
   - "That's a $[value] value — yours free today."
   - Or embed the value in the introduction (see patterns above)

### Step 3: Write Transitions Between Bonuses

8. Between each bonus, write a transition that maintains momentum:
   - Each transition should ESCALATE excitement, not just enumerate
   - Vary the language — never repeat the same transition pattern consecutively

TIER1 bonus-to-bonus transition patterns:
- "But wait, there's more."
- "And I didn't stop there."
- "Plus, you're also going to get..."
- "On top of that, I'm including..."
- "And here's one more thing I think you're going to love..."
- "But that's not all I have for you today."
- "And because I want to make this decision as easy as possible for you..."

9. The transitions should create a "stacking" feeling — the prospect should feel overwhelmed (in a good way) by how much they are getting
10. The LAST bonus transition should signal finality: "And finally..." / "Last but not least..." / "One more thing..."

### Step 4: Calculate and Present Running Value Total

11. After each bonus, the cumulative value should be implicitly building
12. After the LAST bonus, write the cumulative value lead-in:
    - "So when you add it all up — [Product] plus all [N] bonuses — you're looking at a total value of..."
    - This leads directly into the value totaling section (2.3)
13. Calculate and verify:
    - Sum of all bonus values
    - Running total (core product value + cumulative bonus values)
    - This must match the retail_value from the price presentation plan (1.4)

### Step 5: Verify Bonus Section Quality

14. Check each bonus against D-F-W-B-P requirements (at assigned depth):
    - D present with bonus name and dollar value
    - F present with specific contents
    - W present (full/medium depth)
    - B present with personal prospect outcome
    - P present (full depth) or omitted (brief depth)
15. Check overall bonus section quality:
    - Transition from deliverables to bonuses is distinct and emotionally elevated
    - Each bonus feels like a gift (not just more product)
    - Dollar values stated prominently for every bonus
    - Transitions between bonuses are varied and build momentum
    - Cumulative stacking feeling is present
    - Final bonus leads naturally to value totaling
16. Calculate word count per bonus and total bonus section word count
17. Verify total is within the target range from 1.1 section proportions

### Step 6: Package the Bonus Section Draft

18. Compile the complete bonus section output:
    - Bonus transition text
    - Individual bonus texts with word counts and D-F-W-B-P checks
    - Bonus-to-bonus transition texts
    - Total bonus value
    - Full prose text (transition + all bonuses)
    - Connection point to value totaling (how the last bonus leads to total value)

## FAILURE MODES

### Failure Mode 1: Bonuses Sound Like More Deliverables
- **Symptom:** Bonus section reads the same as the deliverable stack — no emotional shift
- **Cause:** Writer did not frame bonuses as gifts; used the same tone as deliverables
- **Prevention:** Bonus language must include "free," "included," "yours at no extra cost," "I'm throwing in"
- **Recovery:** Rewrite bonus introductions with gift framing: "I'm also including [Bonus] — a $97 value — absolutely free."

### Failure Mode 2: Dollar Values Buried or Missing
- **Symptom:** Bonuses described but values not stated — cumulative value perception is weak
- **Cause:** Writer focused on content description and forgot to state the dollar value
- **Prevention:** Dollar value is PART of D-F-W-B-P for bonuses — it is in the Deliverable introduction
- **Recovery:** Add "$[value] value" to every bonus introduction

### Failure Mode 3: Flat Momentum Between Bonuses
- **Symptom:** Each bonus feels the same — no escalation, no building excitement
- **Cause:** Same transition pattern used repeatedly; bonuses presented at equal emotional intensity
- **Prevention:** Vary transitions, increase enthusiasm language with each successive bonus
- **Recovery:** Rewrite transitions with escalating energy: first bonus = "I'm also including..." → second = "But I didn't stop there..." → third = "And here's one more thing I think you're really going to love..."

### Failure Mode 4: Bonus Section Too Long
- **Symptom:** Each bonus gets full-depth D-F-W-B-P treatment, making the section excessively long
- **Cause:** Depth assignments from 1.1 were not respected — all bonuses written at full depth
- **Prevention:** Check depth assignment for each bonus; minor bonuses get brief treatment
- **Recovery:** Trim low-value bonuses to brief depth (D-F-B only, 30-60 words)
