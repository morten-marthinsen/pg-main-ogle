# Skill 13.4.3: Vault Pattern Comparator

**Version:** 1.0
**Skill:** 10 — Root Cause Narrative
**Layer:** 4 (Validation & Assembly)
**Position:** Third validation microskill — compares the generated narrative against elite TIER1 controls
**Type:** Comparative Validation
**Dependencies:** All Layer 3 outputs, tier1_intelligence_index.json (from 0.2), communication_type_classification.json (from 1.1), framing_pattern_selection.json (from 1.2), emotional_arc_design.json (from 1.3)
**Output:** vault_comparison_report.json

---

## PURPOSE

Compare the generated root cause narrative against elite root cause narratives from TIER1 controls across five dimensions: structural match, depth match, evidence density, emotional arc quality, and specificity level. The narrative must demonstrate it follows TIER1 patterns AND shows differentiation — it should match elite patterns but not be a copy. This comparison validates that the generated narrative belongs in the same quality tier as Gundry, Haney, Stansberry, and other TIER1 controls.

---

## IDENTITY

**This microskill IS:**
- A quality benchmarker that measures the generated narrative against the best-in-class
- A pattern matcher that verifies TIER1 structural patterns are present
- A differentiation documenter that ensures the narrative is original, not derivative
- A comparison scorer that produces a numeric quality assessment across 5 dimensions

**This microskill is NOT:**
- A plagiarism checker — it's not looking for copied text; it's comparing patterns and quality
- A content creator — it compares, not generates
- A rules checker (that is 4.1) — this compares against elite examples, not rules
- A final assembler (that is 4.4) — this produces a comparison report, not the final package

## PROCESS

### Step 1: Select Comparison Controls

```
SELECT 3-5 TIER1 controls for comparison:

SELECTION CRITERIA (in priority order):
  1. SAME NICHE — controls from the same or adjacent niche
  2. SAME COMMUNICATION TYPE — controls using the same classified type
  3. SAME FRAMING PATTERN — controls using the same primary frame
  4. HIGH QUALITY — controls recognized as the strongest in the vault

EXAMPLE selections by niche:

  IF health/supplement niche:
    Primary comparisons:
      - Gundry Dark Energy (scientific_explanation + villain_driven)
      - Montezuma (not_your_fault + conspiracy)
      - FloraSource (suppressed_science)
    Secondary: Sears (hidden_truth + science_driven)

  IF golf instruction niche:
    Primary comparisons:
      - Haney "IT'S A LOT OF BULL" (direct_confrontation + villain_driven)
    Secondary: adjacent sports instruction controls if available

  IF financial niche:
    Primary comparisons:
      - Stansberry "End of America" (data_presentation + conspiracy_expose)
    Secondary: adjacent financial controls if available

LOG: "Selected [count] comparison controls: [list]"
LOG: "Selection rationale: [why each was chosen]"
```

### Step 2: Compare Structural Match

```
DIMENSION 1: STRUCTURAL MATCH — Does the narrative follow the universal sequence?

CHECK against the 7-phase universal sequence:

  Phase 1 (Problem Agitation): PRESENT / ABSENT
    Quality: Does it name specific symptoms in prospect language?
    TIER1 benchmark: Gundry names 12+ symptoms in patient vocabulary

  Phase 2 (False Belief Installation): PRESENT / ABSENT
    Quality: Is the false belief precisely articulated?
    TIER1 benchmark: Haney articulates exactly what golfers believe ("grip, stance, backswing")

  Phase 3 (Authority Establishment): PRESENT / ABSENT
    Quality: Is authority sufficient for the counter-intuitive claim?
    TIER1 benchmark: Gundry stacks 5+ credentials before any lectin claims

  Phase 4 (Root Cause Reveal): PRESENT / ABSENT
    Quality: Is the three-part structure delivered clearly?
    TIER1 benchmark: All TIER1 controls deliver all three parts

  Phase 5 (Failure Explanation): PRESENT / ABSENT
    Quality: Are all major failures accounted for?
    TIER1 benchmark: Gundry explains diets, exercise, supplements

  Phase 6 (Villain Deep Dive): PRESENT / ABSENT
    Quality: Is villain named with motive and evidence?
    TIER1 benchmark: Gundry names food industry with 1950 conference evidence

  Phase 7 (Mechanism Bridge): PRESENT / ABSENT
    Quality: Does it bridge cleanly to the mechanism?
    TIER1 benchmark: All controls transition to mechanism or next section

STRUCTURAL_SCORE calculation:
  phases_present = count of phases present (0-7)
  quality_average = average quality score across present phases (1-10)
  STRUCTURAL_SCORE = (phases_present / 7 * 5) + (quality_average / 10 * 5)
  Maximum: 10.0
```

### Step 3: Compare Depth Match

```
DIMENSION 2: DEPTH MATCH — Is the narrative at appropriate depth for its communication type?

DEPTH EXPECTATIONS by communication type:

  scientific_explanation:
    Expected depth: HIGH — specific processes named, mechanisms explained accessibly
    TIER1 benchmark: Gundry names Citrate Lyase enzyme, explains lectin-gut wall interaction,
                     cites institutional research, provides specific biological pathways
    Check: Does the generated narrative reach similar biological/process specificity?

  direct_confrontation:
    Expected depth: MEDIUM-HIGH — blunt assertions backed by clear evidence
    TIER1 benchmark: Haney gives precise geometric explanation (clubface-to-path)
                     but doesn't need extensive scientific citation
    Check: Does the narrative confront directly and provide sufficient backing?

  data_presentation:
    Expected depth: VERY HIGH — cascading statistics, verifiable data points
    TIER1 benchmark: Stansberry provides dollar amounts, dates, percentages,
                     policy references, historical data sequences
    Check: Does the narrative provide comparable data density?

  conspiracy_reveal:
    Expected depth: HIGH — documented evidence, specific timelines, named actors
    TIER1 benchmark: Gundry references 1950 conference, Stansberry names Fed officials
    Check: Does the narrative provide documented conspiracy evidence?

  not_your_fault:
    Expected depth: MEDIUM — focus is on emotional release, not exhaustive proof
    TIER1 benchmark: Montezuma provides enough cause explanation to justify absolution
    Check: Does the narrative provide sufficient externalization evidence?

DEPTH_SCORE:
  Compare generated narrative depth to selected comparison controls
  Score on 1-10 scale:
    8-10: Matches or exceeds TIER1 depth
    6-7: Approaches TIER1 depth; minor gaps
    4-5: Noticeably shallower than TIER1 controls
    1-3: Significantly insufficient depth
```

### Step 4: Compare Evidence Density

```
DIMENSION 3: EVIDENCE DENSITY — Is evidence density comparable to elite controls?

COUNT evidence points in generated narrative:
  FROM evidence_integration_report:
    total_evidence_points: integer
    patterns_used: [which of 7 patterns]

TIER1 BENCHMARKS:
  Gundry Dark Energy: ~12-15 evidence points across root cause section
    Patterns: paradox_proof, institutional_name_drop, specific_number,
              geographic_historical_proof, person_as_proof, villain's_own_data
    Diversity: 6 of 7 patterns used

  Haney: ~5-7 evidence points (direct confrontation uses fewer)
    Patterns: person_as_proof, paradox_proof, specific_number
    Diversity: 3 of 7 patterns used

  Stansberry: ~15-20 evidence points (data-heavy by design)
    Patterns: specific_number, villain's_own_data, institutional_name_drop,
              visual_demonstration, geographic_historical_proof
    Diversity: 5 of 7 patterns used

COMPARISON:
  Compare generated evidence count to appropriate benchmark:
    IF communication type is data_presentation: benchmark = 15+
    IF communication type is scientific_explanation: benchmark = 10+
    IF communication type is direct_confrontation: benchmark = 5+
    IF other types: benchmark = 8+

  Compare pattern diversity:
    TIER1 average: 4-5 of 7 patterns used
    Generated: [count] of 7 patterns used

EVIDENCE_SCORE:
  density_match = (generated_count / benchmark) * 5  # max 5
  diversity_match = (patterns_used / 5) * 5  # max 5 (benchmarked at 5)
  EVIDENCE_SCORE = density_match + diversity_match  # max 10
  CAP at 10.0
```

### Step 5: Compare Emotional Arc Quality

```
DIMENSION 4: EMOTIONAL ARC QUALITY — Does the arc match TIER1 sophistication?

FROM emotional_arc_design:
  arc_type: string
  phase_targets: [7 targets with emotions and intensities]
  peak_moment: object

COMPARISON against TIER1 arc execution:

  ARC SHAPE CHECK:
    TIER1 controls show SHAPED intensity curves — rise, peak, resolve
    Generated narrative should show the same shape
    FLAT arcs = failure (no emotional engagement)
    ALL-PEAK arcs = failure (emotional exhaustion)

  PEAK MOMENT QUALITY:
    TIER1 controls have ONE clear peak moment — worldview shift
    Generated narrative: Is the peak moment clear and powerful?
    Is it at the designed phase? (usually Phase 4)
    Does it deliver the emotional target?

  ENDING STATE QUALITY:
    TIER1 controls end root cause section with reader RECEPTIVE to mechanism
    Generated narrative: Does the ending state match?
    Reader should exit wanting the solution, not in despair or apathy

  EMOTIONAL SPECIFICITY:
    TIER1 controls target SPECIFIC emotions (not "feels good" but "vindicated frustration")
    Generated narrative: Are emotional targets specific?

EMOTIONAL_ARC_SCORE:
  arc_shape: 1-10 (shaped vs. flat)
  peak_quality: 1-10 (clear, powerful, positioned)
  ending_state: 1-10 (receptive to mechanism)
  emotional_specificity: 1-10 (named emotions vs. vague feelings)
  EMOTIONAL_ARC_SCORE = average of 4 sub-scores
```

### Step 6: Compare Specificity Level

```
DIMENSION 5: SPECIFICITY LEVEL — Is the language as specific as elite controls?

TIER1 SPECIFICITY BENCHMARKS:

  NAMING specificity:
    Gundry names: lectins, Citrate Lyase, intestinal permeability, gut microbiome,
                  whole wheat, tomatoes, beans, soy — SPECIFIC substances and processes
    Stansberry names: Federal Reserve, Nixon, 1971, gold standard, specific debt figures
    Haney names: clubface-to-path, impact geometry, specific angular relationships

  NUMBER specificity:
    Gundry: "10-12 pounds of impacted fecal matter," "300+ peer-reviewed articles"
    Stansberry: "$4.7 trillion," "97% devaluation since 1913"
    Montezuma: "70% of immune system in the gut"

  TEMPORAL specificity:
    Gundry: "Since 1950," "16 years as chairman"
    Stansberry: "Since 1971," "Since Nixon"
    Haney: "50 years of wrong instruction"

COMPARISON:
  Count named specifics in generated narrative:
    Named substances/processes: integer
    Specific numbers: integer
    Temporal references: integer
    Named entities: integer

  Compare against same-niche benchmark

SPECIFICITY_SCORE:
  naming: 1-10 (specific terms vs. vague categories)
  numbers: 1-10 (specific figures vs. vague quantities)
  temporal: 1-10 (dates and durations vs. "recently" or "long time")
  entities: 1-10 (named actors vs. "the industry" or "experts")
  SPECIFICITY_SCORE = average of 4 sub-scores
```

### Step 7: Document Differentiation

```
CRITICAL: The narrative must MATCH elite patterns but NOT BE a copy.

DIFFERENTIATION CHECK:
  □ Root cause is DIFFERENT from comparison controls
    (not the same root cause as Gundry, Haney, etc.)
  □ Anchor phrase is ORIGINAL
    (not a reused phrase from TIER1 controls)
  □ Evidence is DIFFERENT
    (uses different specific evidence, even if same patterns)
  □ Villain is DIFFERENT or differently framed
    (not copying Gundry's exact villain narrative)
  □ Voice is appropriate for THIS campaign
    (not mimicking another control's specific voice)

DIFFERENTIATION DOCUMENTATION:
  matches_pattern_from: [list of TIER1 controls whose patterns are followed]
  differs_in:
    root_cause: "Different root cause — [description]"
    anchor_phrase: "Original anchor phrase — [phrase]"
    evidence: "Different evidence set — [summary]"
    villain: "Different or differently framed villain — [description]"
    voice: "Voice adapted for [this campaign's communication type]"

  differentiation_score: 1-10
    8-10: Clearly original while following elite patterns
    5-7: Mostly original with some derivative elements
    1-4: Too closely resembles a specific TIER1 control
```

### Step 8: Calculate Comparison Score and Compile Report

```
CALCULATE overall_comparison_score:
  STRUCTURAL_SCORE: float (0-10)
  DEPTH_SCORE: float (0-10)
  EVIDENCE_SCORE: float (0-10)
  EMOTIONAL_ARC_SCORE: float (0-10)
  SPECIFICITY_SCORE: float (0-10)

  WEIGHTS:
    structural: 0.20
    depth: 0.20
    evidence: 0.20
    emotional_arc: 0.20
    specificity: 0.20

  overall_comparison_score = (STRUCTURAL * 0.20) + (DEPTH * 0.20) +
                             (EVIDENCE * 0.20) + (EMOTIONAL_ARC * 0.20) +
                             (SPECIFICITY * 0.20)

  MINIMUM REQUIRED: 6.0

COMPILE vault_comparison_report:
  comparison_controls: [string]  # which controls were compared against
  selection_rationale: string

  dimension_scores:
    structural_match: float
    depth_match: float
    evidence_density: float
    emotional_arc_quality: float
    specificity_level: float

  overall_comparison_score: float
  minimum_met: boolean  # >= 6.0

  differentiation:
    differentiation_score: float
    matches_pattern_from: [string]
    differs_in: object

  strengths: [string]  # where narrative excels compared to TIER1
  weaknesses: [string]  # where narrative falls short
  improvement_recommendations: [string]  # specific fixes to raise score

  pass: boolean  # overall >= 6.0 AND differentiation >= 5.0

WRITE to: layer-4-outputs/vault_comparison_report.json
```

## QUALITY CRITERIA

1. **Comparison score >= 6.0** — below 6.0 means the narrative does not match TIER1 quality level
2. **Differentiation score >= 5.0** — below 5.0 means the narrative is too derivative
3. **All 5 dimensions scored** — no dimension can be skipped
4. **Comparison controls are relevant** — same niche or adjacent niche controls used
5. **Strengths AND weaknesses documented** — honest assessment of where the narrative excels and falls short
6. **Improvement recommendations are specific** — not "write better" but "add 3 more specific numbers to Phase 4"
7. **Differentiation documented** — clear evidence that the narrative follows patterns but is original

---

## FAILURE MODES

| Failure | Severity | Detection | Handling |
|---------|----------|-----------|----------|
| Overall comparison score < 6.0 | HIGH | Score below minimum | Identify lowest dimension(s); recommend specific fixes |
| Differentiation score < 5.0 | MODERATE | Too derivative of TIER1 control | Rewrite distinctive elements; change specific evidence and anchor phrase |
| Structural match low | HIGH | Missing phases or poor phase quality | Return to Layer 2 for phase rewriting |
| Evidence density too low | MODERATE | Far fewer evidence points than benchmark | Return to 3.1 for additional evidence integration |
| Specificity too low | HIGH | Vague language compared to TIER1 benchmarks | Return to Layer 2 writers with specificity requirements |
| No relevant comparison controls | LOW | Niche not represented in TIER1 vault | Use cross-niche controls with noted limitation |
| Emotional arc flat compared to TIER1 | MODERATE | Arc shape doesn't match TIER1 sophistication | Return to 1.3 for arc redesign or Layer 2 for emotional target reinforcement |
