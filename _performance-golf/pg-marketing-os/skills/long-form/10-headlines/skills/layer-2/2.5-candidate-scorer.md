# 2.5-candidate-scorer.md

> **Layer:** 2 (Generation)
> **Skill:** 10-headlines
> **Purpose:** Score all headline candidates on 6-dimension rubric, select top 5 for refinement
> **Inputs:** All format-adapted candidates from 2.4
> **Outputs:** Scored candidates, top 5 selected for Layer 3

---

## FUNCTION

Apply a consistent 6-dimension scoring rubric to ALL headline candidates, rank them, and select the top 5 for refinement in Layer 3. This is the quality gate between generation and refinement.

**Key Principle:** Scoring must be OBJECTIVE and CONSISTENT. Every candidate gets scored on the same rubric. No favorites, no shortcuts, no "I just feel like this one is better."

---

## THE 6-DIMENSION SCORING RUBRIC

### Dimension 1: CLARITY (Weight: 15%)

**Definition:** How clearly does the headline communicate its promise?

| Score | Description |
|-------|-------------|
| 9-10 | Crystal clear promise, no confusion possible |
| 7-8 | Clear promise, minor ambiguity acceptable |
| 5-6 | Promise present but requires interpretation |
| 3-4 | Confusing, promise unclear |
| 1-2 | Incomprehensible or misleading |

**Scoring Questions:**
- Can a reader understand the core promise in 3 seconds?
- Is there any ambiguity about what's being offered?
- Would two readers interpret this the same way?

---

### Dimension 2: CURIOSITY (Weight: 25%)

**Definition:** How strongly does the headline create a need to know more?

| Score | Description |
|-------|-------------|
| 9-10 | Irresistible curiosity gap, MUST read more |
| 7-8 | Strong curiosity, very likely to continue |
| 5-6 | Moderate curiosity, might continue |
| 3-4 | Weak curiosity, easy to scroll past |
| 1-2 | No curiosity generated |

**Scoring Questions:**
- Does this create an open loop that demands resolution?
- Would I stop scrolling to read this?
- Is the curiosity RELEVANT to the target audience's desires?

---

### Dimension 3: SPECIFICITY (Weight: 20%)

**Definition:** How concrete and specific is the headline?

| Score | Description |
|-------|-------------|
| 9-10 | Multiple specific elements (numbers, names, timeframes) |
| 7-8 | At least one strong specific element |
| 5-6 | Generic but not vague |
| 3-4 | Vague claims, no specifics |
| 1-2 | Completely abstract |

**Scoring Questions:**
- Are there specific numbers? (7 days, 47 pounds, 17 cents)
- Are there named entities? (Harvard, Dr. Smith, Japanese)
- Are there concrete timeframes? (overnight, in 30 minutes)

---

### Dimension 4: EMOTIONAL IMPACT (Weight: 20%)

**Definition:** How strongly does the headline trigger an emotional response?

| Score | Description |
|-------|-------------|
| 9-10 | Visceral emotional response (fear, hope, anger, desire) |
| 7-8 | Clear emotional engagement |
| 5-6 | Mild emotional response |
| 3-4 | Emotionally flat |
| 1-2 | No emotional content |

**Scoring Questions:**
- Does this trigger a gut reaction?
- Which emotion does it target? How strongly?
- Would the target reader FEEL something reading this?

---

### Dimension 5: SCHEMA DISTANCE (Weight: 10%)

**Definition:** How much does the headline violate expectations while remaining plausible?

| Score | Description |
|-------|-------------|
| 9-10 | Strong violation, stops you in your tracks, but believable |
| 7-8 | Good violation, surprising but plausible |
| 5-6 | Mild surprise, somewhat expected |
| 3-4 | No surprise, predictable |
| 1-2 | Either too expected OR too unbelievable |

**Scoring Questions:**
- Does this say something unexpected?
- Is the unexpected claim plausible with support?
- Is there a "wait, really?" moment?

---

### Dimension 6: LEAD CONNECTION (Weight: 10%)

**Definition:** How well does this headline set up the lead and body copy?

| Score | Description |
|-------|-------------|
| 9-10 | Perfect setup for lead, multiple connection points |
| 7-8 | Clear path to lead, good connection |
| 5-6 | Adequate connection, will require bridging |
| 3-4 | Weak connection, lead will feel disconnected |
| 1-2 | No clear path to body copy |

**Scoring Questions:**
- Can the lead directly fulfill or extend this headline's promise?
- Is there a clear emotional trajectory from headline to lead?
- Does this headline create expectations the copy can deliver?

## EXECUTION PROTOCOL

### Step 1: Collect All Candidates

```
GATHER all candidates from:
  - 2.1 formula_candidates
  - 2.2 vault_candidates
  - 2.3 schema_violation_candidates
  - 2.4 format_adapted_candidates

CREATE unified scoring pool
REMOVE exact duplicates
```

### Step 2: Score Each Candidate

```
FOR each candidate in pool:

  SCORE each dimension 1-10:
    clarity_score = [1-10]
    curiosity_score = [1-10]
    specificity_score = [1-10]
    emotional_impact_score = [1-10]
    schema_distance_score = [1-10]
    lead_connection_score = [1-10]

  CALCULATE weighted_total:
    weighted_total = (
      clarity_score * 0.15 +
      curiosity_score * 0.25 +
      specificity_score * 0.20 +
      emotional_impact_score * 0.20 +
      schema_distance_score * 0.10 +
      lead_connection_score * 0.10
    )

  RECORD all scores and total
```

### Step 3: Rank and Filter

```
SORT candidates by weighted_total (descending)

FILTER:
  - REMOVE any candidate with weighted_total < 6.0
  - REMOVE any candidate with any dimension < 4.0 (fatal weakness)

SELECT top 5 candidates for Layer 3 refinement
```

### Step 4: Diversity Check

```
REVIEW top 5 for diversity:
  - Are all 5 using the same pattern? (BAD)
  - Are all 5 from the same generator? (BAD)
  - Is there variety in approach? (GOOD)

IF top 5 lacks diversity:
  - Swap out similar candidates for diverse alternatives from top 10
  - Maintain minimum quality threshold (6.5+)
```

## QUALITY GATES

### Gate 1: Minimum Candidates
```
REQUIRE: At least 5 candidates score above 6.0
IF FAIL: Return to generators for more candidates
```

### Gate 2: No Fatal Weaknesses in Top 5
```
REQUIRE: All top 5 have no dimension below 4.0
IF FAIL: Replace weak candidates with next-highest
```

### Gate 3: Diversity Minimum
```
REQUIRE: Top 5 includes candidates from at least 2 generators
IF FAIL: Swap for diversity while maintaining quality
```

---

## ANTI-PATTERNS (FORBIDDEN)

1. **Subjective Favoritism**
   - "I just like this one better" is not a score
   - Every score must have rationale

2. **Dimension Skipping**
   - All 6 dimensions must be scored
   - No "this one is obviously good" shortcuts

3. **Inflated Scoring**
   - 10s are RARE, not common
   - Most headlines score 6-8

4. **Ignoring Fatal Weaknesses**
   - A single dimension at 2-3 disqualifies
   - Can't average away a fatal flaw
