# 2.1-formula-based-generator.md

> **Layer:** 2 (Generation)
> **Skill:** 10-headlines
> **Purpose:** Generate headline candidates from proven headline formulas
> **Inputs:** `big_idea_distilled`, `pattern_type_selected`, `curiosity_architecture`, `schema_distance_target`
> **Outputs:** 3-4 formula-based headline candidates

---

## FUNCTION

Generate headline candidates by applying proven headline FORMULAS to the distilled Big Idea. This skill uses template patterns as starting points, NOT as rigid structures to fill in.

**Key Principle:** Formulas are scaffolding, not cages. The best headlines RIFF on formulas rather than follow them mechanically.

---

## FORMULA LIBRARY

### Tier 1: Universal Formulas (Work in Any Niche)

#### Formula F1: How To [Benefit] Without [Pain/Effort]
```
TEMPLATE: How to [DESIRED_OUTCOME] without [COMMON_OBSTACLE]
EXAMPLE: "How to Lose 20 Pounds Without Giving Up Your Favorite Foods"
STRENGTH: Dual promise (get benefit + avoid pain)
USE WHEN: Clear benefit + clear objection exists
```

#### Formula F2: The Secret of [Desired Outcome]
```
TEMPLATE: The Secret of [DESIRABLE_STATE/ACHIEVEMENT]
EXAMPLE: "The Secret of Great Business in Bad Times"
STRENGTH: Implies insider knowledge, creates curiosity
USE WHEN: Mechanism can be positioned as "hidden"
```

#### Formula F3: [Number] [Things] That [Benefit/Warning]
```
TEMPLATE: [SPECIFIC_NUMBER] [CATEGORY] that [PROMISE/THREAT]
EXAMPLE: "7 Foods That Fight Cancer" / "5 Mistakes That Kill Your Retirement"
STRENGTH: Specificity + completeness implied
USE WHEN: Content is listable, specificity adds credibility
```

#### Formula F4: Why [Common Belief] Is Wrong
```
TEMPLATE: Why [ACCEPTED_WISDOM] is [WRONG/KILLING YOU/COSTING YOU]
EXAMPLE: "Why Everything You Know About Cholesterol Is Dead Wrong"
STRENGTH: Pattern interrupt, contrarian positioning
USE WHEN: Big Idea challenges conventional wisdom
```

#### Formula F5: [Authority] Reveals [Secret/Discovery]
```
TEMPLATE: [CREDENTIALED_EXPERT] reveals [HIDDEN_KNOWLEDGE]
EXAMPLE: "Harvard Doctor Reveals the Real Cause of Heart Disease"
STRENGTH: Authority transfer + curiosity
USE WHEN: Strong authority figure available
```

### Tier 2: Emotional Formulas

#### Formula F6: They Laughed When I [Did X]... But When I [Result]
```
TEMPLATE: They laughed when I [ACTION]... but when I [SURPRISING_RESULT]
EXAMPLE: "They Laughed When I Sat Down at the Piano... But When I Started to Play!"
STRENGTH: Vindication narrative, underdog story
USE WHEN: Transformation story available, skepticism angle works
```

#### Formula F7: Warning: [Threat] Could [Consequence]
```
TEMPLATE: WARNING: [HIDDEN_DANGER] could [DEVASTATING_OUTCOME]
EXAMPLE: "Warning: This Common Supplement Could Be Destroying Your Liver"
STRENGTH: Fear activation, protective instinct
USE WHEN: Legitimate threat exists, fear-based approach appropriate
```

#### Formula F8: What [Group] Know That You Don't
```
TEMPLATE: What [SUCCESSFUL_GROUP] know about [TOPIC] that you don't
EXAMPLE: "What Billionaires Know About Money That You Don't"
STRENGTH: FOMO, insider knowledge positioning
USE WHEN: Aspirational group reference available
```

### Tier 3: Specificity Formulas

#### Formula F9: [Specific Result] in [Specific Time]
```
TEMPLATE: [QUANTIFIED_BENEFIT] in [BELIEVABLE_TIMEFRAME]
EXAMPLE: "Add 30 Yards to Your Drive in 7 Days"
STRENGTH: Measurable, time-bound promise
USE WHEN: Specific numbers available and defensible
```

#### Formula F10: The [Adjective] [Mechanism] That [Benefit]
```
TEMPLATE: The [SURPRISING_ADJECTIVE] [MECHANISM_NAME] that [SPECIFIC_BENEFIT]
EXAMPLE: "The Amazing 17-Cent Circulation Miracle"
STRENGTH: Mechanism naming + benefit clarity
USE WHEN: Named mechanism exists, adjective adds intrigue
```

## EXECUTION PROTOCOL

### Step 1: Formula Selection (Match to Pattern Type)

```
LOAD pattern_type_selected from Layer 1

MAP pattern_type to formula_set:
  IF pattern_type = "curiosity":
    PRIMARY: F2, F4, F8
    SECONDARY: F5, F10

  IF pattern_type = "benefit":
    PRIMARY: F1, F9, F3
    SECONDARY: F10, F5

  IF pattern_type = "question":
    PRIMARY: F4 (as question), F8 (as question)
    SECONDARY: F1 (as question)

  IF pattern_type = "warning":
    PRIMARY: F7, F4
    SECONDARY: F3 (negative framing)

  IF pattern_type = "story_hook":
    PRIMARY: F6
    SECONDARY: F5, F2

  IF pattern_type = "contrarian":
    PRIMARY: F4, F8
    SECONDARY: F2, F7

SELECT 3-4 formulas from mapped set
```

### Step 2: Big Idea Injection

```
FOR each selected_formula:

  EXTRACT from big_idea_distilled:
    - headline_core → PRIMARY_CONTENT
    - mechanism_hint → MECHANISM_NAME (if applicable)
    - promise_compression → BENEFIT_STATEMENT
    - emotional_hook → EMOTIONAL_ANGLE

  INJECT into formula template:
    - Replace [DESIRED_OUTCOME] with promise_compression
    - Replace [MECHANISM] with mechanism_hint
    - Replace [AUTHORITY] with available authority
    - Replace [NUMBER] with specific proof number

  REFINE:
    - Adjust word count (aim for 8-15 words for main headline)
    - Ensure natural language flow (not template-sounding)
    - Verify Big Idea is expressed, not just hinted
```

### Step 3: Format Adaptation

```
CHECK campaign_context.format:

IF VSL:
  - Favor shorter headlines (8-12 words)
  - Question format often works best
  - Must work as text above video

IF magalog/print:
  - Can be longer (12-20 words)
  - Subheadline expected
  - Can use more complex structures

IF advertorial:
  - Must feel editorial, not promotional
  - News/discovery framing preferred
  - Avoid obvious sales language
```

### Step 4: Quality Check

```
FOR each generated_candidate:

  VERIFY:
    [ ] Big Idea is clearly expressed (not just hinted)
    [ ] Promise is specific, not generic
    [ ] Emotional hook is present
    [ ] Length is appropriate for format
    [ ] Does NOT sound like template fill-in
    [ ] Would stop target reader

  IF any check fails:
    REVISE candidate or DISCARD
```

## ANTI-PATTERNS (FORBIDDEN)

1. **Template Stuffing** — Headlines that obviously follow formula structure
   - BAD: "The Secret of Weight Loss" (too generic)
   - GOOD: "The Secret of Great Business in Bad Times" (specific, unexpected)

2. **Benefit Without Mechanism** — Generic promises without unique angle
   - BAD: "How to Make More Money"
   - GOOD: "How a 17-Cent Capsule Saved My Retirement"

3. **Copying Instead of Riffing** — Using formulas as rigid templates
   - Formulas are STARTING POINTS
   - The best headlines break formula rules creatively

4. **Ignoring Big Idea** — Headlines that could apply to any product
   - Every headline must EXPRESS this specific Big Idea
   - If you could swap in a different product, it's too generic
