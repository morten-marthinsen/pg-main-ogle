# 1.2-pattern-type-selector.md

> **Layer:** 1 (Architecture)
> **Skill:** 10-headlines
> **Purpose:** Select optimal headline pattern type based on niche + vault patterns
> **Outputs:** `selected_pattern` with type, format, and vault reference

---

## FUNCTION

Analyze the Big Idea, niche context, and vault pattern frequencies to select the optimal headline pattern type. Consider what works in this niche while avoiding oversaturation.

---

## PATTERN TYPE TAXONOMY

### Primary Pattern Types

| Type | Description | Best For | Example |
|------|-------------|----------|---------|
| `benefit` | Direct promise of outcome | Clear, specific benefits | "Lose 20 Pounds in 30 Days" |
| `curiosity` | Creates information gap | Mechanism reveals, secrets | "The Strange Reason Your Diet Failed" |
| `question` | Engages reader directly | Problem callouts, identification | "Do You Make These Mistakes?" |
| `news` | Positions as new information | Discoveries, research findings | "Scientists Discover..." |
| `secret` | Promises hidden knowledge | Insider info, suppressed cures | "What Wall Street Doesn't Want You to Know" |
| `warning` | Alerts to threat | Fear-based, avoidance | "Warning: Your Retirement Is at Risk" |
| `story_hook` | Opens narrative loop | Story-driven leads | "They Laughed When I Sat Down..." |
| `command` | Direct instruction | Action-oriented | "Stop Making This Mistake" |
| `contrarian` | Challenges conventional wisdom | Paradigm shifts | "Forget Everything You've Been Told About..." |
| `prophecy` | Predicts future event | Financial, societal trends | "The Coming Crisis of 2025" |

### Format Patterns

| Format | Structure | Use Case |
|--------|-----------|----------|
| `statement` | Declarative sentence | Strong claims, discoveries |
| `question` | Interrogative | Engagement, identification |
| `how_to` | "How to [benefit]" | Educational, self-help |
| `number_lead` | "[Number] ways/secrets/etc" | Lists, compilations |
| `story_hook` | Narrative opening | Story-driven copy |
| `command` | Imperative | Action-oriented |
| `news_flash` | Breaking news format | Timely offers |
| `quote` | Attributed statement | Authority-driven |

## SELECTION ALGORITHM

### Step 1: Niche Pattern Analysis

```
LOAD niche_patterns from vault_intelligence
IDENTIFY niche from upstream_context

FOR each pattern_type in niche_patterns[niche]:
  CALCULATE:
    - frequency_score: How often this pattern appears in niche
    - success_score: Performance of this pattern in niche
    - saturation_risk: Is this pattern oversaturated?

RANK patterns by (success_score * (1 - saturation_risk))
```

### Step 2: Big Idea Fit Analysis

```
FOR each candidate_pattern:
  EVALUATE fit with distilled_big_idea:
    - Does pattern express core_promise effectively?
    - Does pattern accommodate mechanism_hint?
    - Does pattern leverage emotional_hook?

  SCORE fit from 1-10
```

### Step 3: Format Compatibility Check

```
CHECK campaign format (VSL, print, advertorial):
  IF VSL:
    FAVOR: question, story_hook, curiosity
    AVOID: long headlines, complex structures

  IF print/magalog:
    FAVOR: benefit, news, warning
    ALLOW: longer headlines, subheadlines

  IF advertorial:
    FAVOR: news, question, story_hook
    REQUIRE: native appearance
```

### Step 4: Schema Distance Calibration

```
CHECK rsf_scores.schema_distance from upstream:
  IF schema_distance target > 7:
    FAVOR: contrarian, curiosity, prophecy
    AVOID: conventional benefit statements

  IF schema_distance target 5-7:
    FAVOR: curiosity, question, secret
    BALANCE: fresh with familiar

  IF schema_distance target < 5:
    FAVOR: benefit, how_to, news
    PRIORITIZE: clarity over surprise
```

### Step 5: Final Selection

```
COMBINE scores:
  final_score = (niche_fit * 0.3) + (big_idea_fit * 0.4) +
                (format_fit * 0.2) + (schema_calibration * 0.1)

SELECT top 2 pattern types
RECOMMEND primary with backup
```

## NICHE-SPECIFIC RECOMMENDATIONS

### Health Headlines

**High Performing:**
- Doctor authority + discovery
- Symptom callout + solution tease
- Mechanism mystery + promise

**Saturated (use carefully):**
- "Breakthrough" claims
- "Doctors shocked" formulas
- Generic "natural cure" promises

**Whitespace:**
- Patient perspective headlines
- Contrarian health advice
- Specific mechanism naming

### Financial Headlines

**High Performing:**
- Prophecy + insider knowledge
- Specific prediction + timeframe
- Conspiracy + protection offer

**Saturated (use carefully):**
- "Secret investment" claims
- "Wall Street" references
- Generic wealth promises

**Whitespace:**
- Currency/macro angles
- Historical pattern recognition
- Alternative investment mechanisms

### Golf Headlines

**High Performing:**
- Specific yardage promises
- Single swing fix
- Tour pro authority

**Saturated (use carefully):**
- "Add 30 yards" claims
- Generic "secret" promises
- Equipment-focused headlines

**Whitespace:**
- Body mechanics angles
- Mental game elements
- Age-specific promises

---

## FAILURE MODES

| Failure | Diagnosis | Remediation |
|---------|-----------|-------------|
| No clear winner | Multiple patterns tie | Use Big Idea as tiebreaker |
| All patterns saturated | Niche oversaturated | Seek contrarian/whitespace angle |
| Pattern doesn't fit format | Mismatch with campaign type | Select format-compatible alternative |
| Low confidence scores | Insufficient vault data | Use teaching-based defaults |
