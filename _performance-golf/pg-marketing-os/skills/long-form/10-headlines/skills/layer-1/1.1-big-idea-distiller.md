# 1.1-big-idea-distiller.md

> **Layer:** 1 (Architecture)
> **Skill:** 10-headlines
> **Purpose:** Extract headline-ready core from Big Idea
> **Outputs:** `distilled_big_idea` object with headline-ready expressions

---

## FUNCTION

Transform the Big Idea's full expression into headline-ready components. Identify the core promise, mechanism hint, and emotional hook that can be compressed into headline form.

---

## DISTILLATION PROTOCOL

### Step 1: Identify Core Promise

Extract the single most compelling promise from the Big Idea:

```
BIG IDEA: "A forgotten compound from WWII naval research flushes arterial
plaque like a pipe cleaner, potentially adding decades to your life without
surgery, drugs, or lifestyle changes."

CORE PROMISE: "Add decades to your life"
MECHANISM HINT: "Forgotten WWII compound"
EMOTIONAL HOOK: "No surgery, drugs, or lifestyle changes"
```

### Step 2: Compress to Headline Length

Maximum headline length targets:
- Main headline: 8-15 words ideal, 20 words maximum
- Subheadline: 15-25 words ideal

Compression techniques:
1. **Cut backstory:** Save origin story for lead
2. **Imply mechanism:** Hint rather than explain
3. **Strongest promise forward:** Lead with most compelling element
4. **Power word density:** Every word must carry weight

### Step 3: Identify Headline-Ready Expressions

Extract phrases that work directly in headlines:

| Big Idea Element | Headline Expression |
|------------------|---------------------|
| Mechanism name | "The [Mechanism Name] Discovery" |
| Core promise | "[Specific Benefit] in [Timeframe]" |
| Enemy/Villain | "Why [Enemy] Doesn't Want You to Know" |
| Contrarian angle | "The [Opposite of Conventional Wisdom]" |
| Authority source | "[Authority Title] Reveals..." |

### Step 4: Map to Headline Formulas

Identify which formulas fit the Big Idea:

```
IF Big Idea contains mechanism discovery:
  MATCH: "Secret Of", "Discovery", "Breakthrough"

IF Big Idea contains contrarian angle:
  MATCH: "Warning", "They Laughed", "Confessions"

IF Big Idea contains specific promise:
  MATCH: "How To", "Number + Noun", "Better Than"

IF Big Idea contains enemy/villain:
  MATCH: "Why [Enemy]", "What [Authority] Doesn't Tell You"
```

## FAILURE MODES

| Failure | Diagnosis | Remediation |
|---------|-----------|-------------|
| Cannot compress to 15 words | Big Idea too complex | Return to 06-big-idea for simplification |
| No clear promise | Big Idea lacks benefit | Extract implicit benefit or return |
| Mechanism unexpressible | Mechanism too technical | Use category/metaphor instead |
| Multiple competing promises | Big Idea unfocused | Select single primary promise |
