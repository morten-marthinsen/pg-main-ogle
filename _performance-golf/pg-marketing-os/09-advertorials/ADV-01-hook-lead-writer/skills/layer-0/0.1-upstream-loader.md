# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Load advertorial strategy and campaign brief for hook/lead generation
> **Outputs:** `upstream_context` object with strategy and brief data

---

## FUNCTION

Load the advertorial strategy (from ADV-00) and campaign brief package. Extract hook-relevant fields: advertorial type, editorial angle, hook direction, voice register, platform constraints, and mechanism details. Assemble context for hook/lead generation.

---

## REQUIRED UPSTREAM PACKAGES

| Package | Source | Critical Fields | Required |
|---------|--------|-----------------|----------|
| `advertorial-strategy.yaml` | ADV-00 | type, angle, voice, compliance, proof, downstream_instructions.adv_01 | YES |
| `campaign-brief-package.json` | 01-core-message | big_idea, mechanism, promise, avatar | YES |

---

## LOADING PROTOCOL

```
FOR advertorial-strategy.yaml:
  1. LOCATE in ADV-00 outputs/
  2. PARSE YAML structure
  3. EXTRACT:
     - advertorial_type (selected type + constraints)
     - editorial_angle (angle type, hook direction, voice register)
     - platform_compliance (headline rules, char limits)
     - proof_inventory (for optional lead proof teasing)
     - downstream_instructions.adv_01 (specific hook/lead instructions)
  4. VALIDATE: type and angle must be non-null

FOR campaign-brief-package.json:
  1. LOCATE in 01-core-message outputs/
  2. PARSE JSON
  3. EXTRACT: big_idea, mechanism, promise, avatar
  4. VALIDATE: big_idea must exist

IF strategy missing: FAIL -- "ADV-00 must complete before ADV-01"
IF brief missing: FAIL -- "Campaign brief required"
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Strategy missing | HALT -- Execute ADV-00 first |
| CRITICAL | Campaign brief missing | HALT -- Execute 01-core-message first |
| CRITICAL | Type or angle null in strategy | HALT -- Strategy incomplete, re-run ADV-00 |
| DEGRADED | Platform compliance missing | Default to Taboola constraints, flag |
