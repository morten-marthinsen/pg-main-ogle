# 0.2-specimen-calibrator.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Load advertorial specimens and extract hook/lead patterns for calibration
> **Outputs:** `specimen_patterns` object with hook patterns and voice markers

---

## FUNCTION

Load advertorial specimens that match the selected advertorial type. Decompose specimen openings into hook patterns, lead structures, voice markers, and editorial techniques. These extracted patterns serve as calibration material -- they define what "real editorial" sounds like, preventing AI-generated hooks from defaulting to generic or promotional patterns.

---

## SPECIMEN LOADING PROTOCOL

### Step 1: Identify Type-Matched Specimens

```
FROM strategy.advertorial_type:
  MAP to specimen collection:
    listicle -> listicle specimens
    native_article -> native article specimens
    blog_style -> blog-style specimens
    product_review -> review specimens
    PAS -> PAS format specimens
    sponsored_content -> sponsored content specimens
    editorial_feature -> editorial feature specimens

  LOAD specimens matching selected type
  MINIMUM: 3 specimens required for calibration
  IDEAL: 5-8 specimens for strong calibration
```

### Step 2: Decompose Hook Patterns

```
FOR each specimen:
  EXTRACT hook pattern:
    - opening_technique: How the hook opens
      (question, statement, statistic, anecdote, scene-setting, contrast)
    - word_count: Hook length in words
    - curiosity_mechanism: What creates the curiosity gap
    - specificity_elements: Concrete details in hook (numbers, names, places)
    - voice_register: Identified voice (journalistic, conversational, expert, peer)
    - product_mention: Where product first appears (paragraph #, word #)
    - editorial_markers: What makes it read as editorial vs ad

  BUILD pattern library:
    - Most common opening technique across specimens
    - Average hook word count
    - Typical curiosity mechanisms for type
    - Specificity density benchmarks
```

### Step 3: Extract Lead Voice Markers

```
FOR each specimen:
  EXTRACT lead characteristics:
    - paragraph_count: Number of lead paragraphs before body
    - voice_transitions: How voice evolves from hook to lead
    - attribution_style: How sources/experts are cited
    - narrative_entry: How story/narrative begins
    - mechanism_teasing: How mechanism is hinted without naming
    - reader_engagement: What keeps reader moving to body
    - editorial_phrases: Specific phrases that signal "article" not "ad"
      Examples:
        "According to researchers at..."
        "A growing body of evidence suggests..."
        "What [name] discovered was..."
        "The findings, published in..."
```

### Step 4: Build Calibration Reference

```
ASSEMBLE calibration_reference:
  - hook_templates: Abstracted patterns from specimen hooks
  - lead_templates: Abstracted patterns from specimen leads
  - voice_markers: Editorial phrases and attribution styles
  - forbidden_patterns: Patterns NOT found in good specimens
    (direct selling, product features, "buy now" language)
  - quality_benchmarks: What "good" looks like for this type
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Zero specimens available | HALT -- Cannot calibrate without specimens |
| DEGRADED | Fewer than 3 specimens | Proceed with caution, flag reduced calibration |
| DEGRADED | No type-matched specimens | Use closest type, flag mismatch |
| WARNING | Specimens low quality | Note quality concerns, prioritize best specimens |
