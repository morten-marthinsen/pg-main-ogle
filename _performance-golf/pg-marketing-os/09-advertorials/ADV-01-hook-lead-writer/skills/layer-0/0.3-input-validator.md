# 0.3-input-validator.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Validate strategy type and angle present before hook/lead generation
> **Outputs:** `validation_result` object with PASS/FAIL status

---

## FUNCTION

Validate that all required inputs for hook/lead generation are present and complete. The critical gate: strategy must have an advertorial type selected AND an editorial angle defined. Without both, hook formula selection and lead structure planning cannot execute.

---

## VALIDATION PROTOCOL

### Step 1: Strategy Validation

```
REQUIRED (FAIL if missing):
  [ ] strategy.type.selected -- valid type enum
  [ ] strategy.angle.type -- valid angle type enum
  [ ] strategy.angle.hook_direction -- non-empty string
  [ ] strategy.angle.voice_register -- valid register enum
  [ ] strategy.platform.headline_max_chars -- positive integer
  [ ] strategy.adv_01_instructions -- non-empty string

IF any REQUIRED field missing:
  STATUS = FAIL
  LOG: "Missing: [field] -- ADV-00 strategy incomplete"
  HALT: Cannot proceed to Layer 1
```

### Step 2: Specimen Validation

```
REQUIRED (FAIL if missing):
  [ ] specimen_patterns.specimens_loaded >= 3
  [ ] specimen_patterns.hook_patterns -- at least 1 pattern extracted
  [ ] specimen_patterns.voice_markers -- at least 1 marker identified

IF specimens insufficient:
  STATUS = FAIL
  LOG: "Insufficient specimen calibration -- load more specimens"
```

### Step 3: Campaign Brief Validation

```
REQUIRED (FAIL if missing):
  [ ] campaign_brief.big_idea -- non-empty
  [ ] campaign_brief.mechanism.name -- non-empty

QUALITY (WARN if missing):
  [ ] campaign_brief.avatar -- populated
  [ ] campaign_brief.promise -- populated
```

---

## OUTPUT STRUCTURE

```json
{
  "validation_result": {
    "status": "PASS|FAIL",
    "strategy_valid": true,
    "specimens_valid": true,
    "brief_valid": true,
    "gate_0_status": "PASS",
    "proceed_to_layer_1": true,
    "warnings": [],
    "blockers": []
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Strategy type missing | HALT -- Re-run ADV-00 |
| CRITICAL | Strategy angle missing | HALT -- Re-run ADV-00 |
| CRITICAL | Specimens < 3 | HALT -- Load more specimens |
| DEGRADED | Brief avatar missing | Proceed with strategy-only, flag |
