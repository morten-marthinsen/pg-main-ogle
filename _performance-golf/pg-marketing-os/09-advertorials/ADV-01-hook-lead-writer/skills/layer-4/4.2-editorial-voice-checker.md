# 4.2-editorial-voice-checker.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Verify editorial (not promotional) voice in selected hook/lead
> **Outputs:** `voice_check_result` object with PASS/FAIL

---

## FUNCTION

Run the final editorial voice check on the selected hook/lead combination. This is the definitive editorial smell test: remove the product name and any brand references -- does the content still read as a genuine article opening? If not, the hook/lead fails regardless of other scores.

---

## VOICE CHECK PROTOCOL

### Step 1: Promotional Language Scan

```
SCAN for promotional language:
  BANNED PHRASES (auto-fail if found):
    - "Buy now", "Order today", "Limited time"
    - "Our product", "Our formula", "We developed"
    - "Best-selling", "#1", "Award-winning" (without attribution)
    - "Don't miss", "Act fast", "While supplies last"
    - "Amazing results", "Incredible formula"
    - "Unlock", "Transform your life", "Revolutionary"

  WARN PHRASES (flag for review):
    - Product name appearing before paragraph 5
    - Direct benefit claims without attribution
    - Price or discount mentions
    - "Clinically proven" without study citation

  COUNT promotional_phrases_found
  IF count > 0: FAIL -- promotional language detected
```

### Step 2: Editorial Smell Test

```
PERFORM smell test:
  1. CREATE redacted version: Remove all product/brand references
  2. READ redacted version as standalone text
  3. EVALUATE: Does this read as a genuine article opening?

  CRITERIA:
    [ ] Could this appear in a health/lifestyle publication? (Y/N)
    [ ] Does it provide value independent of product? (Y/N)
    [ ] Is the voice consistent with the claimed register? (Y/N)
    [ ] Are sources attributed properly? (Y/N)
    [ ] Would a skeptical reader see this as an article? (Y/N)

  IF any criterion = N: FAIL with specific explanation
```

### Step 3: Voice Register Consistency

```
CHECK voice consistency:
  [ ] Hook and lead use same voice register
  [ ] No register drift between paragraphs
  [ ] Voice markers present in at least 3 of 4 lead paragraphs
  [ ] Attribution style consistent throughout
  [ ] Sentence structure matches claimed register

  SCORE voice_consistency (1-10)
  MINIMUM: 7.0
```

### Step 4: Final Determination

```
DETERMINE overall voice status:
  IF promotional_phrases > 0: FAIL
  IF smell_test_failed: FAIL
  IF voice_consistency < 7.0: FAIL
  ELSE: PASS

  PROVIDE specific evidence for determination
```


---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Promotional language found | Banned phrase detected | Remove/replace phrase, re-check |
| Smell test failed | Redacted version reads as ad | Rewrite through editorial lens |
| Voice drift detected | Register changes between sections | Align all sections to single register |
| Voice markers missing | Too few editorial markers | Inject markers per lead structure plan |
