# 4.1-scroll-stop-validator.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Validate hook scroll-stop quality for native ad platforms
> **Outputs:** `scroll_stop_validation` object with PASS/FAIL

---

## FUNCTION

Validate that the selected hook (from Arena) has genuine scroll-stop quality on the target platform. A hook that reads well but doesn't stop the scroll fails its primary mission. This validation checks for pattern interrupt strength, curiosity gap creation, specificity elements, and platform-specific optimization.

---

## VALIDATION PROTOCOL

### Step 1: Pattern Interrupt Check

```
EVALUATE scroll-stop mechanisms:
  [ ] Does hook contradict a common belief? (pattern interrupt)
  [ ] Does hook present a surprising data point? (statistic interrupt)
  [ ] Does hook open mid-scene? (narrative interrupt)
  [ ] Does hook ask a question reader can't ignore? (question interrupt)

  SCORE pattern_interrupt_strength (1-10):
    9-10: Visceral reaction, impossible to scroll past
    7-8: Strong stop, reader would pause
    5-6: Moderate interest, might stop if topic relevant
    3-4: Weak, blends into feed
    1-2: Zero stopping power

  MINIMUM: 7.0
```

### Step 2: Curiosity Gap Check

```
EVALUATE curiosity quality:
  [ ] Is there a specific information gap created?
  [ ] Is the gap relevant to the reader's interests/problems?
  [ ] Is the gap resolvable (reader expects to learn the answer)?
  [ ] Is the gap specific enough to create urgency?

  SCORE curiosity_gap_strength (1-10)
  MINIMUM: 7.0
```

### Step 3: Platform Optimization Check

```
FOR target platform:
  [ ] Headline version within char limit
  [ ] No forbidden patterns used
  [ ] Thumbnail-compatible (if relevant)
  [ ] Native-feed optimized (doesn't scream "ad")

  SCORE platform_optimization (1-10)
  MINIMUM: 7.0
```

### Step 4: Composite Scroll-Stop Score

```
CALCULATE composite:
  pattern_interrupt * 0.40 +
  curiosity_gap * 0.35 +
  platform_optimization * 0.25

  MINIMUM composite: 7.0
  IF composite < 7.0: FAIL -- hook needs strengthening
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Pattern interrupt weak | Score < 7 | Strengthen specific interrupt mechanism |
| Curiosity gap weak | Score < 7 | Sharpen the information gap |
| Platform non-compliant | Exceeds limits or uses forbidden patterns | Edit to comply |
| Composite below threshold | Overall < 7.0 | Return to Arena for stronger candidate |
