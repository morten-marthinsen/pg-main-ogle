# 4.3-output-packager.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Package validated hook/lead as advertorial-lead-draft.md
> **Outputs:** `advertorial-lead-draft.md` file

---

## FUNCTION

Package the validated, human-selected hook/lead combination into the canonical output file (`advertorial-lead-draft.md`) that ADV-02 (body copy) and ADV-04 (assembly) consume. Include body handoff context and editorial voice documentation.

---

## PACKAGING PROTOCOL

### Step 1: Verify Validations Passed

```
CHECK:
  scroll_stop_validation.status == "PASS"
  voice_check_result.status == "PASS"

IF either FAIL: HALT -- cannot package unvalidated output
```

### Step 2: Assemble Lead Draft

```
BUILD advertorial-lead-draft.md:

  ---
  metadata:
    version: "1.0"
    generated_at: "[ISO timestamp]"
    skill_id: "ADV-01-hook-lead-writer"
    project_id: "[from strategy]"
    advertorial_type: "[type]"
    editorial_angle: "[angle]"
    voice_register: "[register]"
    selection_source: "generated|modified|human_provided"
  ---

  ## Hook
  [Selected hook text]

  ## Lead
  [Selected lead paragraphs]

  ## Body Handoff
  - narrative_thread: [What narrative body must continue]
  - voice_established: [What voice register body must maintain]
  - mechanism_status: [not_yet_introduced|hinted|named]
  - reader_expectation: [What reader expects to learn next]
  - last_tease_level: [0|1|2]

  ## Quality Scores
  - scroll_stop: [score]
  - editorial_voice: [score]
  - overall: [score]

  ## Alternatives
  [Ranked list of alternative hook/lead combinations]
```

### Step 3: Validate Package

```
BEFORE writing:
  [ ] Hook text present and non-empty
  [ ] Lead text present, 100-250 words
  [ ] Body handoff section complete
  [ ] Quality scores populated
  [ ] Metadata complete
  [ ] No placeholder text

IF validation fails: FIX specific issues, re-validate
```

### Step 4: Write File

```
WRITE advertorial-lead-draft.md to:
  [project]/09-advertorials/outputs/advertorial-lead-draft.md

VERIFY file exists and is non-empty
LOG: "advertorial-lead-draft.md written at [timestamp]"
```


---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Validation not passed | Packaging before validation | HALT -- run 4.1 and 4.2 first |
| Incomplete handoff | Body handoff section missing fields | Complete handoff data |
| File write failure | Filesystem error | Check permissions, retry |
