# 2.1-hook-generator.md

> **Layer:** 2 (Generation)
> **Skill:** ADV-01-hook-lead-writer
> **Purpose:** Generate hook candidates using selected formulas and specimen calibration
> **Outputs:** `hook_candidates` array with multiple options per formula

---

## FUNCTION

Generate multiple hook candidates (3-5 per formula) using the selected primary, secondary, and wildcard formulas. Each hook must stop the scroll, establish editorial tone in the first sentence, and create a curiosity gap. Hooks are calibrated against specimen patterns and constrained by platform rules.

---

## GENERATION PROTOCOL

### Step 1: Generate Primary Formula Hooks

```
USING primary_formula + specimen_calibration:
  GENERATE 3-5 hook candidates:

  FOR each candidate:
    1. Apply formula pattern to campaign mechanism/big idea
    2. Inject specificity elements (numbers, names, institutions)
    3. Calibrate against specimen voice markers
    4. Verify within platform char limit
    5. Run editorial voice check (no promotional language)
    6. Verify product name absent

  EXAMPLE (discovery_frame formula):
    "Researchers at a major university medical center have identified
    a compound that appears to reverse a process cardiologists have
    long considered permanent."

    CHECK:
    [ ] Scroll-stop quality? YES -- "reverse a permanent process" creates gap
    [ ] Editorial voice? YES -- institutional attribution, hedging language
    [ ] Product name absent? YES
    [ ] Within char limit? YES (58 chars for headline version)
    [ ] Specificity present? YES -- "major university medical center"
```

### Step 2: Generate Secondary Formula Hooks

```
USING secondary_formula + specimen_calibration:
  GENERATE 2-3 hook candidates:

  Same protocol as primary but with different formula pattern
  Must be genuinely different approach, not variation of primary
```

### Step 3: Generate Wildcard Formula Hooks

```
USING wildcard_formula:
  GENERATE 1-2 hook candidates:

  Higher creative freedom -- formula is outside natural affinity
  Must still meet editorial voice and platform constraints
  These provide diversity for Arena competition
```

### Step 4: Score and Filter

```
FOR each generated hook:
  SCORE on 4 quick criteria (1-10):
    - scroll_stop: Would this stop a feed scroller?
    - editorial_voice: Does this read as article, not ad?
    - curiosity_gap: Is there a gap the reader needs to close?
    - specificity: Are there concrete elements?

  FILTER: Remove any scoring below 6.0 on any criterion
  RANK: Order by average score
  PASS: Top 5-8 hooks to lead generation
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| All hooks score < 6.0 | Formula-campaign mismatch | Re-evaluate formula selection |
| Hooks too long for platform | Exceeding char limits | Compress without losing specificity |
| Editorial voice drift | Promotional language in hooks | Rewrite with specimen voice markers |
| Product name in hook | Automatic rejection | Remove product reference, replace with tease |
| Hooks too similar | Lack of diversity | Force different opening words/techniques per hook |
