# 4.2-output-packager.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-04-advertorial-assembly
> **Purpose:** Package assembled advertorial as advertorial-assembled.md
> **Outputs:** `advertorial-assembled.md` file

---

## FUNCTION

Package the validated assembled advertorial into the canonical output file consumed by ADV-05 (editorial review). The file contains the complete advertorial text plus assembly metadata, quality metrics, and editorial review guidance.

---

## PACKAGING PROTOCOL

### Step 1: Verify Final Validation

```
CHECK final_validation.status == "PASS"
IF FAIL: HALT -- cannot package unvalidated assembly
```

### Step 2: Assemble Output File

```
BUILD advertorial-assembled.md:

  ---
  metadata:
    version: "1.0"
    skill_id: "ADV-04-advertorial-assembly"
    advertorial_type: "[type]"
    voice_register: "[register]"
    total_word_count: [count]
    platform: "[platform]"
    cta_count: 1
    disclosure_placed: true
    assembly_validation: "PASS"
  ---

  [Complete assembled advertorial text]

  ---
  ## Assembly Report
  - Voice consistency: [overall score]
  - Narrative continuity: [overall score]
  - Compliance status: [PASS/FAIL]
  - Seam quality: [min seam score]
  - Proof density: [compliant]

  ## Editorial Review Guidance (for ADV-05)
  - Focus areas: [specific areas to review]
  - Known risks: [any flagged issues]
  - Word count status: [within bounds / needs trimming / needs expansion]
```

### Step 3: Write File

```
WRITE to: [project]/09-advertorials/outputs/advertorial-assembled.md
VERIFY: file exists, non-empty, > 5KB
LOG: "advertorial-assembled.md written at [timestamp]"
```

---

## OUTPUT STRUCTURE

```json
{
  "packaging_result": {
    "status": "complete",
    "file_written": "advertorial-assembled.md",
    "word_count": 1088,
    "validation_status": "PASS",
    "editorial_review_guidance_included": true
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Validation not passed | Packaging before validation | HALT -- run 4.1 |
| File write failure | Filesystem error | Check path, retry |
| Incomplete metadata | Assembly report missing | Complete before write |
