# 4.1-assembly-validator.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-04-advertorial-assembly
> **Purpose:** Final validation pass on assembled advertorial
> **Outputs:** `final_validation` with PASS/FAIL and comprehensive report

---

## FUNCTION

Run the final comprehensive validation on the assembled advertorial before packaging. This is the last gate before the piece goes to editorial review (ADV-05). Check that all upstream validations still hold in the assembled context and that assembly itself introduced no issues.

---

## FINAL VALIDATION CHECKLIST

```
SECTION COMPLETENESS:
  [ ] Hook present and non-empty
  [ ] Lead present (3-5 paragraphs)
  [ ] Body present (all sections from type template)
  [ ] Bridge present (3-4 paragraphs)
  [ ] CTA present (exactly 1, text link)
  [ ] Disclosure present (if required by platform)

VOICE CONSISTENCY:
  [ ] voice_consistency_result.status != FAIL
  [ ] All seam scores >= 7
  [ ] Overall register consistency >= 70%

NARRATIVE CONTINUITY:
  [ ] narrative_validation.status != FAIL
  [ ] Zero major gaps
  [ ] Reader journey scores all >= 7

COMPLIANCE:
  [ ] compliance_result.status != FAIL
  [ ] Zero critical violations
  [ ] Word count within bounds
  [ ] Disclosure properly placed

PROOF DENSITY (re-verify):
  [ ] No proof stacking in assembled piece
  [ ] Max 1 proof per paragraph maintained

CTA (re-verify):
  [ ] Exactly 1 CTA in entire assembled piece
  [ ] CTA is text link format
  [ ] CTA language is recommendation, not sales

5 LAWS (final check):
  [ ] Looks like content, not ad
  [ ] Reads like article, not pitch
  [ ] Passes editorial smell test
  [ ] One CTA only, text link
  [ ] Bridge is recommendation, not close
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Section missing | Incomplete assembly | Return to 1.1 |
| Voice failure | Seam score < 7 | Smooth specific seam |
| Narrative gap | Major gap found | Add transitional element |
| Compliance violation | Critical platform issue | Fix violation |
| Proof stacking | Stacking in assembled piece | Redistribute (flag for ADV-05) |
