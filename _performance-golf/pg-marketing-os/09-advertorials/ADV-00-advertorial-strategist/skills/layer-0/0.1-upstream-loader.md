# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-00-advertorial-strategist
> **Purpose:** Load campaign brief and optional A02 hook angles for advertorial strategy
> **Outputs:** `upstream_context` object for downstream skills

---

## FUNCTION

Load and parse the campaign brief package that provides the strategic foundation for advertorial creation. Optionally load A02 hook angle data if available. Validate presence and structure of required fields. Assemble unified context object for brief parsing.

---

## REQUIRED UPSTREAM PACKAGES

| Package | Source Skill | Critical Fields | Required |
|---------|--------------|-----------------|----------|
| `campaign-brief-package.json` | 01-core-message | `big_idea`, `mechanism`, `promise`, `avatar`, `creative_direction`, `proof_inventory` | YES |
| `hook-angles.json` | A02 (if available) | `angles`, `hook_types`, `angle_rationale` | NO |

---

## LOADING PROTOCOL

```
FOR campaign-brief-package.json:
  1. LOCATE: Find package in 01-core-message outputs/ directory
  2. PARSE: Load JSON and validate top-level structure
  3. EXTRACT: Pull all fields relevant to advertorial strategy:
     - big_idea (core expression, type, mechanism angle)
     - mechanism (name, uniqueness, credibility anchors)
     - promise (primary promise, ceiling, proof pairings)
     - avatar (primary pain, primary desire, awareness level)
     - creative_direction (tone, format preferences)
     - proof_inventory (all available proof elements)
     - story_elements (if present: protagonist, conflict, discovery)
  4. VALIDATE: Confirm non-null values for required fields
  5. STORE: Add to upstream_context object

FOR hook-angles.json (OPTIONAL):
  1. LOCATE: Check A02 outputs/ directory
  2. IF FOUND: Load and extract angle data
  3. IF NOT FOUND: Set hook_angles = null, proceed without degradation
  4. LOG: Whether hook angles were available

IF campaign-brief-package missing:
  FAIL with: "Campaign brief not found. Ensure 01-core-message has completed."
  Cannot proceed -- this is a CRITICAL dependency.

IF any critical field null/empty:
  WARN but continue with degraded mode flag
  LOG: "Field [X] is empty. Strategy quality may be reduced."
```

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | campaign-brief-package missing | Cannot proceed -- return to 01-core-message |
| CRITICAL | campaign brief has no big_idea | Cannot proceed -- Big Idea is foundation |
| DEGRADED | mechanism fields empty | Proceed with Big Idea only, flag in strategy |
| DEGRADED | proof_inventory empty | Proceed with narrative-only strategy |
| NORMAL | hook-angles not found | Proceed without -- not required |
