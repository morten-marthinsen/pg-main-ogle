# 1.2-angle-definer.md

> **Layer:** 1 (Architecture)
> **Skill:** ADV-00-advertorial-strategist
> **Purpose:** Define editorial angle and hook direction for the advertorial
> **Outputs:** `editorial_angle` object with angle type, statement, and hook direction

---

## FUNCTION

Define the editorial angle that gives the advertorial its journalistic reason-to-exist. The angle is the "news peg" or "editorial frame" that makes this content feel like it belongs in a publication rather than in an ad. It determines what the hook promises and how the narrative unfolds.

---

## EDITORIAL ANGLE TYPES

| Angle Type | Description | Best For | Hook Pattern |
|------------|-------------|----------|--------------|
| **Curiosity Gap** | Opens information gap reader must close | Unaware audiences, mechanism-forward | "What [experts] are now saying about [topic]" |
| **Pattern Interrupt** | Contradicts what reader believes | Contrarian Big Ideas, sophisticated audiences | "Why [common belief] is wrong according to [source]" |
| **Story-Driven** | Leads with personal narrative | Blog-style, native articles, empathy-first | "[Person] was about to [dire consequence] when..." |
| **Discovery-Framed** | Frames mechanism as recent finding | Mechanism-forward campaigns, news-style | "Researchers at [institution] have identified..." |

---

## ANGLE DEFINITION PROTOCOL

### Step 1: Match Angle to Campaign Elements

```
ANALYZE:
  - Big Idea type -> angle alignment:
      NEW_MECHANISM -> discovery_framed or curiosity_gap
      CONTRARIAN -> pattern_interrupt
      PROPHECY -> curiosity_gap or story_driven
      TRANSFORMATION -> story_driven

  - Audience awareness -> angle alignment:
      unaware -> curiosity_gap or pattern_interrupt
      problem_aware -> story_driven or discovery_framed
      solution_aware -> discovery_framed or pattern_interrupt
      product_aware -> pattern_interrupt (reframe needed)

  - Advertorial type -> angle compatibility:
      listicle -> curiosity_gap preferred
      native_article -> discovery_framed or curiosity_gap
      blog_style -> story_driven preferred
      product_review -> pattern_interrupt or discovery_framed
      PAS -> story_driven or curiosity_gap
      sponsored_content -> discovery_framed preferred
      editorial_feature -> any angle works
```

### Step 2: Craft Angle Statement

```
CRAFT one-sentence angle statement:
  TEMPLATE: "[Angle type] angle exploring [topic] through [frame] lens"

  EXAMPLES:
    Discovery: "Discovery angle exploring joint inflammation through the lens
    of a compound accidentally found during coral reef research"

    Curiosity Gap: "Curiosity gap angle exploring why conventional cholesterol
    advice may be addressing the wrong biomarker entirely"

    Pattern Interrupt: "Pattern interrupt angle challenging the assumption that
    cardio exercise is the best approach for heart health after 50"

    Story-Driven: "Story-driven angle following a retired teacher who reversed
    15 years of cognitive decline using a method her neurologist had never seen"
```

### Step 3: Define Hook Direction

```
BASED ON angle_type + advertorial_type:
  DEFINE:
    - hook_direction: The specific direction the opening hook takes
    - hook_tone: journalistic | personal | provocative | investigative
    - hook_promise: What the reader expects to learn
    - hook_constraint: What the hook must NOT reveal (saves for body)
    - editorial_voice: How the writer "sounds" in the hook
```

### Step 4: Set Voice Register

```
VOICE REGISTER OPTIONS:
  - journalistic: Third-person, objective tone, source-attributed claims
  - conversational: Second-person, peer-to-peer, relatable language
  - expert: Authoritative but accessible, evidence-forward
  - peer: First-person experience, "I discovered" framing

FORBIDDEN REGISTERS:
  - promotional: "This amazing product..."
  - salesy: "Don't miss this opportunity..."
  - infomercial: "But wait, there's more..."

SET voice_register based on:
  advertorial_type + angle_type + audience_sophistication
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Angle reads promotional | Voice register drifted to salesy | Rewrite through editorial lens |
| Angle doesn't fit type | Angle/type mismatch (e.g., story for listicle) | Re-evaluate type or adjust angle |
| Hook direction too revealing | Hook gives away mechanism/product | Constrain hook to tease only |
| No clear editorial peg | Campaign lacks newsworthy frame | Construct frame from mechanism discovery context |
