# 4.3-output-packager.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-03-cta-bridge-writer
> **Purpose:** Package validated bridge/CTA as advertorial-bridge-draft.md
> **Outputs:** `advertorial-bridge-draft.md` file

---

## FUNCTION

Package the validated bridge/CTA combination into the canonical output file consumed by ADV-04 (assembly). Include recommendation quality metrics, CTA validation results, and assembly handoff context.

---

## PACKAGING PROTOCOL

### Step 1: Verify Validations

```
CHECK:
  tone_check_result.status == "PASS"
  cta_validation_result.status == "PASS"

IF either FAIL: HALT -- cannot package
```

### Step 2: Assemble Bridge Draft

```
BUILD advertorial-bridge-draft.md:

  ---
  metadata:
    version: "1.0"
    skill_id: "ADV-03-cta-bridge-writer"
    bridge_frame: "[frame]"
    cta_count: 1
    cta_format: "text_link"
    recommendation_tone: "PASS"
    word_count: [count]
  ---

  ## Bridge
  [Complete bridge paragraphs]

  ## CTA
  - text: "[CTA text]"
  - url: "[traffic destination]"
  - format: text_link
  - position: [paragraph and sentence position]

  ## Assembly Handoff
  - connects_to_body: [how bridge follows body]
  - voice_consistent: [Y/N]
  - recommendation_quality: [score]
  - transition_gradient: [score]

  ## Alternatives
  [Ranked alternative bridge/CTA combinations]
```

### Step 3: Write File

```
WRITE to: [project]/09-advertorials/outputs/advertorial-bridge-draft.md
VERIFY: file exists, non-empty, CTA count = 1
LOG: "advertorial-bridge-draft.md written at [timestamp]"
```

---

## OUTPUT STRUCTURE

```json
{
  "packaging_result": {
    "status": "complete",
    "file_written": "advertorial-bridge-draft.md",
    "word_count": 140,
    "cta_count_verified": 1,
    "recommendation_tone_verified": true,
    "assembly_handoff_complete": true
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Validation not passed | Packaging before validation | HALT -- run 4.1 and 4.2 |
| CTA count wrong | Package has != 1 CTA | Fix before writing |
| File write failure | Filesystem error | Check path, retry |
