# 4.1-recommendation-tone-checker.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-03-cta-bridge-writer
> **Purpose:** Verify recommendation framing (not "Buy Now") in selected bridge
> **Outputs:** `tone_check_result` with PASS/FAIL

---

## FUNCTION

Run definitive tone analysis on the selected bridge/CTA combination. Verify the bridge uses recommendation framing throughout and contains zero sales, urgency, or close language. This enforces Law 5: bridge = recommendation, not sales close.

---

## TONE CHECK PROTOCOL

### Step 1: Sales Language Scan

```
SCAN for sales language:
  BANNED (auto-fail):
    - buy, purchase, order, add to cart, get yours
    - claim, grab, secure, reserve
    - act now, act fast, hurry, rush
    - limited time, while supplies last, today only
    - don't miss, last chance, expires
    - what are you waiting for, you deserve this
    - sign up, subscribe, join now (in sales context)

  IF any found:
    STATUS = FAIL
    LOG: "Sales language detected: '[phrase]' in paragraph [N]"
```

### Step 2: Recommendation Marker Check

```
SCAN for recommendation markers (at least 1 required):
  APPROVED markers:
    - "If you're interested in...", "For those who want to..."
    - "Based on [research/my experience]...", "I'd recommend..."
    - "You can learn more...", "More information is available..."
    - "[Expert] suggests...", "The evidence points to..."

  IF zero recommendation markers found:
    STATUS = FAIL
    LOG: "No recommendation framing detected -- reads as command, not recommendation"
```

### Step 3: Tone Gradient Verification

```
MEASURE editorial-to-recommendation gradient:
  FOR each bridge paragraph:
    SCORE editorial_percentage (0-100%)
    SCORE sales_signal (0-100%)

  VERIFY:
    [ ] No paragraph exceeds 30% sales signal
    [ ] First paragraph is 100% editorial
    [ ] Gradient is gradual (no sudden jumps > 20%)
    [ ] CTA paragraph maintains >= 70% editorial tone

  IF any verification fails:
    STATUS = FAIL
    LOG: "Tone gradient violation in paragraph [N]"
```

### Step 4: Final Determination

```
IF sales_language_found > 0: FAIL
IF recommendation_markers == 0: FAIL
IF gradient_violations > 0: FAIL
ELSE: PASS
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Sales language found | Banned phrase detected | Replace with recommendation language |
| No recommendation markers | Bridge reads as command | Add recommendation framing |
| Gradient violation | Sudden shift to sales | Smooth transition with intermediary |
