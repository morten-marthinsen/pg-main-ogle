# 0.2-specimen-calibrator.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-03-cta-bridge-writer
> **Purpose:** Load bridge patterns from advertorial specimens
> **Outputs:** `bridge_patterns` object with transition and CTA patterns

---

## FUNCTION

Load advertorial specimens and extract bridge/CTA patterns. Analyze how successful advertorials transition from editorial content to product recommendation. Extract transition techniques, CTA language patterns, bridge framing approaches, and word count benchmarks.

---

## CALIBRATION PROTOCOL

### Step 1: Load Specimen Bridges

```
FROM specimen collection:
  EXTRACT bridge sections (typically last 10-20% of advertorial)
  MINIMUM: 3 specimen bridges for calibration
  FOCUS: The transition from body to recommendation + CTA
```

### Step 2: Decompose Bridge Patterns

```
FOR each specimen bridge:
  EXTRACT:
    - transition_technique: How editorial shifts to recommendation
      (natural extension, expert recommendation, personal discovery, community proof)
    - bridge_length: Word count of bridge section
    - paragraph_count: Number of bridge paragraphs
    - cta_format: How CTA is presented (inline link, standalone link, etc.)
    - cta_language: Exact CTA text used
    - recommendation_markers: Phrases that signal recommendation vs selling
    - sales_language_absent: Confirmation no sales language present
    - transition_gradient: How gradual is the shift (1-10, 10=invisible)
```

### Step 3: Extract CTA Language Patterns

```
FROM specimen CTAs:
  CATALOGUE approved patterns:
    - "Learn more about [approach] at [URL]"
    - "For more information, visit [URL]"
    - "[Name] shares the full details at [URL]"
    - "You can find out more about this approach here"

  CATALOGUE forbidden patterns (from specimens that failed):
    - "Buy now at [URL]"
    - "ORDER TODAY"
    - "Click here for your exclusive discount"
```

### Step 4: Build Bridge Calibration Reference

```
ASSEMBLE:
  - Average bridge word count
  - Most effective transition techniques
  - CTA language that reads as editorial
  - Transition gradient benchmarks
  - Forbidden patterns observed in weak specimens
```

---

## OUTPUT STRUCTURE

```json
{
  "bridge_patterns": {
    "specimens_analyzed": 4,
    "transition_techniques": ["natural_extension", "expert_recommendation"],
    "avg_bridge_words": 120,
    "avg_paragraphs": 3,
    "cta_patterns": {
      "approved": ["Learn more about...", "For more information..."],
      "forbidden": ["Buy now", "Order today"]
    },
    "transition_gradient_benchmark": 8,
    "calibration_quality": "strong"
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Zero specimens | HALT -- Cannot calibrate |
| DEGRADED | < 3 specimens | Proceed with formula guidance, flag |
