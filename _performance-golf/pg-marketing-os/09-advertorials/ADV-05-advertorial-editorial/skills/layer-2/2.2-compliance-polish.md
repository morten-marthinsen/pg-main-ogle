# 2.2-compliance-polish.md

> **Layer:** 2 (Revision)
> **Skill:** ADV-05-advertorial-editorial
> **Purpose:** Platform compliance final pass on revised advertorial
> **Outputs:** `compliance_polish_result` with final compliance status

---

## FUNCTION

Run the final platform compliance pass on the revised advertorial. This catches any compliance issues introduced during revision and verifies all original compliance passes still hold. The output must be submission-ready for the target platform.

---

## COMPLIANCE POLISH PROTOCOL

### Step 1: Re-verify Core Compliance

```
CHECK against platform rules:
  [ ] Word count within bounds
  [ ] Headline/hook within char limit
  [ ] Disclosure present and correctly placed
  [ ] No forbidden patterns in headline
  [ ] No ALL CAPS (except acronyms)
  [ ] No excessive punctuation
  [ ] CTA count = 1, format = text_link
  [ ] All claims attributed to sources
  [ ] No before/after claims without disclaimer
  [ ] No personal attribute assertions (Meta)
```

### Step 2: Check Revision Impact

```
COMPARE pre-revision vs post-revision:
  [ ] Disclosure still present? (revisions may have removed it)
  [ ] Headline unchanged or still compliant?
  [ ] Word count still within bounds?
  [ ] CTA unchanged or still compliant?
  [ ] No new unsubstantiated claims introduced?

IF any check fails:
  FIX: Apply specific compliance correction
  LOG: "Compliance fix applied: [description]"
```

### Step 3: Final Platform-Specific Check

```
FOR target platform:
  RUN platform-specific checklist:

  Taboola:
    [ ] Headline max 60 chars
    [ ] Landing page matches headline promise
    [ ] Sponsored label at top
    [ ] No misleading health claims

  Outbrain:
    [ ] Headline max 65 chars
    [ ] Quality score requirements met
    [ ] Disclosure top and bottom
    [ ] No clickbait triggers

  Meta:
    [ ] Primary text max 40 chars
    [ ] No personal attributes
    [ ] Text overlay max 20% (images)
    [ ] No before/after claims

  Organic:
    [ ] SEO title max 60 chars
    [ ] Meta description 150-160 chars
    [ ] FTC compliance for endorsements
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Disclosure removed by revision | Revision overwrote disclosure | Re-add disclosure |
| Word count exceeded by revision | Revision expanded content | Trim to bounds |
| New claim introduced | Revision added unsubstantiated claim | Add attribution or remove |
| CTA modified | Revision changed CTA | Revert CTA |
