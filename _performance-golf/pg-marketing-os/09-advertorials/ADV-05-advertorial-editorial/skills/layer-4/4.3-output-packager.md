# 4.3-output-packager.md

> **Layer:** 4 (Scoring)
> **Skill:** ADV-05-advertorial-editorial
> **Purpose:** Package final advertorial as advertorial-final.md
> **Outputs:** `advertorial-final.md` -- the publication-ready output

---

## FUNCTION

Package the quality-scored, word-count-validated, editorially-reviewed advertorial as the final canonical output file. This is the PUBLICATION-READY output -- the end product of the entire ADV pipeline. The file contains the complete advertorial text plus quality certification, revision log, and publication metadata.

---

## PACKAGING PROTOCOL

### Step 1: Verify All Gates Passed

```
CHECK:
  quality_score.status == "PASS" (>= 8.0)
  word_count_result.status == "PASS"
  compliance_polish_result.status == "PASS"
  smell_audit.overall_status in ["PASS", "CONDITIONAL"]

IF any FAIL:
  HALT: "Cannot package -- [specific gate] failed"
  ACTION: Return to appropriate layer for remediation
```

### Step 2: Assemble Final File

```
BUILD advertorial-final.md:

  ---
  metadata:
    version: "1.0"
    skill_id: "ADV-05-advertorial-editorial"
    status: "PUBLICATION-READY"
    generated_at: "[ISO timestamp]"
    advertorial_type: "[type]"
    platform: "[platform]"
    voice_register: "[register]"
    total_word_count: [count]
    quality_score: [weighted total]
    five_laws: "ALL PASS"
    smell_test: "PASS"
    cta_count: 1
    cta_format: "text_link"
    disclosure_present: true
  ---

  [Complete final advertorial text]

  ---
  ## Quality Certification
  - Overall Score: [weighted total]/10
  - Editorial Authenticity: [score]/10
  - Hook Strength: [score]/10
  - Voice Unity: [score]/10
  - Proof Integration: [score]/10
  - Bridge Naturalness: [score]/10
  - Compliance: PASS
  - 5 Laws: ALL PASS

  ## Revision Log
  - Revisions Made: [count]
  - Issues Addressed: [list]
  - Approach: [conservative|moderate|deep]

  ## Publication Metadata
  - Platform: [platform]
  - Word Count: [count] (bounds: [min]-[max])
  - Disclosure: [text] at [position]
  - CTA URL: [url]
  - Headline Version: [platform-optimized headline]
```

### Step 3: Write File

```
WRITE to: [project]/09-advertorials/outputs/advertorial-final.md
VERIFY:
  [ ] File exists at expected path
  [ ] File is non-empty
  [ ] File size > 5KB
  [ ] Metadata section complete
  [ ] Quality certification present
  [ ] Advertorial text present and complete

LOG: "advertorial-final.md written at [timestamp] -- PUBLICATION READY"
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Quality gate not passed | Score < 8.0 | Return to revision layer |
| Word count violation | Outside bounds | Return to revision for trim/expansion |
| Compliance failure | Platform rules violated | Return to compliance polish |
| File write failure | Filesystem error | Check path, retry |
| Incomplete file | Missing sections | Complete before write |

---

## POST-PACKAGING VERIFICATION

```
AFTER file written:
  1. READ file back to verify completeness
  2. VERIFY quality score in metadata matches scorer output
  3. VERIFY word count in metadata matches validator output
  4. VERIFY CTA count = 1 in final text
  5. VERIFY disclosure present in final text
  6. LOG: "Post-packaging verification COMPLETE"

  UPDATE PROJECT-STATE.md:
    status: COMPLETE
    output: advertorial-final.md
    quality: [score]
    timestamp: [now]
```
