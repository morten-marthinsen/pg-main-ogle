# 4.2-word-count-validator.md

> **Layer:** 4 (Scoring)
> **Skill:** ADV-05-advertorial-editorial
> **Purpose:** Verify word count within type-specific bounds
> **Outputs:** `word_count_result` with PASS/FAIL

---

## FUNCTION

Verify the final advertorial's word count falls within the type-specific bounds defined in the strategy. Word count bounds are HARD limits -- not approximate guidelines. This is the final word count check before publication.

---

## VALIDATION PROTOCOL

### Step 1: Count Words

```
COUNT total words in final advertorial text:
  INCLUDE: all content text (hook through CTA)
  EXCLUDE: metadata, YAML frontmatter, section markers
  EXCLUDE: disclosure text (not part of editorial content)

word_count = [total]
```

### Step 2: Compare Against Bounds

```
FROM strategy_constraints.word_count_bounds:
  min = [lower bound]
  max = [upper bound]

  IF word_count >= min AND word_count <= max:
    STATUS = PASS
    message = "Word count [N] within bounds [min]-[max]"

  IF word_count < min:
    STATUS = FAIL
    message = "Word count [N] below minimum [min]"
    deficit = min - word_count
    guidance = "Need [deficit] more words -- expand weakest sections with editorial content"

  IF word_count > max:
    STATUS = FAIL
    message = "Word count [N] above maximum [max]"
    excess = word_count - max
    guidance = "Need to cut [excess] words -- trim lowest-value sections"
```

### Step 3: Section Breakdown

```
PROVIDE section-level breakdown:
  hook: [words]
  lead: [words]
  body: [words]
  bridge: [words]
  total: [words]

IDENTIFY which sections contribute most to over/under:
  IF over: "Body accounts for [X]% of total -- primary trim target"
  IF under: "Lead is only [X] words -- primary expansion target"
```

---

## OUTPUT STRUCTURE

```json
{
  "word_count_result": {
    "status": "PASS|FAIL",
    "total_words": 1085,
    "bounds": {"min": 1000, "max": 1500},
    "within_bounds": true,
    "section_breakdown": {
      "hook": 28,
      "lead": 185,
      "body": 720,
      "bridge": 140,
      "disclosure": 12
    },
    "adjustment_needed": false,
    "adjustment_guidance": null
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Below minimum | Word count < min bound | Expand weakest sections |
| Above maximum | Word count > max bound | Trim lowest-value sections |
| Borderline | Within 5% of boundary | Flag but PASS |
