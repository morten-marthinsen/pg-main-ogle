# 0.1-upstream-loader.md

> **Layer:** 0 (Foundation)
> **Skill:** ADV-05-advertorial-editorial
> **Purpose:** Load assembled advertorial for editorial review
> **Outputs:** `upstream_context` with assembled piece and review guidance

---

## FUNCTION

Load the assembled advertorial (from ADV-04) for editorial review. Extract the complete text, assembly report, and editorial review guidance. This is the starting material for the final quality review.

---

## LOADING PROTOCOL

```
1. Load advertorial-assembled.md (from ADV-04)
   EXTRACT:
     - Complete assembled text
     - Assembly metadata (word count, type, voice register)
     - Assembly report (voice consistency, narrative, compliance)
     - Editorial review guidance (focus areas, known risks)
   VERIFY: file exists, non-empty, contains assembled text

IF assembled piece missing:
  FAIL: "Cannot review -- ADV-04 must complete first"
```
