# 4.3-output-packager.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-02-body-copy-writer
> **Purpose:** Package validated body copy as advertorial-body-draft.md
> **Outputs:** `advertorial-body-draft.md` file

---

## FUNCTION

Package the validated body copy into the canonical output file that ADV-03 (CTA/bridge) and ADV-04 (assembly) consume. Include bridge handoff context, proof density report, and editorial quality metrics.

---

## PACKAGING PROTOCOL

### Step 1: Verify Validations

```
CHECK:
  smell_test_result.status == "PASS" or "CONDITIONAL"
  density_check_result.status == "PASS"

IF either FAIL: HALT -- cannot package
```

### Step 2: Assemble Body Draft

```
BUILD advertorial-body-draft.md:

  ---
  metadata:
    version: "1.0"
    skill_id: "ADV-02-body-copy-writer"
    advertorial_type: "[type]"
    voice_register: "[register]"
    smell_test: "[PASS/CONDITIONAL]"
    proof_density: "PASS"
    total_word_count: [count]
  ---

  ## Body Sections
  [All body sections with proof embedded]

  ## Bridge Handoff
  - last_section_theme: [what final section established]
  - reader_state: [convinced|curious|informed]
  - transition_point: [where body ends]
  - mechanism_status: [fully_introduced|partially_explained]
  - recommended_bridge_frame: [discovery|recommendation|expert|community]

  ## Proof Density Report
  - elements_used: [count]
  - max_per_paragraph: 1
  - max_consecutive: [actual]
  - stacking_violations: 0
  - proof_rhythm: [pattern]

  ## Quality Metrics
  - smell_test: [result]
  - voice_consistency: [score]
  - type_structure_compliance: [score]
```

### Step 3: Write File

```
WRITE to: [project]/09-advertorials/outputs/advertorial-body-draft.md
VERIFY: file exists, non-empty, > 3KB
LOG: "advertorial-body-draft.md written at [timestamp]"
```

---

## OUTPUT STRUCTURE

```json
{
  "packaging_result": {
    "status": "complete",
    "file_written": "advertorial-body-draft.md",
    "word_count": 720,
    "bridge_handoff_complete": true,
    "proof_density_compliant": true,
    "smell_test_passed": true
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Validation not passed | Packaging before validation | HALT -- run 4.1 and 4.2 |
| Incomplete handoff | Bridge handoff missing | Complete handoff fields |
| File write failure | Filesystem error | Check path, retry |
