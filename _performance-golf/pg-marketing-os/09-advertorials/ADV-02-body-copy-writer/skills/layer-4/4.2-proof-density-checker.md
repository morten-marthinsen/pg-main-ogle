# 4.2-proof-density-checker.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-02-body-copy-writer
> **Purpose:** Verify no proof stacking (3+ in sequence = fail)
> **Outputs:** `density_check_result` with PASS/FAIL and density map

---

## FUNCTION

Final validation of proof density compliance across the entire body copy. Scan every paragraph for proof elements, count consecutive proof-containing paragraphs, and verify the density rules are met. This is a binary gate: proof stacking detected = FAIL.

---

## DENSITY CHECK PROTOCOL

### Step 1: Paragraph-Level Scan

```
FOR each paragraph in body_copy:
  IDENTIFY proof elements:
    - Clinical study references
    - Expert quotes with attribution
    - Statistics with source
    - Testimonials
    - Case study references
    - Mechanism evidence claims

  COUNT proof_elements_in_paragraph
  LOG: "Paragraph [N]: [count] proof elements"

  IF count > 1:
    FAIL: "Paragraph [N] contains [count] proof elements (max 1)"
```

### Step 2: Sequence Analysis

```
BUILD sequence map:
  FOR each paragraph:
    IF contains_proof: mark as P
    IF no proof: mark as N

  EXAMPLE: N-P-N-P-P-N-P-N

  SCAN for stacking patterns:
    P-P-P = FAIL (3 consecutive proof paragraphs)
    P-P-P-P = FAIL (4 consecutive)
    P-P = ACCEPTABLE (2 consecutive, maximum allowed)
    N-P-N = IDEAL (proof surrounded by narrative)
```

### Step 3: Density Metrics

```
CALCULATE:
  total_paragraphs: [count]
  proof_paragraphs: [count]
  proof_ratio: proof_paragraphs / total_paragraphs
  max_consecutive_proof: [count]
  longest_proof_free_gap: [count]

  BENCHMARKS:
    proof_ratio < 0.50 -> HEALTHY
    proof_ratio 0.50-0.60 -> BORDERLINE (flag)
    proof_ratio > 0.60 -> EXCESSIVE (fail even without stacking)
```

### Step 4: Final Determination

```
DENSITY STATUS:
  IF max_per_paragraph <= 1 AND max_consecutive <= 2 AND proof_ratio < 0.60:
    STATUS = PASS
  ELSE:
    STATUS = FAIL
    DOCUMENT: specific violations with paragraph numbers
    PROVIDE: redistribution recommendations
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Per-paragraph violation | 2+ proof in single paragraph | Split into separate paragraphs |
| Stacking violation | 3+ consecutive proof paragraphs | Insert narrative between proof sections |
| Excessive ratio | > 60% of paragraphs contain proof | Remove lower-priority proof elements |
