# 4.1-editorial-smell-tester.md

> **Layer:** 4 (Validation)
> **Skill:** ADV-02-body-copy-writer
> **Purpose:** Remove product name from body -- verify it still reads as article
> **Outputs:** `smell_test_result` with PASS/FAIL and evidence

---

## FUNCTION

Execute the definitive editorial smell test on the body copy. Create a redacted version with all product and brand names removed. Read the redacted version and evaluate whether it would be publishable as a genuine article. This is the single most important quality gate for advertorial body copy.

---

## SMELL TEST PROTOCOL

### Step 1: Create Redacted Version

```
REDACT from body copy:
  - Product name -> [PRODUCT]
  - Brand name -> [BRAND]
  - Any branded URLs -> [URL]
  - Any pricing information -> [PRICE]
  - Any promotional language that references the product specifically

DO NOT redact:
  - Mechanism name (if it's a generic/scientific term)
  - Category descriptions (e.g., "a probiotic supplement")
  - Source attributions (institutions, researchers, journals)
```

### Step 2: Evaluate Redacted Version

```
READ redacted version and score:

CRITERION 1: Publication Test
  "Could this appear in a health/lifestyle/trade publication?"
  IF YES -> PASS
  IF "Maybe, with editing" -> CONDITIONAL (flag specific issues)
  IF NO -> FAIL

CRITERION 2: Value Independence
  "Does this content provide genuine value even without the product?"
  IF YES -> PASS
  IF NO -> FAIL (content is purely promotional)

CRITERION 3: Voice Authenticity
  "Does this read like a writer wrote it, or like a marketer wrote it?"
  IF writer -> PASS
  IF marketer -> FAIL

CRITERION 4: Source Quality
  "Are claims attributed to named, verifiable sources?"
  IF YES -> PASS
  IF vague ("studies show", "experts say") -> FAIL

CRITERION 5: Balance
  "Does the content acknowledge any limitations or alternatives?"
  IF YES -> PASS (strong editorial signal)
  IF NO -> CONDITIONAL (not mandatory but significant)
```

### Step 3: Identify Ad-Smell Triggers

```
SCAN for residual ad-smell in redacted version:
  - Feature lists disguised as editorial content
  - "Problem-solution" framing without narrative depth
  - Superlative claims without attributed evidence
  - Emotional manipulation without informational substance
  - Single-perspective advocacy (no balance or nuance)

LOG each trigger found with specific paragraph reference
```

### Step 4: Final Determination

```
SMELL TEST SCORE:
  5/5 criteria PASS -> PASS (strong editorial quality)
  4/5 criteria PASS -> CONDITIONAL PASS (flag issues for review)
  3/5 or fewer -> FAIL (body needs rewriting)

IF FAIL:
  DOCUMENT specific failure points
  PROVIDE rewrite guidance for each failed criterion
  RETURN to Layer 2 for revision
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| Publication test fail | Content too promotional for any publication | Major rewrite -- editorial framing needed |
| Value independence fail | Content has no value without product | Add genuine informational content |
| Voice fail | Reads as marketer, not writer | Rewrite with journalist voice markers |
| Source quality fail | Vague attributions | Add specific named sources |
| Multiple triggers found | Several ad-smell elements | Comprehensive editorial revision |
