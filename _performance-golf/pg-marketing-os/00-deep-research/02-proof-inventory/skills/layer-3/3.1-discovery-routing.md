# 3.1 Discovery Routing

## Purpose
Route identified gaps to the appropriate discovery strategy based on gap type, category, and severity. This is the orchestrator that determines which discovery microskill(s) to invoke for each gap.

---

## Input Requirements

From 2.5/2.6 Gap Analysis:
- Prioritized gap list with severity tiers
- Gap categories and types
- Specific gap descriptions

From Context:
- Schwartz stage
- Available research tools/access
- Time constraints

---

## Discovery Strategy Matrix

```
GAP CATEGORY → DISCOVERY STRATEGY ROUTING:

SOCIAL Gaps
├── Missing → Social Discovery (reviews, forums, competitor analysis)
├── Weak (quality) → Testimonial Enhancement Discovery
└── Under-represented → Segment-Specific Search

AUTHORITY Gaps
├── Missing → Study Discovery (3.2)
├── Weak (credibility) → Expert Quote Discovery (3.4)
└── Subtype gap (e.g., no peer-reviewed) → Academic Database Search

DEMONSTRATION Gaps
├── Missing → Visual Proof Discovery
├── Weak → Quality Benchmark Discovery
└── Note: Often requires GENERATION, not discovery

MECHANISM Gaps
├── Missing → Study Discovery (3.2) + Analogous Proof (3.5)
├── Weak → Expert Explanation Discovery (3.4)
└── Technical → Patent/Whitepaper Search

DATA Gaps
├── Missing → Data Discovery (3.3)
├── Weak (credibility) → Official Source Search
└── Outdated → Recency-Focused Search

RISK_REVERSAL Gaps
├── Almost always → GENERATION (not discovery)
└── Competitive benchmark only → Competitor Guarantee Analysis
```

## Execution Protocol

### Step 1: Gap Prioritization for Discovery

```python
def prioritize_for_discovery(gaps, schwartz_stage, time_budget):
    """
    Prioritize gaps for discovery operations.
    Some gaps are better solved by generation — route accordingly.
    """

    discovery_priority = []
    generation_priority = []

    for gap in gaps:
        # Assess discoverability
        discoverability = assess_discoverability(gap, schwartz_stage)

        if discoverability['score'] >= 6:
            discovery_priority.append({
                'gap': gap,
                'discoverability': discoverability,
                'discovery_types': discoverability['strategies'],
                'estimated_effort': discoverability['effort']
            })
        else:
            generation_priority.append({
                'gap': gap,
                'reason': discoverability['generation_rationale'],
                'generation_type': suggest_generation(gap)
            })

    # Sort by impact/effort ratio
    discovery_priority.sort(
        key=lambda g: (g['gap']['severity_score'] / max(g['estimated_effort'], 1)),
        reverse=True
    )

    return {
        'discovery_queue': discovery_priority,
        'generation_queue': generation_priority,
        'total_discovery_gaps': len(discovery_priority),
        'total_generation_gaps': len(generation_priority)
    }


def assess_discoverability(gap, schwartz_stage):
    """
    Score how likely we are to find external proof for this gap.
    """

    DISCOVERABILITY_BASE = {
        'social': 5,      # Testimonials often proprietary
        'authority': 9,   # Studies usually public
        'demonstration': 3,  # Usually must create
        'mechanism': 8,   # Scientific literature available
        'data': 9,        # Statistics often public
        'risk_reversal': 2  # Must create
    }

    category = gap.get('category', '')
    score = DISCOVERABILITY_BASE.get(category, 5)

    # Adjust for specific gap types
    if gap.get('gap_type') == 'quality_gap':
        score -= 2  # Harder to find BETTER quality

    if gap.get('gap_type') == 'subtype_gap':
        score -= 1  # Specific sub-type harder to find

    # Adjust for niche specificity
    if gap.get('niche_specific', False):
        score -= 2  # Niche-specific proof harder to find

    strategies = get_discovery_strategies(gap)

    return {
        'score': max(1, min(10, score)),
        'strategies': strategies,
        'effort': estimate_discovery_effort(strategies),
        'generation_rationale': get_generation_rationale(gap) if score < 6 else None
    }
```

### Step 2: Route to Discovery Microskills

```python
def route_to_discovery(gap):
    """
    Determine which discovery microskill(s) to invoke.
    """

    routing = {
        'gap_id': gap['gap_id'],
        'primary_discovery': None,
        'secondary_discovery': [],
        'fallback_to_generation': False
    }

    category = gap.get('category', '')
    gap_type = gap.get('gap_type', '')

    # AUTHORITY gaps
    if category == 'authority':
        if gap_type in ['missing', 'weak', 'subtype_gap']:
            routing['primary_discovery'] = '3.2-study-discovery'
        if gap_type == 'quality_gap':
            routing['secondary_discovery'].append('3.4-expert-quote-discovery')

    # MECHANISM gaps
    elif category == 'mechanism':
        routing['primary_discovery'] = '3.2-study-discovery'
        routing['secondary_discovery'].append('3.5-analogous-proof-discovery')
        routing['secondary_discovery'].append('3.4-expert-quote-discovery')

    # DATA gaps
    elif category == 'data':
        routing['primary_discovery'] = '3.3-data-discovery'

    # SOCIAL gaps
    elif category == 'social':
        routing['primary_discovery'] = 'social-discovery'  # Inline protocol
        routing['fallback_to_generation'] = True  # Often needs creation

    # DEMONSTRATION gaps
    elif category == 'demonstration':
        routing['primary_discovery'] = None
        routing['fallback_to_generation'] = True

    # RISK_REVERSAL gaps
    elif category == 'risk_reversal':
        routing['primary_discovery'] = None
        routing['fallback_to_generation'] = True

    return routing
```

### Step 3: Parallel vs Sequential Discovery

```python
def plan_discovery_execution(routed_gaps, parallel_allowed=True):
    """
    Plan execution order — some discoveries can run in parallel.
    """

    if parallel_allowed:
        # Group by discovery type for batching
        by_discovery_type = {}
        for routed in routed_gaps:
            primary = routed.get('primary_discovery')
            if primary:
                if primary not in by_discovery_type:
                    by_discovery_type[primary] = []
                by_discovery_type[primary].append(routed)

        # Return batched execution plan
        return {
            'execution_mode': 'parallel',
            'batches': by_discovery_type
        }
    else:
        # Sequential by priority
        return {
            'execution_mode': 'sequential',
            'order': routed_gaps
        }
```
