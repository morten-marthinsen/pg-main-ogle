# 3.3 Data Discovery

## Purpose
Find external statistics, market data, survey results, and quantifiable evidence to fill DATA proof gaps. Focus on government sources, research reports, and authoritative data providers.

---

## Input Requirements

From 3.1 Discovery Routing:
- Gaps routed to data discovery
- Search terms generated
- Data type needed (prevalence, market size, success rates, etc.)

From Context:
- Niche/industry
- Problem being addressed
- Claims requiring statistical support

---

## Data Source Hierarchy

```
DATA SOURCE TIERS:

TIER 1 - Government & Official (Highest Credibility)
├── CDC (health statistics, NHIS, BRFSS)
├── FDA (safety data, adverse events)
├── NIH (research funding, statistics)
├── Bureau of Labor Statistics
├── Census Bureau
├── World Health Organization
├── USDA (nutrition data)
└── Federal Reserve (financial data)

TIER 2 - Academic & Research
├── Pew Research Center
├── Gallup polls
├── University research centers
├── Published survey studies
└── Academic meta-analyses

TIER 3 - Industry Reports
├── Statista
├── IBISWorld
├── Market research firms (Gartner, Forrester)
├── Trade associations
└── Industry white papers

TIER 4 - Media & Aggregated (Use with Citation)
├── News coverage of studies
├── Industry publications
├── Aggregated statistics sites
└── Secondary source citations
```

### Step 1: Data Need Classification

```python
def classify_data_need(gap):
    """
    Determine what type of data is needed.
    """

    DATA_TYPES = {
        'prevalence': {
            'description': 'How common is the problem?',
            'example': '78% of adults experience...',
            'best_sources': ['CDC', 'WHO', 'academic_surveys']
        },
        'market_size': {
            'description': 'How big is the market/industry?',
            'example': '$47 billion industry',
            'best_sources': ['statista', 'ibis', 'trade_associations']
        },
        'success_rate': {
            'description': 'What percentage achieve result?',
            'example': '94% success rate',
            'best_sources': ['clinical_trials', 'internal_data', 'studies']
        },
        'comparison': {
            'description': 'How does X compare to Y?',
            'example': '3x more effective than...',
            'best_sources': ['studies', 'meta_analyses']
        },
        'trend': {
            'description': 'Is the problem growing?',
            'example': 'Up 47% since 2020',
            'best_sources': ['CDC', 'market_research', 'news']
        },
        'cost': {
            'description': 'What is the cost of the problem?',
            'example': 'Americans spend $147B annually on...',
            'best_sources': ['government', 'industry_reports']
        },
        'demographic': {
            'description': 'Who is affected?',
            'example': 'Women aged 35-55 are 2x more likely...',
            'best_sources': ['CDC', 'census', 'academic_surveys']
        }
    }

    # Analyze gap description to determine type
    gap_text = gap.get('description', '').lower()

    for data_type, info in DATA_TYPES.items():
        keywords = get_type_keywords(data_type)
        if any(kw in gap_text for kw in keywords):
            return {
                'data_type': data_type,
                'description': info['description'],
                'best_sources': info['best_sources'],
                'search_strategy': build_search_strategy(data_type, gap)
            }

    # Default
    return {
        'data_type': 'general',
        'description': 'General supporting statistic',
        'best_sources': ['perplexity', 'government', 'studies'],
        'search_strategy': 'broad'
    }
```

### Step 3: Statistic Validation Protocol

```python
def validate_statistic(stat_data):
    """
    Validate discovered statistic for accuracy and usability.
    """

    validation = {
        'stat_id': generate_id(),
        'is_valid': False,
        'credibility_score': 0,
        'recency_score': 0,
        'specificity_score': 0,
        'usability_score': 0,
        'warnings': []
    }

    # CREDIBILITY CHECK
    credibility = 5  # Base

    source_tier = get_source_tier(stat_data['source'])
    tier_scores = {1: 3, 2: 2, 3: 1, 4: 0}
    credibility += tier_scores.get(source_tier, 0)

    if has_methodology(stat_data):
        credibility += 1
    if is_primary_source(stat_data):
        credibility += 1
    if is_verifiable(stat_data):
        credibility += 1

    validation['credibility_score'] = max(1, min(10, credibility))

    # RECENCY CHECK
    year = extract_year(stat_data)
    current_year = 2024

    if year:
        age = current_year - year
        if age <= 1:
            recency = 10
        elif age <= 2:
            recency = 8
        elif age <= 3:
            recency = 6
        elif age <= 5:
            recency = 4
        else:
            recency = 2
            validation['warnings'].append(f'Data is {age} years old')
    else:
        recency = 3
        validation['warnings'].append('Year not specified')

    validation['recency_score'] = recency

    # SPECIFICITY CHECK
    specificity = 5  # Base

    if has_exact_number(stat_data):
        specificity += 2
    if has_sample_size(stat_data):
        specificity += 1
    if has_confidence_interval(stat_data):
        specificity += 1
    if has_demographic_breakdown(stat_data):
        specificity += 1

    validation['specificity_score'] = max(1, min(10, specificity))

    # USABILITY CHECK
    usability = 5  # Base

    if is_quotable(stat_data):
        usability += 2
    if supports_claim(stat_data):
        usability += 2
    if is_surprising(stat_data):  # Novelty value
        usability += 1

    validation['usability_score'] = max(1, min(10, usability))

    # Overall validity
    avg_score = (validation['credibility_score'] +
                 validation['recency_score'] +
                 validation['specificity_score'] +
                 validation['usability_score']) / 4

    validation['is_valid'] = avg_score >= 5
    validation['overall_score'] = avg_score

    return validation
```

### Step 4: Statistic Formatting

```python
def format_statistic(stat_data, validation):
    """
    Format statistic for use in copy.
    """

    formatted = {
        'raw_stat': stat_data['statistic'],
        'copy_ready': None,
        'citation': None,
        'context': None
    }

    # Copy-ready versions
    stat = stat_data['statistic']
    source = stat_data.get('source', '')
    year = stat_data.get('year', '')

    # Version 1: Stat with attribution
    formatted['with_attribution'] = f"{stat} ({source}, {year})"

    # Version 2: Stat with inline source
    formatted['inline_source'] = f"{stat} according to {source}"

    # Version 3: Stat with authority framing
    formatted['authority_framed'] = f"According to {source}, {stat}"

    # Version 4: Stat only (for high-credibility sources)
    if validation['credibility_score'] >= 8:
        formatted['stat_only'] = stat

    # Citation
    formatted['citation'] = build_data_citation(stat_data)

    # Context for copywriter
    formatted['usage_context'] = {
        'credibility_level': 'high' if validation['credibility_score'] >= 7 else 'moderate',
        'needs_citation': validation['credibility_score'] < 8,
        'best_used_in': suggest_copy_position(stat_data, validation)
    }

    return formatted
```

## Data Stacking Strategy

```
PROOF STACKING ORDER FOR MAXIMUM IMPACT:

1. SHOCKING PREVALENCE STAT
   "78% of Americans struggle with..."
   → Establishes you're not alone, problem is real

2. TREND STAT
   "And it's getting worse—up 47% since 2020"
   → Urgency, the problem is growing

3. COST STAT
   "Americans spend $147 billion annually trying to solve..."
   → Financial/emotional cost validation

4. FAILURE STAT
   "Yet 92% of attempts fail within the first year"
   → Existing solutions don't work

5. MECHANISM STAT
   "New research shows the real cause..."
   → Bridge to your mechanism
```
