# 3.5 Analogous Proof Discovery

## Purpose
Find proof for mechanisms, ingredients, or approaches that work via the SAME pathway or principle as the product being promoted. When direct proof is unavailable, analogous proof can bridge the credibility gap by demonstrating that the underlying mechanism is scientifically validated.

---

## Input Requirements

From 3.1 Discovery Routing:
- Mechanism gaps requiring analogous proof
- Mechanism pathway/principle

From 03-Mechanism Skill:
- Mechanism name and type
- Scientific pathway (if known)
- Key ingredients/components
- Mode of action

---

## The Analogous Proof Principle

```
THE LOGIC OF ANALOGOUS PROOF:

IF: [Compound A] works via [Pathway X]
AND: [Your Product] also works via [Pathway X]
THEN: Evidence for [Compound A] supports [Your Product]

EXAMPLE:
IF: "Studies show omega-3 reduces inflammation via COX-2 inhibition"
AND: "Our compound also inhibits COX-2"
THEN: Omega-3 inflammation research indirectly supports our mechanism

KEY REQUIREMENT:
The pathway/mechanism must be IDENTICAL or SUBSTANTIALLY SIMILAR.
Surface-level similarity is NOT sufficient.
```

---

## Analogous Proof Categories

```
ANALOGY STRENGTH TIERS:

TIER 1 - IDENTICAL MECHANISM (Strongest)
├── Same compound, different application
├── Same pathway, different target
├── Same ingredient class, same effect
└── Example: "Curcumin anti-inflammatory" → your curcumin product

TIER 2 - SAME PATHWAY
├── Different compound, same biological pathway
├── Same receptor target
├── Same enzyme inhibition
└── Example: "Green tea EGCG activates AMPK" → your product also activates AMPK

TIER 3 - SIMILAR MECHANISM CLASS
├── Same category of action (antioxidant, adaptogen, etc.)
├── Related but not identical pathways
├── Similar effect through different routes
└── Example: "Adaptogens modulate cortisol" → your adaptogenic blend

TIER 4 - CONCEPTUAL ANALOGY (Weakest)
├── Similar outcomes, different mechanisms
├── Metaphorical similarity
├── Requires careful framing
└── Example: "Probiotics improve gut health" → your gut support product
```

### Step 1: Mechanism Decomposition

```python
def decompose_mechanism(mechanism):
    """
    Break mechanism into searchable pathway components.
    """

    decomposition = {
        'primary_pathway': None,
        'key_receptors': [],
        'enzyme_targets': [],
        'biological_systems': [],
        'active_compounds': [],
        'mechanism_class': None
    }

    # Extract pathway
    decomposition['primary_pathway'] = extract_pathway(mechanism)
    # Example: "gut-brain axis", "AMPK activation", "COX-2 inhibition"

    # Extract receptor targets
    decomposition['key_receptors'] = extract_receptors(mechanism)
    # Example: ["CB1", "serotonin 5-HT1A", "insulin receptor"]

    # Extract enzyme targets
    decomposition['enzyme_targets'] = extract_enzymes(mechanism)
    # Example: ["COX-2", "PDE5", "aromatase"]

    # Extract systems involved
    decomposition['biological_systems'] = extract_systems(mechanism)
    # Example: ["immune", "endocrine", "nervous", "digestive"]

    # Extract active compounds
    decomposition['active_compounds'] = extract_compounds(mechanism)
    # Example: ["curcumin", "berberine", "quercetin"]

    # Classify mechanism type
    decomposition['mechanism_class'] = classify_mechanism(mechanism)
    # Example: "anti-inflammatory", "metabolic", "neuroprotective"

    return decomposition
```

### Step 2: Analogous Compound/Pathway Search

```python
def build_analogy_queries(decomposition):
    """
    Build search queries to find analogous proof.
    """

    queries = []

    # Query 1: Direct pathway search
    pathway = decomposition['primary_pathway']
    if pathway:
        queries.append({
            'type': 'pathway',
            'query': f'{pathway} mechanism study clinical trial',
            'priority': 'high'
        })

    # Query 2: Receptor target search
    for receptor in decomposition['key_receptors']:
        queries.append({
            'type': 'receptor',
            'query': f'{receptor} agonist antagonist clinical evidence',
            'priority': 'high'
        })

    # Query 3: Enzyme inhibition search
    for enzyme in decomposition['enzyme_targets']:
        queries.append({
            'type': 'enzyme',
            'query': f'{enzyme} inhibitor study results efficacy',
            'priority': 'high'
        })

    # Query 4: Similar compound search
    for compound in decomposition['active_compounds']:
        queries.append({
            'type': 'compound_class',
            'query': f'compounds similar to {compound} mechanism action',
            'priority': 'medium'
        })
        queries.append({
            'type': 'compound_study',
            'query': f'{compound} clinical trial peer-reviewed',
            'priority': 'medium'
        })

    # Query 5: Mechanism class search
    mech_class = decomposition['mechanism_class']
    if mech_class:
        queries.append({
            'type': 'class',
            'query': f'{mech_class} compounds natural supplements evidence',
            'priority': 'low'
        })

    return queries
```

### Step 4: Analogy Strength Assessment

```python
def assess_analogy_strength(original_mechanism, analogous_proof):
    """
    Determine how strongly the analogous proof supports the claim.
    """

    assessment = {
        'analogy_id': generate_id(),
        'strength_tier': None,
        'transferability_score': 0,
        'scientific_validity': 0,
        'usage_safety': None,
        'framing_requirements': []
    }

    # Determine analogy tier
    if same_compound_or_ingredient(original_mechanism, analogous_proof):
        assessment['strength_tier'] = 1
        base_score = 9
    elif same_pathway(original_mechanism, analogous_proof):
        assessment['strength_tier'] = 2
        base_score = 7
    elif same_mechanism_class(original_mechanism, analogous_proof):
        assessment['strength_tier'] = 3
        base_score = 5
    else:
        assessment['strength_tier'] = 4
        base_score = 3

    # TRANSFERABILITY SCORING
    transferability = base_score

    # Adjustments
    if analogous_proof['study_quality'] == 'high':
        transferability += 1
    if analogous_proof['sample_size'] >= 100:
        transferability += 0.5
    if analogous_proof['is_human_study']:
        transferability += 0.5
    if original_mechanism['dose'] == analogous_proof['dose']:
        transferability += 1

    # Deductions
    if different_delivery_method(original_mechanism, analogous_proof):
        transferability -= 1
    if different_population(original_mechanism, analogous_proof):
        transferability -= 0.5

    assessment['transferability_score'] = max(1, min(10, transferability))

    # FRAMING REQUIREMENTS
    if assessment['strength_tier'] >= 3:
        assessment['framing_requirements'].append(
            'Must clearly explain the pathway connection'
        )
    if assessment['strength_tier'] == 4:
        assessment['framing_requirements'].append(
            'Frame as "similar approach" not "proven by"'
        )
    if not analogous_proof['is_human_study']:
        assessment['framing_requirements'].append(
            'Note that evidence is preclinical'
        )

    # USAGE SAFETY
    if assessment['transferability_score'] >= 7:
        assessment['usage_safety'] = 'safe_to_cite'
    elif assessment['transferability_score'] >= 5:
        assessment['usage_safety'] = 'cite_with_framing'
    else:
        assessment['usage_safety'] = 'use_cautiously'

    return assessment
```

### Step 5: Analogous Proof Framing

```python
def frame_analogous_proof(analogy, assessment):
    """
    Create copy-ready framing for analogous proof.
    """

    framing = {
        'bridge_statement': None,
        'proof_statement': None,
        'combined': None,
        'variations': []
    }

    pathway = analogy['pathway']
    analogous_compound = analogy['compound']
    your_compound = analogy['your_mechanism']['name']
    key_finding = analogy['key_finding']

    # TIER 1: Direct compound evidence
    if assessment['strength_tier'] == 1:
        framing['bridge_statement'] = None  # No bridge needed
        framing['proof_statement'] = (
            f"Research shows {your_compound} {key_finding}"
        )

    # TIER 2: Same pathway
    elif assessment['strength_tier'] == 2:
        framing['bridge_statement'] = (
            f"The secret is the {pathway} pathway."
        )
        framing['proof_statement'] = (
            f"Studies on {analogous_compound}—which works via the same "
            f"{pathway} mechanism—show {key_finding}. "
            f"[Your mechanism] targets this exact same pathway."
        )

    # TIER 3: Same mechanism class
    elif assessment['strength_tier'] == 3:
        framing['bridge_statement'] = (
            f"This works because of its {analogy['mechanism_class']} properties."
        )
        framing['proof_statement'] = (
            f"Research on {analogy['mechanism_class']} compounds like "
            f"{analogous_compound} demonstrates {key_finding}. "
            f"[Your mechanism] harnesses these same {analogy['mechanism_class']} effects."
        )

    # TIER 4: Conceptual analogy
    else:
        framing['bridge_statement'] = (
            f"The underlying principle is well-established..."
        )
        framing['proof_statement'] = (
            f"Similar approaches have shown {key_finding}. "
            f"[Your mechanism] applies this proven principle."
        )
        framing['warning'] = 'Weaker analogy—use as supporting proof only'

    framing['combined'] = (
        f"{framing['bridge_statement'] or ''} {framing['proof_statement']}"
    ).strip()

    return framing
```

## Analogy Framing Templates

```
FRAMING BY TIER:

TIER 1 (Direct):
"[Study] shows [compound] [result]. [Product] contains this exact [compound]."

TIER 2 (Same Pathway):
"The [pathway] is well-established in science. [Analogous compound] proves
[pathway] works. [Product] targets this same [pathway]."

TIER 3 (Same Class):
"[Mechanism class] compounds have been shown to [effect]. [Product] harnesses
these same [mechanism class] properties."

TIER 4 (Conceptual):
"The principle of [concept] is proven. [Product] applies this proven approach."

NEVER:
- Claim direct proof when analogy is Tier 3-4
- Omit the bridge statement for Tier 2+
- Imply your specific product was tested when citing analogous studies
```
