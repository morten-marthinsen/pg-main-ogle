# 3.2 Study Discovery

## Purpose
Find external research studies, clinical trials, and peer-reviewed papers that can fill AUTHORITY and MECHANISM proof gaps. This microskill includes MCP/API integration protocols for automated research discovery.

---

## Input Requirements

From 3.1 Discovery Routing:
- Gaps routed to study discovery
- Search terms generated
- Category context (authority vs mechanism)

From Context:
- Mechanism to prove
- Claims requiring support
- Niche/industry

## Study Discovery Sources

```
SOURCE HIERARCHY (by credibility):

TIER 1 - Highest Authority
├── PubMed / MEDLINE
├── Cochrane Library (systematic reviews)
├── NIH / Government databases
├── Major university research repositories
└── High-impact factor journals (Nature, Science, NEJM, Lancet)

TIER 2 - Strong Authority
├── Google Scholar
├── Academic databases (JSTOR, ScienceDirect)
├── ResearchGate
├── Peer-reviewed specialty journals
└── Clinical trial registries (clinicaltrials.gov)

TIER 3 - Moderate Authority
├── Industry journals
├── White papers (disclosed funding)
├── Conference proceedings
├── Preprint servers (arXiv, bioRxiv)
└── Institutional research reports

TIER 4 - Limited Authority (use with caution)
├── Company-funded research
├── News coverage of research
├── Secondary citations
└── Non-peer-reviewed publications
```

## Execution Protocol

### Step 1: Search Query Construction

```python
def construct_study_queries(gap, mechanism, context):
    """
    Build search queries optimized for each source.
    """

    queries = {
        'pubmed': [],
        'google_scholar': [],
        'clinical_trials': [],
        'general_web': []
    }

    base_terms = gap.get('search_terms', [])
    mechanism_name = mechanism.get('name', '')
    ingredients = mechanism.get('ingredients', [])
    pathway = mechanism.get('pathway', '')

    # PubMed queries (MeSH-optimized)
    pubmed_queries = []
    for term in base_terms:
        pubmed_queries.append(f'{term}[Title/Abstract]')

    if mechanism_name:
        pubmed_queries.append(f'{mechanism_name}[Title/Abstract]')

    for ingredient in ingredients[:3]:  # Top 3 ingredients
        pubmed_queries.append(f'{ingredient}[Title/Abstract] AND clinical trial[pt]')

    queries['pubmed'] = pubmed_queries

    # Google Scholar queries
    for term in base_terms:
        queries['google_scholar'].append(f'"{term}" filetype:pdf')
        queries['google_scholar'].append(f'{term} systematic review')
        queries['google_scholar'].append(f'{term} meta-analysis')

    # Clinical trials queries
    for ingredient in ingredients:
        queries['clinical_trials'].append(ingredient)

    # General web queries (for expert commentary, news)
    for term in base_terms:
        queries['general_web'].append(f'{term} study research')
        queries['general_web'].append(f'{term} Dr. expert')

    return queries
```

### Step 3: Study Evaluation Protocol

```python
def evaluate_study(study_data):
    """
    Score discovered study for quality and relevance.
    """

    evaluation = {
        'study_id': generate_id(),
        'credibility_score': 0,
        'relevance_score': 0,
        'usability_score': 0,
        'quality_tier': None
    }

    # CREDIBILITY SCORING (1-10)
    credibility = 5  # Base

    # Source quality
    if is_peer_reviewed(study_data):
        credibility += 2
    if is_high_impact_journal(study_data):
        credibility += 2
    if has_large_sample(study_data, threshold=100):
        credibility += 1
    if is_randomized_controlled(study_data):
        credibility += 1
    if is_meta_analysis(study_data):
        credibility += 2

    # Deductions
    if is_company_funded(study_data):
        credibility -= 2
    if is_preprint(study_data):
        credibility -= 1
    if sample_size_small(study_data, threshold=30):
        credibility -= 2

    evaluation['credibility_score'] = max(1, min(10, credibility))

    # RELEVANCE SCORING (1-10)
    relevance = 5  # Base

    if proves_exact_mechanism(study_data):
        relevance += 3
    elif proves_similar_mechanism(study_data):
        relevance += 1

    if same_target_population(study_data):
        relevance += 2
    if proves_claimed_outcome(study_data):
        relevance += 2

    evaluation['relevance_score'] = max(1, min(10, relevance))

    # USABILITY SCORING (1-10)
    usability = 5  # Base

    if has_quotable_statistic(study_data):
        usability += 2
    if has_clear_conclusion(study_data):
        usability += 2
    if is_recent(study_data, years=5):
        usability += 1
    if has_full_citation(study_data):
        usability += 1

    evaluation['usability_score'] = max(1, min(10, usability))

    # QUALITY TIER
    avg_score = (evaluation['credibility_score'] +
                 evaluation['relevance_score'] +
                 evaluation['usability_score']) / 3

    if avg_score >= 8:
        evaluation['quality_tier'] = 'knockout'
    elif avg_score >= 6:
        evaluation['quality_tier'] = 'strong'
    elif avg_score >= 4:
        evaluation['quality_tier'] = 'usable'
    else:
        evaluation['quality_tier'] = 'weak'

    return evaluation
```

### Step 4: Citation Extraction

```python
def extract_citation(study_data):
    """
    Extract properly formatted citation for use in copy.
    """

    citation = {
        'full_apa': None,
        'short_form': None,
        'in_text': None,
        'components': {}
    }

    # Extract components
    citation['components'] = {
        'authors': extract_authors(study_data),
        'year': extract_year(study_data),
        'title': extract_title(study_data),
        'journal': extract_journal(study_data),
        'volume': extract_volume(study_data),
        'pages': extract_pages(study_data),
        'doi': extract_doi(study_data)
    }

    c = citation['components']

    # Build citation formats
    if c['authors'] and c['year'] and c['journal']:
        # Full APA
        citation['full_apa'] = (
            f"{c['authors']} ({c['year']}). {c['title']}. "
            f"{c['journal']}, {c.get('volume', '')}, {c.get('pages', '')}. "
            f"{c.get('doi', '')}"
        ).strip()

        # Short form for copy
        first_author = c['authors'].split(',')[0] if ',' in c['authors'] else c['authors'].split(' ')[0]
        citation['short_form'] = f"{first_author} et al., {c['journal']}, {c['year']}"

        # In-text
        citation['in_text'] = f"({first_author} et al., {c['year']})"

    return citation
```

## MCP Tool Selection Guide

```
WHEN TO USE EACH TOOL:

Perplexity Research (perplexity_research)
├── Best for: Comprehensive academic research with citations
├── When: Need synthesis across multiple sources
├── Output: Summarized findings with source links
└── Use first for broad discovery

Firecrawl Scrape (firecrawl_scrape)
├── Best for: Extracting specific study details from known URLs
├── When: Have a specific PubMed link or study page
├── Output: Full markdown content
└── Use for deep extraction after discovery

Firecrawl Search (firecrawl_search)
├── Best for: Finding studies across the web
├── When: Need to search multiple sources simultaneously
├── Output: Search results with optional content scraping
└── Use for broad web discovery

Apify RAG Browser (apify-slash-rag-web-browser)
├── Best for: Complex multi-page research tasks
├── When: Need to follow links and synthesize
├── Output: Markdown content optimized for LLM processing
└── Use for deep research tasks
```
