# 2.6 Gap Severity Scoring

## Purpose
Score and prioritize gaps by how urgently they need to be filled. Gap severity depends on category importance at the current Schwartz stage, gap type, and impact on promise capability.

---

## Input Requirements

From 2.5 Gap Detection:
- List of all identified gaps
- Gap types and categories
- Schwartz stage context

---

## Severity Scoring Formula

```python
def calculate_gap_severity(gap, schwartz_stage, promise_context=None):
    # Base severity by gap type
    type_severity = BASE_SEVERITY[gap['gap_type']]

    # Stage importance multiplier
    stage_multiplier = STAGE_IMPORTANCE[schwartz_stage][gap['category']]

    # Promise impact (if applicable)
    promise_multiplier = 1.0
    if promise_context and gap_blocks_promise(gap, promise_context):
        promise_multiplier = 1.3

    # Final severity (0-10)
    severity = type_severity * stage_multiplier * promise_multiplier

    return min(round(severity, 1), 10)
```

BASE_SEVERITY = {

    'under_represented': 4, # Too few elements

## Execution Protocol

### Step 1: Score Each Gap

```python
def score_all_gaps(gaps, schwartz_stage, promise_context=None):
    scored_gaps = []

    for gap in gaps:
        base = BASE_SEVERITY[gap['gap_type']]
        stage_mult = STAGE_IMPORTANCE[schwartz_stage].get(gap.get('category'), 1.0)

        # Promise blocking check
        promise_mult = 1.0
        if promise_context:
            if gap_blocks_promise(gap, promise_context):
                promise_mult = 1.3
                gap['blocks_promise'] = True

        severity = min(base * stage_mult * promise_mult, 10)

        scored_gaps.append({
            **gap,
            'severity_score': round(severity, 1),
            'severity_factors': {
                'base_severity': base,
                'stage_multiplier': stage_mult,
                'promise_multiplier': promise_mult
            }
        })

    return scored_gaps
```

### Step 2: Rank by Severity

```python
def rank_gaps_by_severity(scored_gaps):
    # Sort descending by severity score
    ranked = sorted(scored_gaps, key=lambda g: g['severity_score'], reverse=True)

    # Assign priority ranks
    for i, gap in enumerate(ranked):
        gap['priority_rank'] = i + 1

    return ranked
```

### Step 3: Classify Severity Tiers

```python
def classify_severity_tier(severity_score):
    if severity_score >= 8.0:
        return 'critical'
    elif severity_score >= 6.0:
        return 'high'
    elif severity_score >= 4.0:
        return 'medium'
    else:
        return 'low'
```

### Step 4: Generate Priority Actions

```python
def generate_priority_actions(ranked_gaps, top_n=3):
    actions = []

    for gap in ranked_gaps[:top_n]:
        action = {
            'gap_id': gap['gap_id'],
            'priority': gap['priority_rank'],
            'category': gap.get('category', 'general'),
            'gap_type': gap['gap_type'],
            'severity': gap['severity_score'],
            'action': generate_action_text(gap),
            'expected_impact': estimate_impact(gap)
        }
        actions.append(action)

    return actions

def generate_action_text(gap):
    actions = {
        'missing': f"URGENT: Find or create {gap['category']} proof",
        'weak': f"Improve {gap['category']} proof quality (all elements weak)",
        'claim_gap': f"Find proof for claim: {gap.get('claim', 'unspecified')}",
        'quality_gap': f"Find strong (7+) {gap['category']} proof element",
        'under_represented': f"Add more {gap['category']} elements (currently {gap.get('current_count', '?')})",
        'subtype_gap': f"Add {gap.get('missing_subtype', 'specific type')} proof"
    }
    return actions.get(gap['gap_type'], 'Address proof gap')
```

## Severity Score Interpretation

| Score | Tier | Urgency | Action |
|-------|------|---------|--------|
| **9-10** | Critical | Immediate | Must address before making claims |
| **7-8** | High | Soon | Address in Discovery/Generation |
| **5-6** | Medium | When possible | Improve if resources allow |
| **3-4** | Low | Optional | Nice to have, not blocking |
| **1-2** | Minimal | Deprioritize | Only if time permits |
