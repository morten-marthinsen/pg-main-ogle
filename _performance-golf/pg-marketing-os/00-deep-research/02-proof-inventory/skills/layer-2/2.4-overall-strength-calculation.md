# 2.4 Overall Strength Calculation

## Purpose
Calculate the 0-100 overall proof strength score that summarizes the entire inventory's quality. This score drives promise ceiling calculation and provides a single metric for proof health assessment.

---

## Input Requirements

From 2.3 Category Strength Scoring:
- Category strength scores (0-10) for all 6 categories
- Category status (missing/weak/moderate/adequate/robust)
- Schwartz stage context

---

## Overall Strength Formula

```python
def calculate_overall_strength(category_scores, schwartz_stage):
    # Stage-specific category weights
    weights = get_stage_weights(schwartz_stage)

    # Calculate weighted sum
    weighted_sum = sum(
        category_scores[cat]['strength'] * weights[cat]
        for cat in weights.keys()
    )

    # Scale to 0-100 (base max is 10)
    base_score = weighted_sum * 10

    # Apply penalty for missing categories
    missing_count = count_missing_categories(category_scores)
    penalty = missing_count * 10  # -10 points per missing category

    # Apply penalty for critical gaps (stage-specific)
    critical_penalty = calculate_critical_gap_penalty(
        category_scores, schwartz_stage
    )

    # Final score (floor at 0)
    final_score = max(0, base_score - penalty - critical_penalty)

    return round(final_score, 0)
```

## Execution Protocol

### Step 1: Apply Stage Weights

```python
# Example for Stage 3 market
weights = STAGE_WEIGHTS[3]

weighted_sum = (
    social_strength * 0.15 +
    authority_strength * 0.20 +
    demonstration_strength * 0.15 +
    mechanism_strength * 0.25 +  # Weighted highest
    data_strength * 0.15 +
    risk_reversal_strength * 0.10
)

# Example calculation:
# (7.2 * 0.15) + (6.1 * 0.20) + (5.4 * 0.15) + (4.2 * 0.25) + (4.8 * 0.15) + (6.5 * 0.10)
# = 1.08 + 1.22 + 0.81 + 1.05 + 0.72 + 0.65
# = 5.53

base_score = 5.53 * 10 = 55.3
```

### Step 2: Apply Missing Category Penalty

```python
def count_missing_categories(category_scores):
    return sum(
        1 for cat in category_scores
        if category_scores[cat]['status'] == 'missing'
    )

# -10 points per missing category
penalty = missing_count * 10
```

### Step 3: Apply Critical Gap Penalty

Critical gaps are category weaknesses that are especially damaging at the current stage:

```python
CRITICAL_CATEGORIES = {
    1: ['demonstration'],       # Stage 1: Must show it works
    2: ['social'],              # Stage 2: Must show others succeed
    3: ['mechanism'],           # Stage 3: Must prove mechanism
    4: ['mechanism', 'authority'],  # Stage 4: Must prove unique + credible
    5: ['social', 'risk_reversal']  # Stage 5: Must enable identification
}

def calculate_critical_gap_penalty(category_scores, stage):
    critical_cats = CRITICAL_CATEGORIES[stage]
    penalty = 0

    for cat in critical_cats:
        status = category_scores[cat]['status']
        if status == 'missing':
            penalty += 15  # Severe penalty
        elif status == 'weak':
            penalty += 10  # Significant penalty
        elif status == 'moderate':
            penalty += 5   # Mild penalty

    return penalty
```

### Step 4: Calculate Final Score

```python
final_score = max(0, base_score - missing_penalty - critical_penalty)

# Example:
# base_score = 55.3
# missing_penalty = 0 (no missing categories)
# critical_penalty = 10 (mechanism is 'weak' at Stage 3)
# final_score = max(0, 55.3 - 0 - 10) = 45.3 → 45
```
