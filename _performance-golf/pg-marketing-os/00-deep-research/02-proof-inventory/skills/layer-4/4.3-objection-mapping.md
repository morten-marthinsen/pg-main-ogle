# 4.3 Objection Mapping

## Purpose
Map proof elements to specific prospect objections they can overcome. Every major objection should have dedicated proof assigned to answer it. This microskill ensures the inventory addresses the full objection landscape.

---

## Input Requirements

From Layer 1-2:
- All scored proof elements
- Category classifications

From Context:
- Schwartz stage
- Niche/industry
- Known objections (from deep research or root cause)

From 00-Deep-Research:
- Dominant objections in market
- Competitor objection handling

---

## Universal Objection Framework

```
UNIVERSAL OBJECTIONS (Every Market):

1. "DOES IT WORK?"
   └── Proof needed: Results, demonstrations, testimonials

2. "WHY DOES IT WORK?"
   └── Proof needed: Mechanism, science, expert explanation

3. "WILL IT WORK FOR ME?"
   └── Proof needed: Relatable testimonials, variety of success stories

4. "IS IT SAFE?"
   └── Proof needed: Studies, certifications, expert endorsement

5. "CAN I TRUST THESE PEOPLE?"
   └── Proof needed: Credentials, track record, media coverage

6. "IS IT WORTH THE MONEY?"
   └── Proof needed: Value comparison, ROI data, satisfaction stats

7. "WHAT IF IT DOESN'T WORK?"
   └── Proof needed: Guarantee, trial, refund policy

8. "WHY HAVEN'T I HEARD OF THIS?"
   └── Proof needed: Novelty explanation, suppression narrative, recent discovery

9. "WHY IS THIS DIFFERENT?"
   └── Proof needed: Mechanism differentiation, comparison proof

10. "WHY NOW?"
    └── Proof needed: Urgency data, timing evidence, scarcity proof
```

    objections.extend(stage_specific)

    # Add context-specific objections
    niche_objections = get_niche_objections(context.get('niche', ''))
    objections.extend(niche_objections)

    return objections

### Step 2: Objection-to-Proof Matching

```python
def match_proof_to_objections(elements, objections):
    """
    Map proof elements to the objections they can overcome.
    """

    # Category-to-objection mapping
    CATEGORY_OBJECTION_MAP = {
        'OBJ-01': ['demonstration', 'social', 'data'],       # Does it work?
        'OBJ-02': ['mechanism', 'authority'],                 # Why does it work?
        'OBJ-03': ['social'],                                  # Will it work for me?
        'OBJ-04': ['authority', 'data', 'social'],            # Is it safe?
        'OBJ-05': ['authority', 'social'],                     # Can I trust them?
        'OBJ-06': ['data', 'social', 'demonstration'],        # Worth the money?
        'OBJ-07': ['risk_reversal', 'social'],                # What if it doesn't work?
        'OBJ-08': ['mechanism', 'demonstration', 'data'],     # How is this different?
        'OBJ-09': ['authority', 'mechanism'],                  # Why believe mechanism?
        'OBJ-10': ['authority', 'data'],                       # Why haven't I heard?
        'OBJ-11': ['social'],                                  # I've tried everything
        'OBJ-12': ['social']                                   # Nothing works for me
    }

    # Sub-type specificity bonuses
    SUBTYPE_OBJECTION_BONUS = {
        'OBJ-01': {
            'demo_before_after_photo': 2,
            'testimonial_specific_result': 2,
            'study_clinical_trial': 1.5
        },
        'OBJ-02': {
            'mechanism_study': 2,
            'mechanism_expert_explanation': 2,
            'study_peer_reviewed': 1.5
        },
        'OBJ-03': {
            'testimonial_transformation': 2,  # Relatable story
            'case_study': 1.5
        },
        'OBJ-07': {
            'guarantee_unconditional': 2,
            'guarantee_double': 2,
            'trial_free': 2
        },
        'OBJ-11': {
            'testimonial_transformation': 2  # "I tried everything too"
        }
    }

    objection_mapping = {}

    for objection in objections:
        obj_id = objection['id']
        preferred_categories = CATEGORY_OBJECTION_MAP.get(obj_id, [])

        # Find matching elements
        matches = []
        for element in elements:
            if element.category in preferred_categories:
                # Calculate match strength
                base_strength = element.stage_adjusted_score

                # Apply sub-type bonus
                subtype_bonuses = SUBTYPE_OBJECTION_BONUS.get(obj_id, {})
                bonus = subtype_bonuses.get(element.sub_type, 1.0)
                match_strength = base_strength * bonus

                matches.append({
                    'element': element,
                    'match_strength': match_strength,
                    'category': element.category,
                    'sub_type': element.sub_type
                })

        # Sort by match strength
        matches.sort(key=lambda x: x['match_strength'], reverse=True)

        objection_mapping[obj_id] = {
            'objection': objection['objection'],
            'intensity': objection['intensity'],
            'proof_matches': matches[:5],  # Top 5 matches
            'coverage_status': assess_coverage(matches, objection['intensity'])
        }

    return objection_mapping


def assess_coverage(matches, intensity):
    """
    Assess if objection has adequate proof coverage.
    """

    if not matches:
        return 'uncovered'

    top_match = matches[0]['match_strength']

    intensity_thresholds = {
        'very_high': 7.5,
        'high': 6.5,
        'medium': 5.5,
        'low': 4.5
    }

    threshold = intensity_thresholds.get(intensity, 6.0)

    if top_match >= threshold and len(matches) >= 2:
        return 'strong'
    elif top_match >= threshold * 0.8:
        return 'adequate'
    elif matches:
        return 'weak'
    else:
        return 'uncovered'
```

### Step 3: Generate Objection Answer Sequences

```python
def generate_objection_sequences(objection_mapping):
    """
    For each objection, generate a recommended proof sequence.
    """

    sequences = {}

    for obj_id, mapping in objection_mapping.items():
        matches = mapping['proof_matches']

        if not matches:
            sequences[obj_id] = {
                'sequence': [],
                'approach': 'NO PROOF - Objection uncovered',
                'recommendation': 'Generate proof or acknowledge limitation'
            }
            continue

        # Build sequence based on category mix
        sequence = []
        categories_used = set()

        for match in matches[:3]:
            if match['category'] not in categories_used:
                sequence.append(match)
                categories_used.add(match['category'])

        # If only one category, add second element anyway
        if len(sequence) < 2 and len(matches) >= 2:
            sequence.append(matches[1])

        sequences[obj_id] = {
            'sequence': sequence,
            'approach': determine_approach(mapping['objection'], sequence),
            'copy_guidance': generate_objection_copy_guidance(mapping['objection'], sequence)
        }

    return sequences


def determine_approach(objection_text, sequence):
    """
    Determine the best approach to answer this objection.
    """

    if not sequence:
        return 'UNCOVERED'

    categories = [s['category'] for s in sequence]

    if 'demonstration' in categories:
        return 'SHOW-THEN-TELL: Lead with visual/demo proof, then explain'
    elif 'social' in categories and 'authority' in categories:
        return 'STACK: Testimonial + expert for dual validation'
    elif 'mechanism' in categories:
        return 'EXPLAIN: Lead with mechanism proof to build understanding'
    elif 'risk_reversal' in categories:
        return 'REMOVE RISK: Lead with guarantee to eliminate objection'
    else:
        return 'PROVE: Present strongest proof directly'
```
