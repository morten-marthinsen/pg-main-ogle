# Skill 2.1-A: Pattern Analyzer

**Version:** 2.0
**Layer:** 2 (Intelligence)
**Type:** Pattern Recognition
**Dependencies:** scored_quotes.json, market_config.yaml
**Output:** pattern_analysis.json

---

## Purpose

Analyze the quote corpus to identify recurring patterns, themes, and clusters. This surfaces the dominant narratives, common pain points, and repeated experiences that define the market psychology.

---

## Pattern Types

```
PATTERN CATEGORIES:

1. LANGUAGE PATTERNS
   - Repeated phrases
   - Common vocabulary
   - Metaphors and analogies

2. EXPERIENCE PATTERNS
   - Shared journeys
   - Common sequences
   - Typical timelines

3. EMOTIONAL PATTERNS
   - Recurring feelings
   - Emotional progressions
   - Peak moments

4. BEHAVIORAL PATTERNS
   - Common actions
   - Typical mistakes
   - Habitual responses

5. BELIEF PATTERNS
   - Shared worldviews
   - Common assumptions
   - Limiting beliefs
```

## Analysis Logic

### Step 1: Language Pattern Detection

```
ANALYZE across all quotes:

PHRASE FREQUENCY:
- Extract 2-5 word phrases
- Count occurrences across quotes
- Filter for frequency >= 5
- Rank by frequency

VOCABULARY ANALYSIS:
- Identify market-specific terms
- Note emotional vocabulary
- Track metaphor usage

EXAMPLE detection:
- "I can't seem to..." (struggle)
- "No matter what I try..." (futility)
- "I finally..." (breakthrough)
- "The real problem is..." (insight)

OUTPUT language_patterns[] with:
- Pattern text
- Frequency count
- Example quotes
- Significance interpretation
```

### Step 2: Experience Pattern Detection

```
IDENTIFY shared journeys:

TIMELINE PATTERNS:
- How long people struggle
- Common milestones
- Typical progression

SEQUENCE PATTERNS:
- What happens first
- What follows
- Where people get stuck

COMMON JOURNEYS:

"The Trying Loop":
  1. Start with hope
  2. Try approach
  3. Initial success
  4. Eventual failure
  5. Back to start

"The Escalation Pattern":
  1. Start small
  2. Increase risk
  3. Big loss
  4. Emotional decision
  5. Bigger loss

EXTRACT patterns with stages, frequency, examples
```

### Step 3: Emotional Pattern Detection

```
MAP emotional landscape:

EMOTION FREQUENCY:
- Count emotional indicators
- Calculate intensity averages
- Note co-occurrences

EMOTIONAL TRIGGERS:
- What causes each emotion
- Context patterns
- Trigger → emotion mapping

EMOTIONAL PROGRESSIONS:
- Hope → frustration
- Confidence → fear
- Excitement → regret

BUILD emotional_patterns[] with:
- Emotion name
- Frequency
- Average intensity
- Common triggers
- Progression patterns
```

### Step 4: Behavioral Pattern Detection

```
IDENTIFY common behaviors:

ACTION PATTERNS:
- What people do when struggling
- Reactions to setbacks
- Coping mechanisms

MISTAKE PATTERNS:
- Repeated errors
- Self-sabotage behaviors
- Cognitive biases in action

EXAMPLES:
- "Revenge trading" after losses
- "Moving stops" to avoid being wrong
- "Averaging down" on losers
- "Overtrading" when winning

OUTPUT behavioral_patterns[] with:
- Behavior name
- Frequency
- Typical context
- Typical outcome
- Example quotes
```

### Step 5: Belief Pattern Detection

```
SURFACE underlying beliefs:

EXPLICIT BELIEFS:
- Stated worldviews
- "I believe...", "I think..."
- Assertions about how things work

IMPLICIT BELIEFS:
- Assumptions in language
- Unquestioned premises
- Cultural narratives

LIMITING BELIEFS:
- "I'm just not..."
- "People like me can't..."
- "It's impossible to..."

EMPOWERING BELIEFS:
- "I realized..."
- "What changed was..."
- "The key is..."

OUTPUT belief_patterns[] with:
- Belief statement
- Frequency
- Supporting quotes
- Counter-evidence (if any)
- Marketing implication
```

### Step 6: Theme Clustering

```
CLUSTER related patterns:

GROUP patterns by theme:
- Mindset/psychology cluster
- Technical/method cluster
- Resource/time cluster
- Social/identity cluster
- Results/outcome cluster

FOR each cluster:
  - Name the cluster
  - Identify core theme
  - List related patterns
  - Count total quotes
  - Note aspects covered
  - Calculate importance score

IMPORTANCE SCORE:
= (quote_count × 0.4) + (pattern_count × 0.3) + (emotional_intensity × 0.3)
Normalize to 0-100
```

---

## Pattern Significance

```
FOR each pattern:

ASSESS significance:

HIGH significance (priority attention):
- Frequency >= 10% of bucket
- High emotional intensity
- Appears across platforms
- Has clear marketing application

MEDIUM significance:
- Frequency 5-10%
- Moderate intensity
- Platform-specific
- Potential application

LOW significance:
- Frequency < 5%
- Low intensity
- Isolated occurrences
- Limited application
```

---

## Quality Criteria

1. **Coverage** - Patterns explain majority of quotes
2. **Distinctness** - Patterns are meaningfully different
3. **Actionability** - Patterns suggest marketing angles
4. **Validation** - Patterns have sufficient examples
