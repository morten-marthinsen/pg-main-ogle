# Skill 2.1-B: Theme Synthesizer

**Version:** 2.0
**Layer:** 2 (Intelligence)
**Type:** Synthesis
**Dependencies:** pattern_analysis.json, scored_quotes.json
**Output:** theme_synthesis.json

---

## Purpose

Synthesize patterns into cohesive market themes. Create the narrative architecture that explains the market's psychology, connects pain to hope, and reveals the story the market is waiting to hear.

---

## Theme Framework

```
A THEME IS:

A unifying narrative that:
- Connects multiple patterns
- Explains why problems persist
- Reveals underlying truth
- Points toward resolution
- Resonates emotionally

THEME STRUCTURE:
1. The Pain Reality (what's happening)
2. The Hidden Truth (why it persists)
3. The Hope Element (what could change)
4. The Resolution Path (how to get there)
```

## Synthesis Logic

### Step 1: Theme Seed Identification

```
FROM pattern_analysis:

IDENTIFY theme seeds from clusters:

FOR each theme_cluster:
  IF importance_score >= 60:
    CREATE theme_seed:
    {
      name: cluster_name
      core_patterns: related_patterns
      quote_base: quotes in cluster
      initial_insight: cluster's core_theme
    }

ALSO consider:
- High-frequency language patterns
- Strong emotional patterns
- Prevalent behavioral patterns

COMPILE theme_seeds[] (target: 5-10 seeds)
```

### Step 2: Theme Development

```
FOR each theme_seed:

DEVELOP full theme:

1. PAIN REALITY:
   - What is the actual experience?
   - How does it feel?
   - How common is it?
   - What are the consequences?

   PULL from pain quotes matching patterns
   SYNTHESIZE into summary

2. HIDDEN TRUTH:
   - Why does this problem persist?
   - What are people missing?
   - What's the underlying cause they don't see?

   PULL from root_cause quotes
   ANALYZE for deeper meaning
   FORMULATE insight

3. HOPE ELEMENT:
   - What do people wish for?
   - What would success look like?
   - What keeps them trying?

   PULL from hope quotes
   IDENTIFY desired outcomes

4. RESOLUTION PATH:
   - What would actually solve this?
   - What transformation is needed?
   - What barriers exist?

   SYNTHESIZE from all buckets
   Map to potential solution
```

### Step 3: Theme Validation

```
FOR each developed theme:

VALIDATE:
- [ ] Minimum 50 supporting quotes
- [ ] Pain and hope connected
- [ ] Hidden truth is insightful (not obvious)
- [ ] Resolution path is clear
- [ ] Multiple patterns support theme

CALCULATE importance_score:
= (quote_count / 100) × 30 +
  (pattern_count × 10) +
  (emotional_intensity_avg × 5) +
  (cross_aspect_presence × 10)

IF validation fails:
  - Merge with related theme
  - Or demote to sub-theme
```

### Step 4: Marketing Application Mapping

```
FOR each validated theme:

GENERATE marketing_applications:

HEADLINE ANGLES:
- What hook addresses this theme?
- How to open the conversation?
- What pattern interrupt works?

MECHANISM OPPORTUNITIES:
- What solution angle fits?
- How would mechanism address hidden truth?
- What proprietary approach works?

PROOF DIRECTIONS:
- What would prove resolution possible?
- What testimonial would fit?
- What demonstration works?

OUTPUT specific, actionable applications
```

### Step 5: Theme Relationship Mapping

```
ANALYZE relationships between themes:

RELATIONSHIP TYPES:
- Cause-effect (Theme A causes Theme B)
- Sequential (A comes before B)
- Complementary (A and B occur together)
- Contrasting (A and B are opposites)
- Nested (A contains B)

FOR each theme pair:
  IDENTIFY relationship type
  NOTE connection points
  CONSIDER combined messaging

BUILD theme_relationships[]
```

### Step 6: Market Narrative Construction

```
SYNTHESIZE into master narrative:

DETERMINE:
- Dominant theme (highest importance)
- Secondary themes (supporting)
- Theme hierarchy

CONSTRUCT narrative_arc:
"In this market, [dominant theme pain reality].
People experience [secondary theme elements].
The hidden truth is [key insight from themes].
What they're really looking for is [core hope].
The path forward requires [resolution elements]."

OUTPUT as market_narrative
```

---

## Quality Criteria

1. **Coherence** - Themes tell a unified story
2. **Grounding** - All themes have quote evidence
3. **Insight** - Hidden truths are genuinely insightful
4. **Applicability** - Clear marketing uses
