# Skill 2.5-A: Sophistication Analyzer

**Version:** 2.0
**Layer:** 2 (Intelligence)
**Type:** Market Sophistication Diagnosis
**Dependencies:** competitor_analysis.json, awareness_map.json, mechanism_inventory.json
**Output:** sophistication_analysis.json

---

## Purpose

Diagnose the market's sophistication level using Schwartz's Market Sophistication Model. This determines the messaging strategy - what has already been said, what's worn out, and what new ground exists.

---

## Sophistication Framework (Schwartz)

```
5 STAGES OF MARKET SOPHISTICATION:

STAGE 1: VIRGIN MARKET
- First product in category
- Simple, direct claims work
- "This product does X"
- No competition, no skepticism

STAGE 2: ENLARGED CLAIMS
- Competition has entered
- Need bigger/better claims
- "Does X faster/bigger/more"
- Arms race of promises

STAGE 3: MECHANISM ERA
- Claims are exhausted
- Need to explain HOW it works
- "Does X because of [mechanism]"
- Credibility through explanation

STAGE 4: MECHANISM FATIGUE
- Mechanisms are overused
- Need enhanced mechanisms
- "The [new/improved/unique] mechanism"
- Standing out in noise

STAGE 5: IDENTITY/BELIEF ERA
- Everything has been said
- Connect to identity/worldview
- "For people who [identity]"
- Tribal, belief-based appeals

Most mature markets are in Stages 3-5.
```

## Analysis Logic

### Step 1: Identify Stage Indicators

```
ANALYZE market signals for each stage:

STAGE 1 INDICATORS:
- Few competitors
- Simple, direct claims
- High response to basic offers
- Minimal skepticism
- (Rare in established markets)

STAGE 2 INDICATORS:
- Many competitors
- "Bigger/better/faster" claims
- Comparison language
- Promise escalation
- Some skepticism emerging

STAGE 3 INDICATORS:
- Mechanisms prominent in ads
- "This works because..." language
- Educational content marketing
- How/why explanations
- Moderate skepticism

STAGE 4 INDICATORS:
- Multiple competing mechanisms
- Enhanced/unique mechanism claims
- "Unlike other..." language
- Differentiation emphasis
- High skepticism

STAGE 5 INDICATORS:
- Identity-based appeals
- Tribal language
- Worldview alignment
- Experience/transformation focus
- Very high skepticism
```

### Step 2: Evidence Collection

```
FROM competitor_analysis and ads_content:

COUNT and CATEGORIZE:
- Simple claims (Stage 1-2)
- Mechanism explanations (Stage 3)
- Enhanced/unique mechanisms (Stage 4)
- Identity/tribe appeals (Stage 5)

NOTE prevalence of each type

IDENTIFY dominant patterns
```

### Step 3: Claim Landscape Analysis

```
CATALOG all claims in market:

COMMON CLAIMS:
- What do most competitors say?
- What promises are universal?

EXHAUSTED CLAIMS:
- What's been said so many times it's invisible?
- What generates eye-rolls?

STILL EFFECTIVE:
- What claims still work?
- What hasn't been overused?

This reveals where claim innovation is needed
```

### Step 4: Mechanism Saturation Check

```
FROM mechanism_inventory:

SATURATED MECHANISMS:
- Used by 3+ competitors
- Mentioned in most ads
- Customers have heard it

EMERGING MECHANISMS:
- Used by 1-2 competitors
- Gaining traction
- Still has novelty

OPPORTUNITY MECHANISMS:
- Not explicitly used
- Implied by customer needs
- White space for development
```

### Step 5: Stage Determination

```
WEIGH evidence across stages:

CALCULATE stage_score for each:
- Count indicators present
- Weight by prominence
- Adjust for recency

DETERMINE current_stage:
- Highest weighted score
- Consider blend (e.g., "Late Stage 3, early Stage 4")

SET confidence level:
- Clear dominance: High
- Mixed signals: Medium
- Unclear: Low

COMPILE evidence list
```

### Step 6: Strategic Implications

```
BASED on diagnosed stage:

IF Stage 3:
  messaging_strategy: "Lead with unique mechanism"
  claim_approach: "Explain HOW it works differently"
  mechanism_approach: "Develop proprietary mechanism"
  differentiation_path: "Better explanation wins"

IF Stage 4:
  messaging_strategy: "Enhanced mechanism + proof"
  claim_approach: "Acknowledge what they've tried, explain why this is different"
  mechanism_approach: "Unique twist on mechanism, heavy proof"
  differentiation_path: "Credibility + novelty wins"

IF Stage 5:
  messaging_strategy: "Identity/transformation focus"
  claim_approach: "Who you become, not what you get"
  mechanism_approach: "Mechanism serves identity, not product"
  differentiation_path: "Tribal belonging wins"

IF Mixed (e.g., 3-4):
  BLEND strategies appropriately
```

### Step 7: Positioning Recommendations

```
SYNTHESIZE recommendations:

PRIMARY STRATEGY:
- Best approach for this stage
- Core messaging direction

SUPPORTING TACTICS:
- Additional angles
- Proof approaches
- Content strategies

AVOID:
- What not to do
- Exhausted approaches
- Stage-inappropriate tactics
```

## Quality Criteria

1. **Accuracy** - Stage correctly diagnosed
2. **Evidence** - Claims backed by data
3. **Actionability** - Clear strategic implications
4. **Nuance** - Stage blends acknowledged
