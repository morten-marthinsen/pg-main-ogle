# Skill 1.4-F: Ads Library Scraper

**Version:** 2.0
**Layer:** 1 (Infrastructure)
**Type:** Content Extraction
**Dependencies:** market_config.yaml (competitors)
**Output:** Appends to master_raw_content.json

---

## Purpose

Scrape competitor advertisements from Meta Ads Library, Google Ads Transparency, and other ad repositories. Ads reveal proven messaging, positioning, and mechanisms that competitors have invested money to validate.

---

## Target Sources

```
AD LIBRARIES:

1. Meta Ads Library (Facebook/Instagram)
   - URL: facebook.com/08-ads/library
   - Most comprehensive for B2C

2. Google Ads Transparency Center
   - URL: adstransparency.google.com
   - Search and display ads

3. TikTok Creative Center
   - URL: ads.tiktok.com/business/creativecenter
   - Trending ad formats

4. LinkedIn Ads
   - Limited library access
   - Focus on B2B markets
```

---

## Execution Logic

### Step 1: Build Search List

```
FROM market_config.competitors:

FOR each competitor:
  ADD search terms:
  - Company name
  - Product names
  - Key personnel names
  - Domain (for Meta)

ADD category terms:
- Market category names
- Product type terms
- Common problem language
```

### Step 2: Meta Ads Library Scrape

```
TOOL CHAIN (Audit Fix FAIL-004 - Specific Actor Specification):

PRIMARY (Recommended):
mcp__apify__call-actor:
  actor: "apify/facebook-ads-scraper"  # Verified actor name
  input:
    searchTerms: [terms]
    countryCode: "US"
    adActiveStatus: "active"
    adType: "all"
    resultsLimit: 100

ALTERNATIVE ACTOR:
mcp__apify__call-actor:
  actor: "trudax/facebook-ads-library-scraper"  # Community actor
  input:
    searchQuery: [terms]
    country: "US"
    maxItems: 100

FALLBACK 1 (Firecrawl):
mcp__firecrawl-mcp__firecrawl_scrape:
  url: "https://www.facebook.com/08-ads/library/?active_status=active&ad_type=all&country=US&q=[search_term]"
  formats: ["markdown"]
  waitFor: 5000
  actions:
    - type: "scroll"
      direction: "down"
    - type: "wait"
      milliseconds: 2000

FALLBACK 2 (Perplexity):
mcp__perplexity__perplexity_research:
  messages:
    - role: "user"
      content: "Find active Facebook/Meta ads from [competitor_name] in the [market] space. Include: ad copy/text, headlines, calls to action, and estimated run duration if available."

FOR each ad found:
  CAPTURE:
  - Advertiser name
  - Ad ID
  - Start date (longevity signal)
  - Ad creative (text, images, video)
  - CTA button
```

### Step 3: Google Ads Transparency

```
TOOL:

mcp__firecrawl-mcp__firecrawl_scrape:
  url: "https://adstransparency.google.com/?search=[search_term]"
  formats: ["markdown"]
  waitFor: 3000

FOR each ad found:
  CAPTURE:
  - Advertiser
  - Ad format
  - Text content
  - Date range
  - Geographic targeting
```

### Step 4: Analyze Ad Creative

```
FOR each ad:

  EXTRACT hooks:
  - Opening lines
  - Attention-grabbing elements
  - Pattern interrupts

  EXTRACT mechanisms:
  - How it claims to work
  - Technical terms
  - Proprietary names
  - Framework language

  EXTRACT proof elements:
  - Testimonials
  - Statistics
  - Authority markers
  - Before/after

  EXTRACT emotional triggers:
  - Pain amplification
  - Hope creation
  - Fear of missing out
  - Identity appeals

  CLASSIFY CTA:
  - Direct (Buy, Sign Up)
  - Soft (Learn More, Watch)
  - Interactive (Quiz, Assessment)
```

### Step 5: Pattern Recognition

```
ACROSS all ads:

IDENTIFY messaging patterns:
- Common headline structures
- Repeated claims
- Shared proof types
- Similar CTAs

CALCULATE frequency:
- How often pattern appears
- Which competitors use it
- Apparent effectiveness (run duration)

FLAG:
- Overused patterns (saturated)
- Unique approaches (differentiation opportunity)
- Evolving trends (recent vs older ads)
```

### Step 6: Mechanism Inventory

```
COMPILE all mechanisms mentioned:

FOR each unique mechanism:
  RECORD:
  - Mechanism name/description
  - Which advertiser uses it
  - How it's positioned
  - Frequency across ads

CATEGORIZE:
- Root cause mechanisms
- Solution mechanisms
- Differentiating mechanisms
- Commodity mechanisms (everyone uses)
```

---

## Longevity Analysis

```
AD LONGEVITY = VALIDATION

Long-running ads (3+ months):
- Proven messaging
- Validated hooks
- Working mechanisms

Recently started ads:
- Testing new angles
- Trend responses
- Seasonal variations

Track run_duration for each ad
FLAG ads running 6+ months as "proven"
```

---

## Tool Resilience

```
AD LIBRARY SCRAPING FALLBACK:

Meta:
1. apify/facebook-ads-library-scraper
2. Firecrawl direct scrape
3. Manual screenshot capture (last resort)

Google:
1. Firecrawl scrape
2. Perplexity with site:adstransparency.google.com

TikTok:
1. Firecrawl on Creative Center
2. Manual review

IF automated scraping fails:
- Log limitation
- Provide manual search URLs
- Focus on available platforms
```

---

## Quality Criteria

1. **Recency** - Focus on active ads
2. **Depth** - Full creative text, not just headlines
3. **Attribution** - Link to specific competitors
4. **Analysis** - Extract patterns, not just collect
