## Execution Protocol

### Step 1: Verify All Validators Passed

Before assembly, confirm:

```python
def verify_validators(validations):
    checks = {
        "logical_coherence": validations["logical_coherence"]["pass"],
        "persuasion_flow": validations["persuasion_flow"]["pass"],
        "anti_slop": validations["anti_slop"]["status"] in ["pass", "pass_with_warnings"],
        "vault_comparison": validations["vault_comparison"]["pass"],
    }

    all_passed = all(checks.values())
    if not all_passed:
        failed = [k for k, v in checks.items() if not v]
        return {"status": "blocked", "failed_validators": failed}
    return {"status": "clear_to_assemble"}
```

**If any validator failed:** Do NOT assemble. Return to the appropriate layer for remediation.

### Step 3: Assemble Campaign Thesis Section

### Step 4: Assemble Argument Strategy Section

Pull from Layer 1.4 output:

### Step 5: Assemble Prospect Gap Map Section

Pull from Layer 1.2 output:

### Step 6: Assemble CPB Chunks Section

For each chunk in the finalized sequence order (from Layer 3.1), compile:
- chunk_id, gap_question, gap_id, position (from final sequence)
- claim: statement, type, scores, opinion_test (from Layer 2.1)
- proof_elements: proof_id, type, content_summary, strength, source_id (from Layer 2.2)
- benefit: functional, dimensionalized, emotional, clarity_score (from Layer 2.3)
- chunk_coherence_score (from Layer 2.4)

### Step 7: Assemble Flow Architecture Section

Compile:
- simple_segue: text, promise_echo, mechanism_reference (from Layer 3.2)
- chunk_sequence: order, sequence_logic (from Layer 3.1)
- connectors: between, text, type (from Layer 3.3)
- coherence_markers: position, text, type (from Layer 3.4)
- sin_segue: text, mechanism_callback, result_callback, offer_bridge (from Layer 3.5)

### Step 8: Assemble Validation Scores Section

Compile all validation scores into a unified scoring block:

```yaml
validation_scores:
  logical_coherence: [from 4.1]
  persuasion_flow: [from 4.2]
  claim_defensibility: [average from 2.1]
  proof_density: [from proof distribution check]
  benefit_clarity: [average from 2.3]
  conversational_flow: [from 3.4/4.2]
  anti_slop: [pass/fail from 4.3]
  anti_slop_violations: [count from 4.3]
  vault_comparison: [from 4.4]
  vault_differentiation_notes: [summary from 4.4]
  overall_weighted_average: [calculated]
```

**Overall weighted average calculation:**
```
overall = (logical_coherence × 0.20) +
          (persuasion_flow × 0.15) +
          (claim_defensibility × 0.15) +
          (proof_density × 0.10) +
          (benefit_clarity × 0.10) +
          (conversational_flow × 0.15) +
          (vault_comparison × 0.05) +
          (anti_slop_bonus × 0.10)
          # anti_slop_bonus: 10 if pass, 7 if pass_with_warnings, 0 if fail
```

### Step 10: Final Completeness Check

Run the post-processing checkpoint from STRUCTURE-AGENT.md:

```python
def final_completeness_check(package):
    checks = [
        ("thesis_single_sentence", len(package["campaign_thesis"]["statement"].split(". ")) <= 2),
        ("all_gaps_mapped", package["prospect_gap_map"]["total_gaps"] >= 5),
        ("all_chunks_complete", all(has_cpb(c) for c in package["cpb_chunks"])),
        ("all_claims_pass_opinion", all(c["claim"]["opinion_test"] == "pass" for c in package["cpb_chunks"])),
        ("all_proofs_traced", all(has_valid_proof(c) for c in package["cpb_chunks"])),
        ("access_chunk_last", package["cpb_chunks"][-1]["gap_type"] == "access" or package["cpb_chunks"][-1]["gap_question"].startswith("How can I get")),
        ("simple_segue_present", bool(package["flow_architecture"]["simple_segue"]["text"])),
        ("sin_segue_present", bool(package["flow_architecture"]["sin_segue"]["text"])),
        ("all_scores_above_threshold", package["validation_scores"]["overall_weighted_average"] >= 7.0),
        ("anti_slop_pass", package["validation_scores"]["anti_slop"] in ["pass", "pass_with_warnings"]),
        ("handoffs_populated", all(bool(h) for h in package["downstream_handoffs"].values())),
    ]

    failed = [name for name, result in checks if not result]
    return {"all_passed": len(failed) == 0, "failed_checks": failed}
```
