# Layer 1.1 — Campaign Thesis Crystallizer

## Purpose

Formulate the single-sentence campaign thesis that the entire argument must prove. The thesis is the intersection of mechanism + promise + root cause — the core belief the prospect must hold by the end of the campaign argument.

**Primary Persona:** Alex Rivera — Strategic Integration Director
**Secondary Persona:** Dr. James Liu — Research Director

## Execution Protocol

### Step 1: Identify the Three Thesis Pillars

The campaign thesis sits at the intersection of three upstream elements:

```
     ROOT CAUSE
    (The real problem)
          │
          ▼
    ┌───────────┐
    │  CAMPAIGN  │
    │   THESIS   │
    └───────────┘
     ▲           ▲
     │           │
MECHANISM    PROMISE
(The solution)  (The result)
```

Extract:
- **Root cause connection:** What does the root cause reveal that makes the mechanism necessary?
- **Mechanism connection:** How does the mechanism address the root cause in a way nothing else does?
- **Promise connection:** What result becomes possible because the mechanism addresses the root cause?

### Step 2: Draft Thesis Candidates

Generate minimum 3 thesis candidates. Each must:
- Be expressible in a single sentence (hard limit: 2 sentences max, target: 1)
- Connect all three pillars (root cause → mechanism → promise)
- Be provable through evidence and education (not mere assertion)
- Sound like a prosecutor's opening statement, not a sales pitch

**Thesis Type Classification:**
- **Mechanism Superiority:** "[Mechanism] is the most effective way to [result] because [root cause connection]"
- **Result Inevitability:** "When you address [root cause] with [mechanism], [result] becomes inevitable"
- **Root Cause Discovery:** "The real reason [problem] persists is [root cause], and [mechanism] is the only approach that addresses it"
- **Paradigm Shift:** "Everything you've been told about [topic] is wrong — [root cause truth] changes everything, and [mechanism] is why"

### Step 3: Evaluate Thesis Candidates

Score each candidate on:

| Dimension | Weight | Description |
|-----------|--------|-------------|
| Single-sentence clarity | 0.25 | Can it be stated in one clear sentence? |
| Pillar integration | 0.25 | Does it connect root cause + mechanism + promise? |
| Provability | 0.20 | Can it be proven through evidence and education? |
| Prosecutor quality | 0.15 | Does it sound like a case, not a pitch? |
| Emotional resonance | 0.15 | Does it create urgency to learn more? |

**Minimum threshold:** Weighted average ≥ 7.0/10

### Step 4: Select Winner

Choose the highest-scoring candidate. If multiple candidates score within 0.5 points, select the one with the strongest provability score (the argument must be defensible).

### Step 5: Validate Thesis Against Schwartz Stage

Cross-reference thesis against market sophistication level:
- **Stage 1-2:** Thesis can lead with promise + mechanism (market hasn't heard these claims)
- **Stage 3:** Thesis must emphasize mechanism uniqueness (market has heard similar promises)
- **Stage 4:** Thesis must lead with root cause discovery (market skeptical of mechanisms)
- **Stage 5:** Thesis must create paradigm shift (market skeptical of everything)

If thesis type doesn't align with Schwartz stage, revise or select alternate candidate.

## Quality Gates

- [ ] Thesis is expressible in a single sentence (2 sentences absolute max)
- [ ] Root cause, mechanism, and promise ALL connected in the thesis
- [ ] Thesis is provable through evidence (not mere assertion)
- [ ] Thesis sounds like a prosecutor's case, not a sales pitch
- [ ] Weighted average ≥ 7.0/10
- [ ] Thesis type aligns with Schwartz sophistication stage
- [ ] At least 3 candidates evaluated before selection
- [ ] Runner-up documented with reasoning

---

## Anti-Patterns

- **Product thesis:** "Our product is the best solution for X" — this proves a product, not a thesis
- **Feature thesis:** "This system has 7 steps that..." — this describes features, not a prosecutable thesis
- **Vague thesis:** "This approach will change your life" — unprovable, non-specific
- **Multi-thesis:** "This works because of X, and also Y is important, and Z matters too" — multiple theses dilute the argument
- **Salesperson thesis:** "You need to try this amazing breakthrough" — sounds like selling, not prosecuting
