# 08: CAMPAIGN STRUCTURE

## Format, Channels, and Element Architecture

---

## PURPOSE

Determine the structural context for the campaign. Structure decisions affect
every downstream writing skill — a VSL requires different pacing than a long-form
sales letter, and both differ from a landing page.

**Output:** `structure-package.json`
**Unlocks:** Skill 09 (Campaign Brief)

**Success Criteria:**
- Section-by-section outline maps the full VSL/sales page flow
- Each section has a defined purpose and emotional target
- Proof elements assigned to specific sections
- Format and channel decisions finalized
- Output file produced: `structure-package.json`
