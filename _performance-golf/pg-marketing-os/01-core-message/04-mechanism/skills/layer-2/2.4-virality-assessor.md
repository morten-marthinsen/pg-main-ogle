# Skill 2.4: Virality Assessor

**Version:** 1.0
**Layer:** 2 (Scorecard Optimization)
**Type:** Dimension Evaluation + Optimization
**Dimension:** 4 of 12 — VIRALITY
**Dependencies:** gate_1_report.json, naming_candidates.json, explanation_structures.json
**Output:** virality_scores.json

---

## Purpose

Evaluate and optimize each mechanism candidate on VIRALITY: Would someone want to TELL someone else about this? The best mechanisms are inherently shareable — they contain a surprising insight, a counter-intuitive truth, or a "wait, really?" moment that compels the prospect to tell others.

---

## The Question

**"Would someone want to TELL their friend about this over dinner?"**

---

## Scoring Rubric

```
1-3 (NOT SHAREABLE): Boring, expected, nothing to tell.
     - Generic health advice with a new name
     - Nothing surprising or counter-intuitive
     - Reader thinks "sure, whatever" and moves on

4-6 (MILDLY INTERESTING): Some novelty but not compelling to share.
     - Contains a fact but not a surprising one
     - Interesting but not "wait until you hear this" level
     - Would mention if topic came up, wouldn't bring it up

7-8 (SHAREABLE): Contains a genuine "did you know?" moment.
     - Surprising fact or counter-intuitive insight
     - Reader wants to seem smart by sharing this
     - "Hey, I just learned something crazy..."

9-10 (VIRAL): Compulsively shareable. Can't NOT tell someone.
     - Challenges a widely-held belief dramatically
     - Contains a "holy shit" revelation
     - Makes the sharer look brilliant for knowing this
     - Creates conversational currency
```

---

## Virality Triggers

```
WHAT MAKES SOMETHING SHAREABLE:

1. SURPRISING TRUTH — Contradicts what everyone "knows"
2. SPECIFIC NUMBER — A precise statistic that shocks
3. ENEMY NAMING — Identifies a villain people suspected but couldn't name
4. STATUS SIGNAL — Knowing this makes you look smart/informed
5. EMOTIONAL PUNCH — Makes you feel something strongly
6. EXPLANATORY POWER — Finally explains something mysterious
7. CONTRARIAN POSITION — Goes against the crowd (and is right)
```

---

## Evaluation Logic

### Step 1: Identify Shareability Elements

```
FOR each candidate:

  SCAN mechanism name + explanation for:

  SURPRISE ELEMENT:
    - Does this contradict common belief?
    - Would the audience say "really??" hearing this?
    - Is there a "plot twist" in the explanation?

  SPECIFICITY ELEMENT:
    - Is there a precise number that shocks?
    - Is there a named entity (person, institution, place)?
    - Is there a measurable claim?

  EMOTIONAL ELEMENT:
    - Does this trigger anger? (enemy naming)
    - Does this trigger hope? (new possibility)
    - Does this trigger vindication? (I was right all along)

  STATUS ELEMENT:
    - Does knowing this make someone feel informed?
    - Would sharing this make someone look smart?
    - Is this "insider knowledge"?

  COUNT shareability elements present (0-7)
  MORE elements = higher score
```

### Step 2: The Dinner Table Test

```
FOR each candidate:

  SIMULATE: Prospect is at dinner with friends.
  Topic of [their problem] comes up.

  WOULD THEY SAY:
    A) Nothing about the mechanism (score 1-3)
    B) "I read something about that" without detail (score 4-5)
    C) "Oh, I just learned about this thing called [name]..." (score 6-7)
    D) "Wait, you guys NEED to hear this — did you know that..." (score 8-9)
    E) They bring it up even when the topic DOESN'T come up (score 10)

  RECORD which level and WHY
```

### Step 3: Optimize If Below Threshold

```
IF current_score < 7:

  TECHNIQUE 1: Add Surprise
    FIND the most counter-intuitive aspect of the mechanism
    LEAD with it instead of burying it
    "What if the real reason [common belief] is actually [opposite]?"

  TECHNIQUE 2: Add Specific Shocking Number
    FIND or calculate a specific statistic:
    "97% of people have this problem and don't know it"
    "Your body does [specific action] 47,000 times per day"

  TECHNIQUE 3: Name the Enemy
    IF mechanism implies a villain:
      NAME it explicitly
      "The [industry/substance/habit] has been causing this for decades"

  TECHNIQUE 4: Create Status Signal
    FRAME mechanism as insider knowledge:
    "What [elite group] has known for years but never told the public"
    "The discovery that [prestigious institution] tried to suppress"

  TECHNIQUE 5: Strengthen Emotional Punch
    ADD consequence language:
    "And every day you don't know this, [bad thing] keeps happening"
    ADD vindication language:
    "So it was NEVER your fault — it was [enemy] all along"

  RE-SCORE after optimization
```

---

## Quality Criteria

1. Every candidate assessed on all 7 shareability triggers
2. Dinner table test produces a concrete script
3. Candidates scoring < 7 receive at least 2 optimization techniques
4. Top candidates have ≥ 3 shareability elements present
5. No candidate proceeds without at least 1 surprise element
