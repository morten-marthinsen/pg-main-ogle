# Skill 2.1: Image Strength Optimizer

**Version:** 1.0
**Layer:** 2 (Scorecard Optimization)
**Type:** Dimension Evaluation + Optimization
**Dimension:** 1 of 12 — IMAGE STRENGTH
**Dependencies:** gate_1_report.json (viable candidates), mechanism_analogies.json, explanation_structures.json
**Output:** image_strength_scores.json

---

## Purpose

Evaluate and optimize each mechanism candidate on IMAGE STRENGTH: Can the prospect PICTURE this mechanism working? The strongest mechanisms create a vivid mental movie — the prospect can see, feel, and almost experience the mechanism in action before they buy.

---

## The Question

**"Can the prospect PICTURE this mechanism working inside them / for them?"**

---

## Scoring Rubric

```
1-3 (WEAK): Abstract concept. No mental image. Reader thinks "what does this mean?"
     - "Optimizes cellular function" — can't picture this
     - "Synergistic approach" — meaningless image
     - "Holistic system" — nothing to see

4-6 (MODERATE): Vague image. Reader gets the general idea but can't see specifics.
     - "Clears toxins from your body" — general but not vivid
     - "Rebalances your system" — directionally clear, not specific
     - "Strengthens the connection" — what connection? what does it look like?

7-8 (STRONG): Clear image. Reader can picture the mechanism in action.
     - "Flips the cortisol switch from ON to OFF" — can see a switch
     - "Dissolves the plaque blocking your arteries" — can picture dissolving
     - "Rebuilds the neural pathways one connection at a time" — can see building

9-10 (EXCEPTIONAL): Vivid mental movie. Reader can FEEL it happening.
     - "Like a reset button in your brain — press it, feel the fog lift,
        watch your clarity return in seconds" — full sensory experience
     - "The compound enters your bloodstream, finds the inflamed cells,
        wraps around them like a cooling blanket" — movie-like
```

## Evaluation Logic

### Step 1: Assess Current Image Strength

```
FOR each viable candidate:

  READ explanation_structures.level_2_paragraph
  READ mechanism_analogies.primary_analogy

  EVALUATE:
    1. Is there a VISUAL VERB? (switches, dissolves, builds, resets, unlocks)
       vs. abstract verb (optimizes, enhances, supports, facilitates)

    2. Is there a TANGIBLE OBJECT? (a switch, a wall, a pathway, a filter)
       vs. abstract concept (function, balance, wellness, synergy)

    3. Are there SENSORY DETAILS? (feel the warmth, see the clarity, notice the calm)
       vs. intellectual descriptions (improved biomarkers, enhanced function)

    4. Is there TEMPORAL SEQUENCE? (first X happens, then Y, finally Z)
       vs. static description (it works by doing X)

  SCORE based on rubric above
```

### Step 2: Identify Image Weaknesses

```
IF current_score < 7:

  DIAGNOSE what's missing:

  MISSING VISUAL VERB:
    → Replace abstract verbs with concrete action verbs
    → "optimizes" → "rebuilds" / "switches" / "dissolves"
    → "supports" → "wraps around" / "shields" / "feeds"

  MISSING TANGIBLE OBJECT:
    → Add physical metaphor to abstract concept
    → "cellular function" → "the power generators inside each cell"
    → "neural pathway" → "the highway your thoughts travel on"

  MISSING SENSORY DETAILS:
    → Add what you FEEL, SEE, or NOTICE
    → "improved energy" → "feel the fog lift by 10am"
    → "reduced inflammation" → "notice the swelling shrink overnight"

  MISSING TEMPORAL SEQUENCE:
    → Add first/then/finally progression
    → "it reduces cortisol" → "first it calms the alarm signal,
       then cortisol drops, then your body shifts from panic to recovery"
```

### Step 3: Optimize If Below Threshold

```
IF current_score < 7:

  REWRITE the Level 2 explanation with:
    - Concrete visual verb replacing every abstract verb
    - Tangible object for every concept
    - At least 2 sensory details
    - Temporal sequence showing progression

  REWRITE the analogy with:
    - More vivid primary metaphor
    - Sensory language in the extended version
    - "Can you picture..." framing

  RE-SCORE the optimized version
  SET optimization_applied = true
  RECORD what_changed
```

### Step 4: Image Test

```
FOR each candidate (original or optimized):

  SIMULATE prospect reading:
    - After reading the mechanism explanation, what do they SEE?
    - Can they picture it working INSIDE THEM / FOR THEM?
    - What EMOTION does the image create?

  RECORD:
    can_picture_it: true/false
    what_they_picture: description of mental image
    emotional_response: what they feel seeing it
```

---

## Quality Criteria

1. Every candidate has a current_score
2. Candidates scoring < 7 receive optimization
3. Optimized versions score ≥ 7 (or flag as structurally weak)
4. Image test produces concrete "what they picture" description
5. No abstract verbs remain in optimized explanations
