# Skill 1.2: Naming Generator

**Version:** 1.0
**Layer:** 1 (Mechanism Ideation)
**Type:** Creative Generation
**Dependencies:** type_candidates.json, emphasis_strategy.json, vault_patterns.json, teachings_framework.json, upstream_context.json
**Output:** naming_candidates.json

---

## Purpose

Generate 15+ mechanism naming candidates using verbalized sampling for creative diversity. Names are generated across multiple naming patterns, informed by vault exemplars and evaluated against naming principles. This is a HIGH-CREATIVITY skill — verbalized sampling is MANDATORY.

---

## Verbalized Sampling Requirement

```
THIS SKILL REQUIRES DISTRIBUTION-LEVEL PROMPTING.

Standard generation produces mode-collapsed names (same patterns, same structures).
Verbalized sampling produces 1.6-2.1x more diverse candidates.

MINIMUM: 4 separate generation passes with different sampling instructions:
  Pass 1: "Generate names that feel SCIENTIFIC and PRECISE"
  Pass 2: "Generate names that feel VISCERAL and EMOTIONAL"
  Pass 3: "Generate names that feel MYSTERIOUS and INTRIGUING"
  Pass 4: "Generate names using E5 creative strategies (UNSPOKEN/TRANSSUBSTANTIATED)"

COMBINE all passes, then deduplicate and score.
```

## Generation Logic

### Step 1: Extract Naming Seeds

```
FROM upstream_context:
  ROOT CAUSE seeds:
    - Key nouns from real_root_cause
    - Action verbs implied by the reframe
    - Scientific terms related to the cause

  BIG IDEA seeds:
    - Core concept words
    - Promise-related terms
    - Emotional core language

  AUDIENCE seeds:
    - Terms the audience uses naturally
    - Technical vocabulary they respect
    - Metaphors they already use

FROM vault_patterns.naming_patterns:
  EXEMPLAR seeds:
    - Top-scoring mechanism names
    - Structural patterns (not the names themselves)
    - Power words from top-25 exemplars

FROM emphasis_strategy:
  EMPHASIS seeds:
    - If ROOT_CAUSE emphasis: name should evoke the problem/reframe
    - If SCIENCE emphasis: name should evoke the process/discovery
    - If DELIVERY emphasis: name should evoke the tangible action/innovation
```

### Step 1.5: E5 Creative Strategy Layer

```
FROM TODD BROWN E5 FRAMEWORK:

Before generating names, identify which E5 CREATIVE STRATEGY applies
to this mechanism. This shapes HOW you name it.

THREE E5 NAMING STRATEGIES:

1. ACTUAL MECHANISM NAMING:
   The mechanism IS genuinely unique. Name it for what it actually is.
   Strategy: Invent a name that captures the real innovation.
   Examples: "Muscle Confusion" (P90X), "Product Launch Formula" (Jeff Walker)
   Naming approach: Compound nouns or invented terms that describe
   the actual new thing you've created.

2. UNSPOKEN MECHANISM NAMING (Hopkins/Schlitz Principle):
   The mechanism is what everyone does, but nobody has SAID IT before.
   Strategy: Articulate the common process with a powerful name.
   By naming what's already happening, you OWN it.
   Classic example: "We wash our bottles with live steam"
   (all breweries did this — Schlitz SAID it first and owned it)
   Naming approach: Take the existing process and give it a proprietary-
   sounding name. Make the ordinary feel extraordinary by ARTICULATING it.
   - "Slow-Release Absorption" (every capsule does this, but now it's NAMED)
   - "Neural Pathway Clearing" (meditation does this, now it sounds clinical)
   - "Triple-Phase Digestion" (normal digestion, articulated as a system)

3. TRANSSUBSTANTIATED MECHANISM NAMING:
   Take an ordinary concept and rename it to feel extraordinary.
   Strategy: Give a common idea a scientific, powerful, or exotic name.
   The concept stays the same — the LANGUAGE elevates it.
   Examples: "Trip Wire" = low-ticket offer, "Erection Muscle" = PC muscle
   Naming approach: Find the ordinary version, then rename with:
   - Clinical/Latin terminology
   - Power vocabulary
   - Specificity that implies precision
   - "PC muscle exercises" → "The Kegel Protocol"
   - "Eating less sugar" → "Glycemic Reset Method"
   - "Walking daily" → "Metabolic Activation Sequence"

DETERMINE which strategy applies to THIS mechanism:
  IF product has genuine innovation → ACTUAL
  IF product does what others do but nobody has articulated it → UNSPOKEN
  IF product concept is ordinary but can be elevated linguistically → TRANSSUBSTANTIATED
  IF unclear → Generate candidates from ALL three strategies

RECORD e5_strategy for each candidate generated.
```

### Step 2: Generate Pass 1 — Scientific/Precise

```
SAMPLING INSTRUCTION: "Generate mechanism names that feel SCIENTIFIC,
PRECISE, and CREDIBLE. Think research papers, medical terminology,
biological processes. The name should make the reader think 'this is
based on real science.'"

GENERATE 8-10 candidates:
  FOR each primary + secondary mechanism type:
    APPLY clinical pattern: [Medical/Scientific Term] + [Process Noun]
    APPLY branded_format pattern: [Proprietary Name] + [System/Method/Protocol]
    APPLY invented pattern: [Latinate/Greek root combinations]
    APPLY numbered pattern: [Specific Number] + [Clinical Element]

EXAMPLES of the feel:
  - "Cortisol Switch" (clinical, scientific type)
  - "Neural Activation Protocol" (clinical + branded_format)
  - "Sirtuin Gene Activation" (clinical, precise)
```

### Step 3: Generate Pass 2 — Visceral/Emotional

```
SAMPLING INSTRUCTION: "Generate mechanism names that feel VISCERAL,
EMOTIONAL, and PHYSICAL. The name should make the reader FEEL something
in their body. Think power, movement, force, sensation."

GENERATE 8-10 candidates:
  FOR each primary + secondary mechanism type:
    APPLY experiential pattern: [Action Verb/Sensation] + [Body Term]
    APPLY metaphor pattern: [Physical Force/Movement metaphor]
    APPLY clinical pattern: [Visceral Scientific Term] + [Action Noun]

EXAMPLES of the feel:
  - "The Gut Punch Effect" (metaphor, visceral)
  - "Primal Override" (experiential, physical)
  - "The Compression Method" (experiential + branded_format)
```

### Step 4: Generate Pass 3 — Mysterious/Intriguing

```
SAMPLING INSTRUCTION: "Generate mechanism names that feel MYSTERIOUS,
SECRET, and EXCLUSIVE. The name should make the reader think 'I need
to know what this is.' Think hidden knowledge, ancient secrets,
forbidden discoveries."

GENERATE 8-10 candidates:
  FOR each primary + secondary mechanism type:
    APPLY origin pattern: [Geographic/Historical] + [Knowledge Term]
    APPLY metaphor pattern: [Mystery/Discovery metaphor]
    APPLY branded_format pattern: [Exotic Name] + [System Word]
    APPLY numbered pattern: [Ancient Number/Date] + [Revelation Term]

EXAMPLES of the feel:
  - "The Okinawan Secret" (origin, mysterious)
  - "The Forbidden Protocol" (branded_format, exclusive)
  - "The 4,000-Year Paradox" (numbered + origin)
```

### Step 5: Generate Pass 4 — E5 Strategy Names

```
SAMPLING INSTRUCTION: "Generate mechanism names using the identified
E5 creative strategy. If UNSPOKEN: articulate ordinary processes with
proprietary-sounding names. If TRANSSUBSTANTIATED: elevate common
concepts with scientific/power vocabulary. If ACTUAL: name the genuine
innovation directly."

GENERATE 6-8 candidates:

  FOR UNSPOKEN strategy:
    IDENTIFY: What does this product do that ALL products in the category do?
    NAME IT: Give that common process a proprietary, clinical, or systematic name.
    PATTERNS:
      - "[Clinical Term] + [Process Noun]" → "Bioavailability Cascade"
      - "The [Adjective] + [Common Action]" → "The Sequential Absorption Method"
      - "[Invented Term] for [Ordinary Thing]" → "Chrono-Release Technology"

  FOR TRANSSUBSTANTIATED strategy:
    IDENTIFY: What's the plain-language version of what this does?
    RENAME IT: Elevate with scientific, military, or technical vocabulary.
    PATTERNS:
      - Plain → Clinical: "stretching" → "Fascial Release Protocol"
      - Plain → Technical: "eating right" → "Macro Calibration System"
      - Plain → Powerful: "relaxation" → "Parasympathetic Override"

  FOR ACTUAL strategy:
    IDENTIFY: What's genuinely new about this mechanism?
    NAME IT: Create a term that captures the real innovation.
    PATTERNS:
      - Describe the novel element directly
      - Combine the unique components into a new term
      - Create a category name for something that didn't have one

TAG each candidate with e5_strategy: ACTUAL|UNSPOKEN|TRANSSUBSTANTIATED
```

### Step 6: Deduplicate and Quality Filter

```
COMBINE all candidates from 4 passes

DEDUPLICATE:
  - Remove exact matches
  - Remove near-matches (same root words, different order)
  - Merge variants into strongest version

QUALITY FILTER (remove if):
  - Name is generic (could apply to any product)
  - Name contains banned slop phrases
  - Name is > 5 words (too long)
  - Name has no embedded benefit or image
  - Name triggers > 2 anti-pattern flags

ANTI-PATTERNS:
  - Too vague: "The Solution", "The Method"
  - Too complex: 4+ word compound phrases
  - Unpronounceable: invented words without phonetic logic
  - Already exists: matches known competitor mechanism
  - Cliché: "Secret", "Hack", "Trick" (overused in market)
```

### Step 7: Score Remaining Candidates

```
FOR each surviving candidate:

  memorability_score (1-10):
    - Is it easy to remember after one reading?
    - Does it have phonetic appeal?
    - Is it distinct from other mechanisms?

  specificity_score (1-10):
    - Does it point to something concrete?
    - Could you explain what this IS from the name alone?
    - Does it narrow vs. broaden?

  ownability_score (1-10):
    - Could this be trademarked?
    - Is it unique to this product/offer?
    - Would it be hard for a competitor to copy?

  embedded_benefits:
    - What benefits does the NAME ITSELF imply?
    - (e.g., "Cortisol Switch" implies stress control)

  image_evoked:
    - What mental picture does the name create?
    - (e.g., "The Domino Effect" → falling dominoes)

  scientific_credibility_score (1-10): # v1.1 ADDITION
    - Does the name use real biological/scientific terminology?
    - Could this be a term in a medical textbook or research paper?
    - Does it reference actual physiological processes?
    - Would a skeptic accept this as "real science"?
    - Scoring guide:
      9-10: Sounds like established medical/scientific term
      7-8:  Uses credible scientific vocabulary
      5-6:  Neutral — neither scientific nor marketing
      3-4:  Sounds more like marketing than science
      1-2:  Obvious marketing speak, would trigger skeptic red flags

  CALCULATE composite_score:

    STANDARD (skepticism_level = low or medium):
      (memorability × 0.30) + (specificity × 0.30) +
      (ownability × 0.25) + (embedded_benefit_count × 0.15)

    HIGH SKEPTICISM MARKET (skepticism_level = high, type = scientific_validity):
      (memorability × 0.15) + (specificity × 0.20) +
      (ownability × 0.15) + (embedded_benefit_count × 0.10) +
      (scientific_credibility × 0.40)  # Heavy weight on credibility

    ADDITIONALLY, when skepticism is high:
      - Apply +0.3 boost to candidates from Pass 1 (Scientific/Precise)
      - Apply -0.2 penalty to candidates from Pass 2 (Visceral/Emotional)
      - Apply -0.3 penalty to candidates from Pass 3 (Mysterious/Intriguing)
```

### Step 8: Compile Top 5

```
RANK by composite_score
SELECT top 5 with rationale for each

ENSURE diversity:
  - Top 5 should include ≥ 3 different naming patterns
  - Top 5 should include candidates from ≥ 2 generation passes
  - If top 5 is too homogeneous, swap lowest for best from underrepresented pattern
```

---

## Quality Criteria

1. ≥ 15 candidates after dedup and quality filter
2. All 4 generation passes completed (verbalized sampling mandatory)
3. ≥ 3 naming patterns represented in top 5
4. No candidate in top 5 triggers anti-pattern flags
5. Each top-5 candidate has clear embedded benefits and image
6. Pattern distribution shows diversity (no single pattern > 50%)
7. E5 strategy identified for each candidate (actual, unspoken, or transsubstantiated)
8. Top 5 includes at least 1 candidate from a different E5 strategy than the majority
