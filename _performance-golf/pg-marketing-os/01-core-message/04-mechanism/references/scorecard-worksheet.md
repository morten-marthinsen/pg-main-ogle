## Step 4: Interpret Results

### Score Ranges

**85-100%:** Mechanism is ready. Strong foundation for campaign. Proceed.

**70-84%:** Mechanism is viable but has gaps. Address weak categories before proceeding.

**55-69%:** Mechanism needs significant work. Do not proceed until improved.

**Below 55%:** Return to mechanism development. Current mechanism unlikely to succeed.

## Step 6: Quick Fixes Reference

| Low Score In | Quick Fix Approach |
|--------------|-------------------|
| Image Strength | Add metaphor or concrete analogy. Connect to something visual. |
| Simplicity | Find the ONE thing. Strip to seven words. Remove jargon. |
| Proof | Stack proof types: studies + testimonials + demonstrations + logic |
| Virality | Add surprise element. Make counter-intuitive. Create conversation. |
| Ease of Use | Reframe language. Make mechanism sound like it does the work. |
| Differentiation | Check competitors. Rename or reposition to create distance. |
| Embedded Benefits | Revise name to imply outcomes without stating them. |
| Doomsday Scenario | Name the enemy. Define specific consequences of not using. |
| Belief Compatibility | Find overlap with existing beliefs. Use "You've always suspected..." |
| Central Thesis Cohesion | Strengthen product-mechanism connection. Make product THE vehicle. |
| Super Power | Reframe as capability given, not just result delivered. |
| "Can Ya Feel It??" | Test names out loud. Find what creates genuine excitement. |

## Step 7: Validation Tests

Before finalizing, pass these tests:

### The Competitor Test
> "Could a competitor use this exact mechanism without modification?"

□ Yes (FAIL - return to differentiation)
□ No (PASS)

### The Bar Talk Test
> "Could someone explain this to a friend in 30 seconds?"

□ No (FAIL - return to simplicity)
□ Yes (PASS)

### The Belief Test
> "After hearing this, would prospects believe the promise?"

□ No (FAIL - strengthen proof and explanation)
□ Yes (PASS)

### The "So What" Test
> "Does the prospect care about this mechanism?"

□ No (FAIL - reconnect to their desires/pains)
□ Yes (PASS)
