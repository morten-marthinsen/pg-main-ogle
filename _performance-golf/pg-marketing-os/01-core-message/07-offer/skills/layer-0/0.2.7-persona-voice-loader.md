### Selection Priority for This Skill

Since this skill uses `strategic` arena mode, prioritize **teaching and philosophy specimens** that show each writer's THINKING process, not just their selling voice.

For offer specifically, prioritize specimens that demonstrate how each writer **constructs offers and builds value perception**. The offer skill needs to design the complete commercial structure — what the prospect buys, how it's priced, what bonuses enhance it, what guarantees eliminate risk, and how value is demonstrated. The best specimens for this skill show:

1. **Offer construction** — How does the writer build a value stack that makes the price feel like a steal?
2. **Bonus design** — How does the writer create bonuses that address specific objections?
3. **Guarantee branding** — How does the writer turn risk reversal into a feature, not a policy?
4. **Price psychology** — How does the writer use anchoring, comparison, and value demonstration to frame the price?

Bencivenga's 100-Seminar Letter is essential here — it is a 46-page masterclass in high-ticket offer architecture with qualification selling, the "7 Master Secrets" structure, and 35+ testimonials integrated into the offer. Makepeace's controls show multi-buy cascades and value stacking where each component is individually valued before being combined at a discount. Ogilvy's ads show premium positioning techniques where the offer itself signals quality.
