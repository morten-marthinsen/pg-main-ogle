# 07: OFFER STACK DESIGN

## Make Saying Yes the Only Rational Choice

---

## PURPOSE

Architect the complete offer that transforms the Big Idea into a purchasable
package. The offer stack determines conversion — everything upstream creates
desire; the offer converts it.

**Output:** `offer-package.json`
**Unlocks:** Skill 08 (Structure)

**Success Criteria:**
- Complete offer architecture defined (core + bonuses + guarantee)
- Each component has clear value articulation backed by proof elements from the proof inventory
- Price-to-value ratio is compelling at target price point
- Urgency and scarcity mechanisms specified
- Output file produced: `offer-package.json`
