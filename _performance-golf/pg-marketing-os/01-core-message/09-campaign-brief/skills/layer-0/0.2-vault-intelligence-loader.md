## Execution Steps

### Step 1: Scan Extraction Index

Load available TIER1 extraction files. Prioritize extractions with:
- `metadata.quality_score` >= 85
- `metadata.extraction_confidence` >= 85
- Niche alignment with current project (when available)

### Step 2: Extract Campaign Coherence Patterns

From each qualifying extraction, identify:
- **Element alignment patterns** — how root cause, mechanism, promise, and Big Idea connect in elite controls
- **Coherence indicators** — linguistic and structural markers of tight campaign coherence
- **Incoherence patterns** — what disconnection looks like in weaker promotions (anti-examples)
- **Threading techniques** — how elite controls maintain narrative continuity across sections

### Step 3: Extract Lead Type Correlations

Build frequency maps:
- **Lead type by mechanism type** — which lead types pair with which mechanism types in elite controls
- **Lead type by awareness level** — which lead types match which Schwartz awareness stages
- **Lead type by Big Idea angle** — which lead types serve which Big Idea framings
- **Lead type success patterns** — which lead types appear in the highest-quality extractions

### Step 4: Extract Story Type Correlations

Build frequency maps:
- **Story type by proof asset strength** — which story types pair with which proof profiles
- **Story type by narrative need** — which story types serve blame externalization, credibility building, emotional buy-in
- **Story type by mechanism type** — which stories showcase which mechanism types
- **Story function catalog** — discovery, proof, origin, transformation, revelation, warning, vulnerability

### Step 5: Extract Emotional Arc Patterns

From the highest-converting promotions, map:
- **Opening emotions** — what prospects feel at the lead (curiosity, recognition, alarm)
- **Emotional transitions** — how emotion shifts between sections
- **Tone patterns** — dominant/secondary tones in elite controls by niche and Big Idea type
- **Pacing patterns** — emotional intensity curves (where peaks and valleys occur)

### Step 6: Specimen Tiering

```
FOR each campaign coherence specimen in loaded data:

  CALCULATE coherence_completeness_score:
    +2 IF element_alignment documented (root cause + mechanism + promise + Big Idea mapped)
    +1 IF lead_type classified with rationale
    +1 IF story_type classified with rationale
    +1 IF emotional_arc traced across sections
    +1 IF tone_pattern documented
    +1 IF threading_techniques present with 2+ entries
    +1 IF quality_score >= 85 (from extraction metadata)
    MAX_SCORE: 8

  SORT specimens by coherence_completeness_score DESC
  ASSIGN tier:
    Gold = top 10% (coherence_completeness_score >= 7)
    Silver = next 15% (coherence_completeness_score >= 5)
    Standard = remainder
```

### Step 7: Build Vault Intelligence Object

Assemble all extracted patterns into a structured intelligence object.

```yaml
vault_intelligence:
  coherence_patterns:
    - source: string
      element_alignment: object
      coherence_markers: [string]
      threading_techniques: [string]
      quality_score: integer
  lead_type_correlations:
    by_mechanism_type: object
    by_awareness_level: object
    by_big_idea_angle: object
  story_type_correlations:
    by_proof_strength: object
    by_narrative_need: object
    by_mechanism_type: object
  emotional_arc_patterns:
    - source: string
      opening_emotion: string
      transition_map: object
      tone_profile: object
      pacing_curve: string
  tier_distribution:
    total_specimens: integer
    gold_count: integer
    silver_count: integer
    standard_count: integer
  loading_status: enum[complete, partial, empty]
```

## Quality Gate
