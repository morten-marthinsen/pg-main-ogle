# Layer 2.3 — Coherence Narrative Generator

**Skill:** 09 — Campaign Brief
**Microskill:** 2.3 — Coherence Narrative Generator
**Layer:** 2
**Model:** opus
**Purpose:** Generate a human-readable coherence assessment narrative that synthesizes dimension scores, weaknesses, and strengths into a clear, actionable document the human can review.

---

## Input

| Source | File | Required |
|--------|------|----------|
| 2.1-coherence-dimension-scorer | `2.1-coherence-scores.json` | Yes |
| 2.2-weakness-detector | `2.2-weakness-report.json` | Yes |
| 1.1-campaign-summary-generator | `1.1-campaign-summary.json` | Yes |

## Output

| File | Format | Min Size |
|------|--------|----------|
| `2.3-coherence-narrative.md` | Markdown | 2KB |

## Execution Steps

### Step 1: Write the Executive Summary

Open with a 3-4 sentence summary that captures:
- The overall coherence score and what it means
- The number and nature of strengths vs. weaknesses
- The proceed/revise/halt recommendation
- The single most important thing the human should know

Example structure (adapt to actual data):
```
"This campaign scores [X]/10 on overall coherence. The [strongest dimension]
connection is the foundation — [specific reason]. The primary weakness is
[weakest dimension]: [specific issue]. [Recommendation]."
```

### Step 2: Write the Dimension-by-Dimension Breakdown

For each of the 10 dimensions, write a brief paragraph:

```markdown
### Dimension [N]: [Name] — Score: [X]/10

[2-3 sentence assessment from 2.1-coherence-scores]

[If weakness flagged: "WEAKNESS FLAG: [specific issue and improvement suggestion]"]
[If strength: "STRENGTH: [specific reason this dimension works well]"]
```

Order dimensions by strategic importance:
1. Dimensions 1-4 first (strategy chain — load-bearing, 1.5x weight)
2. Dimensions 5-8 second (framing alignment — 1.0x weight)
3. Dimensions 9-10 last (creative direction — 0.75x weight)

### Step 3: Write the Weakness Analysis Section

If weaknesses exist, write a clear analysis:

```markdown
## Critical Weaknesses

[For each weakness from 2.2-weakness-report:]

### [Weakness Name]
- **Dimension(s) affected:** [which dimensions]
- **The problem:** [specific description of the disconnection]
- **Why it matters:** [downstream consequence for the campaign]
- **Cascading effects:** [if any, from cascade analysis]
- **Remediation:** [specific fix, either upstream or at brief level]
```

### Step 4: Write the Contradiction Report

If contradictions exist:

```markdown
## Contradictions Detected

### [Topic]
- **Element A** ([name]) says: [implication]
- **Element B** ([name]) says: [implication]
- **Type:** [direct/implicit/tonal]
- **Resolution:** [suggestion]
```

### Step 5: Write the Strengths Section

```markdown
## Campaign Strengths

[For each of top 3 strengths:]
- **[Strength name]:** [What's working and why]
- **Leverage opportunity:** [How to build on this in downstream writing]
```

### Step 6: Write the Recommendation

```markdown
## Recommendation

**Overall Coherence Score:** [X]/10
**Status:** [PASS / PASS WITH CAUTION / REQUIRES REVISION / HALT]

[2-3 sentences explaining the recommendation and what the human should focus on
during their review. If borderline (6.5-7.0), explicitly note: "This score is
borderline. The strategy may work but has identified weaknesses. You may approve
with awareness of these risks, or request upstream revision."]
```

### Step 7: Anti-Slop Validation

Before finalizing, scan the entire narrative for anti-slop violations:
- No vague qualifiers (many, often, most, some, several)
- No AI telltales (revolutionary, game-changing, unlock, harness)
- No corporate filler (comprehensive, robust, innovative)
- No hedge words (might, could potentially, should consider)
- No generic brief language ("write compelling copy about...")

If violations found, replace with specific, concrete language.

## Quality Gate

- [ ] Executive summary is 3-4 sentences, captures the essential picture
- [ ] All 10 dimensions covered in the breakdown with specific assessments
- [ ] Weaknesses analyzed with specific problems, consequences, and remediations
- [ ] Contradictions reported if any exist
- [ ] Top 3 strengths identified with leverage opportunities
- [ ] Clear recommendation issued with coherence score
- [ ] Borderline scores (6.5-7.0) explicitly flagged
- [ ] Anti-slop validation passed — zero violations in final narrative
- [ ] Narrative reads as specific to THIS campaign (not generic template)

---

## Anti-Patterns

- **Template narrative:** "The campaign shows strong coherence across most dimensions" — REJECT. The narrative must be specific to the actual scores and elements.
- **Buried weaknesses:** Mentioning weaknesses in passing within a positive-sounding narrative — weaknesses must be prominent and clear
- **Anti-slop violations:** Any use of forbidden language in the coherence narrative — auto-reject and rewrite

## Handoff

Pass `coherence_narrative` to:

- **4.3-human-review-interface.md** (narrative presented to human for review)
