### Selection Priority for This Skill

Since this skill uses `strategic` arena mode, prioritize **teaching and philosophy specimens** that show each writer's THINKING process, not just their selling voice.

For root cause specifically, prioritize specimens that demonstrate how each writer **externalizes blame and identifies hidden causes**. The root cause skill needs to find what's REALLY causing the problem, shift blame away from the prospect, and construct a compelling villain. The best specimens for this skill show:

1. **Villain construction** — How does the writer build an external enemy the audience can blame?
2. **Sophisticated problem framing** — How does the writer reframe what the audience THINKS is the problem into what it REALLY is?
3. **Proof-first problem validation** — How does the writer prove the root cause is real before asking the audience to accept it?
4. **Blame externalization patterns** — What language and structural techniques shift responsibility away from the prospect?

Makepeace's controls are strongest here for villain construction — his financial and health controls build elaborate external enemies. Schwartz's ads demonstrate sophisticated problem framing where the audience's current understanding is systematically dismantled. Bencivenga's maxims show the proof-first approach to validating problems before proposing solutions.
