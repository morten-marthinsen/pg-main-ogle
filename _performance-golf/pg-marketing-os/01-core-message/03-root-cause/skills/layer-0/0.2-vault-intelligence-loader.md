## Loading Protocol

### Step 1: Load V3 Vault Intelligence File

```
LOCATE: vault-intelligence/root-cause-vault-intelligence.json

VALIDATE:
  - File exists and is valid JSON
  - Contains "version": "v3"
  - Contains "specimens" section with non-empty content
  - Contains "patterns" section with frequency distributions

IF file missing or invalid:
  SET status = FAIL
  LOG "V3 vault-intelligence build must run first"
  HALT
```

### Step 2: Extract Specimens

```
FROM specimens section:
  LOAD all verbatim content specimens
  INDEX by category/type for downstream lookup
  VERIFY verbatim_quote/text fields are present (not summaries)
```

### Step 3: Extract Patterns

```
FROM patterns section:
  LOAD all frequency distributions
  IDENTIFY top-frequency categories for defaults
  CALCULATE niche-specific pattern weightings
```

### Step 4: Extract Statistics

```
FROM statistics section:
  LOAD aggregate metrics
  SET quality baselines for validation gates
```

### Step 5: Specimen Tiering

```
FOR each specimen in loaded data:

  CALCULATE reframe_depth_score:
    +2 IF what_they_think present AND specific (not generic)
    +2 IF real_problem present AND specific (not generic)
    +1 IF why_nothing_worked present and articulated
    +1 IF reframe_technique identified (named technique)
    +1 IF emphasis_level tagged
    +1 IF full reframe arc complete (all three parts present)
    MAX_SCORE: 8

  SORT specimens by reframe_depth_score DESC
  ASSIGN tier:
    Gold = top 10% (reframe_depth_score >= 7)
    Silver = next 15% (reframe_depth_score >= 5)
    Standard = remainder

  TAG each specimen with:
    tier: enum[gold, silver, standard]
    reframe_depth_score: integer (0-8)

  OUTPUT tier_distribution:
    total_specimens: integer
    gold_count: integer
    silver_count: integer
    standard_count: integer
```

### Step 6: Type-Matched Retrieval

```
DEFINE match_fields:
  primary: reframe_technique (e.g., paradigm_shift, blame_redirect, hidden_cause)
  secondary: niche (e.g., health, finance, survival, self-improvement)
  tertiary: emphasis_level (e.g., high, medium, low)

WHEN campaign_context is available:
  STRATEGY 1 — exact_match:
    FILTER specimens WHERE reframe_technique == target AND niche == target_niche
    PRIORITIZE Gold tier first, then Silver

  STRATEGY 2 — technique_match:
    FILTER specimens WHERE reframe_technique == target (any niche)
    PRIORITIZE Gold tier, same-niche Silver, then cross-niche Silver

  STRATEGY 3 — cross_technique:
    RETURN all Gold specimens across reframe techniques
    INCLUDE Silver specimens from target niche

WHEN campaign_context is NOT available:
  RETURN full specimen set with tier metadata attached
  SORT by reframe_depth_score DESC

OUTPUT retrieval_result:
  matched_specimens: [specimen]
  match_strategy_used: enum[exact_match, technique_match, cross_technique, full_set]
  gold_specimens_available: integer
  silver_specimens_available: integer
```
