# 4.4 Output Synthesis

## Purpose
Compile ALL root cause skill outputs into the FINAL DELIVERABLE PACKAGE. This is the complete output of the Root Cause Skill—everything needed for copy integration and downstream skill handoff.

---

## Input Requirements

From Layer 3:
- Validation Synthesis Package

From Layer 4:
- 4.1 Copy Block Package
- 4.2 Downstream Handoff Package
- 4.3 Integration Guidance Document

---

## The Final Package

The Root Cause Skill produces a single comprehensive output that contains:

1. **Core Root Cause** - The validated, expressed root cause
2. **Copy Blocks** - Formatted versions for direct use
3. **Validation Summary** - Quality scores and confidence
4. **Downstream Handoffs** - Packages for mechanism, promise, big idea
5. **Integration Guidance** - How to use in different formats
6. **Metadata** - Tracking and reference information
