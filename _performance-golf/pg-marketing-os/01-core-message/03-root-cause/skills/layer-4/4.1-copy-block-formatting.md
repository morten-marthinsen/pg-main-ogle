# 4.1 Copy Block Formatting

## Purpose
Format the validated root cause into COPY-READY BLOCKS that can be directly integrated into different copy formats (VSL, sales letter, email, landing page). This microskill transforms the raw expression into polished, deployable copy elements.

---

## The Formatting Principle

**Raw Expression → Formatted Copy Blocks**

The validated expression needs to be formatted for:
1. Different copy formats (VSL vs. sales letter vs. email)
2. Different placement points (lead, body, close)
3. Different length requirements (full vs. compressed)

---

## Execution Protocol

### Step 1: Review the Validated Expression

Pull from Layer 3 Validation Synthesis:

```
VALIDATED ROOT CAUSE:
├── Core Statement: "[1-2 sentences]"
├── What They Think: "[False understanding]"
├── What's Real: "[Expressed root cause]"
├── Why Nothing Worked: "[Failure explanation]"
└── Full Copy Block: "[Complete expression]"
```

---

### Step 2: Create the FULL REVELATION Block

The complete root cause reveal for use in body copy:

```
FULL REVELATION BLOCK (200-300 words):

[SETUP - Acknowledge their experience]
[2-3 sentences validating what they've tried/experienced]

[TRANSITION - The turn]
[1 sentence signaling the reveal: "But here's what's really happening..."]

[REVELATION - The root cause]
[3-5 sentences explaining the actual root cause]

[MECHANISM - How it works]
[2-4 sentences showing how root cause creates symptoms]

[FAILURE EXPLANATION - Why nothing worked]
[2-3 sentences connecting to failed solutions]

[HOPE BRIDGE - Implying solution]
[1-2 sentences creating receptivity to mechanism]
```

---

### Step 3: Create the COMPRESSED Block

Shorter version for emails, landing pages, or space-constrained formats:

```
COMPRESSED BLOCK (75-100 words):

[ONE SENTENCE setup]
[THE ROOT CAUSE in 1-2 sentences]
[WHY IT MATTERS in 1-2 sentences]
[HOPE BRIDGE in 1 sentence]
```

**Example**:
```
You've tried everything—the diets, the exercise, the willpower.
But here's what almost no one tells you: your hormones have been
working against you since puberty. They've literally hardwired your body
to store fat, and no amount of dieting can override that programming.
That's why nothing has worked. The good news? Once you know how to
reset your hormones, everything changes.
```

---

### Step 4: Create the HEADLINE/SUBHEAD Block

Root cause distilled to headline-length:

```
HEADLINE VERSIONS:

Main Headline Option 1:
"[X words or fewer—the core insight as hook]"

Main Headline Option 2:
"[Alternative angle]"

Subheadline:
"[Expands on headline—15-20 words]"

Deck Copy (if applicable):
"[2-3 sentences bridging headline to body]"
```

**Example**:
```
Headline: "There's Only ONE Reason Your Diet Keeps Failing"
Subheadline: "It's Not Willpower—It's Your Hormones Working Against You Since Puberty"
Deck: "Everything you've been told about weight loss is backwards.
The real problem isn't what you eat—it's what your body does with it."
```

---

### Step 5: Create the BULLET POINT Block

Root cause as scannable bullet points:

```
BULLET POINT VERSION:

• [The core reframe in bullet form]
• [Why they've struggled]
• [What's really going on]
• [Why nothing worked]
• [The hope/solution hint]
```

**Example**:
```
• You've been told weight loss is about calories in vs. calories out
• But your hormones have been programmed for fat storage since puberty
• Three specific hormones override every diet you try
• That's why willpower NEVER works long-term
• The solution isn't trying harder—it's resetting the hormones
```

---

### Step 6: Create the VSL SCRIPT Block

Root cause formatted for spoken/video delivery:

```
VSL SCRIPT BLOCK:

[BEAT 1 - SETUP]
"[Conversational acknowledgment of their struggle]"
[PAUSE]

[BEAT 2 - THE TURN]
"But here's what I discovered..."
[PAUSE]

[BEAT 3 - REVELATION]
"[Root cause stated simply, with pauses for emphasis]"
[PAUSE]

[BEAT 4 - MECHANISM]
"See, what happens is..."
"[Explain mechanism conversationally]"
[PAUSE]

[BEAT 5 - FAILURE EXPLANATION]
"That's why [solution] didn't work..."
"And why [solution] didn't work either..."
[PAUSE]

[BEAT 6 - HOPE]
"But here's the good news..."
```

---

### Step 7: Create the EMAIL TEASER Block

Root cause as curiosity-building teaser:

```
EMAIL TEASER BLOCK:

Subject Line Option 1: "[Curiosity hook on root cause]"
Subject Line Option 2: "[Pattern interrupt on root cause]"

Preview Text: "[First line that expands curiosity]"

Opening Hook:
"[1-2 sentences that tease the root cause without fully revealing]"

Transition to Click:
"[Creates desire to learn the full root cause on landing page]"
```

**Example**:
```
Subject: The real reason diets fail (it's not what you think)
Preview: Your willpower isn't the problem...

If you've ever started a diet with total commitment...
only to find yourself right back where you started...

There's a reason for that.

And it has nothing to do with discipline, calories, or "trying harder."

I'll explain everything here: [LINK]
```

---

## Quality Gates

Before proceeding:

- [ ] Full revelation block is 200-300 words
- [ ] Compressed block is 75-100 words
- [ ] At least 2 headline options provided
- [ ] VSL script has natural spoken rhythm
- [ ] Email teaser creates genuine curiosity
- [ ] All blocks maintain the same core message

---

## Formatting Best Practices

| Format | Key Requirement |
|--------|----------------|
| Full Revelation | Complete logic, all three parts covered |
| Compressed | Essence only, no redundancy |
| Headlines | Curiosity or benefit, not explanation |
| Bullets | Scannable, parallel structure |
| VSL Script | Natural speech rhythm, pauses marked |
| Email Teaser | Curiosity gap that demands click |
