# 1.5 — Mechanism Constraint Check

**Microskill:** 1.5-mechanism-constraint-check
**Input:** Root cause candidates from 1.1-1.4 + PG1 product architecture
**Purpose:** Verify root cause candidates can be solved by PG1's mechanism

---

## PG1's Mechanism (from Brief + Keynote)

PG1 Pulse delivers:
1. **SwingScan AI** — Trained on 2M+ swings, 96% accuracy, identifies 77+ swing flaw patterns. Diagnoses the golfer's single root swing flaw.
2. **Six Elite PGA Coaches** — Expertise built INTO the AI plus direct access via Coach Chat. Human coaching layer on top of AI diagnosis.
3. **Adaptive Improvement Plan** — Sequenced, personalized plan that evolves based on progress. Not a generic program — adapts to the individual.
4. **Intelligence Layer** — Gets smarter the more the golfer uses it (Netflix/Spotify personalization model applied to golf).

## Constraint Check: Does the mechanism solve the root cause?

### Root Cause Candidate: "The golf improvement industry operates on treatment-without-diagnosis"

| Mechanism Component | Solves Root Cause? | How |
|--------------------|--------------------|-----|
| SwingScan AI (diagnosis) | **YES — DIRECTLY** | This IS the diagnosis. 2M+ swings, 96% accuracy, 77+ flaw patterns. The golfer's root flaw is identified with precision no human instructor can match. |
| Elite Coaches (treatment) | YES — COMPLETES | After diagnosis, the coaches provide the RIGHT treatment. Not random treatment — treatment prescribed FOR the diagnosed flaw. |
| Adaptive Plan (sequencing) | YES — SUSTAINS | The plan sequences improvement in the right order based on the diagnosis. Not "work on everything" — work on THE thing first, then cascade. |
| Intelligence Layer (learning) | YES — COMPOUNDS | As the golfer practices, the system learns whether the root flaw is resolving and adjusts. The diagnosis stays alive and evolving. |

**Mechanism alignment: PERFECT (10/10)**

The root cause is "nobody diagnoses first." PG1's mechanism STARTS with diagnosis. The alignment is not incidental — the product was literally built to fill this gap.

### Alternative Root Cause: "Too much conflicting information"
| Mechanism Component | Solves? | Assessment |
|--------------------|--------------------|-----|
| SwingScan AI | Partially — reduces confusion by providing ONE answer | Indirect |
| Elite Coaches | Partially — one trusted source vs. many | Indirect |
| Adaptive Plan | Yes — replaces chaos with structure | Direct |
| Intelligence Layer | Partially — learns preferences | Indirect |

**Mechanism alignment: MODERATE (6/10).** The mechanism helps but doesn't directly solve "information overload" — it bypasses it. This is a weaker root cause candidate because the mechanism connection is indirect.

### Alternative Root Cause: "Golfers internalize blame for systemic failure"
| Mechanism Component | Solves? | Assessment |
|--------------------|--------------------|-----|
| SwingScan AI | Partially — externalizes the problem by identifying a FLAW, not a FAILURE | Indirect |
| Elite Coaches | Partially — human validation | Indirect |
| Adaptive Plan | No — doesn't address self-blame directly | No |

**Mechanism alignment: WEAK (4/10).** Self-blame is a symptom (Layer 4), not the root cause. The mechanism doesn't solve self-blame directly — the FRAMING does. But the mechanism solves the structural problem (absent diagnosis) that CREATED the self-blame.

---

## Constraint Verdict

**"Treatment-without-diagnosis" root cause passes mechanism constraint at 10/10.** PG1's entire architecture — from SwingScan AI diagnosis through adaptive coaching plan — is built to solve this exact problem. The mechanism-root cause alignment is the strongest possible: the product IS the diagnosis the market never provided.

The alternative candidates ("information overload," "self-blame") are symptoms that the primary root cause explains. They fail mechanism constraint as standalone root causes but function as emotional/narrative layers in the expression phase.
