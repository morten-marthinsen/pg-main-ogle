# 1.4 Mechanism Connection

## Purpose
Connect each promise to the mechanism. The promise tells WHAT they'll achieve; the mechanism tells HOW they'll achieve it. This connection is critical for the Campaign Thesis and for Stage 3+ markets where mechanism IS the differentiator.

**Key Formula:** "The [superiority] way to [promise] is with [mechanism]"

---

## Connection Patterns

```
MECHANISM CONNECTION PATTERNS:

PATTERN 1: Promise + Mechanism
"[Promise] with the [Mechanism]"
Example: "End your cravings with the Gut Reset Protocol"

PATTERN 2: Mechanism + Promise
"The [Mechanism] [delivers/unlocks/enables] [promise]"
Example: "The Gut Reset Protocol ends your cravings in 7 days"

PATTERN 3: Superiority + Promise + Mechanism (Campaign Thesis)
"The [only/best/fastest] way to [promise] is with [mechanism]"
Example: "The ONLY way to end your cravings is with the Gut Reset Protocol"

PATTERN 4: Villain + Promise + Mechanism
"[Defeat villain] to [promise] with [mechanism]"
Example: "Defeat the Bad Bugs to end your cravings with the Gut Reset Protocol"

PATTERN 5: Question + Promise + Mechanism
"What if the [superiority] way to [promise] was simply [mechanism action]?"
Example: "What if the only way to end cravings was to feed the right bacteria?"
```

## Execution Protocol

### Step 1: Verify Mechanism Alignment

```python
def verify_mechanism_alignment(promise, mechanism):
    """
    Check if the mechanism can actually deliver the promise.
    """
    mechanism_capabilities = mechanism.get('capabilities', {})
    direct_outcomes = mechanism_capabilities.get('direct_outcomes', [])
    indirect_outcomes = mechanism_capabilities.get('indirect_outcomes', [])

    promise_outcome = extract_outcome(promise)

    alignment = {
        'promise': promise,
        'mechanism': mechanism['name'],
        'is_direct': promise_outcome in direct_outcomes,
        'is_indirect': promise_outcome in indirect_outcomes,
        'alignment_type': None,
        'bridge_needed': False
    }

    if alignment['is_direct']:
        alignment['alignment_type'] = 'direct'
        alignment['connection_strength'] = 'strong'
    elif alignment['is_indirect']:
        alignment['alignment_type'] = 'indirect'
        alignment['connection_strength'] = 'moderate'
        alignment['bridge_needed'] = True
        alignment['bridge_explanation'] = explain_indirect_connection(
            promise_outcome, mechanism
        )
    else:
        alignment['alignment_type'] = 'none'
        alignment['connection_strength'] = 'weak'
        alignment['warning'] = "Promise may not be deliverable by mechanism"

    return alignment
```

### Step 2: Generate Connected Versions

```python
def generate_mechanism_connections(promise, mechanism, schwartz_stage):
    """
    Generate promise versions with mechanism connection.
    """
    connected = []

    # PATTERN 1: Promise + Mechanism (simple append)
    connected.append({
        'pattern': 'promise_plus_mechanism',
        'statement': f"{promise} with the {mechanism['name']}",
        'mechanism_emphasis': 'light',
        'best_for_stage': [1, 2, 3]
    })

    # PATTERN 2: Mechanism + Promise (mechanism-led)
    connected.append({
        'pattern': 'mechanism_plus_promise',
        'statement': f"The {mechanism['name']} {mechanism['what_it_does']} so you can {promise}",
        'mechanism_emphasis': 'medium',
        'best_for_stage': [3, 4]
    })

    # PATTERN 3: Campaign Thesis (superiority)
    for superiority in ['only', 'best', 'fastest', 'easiest']:
        connected.append({
            'pattern': 'campaign_thesis',
            'statement': f"The {superiority.upper()} way to {promise} is with the {mechanism['name']}",
            'mechanism_emphasis': 'heavy',
            'superiority_claim': superiority,
            'best_for_stage': [3, 4, 5]
        })

    # PATTERN 4: Villain + Promise + Mechanism
    if mechanism.get('villain'):
        villain_name = mechanism['villain']['name']
        defeat_action = mechanism.get('villain_defeat_action', f"defeat {villain_name}")

        connected.append({
            'pattern': 'villain_inclusive',
            'statement': f"{defeat_action.capitalize()} to finally {promise}",
            'mechanism_emphasis': 'implied',
            'villain_present': True,
            'best_for_stage': [3, 4]
        })

        connected.append({
            'pattern': 'villain_thesis',
            'statement': f"The only way to {promise} is to {defeat_action} — and the {mechanism['name']} does exactly that",
            'mechanism_emphasis': 'heavy',
            'villain_present': True,
            'best_for_stage': [4, 5]
        })

    # PATTERN 5: Question format
    connected.append({
        'pattern': 'question_thesis',
        'statement': f"What if the only way to {promise} was simply {mechanism['simplified_action']}?",
        'mechanism_emphasis': 'curiosity',
        'best_for_stage': [3, 4, 5]
    })

    return connected
```

### Step 3: Stage-Appropriate Selection

```python
def select_for_stage(connected_promises, schwartz_stage):
    """
    Filter and rank connected promises for the market's Schwartz stage.
    """
    stage_appropriate = []

    for connected in connected_promises:
        best_stages = connected.get('best_for_stage', [])

        if schwartz_stage in best_stages:
            # Perfect fit
            connected['stage_fit'] = 'optimal'
            connected['stage_fit_score'] = 10
        elif schwartz_stage in [s-1 for s in best_stages] or schwartz_stage in [s+1 for s in best_stages]:
            # Adjacent fit
            connected['stage_fit'] = 'acceptable'
            connected['stage_fit_score'] = 7
        else:
            # Poor fit
            connected['stage_fit'] = 'poor'
            connected['stage_fit_score'] = 3

        stage_appropriate.append(connected)

    # Sort by stage fit
    return sorted(stage_appropriate, key=lambda x: x['stage_fit_score'], reverse=True)
```

## Campaign Thesis Generation

The Campaign Thesis is the core positioning statement for the entire campaign:

```yaml
campaign_thesis_formula:
  formula: "The [superiority] way to [promise] is with [mechanism]"

  superiority_options:
    exclusive: ["only", "single", "sole"]
    comparative: ["fastest", "easiest", "most effective", "best"]
    novel: ["new", "breakthrough", "revolutionary"]
    scientific: ["scientifically-proven", "research-backed"]

  stage_guidance:
    stage_1_2:
      superiority: "best" or "fastest"
      mechanism_emphasis: light
      example: "The fastest way to lose weight is with XYZ"

    stage_3:
      superiority: "only" or "best"
      mechanism_emphasis: medium-heavy
      example: "The only way to end cravings is with the Gut Reset Protocol"

    stage_4:
      superiority: "only" (required)
      mechanism_emphasis: heavy + unique angle
      example: "The ONLY way to end cravings is to defeat the Bad Bugs"

    stage_5:
      superiority: context-dependent
      mechanism_emphasis: identity-wrapped
      example: "For anyone who's tried everything... the only way that works"
```
