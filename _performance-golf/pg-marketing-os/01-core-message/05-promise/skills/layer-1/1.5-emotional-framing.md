# 1.5 Emotional Framing

## Purpose
Wrap technical or functional promises in emotional language that resonates with the prospect's psychological state. The promise must connect to how they FEEL, not just what they'll GET.

---

## Input Requirements

From 1.1-1.4:
- Promise candidates (blue sky, specific, typed, mechanism-connected)

From 01-research:
- market_psychology (hope, fear, frustration, exhaustion levels)
- customer_language (how they describe success, feelings)
- emotional_drivers

From 03-root-cause:
- villain_name (for fear/frustration framing)

---

## Emotional Frame Categories

```
EMOTIONAL FRAMES:

HOPE Frame
"Finally... it's possible"
Best for: High-hope markets, Stage 1-2
Trigger: When prospects believe transformation is possible

FRUSTRATION RELIEF Frame
"After years of trying... finally"
Best for: High-frustration markets, burned markets
Trigger: When prospects have tried and failed many times

FEAR ESCAPE Frame
"Before it's too late..."
Best for: High-fear markets, health/prevention niches
Trigger: When prospects fear negative consequences

VINDICATION Frame
"Prove them wrong..."
Best for: Identity-driven markets, competitive niches
Trigger: When prospects want to show others they can succeed

FREEDOM Frame
"Finally be free from..."
Best for: Constraint-aware markets, relief-seeking
Trigger: When prospects feel trapped by their situation

CONTROL Frame
"Take back control of..."
Best for: Overwhelmed markets, health/money niches
Trigger: When prospects feel out of control
```

## Execution Protocol

### Step 1: Match Frame to Market Psychology

```python
def select_emotional_frame(market_psychology):
    """
    Select primary and secondary emotional frames based on market psychology.
    """
    frames = {
        'hope': market_psychology.get('hope_level', 5),
        'frustration': market_psychology.get('frustration_level', 5),
        'fear': market_psychology.get('fear_level', 5),
        'exhaustion': market_psychology.get('exhaustion_level', 5),
        'control_desire': market_psychology.get('control_desire', 5)
    }

    # Determine primary frame
    if frames['hope'] >= 8:
        primary = 'hope'
    elif frames['frustration'] >= 8:
        primary = 'frustration_relief'
    elif frames['fear'] >= 8:
        primary = 'fear_escape'
    elif frames['exhaustion'] >= 8:
        primary = 'frustration_relief'
    elif frames['control_desire'] >= 8:
        primary = 'control'
    else:
        primary = 'hope'  # Default

    # Determine secondary frame
    sorted_frames = sorted(frames.items(), key=lambda x: x[1], reverse=True)
    secondary = sorted_frames[1][0] if sorted_frames[1][1] >= 6 else None

    return {
        'primary_frame': primary,
        'secondary_frame': secondary,
        'frame_scores': frames
    }
```

### Step 3: Generate Emotionally-Framed Versions

```python
def apply_emotional_framing(promise, frame, templates, context):
    """
    Apply emotional framing to a promise.
    """
    framed_versions = []

    # Apply full templates
    for template in templates['full_templates']:
        framed = template.format(
            promise=promise,
            fear=context.get('fear', 'failure'),
            negative=context.get('negative', 'the struggle'),
            constraint=context.get('constraint', 'limitations'),
            villain=context.get('villain', 'the problem'),
            pain=context.get('pain', 'the frustration'),
            problem=context.get('problem', 'the issue'),
            topic=context.get('topic', 'your life'),
            result=context.get('result', 'results')
        )

        framed_versions.append({
            'original_promise': promise,
            'framed_statement': framed,
            'frame_type': frame,
            'template_used': template
        })

    # Apply opener + promise + closer combinations
    for opener in templates['openers'][:2]:
        for closer in templates['closers'][:2]:
            framed = f"{opener} {promise} {closer}"
            framed_versions.append({
                'original_promise': promise,
                'framed_statement': framed,
                'frame_type': frame,
                'template_used': f"{opener} + {closer}"
            })

    return framed_versions
```

## Emotional Intensity Calibration

```yaml
emotional_intensity_levels:

  high_intensity:
    when_to_use:
      - Breakthrough moments in copy
      - Headlines and hooks
      - Close and CTA
    markers:
      - Exclamation language ("Finally!")
      - Emphatic words ("never again", "once and for all")
      - All caps for emphasis ("FINALLY")
    example: "FINALLY End Your Cravings — Once and For All!"

  medium_intensity:
    when_to_use:
      - Body copy promises
      - Supporting points
      - Benefit bullets
    markers:
      - Standard emotional framing
      - Moderate emphasis
    example: "Finally end your cravings and take back control"

  low_intensity:
    when_to_use:
      - Technical explanations
      - Proof sections
      - Scientific framing
    markers:
      - Subtle emotional undertones
      - Factual but resonant
    example: "End the cravings that have controlled your eating"
```
