# 0.3 — Specimen Proof Loader

**Skill:** LP-05-social-proof-architecture
**Position:** Layer 0, Microskill 3
**Purpose:** Load proof architecture patterns from specimens matching this project's vertical and page type. Extract real-world proof density patterns, placement strategies, and format choices that inform Layer 1 and Layer 2 decisions.
**Effort Level:** medium
**Dependencies:** `brief-and-sequence-loaded.md` (0.1 — for page type and vertical)
**Output:** `specimen-proof-patterns.md`

---

## Input Schema

```yaml
input:
  page_type: "[type_a | type_b | hybrid]"
  vertical: "[from page-brief.json vertical.primary]"
  specimen_index_path: "04-page-builder/specimens/specimen-index.md"
  cross_page_analysis_path: "04-page-builder/specimens/cross-page-pattern-analysis.md"
```

---

## Execution Logic

**Step 1:** Read specimen-index.md. Filter specimens by:
- Primary match: same page type AND same vertical
- Secondary match: same page type, different vertical (proof patterns transfer across verticals)
- Tertiary match: same vertical, different page type (vertical-specific proof norms)

**Step 2:** Read cross-page-pattern-analysis.md, Section 2 (Proof Architecture Patterns). Extract:
- Proof density distribution (ZERO / LOW / MEDIUM / HIGH / EXTREME by specimen)
- First proof position (above fold / section 2-3 / section 4-5 / none on page)
- Proof type frequency (which types appear across specimens)
- Bimodal proof distribution finding (proof-saturated vs. proof-absent)

**Step 3:** For each matching specimen, note:
- Proof density level
- Where first proof appears
- What proof types are used
- How proof is formatted (widget, cascade, video, text, etc.)
- Any unique proof patterns worth replicating

**Step 4:** Identify the specimen closest to this project's profile. Flag it as the primary reference for LP-05.

**Step 5:** Extract the key strategic finding: Is this project a PROOF SATURATION candidate or an AUTHORITY SUBSTITUTION candidate based on specimen patterns for this vertical?

## Quality Criteria
- At least 1 matching specimen identified (primary, secondary, or tertiary)
- Cross-page pattern analysis findings incorporated
- Strategic recommendation aligns with bimodal proof distribution finding
- Primary reference specimen selected with rationale

## Failure Modes
- No specimens match vertical -> Use cross-page patterns only; note: "No vertical-matched specimens. Architecture based on cross-page analysis + page type norms."
- Specimen files not readable -> Skip that specimen, continue with remaining matches
