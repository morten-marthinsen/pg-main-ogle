## Loading Instructions

### Selection Priority for This Skill

Since this skill generates FEATURE NAMES (not full copy), prioritize specimens that demonstrate each writer's approach to **naming mechanisms, labeling product features, and creating proprietary terminology**. Feature naming is compressed persuasion — every word in a 2-4 word feature name carries maximum weight.

The key specimen signal for feature naming: look for how each writer NAMES things — what they call their mechanisms, how they label product components, and how they create proprietary-sounding terms from common ingredients or technologies. Makepeace's controls name mechanisms with dramatic specificity. Schwartz's ads name processes that make benefits concrete. Hopkins-style preemptive naming (naming a common process first) is visible across all specimen libraries.

## E-Comm Feature Naming Specimen Focus

When loading specimens for feature naming Arena, instruct each persona to pay special attention to:

1. **How the writer names mechanisms** — What proprietary labels do they create?
2. **How the writer describes ingredients/components** — Do they use technical names, benefit names, or hybrid?
3. **How the writer creates perceived exclusivity** — What makes their named features feel OWNABLE?
4. **Word economy** — Feature names are 2-4 words. How does the writer compress meaning into minimal words?
