# 3.3 — Anti-Slop Check

**Skill:** LP-03-above-fold-architecture
**Position:** Layer 3, Microskill 3
**Purpose:** Scan all Layer 2 outputs for forbidden AI-generated language patterns. A single instance of generic AI telltale language is a FAIL. This gate prevents downstream writing skills (LP-07) from inheriting slop in their creative briefs.
**Effort Level:** low
**Dependencies:** All Layer 2 outputs (2.1 through 2.6)
**Output:** `anti-slop-check.md`

---

## Input Schema

```yaml
input:
  layer_2_outputs:
    - "[path to headline-type.md]"
    - "[path to deck-direction.md]"
    - "[path to hero-image-brief.md]"
    - "[path to trust-architecture.md]"
    - "[path to cta-direction.md]"
    - "[path to price-display.md]"  # if Type B
```

---

## Execution Logic

**Step 1:** Read all Layer 2 output files into a single scan corpus.

**Step 2:** Scan for the FORBIDDEN WORD LIST. These are AI-generated telltale words and phrases that signal generic, undifferentiated output:

**Tier 1 — Instant FAIL (single instance):**
- "revolutionary"
- "game-changing" / "game-changer"
- "transform your" / "transformative"
- "unlock your" / "unlock the"
- "unleash"
- "harness the power"
- "cutting-edge"
- "groundbreaking"
- "elevate your"
- "empower" / "empowering"
- "seamless" / "seamlessly"
- "leverage" (as a verb meaning "use")
- "paradigm shift"
- "synergy"
- "holistic approach"

**Tier 2 — Flagged (2+ instances across outputs = FAIL):**
- "journey" (when meaning "experience" or "process")
- "dive deep" / "deep dive"
- "landscape" (when meaning "market" or "space")
- "robust"
- "comprehensive"
- "innovative"
- "state-of-the-art"
- "best-in-class"
- "world-class"
- "next-level"
- "premium quality"
- "curated"
- "bespoke"

**Step 3:** For each match found, document:
- The forbidden word/phrase
- Which file it appears in
- The full sentence containing it
- The tier (1 or 2)

**Step 4:** Determine result:
- Zero Tier 1 matches AND fewer than 2 Tier 2 matches = PASS
- Any Tier 1 match = FAIL
- 2+ Tier 2 matches across all outputs = FAIL

**Step 5:** For each FAIL, write a specific replacement recommendation:
- Original: "This revolutionary supplement transforms your health"
- Fix: Remove the adjective, make the claim specific: "This supplement reduced joint pain in 87% of clinical trial participants"

**Step 6:** Write `anti-slop-check.md`.

## Quality Criteria
- All Layer 2 files scanned (not just a subset)
- Both Tier 1 and Tier 2 word lists checked completely
- Each match includes the full sentence for context (word might be used legitimately in rare cases)
- Replacement recommendations are specific and concrete (not "use a better word")
- PASS requires zero Tier 1 and fewer than 2 Tier 2 matches
- File exists after execution

## Failure Modes
- Layer 2 outputs not all present → List which files were scanned and which were missing; scan available files
- Word appears in a legitimate context (e.g., "journey" in a direct audience quote) → Note as legitimate use, do not count as match
- Replacement recommendations are equally generic → REJECT the recommendation, rewrite with specific language
- Zero matches found in a large corpus → Confirm scan was thorough; verify word list was fully checked
