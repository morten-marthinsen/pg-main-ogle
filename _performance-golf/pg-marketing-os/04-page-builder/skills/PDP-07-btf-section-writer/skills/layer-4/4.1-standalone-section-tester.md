# 4.1-standalone-section-tester.md

> **Layer:** 4 (Validation)
> **Skill:** PDP-07-section-copy
> **Purpose:** Shuffle test -- each section standalone?
> **Outputs:** `standalone_test_result` object with PASS/FAIL per section

---

## FUNCTION

Execute the shuffle test: randomize the order of all BTF sections and verify that each section makes complete sense when read independently, in any order, without context from any other section.

---

## TESTING PROTOCOL

### Step 1: Shuffle Test

```
RANDOMIZE section order
FOR each section (in random order):
  READ section in isolation
  ASK:
    Q1: "Do I understand what product this section is about?" [Y/N]
    Q2: "Do I understand the specific claim/benefit?" [Y/N]
    Q3: "Does the section reference any other section?" [Y/N -- N is correct]
    Q4: "Could I read ONLY this section and decide to buy?" [Y/N]

  SCORE:
    Q1=Y, Q2=Y, Q3=N, Q4=Y -> FULL STANDALONE (4/4)
    Q1=Y, Q2=Y, Q3=N, Q4=N -> STRONG STANDALONE (3/4) -- acceptable
    Q1=Y, Q2=N, Q3=N, Q4=N -> PARTIAL STANDALONE (2/4) -- needs work
    Any Q3=Y -> DEPENDENCY DETECTED -- must fix

  MINIMUM THRESHOLD:
    Every section must achieve >= 3/4
    Q3 must be N for ALL sections (zero cross-references)
```

### Step 2: Cross-Reference Detection

```
SCAN all section copy for cross-reference phrases:
  FORBIDDEN PHRASES:
    "as mentioned above"
    "as we discussed"
    "see the [X] section"
    "building on that"
    "as you'll see below"
    "earlier we showed"
    "remember when"
    "going back to"
    "in the previous section"
    "this connects to"
    "furthermore" (when bridging to prior content)

  IF any found: FAIL -- remove and rewrite with built-in context
```

### Step 3: Context Sufficiency Check

```
FOR each section:
  CHECK first sentence:
    [ ] Establishes WHAT the section is about
    [ ] Establishes WHY the reader should care
    [ ] Does NOT assume prior reading

  GOOD first sentence: "Back pain costs Americans $100 billion annually -- and most treatments target the symptom, not the cause."
  BAD first sentence: "Now that you understand the problem, let's look at how [Product] addresses it."
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Any section has cross-reference | Remove reference, add built-in context |
| CRITICAL | Any section scores 2/4 or below | Rewrite section opening with context |
| WARNING | Section scores 3/4 | Strengthen context establishment |
| INFO | All sections 4/4 | Best case -- proceed |
