## FAILURE MODES

- Execution log written before all steps complete -> This should be the LAST file written. Verify all other outputs exist first.
- Missing scrape attempt details -> Reconstruct from scrape-[N].md files (they contain scrape metadata).
- Gate result not recorded -> Re-check each gate condition against existing files and record the result.
