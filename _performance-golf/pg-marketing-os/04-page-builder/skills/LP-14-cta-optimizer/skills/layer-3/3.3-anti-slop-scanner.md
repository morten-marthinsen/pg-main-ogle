# 3.3 — Anti-Slop Scanner

> Version: 1.0
> Layer: 3
> Model: sonnet
> Input: CTA_SELECTION.yaml, final-close-draft.md, ps-copy-draft.md, micro-cta-candidates.md
> Output: anti-slop-scan.md

**Skill:** LP-14-cta-optimizer
**Position:** Layer 3, Microskill 3
**Purpose:** Systematically scan every CTA copy element against the 5-category forbidden word list from LP-14-AGENT.md. A single instance of any forbidden word or phrase = FAIL. This is the quality gate that prevents AI telltales and fabricated urgency from reaching the final package.
**Effort Level:** low
**Dependencies:** `CTA_SELECTION.yaml`, selected P.S. copy, selected final close, selected micro-CTAs
**Output:** `anti-slop-scan.md`

---

## Input Schema

```yaml
input:
  elements_to_scan:
    primary_cta_button: "[text from CTA_SELECTION.yaml]"
    primary_cta_context: "[text or: None]"
    secondary_cta_buttons: "[all secondary button texts]"
    secondary_cta_contexts: "[all secondary context texts]"
    ps_copy: "[all P.S. lines]"
    final_close: "[full final close paragraph]"
    micro_ctas: "[all selected micro-CTA texts]"
```

## Execution Logic

1. **Load the 5-category forbidden word list** from LP-14-AGENT.md:

   **Category 1 -- AI Telltales:**
   revolutionary, game-changing, groundbreaking, cutting-edge, breakthrough, innovative, state-of-the-art, next-level, unprecedented, transformative, unlock your potential, harness the power, leverage, synergy, holistic approach

   **Category 2 -- Vague CTA Language:**
   click here, learn more (as button text), submit, continue, discover the secret, find out how, see more, read more (as primary CTA)

   **Category 3 -- Fabricated Urgency:**
   act now (without justified deadline), limited time (without LP-13 urgency data), don't miss out (without specific consequence), hurry (without deadline), last chance (without verified scarcity from LP-13)

   **Category 4 -- Unearned Claims:**
   the best, the most effective, the #1 (without sourced data), proven (without specifying what was proven), guaranteed results (without specific guarantee terms from LP-06)

   **Category 5 -- Passive/Hedge Language:**
   may help, could potentially, might support, has been known to, try it and see, give it a shot, why not try

2. **Scan each element independently.** For each element:
   - Check every word and phrase against all 5 categories
   - Record ANY match with: the forbidden word/phrase, the element it was found in, and the category

3. **Special urgency check:** For Category 3, cross-reference with urgency permission:
   - If urgency is PERMITTED (LP-13 data exists): Category 3 words are allowed ONLY if they reference specific LP-13 data (actual deadline, actual quantity)
   - If urgency is FORBIDDEN: ALL Category 3 words are banned regardless of context

4. **Check for near-misses:**
   - "Unlock" (any usage)
   - "Empower" / "empowering"
   - "Journey" (metaphorical)
   - "Seamless" / "seamlessly"
   - "Robust"
   - "Elevate" / "elevated"
   - "Transform" / "transformation" (in generic sense)

5. **Determine scan result:**
   - Zero forbidden words AND zero near-misses -> **PASS**
   - Zero forbidden words but near-misses found -> **PASS WITH FLAG**
   - One or more forbidden words found -> **FAIL**

6. **If FAIL:** For each forbidden word found, provide a specific replacement suggestion.

## Quality Criteria
- Every CTA copy element scanned -- no elements skipped
- All 5 categories checked explicitly (each listed in output even if "None found")
- Near-miss check conducted
- Category 3 cross-referenced with urgency permission
- FAIL result includes specific replacement suggestions for every instance
- Scan result is binary: any forbidden word = FAIL, no exceptions

## Failure Modes
- Scanning only button texts and skipping context/P.S./final close -> All elements must be scanned. P.S. and final close are the most likely locations for AI telltales.
- Marking a Category 3 word as acceptable without LP-13 data -> "Limited Time" is FORBIDDEN without LP-13 justification. No exceptions.
- Missing compound phrases -> "unlock your potential" must be caught as a unit, not missed because "unlock" and "potential" pass individually.
- Near-miss found but not flagged -> Near-misses should be noted even on PASS. Human may want to revise.
- Accepting "Learn More" as primary CTA button text -> It is on the Category 2 forbidden list for primary CTAs. It can only be used for above-fold secondary CTA on cold traffic pages.
