# 1.3 — CTA Strategy Planner

**Skill:** LP-06-offer-cta-architecture
**Position:** Layer 1, Microskill 3
**Purpose:** Set the CTA strategy for the page — placement count (minimum 3), page zone assignments, and emotional appeal category selections. This is the strategic PLAN that Layer 2 microskills (2.3 CTA Placement Mapper and 2.5 CTA Language Direction) execute in detail.
**Effort Level:** medium
**Dependencies:** `brief-loaded.md` (0.1 — CTA sections from LP-04, awareness stage, traffic temperature), `offer-type-classification.md` (1.1), `specimen-offers-loaded.md` (0.3)
**Output:** `cta-strategy.md`

## CTA Count Guidelines

| Page Type | Minimum | Recommended | Maximum | Notes |
|-----------|---------|-------------|---------|-------|
| Type A (Long-Form) | 3 | 5-8 | 10 | More sections = more natural CTA positions |
| Type B (Ecomm/PDP) | 3 | 3-5 | 6 | Above-fold ATC + mid-page + sticky mobile |
| Hybrid | 3 | 4-6 | 8 | Type B above fold + Type A below fold |

## CTA Page Zone Model

Every CTA sits in one of 5 page zones:

| Zone | Position | Visitor State | Best Emotional Appeals |
|------|----------|--------------|----------------------|
| **Above-Fold** | First screen | Impulse/ready-to-buy | Desire/Confidence, Specificity |
| **Early** | Top 25% of page | Interested but uneducated | Transformation, Belonging |
| **Mid-Page** | 25-60% of page | Educated, building belief | Risk Reversal, Belonging |
| **Post-Proof** | 60-80% of page | Convinced, trust established | Desire/Confidence, Specificity |
| **Close** | Bottom 20% | Final decision moment | Consequence, Risk Reversal, Specificity |

---

## Execution Logic

**Step 1:** Determine CTA count based on page type and sections. Start with LP-04 CTA placements (if available). Add additional placements if below minimum or if page length warrants more.

**Step 2:** Map each planned CTA to a page zone. Every page needs representation in at least 3 zones.

**Step 3:** For each CTA position, select 2-3 candidate emotional appeal categories from the 6-appeal framework:
- Desire/Confidence — ownership, certainty
- Transformation — aspiration, identity shift
- Consequence — fear of missing out, loss aversion
- Belonging — social validation, community
- Risk Reversal — safety, zero risk
- Specificity — precision, deal, savings

**Step 4:** Determine CTA text consistency strategy:
- **Consistent text** (same CTA wording throughout) — simpler, reinforces single message. Works for: trial offers, simple products, high-awareness audiences.
- **Varying text** (different wording at each placement) — matches emotional state at that scroll position. Works for: long-form pages, complex offers, multi-stage persuasion.

**Step 5:** Determine inline risk reversal strategy:
- Which CTAs include guarantee reference beneath/beside the button?
- Which CTAs include "cancel anytime" or "free trial" reassurance?

**Step 6:** Determine sticky mobile CTA requirements:
- What elements appear in the sticky bar (product name, price, rating, button)?
- When does the sticky bar activate (after initial CTA scrolls out of view)?

## Quality Criteria
- CTA count >= 3
- At least 3 page zones covered
- Each CTA has 2-3 candidate emotional appeal categories
- Text strategy (consistent vs. varying) declared with rationale
- Sticky mobile CTA addressed (enabled or explicitly disabled with reason)
- Awareness stage and traffic temperature alignment documented

## Failure Modes
- LP-04 section-sequence.json missing CTA placements -> Use default zone-based placements: above-fold + mid-page + close. Note that specific section numbers will be assigned in 2.3.
- Awareness stage = 1 (Unaware) with Type B page -> Conflict detected. Document: "Above-fold buy CTA may underperform for unaware audience. Consider delaying first buy CTA or adding educational content above fold." Flag for human review.
- Offer type = OTO (One-Time Offer) -> All CTAs use Consequence appeal. Time-limited framing across every placement. Note this overrides normal multi-appeal strategy.
