# Microskill 4.5: Distinctiveness Pass (Page Builder)

**Layer:** 4 (Fix Application)
**Purpose:** Detect whether conversion optimization has drifted landing page copy toward generic template language — structurally sound but voiceless
**Sequence:** Runs AFTER audit compilation (4.1), summary (4.2), and log writing (4.3)

---

## WHY THIS PASS EXISTS

Page builder skills optimize for conversion architecture, mobile responsiveness, and scan optimization — all pushing copy toward proven-template language that works for any product. This pass checks whether the campaign's distinctive voice persists across page sections.

**References:**
- `~system/humanization-specimens/SIGNATURE-MOVES.md` (Page Builder section)
- `~system/humanization-specimens/engine-packs/page-builder-pack.md`
- Campaign Soul.md

  convergence_indicators:
