# 2.4 — Badge Copy Writer

**Skill:** LP-08-trust-elements
**Position:** Layer 2, Microskill 4
**Purpose:** Write the text that appears ON and UNDER trust badges. Badges are visual trust signals — the copy is 3–7 words that must communicate ONE trust message per badge at a glance. This microskill also writes optional tooltip expansions (1 sentence) that appear on hover or below the badge.
**Effort Level:** high (when applicable)
**Dependencies:** `trust-classification.md` (1.1), `trust-placement-map.md` (1.3), `trust-inventory.md` (0.2), `certification-copy.md` (2.3 — if certifications exist, badge labels may reference them)
**Output:** `badge-copy.md`

---

## Input Schema

```yaml
input:
  badge_status: "[from trust-classification.md — included | not_applicable]"
  trust_inventory: "[from trust-inventory.md — available badge-worthy signals]"
  placement_map: "[from trust-placement-map.md — where badges appear]"
  certification_copy: "[from certification-copy.md — badge labels if certifications were written]"
  page_type: "[type_a | type_b | hybrid]"
  product_name: "[from packages-loaded.md]"
```

---

## Execution Logic

### If `not_applicable`:
Write `badge-copy.md` with:
```
Status: not_applicable
Reason: No badge-worthy trust signals in inventory for this page type.
```
Terminate this microskill.

### If `included`:

1. **Identify all badge-worthy signals from the trust inventory.** Badge candidates include:
   - Guarantee (type + duration)
   - Certifications (badge label form from 2.3)
   - Shipping policy
   - Security/payment
   - Formula attributes (Non-GMO, Gluten-Free, Vegan, etc.)
   - Quality testing (Third-Party Tested, Lab Verified)

2. **Determine badge count:**
   - Type A: 2–3 badges (near guarantee section or CTA)
   - Type B: 3–5 badges (below ATC, above fold — critical placement)
   - Hybrid: 3–4 badges

3. **Write badge copy for each selected badge:**

   **Badge text (ON the badge):** 3–7 words maximum
   - Must communicate ONE trust signal
   - Must be scannable at 0.5-second glance
   - Use icon-friendly language (short nouns + qualifiers)

   **Tooltip text (BELOW the badge or on hover):** 1 sentence maximum
   - Expands on the badge text for visitors who want more detail
   - Optional — some badges are self-explanatory ("Free Shipping")

4. **Apply badge-type writing rules:**

   | Badge Type | Badge Text Example | Tooltip Example |
   |-----------|-------------------|----------------|
   | Guarantee | "365-Day Money Back" | "Try it for a full year. If you're not satisfied for any reason, we'll refund every penny." |
   | Manufacturing | "GMP Certified Facility" | "Manufactured in a facility meeting pharmaceutical-grade standards." |
   | Testing | "Third-Party Lab Tested" | "Every batch independently tested for purity and potency." |
   | Shipping | "Free Shipping" | "Free shipping on all US orders. International shipping available." |
   | Security | "Secure Checkout" | "Your payment is protected by 256-bit SSL encryption." |
   | Formula | "Non-GMO · Gluten Free" | "Made with clean, non-GMO ingredients. No gluten, soy, or artificial additives." |
   | Results | "Clinically Studied Ingredients" | "Key ingredients backed by peer-reviewed clinical research." |

5. **Verify no badge text exceeds 7 words.** Count each badge.

6. **Verify each badge communicates exactly ONE trust signal.** A badge that says "GMP Certified Free Shipping Guaranteed" is trying to do three things at once and fails at all of them.

7. **If certification badge labels were written in 2.3, reference them** — do not rewrite certification badges from scratch. Use the badge label already created.

8. **Write `badge-copy.md`.**

## Quality Criteria
- Badge text is 3–7 words per badge (hard constraint)
- Each badge communicates exactly 1 trust signal (not bundled signals)
- Tooltip text adds genuine value (not just restating the badge)
- Badge count is appropriate for page type (2–3 Type A, 3–5 Type B)
- Certification badges reference 2.3 output (no redundant rewrites)
- Every badge traceable to trust inventory data

## Failure Modes
- Badge text over 7 words → Too wordy for visual badge. Trim or split into 2 badges.
- Badge combining multiple signals → "Free Shipping + Money-Back Guarantee" is 2 signals, not 1. Split them.
- Tooltip that just restates the badge → "Free Shipping" badge with tooltip "We offer free shipping" adds nothing. Either add value or mark as self-explanatory.
- Inventing certifications or guarantees → Every badge must source from trust inventory.
- Type B page with fewer than 3 badges near ATC → Critical placement requires minimum 3 badges for Type B.
