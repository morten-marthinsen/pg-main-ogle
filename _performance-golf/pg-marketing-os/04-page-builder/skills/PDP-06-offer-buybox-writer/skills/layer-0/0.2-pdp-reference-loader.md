# 0.2 — PDP Reference Loader

**Skill:** PDP-06-offer-buybox-writer
**Position:** Layer 0, Microskill 2
**Purpose:** Load the buy box section of PDP-BEST-PRACTICES-REFERENCE.md. Extract Baymard/NN-g copy standards for pricing display, ATC labels, subscription tiles, micro-trust signals, and savings formatting that govern every PDP-06 decision.
**Effort Level:** low
**Dependencies:** 0.1 input-verification.md (to confirm page type and blueprint status)
**Output:** `pdp-reference-load.md`

---

## Input Schema

```yaml
input:
  input_verification: "[path to input-verification.md]"
  pdp_reference: "[path to reference/PDP-BEST-PRACTICES-REFERENCE.md]"
  pdp_engine_claude: "[path to PDP-ENGINE.md]"
```

---

## Execution Logic

**Step 1:** Verify `input-verification.md` exists and GATE_0 prerequisites are on track. If not -> HALT.

**Step 2:** Read `PDP-BEST-PRACTICES-REFERENCE.md`. Extract the buy box copy standards section.

**Step 3:** Extract ATC Button Copy Standards:
- Standard label: "Add to Cart"
- All state labels: loading, success, out-of-stock, disabled
- Full-width requirement
- Post-click behavior: slide-out mini-cart (not cart redirect)
- Sticky ATC bar requirements

**Step 4:** Extract Pricing Display Standards:
- Anchor price (struck-through original)
- Sale price (bold/accent)
- Savings in both % and $ amount
- Per-unit pricing for multi-packs
- Shipping proximity to price

**Step 5:** Extract Subscription Tile Standards:
- OTP tile copy template
- Subscribe & Save tile copy template
- "Cancel anytime" requirement
- Frequency selector copy
- Explicit savings inside tile
- Neither tile pre-selected

**Step 6:** Extract Micro-Trust Signal Standards:
- 3-4 signals maximum
- Placement: below ATC
- Format: icon + short factual statement
- Forbidden: marketing language, generic claims
- Common signals by vertical (security, shipping, guarantee, certification)

**Step 7:** Extract Variant Selector Copy Standards:
- Exposed chip labels (not dropdowns)
- Stock status indicators
- Price differential inline display
- Per-variant pricing rules

**Step 8:** Extract Mini-Cart Copy Standards:
- Slide-out cart elements
- Cross-sell suggestion placement
- "Continue Shopping" + "Checkout" actions
- Product name + variant + quantity + price display

**Step 9:** If PDP-BEST-PRACTICES-REFERENCE.md is not found, fall back to PDP-06-AGENT.md standards tables (Buy Box Copy Standards section). Document the fallback.

**Step 10:** Write `pdp-reference-load.md`.
