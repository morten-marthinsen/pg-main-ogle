# 2.3 — Micro-Trust Signal Copy

**Skill:** PDP-06-offer-buybox-writer
**Position:** Layer 2, Microskill 3
**Purpose:** Write 3-4 micro-trust signal lines that appear in small gray text below the ATC button. Each signal resolves a specific last-second purchase anxiety: security, shipping, returns, or product legitimacy. Exactly 3-4 signals -- not a wall of badges, not just 1-2.
**Effort Level:** medium
**Dependencies:** 0.1 input-verification.md (shipping, guarantee, certifications), 0.3 blueprint-extraction.md (micro-trust spec from PDP-03), 0.2 pdp-reference-load.md (micro-trust standards)
**Output:** `micro-trust-copy.md`

---

## Input Schema

```yaml
input:
  input_verification: "[path to input-verification.md]"
  blueprint_extraction: "[path to blueprint-extraction.md]"
  pdp_reference_load: "[path to pdp-reference-load.md]"
```

---

## Execution Logic

**Step 1:** Read `input-verification.md`. Extract trust-relevant product data:
- Shipping policy (free, threshold, paid)
- Guarantee type and duration
- Certifications (GMP, Third-Party Tested, NSF, USDA Organic, etc.)
- Manufacturing location (if "Made in USA" or similar)
- Return policy details

**Step 2:** Read `blueprint-extraction.md`. Extract PDP-03's micro-trust signal spec:
- Number of signals PDP-03 specified (must be 3-4)
- Signal types PDP-03 suggested
- Placement confirmation (below ATC)

**Step 3:** Read `pdp-reference-load.md`. Confirm micro-trust standards:
- 3-4 signals max
- Small gray text, icon + factual statement
- No marketing language
- Specific and factual

**Step 4:** Select 3-4 signals from available product data, prioritized by anxiety-resolution power:

```
Priority hierarchy (select top 3-4):
  1. Security:    "Secure Checkout" (always include -- universal anxiety)
  2. Shipping:    "Free Shipping" or "Free Shipping on Orders $X+"
  3. Returns:     "[X]-Day Money-Back Guarantee" (specific duration)
  4. Certification: "Third-Party Tested" or "GMP Certified" or "NSF Approved"
  5. Origin:      "Made in USA" or "Manufactured in [Country]"
  6. Support:     "24/7 Customer Support" or "Live Chat Support"
```

**Step 5:** Write each signal as icon + short factual statement:
```
Format: [icon] [factual statement]
Examples:
  🔒 Secure Checkout
  🚚 Free Shipping on Orders $50+
  ↩️  30-Day Money-Back Guarantee
  ✓  Third-Party Tested for Purity
```
- Each signal is ONE short line (3-7 words)
- No marketing language ("Premium Quality", "Trusted by Millions")
- No vague claims ("Satisfaction Guaranteed" without duration)
- Every claim must be verifiable

**Step 6:** Validate each signal:
- Is it factual? (Can the company back this claim?)
- Is it specific? (Not "Money-Back Guarantee" but "30-Day Money-Back Guarantee")
- Is it concise? (Under 7 words)
- Is it relevant to last-second purchase anxiety?

**Step 7:** Ensure no overlap with express payment security note (from 2.2). If 2.2 includes "All transactions are secure and encrypted", adjust micro-trust to avoid duplication.

**Step 8:** Write `micro-trust-copy.md`.

## Quality Criteria
- Exactly 3-4 signals (not fewer, not more)
- Every signal is a specific, verifiable factual statement
- No generic claims ("Premium Quality", "Best in Class", "Satisfaction Guaranteed" without specifics)
- No marketing language or superlatives
- Each signal resolves a distinct purchase anxiety (no two signals addressing the same anxiety)
- Concise: 3-7 words each
- No duplication with express payment section (2.2)
- File exists after execution

## Failure Modes
- Only 1-2 trust data points available from brief -> Use "Secure Checkout" (always applicable) + available data to reach 3 minimum; flag remaining gaps for human
- Brief has 6+ certifications -> Select top 3-4 by anxiety-resolution power; do NOT include all
- Signal text gets marketing-heavy ("Your Satisfaction is Our #1 Priority") -> Rewrite as factual statement ("30-Day Money-Back Guarantee")
- Shipping policy unclear -> Use "Shipping calculated at checkout" only as last resort; flag for human clarification
- Guarantee duration missing -> Do NOT write "Money-Back Guarantee" without duration. HALT and request duration from human.
