# 4.2-output-packager.md

> **Layer:** 4 (Validation & Packaging)
> **Skill:** LP-15-ecomm-assembly
> **Purpose:** Package ecomm-copy-assembled.md + page-builder-handoff.yaml
> **Outputs:** Two files in project output directory

---

## FUNCTION

Package the final assembled page copy and page builder handoff into their output files.

---

## PACKAGING PROTOCOL

### Step 1: Verify Validation
```
CHECK validation_result.overall_status: MUST be PASS
IF FAIL: HALT
```

### Step 2: Write ecomm-copy-assembled.md
```
BUILD markdown document:
  # [Product Name] -- E-Commerce Copy (Assembled)

  _Generated: [timestamp] | Skill: LP-15 | Total words: [count]_

  FOR each section (in priority order):
    ## [Section Name]
    [Full section copy]

    **Design Notes:** [layout, mobile, visuals]
    **Video:** [micro-script reference if applicable]

    ---
```

### Step 3: Verify Both Files Written
```
CHECK:
  [ ] ecomm-copy-assembled.md written
  [ ] page-builder-handoff.yaml written
  [ ] Both files at expected path
  [ ] Both files non-empty
  [ ] Markdown properly formatted
  [ ] YAML properly formatted
```
