# 1.3-design-note-integrator.md

> **Layer:** 1 (Assembly)
> **Skill:** LP-15-ecomm-assembly
> **Purpose:** Integrate UX/design notes per section
> **Outputs:** Updated assembled page with design notes per section

---

## FUNCTION

Integrate the design notes from PDP-07 (section-level) and PDP-01 (strategy-level seeds) into the assembled page document. Every section must have actionable design notes for the page builder.

---

## INTEGRATION PROTOCOL

### Step 1: Merge Design Note Sources
```
FOR each section:
  LOAD design notes from:
    1. PDP-07 section-copy-package.json (section-level design notes)
    2. PDP-01 ecomm-strategy.yaml (design note seeds)
    3. PDP-03 hero-copy-draft.md (hero design notes)

  MERGE: PDP-07 notes take priority, PDP-01 seeds fill gaps
```

### Step 2: Verify Completeness
```
FOR each section:
  CHECK design notes contain:
    [ ] layout: specified (full-width, two-column, card, etc.)
    [ ] mobile_behavior: specified (stack, swipe, collapse, etc.)
    [ ] visual_elements: specified (images, icons, charts, etc.)

  IF any missing: FILL from defaults based on section type
```

### Step 3: Add Micro-Script Placement Notes
```
FOR each section with a mapped micro-script:
  ADD to design notes:
    video_placement: "Embed [script_type] video here"
    video_duration: "[X] seconds"
    video_script_id: "[MS-XXX]"
```
