# 2.1-page-builder-handoff-generator.md

> **Layer:** 2 (Handoff)
> **Skill:** LP-15-ecomm-assembly
> **Purpose:** Generate page-builder-handoff.yaml
> **Outputs:** `page-builder-handoff.yaml` file

---

## FUNCTION

Generate the page builder handoff document -- a structured YAML file that gives the page builder (04-page-builder / LP-00) everything needed to construct the page: section order, copy references, design specs, visual requirements, and micro-script placements.

---

## GENERATION PROTOCOL

### Step 1: Build Section Specs
```
FOR each section in assembled page:
  BUILD spec:
    section_number: [N]
    section_name: [name]
    priority: [from PDP-01]
    copy_source: [PDP-03 or PDP-07]
    copy_word_count: [actual]
    layout: [from design notes]
    mobile_behavior: [from design notes]
    visual_elements: [from design notes]
    video_element: [micro-script reference if applicable]
    proof_element: [proof type and content]
    notes: [any special handling instructions]
```

### Step 2: Build Page-Level Specs
```
ASK the user for the page URL slug before generating the YAML.
  Base URL (constant, do NOT include in YAML): pg1.performancegolf.com/pages/
  The user provides the slug that follows the final slash.
  Example: user says "rs1-media-product-sc" → page_url: "rs1-media-product-sc"

BUILD page_spec:
  page_type: [pdp|collection|bundle]
  page_url: [slug from user — REQUIRED]
  total_sections: [count]
  total_word_count: [sum]
  estimated_scroll_depth: [based on content volume]
  mobile_priority_sections: [top 5 sections for mobile]
  cta_placements: [section numbers with CTAs]
  video_placements: [section numbers with videos]
```

### Step 3: Generate YAML
```
WRITE page-builder-handoff.yaml:
  page_spec at top
  section_specs in priority order
  appendix with feature name registry (for consistency)
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Section specs incomplete | Fill missing specs before writing |
| WARNING | Design notes sparse for some sections | Note as "needs designer input" |
