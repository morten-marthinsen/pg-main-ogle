# 1.2-thumbnail-story-planner.md

> **Layer:** 1 (Hero Architecture)
> **Skill:** PDP-03-hero-value-prop
> **Purpose:** Plan 10-thumbnail hero carousel narrative
> **Outputs:** `carousel_plan` object with 10-position narrative structure

---

## FUNCTION

Plan the narrative arc for the 10-thumbnail hero carousel. Each position has a specific role in the product story. The plan defines what each thumbnail communicates and what visual + copy elements are needed.

---

## PLANNING PROTOCOL

### Step 1: Select Carousel Template

```
FROM specimen_patterns.carousel_templates:
  EVALUATE against product type and available assets:

  Product Story Arc: Best for PDP with clear mechanism
  Feature Showcase: Best for tech products with multiple standout features
  Before/After Journey: Best for transformation products (health, beauty, fitness)

  SELECT template with highest fit
```

### Step 2: Assign Position Content

```
FOR each position (1-10):
  DEFINE:
    position_role: What this thumbnail accomplishes in the narrative
    copy_direction: What text overlay should communicate
    visual_direction: What the image should show
    feature_reference: Which PDP-02 feature is relevant (if any)
    word_budget: 5-15 words per thumbnail overlay

  MANDATORY POSITIONS:
    Position 1: Product hero shot + headline (ALWAYS)
    Position 10: CTA + offer summary (ALWAYS)

  RECOMMENDED STRUCTURE:
    Positions 2-3: Problem/hero feature introduction
    Positions 4-6: Key benefits + supporting features
    Position 7: Social proof element
    Positions 8-9: Trust/guarantee/use case
```

### Step 3: Verify Narrative Flow

```
READ positions 1-10 in sequence:
  [ ] Does the story progress logically?
  [ ] Does each position add NEW information?
  [ ] Is the hero feature introduced by position 3?
  [ ] Is there a proof element by position 7?
  [ ] Does position 10 create urgency to act?
  [ ] Could a user who only sees 3-4 thumbnails still get the core message?
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Fewer than 10 positions planned | Complete all positions |
| WARNING | Narrative doesn't flow logically | Reorder positions |
| WARNING | Hero feature not introduced by position 3 | Move feature introduction earlier |
| INFO | Multiple templates fit equally well | Select most common for product category |
