# 4.2-word-budget-checker.md

> **Layer:** 4 (Validation)
> **Skill:** PDP-03-hero-value-prop
> **Purpose:** Verify word budgets (headline 5-12w, subhead 10-20w, etc.)
> **Outputs:** `word_budget_result` object with PASS/FAIL per component

---

## FUNCTION

Verify that every hero component meets its word budget. Word budgets are HARD LIMITS, not guidelines. Exceeding any budget degrades scan optimization and dilutes hero impact.

---

## BUDGET CHECK PROTOCOL

```
FOR each hero component:

  HEADLINE:
    actual_words: [count]
    min: 5, max: 12
    status: PASS if 5 <= actual <= 12, else FAIL
    IF FAIL: "Headline is [X] words. Must be 5-12. [Compress/Expand]."

  SUBHEAD:
    actual_words: [count]
    min: 10, max: 20
    status: PASS if 10 <= actual <= 20, else FAIL
    IF FAIL: "Subhead is [X] words. Must be 10-20. [Compress/Expand]."

  VALUE PROP:
    actual_words: [count]
    min: 15, max: 30
    status: PASS if 15 <= actual <= 30, else FAIL
    IF FAIL: "Value prop is [X] words. Must be 15-30. [Compress/Expand]."

  CTA:
    actual_words: [count]
    min: 2, max: 5
    status: PASS if 2 <= actual <= 5, else FAIL

  HIGHLIGHTS TOTAL:
    actual_words: [sum of all highlights]
    min: 30, max: 100
    status: PASS if 30 <= actual <= 100, else FAIL

  CAROUSEL TOTAL:
    actual_words: [sum of all 10 positions]
    max: 150
    status: PASS if actual <= 150, else FAIL

  GRAND TOTAL (all hero copy):
    actual_words: [sum]
    max: 350
    status: PASS if actual <= 350, else FAIL
    NOTE: Hero should be LEAN. Under 250 is ideal.
```

---

## OUTPUT STRUCTURE

```json
{
  "word_budget_result": {
    "overall_status": "PASS|FAIL",
    "components": {
      "headline": { "actual": 8, "min": 5, "max": 12, "status": "PASS" },
      "subhead": { "actual": 15, "min": 10, "max": 20, "status": "PASS" },
      "value_prop": { "actual": 22, "min": 15, "max": 30, "status": "PASS" },
      "cta": { "actual": 3, "min": 2, "max": 5, "status": "PASS" },
      "highlights_total": { "actual": 56, "min": 30, "max": 100, "status": "PASS" },
      "carousel_total": { "actual": 85, "max": 150, "status": "PASS" }
    },
    "grand_total": { "actual": 189, "max": 350, "status": "PASS" },
    "violations": []
  }
}
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Headline exceeds 12 words | COMPRESS -- every word must earn its place |
| CRITICAL | Grand total exceeds 350 words | REDUCE -- hero is bloated |
| WARNING | Any component at exact boundary | Monitor -- slight edits could push over |
| INFO | All components well within budget | Ideal state |
