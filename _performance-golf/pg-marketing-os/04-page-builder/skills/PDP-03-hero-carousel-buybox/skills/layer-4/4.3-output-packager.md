# 4.3-output-packager.md

> **Layer:** 4 (Validation & Packaging)
> **Skill:** PDP-03-hero-value-prop
> **Purpose:** Package hero-copy-draft.md
> **Outputs:** `hero-copy-draft.md` file in project output directory

---

## FUNCTION

Package the validated hero section into `hero-copy-draft.md` -- a human-readable markdown document that contains all hero copy components with design notes. This file feeds LP-15 (Assembly) and PDP-17 (Editorial).

---

## PACKAGING PROTOCOL

### Step 1: Verify All Validations Passed
```
CHECK:
  scroll_stop_result.overall_status: MUST be PASS
  word_budget_result.overall_status: MUST be PASS
  arena_selection: MUST have human selection recorded

IF any fails: HALT
```

### Step 2: Assemble Markdown Document
```
BUILD hero-copy-draft.md:

  # Hero Section -- [Product Name]

  ## Headline
  [Selected headline text]
  _Word count: X | Approach: [approach type]_

  ## Subhead
  [Selected subhead text]
  _Word count: X_

  ## Value Proposition
  [Value prop text]
  _Word count: X_

  ## CTA
  **[CTA text]**

  ## Product Highlights TLDR
  1. **[Feature Name]** -- [highlight text]
  2. **[Feature Name]** -- [highlight text]
  ...

  ## Hero Carousel (10 Thumbnails)
  | Position | Copy | Visual Direction |
  |----------|------|-----------------|
  | 1 | [copy] | [visual] |
  ...

  ## Design Notes
  - Layout: [hero layout recommendation]
  - Mobile: [mobile behavior]
  - Image: [hero image requirements]

  ## Metadata
  - Generated: [timestamp]
  - Skill: PDP-03-hero-value-prop
  - Arena selection: [selection source]
  - Total word count: [grand total]
```

### Step 3: Write and Verify
```
WRITE hero-copy-draft.md to:
  ~outputs/[project-code]/e-comm/hero-copy-draft.md

VERIFY:
  [ ] File written successfully
  [ ] All sections present
  [ ] Word counts accurate
  [ ] Feature names match PDP-02 exactly
  [ ] No placeholder text
```

---

## FAILURE MODES

| Mode | Trigger | Remediation |
|------|---------|-------------|
| CRITICAL | Validation not passed | Return to validator, fix issues |
| CRITICAL | File write fails | Check directory and permissions |
| WARNING | Feature names don't match PDP-02 | Correct to exact PDP-02 names |
