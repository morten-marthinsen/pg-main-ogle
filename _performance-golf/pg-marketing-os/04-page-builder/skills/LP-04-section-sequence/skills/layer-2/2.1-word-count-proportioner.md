# 2.1 — Word Count Proportioner

**Skill:** LP-04-section-sequence
**Position:** Layer 2, Microskill 1
**Purpose:** Assign a specific word count target to every section based on benchmark ranges, page type, and awareness stage adjustments. Every section gets one number — not a range — as its target.
**Effort Level:** medium
**Dependencies:** `optional-decisions.md` (Layer 1), `pacing-adjustment.md` (2.5 — but use awareness stage from brief to pre-configure), `page-brief.json`
**Output:** `word-count-plan.md`

---

## Input Schema

```yaml
input:
  final_sections_list: "[from optional-decisions.md — includes all included sections]"
  page_type: "[type_a | type_b | hybrid]"
  awareness_stage: "[1-5]"
  traffic_temperature: "[cold | warm | hot]"
  page_length_preference: "[short | standard | long | not_specified]"
```

---

## Word Count Selection Logic

**Step 1: Determine page length tier**

```
IF page_length_preference specified → use that tier
ELSE:
  Cold traffic + Awareness stage 1-2 → standard to long
  Cold traffic + Awareness stage 3 → standard
  Warm traffic + Awareness stage 3-4 → short to standard
  Hot traffic + Awareness stage 4-5 → short
  If high-ticket offer ($100+) → add +25% to all targets
```

**Step 2: Load benchmark table for page type** (from AGENT.md WORD COUNT BENCHMARK TABLE)

**Step 3: For each section, select a SINGLE target from the range**
- Longer page tier → pick upper-mid of range
- Standard tier → pick midpoint of range
- Shorter tier → pick lower-mid of range
- Adjust for: ingredient count (multiply per-ingredient target × ingredient count)

**Step 4: Sum all targets to get total page word count estimate**

**Step 5: Sanity check totals**
- Type A total: must fall 4,000–18,000. If outside → adjust proportions.
- Type B total: must fall 1,500–7,000. If outside → adjust proportions.
- No single section should exceed 30% of total.

---

## Quality Criteria
- Every section has a single specific target number
- No section has only a range (ranges are inputs; targets are outputs)
- Total word count within benchmark range for page type
- No section exceeds 30% of total
- Adjustments from benchmarks are documented with rationale

## Failure Modes
- Section list is incomplete (missing mandatory sections) → HALT — this is a Layer 1 failure
- Targets produce total outside range → Redistribute proportionally until total is within range
