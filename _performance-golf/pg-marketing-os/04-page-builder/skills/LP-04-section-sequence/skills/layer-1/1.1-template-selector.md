# 1.1 — Template Selector

**Skill:** LP-04-section-sequence
**Position:** Layer 1, Microskill 1
**Purpose:** Select the correct base template (Type A, Type B, or Hybrid) based on the page type declaration from LP-00. Load the full template with all mandatory and optional sections pre-populated.
**Effort Level:** low
**Dependencies:** `page-brief.json` (LP-00 output) — specifically `page_type.classification`
**Output:** `template-selection.md`

---

## Input Schema

```yaml
input:
  page_type: "[type_a | type_b | hybrid — from page-brief.json]"
  hybrid_zone_split: "[if hybrid — zone split description from LP-00]"
  vertical: "[from page-brief.json]"
  awareness_stage: "[1-5 — from page-brief.json]"
```

---

## Execution Logic

**Step 1:** Read `page_type.classification` from `page-brief.json`. This is the ONLY input that determines template selection.

**Step 2:** Load the corresponding base template from the AGENT.md `BASE SECTION TEMPLATES` section:
- `type_a` → Load Type A Template (18 sections, 12–14 mandatory)
- `type_b` → Load Type B Template (15 sections, 10–12 mandatory)
- `hybrid` → Load Hybrid Template (zones defined by LP-00 hybrid_zone_split)

**Step 3:** Mark vertical-specific mandatory overrides:
- `health-supplements` vertical: `ingredient_deep_dives` becomes **mandatory** for both Type A and Type B
- `info-products` vertical: `story` becomes **mandatory** for Type A; `how_it_works` becomes **optional** for Type B
- `coaching-services` vertical: `comparison_table` becomes recommended for Type A
- `beauty-wellness` vertical: `before_after` becomes **mandatory** for Type B if any transformation claims exist

**Step 4:** Note awareness stage modifier for pacing (will be applied in 2.5):
- Stage 1–2: Flag for extended lead and story sections
- Stage 4–5: Flag for compressed or skipped story; earlier CTA placement

---

## Quality Criteria
- Correct template selected for declared page type
- All sections from template listed (mandatory + optional)
- Vertical overrides applied
- Awareness stage modifier noted for Layer 2

## Failure Modes
- page_brief.json missing or page_type field empty → HALT, cannot proceed
- Hybrid selected but no zone split available from LP-00 → Request zone split from user before proceeding
