# Landing Page Engine — Design Tokens Reference

**Version:** 1.0
**Date:** 2026-03-06
**Source:** Firecrawl branding extraction from S-tier specimens
**Purpose:** Give Claude concrete design DNA from real high-converting pages so generated landing pages don't default to generic "AI website" styling. Load this file during any page generation skill.

---

## How To Use This File

When generating landing page code, **do NOT use default Tailwind/generic values**. Instead:

1. Match the project's vertical to the closest specimen below
2. Use that specimen's actual design tokens (colors, fonts, spacing, button styles)
3. If building a net-new brand, blend tokens from 2-3 relevant specimens
4. The goal is pages that look **designed by a human**, not generated by AI

## Anti-Patterns (What Makes Pages Look "AI-Generated")

These are the default Claude tendencies to actively avoid:

1. **Generic blue** (`blue-600`, `indigo-600`) — Real brands use distinctive colors
2. **Uniform border-radius** (`rounded-lg` everywhere) — Real pages mix sharp and round
3. **Uniform spacing** (`py-12 px-6` on every section) — Real pages vary rhythm
4. **System fonts only** (`font-sans`) — Real brands use custom typefaces
5. **No shadows or all shadows** — Real pages use shadows sparingly and specifically
6. **Centered everything** — Real pages use asymmetric layouts, left-aligned text
7. **Every section same background** — Real pages alternate white/off-white/colored
8. **Generic button text** ("Get Started", "Learn More") — Real pages use specific CTAs
9. **No visual tension** — Real pages have one oversized element per section
10. **Cookie-cutter card grids** — Real pages vary their content presentation

## Extraction Method

All tokens extracted via Firecrawl `branding` format, which uses AI to identify:
- Actual computed CSS values from rendered pages
- Primary/secondary button styles from real interactive elements
- Font families from CSS `@font-face` declarations and computed styles
- Color values from dominant page elements

Confidence scores: 0.85-0.925 across all extractions.
