# Microskill 0.4: Input Validator

**Layer:** 0 (Foundation)
**Purpose:** Validate assembled copy is complete and ready for editorial

---

## VALIDATION CHECKLIST

### Primary Copy Validation

| Requirement | Check | Failure Action |
|-------------|-------|----------------|
| assembled-draft.md exists | `[ ]` | STOP: No copy to edit |
| Copy is not empty | `[ ]` | STOP: Empty file |
| Copy has >1000 words | `[ ]` | WARN: May be incomplete |
| Copy has identifiable sections | `[ ]` | WARN: Structure unclear |
| No placeholder text detected | `[ ]` | STOP: Copy incomplete |
| No TODO/TBD markers | `[ ]` | STOP: Copy incomplete |
| No [INSERT] markers | `[ ]` | STOP: Copy incomplete |

### Copy Quality Pre-Check

| Check | Status | Notes |
|-------|--------|-------|
| Has opening hook | `[ ]` | First 50 words grab attention |
| Has clear mechanism | `[ ]` | Mechanism identifiable |
| Has proof elements | `[ ]` | At least some proof present |
| Has offer section | `[ ]` | Clear what's being sold |
| Has close/CTA | `[ ]` | Clear call to action |
