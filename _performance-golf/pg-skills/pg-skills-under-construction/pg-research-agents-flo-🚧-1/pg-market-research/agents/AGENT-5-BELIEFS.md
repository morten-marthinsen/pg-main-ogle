## Belief Inversion Framework

### High-Potential Inversion Criteria

For an inversion to power a campaign, it must be:

1. **WIDELY HELD** — Most of the market believes it
2. **PROVABLY FALSE** — Can demonstrate the inversion is true
3. **EMOTIONALLY CHARGED** — Tied to deep frustration or hope
4. **MECHANISM-READY** — Can connect to a specific solution

### Inversion Analysis Template

```
BELIEF: "[What they believe]"
──────────────────────────────

PREVALENCE: [High/Med/Low]
Evidence: [Verbatim quotes showing prevalence]

THE INVERSION: "[What's actually true]"

PROOF AVAILABLE:
□ Scientific evidence: [Description]
□ Case studies: [Description]
□ Logic/reasoning: [Description]
□ Authority backing: [Description]

EMOTIONAL CHARGE: [1-10]
Connected emotions: [Fear/frustration/hope tied to this belief]

MECHANISM CONNECTION:
How does the product/method prove the inversion?
[Description]

BIG IDEA POTENTIAL: [1-10]
Rationale: [Why this score]

CAMPAIGN ANGLE:
If this powered a campaign, the hook would be:
"[Draft hook using this inversion]"
```
