# Artifact Sequence Patterns by Request Type

Detailed patterns for generating the 16 artifacts based on different types of user requests.

## Overview

The Infinite Prompting System adapts its artifact sequence based on what the user is asking for. While the core structure remains (15 exploration artifacts + 1 synthesis), the specific focus of each artifact changes dramatically.

## Writing / Content Creation

### Use When
- User requests articles, blog posts, essays
- Content needs to inform, persuade, or entertain
- Tone, style, and audience are critical
- Structure and flow matter

### Artifact Sequence

**Artifacts 1-3: Foundation**
1. **Topic Analysis**
   - Core subject matter
   - Key concepts to cover
   - Boundaries and scope
   - Example: "The topic 'AI in healthcare' encompasses diagnosis, treatment planning, patient monitoring, ethical considerations..."

2. **Audience Understanding**
   - Who will read this
   - Prior knowledge level
   - Motivations and pain points
   - Reading context
   - Example: "The target audience is healthcare professionals with moderate tech literacy, seeking practical applications..."

3. **Key Themes Identification**
   - Main messages
   - Supporting points
   - Potential objections
   - Narrative arc
   - Example: "Three key themes emerge: efficiency gains, quality improvements, ethical guardrails..."

**Artifacts 4-6: Structure and Evidence**
4. **Structure Development**
   - Introduction strategy
   - Section organization
   - Transition planning
   - Conclusion approach
   - Example: "Hook with surprising statistic → Problem definition → Solution exploration → Implementation guide → Future outlook"

5. **Evidence Gathering**
   - Data points
   - Examples
   - Expert citations
   - Case studies
   - Example: "Mayo Clinic's AI diagnostic tool reduced false negatives by 23%..."

6. **Counter-Arguments**
   - Anticipated objections
   - Rebuttals
   - Balanced perspective
   - Credibility building
   - Example: "Privacy concerns are valid; address with specific technical safeguards..."

**Artifacts 7-9: Differentiation and Flow**
7. **Unique Angle Discovery**
   - Fresh perspective
   - Unexpected insights
   - What hasn't been said
   - Contrarian elements
   - Example: "Instead of focusing on AI replacing doctors, explore AI as 'cognitive prosthetic'..."

8. **Narrative Flow**
   - Story elements
   - Pacing
   - Tension and release
   - Reader journey
   - Example: "Open with doctor's frustration → Introduce AI tool → Show transformation → Scale to systemic impact"

9. **Emotional Resonance**
   - Emotional hooks
   - Human elements
   - Relatable moments
   - Empathy building
   - Example: "Patient story: early cancer detection that wouldn't have happened without AI..."

**Artifacts 10-12: Polish and Impact**
10. **Example Development**
    - Concrete illustrations
    - Vivid details
    - Memorable stories
    - Practical demonstrations
    - Example: "Dr. Sarah Chen used AI to review 1,000 mammograms in 2 hours—previous process took 2 weeks..."

11. **Hook Creation**
    - Opening lines
    - Subheadings
    - Pull quotes
    - Attention grabbers
    - Example: "'Your doctor's new assistant never sleeps, never misses a detail, and has reviewed 10 million cases. It's also not human.'"

12. **Clarity Optimization**
    - Simplification
    - Jargon removal
    - Transition smoothing
    - Readability enhancement
    - Example: "Replace 'algorithmic decision-making' with 'how the AI makes choices'..."

**Artifacts 13-15: Final Preparation**
13. **Impact Amplification**
    - Memorable moments
    - Quotability
    - Shareability
    - Lasting impression
    - Example: "End with: 'The question isn't whether AI will transform healthcare. It's whether we'll do it wisely.'"

14. **Final Polish**
    - Grammar and style
    - Consistency check
    - Formatting
    - Flow optimization
    - Example: "Standardize all statistics format, ensure consistent tone throughout..."

15. **Meta-Reflection**
    - Overall effectiveness
    - Gaps or weaknesses
    - Surprise insights
    - Lessons learned
    - Example: "The angle of 'cognitive prosthetic' emerged in Artifact 7 and became the unifying metaphor..."

**Artifact 16: Complete Content**
- Fully written piece
- All insights incorporated
- Ready to publish
- Includes unexpected depth from exploration process

## Problem-Solving

- User faces a specific challenge
- Solution needs to be practical and actionable

3. **Root Cause Analysis**
   - 5 Whys analysis
   - Fishbone diagram thinking
   - System dynamics
   - Historical context

**Artifacts 4-6: Solution Generation**
4. **Solution Brainstorming**
   - Multiple approaches
   - Creative alternatives
   - Conventional and unconventional
   - Quick wins and long-term solutions

5. **Feasibility Assessment**
   - Technical feasibility
   - Economic feasibility
   - Timeline feasibility
   - Political feasibility

6. **Risk Identification**
   - What could go wrong
   - Probability and impact
   - Mitigation strategies
   - Early warning signs

**Artifacts 7-9: Refinement**
7. **Alternative Approaches**
   - Different paradigms
   - Lateral solutions
   - Industry analogies
   - Unconventional methods

8. **Resource Optimization**
   - Maximize efficiency
   - Minimize waste
   - Leverage existing assets
   - Creative resource allocation

9. **Implementation Planning**
   - Step-by-step roadmap
   - Dependencies
   - Critical path
   - Quick wins

**Artifacts 10-12: Validation**
10. **Stakeholder Impact**
    - Who is affected
    - Buy-in requirements
    - Communication strategy

11. **Success Metrics**
    - Quantitative measures
    - Qualitative indicators

12. **Contingency Planning**

**Artifacts 13-15: Synthesis**

    - Weighting factors

    - Recommendation logic

14. **Action Prioritization**
    - High-impact actions
    - Quick wins
    - Long-term investments
    - Sequencing strategy

15. **Validation Method**
    - How to test solution
    - Pilot approach
    - Success criteria
    - Iteration plan

**Artifact 16: Complete Solution**
- Recommended solution
- Implementation roadmap
- Risk mitigation strategies
- Success metrics
- Contingencies

   - Gaps in evidence

5. **Framework Selection**

   - Justification for choices

6. **Deep Dive Analysis**
   - Detailed examination
   - Layer-by-layer exploration
   - Mechanism understanding
   - Causal relationships

**Artifacts 7-9: Connections**

   - Analogies from other fields

   - Unexpected parallels
   - Meta-patterns

8. **Contradiction Exploration**
   - Conflicting evidence
   - Paradoxes
   - Resolution attempts
   - Productive tensions

12. **Implication Mapping**

    - Practical applications

**Artifacts 13-15: Validation**

    - Strength of evidence
    - Certainty levels

14. **Recommendation Development**
    - Based on analysis
    - Actionable guidance

    - Logical consistency
    - Evidence adequacy

**Artifact 16: Complete Analysis**
- Executive summary
- Detailed findings
- Supporting evidence
- Implications
- Recommendations
- Limitations and caveats

## Hybrid Requests

Many requests combine multiple types. For example:

**"Write an analysis of this problem and propose creative solutions"**
- Artifacts 1-5: Analysis pattern
- Artifacts 6-10: Problem-solving pattern
- Artifacts 11-14: Creative pattern
- Artifact 15: Integration
- Artifact 16: Complete hybrid response

## Adaptation Principles

1. **Identify Primary Request Type** - What is the user mainly asking for?
2. **Note Secondary Elements** - What other aspects are important?
3. **Choose Base Pattern** - Start with the primary type's pattern
4. **Integrate Secondary Patterns** - Blend in elements from other patterns
5. **Maintain Momentum** - Ensure artifacts still build progressively
6. **Explicit References** - Each artifact cites insights from previous ones

## Quality Indicators

Regardless of request type, quality artifacts:
- Are substantial (200-800 words)
- Build explicitly on previous artifacts
- Add new synthetic data
- Use specific reasoning methods
- Generate genuine insights
- Contribute to momentum
- Lead to breakthrough understanding in final artifact
