## Analysis Framework Quick Checklist

### 1. Headline & Hook Analysis ✓

- [ ] Primary headline extracted (exact copy)
- [ ] Subheadline extracted (exact copy)
- [ ] Hook type identified (fear/hope/curiosity/pride)
- [ ] Unique mechanism mentioned
- [ ] Emotional trigger documented

**Quick Assessment:**
- Does headline stop scroll in 3 seconds?
- Is unique mechanism present?
- Does it promise specific benefit?

### 2. Problem/Solution Framework ✓

- [ ] Problem introduction method
- [ ] Pain points listed (specific)
- [ ] Villain identified (if present)
- [ ] Solution positioning
- [ ] Mechanism explanation
- [ ] Differentiation from alternatives

**Quick Assessment:**
- Problem agitation level: Low / Medium / High
- Time on problem vs. solution: __%  / __%
- Unique mechanism clarity: Weak / Good / Strong

### 3. Social Proof Strategy ✓

**Types to Check:**
- [ ] Customer testimonials (count: ___)
- [ ] Video testimonials (count: ___)
- [ ] Star ratings/review count
- [ ] Expert endorsements
- [ ] Media mentions
- [ ] Before/after transformations
- [ ] Clinical study references
- [ ] Customer count claims

**Quick Assessment:**
- Total proof elements: ___
- Specificity level: Generic / Mixed / Specific
- Visual proof present: Yes / No

### 4. Scarcity & Urgency Tactics ✓

**Check for:**
- [ ] Countdown timer
- [ ] Limited stock claims
- [ ] Sale/discount ending
- [ ] Seasonal/batch scarcity
- [ ] Exclusive access claims

**Quick Assessment:**
- Urgency level: None / Subtle / Moderate / Aggressive
- Placement: Above fold / Near CTA / Throughout
- Authenticity feel: Real / Artificial

### 5. Scientific Backing ✓

- [ ] Clinical studies mentioned
- [ ] Ingredient research citations
- [ ] Mechanism explanation depth
- [ ] Doctor endorsement/formulation
- [ ] Third-party testing
- [ ] Quality certifications
- [ ] Sourcing transparency

**Quick Assessment:**
- Science approach: Absent / Basic / Moderate / Comprehensive
- Accessibility: Too complex / Good balance / Too simple

### 6. Pricing Psychology ✓

- [ ] Price format (one-time/$X per day/etc.)
- [ ] Anchor pricing used
- [ ] Bundle discounts
- [ ] Subscription option
- [ ] Guarantee terms
- [ ] Value stacking
- [ ] Payment plans

**Quick Assessment:**
- Price point: $___
- Anchor used: Yes / No
- Guarantee prominence: Weak / Moderate / Strong

## Scoring Quick Reference

### Category Weights

| Category | Weight | 1-2 (Weak) | 3-4 (Poor) | 5-6 (Average) | 7-8 (Good) | 9-10 (Excellent) |
|----------|--------|------------|------------|---------------|------------|------------------|
| **Conversion** | 25% | Fails to engage | Weak opening | Decent but missing | Strong opener | Immediately compelling |
| **Copywriting** | 20% | Generic copy | Weak headlines | Adequate copy | Strong copy | Exceptional copy |
| **Visual/UX** | 15% | Poor design | Weak layout | Acceptable | Good design | Perfect hierarchy |
| **Social Proof** | 15% | Little/no proof | Minimal proof | Basic proof | Good variety | Multiple strong types |
| **Scientific** | 10% | No backing | Vague claims | Basic explanation | Good credibility | Specific studies |
| **Mobile** | 8% | Broken mobile | Poor mobile | Acceptable | Good mobile | Perfect mobile |
| **Trust** | 7% | No signals | Minimal signals | Basic signals | Good signals | Multiple strong signals |

### Total Score Ranges

- **9.0-10.0:** Exceptional - Study extensively, top tier
- **8.0-8.9:** Excellent - Strong performer, many lessons
- **7.0-7.9:** Good - Solid page, specific elements worth studying
- **6.0-6.9:** Average - Meets baseline, nothing special
- **Below 6.0:** Weak - Avoid patterns, study what NOT to do

## Common Headline Patterns

| Pattern | Formula | Example |
|---------|---------|---------|
| **Unique Mechanism** | [Ingredient] + [Benefit] + [Without Negative] | "Mushroom Coffee That Boosts Energy Without Jitters" |
| **Problem Agitation** | [Problem] + [Amplification] + [Solution Tease] | "Tired of Bloating? This Ancient Root Fixes Your Gut (Without Probiotics)" |
| **Curiosity Gap** | [Surprising Claim] + [Incomplete Explanation] | "The Nobel Prize-Winning Molecule Your Body Stopped Making at 25" |
| **Social Proof** | [X People] + [Result] + [Product] | "47,000+ People Trust This for Daily Energy (Here's Why)" |
| **Villain/Hero** | [Villain Identified] + [Product as Hero] | "Big Pharma Doesn't Want You to Know About This Natural Alternative" |

## Social Proof Hierarchy (Effectiveness)

**Rank 1:** Video testimonials with specificity
- Name, face, specific results
- "Sarah from Texas: Lost 23 pounds in 8 weeks"

**Rank 2:** Before/after transformations
- Visual proof of change
- Specific timeframe

**Rank 3:** Expert/doctor endorsements
- Credentialed professionals
- Formulation involvement

**Rank 4:** Aggregate numbers
- "127,000+ customers served"
- Star ratings with count

**Rank 5:** Media mentions
- "As seen in" logos
- Third-party credibility

**Rank 6:** Generic testimonials
- No photo, generic name
- Vague results

## Scarcity Tactics by Frequency

| Tactic | Typical Frequency | Implementation | Effectiveness |
|--------|-------------------|----------------|---------------|
| **Sale Ending** | 60-70% of pages | "Sale ends tonight" | High if authentic |
| **Limited Stock** | 40-50% of pages | "Only 47 left in stock" | Medium (often doubted) |
| **Countdown Timer** | 30-40% of pages | Visual timer above CTA | High if real |
| **Seasonal/Batch** | 20-30% of pages | "Limited spring harvest" | High authenticity |
| **Exclusive Access** | 15-25% of pages | "Members only pricing" | Medium |

## Pricing Psychology Quick Reference

| Strategy | Implementation | Psychological Principle |
|----------|----------------|------------------------|
| **Daily Cost** | "$1.47/day" | Makes large price feel small |
| **Anchor High** | "Normally $147, today $47" | Contrast makes deal feel bigger |
| **Bundle Discount** | "Buy 3, save 30%" | Rewards bulk purchase |
| **Value Stack** | "$200 value for $47" | Perceived ROI |
| **Free Bonuses** | "Plus get $97 bonus free" | Increases perceived value |
| **Comparison** | "Less than one doctor visit" | Relative value framing |
| **Subscription Discount** | "Subscribe and save 20%" | Ongoing relationship incentive |

## CTA Language Patterns

| Pattern | When to Use | Example |
|---------|-------------|---------|
| **Benefit-Driven** | High-awareness audience | "Boost My Energy Now" |
| **Action-Focused** | Low friction offer | "Get Started Today" |
| **Risk-Reversal** | High ticket or skeptical | "Try Risk-Free for 60 Days" |
| **Urgency-Laden** | Time-sensitive offer | "Claim My Discount Before Midnight" |
| **Curiosity** | Educational offers | "Show Me How It Works" |
| **Exclusive** | Premium positioning | "Join 10,000+ Members" |

## Output Format Requirements

### Every Analysis Must Include:

**1. Executive Summary** (2-3 sentences)
- Highest-level insights
- Key patterns discovered
- Main competitive intelligence

**2. Top 5-10 Highest-Performing Pages**
Each with:
- Company, product, URL, overall score
- 3-5 standout elements
- Conversion strategy summary
- Copywriting insights
- Category scores

**3. Pattern Analysis**
- Winning headline formulas
- Most effective social proof types
- Common scarcity tactics
- Scientific positioning strategies
- Pricing psychology patterns

**4. Actionable Recommendations**
- Top 3 conversion elements
- Top 3 copywriting patterns
- Top 3 design/UX elements
- Red flags to avoid

**5. Competitive Intelligence**
- Market positioning patterns
- Target audience insights
- Industry trends

## Common Mistakes Quick Fix

| Mistake | Fix |
|---------|-----|
| Analyzing homepages | Navigate to actual product sales pages |
| Paraphrasing copy | Copy/paste exact headlines and CTAs |
| Generic observations | Always cite specific examples with URLs |
| Outdated pages | Verify all URLs before analyzing |
| Skipping mobile | Test every page on mobile device |
| Missing "why" | Explain psychological reasoning for each element |
| No cross-reference | Track patterns across multiple pages |

## Pattern Recognition Speed Tips

**Headline Patterns:**
1. Read all headlines
2. Group by structure type
3. Note common words/phrases
4. Identify unique mechanisms
5. Document formulas

**Social Proof Patterns:**
1. Count each type
2. Note placement
3. Assess specificity
4. Rank by frequency
5. Identify most effective

**Urgency Patterns:**
1. List all scarcity tactics
2. Count frequency
3. Note implementation
4. Assess authenticity
5. Rank effectiveness

## One-Liner Reminders

- **URL Rule:** Every page must have exact, working URL
- **Quote Rule:** All copy must be exact (copy/paste only)
- **Specific Rule:** No generic observations without examples
- **Why Rule:** Always explain psychological mechanism
- **Pattern Rule:** Track what appears across multiple pages
- **Action Rule:** Every insight must be actionable
- **Quality Rule:** CEO-level deliverable standard
- **Verify Rule:** Test all URLs before final delivery
- **Mobile Rule:** Check every page on mobile
- **Complete Rule:** All scoring categories must be filled

## Emergency Recovery

If analysis seems incomplete:

1. **Identify gap** - Which of 10 categories missing?
2. **Return to page** - Re-analyze specific element
3. **Extract verbatim** - Copy exact text
4. **Document why** - Explain mechanism
5. **Find pattern** - Check if appears on other pages
6. **Verify quality** - Meet CEO standard?
