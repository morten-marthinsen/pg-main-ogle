# Core Skill 0.3: Authenticity Validator

**Layer:** 0 - Core Infrastructure
**Purpose:** Verify extracted quotes are REAL, not hallucinated or fabricated
**Dependency:** Raw scrape data must exist
**Version:** 1.0
**Type:** BLOCKING GATE - Prevents fake data from entering pipeline

---

## CRITICAL: This Skill Exists Because of a Real Failure

**What Happened:**
- System reported 92 extracted quotes
- Validation checked structure (fields exist, not empty)
- All checks passed
- Later discovered: quotes were fabricated, not from actual sources
- 91% data loss - entire analysis was invalid

**Root Cause:**
- Validation checked IF data existed, not IF data was AUTHENTIC
- No cross-reference against raw scrape files
- No source verification

**This Skill Ensures:**
1. Every quote traces to a raw scrape file
2. Quote text appears verbatim in raw data
3. Random sample verified against live sources
4. Hash chain maintains data integrity

---

## Authenticity Validation Architecture

```
RAW SCRAPE DATA (Source of Truth)
       ↓
   Hash on creation
       ↓
CONTENT EXTRACTOR (1.5-A)
       ↓
   Quote must reference source hash
       ↓
AUTHENTICITY VALIDATOR (This Skill)
       ↓
   Cross-reference every quote
       ↓
   Random sample live verification
       ↓
VALIDATED QUOTES → Layer 2
```

## Validation Layers

### Layer 1: Existence Verification

**Check:** Does the quote exist in raw data?

```yaml
existence_check:
  for_each_quote:
    - Load raw source file referenced by quote
    - Verify file exists
    - Verify file hash matches recorded hash
    - Search for quote text in file content
    - Record: found/not_found, location, confidence

  pass_criteria:
    - 100% of quotes must have valid source file reference
    - 95% of quotes must be found in source file (5% tolerance for extraction variations)

  fail_actions:
    - If quote not in source: Flag as "UNVERIFIABLE"
    - If >5% unverifiable: BLOCK progression, require investigation
```

### Layer 2: Content Matching

**Check:** Does the quote text match the source verbatim?

```yaml
content_matching:
  matching_modes:
    exact:
      description: "Quote is character-for-character identical"
      tolerance: 0
      confidence: 1.0

    normalized:
      description: "Quote matches after whitespace/punctuation normalization"
      operations: [trim, collapse_whitespace, normalize_quotes, remove_ellipsis]
      confidence: 0.95

    fuzzy:
      description: "Quote substantially matches (minor extraction variations)"
      threshold: 0.90  # 90% similarity
      confidence: 0.85

    partial:
      description: "Quote is subset of larger text in source"
      min_overlap: 0.80
      confidence: 0.75

  pass_criteria:
    - 80% exact or normalized match
    - 95% at least fuzzy match
    - 0% complete mismatch (no trace in source)

  mismatch_handling:
    IF exact_match: confidence = 1.0
    ELIF normalized_match: confidence = 0.95
    ELIF fuzzy_match: confidence = 0.85, FLAG for review
    ELIF partial_match: confidence = 0.75, FLAG for review
    ELSE: REJECT quote, log as "FABRICATION_SUSPECTED"
```

### Layer 3: Source Hash Chain

**Check:** Has raw data been modified since scraping?

```yaml
hash_chain:
  on_scrape_complete:
    - Generate SHA-256 hash of raw scrape file
    - Store in: raw-data/{file}_hash.json
    - Record: filename, hash, timestamp, scraper_id, source_count

  on_extraction:
    - For each source file read:
      - Recalculate hash
      - Compare to stored hash
      - IF mismatch: HALT, report "SOURCE_FILE_MODIFIED"

  on_validation:
    - Verify hash chain integrity
    - Ensure no gaps (every extraction references valid hash)

  hash_record_format:
    {
      "file": "raw-data/reddit_scrape_001.json",
      "hash": "sha256:abc123def456...",
      "created_at": "2026-01-17T10:00:00Z",
      "scraper_id": "1.4-C",
      "items_count": 150,
      "size_bytes": 245678
    }
```

### Layer 4: Live Source Verification (Sampling)

**Check:** Does quote actually exist at the claimed source URL?

```yaml
live_verification:
  sample_size: 10  # Random quotes to verify
  sample_selection:
    - 3 from highest confidence quotes
    - 3 from lowest confidence quotes
    - 4 random from remaining

  verification_process:
    for_each_sample:
      1. Get quote's source URL
      2. Fetch current page content (using tool chain)
      3. Search for quote text in fetched content
      4. Record: found/not_found, current_content_sample

  handling_not_found:
    possible_reasons:
      - Page content changed (legitimate)
      - Quote was from dynamic content (comments loaded via JS)
      - Quote was fabricated (CRITICAL)

    differentiation:
      IF page_structure_similar AND other_content_matches:
        → Likely page update, not fabrication
        → Confidence penalty: -0.1
      ELIF page_completely_different:
        → Likely URL changed or site restructured
        → Confidence penalty: -0.2
      ELIF page_accessible BUT quote_nowhere:
        → SUSPICIOUS - flag for human review
        → Confidence penalty: -0.5

  pass_criteria:
    - 60% of samples found at live source
    - No more than 1 sample with "SUSPICIOUS" flag
    - Average confidence after penalties > 0.7

  fail_action:
    IF < 60% found AND NOT due to site changes:
      → CRITICAL FAIL
      → HALT pipeline
      → Flag: "SYSTEMATIC_FABRICATION_SUSPECTED"
      → Require human investigation
```

## Implementation Protocol

### On Every Extraction

```python
def validate_extraction_authenticity(extracted_quotes: list, raw_data_dir: str) -> ValidationResult:
    """
    Validate that extracted quotes are authentic.

    Args:
        extracted_quotes: List of Quote objects from 1.5-A
        raw_data_dir: Path to raw scrape data

    Returns:
        ValidationResult with status and flagged quotes
    """

    results = {
        "layer_1": validate_existence(extracted_quotes, raw_data_dir),
        "layer_2": validate_content_matching(extracted_quotes, raw_data_dir),
        "layer_3": validate_hash_chain(raw_data_dir),
        "layer_4": validate_live_sources(extracted_quotes, sample_size=10)
    }

    # Aggregate results
    flagged_quotes = []
    for layer_result in results.values():
        flagged_quotes.extend(layer_result.flagged)

    # Calculate overall status
    authenticity_rate = calculate_authenticity_rate(extracted_quotes, flagged_quotes)

    if authenticity_rate >= 0.95:
        status = "PASS"
    elif authenticity_rate >= 0.90:
        status = "WARN"
    elif authenticity_rate >= 0.80:
        status = "FAIL"
    else:
        status = "CRITICAL_FAIL"

    return ValidationResult(
        status=status,
        authenticity_rate=authenticity_rate,
        flagged_quotes=flagged_quotes,
        layer_results=results
    )
```

### Existence Validation

```python
def validate_existence(quotes: list, raw_data_dir: str) -> LayerResult:
    """
    Layer 1: Verify each quote has a valid source file.
    """
    results = []

    for quote in quotes:
        source_file = quote.source_file
        source_path = f"{raw_data_dir}/{source_file}"

        if not file_exists(source_path):
            results.append({
                "quote_id": quote.id,
                "status": "FAIL",
                "reason": "source_file_missing",
                "source_file": source_file
            })
            continue

        # Verify hash
        expected_hash = get_stored_hash(source_file)
        actual_hash = calculate_hash(source_path)

        if expected_hash != actual_hash:
            results.append({
                "quote_id": quote.id,
                "status": "FAIL",
                "reason": "source_file_modified",
                "expected_hash": expected_hash,
                "actual_hash": actual_hash
            })
            continue

        results.append({
            "quote_id": quote.id,
            "status": "PASS"
        })

    failed = [r for r in results if r["status"] == "FAIL"]
    return LayerResult(
        status="PASS" if len(failed) == 0 else "FAIL",
        passed=len(results) - len(failed),
        failed=len(failed),
        flagged=[r for r in results if r["status"] == "FAIL"]
    )
```

### Content Matching

```python
def validate_content_matching(quotes: list, raw_data_dir: str) -> LayerResult:
    """
    Layer 2: Verify quote text appears in source file.
    """
    results = []

    for quote in quotes:
        source_content = load_source_content(raw_data_dir, quote.source_file)
        quote_text = quote.text

        # Try exact match
        if quote_text in source_content:
            results.append({
                "quote_id": quote.id,
                "match_type": "exact",
                "confidence": 1.0
            })
            continue

        # Try normalized match
        normalized_quote = normalize_text(quote_text)
        normalized_source = normalize_text(source_content)

        if normalized_quote in normalized_source:
            results.append({
                "quote_id": quote.id,
                "match_type": "normalized",
                "confidence": 0.95
            })
            continue

        # Try fuzzy match
        similarity = fuzzy_search(quote_text, source_content)
        if similarity >= 0.90:
            results.append({
                "quote_id": quote.id,
                "match_type": "fuzzy",
                "confidence": 0.85,
                "similarity": similarity
            })
            continue

        # Try partial match
        overlap = find_max_overlap(quote_text, source_content)
        if overlap >= 0.80:
            results.append({
                "quote_id": quote.id,
                "match_type": "partial",
                "confidence": 0.75,
                "overlap": overlap
            })
            continue

        # No match found
        results.append({
            "quote_id": quote.id,
            "match_type": "none",
            "confidence": 0.0,
            "flag": "FABRICATION_SUSPECTED"
        })

    no_match = [r for r in results if r["match_type"] == "none"]
    return LayerResult(
        status="PASS" if len(no_match) == 0 else "FAIL" if len(no_match) > len(quotes) * 0.05 else "WARN",
        flagged=no_match
    )
```

### Live Source Verification

```python
def validate_live_sources(quotes: list, sample_size: int = 10) -> LayerResult:
    """
    Layer 4: Verify sample of quotes against live sources.
    """
    # Select diverse sample
    sample = select_verification_sample(quotes, sample_size)

    results = []
    for quote in sample:
        url = quote.source_url

        # Skip if no URL (some sources don't have direct URLs)
        if not url:
            results.append({
                "quote_id": quote.id,
                "status": "SKIPPED",
                "reason": "no_url"
            })
            continue

        # Fetch current content using tool chain
        try:
            current_content = execute_chain(
                chain_id="COMPETITOR_PAGES",  # Generic fetch chain
                params={"url": url}
            )
        except Exception as e:
            results.append({
                "quote_id": quote.id,
                "status": "FETCH_FAILED",
                "error": str(e)
            })
            continue

        # Search for quote in current content
        if quote.text in current_content.data:
            results.append({
                "quote_id": quote.id,
                "status": "FOUND",
                "confidence_impact": 0
            })
        elif fuzzy_search(quote.text, current_content.data) >= 0.85:
            results.append({
                "quote_id": quote.id,
                "status": "FUZZY_FOUND",
                "confidence_impact": -0.05
            })
        else:
            # Not found - determine if suspicious
            page_similarity = compare_page_structure(
                quote.source_content_sample,
                current_content.data
            )

            if page_similarity > 0.7:
                # Page similar but quote gone - might be updated
                results.append({
                    "quote_id": quote.id,
                    "status": "NOT_FOUND_PAGE_CHANGED",
                    "confidence_impact": -0.1
                })
            else:
                # Page completely different - suspicious
                results.append({
                    "quote_id": quote.id,
                    "status": "SUSPICIOUS",
                    "confidence_impact": -0.5,
                    "flag": "REQUIRES_HUMAN_REVIEW"
                })

    found = [r for r in results if r["status"] in ["FOUND", "FUZZY_FOUND"]]
    suspicious = [r for r in results if r["status"] == "SUSPICIOUS"]

    return LayerResult(
        status="PASS" if len(found) >= sample_size * 0.6 and len(suspicious) <= 1 else "FAIL",
        verification_rate=len(found) / len([r for r in results if r["status"] != "SKIPPED"]),
        flagged=suspicious
    )
```

## Integration Points

### Before Layer 2 Proceeds

```yaml
integration:
  after_1.5-A_extraction:
    - AUTHENTICITY_VALIDATOR runs automatically
    - Reads: extracted_quotes.json, raw-data/*
    - Outputs: authenticity_report.json, validated_quotes.json

  gate_check:
    - 1.6-B Data Sufficiency Validator now REQUIRES authenticity_report.json
    - 1.6-B checks: authenticity_rate >= 0.90
    - IF authenticity_rate < 0.90: BLOCK Layer 2

  quote_modification:
    - Quotes flagged as FABRICATION_SUSPECTED are REMOVED from dataset
    - Quotes with reduced confidence have confidence_score updated
    - validated_quotes.json only contains authenticated quotes
```

### Required Raw Data Structure

```yaml
raw_data_requirements:
  on_scrape:
    - Save full response to raw-data/{platform}_{timestamp}.json
    - Generate hash and save to raw-data/{platform}_{timestamp}_hash.json
    - Include metadata: scrape_time, tool_used, params, item_count

  during_extraction:
    - Record source_file for every extracted quote
    - Record approximate_location (character offset or item index)
    - Record source_content_sample (surrounding 200 chars)

  for_validation:
    - raw-data/ must be preserved until research complete
    - NEVER modify raw-data/ after hash generation
    - Hash files are the source of truth for integrity
```

## Failure Handling

### On Validation Failure

```yaml
failure_protocol:
  IF authenticity_rate < 0.95 AND >= 0.90:
    - Status: WARN
    - Action: Proceed with documented risk
    - Log: Which quotes are suspect, why

  IF authenticity_rate < 0.90 AND >= 0.80:
    - Status: FAIL
    - Action: BLOCK Layer 2
    - Require: Human review of flagged quotes
    - Options:
      - Remove flagged quotes and re-validate
      - Investigate extraction process
      - Re-scrape sources with issues

  IF authenticity_rate < 0.80:
    - Status: CRITICAL_FAIL
    - Action: HALT pipeline completely
    - Require: Full investigation
    - Suspect: Systematic extraction failure or hallucination
    - Do NOT proceed under any circumstances

  IF hash_chain_broken:
    - Status: CRITICAL_FAIL
    - Action: HALT pipeline
    - Reason: Data integrity compromised
    - Require: Determine if raw data modified intentionally or corruption
```

## Quality Criteria

- [ ] 100% of quotes have source file reference
- [ ] 100% of source files have valid hash
- [ ] 95%+ of quotes found in source files
- [ ] 0% of quotes have no trace in any source (FABRICATION)
- [ ] Live verification passes for 60%+ of samples
- [ ] No more than 1 "SUSPICIOUS" flag in live verification
- [ ] Hash chain is intact with no modifications

## Notes

- This validator is the defense against the exact failure that broke Ion Golf Ball analysis
- Raw data is SACRED - never modify, never delete until research complete
- Hash chain provides audit trail and integrity verification
- Live verification is sampling - can't check every quote but catches systematic issues
- When in doubt, flag for human review rather than pass through
- Better to have fewer verified quotes than many unverified ones
