**Purpose:** Generate human-reviewable Markdown file showing ALL quotes by category with tallies for validation before Final Handoff

---

## Purpose

This skill generates **SYNTHESIS_VALIDATION.md** - a human-reviewable Markdown document that:
1. Shows ALL 1,000+ quotes organized by bucket and subcategory
2. Displays quotes as readable lines (NOT tables)
3. Provides tally counts for every category
4. Summarizes the BIGGEST patterns identified
5. Serves as a validation checkpoint before Final Handoff

**This file is the human's last opportunity to verify synthesis quality before the system proceeds to assemble FINAL_HANDOFF.md.**

## CRITICAL: Quote Display Format

Quotes MUST be displayed as readable blockquote lines, NOT in tables:

**CORRECT FORMAT:**
```markdown
> "Quote text here exactly as collected" — Source [ID] [PRIORITY] [tag1, tag2]
```

**INCORRECT FORMAT (DO NOT USE):**
```markdown
| ID | Quote | Source | Priority | Tags |
|----|-------|--------|----------|------|
| P-021 | Quote text... | GolfWRX | GOLD | desperation |
```

The readable line format ensures:
- Quotes flow naturally when reading
- Human can quickly scan quote quality
- Tags are visible but don't interrupt the quote
- Easier to spot miscategorized quotes

## Validation Rules

### Completeness

- [ ] Every quote from two_tier_quotes.json is listed
- [ ] Every bucket section is present
- [ ] Every subcategory has a header with count
- [ ] All transformation pairs listed
- [ ] All educational pairs listed
- [ ] Pattern analysis section complete
- [ ] Validation checklist present

### Format Compliance

- [ ] Quotes displayed as blockquotes (>), NOT in tables
- [ ] Each quote shows: full text, source, ID, priority, emotional tags
- [ ] Subcategory headers show count and BIGGEST marker if applicable
- [ ] Tables used ONLY for summary stats, NOT for quotes

### Content Accuracy

- [ ] Total quote count matches two_tier_quotes.json
- [ ] Bucket counts sum to total
- [ ] Subcategory counts sum to bucket counts
- [ ] Transformation pair count matches JSON
- [ ] Educational pair count matches JSON
