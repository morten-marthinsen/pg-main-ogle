## Purpose

This skill ensures **every quote** has proper two-tier categorization:
1. **Primary:** Physical Problem tag (what product solves)
2. **Secondary:** Emotional tags (how it feels)

This categorization is critical for:
1. Bucket section assembly in FINAL_HANDOFF
2. Subcategory breakdown tables
3. Emotional distribution analysis
4. Copy targeting by problem type

---

## CRITICAL: Full Analysis Requirement

**ALL 1,000+ quotes MUST be categorized.**

This skill is NOT about finding a subset of quotes. It MUST:
1. Process EVERY quote from Layer 1 outputs
2. Assign physical_tag to EVERY quote
3. Assign emotional_tags[] to EVERY quote
4. Tally ALL subcategories to identify the BIGGEST patterns
5. NO quotes left uncategorized

The goal is to provide the data foundation for:
- Identifying the **BIGGEST Pain** (most frequent physical problem)
- Identifying the **BIGGEST Hope** (most frequent resolution)
- Identifying the **BIGGEST Why** (most frequent root cause)
- Identifying the **BIGGEST How** (most frequent mechanism)

## Two-Tier System Explained

### Tier 1: Physical Tag (Primary)
**What physical problem does the product solve?**

This is NOT an emotional category. It's the tangible issue.

**For PAIN bucket:**
- SLICE - Ball curves right
- CONFIDENCE_COLLAPSE - Mental paralysis
- DISTANCE_INCONSISTENCY - Can't hit it far reliably
- SOCIAL_EMBARRASSMENT - Affects playing with others
- EQUIPMENT_CHURN - Constant searching for solution

**For HOPE bucket:**
- STRAIGHT_SHOTS - Ball goes straight
- FAIRWAY_ACCURACY - Hitting fairways
- DISTANCE_MAINTAINED - No distance sacrifice
- CONFIDENCE_RESTORED - Trust in the club
- SCORE_IMPROVEMENT - Lower scores

**For ROOT_CAUSE bucket:**
- MECHANICAL_CAUSE - Physical swing issue
- MENTAL_CAUSE - Psychological belief
- EQUIPMENT_MISMATCH - Wrong equipment
- TECHNIQUE_BELIEF - Believe technique is problem

**For SOLUTIONS_TRIED bucket:**
- LESSONS - Instruction attempts
- EQUIPMENT_CHANGES - New clubs/shafts
- SWING_ADJUSTMENTS - DIY fixes
- PRACTICE - Range time

**For COMPETITOR_MECHANISM bucket:**
- DRAW_BIAS - Draw-biased clubs
- MINI_DRIVER - Smaller heads
- SHORTER_SHAFT - Cut down shafts
- FITTING - Professional fitting

**For VILLAIN bucket:**
- GOLF_INDUSTRY - OEM marketing
- INFOMERCIAL_CLUBS - Gimmick products
- INSTRUCTION_INDUSTRY - Lessons that don't work
- PEER_PRESSURE - Social expectations

### Tier 2: Emotional Tags (Secondary)
**How does this quote express emotion?**

Common emotional tags:
- frustration
- fear
- embarrassment
- hope
- relief
- confidence
- desperation
- transformation
- skepticism
- excitement
- dread
- mental_defeat

## Priority Assignment Rules

### GOLD Priority
Quote is GOLD if it has:
- Vivid imagery ("2 fairways to the right")
- Quantified data ("96 percent", "33% fairways")
- Academic/research validation
- Product testimonial (specific to researched product)
- Directly validates product mechanism

### HIGH Priority
Quote is HIGH if it has:
- Strong emotional language
- Relatable scenario
- Clear problem articulation
- Useful for copy

### MEDIUM Priority
Quote is MEDIUM if it:
- Supports a pattern
- Adds volume to a category
- Less vivid but still useful

### STANDARD Priority
Quote is STANDARD if it:
- Basic data point
- Repetitive of stronger quotes
- Lower emotional impact

## Validation Rules

### Completeness Validation
Every quote MUST have:
- [ ] `id` (format: P-XXX, H-XXX, RC-XXX, ST-XXX, CM-XXX, V-XXX)
- [ ] `quote` (verbatim text)
- [ ] `bucket` (PAIN, HOPE, ROOT_CAUSE, SOLUTIONS_TRIED, COMPETITOR_MECHANISM, VILLAIN)
- [ ] `physical_tag` (from bucket-specific tag list)
- [ ] `emotional_tags` (array with 1-3 tags)
- [ ] `source` (where it came from)
- [ ] `priority` (GOLD, HIGH, MEDIUM, STANDARD)

### No Missing Tags
- [ ] 0 quotes with empty physical_tag
- [ ] 0 quotes with empty emotional_tags array
- [ ] 0 quotes with "uncategorized" or similar

### Distribution Sanity
- [ ] Each bucket has quotes
- [ ] Each physical_tag within bucket has quotes
- [ ] No single physical_tag dominates (>60% of bucket)
