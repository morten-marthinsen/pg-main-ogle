**Version:** 1.0

**Layer:** 2.5 - Synthesis
**Purpose:** Generate WEB Analysis (Wants-Emotions-Beliefs) using Vic Schwab framework

---

## Purpose

This skill creates the **WEB Analysis** - a comprehensive mapping of prospect psychology organized into:
- **W**ants (using Vic Schwab's 5 categories)
- **E**motions (current state vs desired state)
- **B**eliefs (to disrupt and to align with)

This analysis is critical for:
1. Messaging strategy foundation
2. Emotional targeting in copy
3. Belief disruption sequences
4. Desire articulation

## Vic Schwab WANTS Framework

### GAIN
What do they want to **acquire** or **achieve**?
- Physical outcomes (straight drives, more fairways)
- Performance improvements (better scores, longer drives)
- Skills (consistent tee shots)

### BE
What do they want to **become** or **be seen as**?
- Identity (confident golfer, fairway finder)
- Status (respected player, not the weak link)
- Reputation (the guy who stripes it)

### DO
What do they want to **do** or **experience**?
- Activities (hit tee shots without dread)
- Experiences (enjoy the first tee)
- Actions (keep up with playing partners)

### SAVE
What do they want to **preserve** or **protect**?
- Resources (strokes, money, time)
- Face (avoid embarrassment)
- Relationships (not slow down the group)

### AVOID
What do they want to **escape** or **eliminate**?
- Situations (looking for ball in wrong fairway)
- Feelings (dread, embarrassment, frustration)
- Outcomes (penalties, lost balls, bad scores)

## Validation Rules

### WANTS Validation
- [ ] All 5 categories (GAIN, BE, DO, SAVE, AVOID) populated
- [ ] Each category has 2+ items minimum
- [ ] Each item has evidence_count and sample_quote_id
- [ ] Summary statement for each category

### EMOTIONS Validation
- [ ] current_state has 3+ primary emotions
- [ ] desired_state has 3+ primary emotions
- [ ] emotional_journey narrative present for both
- [ ] trigger_moments and desired_moments lists present

### BELIEFS Validation
- [ ] 3+ beliefs to disrupt
- [ ] 3+ beliefs to align
- [ ] Each belief has counter/validation
- [ ] Each has evidence_quote_id
- [ ] Each has disruption/alignment angle
