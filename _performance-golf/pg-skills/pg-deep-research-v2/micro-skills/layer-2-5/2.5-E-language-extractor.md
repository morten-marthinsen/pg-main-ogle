# 2.5-E Language Pattern Extractor

**Version:** 1.0
**Created:** January 20, 2026
**Layer:** 2.5 - Synthesis
**Purpose:** Extract exact language patterns and gold phrases organized by physical problem category

---

## Purpose

This skill extracts **verbatim language patterns** from prospect quotes, organized by category. These patterns are critical for:
1. Copy authenticity (using prospect's own words)
2. Headline generation
3. Ad hook creation
4. Objection language matching

---

| Input | Source | Required |
|-------|--------|----------|
| All quotes with tags | layer-1-outputs/raw_quotes_batch*.json | YES |
| Emotional intensity map | layer-2-outputs/E5.2_emotional_intensity_map.json | YES |
| PAIN subcategory names | From theme clustering | YES |
| Two-tier quotes | layer-2-5-outputs/two_tier_quotes.json (if exists) | OPTIONAL |

## Extraction Rules

### Physical Problem Language
For each PAIN subcategory:
1. Extract exact phrases (3-8 words) that describe the problem
2. Identify vivid imagery phrases (most visual/emotional)
3. Count frequency across all quotes
4. Tag each phrase with source quote_id

### Transformation Language
1. Extract FROM words (pain state descriptors)
2. Extract TO words (hope state descriptors)
3. Create transformation pairs showing the journey

### GOLD Phrase Criteria
A phrase is GOLD if it has:
- Vivid imagery (visual or emotional)
- High memorability
- Direct quotability
- Clear problem/solution connection
- Unique phrasing (not generic)

---

## Validation Rules

### Minimum Counts
- [ ] 5+ physical problem categories
- [ ] 10+ gold phrases
- [ ] Each category has 5+ exact phrases

### Quality Validation
- [ ] All gold phrases have quote_id
- [ ] All gold phrases have use_for tags
- [ ] No duplicate phrases
- [ ] Phrases are actually verbatim (not paraphrased)

## Failure Handling

If categories < 5 OR gold phrases < 10:
1. Log: "Language patterns insufficient: {details}"
2. Review raw quotes for missed patterns
3. Lower threshold for "gold" temporarily if needed
4. DO NOT proceed to Gate 2.5 until minimums met
