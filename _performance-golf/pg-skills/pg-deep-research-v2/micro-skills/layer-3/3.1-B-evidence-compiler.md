# Micro-Skill 3.1-B: Evidence Compiler

**Layer:** 3 - Opportunity Surfacing
**Purpose:** Compile comprehensive evidence packages for top opportunities
**Dependency:** 3.1-A Opportunity Scorer + all Layer 1 raw data
**Output:** Evidence packages ready for handoff

---


## Compilation Scope

### Which Opportunities Get Full Packages?

**Full Evidence Package:** Tier 1 opportunities (composite ≥ 7.5)
- Complete quote collection
- All pattern evidence
- Competitive context
- Audience mapping

**Summary Package:** Tier 2 opportunities (composite 5.0-7.4)
- Top 20 quotes
- Key patterns only
- Brief competitive notes

**Minimal Package:** Tier 3 opportunities (composite < 5.0)
- Top 5 quotes
- Why it didn't rank higher
- Conditions that might elevate it

---

## Evidence Compilation Process

### Step 1: Quote Collection

For each opportunity, gather ALL supporting quotes from Layer 1:

#### Quote Selection Criteria
1. Directly supports the opportunity
2. SQS ≥ 5 (medium quality or better)
3. Not duplicate/near-duplicate of already included quote

#### Quote Organization
Group quotes by:
- **Platform** (Reddit, YouTube, Forums, etc.)
- **Sentiment** (positive, negative, mixed)
- **Aspect** (pain, desire, frustration, hope, fear, identity)
- **Strength** (SQS score)

#### Quote Formatting
```json
{
  "quote_id": "q_00847",
  "text": "Full quote text...",
  "source": {
    "platform": "Reddit",
    "source_name": "r/golf",
    "url": "...",
    "date": "2024-12-10"
  },
  "tags": {
    "sentiment": "negative",
    "aspects": ["frustration", "shame"],
    "sqs": 8
  },
  "relevance_to_opportunity": "Directly expresses ball loss anxiety with financial and emotional components"
}
```

### Step 2: Pattern Evidence

Compile evidence from each relevant Layer 2 analysis:

#### From Effect Mapping (2.1)
- Which effects support this opportunity?
- Consequence chain evidence
- Validation confidence

#### From Audience Analysis (2.2)
- Which segments most affected?
- Intensity by segment
- Jolie Pattern evidence (if applicable)

#### From Gap Analysis (2.3)
- Gap ratio data
- Symptom-cause disconnect evidence
- Opportunity size metrics

#### From Competitive Analysis (2.4)
- Current competitor positioning
- Saturation evidence
- White space confirmation

#### From Timing Analysis (2.5)
- Trend data
- Timing signal evidence
- Window duration estimate

#### From Advanced Patterns (2.6)
- Contrarian signal evidence
- Villain identification
- Transformation arc evidence
- Category creation signals

## Compilation Quality Criteria

- [ ] All Tier 1 opportunities have full packages
- [ ] Quote counts match evidence summary
- [ ] No duplicate quotes within package
- [ ] All quotes trace to Layer 1 source
- [ ] Pattern evidence cites specific Layer 2 outputs
- [ ] Competitive context is current and accurate
- [ ] Handoff notes are actionable
- [ ] Packages are complete (no missing sections)

---

## Evidence Integrity Rules

### Quote Handling
- NEVER paraphrase - exact text only
- ALWAYS include source attribution
- NEVER include quotes with SQS < 3
- ALWAYS note if quote is truncated

### Pattern Handling
- ALWAYS cite source analysis
- NEVER overstate confidence
- ALWAYS note uncertainty
- NEVER invent patterns not found in Layer 2

### Competitive Handling
- USE current data only (from scraping)
- NEVER make assumptions about competitor strategy
- ALWAYS note information gaps

---

## Notes

- Evidence packages are for HANDOFF, not for this system to act on
- Big Idea development is a SEPARATE process
- Packages should enable copywriter to understand opportunity deeply
- Quality > Quantity - better fewer strong quotes than many weak ones
- Output feeds to 3.2-A Handoff Packager
