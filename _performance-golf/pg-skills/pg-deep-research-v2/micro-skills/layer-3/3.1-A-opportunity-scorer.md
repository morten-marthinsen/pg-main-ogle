**Layer:** 3 - Opportunity Surfacing

**Dependency:** All Layer 2 outputs

## Scoring Framework

### Component Weights

| Component | Weight | Source |
|-----------|--------|--------|
| Market Evidence | 25% | Layer 1 quote volume + SQS |
| Emotional Intensity | 20% | Aspect analysis + loss aversion |
| Gap Size | 15% | Gap assessment scores |
| Competitive Space | 15% | Saturation scores |
| Timing Window | 10% | Timing assessment |
| Pattern Strength | 15% | Advanced pattern detection |

**Total: 100%**

---

## Scoring Methodology

### Step 1: Aggregate Opportunities

Collect all opportunities surfaced across Layer 2:
- Effect-based opportunities (from 2.1-B)
- Audience-effect matches (from 2.2-B, including Jolie Pattern)
- Gaps (from 2.3-B)
- White space opportunities (from 2.4-B)
- Timing opportunities (from 2.5-B)
- Pattern-based opportunities (from 2.6 skills)

**Note:** Some opportunities will appear in multiple analyses. Consolidate these.

### Step 2: Score Each Component

#### A. Market Evidence Score (0-10)

```
Base = (Quote Count Supporting / 100) × 5
+ (Average SQS of Supporting Quotes / 10) × 3
+ (Platform Diversity Bonus: 1 per platform, max 2)

Cap at 10
```

**Scoring Guide:**
- 9-10: 100+ quotes, high SQS, multi-platform
- 7-8: 50-100 quotes, good SQS, 2+ platforms
- 5-6: 20-50 quotes, medium SQS
- 3-4: 10-20 quotes, some support
- 1-2: <10 quotes, weak support

#### B. Emotional Intensity Score (0-10)

From aspect analysis and loss aversion:
```
Base = Average Aspect Intensity Score
× Loss Aversion Factor (from 2.6-F)

Cap at 10
```

**Intensity Indicators:**
- Extreme language (hate, love, desperate, finally)
- Personal stakes mentioned
- Repeated emphasis
- Identity-level language

#### C. Gap Size Score (0-10)

From gap assessment (2.3-B):
```
Direct transfer of gap_opportunity_score
```

**Gap Scoring:**
- 9-10: Gap Ratio >5:1, high symptom volume
- 7-8: Gap Ratio 3-5:1, clear pattern
- 5-6: Gap Ratio 2-3:1, moderate gap
- 3-4: Gap Ratio 1-2:1, small gap
- 1-2: Gap Ratio <1:1 or no clear gap

#### D. Competitive Space Score (0-10)

From saturation scoring (2.4-B):

| Saturation Level | Score |
|------------------|-------|
| OPEN | 10 |
| EMERGING | 8 |
| PARTIAL | 6 |
| CROWDED | 3 |
| DEAD | 1 |

#### E. Timing Window Score (0-10)

From timing assessment (2.5-B):

| Timing | Score |
|--------|-------|
| NOW | 10 |
| SOON | 7 |
| WAIT | 4 |
| AVOID | 1 |

#### F. Pattern Strength Score (0-10)

Average of relevant pattern scores from 2.6 skills:
- Contrarian signal strength
- Pattern link confidence
- Villain intensity
- Transformation clarity
- Category creation signals

```
Sum of applicable pattern scores / Number of applicable patterns
```

### Step 3: Calculate Composite Score

```
Composite = (Market Evidence × 0.25)
          + (Emotional Intensity × 0.20)
          + (Gap Size × 0.15)
          + (Competitive Space × 0.15)
          + (Timing Window × 0.10)
          + (Pattern Strength × 0.15)
```

### Step 4: Rank and Tier

**Tier 1 (Pursue):** Composite ≥ 7.5
- Strong across multiple dimensions
- Clear evidence base
- Favorable competitive/timing conditions

**Tier 2 (Consider):** Composite 5.0-7.4
- Solid potential
- May have weaknesses in 1-2 dimensions
- Worth further exploration

**Tier 3 (Monitor):** Composite < 5.0
- Weak or mixed signals
- May develop over time
- Not recommended for immediate pursuit

## Validation Rules

### Score Validation
- All component scores must be 0-10
- Composite score must equal weighted sum
- Tier must match composite score threshold

### Evidence Validation
- Each opportunity must trace back to Layer 2 source
- Supporting data must reference actual Layer 1/2 outputs
- No opportunity scored without evidence base

### Consistency Validation
- Loss aversion weighting already applied (don't double-count)
- Opportunities appearing in multiple analyses consolidated (not duplicated)

---

## Quality Criteria

- [ ] All Layer 2 opportunities aggregated
- [ ] Duplicate opportunities consolidated
- [ ] All components scored with evidence
- [ ] Composite scores calculated correctly
- [ ] Opportunities tiered appropriately
- [ ] Rankings make intuitive sense
- [ ] Key insights generated

---

## Edge Cases

### Low Evidence Opportunities
If an opportunity has strong pattern signal but weak market evidence:
- Score market evidence low
- Note in recommendation: "pattern-based, needs validation"
- May still rank highly if patterns are strong

### Conflicting Signals
If dimensions conflict (e.g., high emotion but crowded space):
- Let composite score reflect balance
- Note conflict in key_insight
- Flag for human review

### New Category Opportunities
If opportunity is truly novel (from 2.6-E):
- Competitive space = 10 (OPEN by definition)
- Gap size may be hard to measure (use pattern strength)
- Note higher uncertainty in confidence level

---

## Notes

- This skill SCORES and RANKS - it does NOT create angles or ideas
- Opportunities remain as surfaced by Layer 2 - no interpretation added
- Output feeds directly to 3.1-B Evidence Compiler
- Human can override scores/rankings after review
- Tier 1 opportunities should be manageable number (3-7 typically)
