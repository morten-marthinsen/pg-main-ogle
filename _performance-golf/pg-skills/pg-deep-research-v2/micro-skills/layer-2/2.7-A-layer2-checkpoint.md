## Two-Phase Review Process

### Phase 1: Agent Critic Review (Automatic)

The agent evaluates:

1. **Confidence Distribution**
2. **Evidence Quality**
3. **Pattern Coherence**
4. **Analysis Coverage**
5. **Potential Biases**

### Phase 2: Human Review (Conditional)

Human review triggered if:
- Any item below confidence threshold
- Conflicting patterns detected
- High-stakes items flagged
- Agent recommends review

---

## Phase 1: Agent Review Criteria

### 1. Confidence Distribution Check

```
FOR EACH Layer 2 output:
  Calculate confidence_score (0-1)

THRESHOLDS:
  High confidence: ≥0.75
  Medium confidence: 0.60-0.74
  Low confidence: <0.60

FLAGS:
  - If >20% of items are low confidence
  - If any Tier 1 opportunity is low confidence
  - If overall average confidence <0.70
```

### 2. Evidence Quality Check

```
FOR EACH pattern/opportunity:
  Count supporting quotes
  Assess quote SQS average
  Check quote diversity (sources)

FLAGS:
  - Pattern with <5 supporting quotes
  - Pattern with average SQS <5
  - Pattern supported by single source only
```

### 3. Pattern Coherence Check

```
FOR EACH cross-pattern insight:
  Verify logical connection
  Check for contradictions
  Assess strategic value

FLAGS:
  - Contradictory patterns (A says X, B says not-X)
  - Orphaned patterns (no connection to opportunities)
  - Circular reasoning detected
```

### 4. Analysis Coverage Check

```
CHECK:
  - All Layer 2 skills produced output
  - All opportunity types represented
  - No major gaps in analysis

FLAGS:
  - Missing skill output
  - Zero results for any skill
  - Imbalanced coverage (e.g., all villains, no transformations)
```

### 5. Potential Bias Check

```
CHECK:
  - Platform bias (patterns from single platform)
  - Sentiment bias (patterns all same sentiment)
  - Recency bias (patterns all from recent data)
  - Confirmation bias (patterns all confirm brief assumptions)

FLAGS:
  - >70% of evidence from single platform
  - >80% of patterns confirm pre-existing assumptions
  - Contradicting evidence ignored or underweighted
```

## Status Determination

### Automatic Pass (No Human Review)
```
IF:
  - All items ≥60% confidence
  - No contradictions
  - No critical flags
  - Coverage complete
THEN:
  status = "approved"
  proceed_to_layer3 = true
  human_review_required = false
```

### Human Review Required
```
IF:
  - Any item <60% confidence
  - Any critical flags raised
  - Agent recommends review
THEN:
  status = "pending_human"
  proceed_to_layer3 = false
  human_review_required = true
  WAIT for human decision
```

### After Human Review
```
IF human approves (with or without modifications):
  status = "approved"
  proceed_to_layer3 = true
  Apply any modifications to advanced_patterns.json

IF human rejects:
  status = "rejected"
  proceed_to_layer3 = false
  Document rejection reason
  Return to relevant Layer 2 skill for revision
```

## Timeout Protocol

```
IF human_review_required AND no response within 24 hours:
  Send reminder notification

IF no response within 48 hours:
  Auto-escalate: Present summary to human with option to:
  - Approve all flagged items
  - Exclude all flagged items
  - Continue waiting

IF no response within 72 hours:
  Default action: Exclude all low-confidence items
  Proceed with reduced opportunity set
  Document timeout in checkpoint log
```
