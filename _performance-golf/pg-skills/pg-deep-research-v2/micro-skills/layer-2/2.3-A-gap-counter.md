## What We're Counting

### Symptoms
The experience/manifestation of a problem:
- Behaviors observed
- Feelings experienced
- Outcomes encountered
- Effects noticed

### Causes
The named problem/root issue:
- Named condition
- Identified cause
- Labeled problem
- Recognized issue

### The Gap
When symptoms are mentioned frequently but the cause rarely named:
- High symptom count
- Low cause count
- = Opportunity to name the problem

---

## Gap Ratio Logic

```
Gap Ratio = Symptom Mentions / Direct Cause Mentions

High ratio (>10): People feel it but don't name it
Medium ratio (5-10): Some awareness but not widespread
Low ratio (<5): Problem is well-understood and named
```

### Why This Matters
- High gap ratio = opportunity to be the one who names the problem
- "If you can name it, you can claim it"
- People are aware of symptoms but don't have language for cause
- Provide the frame, own the solution

---

## Counting Process

### Step 1: Symptom Identification
Identify symptom language patterns:
```
Golf Ball Example:
- "I keep losing balls" (symptom)
- "Putting one in the water" (symptom)
- "Can't find my ball" (symptom)
- "$4 gone every time" (symptom)
- "Had to drop again" (symptom)
```

### Step 2: Cause Identification
Identify direct cause naming:
```
Golf Ball Example:
- "ball loss problem" (cause named)
- "my ball-losing habit" (cause named)
- "ball loss anxiety" (cause named)
- "the lost ball issue" (cause named)
```

### Step 3: Count Both
For each symptom/cause pair:
1. Search quotes for symptom language
2. Search quotes for cause language
3. Count distinct quotes for each
4. Calculate ratio

### Step 4: Significance Assessment
Rate gap significance:
- **High:** Ratio >10, symptom count >50
- **Medium:** Ratio 5-10, symptom count >20
- **Low:** Ratio <5, or symptom count <20

---

## Common Gap Patterns (Golf Example)

| Symptom Language | Cause Language | Gap Ratio |
|-----------------|----------------|-----------|
| "losing balls," "put one in water," "can't find it" | "ball loss," "lost ball problem" | High (~12:1) |
| "slice keeps coming back," "can't stop hooking" | "swing fault," "path issue" | Medium (~6:1) |
| "feel nervous over short putts" | "yips," "putting anxiety" | Medium (~8:1) |
| "don't know what club to hit" | "distance uncertainty" | Low (~4:1) |

## Quality Criteria

- [ ] Symptom patterns identified with variations
- [ ] Cause patterns identified with variations
- [ ] Counts are from distinct quotes (no double-counting)
- [ ] Ratios calculated correctly
- [ ] Significance assessed
- [ ] Representative quotes attached

---

## Failure Modes

- **Conflating symptom and cause:** Counting same language in both
- **Missing variations:** Only counting exact matches
- **Double counting:** Same quote counted multiple times
- **Threshold misapplication:** High ratio on tiny sample size
