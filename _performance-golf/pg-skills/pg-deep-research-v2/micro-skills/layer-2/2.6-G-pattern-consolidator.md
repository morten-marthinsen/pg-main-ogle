Before consolidating, verify all 2.6 skills completed:

```

- 2.6-A contrarian_analysis.json
- 2.6-B pattern_links.json
- 2.6-C villain_analysis.json
- 2.6-D transformation_maps.json
- 2.6-E category_creation_signals.json

IF any input missing:
  HALT consolidation
  Report which input(s) missing
```

## Failure Protocol

```
IF any 2.6 skill output missing:
  HALT
  Report: "Cannot consolidate - missing {skill} output"
  DO NOT create partial advanced_patterns.json

IF any 2.6 skill output malformed:
  HALT
  Report: "Cannot consolidate - {skill} output invalid: {error}"
  DO NOT proceed

IF consolidation completes but >50% items below confidence threshold:
  Set overall_layer2_confidence = "low"
  Flag for mandatory Checkpoint 3 human review
```
