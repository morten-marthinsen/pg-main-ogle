## Assessment Dimensions

### 1. Gap Magnitude
How big is the disconnect?
- **Large:** Ratio >10, high symptom volume
- **Medium:** Ratio 5-10, moderate volume
- **Small:** Ratio <5, or low volume

### 2. Emotional Intensity
How much do people care?
- Scan symptom quotes for emotional language
- Check SQS distribution of symptom quotes
- Note frustration/pain intensity

### 3. Naming Potential
Can this gap be named effectively?
- Is there natural language to build on?
- Would a name resonate?
- Is it concrete enough to visualize?

### 4. Audience Alignment
Which segments experience this gap?
- Map gap to audience segments
- Check intensity per segment
- Identify primary segment affected

### 5. Competitive Context
Is anyone else naming this?
- Check if competitors address this gap
- Is there white space for naming?
- Would naming be differentiated?

---

## Opportunity Score Calculation

| Factor | Weight | Scoring |
|--------|--------|---------|
| Gap magnitude | 25% | Large=10, Medium=6, Small=3 |
| Emotional intensity | 25% | High=10, Medium=6, Low=3 |
| Naming potential | 20% | High=10, Medium=6, Low=3 |
| Audience alignment | 15% | Clear segment=10, Mixed=6, Unclear=3 |
| Competitive white space | 15% | Open=10, Partial=6, Crowded=3 |

**Opportunity Score = Weighted sum (1-10 scale)**

---

## Assessment Process

### Step 1: Review Gap Data
For each gap from 2.3-A:
1. Confirm ratio and counts
2. Review symptom quote samples
3. Assess initial opportunity potential

### Step 2: Emotional Analysis
Analyze symptom quotes:
1. Scan for emotional language
2. Rate emotional intensity
3. Identify dominant emotions

### Step 3: Naming Exploration
Explore potential names:
1. What language do people use?
2. What metaphors emerge?
3. What name would resonate?

### Step 4: Segment Mapping
If audience segments available:
1. Match gap to segments
2. Check segment intensity
3. Identify primary audience

### Step 5: Evidence Chain Building
Build traceable evidence:
1. Select representative quotes
2. Link to source data
3. Create citation chain

### Step 6: Opportunity Scoring
Calculate weighted score:
1. Score each dimension
2. Apply weights
3. Calculate composite

---

## Naming Frame Development

For high-opportunity gaps, develop potential frames:

### Frame Components
- **Name:** What to call the problem
- **Visual:** How to make it concrete
- **Emotional hook:** What feeling it captures
- **Solution link:** How product solves it

### Example: Ball Loss Gap

```
Raw symptom: "I keep putting balls in the water"

Potential frames:
1. "Ball Loss Anxiety" - Names the emotional state
2. "The $4 Splash" - Makes the cost vivid
3. "Premium Ball Paralysis" - Names the hesitation
4. "Lost Ball Tax" - Frames as ongoing cost
```

## Confidence Levels

**High Confidence:**
- Large sample size (50+ quotes)
- Consistent pattern across sources
- Clear emotional signal
- Strong evidence chain

**Medium Confidence:**
- Moderate sample (20-50 quotes)
- Some source variation
- Mixed emotional signal
- Adequate evidence

**Low Confidence:**
- Small sample (<20 quotes)
- Limited sources
- Unclear signal
- Weak evidence - flag for review

## Notes

- This is interpretation phase - build on 2.3-A quantification
- Naming frames are suggestions, not final
- Evidence chains must be traceable
- Flag low confidence for human review
