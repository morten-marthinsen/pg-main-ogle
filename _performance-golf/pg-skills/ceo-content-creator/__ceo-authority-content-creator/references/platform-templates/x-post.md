# X/Twitter Post Template

**Formats:** Single Tweet, Thread
**Voice Calibration:** Punchy, direct, high density of insight per word

---

## Format 1: Single Tweet (<280 characters)

### Structure Options

**Option A: The Insight**
```
[Counterintuitive observation].

Most think X.
But Y is true because Z.
```

**Option B: The List**
```
[Number] things that [outcome]:

• Item 1
• Item 2
• Item 3
• Item 4
• Item 5
```

**Option C: The Hot Take**
```
Unpopular opinion: [Statement].

Here's why: [reasoning in 1-2 sentences]
```

**Option D: The Question**
```
[Provocative question]?

Genuine question. Reply with your take.
```

**Option E: The Proof**
```
[Result achieved].

Here's exactly how:
• Step 1
• Step 2
• Step 3
```

---

### Single Tweet Examples

**The Insight:**
```
The best golfers don't practice more.

They practice less—on the right things.

Systems > hours.
```

**The Hot Take:**
```
Unpopular opinion: Most swing tips make you worse.

Not because they're wrong. Because they conflict with the 5 other tips you're trying to remember.

One system beats 12 tips. Every time.
```

**The Question:**
```
Why does your swing work on the range but fall apart on the course?

It's not nerves. It's something else.

(Reply if you want me to break it down)
```

## Format 2: Thread (8-15 Tweets)

### Structure

| Tweet | Purpose | Content |
|-------|---------|---------|
| 1 | Hook | Bold claim + promise + 🧵 |
| 2 | Setup | Why this matters, context |
| 3-X | Content | Numbered points, one per tweet |
| X-1 | TL;DR | 5 bullet summary |
| X | CTA | Follow, RT, link |

---

### Thread Template

**TWEET 1 (Hook):**
```
[BOLD CLAIM OR PROMISE]

[What they'll learn in 1 sentence]

🧵 Thread:
```

**Example:**
```
I've spent 10 years running experiments on how to get good at anything fast.

Here are 7 lessons that changed everything:

🧵
```

---

**TWEET 2 (Setup):**
```
First, context:

[Why you're qualified to speak on this]
[What led you to these insights]
[What's at stake for the reader]
```

**Example:**
```
First, some context:

At 6, I had a speech impediment. Therapist said I'd need a year.

I did it in a week.

That moment sparked a 20-year obsession: What's the fastest path to mastery?

Here's what I've learned:
```

---

**TWEETS 3-X (Content):**
```
[NUMBER]. [Point in 5-10 words]

[2-3 sentences explaining]

[Optional: specific example or data point]
```

**Example:**
```
1. If everyone says it takes 5 years, ask how to do it in 6 months.

Not by working harder. By identifying the 20% that drives 80% of results.

Most people spread effort thin. Concentrated effort compounds.
```

---

**TWEET X-1 (TL;DR):**
```
TL;DR:

• Point 1 (5 words)
• Point 2 (5 words)
• Point 3 (5 words)
• Point 4 (5 words)
• Point 5 (5 words)
```

---

**TWEET X (CTA):**
```
If this thread was valuable:

1. Follow @brixtonalbert for more on [topic]
2. RT the first tweet to share with others

[Optional: Link to longer content]
```

## X Voice Adjustments

### Compared to LinkedIn:
- Shorter sentences (10 words average)
- More fragments
- Higher opinion density
- Less hedging, more conviction
- Punchier, more direct

### Signature Phrases (Adapted for X):
- "Systems > effort" — Works perfectly
- "Right things, right order" — Shorten to "Right things. Right order."
- "Less guesswork. More fireworks." — Works perfectly

### What to AVOID:
- Long paragraphs
- Hedging language ("I think", "perhaps")
- Links in first tweet (kills reach)
- Hashtags (not X culture)
- Thread longer than 15 tweets

---

## X Algorithm Signals (2025)

**Optimize for:**
- Replies (highest weight)
- Quote tweets (2x engagement value)
- Time spent on tweet
- Profile clicks
- Early engagement (first 30 min)

**Best Practices:**
- Threads: 8-15 tweets optimal
- Quote tweets > retweets
- Reply to comments quickly
- No links in main tweets (put in reply)
- Post 2-4x daily for growth

## Content Ideas by Pillar

- "I've been running a 10-year experiment on [topic]. Here's what I learned:"
- "If everyone says it takes [X], ask how to do it in [Y]."
- "The week vs. year story" (speech impediment) — works great as thread

### The Arena
- Match recap threads
- "This round taught me [X]"
- "When pressure revealed [insight]"
- Hot takes on golf/competition

### The Innovator
- "Why [common approach] doesn't work"
- "The system I wish existed"
- "How to think about [topic]" frameworks
- "The Scattered Playbook" breakdown
