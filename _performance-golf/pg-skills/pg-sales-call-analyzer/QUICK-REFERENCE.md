# Sales Call Analyzer - Quick Reference Guide

Fast lookup for the sales call analysis workflow.

## Workflow at a Glance

```
1. Structural Architecture → 2. Language Extraction → 3. Psychological Analysis →
4. Timing & Pacing → 5. Objection Handling → 6. Customer Psychology →
7. Value Proposition → 8. Trust Building → 9. Closing Strategy →
10. Implementation Output
```

## The 6-Phase Conversion Framework

| Phase | Duration | Purpose | Key Elements |
|-------|----------|---------|--------------|
| **1. Rapport & Context** | 5-10 min | Establish comfort | Greeting, confirmation, recording consent, preview |
| **2. Deep Discovery** | 15-20 min | Uncover pain | Questions, excavation, quantification, history |
| **3. Problem Amplification** | 10-15 min | Quantify pain | Statistics, consequences, patterns, cost |
| **4. Solution Architecture** | 10-15 min | Position solution | Root flaw, systematic approach, technology |
| **5. Authority Building** | 10-15 min | Build credibility | Coach positioning, tour players, rankings |
| **6. Value & Price** | 15-20 min | Stack value, close | Components, anchor, discount, payment |

## Critical Transition Phrases

| Phrase | Purpose | When to Use |
|--------|---------|-------------|
| "Based on what you've told me..." | Analytical bridge | After discovery, before solution |
| "That being said..." | Logical transition | Between phases |
| "Here's what we do..." | Solution introduction | After problem amplification |
| "The way I see it..." | Expert positioning | During analysis |
| "Let me tell you about our process..." | Authority establishment | Before value stacking |

## Language Pattern Categories

### Discovery Phase Power Questions

- "Tell me about your golf game" (not "how's")
- "What are your biggest missed opportunities?"
- "Where do you see strokes being left out there?"
- "What would be a big success story for you?"
- "How long have you been struggling with this?"
- "What have you tried before?"

### Problem Amplification Phrases

- "You're missing 40% of your shots off the tee"
- "That would save me 5-6 strokes easily"
- "Having to get back out to the fairway"
- "One or two holes that were just my nemesis holes"
- "Throwing a double up there"
- "Kick me out of the seventies"

### Authority Building Language

- "The Sultan of the Swing" (memorable positioning)
- "They call him..." (third-party positioning)
- "Hand selected... ranked top 50 in the world"
- "Gets flown out privately... $30,000 a day"
- "Best young coach in America"
- "We work with Tiger Woods, Dustin Johnson, Phil Mickelson"

### Root Flaw Positioning Scripts

- "Every golfer has 5-7 swing flaws, but there's ONE root flaw"
- "Until you fix the root cause, symptoms never go away"
- "Working on swing flaws in the wrong order"
- "Like a combination lock - all numbers but wrong sequence"
- "Treating symptoms vs. fixing the core health issue"

### Assumption-Based Closing Language

- "What would hold you back from pulling the trigger today?"
- "As long as we get it done over the next couple hours"
- "When you go to the academy..."
- "After you leave the golf school..."
- "You're gonna have an incredible experience..."

## Cialdini's Principles Quick Reference

| Principle | How Applied | Example from Calls |
|-----------|-------------|-------------------|
| **Social Proof** | Tour players, rankings, elite pricing | "Tiger, DJ, Phil" / "$30K/day" |
| **Authority** | Third-party positioning, credentials | "They call him the Sultan" |
| **Scarcity** | Limited spots, coach availability | "I have only one more spot" |
| **Commitment** | Small agreements throughout | "Makes sense?" "Right?" |
| **Liking** | Rapport, shared golf passion | Warm greeting, empathy |
| **Reciprocity** | Free value, insights first | Education before selling |

## Psychological Techniques Quick Reference

| Technique | Application | When to Use |
|-----------|-------------|-------------|
| **Loss Aversion** | Past performance vs. current decline | Discovery phase |
| **Anchoring** | $15K/$12K before $11K reveal | Value stacking phase |
| **Contrast** | Elite vs. local instruction | Authority building |
| **Cognitive Dissonance** | Root flaw explains past failures | Solution architecture |
| **Pain/Relief** | Amplify then solve | Throughout call |

## Timing Blueprint Quick Reference

| Minutes | Phase | Accomplish |
|---------|-------|------------|
| **0-5** | Rapport | Warm greeting, recording consent, preview |
| **5-20** | Discovery | All pain points uncovered, quantified |
| **20-35** | Amplification + Solution | Problems systematized, root flaw introduced |
| **35-50** | Authority + Value | Coach credentials, value stacking |
| **50-60** | Price + Close | Anchor, discount, payment processing |
| **60+** | Confirmation | Next steps, excitement building |

## Objection Handling Quick Matrix

| Objection Type | Neutralization Strategy | Key Phrases |
|----------------|------------------------|-------------|
| **Price** | Scholarship reframe, value stacking, anchoring | "For guys outside PG, it's 15K. We give you 11K" |
| **Timing** | Flexibility positioning, travel as value | "35 locations, 20 coaches" / "Get away for a couple days" |
| **Skepticism** | Differentiator positioning, process transparency | "Root flaw vs. symptoms" / "Wrong order/sequence" |
| **Spouse** | Permission-based processing, involvement | "What card would you like to use?" |

## Elite Closing System Quick Reference

### Building to Inevitable Decision

1. No "Would you like to buy?" questions
2. Assumes decision is logical conclusion
3. "What would hold you back?" presumes readiness
4. Direct transition to logistics
5. Hesitation as administrative details

### Multiple Commitment Layers

- Interest confirmation
- Scheduling availability
- Coach fit acknowledgment
- Investment acceptance
- Payment method provision
- Future session booking

### Payment Processing

- Administrative detail treatment
- Matter-of-fact card requests
- Verification steps increase commitment
- Never reverse direction
