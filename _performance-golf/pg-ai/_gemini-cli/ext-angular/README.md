# Angular Extension for Gemini CLI
A Gemini CLI extension to help you build your next great web application with Angular.

## Prerequisites
1. Gemini CLI installed and configured
2. Node.js 20+ and npm

## Installation
### 1. Install the extension
Install the extension using the `gemini extensions install` command:
`gemini extensions install https://github.com/gemini-cli-extensions/angular`

### 2. Activate
- Restart Gemini CLI
- After restarting Gemini CLI the following will be configured and available:
    - [Angular MCP Server](https://angular.dev/ai/mcp)
    - [Angular System Instructions and Best Practices prompts](https://angular.dev/ai/develop-with-ai#custom-prompts-and-system-instructions)
