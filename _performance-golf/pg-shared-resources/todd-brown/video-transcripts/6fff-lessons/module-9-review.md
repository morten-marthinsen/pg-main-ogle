# Module 9 Review

**Date:** 2014-03-01 | **Duration:** 1m | **Folder:** 6FFF Lessons

---

Okay, so here's what we covered in this module. We started by talking about why tracking is mandatory. Then I showed you how to get started with HyperTracker, how to set up a tracking campaign,
how to track opt-ins, sales, how to track your individual transaction maximization offers,
and what the different tracking metrics mean. Here's what to do now. First, create a HyperTracker account.

Second, add the action integration code on your customer captivation offer. This is to track opt-ins. Then add the appropriate sales integration code on each of your transaction maximization pages. That's to track sales for each of those offers. Just be sure that you remove the little and sales equals 1 from the sales integration code
that I demonstrated for you in lesson 2.

This way, the total sales number remains accurate.
