# Module 6 Review

**Date:** 2014-03-01 | **Duration:** 1m | **Folder:** 6FFF Lessons

---

Okay, so here's what we covered in this module. We started out talking about the purpose of the
transaction maximization sequence. Then we went through the reverse add-on, the soft add-on,
and then the mid-tier add-on. Then we went through the key elements of transaction maximization
pages, and then we covered the thank you page gift offer add-on. Here's what to do now.

First,
decide what your reverse add-on offer is going to be. Then do the same for the soft add-on offer,
the mid-tier add-on offer, and then finally the thank you page gift offer. If you're using SamCart,
set up each of the transaction maximization offers and pages inside of SamCart. And if you're not
using SamCart, then do the same thing on your own domain with your own CRM. Now a quick important
note, you do not, I repeat, do not have to start with all three transaction maximization offers.

You can add them as you create them. I recommend that you start one at a time and add them in as
you build them. There's no need to wait until you have all three transaction maximization offers built. Get started and add them in as you create them.
