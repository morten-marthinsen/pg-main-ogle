# Module 8 Lesson 2

**Date:** 2014-03-12 | **Duration:** 1m | **Folder:** 6FFF Lessons

---

In this lesson, I want to show you how easy it is to password protect a web page or post
in WordPress. Now, like I said in our prior lesson, this is the method that I recommend
you use to secure your digital content, whether it be videos, audios, software, anything digitally
delivered. This is an easy, fast way to secure it when you're first launching your new funnel. Now, I'm inside of WordPress and what you're about to see can be used for a post or a page. Now, what you'll see if you're familiar with WordPress is this section over here that says
publish.

Now, right now, the visibility is set to public, which means that if somebody went to this
page, they'd be able to view all of the content. But if we simply click on edit, you'll see that
there is public and there's password protected and there's private. Well, to make this a password
protected page, all we do is click this little radio button and then you enter password right
here. Then you click OK, click update, and that's all there is to it. And now this page
is password protected.

Now, when somebody goes to view this page, what they'll see is a little blurb
that says the content you're trying to view is password protected, and there'll be a little field
for them to enter the password. The key is that you take this password and you put it either on the
thank you page that new customers come to after they purchase your product and or you include it in
the thank you, welcome email for new customers so they know exactly what this password is. Again,
start with this and then you can move on to the more expensive options.
