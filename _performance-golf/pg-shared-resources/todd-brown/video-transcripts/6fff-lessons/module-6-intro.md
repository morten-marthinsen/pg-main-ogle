# Module 6 Intro

**Date:** 2014-03-01 | **Duration:** 0m | **Folder:** 6FFF Lessons

---

Welcome to Module 6. In this module, we're going to cover the Six Figure Funnel Formula Transaction
Maximization Sequence. This is a special sequence that comes after the customer captivation offer
that's designed to increase the average transaction size of every new customer. Now, we're going to start by talking about the purpose of the transaction maximization sequence. Then we're going to get into maximization offer number one called the reverse add-on.

We're going to talk about maximization offer number two, the soft add-on. Maximization offer
number three, the mid-tier add-on. We're going to talk about the key elements of transaction
maximization pages. And then we're going to talk about something called the thank you page
gift offer add-on.
