In this lesson, we're going to cover how to use split testing for campaign enhancement. So what is split testing? Well, it's also called A-B testing, and you'll understand why in just
a second. And this is the process of showing two variants of the same webpage to different
segments of website visitors at the same time, and then comparing which variant drives more
conversions. And this is really the beauty of direct response marketing, because everything
is trackable and measurable and quantifiable, because everything is based on numbers or driven
with data, produces metrics.

We're able to change any piece or part or component of any marketing
campaign and then use split testing to see if that change had any impact on the performance
of the campaign. This is why I say that this split testing is at the foundation of being
able to improve the performance of any campaign and do it scientifically, do it intelligently,
do it based on numbers, because split testing is based on data and it gives us an objective
result. So we don't need to guess anymore of whether a new headline or a different headline
will improve performance or hurt performance. We're able to just use split testing and test
the current headline against what we think is a better headline and then get the data back
from the split testing and see which one was better. And so this is how we are able to over
time strategically and methodically improve the result of any campaign.

Now, the way this
works is really simple. What happens is, let's say you're running an ad. Well, what happens

another, or we could simply split it 50-50.

So we're taking the traffic from an ad. And
at the very same time, we're splitting it between two different versions of the same webpage. Now, the reason why this is called A-B testing is because one version is considered A and the
other version is considered B. Now, to get a little bit more formal for a second, the current
version that you have in place at any given point, the version of a webpage, the version
