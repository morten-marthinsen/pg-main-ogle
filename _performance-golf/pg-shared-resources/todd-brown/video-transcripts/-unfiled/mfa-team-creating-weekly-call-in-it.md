# MFA Team - Creating Weekly Call in IT

**Date:** 2014-12-20 | **Duration:** 1m | **Folder:** _unfiled

---

To set up a weekly call inside of Instant Teleseminar, the first thing we do is log in. And you'll have a link and username and password to do that. Once inside, the first thing we want to do is left-click on Add New Event. We want to go over here where it says Event Name and we want to change it to or type in MFA Weekly Call. This is a call that happens every week on Friday at 3 p.m.

Eastern Time. So, then you want to go under Event Date and you want to make sure that you're choosing the upcoming Friday. So, in this case, I'm going to schedule this for Friday the 26th. The official start time is always 3 p.m. Eastern Time.

Under Duration, you want to left-click on this and you want to choose 4 hours. This is just to make sure that we set it up so that we can go as long as is necessary. Then, under this section over here, which instead of Guest Mode, we click on this and we choose Host Mode. And then, under this section, one more section, we choose Phone Only Includes Skype. So, we left-click on this radio button and we click Next and that's it.

Once we see this page that says Congratulations, the event is set up. And so, we need an event created once a week every Friday in the morning or Thursday night for 3 p.m. Eastern Time.
