# Measuring investor sentiment

at or near the highest price for the last  months, the System
makes the assumption that the sentiment is at 00%. In a market at

Let’s say prices have been moving up. In that case, the current
price relative to the price for the last  months is an important part
of the calculation, and it should yield a number above 0%. When

-- and any sign of a new move up in price is the signal to get back in
for another possible leg up.
But if the sentiment has been running very high -- above 90% --
then any pullback might be dangerous, because it may be followed
by one final exhaustive run to the top, which would be a sell signal.
