## Proof and Claim Rules (Non-Negotiable)

- Every quantified claim must be classed as verified or qualified.
- No guaranteed mph/fairway claims without proprietary validation.
- Comparative claims must be evidence-backed or explicitly qualified.
- Mechanism language must map to directional control and transfer, not raw speed only.

## Output Package (What This Document Must Produce)

When used correctly, this blueprint should output:

1. Section-by-section copy brief (16 sections)
2. Claim table (verified / qualified / blocked)
3. Evidence map (which proof supports which claim)
4. Comparison table (SPD vs speed-only alternatives)
5. Draft-ready page outline for:
- front-end sales page
- PDP variant (hero delta only)
